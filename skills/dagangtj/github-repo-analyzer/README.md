# GitHub Repo Analyzer

Analyze any GitHub repository instantly - stars, issues, contributors, activity, and tech stack.

## Features

- 📊 Repository statistics (stars, forks, issues)
- 👥 Contributor analysis
- 📈 Activity trends
- 🔧 Tech stack detection
- 🐛 Issue analysis

## Installation

```bash
clawhub install github-repo-analyzer
```

## Usage

- "Analyze github.com/facebook/react"
- "Show me stats for vercel/next.js"
- "What tech stack does tailwindcss use?"
