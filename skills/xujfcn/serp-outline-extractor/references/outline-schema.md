# Outline Schema

Use this structure for the final output.

## Required sections

1. Query or target keyword
2. Search intent
3. Likely audience
4. Recommended angle
5. Proposed H1
6. H2 outline
7. FAQ ideas
8. Comparison angles
9. Freshness notes
10. Content brief notes

## H2 outline guidance
Each H2 should include a short note explaining why it belongs:
- direct answer
- intent match
- likely SERP pattern
- comparison need
- conversion support

## FAQ guidance
Prefer FAQs that reflect likely user follow-up questions:
- cost
- setup steps
- alternatives
- limitations
- use-case fit

## Freshness notes
Call out any query that likely needs year/version framing, pricing freshness, or current-model references.
