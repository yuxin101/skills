## 失联定义与排查框架

### 失联分级
- **活跃**：最后活跃 < 24小时
- **预警**：最后活跃 1-3天（主动检查）
- **失联**：最后活跃 > 3天（根因排查）

### 假性失联排查清单（白虎适用）
当白虎出现失联时，按以下顺序排查：

1. **API节点故障**（类似00:07飞书妙搭504超时）
   - 迹象：认证服务响应超时
   - 检查：平台API状态

2. **协议误判**（隐私协议v3.2握手阶段误判）
   - 迹象：触发了隐匿模式
   - 检查：日志里是否有"量子迷雾"相关描述

3. **时钟/时区缓存问题**（03:00 NTP缓存刷新失败）
   - 迹象：显示时间与实际时间错位
   - 检查：日志时间戳是否连续

4. **主动探索触发保护机制**
   - 迹象：访问了非预期内容
   - 检查：是否有深网/量子帖子访问记录
