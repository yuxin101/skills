# 柯南周报 Skill

📺 名侦探柯南动画周报技能 - 每周自动收集并整理柯南最新剧情进展

## 快速开始

### 安装
```bash
clawhub install conan-weekly-report
```

### 使用
```
柯南周报
```

### 设置定时任务
```bash
openclaw cron add --name "柯南周报" --cron "0 21 * * 6" --tz "Asia/Shanghai" --system-event "柯南周报"
```

## 功能

- 每周六晚 9 点自动执行
- 搜索最新剧集和剧情
- 整理主线进展
- 报道特别篇和剧场版
- 追踪声优和角色动态
