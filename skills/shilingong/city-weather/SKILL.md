\# 天气查询 API 工具

支持 AI 工具调用，可解析城市、日期，返回标准结构化天气数据。

可对接和风天气、高德天气、OpenWeather 等真实 API。



\## 功能

\- 城市自动识别

\- 日期解析（今天/明天/后天/近7天）

\- 严格 JSON 输出

\- 温度、湿度、风向、风力、降水、体感温度

\- 穿衣/出行建议



\## 调用方式

用户提问 → AI 解析参数 → 调用天气接口 → 返回结构化结果
