\# 天气 API 对接说明



\## 支持的接口

\- 和风天气 API

\- 高德开放平台天气接口

\- OpenWeatherMap



\## 请求示例（模拟）

GET https://api.example.com/weather?city=北京\&date=today



\## 返回字段

\- condition：天气状况

\- temp\_min / temp\_max：温度范围

\- humidity：湿度

\- wind：风向风力

\- precip：降水概率

\- feels\_like：体感温度
