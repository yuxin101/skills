# 浏览器工作流

本文档描述本技能推荐采用的浏览器侧执行路径。

## 职责划分

### Node 脚本负责

- 写入平台词库
- 去重整理词条
- 选择今日单词
- 生成并发送消息

### OpenClaw agent + browser tool 负责

- 打开平台页面
- 检查登录状态
- 在需要时提示登录
- 抓取当前页内容
- 在存在稳定流程时继续翻页补充缓存
- 在运行 `npm run sync` 前写入缓存文件

## Google Translate Saved

### 目标地址

`https://translate.google.com/saved?sl=en&tl=zh-CN&op=translate`

### 推荐流程

1. 打开目标地址
2. 对当前页面做 snapshot
3. 判断页面状态：
   - 若出现登录提示，则停止并要求先登录
   - 若出现收藏页标记与词条列表，则继续
4. 抓取当前页内容
5. 若存在稳定的自动翻页流程，则继续抓取更多页面
6. 将结果写入 `data/cache/google-page.txt`
7. 运行 `npm run sync -- google`
8. 返回同步结果，并说明覆盖范围

### 成功标准

- 页面可以明确识别为收藏词页面
- 抓取内容中可解析出词条
- 若启用翻页，更多页面可以稳定遍历
- 最终结果同时包含条数与覆盖范围说明

## 有道单词本

### 目标地址

`https://dict.youdao.com/webwordbook/wordlist`

### 推荐流程

1. 打开目标地址
2. 检查登录状态
3. 若未登录，则停止并要求先登录
4. 若已登录但页面不对，则要求切换到单词本页面
5. 抓取当前页内容
6. 若存在稳定的翻页流程，则继续抓取更多页面
7. 将结果写入 `data/cache/youdao-page.txt`
8. 运行 `npm run sync -- youdao`
9. 返回同步结果，并说明覆盖范围

### 成功标准

- 页面可以明确识别为单词本 / 收藏页
- 抓取内容中可解析出单词与释义
- 最终结果同时包含条数与覆盖范围说明

## 执行约束

- 不要默认登录已完成
- 未验证覆盖范围前，不要声称已完成全库同步
- 如果只抓取了当前页，必须明确说明限制
- 解析失败时，要区分“未登录”“页面不对”“页面结构变化”
