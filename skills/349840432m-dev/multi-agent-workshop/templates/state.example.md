**目的**：在投入选角与多轮讨论前，锁定 **交付物类型**，避免「要方案却给成稿」或相反。

**期望交付物类型**（必选其一或写明组合；禁止长期留白「待议」）：

- [ ] **`methodology`** — 方法论 / 流程 / SOP / 检查清单（**教怎么做**，默认**不**交付完整成稿）
- [ ] **`artifact`** — 成稿正文（单篇可发布文章、对外声明、逐字脚本等）
- [ ] **`outline_sample`** — 大纲 + 样例段落（非全文）

- [ ] **`mixed`** — 混合：〈方案部分〉 + 〈成稿部分〉（写清边界与先后顺序）
