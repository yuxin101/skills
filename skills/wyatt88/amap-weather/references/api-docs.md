# 高德天气 API 参考

## API 端点

```
GET https://restapi.amap.com/v3/weather/weatherInfo
```

## 请求参数

| 参数 | 必填 | 说明 |
|------|------|------|
| key | 是 | 高德 Web 服务 API Key |
| city | 是 | 城市 adcode（6位数字） |
| extensions | 否 | `base`（实况）或 `all`（预报） |
| output | 否 | `JSON`（默认）或 `XML` |

## adcode 说明

adcode 是高德定义的区域编码（6位数字），可精确到区县级别。

常用城市 adcode：
- 北京 110000 / 朝阳 110105 / 海淀 110108
- 上海 310000 / 浦东 310115
- 广州 440100 / 深圳 440300
- 杭州 330100 / 成都 510100 / 武汉 420100

完整列表: https://lbs.amap.com/api/webservice/download (AMap_adcode_citycode.xlsx)

## 返回字段

### 实况天气 (extensions=base → `lives[]`)

| 字段 | 说明 | 示例 |
|------|------|------|
| province | 省份 | 北京 |
| city | 城市 | 北京市 |
| adcode | 区域编码 | 110000 |
| weather | 天气现象 | 晴 |
| temperature | 实时气温 (°C) | 28 |
| winddirection | 风向 | 南 |
| windpower | 风力 (级) | ≤3 |
| humidity | 湿度 (%) | 45 |
| reporttime | 数据发布时间 | 2026-03-23 17:00:00 |

### 预报天气 (extensions=all → `forecasts[].casts[]`)

| 字段 | 说明 |
|------|------|
| date | 日期 |
| week | 星期 (1-7) |
| dayweather / nightweather | 白天/夜间天气 |
| daytemp / nighttemp | 白天/夜间温度 (°C) |
| daywind / nightwind | 白天/夜间风向 |
| daypower / nightpower | 白天/夜间风力 (级) |

预报数据包含当天 + 未来3天，共4天。

## 错误码

| infocode | 含义 |
|----------|------|
| 10000 | 成功 |
| 10001 | key 不正确或过期 |
| 10002 | 没有权限使用相应的服务 |
| 10003 | 访问量超限 |
| 10004 | 缺少必填参数 |
| 20000 | 请求参数非法 |
| 20800 | IP 白名单限制 |

## 配额

免费额度: 每日 300,000 次（2025年数据）。
更新频率: 实况每小时多次, 预报每天 8:00/11:00/18:00 左右更新。
