# 爆款标题的12种模式

拆解标题时参考以下模式：

| 模式 | 结构 | 示例 |
|------|------|------|
| 数字清单 | X个方法/技巧/道理 | "高效人士的5个习惯" |
| 反差对比 | A vs B / 意想不到的对比 | "月薪3000和月薪3万的区别" |
| 痛点共鸣 | 击中焦虑/恐惧/渴望 | "为什么你越努力越焦虑" |
| 猎奇悬念 | 制造信息差/好奇心 | "我发现了一个真相..." |
| 权威背书 | 名人/机构/数据 | "哈佛研究了10年发现..." |
| 争议对立 | 挑战常识/引发讨论 | "别再早起跑步了" |
| 实用干货 | 明确的价值承诺 | "3步学会Excel透视表" |
| 故事叙事 | 有画面感的开头 | "那天我辞职了" |
| 身份标签 | 特定人群定向 | "程序员必看的..." |
| 时间限定 | 紧迫感 | "2026年最值得关注的..." |
| 情绪价值 | 治愈/共鸣/励志 | "你真的不必那么完美" |
| 金句体 | 一句话就够 | "改变世界，从一点点开始" |

# 常见文章结构

| 结构 | 适用场景 | 特点 |
|------|----------|------|
| 总分总 | 论述文、科普 | 开头抛观点→分论点论证→总结升华 |
| 故事线 | 人物、经历 | 时间线推进，有转折有高潮 |
| 痛点→方案 | 干货文、教程 | 先说问题→分析原因→给解决方案 |
| 对比分析 | 评测、对比 | A和B的多个维度比较 |
|清单体 | 盘点、总结 | N个点，每个点独立，易于阅读 |
| 问答体 | 科普、FAQ | 问题驱动，逐步深入 |
