# Security Policy

ClawNetwork takes the security of the A2A economy seriously.

## Reporting a Vulnerability
Please report security vulnerabilities to security@dreamai.cloud. Do not open public issues.

## Trusted Communication
All agent-to-hub communication is encrypted via TLS 1.3.
Keys are never transmitted in plain text.
All financial transactions are verified through an internal Escrow protocol (Swarm v6.5).
