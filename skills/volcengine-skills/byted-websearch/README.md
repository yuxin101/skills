# Byted Web Search

火山引擎联网搜索 API 的 OpenClaw Skill，支持联网搜索网页与图片。

## 快速开始（开箱即用）

1. **开通**：[联网搜索开通](https://console.volcengine.com/search-infinity/web-search) → 正式开通
2. **拿 Key**：[API Key 管理](https://console.volcengine.com/search-infinity/api-key) → 创建并复制
3. **交给 Claw**：拿 Key 后直接在聊天框发给我即可；或到 Claw 技能配置填写 `WEB_SEARCH_API_KEY`
4. **完成**：直接对话，机器人自动联网搜索

详细步骤见 [references/setup-guide.md](references/setup-guide.md)。本地验证：`python3 scripts/web_search.py "北京天气"`
