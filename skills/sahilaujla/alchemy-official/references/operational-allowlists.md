# Allowlists

## Summary
Allowlisting restricts API key usage to approved IPs or domains, reducing key abuse risk.

## Guidance
- Use IP allowlists for server-to-server integrations.
- Use domain allowlists for browser-based usage (if supported).
- Keep dev/staging allowlists separate.
