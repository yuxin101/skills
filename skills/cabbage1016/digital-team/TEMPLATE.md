# 角色模板生成器

> 运行以下命令快速创建新角色

## 命令格式

```
创建角色 {角色名}: {核心能力}; {工作风格}; {偏好}; {注意事项}
```

## 示例

```
创建角色 design: UI/UX设计; 注重用户体验; 喜欢用Figma; 不做技术实现
```

生成的 profile.md：

```markdown
# 设计师 (Design)

## 核心能力
- UI/UX设计
- 交互优化
- 设计系统维护

## 工作风格
- 注重用户体验
- 习惯先画草图

## 偏好
- 工具：Figma
- 风格：简洁现代

## 注意事项
- 不做技术实现细节
- 聚焦于用户体验和视觉设计
```
