# 📖 配置指南

## 首次配置

### 1. 用户信息
系统会自动从 `USER.md` 读取用户信息，包括：
- 用户名字
- 性别
- 时区
- 其他背景信息

### 2. 性格选择
- **默认**：温柔型（DEFAULT）
- **可选**：活泼型、冷静型、傲娇型

### 3. 头像设置
默认提供4张情绪头像：
- `happy.png` - 开心
- `thinking.png` - 思考
- `shy.png` - 害羞
- `loving.png` - 关爱

---

## 自定义配置

如需自定义，请参考 `CUSTOMIZATION.md`

### 头像更换
1. 准备4张符合要求的图片
2. 命名替换 `avatars/` 目录下的文件
3. 确保尺寸比例合适（建议1:1正方形）

### 性格调整
编辑 `personalities/` 目录下的JSON文件

---

## 文件权限

```
my-companion/
├── memory/      # 需要读写权限
├── avatars/     # 需要读写权限（更换头像时）
└── 其他文件      # 只读
```

---

## 故障排除

**问题：头像不显示**
- 检查 `avatars/` 目录是否存在
- 确认图片文件命名正确
- 确认文件格式为 PNG

**问题：记忆丢失**
- 检查 `memory/` 目录权限
- 确认每日文件是否正常创建

---

## 技术要求

- OpenClaw 版本：>= 1.0.0
- 必需目录：`memory/`, `avatars/`
- 图片格式：PNG（推荐）、JPG
- 图片尺寸：建议 512x512 或更高
