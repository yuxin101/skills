## ⚠️ 重要说明

### 免费 vs 付费

| 类型 | 音色前缀 | 状态 |
|------|---------|------|
| **免费通用** | `zh_female_*_uranus_bigtts` | ✅ 默认可用 |
| **免费通用** | `zh_male_*_uranus_bigtts` | ✅ 默认可用 |
| **角色扮演** | `saturn_zh_*_tob` | ⚠️ 需开通 |
| **特殊音色** | `zh_female_peiqi_*` | ⚠️ 可能需授权 |

### 如何确认音色状态

1. 登录火山引擎控制台
2. 进入「语音合成」→「音色管理」
3. 查看音色状态：
   - ✅ 已开通 → 可直接使用
   - ⚠️ 未开通 → 点击开通（可能付费）
   - ❌ 已过期 → 需续费

## 🧪 测试建议

### 第一步：测试免费音色
```bash
python3 scripts/seedtts2-demo.py
# 默认测试：Vivi 2.0 + 儒雅逸辰 2.0
```

### 第二步：逐个测试兴趣音色
```python
from tts_client import SeedTTS2

tts = SeedTTS2()

# 测试女声
tts.say("你好，这是测试", speaker="zh_female_vv_uranus_bigtts")

# 测试男声
tts.say("你好，这是测试", speaker="zh_male_ruyayichen_uranus_bigtts")

# 测试英文
tts.say("Hello, this is a test", speaker="en_female_dacey_uranus_bigtts")
```

### 第三步：开通特殊音色（如需要）
1. 控制台确认音色状态
2. 点击开通（注意费用）
3. 等待生效（通常即时）
4. 测试调用

## 📞 音色选择建议

**问自己三个问题**：

1. **目标听众是谁？**
   - 儿童 → 佩奇猪、儿童绘本
   - 年轻人 → 甜美小源、爽朗少年
   - 商务人士 → 儒雅逸辰、知性灿灿

2. **内容类型是什么？**
   - 情感类 → Vivi 2.0、魅力女友
   - 知识类 → 儒雅逸辰、大壹
   - 搞笑类 → 猴哥、调皮公主

3. **预算是多少？**
   - 零预算 → 通用场景音色（19 种免费）
