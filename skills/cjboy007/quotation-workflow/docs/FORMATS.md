# 📊 报价单格式对比

**选择最适合你的格式**

---

## 🎯 快速选择指南

| 场景 | 推荐格式 | 理由 |
|------|----------|------|
| **发送给客户（正式）** | 📄 PDF | 专业、不可编辑、通用 |
| **发送给客户（现代）** | 🌐 HTML | 美观、可交互、可打印 |
| **内部存档/编辑** | 📗 Excel | 可编辑、公式计算、数据分析 |
| **正式文档/打印** | 📘 Word | 格式稳定、易编辑、兼容性好 |
| **快速预览** | 🌐 HTML | 浏览器直接打开、无需软件 |

---

## 📗 Excel (.xlsx)

**优势：**
- ✅ 可编辑，方便修改数据
- ✅ 自动计算公式（单价 × 数量 = 总额）
- ✅ 支持数据透视表、图表分析
- ✅ 便于批量处理多个报价单
- ✅ 可与 OKKI 产品库联动

**劣势：**
- ❌ 格式可能因 Excel 版本不同而偏移
- ❌ 不够正式（客户可能觉得不专业）
- ❌ 需要 Excel 或 LibreOffice 打开

**适用场景：**
- 内部存档和编辑
- 需要频繁修改价格/数量
- 需要计算多个版本对比
- 与 ERP/CRM 系统集成

**生成命令：**
```bash
python3 generate_quotation.py --data data.json -o QT-001.xlsx
```

---

## 📘 Word (.docx)

**优势：**
- ✅ 格式稳定，兼容性好
- ✅ 易于编辑文字内容（条款、备注）
- ✅ 适合正式文档打印
- ✅ 大多数电脑都有 Word 或兼容软件

**劣势：**
- ❌ 表格计算不如 Excel 方便
- ❌ 修改数据需要手动重新计算
- ❌ 文件体积较大

**适用场景：**
- 正式报价文档
- 需要打印签字
- 需要频繁修改文字条款
- 客户明确要求 Word 格式

**生成命令：**
```bash
python3 generate_quotation_docx.py --data data.json -o QT-001.docx
```

---

## 🌐 HTML（推荐 ⭐）

**优势：**
- ✅ **现代设计**（Tailwind CSS，专业美观）
- ✅ **无需安装软件**，浏览器直接打开
- ✅ **一键导出 PDF**（内置打印按钮）
- ✅ **响应式设计**，手机/平板友好
- ✅ **可交互**（hover 效果、链接）
- ✅ **文件体积小**（14KB vs PDF 160KB）

**劣势：**
- ❌ 需要浏览器查看
- ❌ 离线环境可能无法加载字体/图标（CDN）
- ❌ 不可直接编辑（需要修改 HTML 或 JSON 源数据）

**适用场景：**
- ⭐ **快速预览和分享**
- ⭐ **邮件附件（体积小）**
- ⭐ **客户在线查看**
- ⭐ **高质量 PDF 导出源文件**

**生成命令：**
```bash
# 只生成 HTML
python3 generate_quotation_html.py --data data.json -o QT-001.html

# 或一键生成（包含 HTML）
./generate-all.sh data.json QT-001
```

**最佳实践：**
1. 生成 HTML 文件
2. 在浏览器打开
3. 点击 "Export to PDF" 按钮
4. 获得高质量 PDF（比 LibreOffice 转换更清晰）

---

## 📄 PDF

**优势：**
- ✅ **最正式**，商务标准格式
- ✅ **不可编辑**，防止篡改
- ✅ **格式固定**，任何设备显示一致
- ✅ **通用兼容**，所有设备都能打开
- ✅ **适合打印**，分辨率高

**劣势：**
- ❌ 不可编辑（需要重新生成）
- ❌ 文件体积较大（150-200KB）
- ❌ 修改需要重新生成整个文件

**适用场景：**
- ⭐ **正式发送给客户**
- ⭐ **合同附件**
- ⭐ **打印签字**
- ⭐ **归档保存**

**生成方式：**

**方式 1：从 HTML 导出（推荐，质量最好）**
```bash
# 浏览器打开 HTML，点击 "Export to PDF"
open QT-001.html
```

**方式 2：从 Excel/Word 转换**
```bash
soffice --headless --convert-to pdf QT-001.xlsx --outdir ./
soffice --headless --convert-to pdf QT-001.docx --outdir ./
```

**方式 3：一键生成（自动包含 PDF）**
```bash
./generate-all.sh data.json QT-001
```

## 🔄 推荐工作流

### 方案 A：快速模式（推荐）
```bash
# 1. 一键生成所有格式
./generate-all.sh data.json QT-001

# 2. 在浏览器打开 HTML 预览
open QT-001.html

# 3. 点击 "Export to PDF" 导出高质量 PDF

# 4. 发送 PDF 给客户
```

**优点：** 一次生成，多种格式，灵活选择

---

### 方案 B：HTML 优先（现代）
```bash
# 1. 只生成 HTML
./generate-all.sh data.json QT-001 --html-only

# 2. 浏览器打开，检查内容
open QT-001.html

# 3. 导出 PDF（如需）

# 4. 发送 HTML 链接或 PDF
```

**优点：** 快速、文件小、现代美观

---

### 方案 C：传统模式（兼容）
```bash
# 1. 生成 Excel（用于计算和编辑）
python3 generate_quotation.py --data data.json -o QT-001.xlsx

# 2. 转换为 PDF
soffice --headless --convert-to pdf QT-001.xlsx

# 3. 发送 PDF 给客户
```

**优点：** 兼容性好，适合传统企业

## 💡 最佳实践建议

### 对内（内部使用）
- **编辑/计算** → Excel
- **存档** → Excel + PDF 双格式
- **快速预览** → HTML

### 对外（发送给客户）
- **首次询价** → PDF（正式）
- **多次沟通** → HTML（方便快速查看）
- **正式合同** → PDF（打印签字）
- **技术规格** → Excel（便于客户导入系统）

### 特殊场景
- **展会现场** → HTML（平板展示，交互效果好）
- **邮件营销** → HTML（可直接嵌入邮件）
- **归档备份** → PDF + JSON（源数据）
