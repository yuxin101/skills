# 常见变现模式

## 1. 订阅制 (SaaS)

**适用**：工具类产品、效率软件、付费功能明确

**案例**：
- Notion、Notion AI
- Figma、Raycast
- 少数派会员

**模式**：
- 月付/年付
- 免费版 + 付费版
- 按功能阶梯定价

**优点**：稳定收入、可预测、用户粘性高
**难点**：需要持续更新、留存率关键

---

## 2. 付费下载

**适用**：桌面/手机应用、模板、素材

**案例**：
- App Store 应用
- Gumroad 数字产品
- 站酷/UI中国素材

**模式**：
- 一次性付费
- 免费+付费（Freemium）
- 买断制

**优点**：简单直接、边际成本低
**难点**：需要持续推广、盗版问题

---

## 3. 付费咨询/服务

**适用**：专业服务、技能培训

**案例**：
- 一对一咨询
- 训练营/课程
- 代运营服务

**模式**：
- 按小时/项目收费
- 社群会员制
- 课程销售

**优点**：客单价高、边际成本低
**难点**：时间有限、难以规模化

---

## 4. 广告/联盟

**适用**：流量大的内容产品

**案例**：
- YouTube、B站 UP主
- 公众号/博客
- 小红书博主

**模式**：
- 广告联盟（Google AdSense、百度联盟）
- 品牌合作
- 带货分成

**优点**：流量变现、规模效应
**难点**：需要流量积累、平台规则

---

## 5. 电商卖货

**适用**：有实体/虚拟产品

**案例**：
- 淘宝/拼多多店铺
- 有赞/微店
- 知识付费

**模式**：
- 自有产品
- 代发货（Dropshipping）
- 课程/电子书

**优点**：直接变现
**难点**：供应链、客服、库存

---

## 6. 开源赞助 (Open Source)

**适用**：开发者工具、开源软件

**案例**：
- GitHub Sponsors
- Open Collective
- 爱发电

**模式**：
- 接受赞助
- 企业版付费
- 捐赠

**优点**：社区认可、技术名片
**难点**：需要技术实力、难以维持

---

## 选哪个？

| 你的情况 | 推荐模式 |
|---------|---------|
| 有技术、想做工具 | 订阅制 |
| 做模板/素材 | 付费下载 |
| 有专业技能 | 咨询/课程 |
| 有流量基础 | 广告/联盟 |
| 有产品资源 | 电商 |
| 技术大牛 | 开源赞助 |
