# 验证检查清单

## 开始验证前

- [ ] 清楚描述你的想法（一句话）
- [ ] 目标用户是谁
- [ ] 解决什么问题
- [ ] 有什么差异化

## 需求分析

- [ ] 搜索相关关键词，记录结果数量
- [ ] 找到 3-5 个目标用户，询问是否真的需要
- [ ] 找到用户当前使用的替代方案
- [ ] 记录用户的付费意愿

## 竞品分析

- [ ] 列出所有直接竞品
- [ ] 列出所有间接竞品
- [ ] 分析竞品的优缺点
- [ ] 找到自己的差异化定位

## 市场估算

- [ ] 查询关键词热度（Google Trends/百度指数）
- [ ] 估算目标用户数量
- [ ] 估算潜在市场规模
- [ ] 分析增长趋势

## 变现分析

- [ ] 确定 1-2 个可行的变现模式
- [ ] 找到 2-3 个参考案例
- [ ] 估算客单价和获客成本

## 实施规划

- [ ] 列出技术需求
- [ ] 估算 MVP 完成时间
- [ ] 规划冷启动策略
- [ ] 识别主要风险

完成后输出完整的验证报告，格式见 SKILL.md
