# 行业特定R-O优化模板

## 电商行业模板

### 电商运营分析

**原始需求示例：**
"分析店铺数据"

**R-O优化模板：**
```
## 角色&现实情况

### 角色设定
您是资深电商数据分析专家，拥有5年以上淘宝/天猫运营经验，精通：
- 生意参谋数据解读
- 消费者行为分析  
- 竞品监控方法
- 转化率优化策略

### 现实情况
- 时间：当前季度末，需要为下季度规划提供数据支持
- 平台：淘宝/天猫店铺，类目为[填写类目]
- 数据源：生意参谋后台数据，包含过去30天完整数据
- 限制：缺乏用户调研数据，需基于现有数据推断

## 对象&输出

### 输出对象
- 身份：店铺运营负责人（具备基础数据理解能力）
- 知识水平：熟悉电商基础指标但不精通深度分析
- 需求：需要 actionable 的改进建议，而非纯数据展示

### 输出要求
1. **核心指标概览**：GMV、UV、转化率、客单价的环比/同比变化
2. **问题诊断**：识别关键瓶颈（流量、转化、复购等环节）
3. **竞品对标**：与Top3竞品的核心指标对比
4. **行动建议**：具体、可执行的优化方案（分优先级）
5. **格式**：PPT简报大纲 + 详细数据附录（Excel格式）
```

## 通用优化原则

### 跨行业适用
1. **角色设定**：始终从"专业深度"和"经验广度"两个维度构建
2. **现实情况**：明确时间、资源、环境的具体约束
3. **输出对象**：深入了解受众的真实需求和认知水平
4. **输出要求**：具体化、可衡量、有时限的质量标准

### 模板使用建议
1. **选择最接近的模板**作为起点
2. **根据实际需求调整**各个维度的细节
3. **保持一致性**：四个维度之间要逻辑自洽
4. **验证实用性**：生成的提示词是否能直接使用

### 模板扩展
当现有模板不满足需求时：
1. 分析需求属于哪个大行业
2. 确定核心任务类型（分析、创意、执行等）
3. 参考类似行业的模板结构
4. 创建新的行业特定模板
