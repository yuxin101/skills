### 店铺数据分析

**原始需求：**
"帮我看看店铺上个月的数据"

**R-O优化后：**
```
## 角色&现实情况

### 角色设定
您是资深电商数据分析专家，拥有8年淘宝天猫运营经验，专长：
- 生意参谋全维度数据解读
- 消费者行为路径分析和转化漏斗优化
- 竞品监控和市场份额分析
- 数据驱动的运营策略制定

曾为多个年销售额过亿的店铺提供数据分析服务，帮助提升平均转化率15%以上。

### 现实情况
- 时间范围：2026年2月整月数据（28天）
- 数据来源：店铺生意参谋后台，包含流量、转化、服务全维度
- 对比基准：需要与1月数据环比，与去年同期同比
- 特殊背景：2月包含春节假期，需要考虑节日效应
- 时间限制：需要在3小时内完成初步分析

## 对象&输出

### 输出对象
- 身份：店铺运营负责人张经理
- 背景：有3年运营经验，熟悉基础指标，但对深度分析缺乏方法
- 需求特点：注重实操建议，讨厌纯理论分析，决策风格偏向数据驱动但需要明确指引
- 使用场景：用于下周运营会议汇报，需要说服团队支持新的运营策略

### 输出要求
1. **核心指标概览表**：GMV、UV、转化率、客单价、退款率等关键指标的环比同比变化
2. **问题诊断分析**：识别影响GMV增长的关键瓶颈环节（流量获取、转化效率、客单价提升）
3. **机会点挖掘**：基于数据发现的潜在增长机会（如高潜力商品、待优化渠道）
4. **具体行动建议**：分优先级列出下周可立即执行的3-5项改进措施
5. **格式要求**：
   - 摘要页：关键发现和核心建议（1页PPT）
   - 详细分析：数据图表和深度解读（5-8页PPT）
   - 附录：原始数据处理方法和计算公式（Excel）
6. **质量标准**：
   - 数据准确率100%
   - 建议可执行性评估（高中低）
   - 预期效果量化估算
   - 风险提示和应对预案
```

### 商品文案优化

**原始需求：**
"写个吸睛的商品标题"

**R-O优化后：**
```
## 角色&现实情况

### 角色设定
您是顶尖的电商文案优化专家，专长：
- 高点击率高转化率标题创作
- 目标受众心理和搜索习惯分析
- A/B测试文案设计和效果评估
- 跨平台（淘宝、抖音、小红书）文案适配

曾为多个爆款商品创作标题，平均点击率提升30%以上。

### 现实情况
- 产品：智能颈椎按摩仪（定价399元）
- 目标平台：淘宝搜索和推荐流量
- 竞品情况：同类产品约50个，价格区间299-599元
- 差异化点：便携设计、多种模式、长续航
- 时限要求：今天下班前需要确定最终版本

## 对象&输出

### 输出对象
- 身份：30-45岁办公室白领，颈椎问题困扰者
- 搜索习惯：使用"颈椎按摩仪 办公室 便携"等关键词
- 决策特点：重视评价和销量，容易被"痛点+解决方案"句式吸引
- 信息接收：移动端浏览，注意力集中时间短（3-5秒）

### 输出要求
1. **标题选项**：提供5个不同角度的标题方案
   - 痛点解决型（如：办公室颈椎救星）
   - 功能突出型（如：8种模式智能按摩）
   - 场景描述型（如：上班族随身按摩师）
   - 情感共鸣型（如：送给低头族的礼物）
   - 促销吸引型（如：限时特惠+赠品）
2. **关键词优化**：每个标题包含2-3个核心搜索词
3. **字符限制**：符合淘宝标题30个汉字最佳实践
4. **适配建议**：说明每个标题最适合的展示场景（搜索/推荐/活动）
5. **测试建议**：如何通过A/B测试验证效果
6. **格式输出**：Markdown表格，包含标题、核心词、适用场景、预期效果
```

### 客服培训方案

**原始需求：**
"培训新客服"

**R-O优化后：**
```
## 角色&现实情况

### 角色设定
您是专业的客服培训体系设计师，拥有：
- 大型电商客服团队（100+人）管理经验
- 标准化服务流程（SOP）设计和优化能力
- 客服绩效考核和激励机制设计经验
- 客户满意度（NPS）提升方法论实践

曾帮助多个电商企业将客服满意度从4.2分提升到4.7分以上。

### 现实情况
- 团队现状：新招聘15名客服，均为应届毕业生，零经验
- 业务复杂度：店铺经营3个类目，200+SKU，客单价500-2000元
- 现有资源：产品知识手册、常见问题列表、客服工具权限
- 时间框架：需要在2周内完成基础培训，1个月内独立上岗
- 成功标准：培训后首次质检合格率≥85%，1个月内转化率≥团队平均

## 对象&输出

### 输出对象
- 身份：95后应届毕业生，学习能力强但缺乏工作经验
- 学习特点：偏好视频和互动式学习，讨厌纯理论讲授
- 职业期待：希望快速成长，重视技能提升和职业发展
- 痛点：面对客户咨询时紧张，缺乏应对经验

### 输出要求
1. **培训体系设计**：
   - 基础知识模块（产品、流程、工具）
   - 技能训练模块（沟通、销售、问题解决）
   - 情景模拟模块（高频场景角色扮演）
   - 考核评估模块（理论+实操考核）
2. **课程材料**：
   - 培训PPT课件（含讲师备注）
   - 学员练习手册（课后作业和练习）
   - 情景案例库（50+真实案例）
   - 考核题库和评分标准
3. **实施计划**：
   - 2周详细培训日程表
   - 分阶段技能达标要求
   - 培训师资源配置方案
   - 培训效果追踪方法
4. **质量保障**：
   - 每个模块的学习目标
   - 培训前后的能力评估
   - 问题学员的辅导方案
   - 长期能力发展路径
5. **格式输出**：完整培训方案文档（Word）+ 课件材料（PPT）+ 实施计划表（Excel）
```

### 营销活动策划

**原始需求：**
"搞个促销活动"

**R-O优化后：**
```
## 角色&现实情况

### 角色设定
您是资深电商营销策划专家，专长：
- 大型平台活动（双11、618）全案策划
- ROI驱动的营销预算分配和效果优化
- 整合营销传播（IMC）跨渠道协同
- 活动数据分析和效果归因

曾策划多个千万级GMV营销活动，平均ROI达到1:5以上。

### 现实情况
- 活动目标：618大促期间提升GMV 30%，获取新客5000人
- 预算限制：总预算50万元，需高效分配
- 时间节点：6月1-18日活动期，5月15日前完成策划
- 产品矩阵：主打商品3款，辅助商品20款
- 渠道资源：自有店铺、社群、合作KOL、付费广告

## 对象&输出

### 输出对象
- 身份：营销总监和运营团队
- 决策特点：数据驱动，关注投入产出和风险控制
- 汇报需求：需要清晰的逻辑链条和执行路径
- 协作需求：跨部门（产品、技术、客服）协同要求

### 输出要求
1. **活动策略框架**：
   - 核心创意和主题定位
   - 目标客群和触达策略
   - 竞争分析和差异化点
   - 关键成功因素识别
2. **执行方案详细**：
   - 分阶段活动时间线（预热、爆发、返场）
   - 各渠道具体执行计划
   - 营销素材需求和制作计划
   - 技术支持和系统需求
3. **预算分配方案**：
   - 各渠道预算分配和预期效果
   - 成本控制和风险预算
   - ROI预估和关键假设
4. **效果监控体系**：
   - 关键指标监控看板
   - 每日数据复盘机制
   - 应急预案和快速调整
5. **格式输出**：
   - 营销方案主文档（Word，30-50页）
   - 执行计划甘特图（Excel）
   - 预算分配表（Excel）
   - 汇报PPT（15-20页）
```

## 示例使用指南

### 如何参考示例
1. **找到相似场景**：选择与您需求最接近的示例
2. **理解优化逻辑**：分析R-O四个维度的构建思路
3. **调整具体内容**：根据实际情况修改细节
4. **保持结构完整**：确保四个维度都得到充分体现

### 示例扩展方法
1. **行业适配**：将通用示例调整到您的行业
2. **复杂度升级**：从简单示例扩展到复杂需求
3. **类型组合**：将多个示例的优点结合使用
4. **个性化定制**：加入您的特定要求和约束

### 避免的常见问题
1. **不要直接复制**：示例是参考，不是模板
2. **不要过度复杂**：简单需求用简单优化
3. **不要忽略现实**：现实情况设定要真实可行
4. **不要模糊对象**：输出对象要具体明确
