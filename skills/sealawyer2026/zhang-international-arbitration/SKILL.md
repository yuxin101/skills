# 国际仲裁全流程AI助手 🌍

> 专业国际仲裁律师AI助手 | ICC/LCIA/SIAC/HKIAC | 跨境争议解决

## 🎯 适用场景

- ICC国际商会仲裁
- LCIA伦敦国际仲裁
- SIAC新加坡国际仲裁
- HKIAC香港国际仲裁
- 投资仲裁（ICSID）
- 临时仲裁（Ad Hoc）
- 仲裁裁决跨境承认执行（纽约公约）

## 📋 阶段一：国际仲裁策略制定

### 1.1 仲裁地选择策略
```
请制定国际仲裁地选择策略：

案件因素：
- 当事人国籍：[A国/B国]
- 合同履行地：[地点]
- 争议标的：[金额/类型]
- 执行便利性考量：[地点]

候选仲裁地分析：
| 仲裁地 | 优势 | 劣势 | 适用性评分 |
|--------|------|------|-----------|
| 新加坡 | 中立、高效、执行好 | 费用较高 | ★★★★★ |
| 香港 | 中国熟悉、双语 | 政治因素 | ★★★★☆ |
| 伦敦 | 经验丰富、法律完善 | 费用高、时间长 | ★★★★☆ |
| 巴黎 | 大陆法系友好 | 语言障碍 | ★★★☆☆ |
| 其他 | ... | ... | ... |

推荐结论：
```

### 1.2 仲裁机构选择
```
请推荐适合的国际仲裁机构：

案件特点：

- 当事人背景：[国企/民企/外企]
- 保密需求：[高/中/低]

| SIAC | 中 | 较快 | 亚洲首选 | ★★★★★ |
| HKIAC | 中 | 较快 | 中国相关 | ★★★★☆ |
| SCC | 中 | 中等 | 欧洲优选 | ★★★☆☆ |

推荐结论：
```

### 1.3 国际仲裁条款起草
```
请起草国际仲裁条款：

合同类型：[贸易合同/投资协议/建设工程/等]
当事人：[A国公司/B国公司]
优先仲裁地：[地点]

条款草案：
```
第X条 争议解决

X.1 凡因本合同引起的或与本合同有关的任何争议，均应提交[仲裁机构]根据其届时有效的仲裁规则进行仲裁。

X.2 仲裁地点为[城市]，仲裁语言为[中文/英文/双语]。

X.3 仲裁庭由[一/三]名仲裁员组成。

X.4 仲裁裁决是终局的，对双方均有约束力。

X.5 本条款适用[准据法]法律。
```

条款要点说明：
1. 仲裁意愿明确性
2. 机构/规则确定性
3. 仲裁地重要性
4. 语言选择策略
5. 适用法律约定
```

## 📋 阶段二：国际仲裁启动

### 2.1 仲裁申请书（国际）
```
请起草国际仲裁申请书：

申请人：[公司名/国籍]
被申请人：[公司名/国籍]
仲裁机构：[ICC/LCIA/SIAC等]
仲裁地：[城市]

申请书应包含：
1. 当事人信息（完整）
2. 仲裁协议依据
3. 仲裁请求（明确具体）
4. 争议事实（时间线）
5. 法律依据（准据法）
6. 证据清单（分类）
7. 仲裁员选定意见
8. 程序性申请（如有）
```

### 2.2 仲裁答辩书（国际）
```
请起草国际仲裁答辩书：

被申请人：[公司名]
申请人：[公司名]
仲裁机构：[机构]

答辩书应包含：
1. 当事人信息
2. 对请求的回应
3. 事实陈述
4. 法律抗辩
5. 反请求（如有）
6. 仲裁员选定意见
7. 程序性申请
```

### 2.3 紧急仲裁员申请
```
请起草紧急仲裁员申请：

申请事项：[临时措施/禁令]
紧急性说明：[理由]

申请书应包含：
1. 当事人信息
2. 紧急事项说明
3. 紧急性论证
4. 申请措施
5. 担保提供
6. 法律依据
```

## 📋 阶段三：仲裁庭组成

### 3.1 仲裁员选定策略
```
请制定仲裁员选定策略：

案件特点：
- 争议焦点：[技术/法律/商业]
- 准据法：[法律体系]
- 当事人背景：[文化因素]

策略分析：
1. 首席仲裁员选择
   - 专业背景
   - 国籍考量
   - 经验水平
   - 可用时间

2. 边裁选择
   - 我方边裁
   - 对方边裁预判
   - 协调策略

3. 回避考量
   - 利益冲突审查
   - 独立性评估
```

### 3.2 仲裁员挑战申请
```
请起草仲裁员挑战申请：

被挑战人：[仲裁员姓名]
挑战理由：[独立性/公正性]

申请书应包含：
1. 案件信息
2. 被挑战仲裁员信息
3. 挑战理由
   - 利益冲突
   - 偏见表现
   - 其他情形
4. 证据支持
5. 法律依据
6. 申请结论
```

## 📋 阶段四：国际仲裁程序

### 4.1 程序令（Procedural Order）分析
```
请分析以下程序令：

[粘贴程序令内容]

分析要点：
1. 时间表安排
2. 文件提交要求
3. 证据开示范围
4. 听证安排
5. 费用安排
6. 程序性权利
7. 应对建议
```

### 4.2 证据开示策略
```
请制定证据开示策略：

案件阶段：[文件提交/证据开示]
对方请求：[请求内容]

策略分析：
1. 开示范围界定
2. 保密保护申请
3. 特权主张（律师-客户特权）
4. 负担抗辩
5. 请求对方开示
6. 开示时间表
```

### 4.3 专家证人管理
```
请制定专家证人策略：

争议领域：[技术/财务/行业]

策略内容：
1. 专家选择
   - 专业资质
   - 经验水平
   - 独立性
   - 沟通能力

2. 专家报告
   - 报告结构
   - 技术内容
   - 可理解性

3. 专家听证
   - 直接询问
   - 交叉询问准备
   - 热 tubbing（专家对峙）
```

### 4.4 国际仲裁听证
```
请准备国际仲裁听证：

听证安排：
- 时间：[日期]
- 地点：[城市]
- 形式：[现场/视频]

准备清单：
1. 开场陈述
2. 证人安排
   - 事实证人
   - 专家证人
3. 证据展示
4. 辩论要点
5. 技术设备
6. 翻译安排（如需）
```

## 📋 阶段七：投资仲裁专项

### 7.1 ICSID仲裁申请
```
请起草ICSID仲裁申请：

申请人：[投资者国籍]
被申请人：[东道国]
投资条约：[BIT/ICSID公约]

申请书应包含：
1. 当事人信息
2. 管辖权依据
   - 国籍规划
   - 投资定义
   - 同意表示
3. 仲裁请求
4. 争议事实
5. 违约指控
6. 损害赔偿计算
7. 证据清单
```

### 7.2 投资仲裁抗辩
```
请起草投资仲裁抗辩：

被申请人：[东道国]
申请人：[投资者]

抗辩要点：
1. 管辖权抗辩
   - 国籍不符
   - 非投资性质
   - 同意缺失
2. 实体抗辩
   - 未违反条约
   - 国家防卫
   - 合法性审查
3. 损害赔偿抗辩
   - 因果关系
   - 计算方法
   - 减轻义务
```
