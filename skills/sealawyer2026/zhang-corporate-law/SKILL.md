# 公司法律助手 🏛️

> 公司全流程法律服务 - 设立/治理/变更/解散/清算

## 🎯 适用场景

- 专业法律咨询与方案制定
- 法律文书起草与审查
- 案件策略分析与规划
- 合规风险评估与建议
- 诉讼仲裁全流程支持

## 📋 核心功能

### 1.1 智能咨询
```
你是一位资深公司法律助手专家。请根据以下情况提供专业建议：

案件/项目背景：[描述]
当事人/企业情况：[描述]
核心诉求：[描述]
争议焦点：[如有]

请提供：
1. 法律问题分析
2. 可行方案建议（2-3个）
3. 各方案优劣对比
4. 风险提示
5. 下一步行动建议
```

### 1.2 文书起草
```
请根据以下信息起草法律文书：

文书类型：[类型]
当事人信息：[信息]
案件/项目情况：[描述]
具体诉求：[诉求]

要求：
1. 格式规范
2. 法律依据准确
3. 事实陈述清晰
4. 论证逻辑严密
5. 语言专业得体
```

### 1.3 合同审查
```
请审查以下合同，识别风险点：

[粘贴合同内容]

审查要点：
1. 主体资格审查
2. 标的条款审查
3. 价款/报酬条款
4. 履行条款审查
5. 违约责任审查
6. 争议解决条款
7. 其他风险点

输出格式：
- 🔴 高风险：[说明]
- 🟡 中风险：[说明]
- 🟢 建议优化：[说明]
```

### 1.4 案例分析
```
请分析以下案例/项目：

案情/项目简介：[描述]
争议焦点/关键问题：[描述]
已有证据/材料：[描述]

分析内容：
1. 法律关系梳理
2. 争议焦点分析
3. 有利/不利因素
4. 类似案例参考
5. 策略建议
6. 预期结果评估
```

### 1.5 法规检索
```
请检索与以下问题相关的法律法规：

问题描述：[描述]
适用领域：[领域]
效力层级：[法律/行政法规/部门规章/地方性法规/司法解释]

输出：
1. 核心法律依据（条文引用）
2. 相关司法解释
3. 部门规章/规范性文件
4. 地方性规定（如有）
5. 最新立法动态
```
