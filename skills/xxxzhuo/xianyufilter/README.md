# 闲鱼搜索技能 - 使用说明

## 🚀 快速开始

### 基本用法

```
闲鱼搜索 [产品名称] [规格参数]
```

### 示例

```
闲鱼搜索 金士顿 DDR5 16G 6000 台式机内存
闲鱼搜索  RTX4060 显卡
闲鱼搜索  iPhone15 256G 二手
```

## ✅ 默认严格筛选

技能自动应用以下筛选条件，无需用户指定：

| 筛选条件 | 说明 |
|----------|------|
| 👤 个人闲置 | 排除商家/鱼小铺，只显示个人卖家 |
| 💰 单一价格 | 排除价格区间，只显示明确定价 |
| 📦 规格匹配 | 确保容量/频率/型号完全匹配 |
| 🖥️ 台式机 | 排除笔记本/服务器配件 |

## 📋 输出内容

每次搜索返回：

1. **TOP10 商品列表** - 按价格升序排列
2. **价格统计** - 最低价/最高价/平均价
3. **购买链接** - 直接点击跳转商品页
4. **筛选说明** - 显示应用的筛选条件

## 🎯 高级用法

### 价格限制

```
闲鱼搜索 金士顿 DDR5 16G --max-price 1200
```

### 地区限制

```
闲鱼搜索 金士顿 DDR5 16G --region 广东
```

### 成色要求

```
闲鱼搜索 金士顿 DDR5 16G --condition 全新
```

### 组合条件

```
闲鱼搜索 金士顿 DDR5 16G 6000 --max-price 1200 --region 广东 --shipping
```

### 支持的条件

| 条件 | 说明 | 示例 |
|------|------|------|
| `--max-price XXX` | 最高价格 | `--max-price 1200` |
| `--min-price XXX` | 最低价格 | `--min-price 800` |
| `--region XXX` | 指定地区 | `--region 广东` |
| `--condition XXX` | 成色要求 | `--condition 全新` |
| `--shipping` | 仅包邮 | `--shipping` |
| `--verified` | 仅验货宝 | `--verified` |

## ⚠️ 注意事项

### 价格说明

- 所有价格均为**单件价格**
- 套条商品会标注"标价为单根价格"或"标价为整套价格"
- 价格区间商品（如"¥1000-¥1500"）会被排除

### 卖家类型

- **个人闲置** - 个人卖家，通常价格更优惠
- **鱼小铺** - 认证商家，可能有保修
- **超赞鱼小铺** - 优质商家，服务更好

默认只显示"个人闲置"，如需查看商家商品，请说明"接受商家商品"。

### 成色说明

| 成色 | 说明 |
|------|------|
| 全新 | 未拆封/未使用 |
| 几乎全新 | 使用极少，接近全新 |
| 95 新 | 轻微使用痕迹 |
| 9 成新 | 正常使用痕迹 |
| 二手 | 明显使用痕迹 |

## 🔧 故障排除

### 无搜索结果

**可能原因：**
- 搜索关键词太具体
- 筛选条件太严格
- 商品确实稀缺

**解决方法：**
1. 简化搜索关键词
2. 放宽价格限制
3. 接受商家商品

### 价格异常

**可能原因：**
- 低价引流（实际价格更高）
- 套条价标为单根价
- 商品已下架

**解决方法：**
1. 点击链接查看实际价格
2. 私信卖家确认
3. 查看商品发布时间

### 链接失效

**可能原因：**
- 商品已卖出
- 卖家下架商品
- 链接过期

**解决方法：**
1. 刷新搜索结果
2. 查看其他商品
3. 设置降价提醒

## 💡 使用技巧

1. **精确关键词** - 包含品牌 + 型号 + 容量 + 频率
2. **价格参考** - 先搜索了解市场行情
3. **对比多家** - 查看 TOP10 对比价格和成色
4. **及时下手** - 个人闲置好物容易被秒
