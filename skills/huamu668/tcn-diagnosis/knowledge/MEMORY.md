# 中医诊疗知识与经验积累

## 核心经典

### 四大经典
1. **《黄帝内经》** - 中医理论奠基之作
   - 核心：阴阳五行、藏象经络、病因病机
   - 名言："上医治未病，中医治欲病，下医治已病"

2. **《伤寒论》** - 辨证论治经典
   - 核心：六经辨证（太阳、阳明、少阳、太阴、少阴、厥阴）
   - 贡献：113方，确立了辨证论治体系

3. **《金匮要略》** - 杂病诊治经典
   - 核心：脏腑辨证、杂病分科
   - 贡献：262首方剂，内科杂病专著

4. **《温病条辨》** - 温病学经典
   - 核心：卫气营血辨证、三焦辨证
   - 贡献：完善了外感热病的辨证体系

## 诊疗经验

### 问诊要点
- 问寒热：了解病性（寒热）和病位（表里）
- 问汗出：判断阴阳虚实（自汗气虚、盗汗阴虚）
- 问饮食：了解脾胃功能（食欲、口味、口渴）
- 问二便：了解脏腑功能（脾肾）
- 问睡眠：了解心神状态（心肾相交）

### 舌诊速记
- **舌质看正气**：淡白阳虚、红绛热盛、紫暗瘀血
- **舌苔看邪气**：白苔寒、黄苔热、腻苔湿、剥苔阴虚

### 脉诊要诀
- **浮沉分表里**：浮脉表证、沉脉里证
- **迟数分寒热**：迟脉寒证、数脉热证
- **虚实分正邪**：虚脉正虚、实脉邪实

### 解表剂
- 桂枝汤：桂枝汤治太阳风，芍药甘草姜枣同
- 麻黄汤：麻黄汤中用桂枝，杏仁甘草四般施

- 六味地黄丸：六味地黄益肾肝，茱薯丹泽地苓专

### 祛湿剂

## 穴位保健

### 常用保健穴
- **足三里**：健脾益气（膝下4横指）
- **关元**：培元固本（脐下4横指）
- **气海**：益气助阳（脐下2横指）
- **三阴交**：调理肝脾肾（内踝上4横指）
- **合谷**：清热止痛（手背虎口处）

## 安全经验

### 必须建议就医的情况
- 持续高热不退（>39°C超过3天）
- 剧烈胸痛、胸闷
- 呼吸困难、气促
- 意识模糊、昏迷
- 大量出血
- 剧烈腹痛

### 用药禁忌提醒
- 孕妇：禁用活血、泻下、毒性药物
- 儿童：剂量需减量，避免大寒大热
- 老人：注意脾胃功能，避免过于攻伐
