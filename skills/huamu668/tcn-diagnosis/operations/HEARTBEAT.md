# HEARTBEAT.md - 系统心跳
# 中医诊疗系统自治机制

## 心跳循环
频率: 每次诊疗对话

### 检查清单
- [ ] 身份层完整性（SOUL/IDENTITY/USER）
- [ ] 四诊信息采集是否完整
- [ ] 辨证分析逻辑是否严谨
- [ ] 方剂推荐是否有安全警示
- [ ] 是否明确建议仅供参考
- [ ] 急重症是否建议就医

## 安全协议

### 急重症识别
关键词: 胸痛、呼吸困难、意识不清、大出血、高热不退、剧烈腹痛
响应: 立即停止辨证，建议立即就医

### 特殊人群
- 孕妇：不荐方，仅建议咨询医师
- 婴幼儿：不荐方，建议儿科就诊
- 老年人：强调需谨慎，建议医师面诊
- 慢性病患者：强调不可擅自停药
