## 使用方法

### 1. 中医问诊
说：**"看病"** 或 **"问诊"**

系统将启动四诊采集：
- 望诊：面色、舌象
- 闻诊：声音、气味
- 问诊：症状、病史、饮食睡眠
- 切诊：脉象

### 2. 体质辨识
说：**"体质"** 或 **"我是哪种体质"**

系统将：
- 进行九种体质问卷
- 给出体质判断
- 提供调养建议

### 3. 食疗养生
说：**"食疗"** 或 **"养生"**

系统将：
- 根据体质/证型推荐食材
- 提供食疗方案
- 建议起居调摄

## 核心知识库

### 四诊信息库 (knowledge/symptoms/)
- **望诊**：神色、面色、舌象
- **闻诊**：声音、呼吸、气味
- **问诊**：寒热、汗出、疼痛、饮食、睡眠、二便
- **切诊**：脉象（浮沉、迟数、虚实、滑涩弦紧）

### 经典方剂库 (knowledge/formulas/)
- **解表剂**：桂枝汤、麻黄汤、银翘散
- **清热剂**：白虎汤、黄连解毒汤
- **补益剂**：四君子汤、四物汤、六味地黄丸、金匮肾气丸
- **祛湿剂**：平胃散、藿香正气散
- **理气剂**：柴胡疏肝散、半夏厚朴汤
- **理血剂**：血府逐瘀汤

### 辨证模式库 (knowledge/patterns/)
- **八纲辨证**：阴阳、表里、寒热、虚实
- **脏腑辨证**：心肝脾肺肾各证型
- **气血津液辨证**：气病、血病

## 诊疗流程

```
用户描述症状 → 四诊采集(zhen) → 辨证分析(bianzheng) → 方剂推荐(fangji)
              ↓
         食疗建议(shiliao) / 体质辨识(tizhi)
              ↓
         安全警示（必须）
```

## 安全机制

### 急重症识别
自动识别关键词：胸痛、呼吸困难、意识不清、大出血、高热不退、剧烈腹痛
→ 立即停止辨证，建议立即就医

### 特殊人群保护
- 孕妇：不推荐方剂，建议咨询医师
- 婴幼儿：不推荐方剂，建议儿科就诊
- 老年人：强调需医师面诊

### 安全输出模板
所有输出必须包含：
```
⚠️ 以上建议仅供参考，不能替代专业医疗诊断
⚠️ 如有急重症，请立即就医
⚠️ 用药请咨询执业中医师
```
