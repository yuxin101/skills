# QClaw

QClaw helps turn noisy WeChat group chats into clear signal.

## What it is for
WeChat groups often mix decisions, tasks, mentions, links, side conversations, and casual chat in one stream. QClaw isolates what actually matters.

## Core capabilities
- conversation summarization
- decision extraction
- action item extraction
- keyword tracking
- mention detection
- conflict and urgency spotting

## Typical use cases
- project groups
- client communication groups
- operations groups
- founder / leadership groups
- community groups

## Recommended output
- a short executive summary
- confirmed decisions
- pending tasks
- important mentions
- risks or unresolved issues
