# VetClaw 工作流示例

## 工作流1：完整客户旅程

```
┌─────────────┐
│  客户来电    │
└──────┬──────┘
       ↓
┌──────────────┐
│ vet-phone-ai │  AI接听，判断意图
└──────┬───────┘
       ↓
┌──────────────┐
│  vet-intake  │  新客户信息采集
└──────┬───────┘
       ↓
┌──────────────────┐
│ vet-appointment  │  安排预约
└──────┬───────────┘
       ↓
┌──────────────┐
│ vet-reminder │  就诊前1h短信提醒
└──────┬───────┘
       ↓
┌────────────────────┐
│ [客户到院就诊]      │
│ vet-record-create  │  创建病历
└──────┬─────────────┘
       ↓
┌──────────────────┐
│ vet-prescription │  开具处方
└──────┬───────────┘
       ↓
┌─────────────┐
│ vet-invoice │  生成费用清单
└──────┬──────┘
       ↓
┌─────────────┐
│ vet-payment │  收款
└──────┬──────┘
       ↓
┌─────────────┐
│ vet-discharge│  出院指导
└──────┬──────┘
       ↓
┌─────────────┐
│ vet-followup │  设置复诊提醒(7天后)
└──────┬──────┘
       ↓
┌─────────────┐
│ vet-feedback │  就诊后满意度调查
└─────────────┘
```

## 工作流2：疫苗全周期管理

```
┌──────────────────┐
│ vet-intake       │  录入宠物信息(含出生日期)
└──────┬───────────┘
       ↓
┌──────────────────────┐
│ vet-vaccine-schedule │  根据年龄生成疫苗计划
└──────┬───────────────┘
       ↓
┌──────────────┐
│ vet-reminder │  每针前1天提醒
└──────┬───────┘
       ↓
┌───────────────────┐
│ [到院接种疫苗]     │
│ vet-record-create │  记录接种信息
└──────┬────────────┘
       ↓
┌──────────────┐
│ vet-invoice  │  收费
└──────┬───────┘
       ↓
┌──────────────┐
│ vet-followup │  设置下一针提醒
└──────────────┘
```

## 技能触发优先级

当用户消息匹配多个技能时，按以下优先级处理：

1. 🔴 **紧急类**: vet-emergency（最高优先级）
2. 🟠 **预约类**: vet-appointment, vet-reminder
3. 🟡 **查询类**: vet-price-lookup, vet-record-lookup, vet-hours
4. 🟢 **业务类**: vet-intake, vet-prescription, vet-invoice
5. ⚪ **辅助类**: 其他技能

## 集成说明

- **微信**: 通过企业微信API接收/发送消息
- **短信**: 阿里云短信API，支持模板消息
- **电话**: Vapi/阿里云电话API，支持AI接听
- **支付**: 支付宝/微信支付SDK
- **打印**: ESC/POS指令，支持热敏打印机
