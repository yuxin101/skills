## 概述

本技能包为中国企业法务团队提供50个AI驱动的法律合规自动化技能，覆盖合同审查、法规查询、劳动合规、知识产权、数据保护等核心领域。

基于中国现行法律法规知识库，每个技能可独立运行或组合使用。

## 快速开始

```bash
# 安装技能包
clawhub install legal-compliance-bundle

# 配置
cp config/legal-config.yaml.template config/legal-config.yaml
# 编辑 config/legal-config.yaml 填入你的企业信息

# 使用单个技能
openclaw run contract-review --input contract.pdf

# 使用技能组合
openclaw run compliance-audit-checklist --scope "data+labor+ip"
```
