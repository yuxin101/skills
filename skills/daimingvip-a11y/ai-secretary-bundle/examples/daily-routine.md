## 自定义快捷指令

你可以创建自己的快捷指令：

| 指令 | 功能 |
|------|------|
| `开始我的一天` | 执行完整晨间简报 |
| `检查邮件` | 快速邮件摘要 |
| `今天的新闻` | AI资讯速递 |
| `我的待办` | 任务列表 |
| `结束今天` | 晚间回顾 |
| `这周怎么样` | 周回顾 |

## 进阶用法

### 批量处理邮件

```
帮我处理所有来自客户A的未读邮件
```

### 智能提醒

```
提醒我明天下午3点前完成报告
```

### 新闻筛选

```
只给我看关于大模型的新闻
```

### 任务管理

```
把客户提案设为本周末前完成，优先级高
```
