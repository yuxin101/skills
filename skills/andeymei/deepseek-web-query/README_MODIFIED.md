# DeepSeek Web Query 技能修改说明

## 修改时间
2026-03-21 01:02

## 修改内容

根据用户要求，对 DeepSeek 网页查询技能进行了以下修改：

### 1. 深度思考按钮智能处理
- **修改前**：总是点击深度思考按钮
- **修改后**：检查按钮状态，如果已启用则不重复点击
- **实现**：检查按钮的 `data-active` 属性或 `active` 类名

### 2. 输入框焦点处理
- **修改前**：直接输入内容，不确保焦点
- **修改后**：先点击输入框获取焦点，再输入内容
- **实现**：
  1. 点击输入框获取焦点
  2. 清空输入框中的旧内容
  3. 输入新问题
  4. 按 Enter 发送

### 3. 回答提取方式优化
- **修改前**：通过页面元素提取文本内容
- **修改后**：使用拷贝按钮获取 Markdown 格式内容
- **实现**：
  1. 查找回答下方的拷贝按钮（最后一个）
  2. 点击拷贝按钮
  3. 从剪贴板读取 Markdown 格式内容
  4. 如果失败则回退到页面提取

### 4. 结果输出策略
- **修改前**：可能进行整理或总结
- **修改后**：直接输出原始内容，不进行任何处理
- **实现**：将获取的 Markdown 内容直接输出，不添加解释、总结或格式化

## 使用示例

### 用户输入
```
ds:什么是MPI
```

### 处理流程
1. 移除 `ds:` 前缀 → `什么是MPI`
2. 检查深度思考按钮状态（如果未启用则点击）
3. 点击输入框获取焦点
4. 输入问题并发送
5. 等待回答完成
6. 点击拷贝按钮获取 Markdown 内容
7. 从剪贴板读取内容
8. **直接输出原始 Markdown 内容**（不整理、不总结）

### 输出示例
```
MPI（Message Passing Interface，消息传递接口）是...

## 主要特点
1. 独立进程，不共享内存
2. 显式通信
3. 单程序，多数据（SPMD）

## 应用领域
- 天气预报
- 流体力学模拟
- 分子动力学
...
```

## 注意事项

1. **首次使用需要登录**：需要手动登录 DeepSeek 账号
2. **浏览器保持打开**：会话会保持打开状态供后续查询使用
3. **剪贴板权限**：需要浏览器允许剪贴板访问权限
4. **网络稳定性**：需要稳定的网络连接
