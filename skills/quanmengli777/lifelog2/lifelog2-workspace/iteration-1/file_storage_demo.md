# 暂存文件示例


## 文件格式
```markdown
- [14:30] 今天天气真好，心情不错
- [15:45] 下午开会讨论了 Q2 的目标，感觉很有挑战性
- [19:20] 晚上吃了火锅，太好吃了！
```

## 工作流程演示

### 1. 记录第一条
**用户**: 记日记 今天天气真好，心情不错

**技能操作**:
- 获取日期：2026-03-25
- 获取时间：14:30
- 创建/打开文件：`~/.trae/skills/lifelog2/drafts/2026-03-25.md`
- 追加内容：`- [14:30] 今天天气真好，心情不错`
- 回复：✓ 已记录 [14:30]

### 2. 记录第二条
**用户**: 记日记 下午开会讨论了 Q2 的目标，感觉很有挑战性

**技能操作**:
- 获取时间：15:45
- 追加到同一文件：`- [15:45] 下午开会讨论了 Q2 的目标，感觉很有挑战性`
- 回复：✓ 已记录 [15:45]

### 3. 汇总日记
**用户**: 汇总日记

**技能操作**:
- 读取文件：`~/.trae/skills/lifelog2/drafts/2026-03-25.md`
- 添加标题：`# 2026 年 3 月 25 日#lifelog`
- 输出完整内容

### 4. 存档日记
**用户**: 日记存档

**技能操作**:
1. 调用 flomo-notes 技能写入内容
2. 调用 ima-skill 技能写入内容
3. 删除文件：`~/.trae/skills/lifelog2/drafts/2026-03-25.md`
4. 回复：✓ 日记已存档

## 优点对比

| 特性 | 内存存储 | 文件存储 |
|------|---------|---------|
| 持久化 | ❌ 会话结束即丢失 | ✅ 永久保存 |
| 跨会话 | ❌ 不支持 | ✅ 支持 |
| 可靠性 | ❌ 低 | ✅ 高 |
| 可备份 | ❌ 否 | ✅ 是 |
| 易管理 | ⚠️ 中等 | ✅ 简单 |
| 占用资源 | ✅ 低 | ✅ 低 |
