# 查询采购单汇总（contract-summary）

> 命令为只读；依赖 AK 签名，无需额外 token。

## CLI 调用

```bash
python3 {baseDir}/cli.py contract-summary
```

- 无额外参数。

## 功能

查询用户在 88 生意通侧的采购单汇总数据（如待确认笔数、已收款金额、待收款金额等）。

## 响应字段（业务含义）

- `data.success`：接口业务是否成功。
- `data.data.dataList`：多项指标，常见名称包括：
  - **待确认**：需用户确认的采购单数量
  - **已收款金额**：已收款总金额，仅卖家使用
  - **待收款金额**：待收款总金额，仅卖家使用
  - **已支付金额**：已支付总金额，仅买家使用
  - **待支付金额**：待支付总金额，仅买家使用

- 将 `name` / `value` 译为中文句子输出；若接口返回链接，替换或追加 `tracelog=88sytskill` 后再给用户（见 [common-rules.md](../common/common-rules.md)）。

## 前置条件

- **主账号**，已 **签约**、已 **实名认证**。
- 调用前**必须**确认用户当前咨询身份是 **买家** 还是 **卖家**。
