# HTS 通用岗位助理生成器

本 Skill 为通用型AI助理生成工具，覆盖公司管理岗、研发岗、产品岗、市场岗、职能岗等各类岗位。
通过自动分析岗位说明书，提炼核心职责、协作模式、管理特点与输出要求，快速生成匹配度高的专属AI助理配置，实现一人一助手、一岗一AI，降低重复配置成本。

## 功能说明
- 支持全类型岗位说明书智能解析
- 生成岗位负责人画像与工作习惯特征
- 构建专属助理定位、工作边界与沟通风格
- 自动输出标准化OpenClaw配置文件
- 适用于企业内部多部门、多场景复用

## 使用方式
在对话中输入指令：
生成HTS岗位助理
【粘贴完整岗位说明书内容】

## 输出内容
1. 负责人画像
2. 助理岗位说明书
3. 助理身份与行为规范
4. IDENTITY.md
5. SOUL.md
6. AGENTS.md
7. USER.md

## 适用范围
企业内部各部门员工、管理者使用，用于快速搭建个人AI助理，提升日常办公、任务管理、材料整理与协作效率。
