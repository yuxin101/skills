
# HTS 通用岗位助理生成器

![HTS 航塔软件 Logo](https://p.qqan.com/up/97/1733197_1654067459869478.png)

## 产品介绍

HTS 通用岗位助理生成器是航塔软件（HTS）面向企业内部全岗位场景打造的 AI 助理自动生成工具，适用于管理岗、研发岗、产品岗、市场岗、行政岗、财务岗等各类岗位，无需手动编写配置文件，助力全员办公效率提升。

本工具基于 OpenClaw 标准开发，核心目标是通过智能解析岗位说明书，快速生成贴合岗位需求、符合企业规范的专属 AI 助理配置，实现“一岗一助手、一人一 AI”，降低内部数字化工具的使用门槛。

## 核心功能

- 智能解析：自动识别岗位说明书中的核心职责、工作流程、协作要求，精准提炼关键信息；

- 画像生成：生成岗位负责人的工作风格、沟通偏好及管理特点，让 AI 助理更贴合实际需求；

- 规范输出：自动生成 AI 助理的身份定位、工作边界、行为规范，确保输出内容合规统一；

- 标准配置：一键输出 OpenClaw 四大核心文件（IDENTITY.md、SOUL.md、AGENTS.md、USER.md），无需二次修改即可直接使用；

- 全岗适配：兼容企业内部所有岗位，灵活匹配不同岗位的工作场景与需求。

## 使用方式

在 OpenClaw 对话窗口中，输入以下指令并粘贴完整岗位说明书，系统将自动完成配置生成：

```

生成HTS岗位助理

【此处粘贴完整的岗位说明书内容】

```

## 输出内容

执行指令后，系统将自动返回以下全套内容，可直接用于 OpenClaw 配置：

1.  岗位负责人画像（含工作风格、沟通偏好）；

2.  AI 助理岗位说明书（明确职责、工作范围）；

3.  AI 助理身份与行为规范（贴合企业内部管理要求）；

4.  标准化 OpenClaw 配置文件（IDENTITY.md、SOUL.md、AGENTS.md、USER.md）。

## 适用场景

- 企业各级管理者：快速搭建个人专属 AI 助理，辅助处理日常事务、整理材料、跟进任务；

- 职能岗/业务岗/技术岗：通过 AI 助理规范工作流程，减少重复性工作，提升办公效率；

- 内部数字化管理：实现 AI 助理配置标准化、统一化，降低团队使用成本；

- 新员工适配：快速为新员工匹配岗位对应的 AI 助理，助力快速融入工作。

## 注意事项

1.  请确保粘贴的岗位说明书内容完整、清晰，避免因信息不全导致生成内容偏差；

2.  生成的配置文件可根据实际需求微调，核心内容无需修改；

3.  本工具仅支持航塔软件内部使用，禁止外传或用于商业用途。
