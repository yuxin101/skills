## 使用示例

### Python代码示例
```python
# 常用音色映射字典
voice_map = {
    "小明": 1,      # 度小宇
    "小红": 0,      # 度小美  
    "老师": 106,    # 度博文
    "孩子": 4,      # 度丫丫
    "旁白": 3,      # 度逍遥
}

# 特殊场景音色
emotional_voices = {
    "高兴": 5,      # 度小娇
    "悲伤": 103,    # 度米朵
    "紧急": 1,      # 度小宇
    "温柔": 0,      # 度小美 (spd=4, pit=5)
}
```

### SSML示例
```xml
<speak>
  <!-- 男女对话 -->
  <voice name="度小宇">小明：你好，今天天气不错。</voice>
  <break time="300ms"/>
  <voice name="度小美">小红：是的，适合出去散步。</voice>
  
  <!-- 情感表达 -->
  <break time="500ms"/>
  <voice name="度小娇" spd="7" pit="6">太棒了！我们快去吧！</voice>
</speak>
```

## 注意事项

1. **免费额度**：标准音色每月有5万字符免费额度，精品音色可能消耗更多额度。
2. **SSML支持**：音色代码511 (`per=511`) 用于SSML多音色模式，允许在同一请求中使用多个音色。
3. **音色可用性**：不同区域和API版本可能支持不同音色，建议测试确认。
4. **性能考虑**：复杂音色（如情感音色）可能需要更多处理时间。
