# Border Hops and Neighbor Countries

## Brazil Side Trips Need Re-Entry Logic

- The easiest Brazil border add-on is usually Iguacu with the Argentine side of the falls.
- The hardest mistake is assuming that leaving Brazil briefly changes nothing for visa or passport requirements.

## Common Side Trips

- Argentina side of Iguazu from Foz do Iguacu
- Paraguay shopping or Itaipu area extensions from the triple-border zone
- Uruguay or Argentina add-ons from the south on longer trips

## Planning Rules

- Use the same passport for all related entries and exits.
- Check whether the traveler's nationality needs visas for Brazil re-entry after the side trip.
- Separate border days from flights when possible.
- Carry the original passport, not just a digital copy, for land or air border crossings.

## Recommendation Pattern

- If the user mainly wants the waterfalls, base in Foz and decide whether the extra border crossing is worth the friction.
- If the user is already visa-sensitive, bias toward a Brazil-only route unless the side trip changes the trip materially.
