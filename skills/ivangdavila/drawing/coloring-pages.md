# Coloring Pages — Drawing

Coloring pages fail when the page looks "beautiful" to adults but exhausting to children. Simplify more than your instinct suggests.

## Age Profiles

| Age | Target complexity | Good defaults |
|-----|-------------------|---------------|
| 3-5 | very low | 1 main subject, very thick outlines, almost blank background |
| 5-7 | low | 1-2 subjects, 1-2 supporting objects, clean ground line or border |
| 7-9 | medium | 2-3 subjects, moderate patterns, simple scene depth |
| 9+ | medium-high | more texture allowed, but still clear silhouette and closed shapes |

## Printable Rules

Always ask for:
- black outlines only
- white background
- large closed shapes
- minimal background
- uncropped subject
- no gray fill, no halftones, no paper texture

If the user wants home printing, add:
- centered composition
- generous outer margins
- portrait or landscape specified up front

## Prompt Fragments That Help

Useful:
- printable black-and-white coloring page
- thick clean outlines
- large simple shapes
- minimal background
- easy for children to color
- white page with no shading

Risky:
- intricate
- ornate
- highly detailed
- realistic texture
- cinematic lighting
- rich background

## Repair Moves

If the page is too busy:
- remove background first
- reduce subject count to one
- replace "detailed" with "simple"

If line art looks muddy:
- ask for thicker outlines
- remove fur, fabric, or texture details
- restate "no shading, black outlines only"

If anatomy breaks:
- simplify pose
- face the subject forward or three-quarter view
- reduce gesture intensity

## Classroom and Activity Packs

For a themed set, keep these stable across pages:
- same outline thickness
- same border treatment
- same complexity tier
- same page orientation
- same style language
