# Color for Print

## Core Rule

- Screen color is a preview; print color is a production workflow with materials, inks, and tolerances.

## RGB to CMYK Reality

- Many vivid screen colors cannot be reproduced exactly in CMYK.
- Blues, greens, and saturated neons often shift the most.
- A print-safe palette usually needs approved alternatives, not blind conversion.
- Always check the actual print intent: brochure, packaging, large-format, label, or office printer are different workflows.

## Spot vs Process Color

| Method | Best for | Watch out for |
|--------|----------|---------------|
| CMYK process | Standard full-color print | Brand matches vary by device and stock |
| Spot color | Brand-critical consistency, special inks | Extra cost and production complexity |
| Mixed workflow | Brand anchors plus process imagery | Needs clear prepress coordination |

## Stock and Finish Effects

- Coated stock keeps colors sharper and brighter than uncoated stock.
- Matte finishes can mute contrast compared with glossy outputs.
- Paper tone affects perceived neutrality, especially creams and off-whites.
- Packaging materials can darken or flatten expected colors.

## Proofing Rules

- If the result matters commercially, review a proof instead of trusting the monitor.
- Soft proofing helps, but it is still not the final material.
- Black text and rich black fills are different production choices.
- Overprint, trapping, bleed, and total ink coverage can become real issues in press workflows.

## Print Handoff Checklist

```
□ Output profile matches the printer or production target
□ Brand-critical colors have approved print values
□ Bleed and trim constraints already considered
□ Rich black vs text black decided intentionally
□ Proof reviewed when quality matters
```

## Common Print Traps

- Delivering only hex values to a print vendor.
- Assuming the same CMYK numbers work equally on every substrate.
- Using deep black builds for small text.
- Forgetting that proof lighting changes color perception.
- Sending a digital-first palette to press without checking out-of-gamut shifts.
