# Brand Color Systems

## Core Rule

- Brand color is not one hero hue; it is a repeatable system that survives logos, campaigns, UI, packaging, and cheap production conditions.

## Build the Brand Palette in Layers

1. Anchor color: the hue users should remember first.
2. Support colors: 1-3 colors that expand range without diluting recognition.
3. Neutral system: darks, lights, and muted surfaces that let the brand colors breathe.
4. Signal colors: success, warning, error, info, kept distinct from brand accents.

- If every color is "brand," nothing is brand.
- Most systems need stronger neutrals than stakeholders expect.

## Recognition vs Variety

- Repetition builds recognition faster than adding more palette options.
- Campaign palettes can flex, but the core anchor needs a stable home.
- The same brand color may need different tonal variants for dark backgrounds, print, and UI text.
- A color that works in a logo does not automatically work as a button fill or chart series.

## Logo and Environment Rules

- Validate the logo on light, dark, photographic, and low-contrast environments.
- Protect enough clear space that color is not doing all the separation work.
- When the brand mark depends on color contrast internally, test small sizes aggressively.
- Do not let social avatars, favicons, or stickers force a weaker crop or weaker color relationship.

## Campaign and Marketing Use

- Use campaign accents to support the message, not replace the brand anchor.
- If a campaign palette is louder than the parent brand, define exactly where it is allowed.
- Marketing gradients should still leave room for readable headlines, CTAs, and legal text.
- Seasonal campaigns can bend the system, but they should still feel like the same company.

## Packaging and Physical Materials

- Saturated digital mockups often print duller or darker.
- Paper, fabric, plastic, metal, and matte coatings all shift perceived color.
- If color accuracy matters, define approved print values and proof them.
- Metallic or fluorescent inks are special production workflows, not normal CMYK substitutions.

## Common Branding Traps

- Building a brand palette with no neutral strategy.
- Letting trend colors overpower the recognizability of the anchor hue.
- Using the same color formula for web UI, social graphics, and print without adjustment.
- Asking one saturated accent to carry brand, hierarchy, and accessibility at the same time.
- Treating campaign palettes as free-form art direction with no guardrails.
