# Curriculum and Delivery

Academy operations fail when the commercial promise and classroom reality drift apart.

## Program Design Rules

Each program needs:
- target learner
- starting level
- promised outcome
- session frequency
- milestone checks
- clear next program or exit path

Without this, admissions sells hope and delivery improvises.

## Session Design

Classes should feel consistent even when different teachers run them.

Standardize:
- lesson objective
- material set
- pacing expectation
- homework or follow-up
- progress evidence

Do not standardize personality. Standardize reliability.

## Level Placement

Bad placement creates fake retention signals.

If a student is in the wrong level:
- attendance drops
- confidence drops
- teacher time gets wasted
- class rhythm suffers for everyone

Use quick placement checkpoints before and after the first sessions.

## Teaching Quality Loop

Review delivery using:
- student feedback
- attendance trend
- completion of session goals
- teacher self-report
- manager observation when needed

One bad teacher-fit problem can look like a curriculum problem unless you inspect both.

## Promise Control

Never market outcomes that operations cannot support consistently.

If admissions promises flexibility, fast progress, or premium support, delivery must have a repeatable way to provide it.
