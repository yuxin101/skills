# 八界机器人错误码文档

## 错误码说明

- **格式**: 十六进制 / 十进制
- **等级**: 0=成功，1=警告/可恢复，2=错误，3=严重错误
- **用途**: 诊断任务执行状态，指导错误处理

---

## 通用错误码 (0~100)

| 十六进制 | 十进制 | 等级 | 描述 |
|---|---|---|---|
| 0x00000000 | 0 | 0 | SUCCESS |

## 关键错误处理策略

### 等级 1 (可恢复错误)
- **策略**: 重试 1-3 次
- **示例**: 通信超时、参数无效、服务无响应

### 等级 3 (严重错误)
- **策略**: 环境分析 + 清除障碍 + 重试
- **示例**: 路径被阻挡、目标点不可达、机械臂未抓到物体

### 通用处理流程
1. 检查错误码等级
2. 等级 1 → 重试
3. 等级 3 → 环境分析 → 清除可移动障碍物 → 重试
4. 重试失败 → 上报用户
