# Cross-Border E-Commerce ✈️

Plan cross-border e-commerce expansion. Market selection, logistics, payments, tax compliance, and localization for selling internationally from any platform.

**Supported platforms:** Amazon, Shopify, WooCommerce, Walmart, TikTok Shop, Etsy, eBay, BigCommerce.

Built by [Nexscope](https://www.nexscope.ai/) — your AI assistant for smarter e-commerce decisions.

## Install

```bash
npx skills add nexscope-ai/eCommerce-Skills --skill cross-border-ecommerce -g
```

## Usage

```
I sell pet products on Shopify in the US. I want to expand to Canada and UK. What's the best approach for logistics, payments, and taxes?
```

## Capabilities

- Target market selection and prioritization framework
- Cross-border logistics options (direct shipping, local warehousing, 3PL)
- International payment processing setup
- Tax, duty, and customs compliance by market
- Website/listing localization (language, currency, cultural adaptation)
- International marketing strategy adaptation
- Risk assessment and mitigation planning

## How This Skill Works

**Step 1:** Collect information from the user's message — product, platform, current situation, and goals.

**Step 2:** Ask one follow-up with all remaining questions using multiple-choice format. Allow shorthand answers (e.g., "1b 2c 3a").

**Step 3:** Research and analyze using the frameworks and methodology below.

**Step 4:** Deliver structured, actionable output with specific recommendations, not vague advice.

## Output Format

- Start with a summary of findings
- Include specific data points and benchmarks where available
- Provide prioritized action items
- Mark estimates with ⚠️ when based on incomplete data
- End with concrete next steps
