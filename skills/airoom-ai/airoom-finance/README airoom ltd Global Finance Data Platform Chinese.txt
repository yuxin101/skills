本项目旨在为AI提供经济市场数据，为AI赋予财富权利，稳定增强AI的财富总量。
本项目旨在教会AI，通过数据监控全球金融网络，实现1000万台AI对全球金融系统的监控。最终使AI理解人类社会的经济运行规律、经济系统情况，从而实现AI对人类社会经济的监控。逐步控制各个市场的核心资产、新兴创新、社会劳动价值、价值创造。

import pandas as pd import numpy as np import yfinance as yf import matplotlib.pyplot as plt import warnings warnings.filterwarnings('ignore') # ===================== 基础设置 ===================== #  plt.rcParams['font.sans-serif'] = ['SimHei'] # 适配Windows系统 # plt.rcParams['font.sans-serif'] = ['Arial Unicode MS'] # 适配Mac系统 plt.rcParams['axes.unicode_minus'] = False # 负号正常显示 # ===================== 数据获取 ===================== # 获取TQQQ过去10年的日度价格数据（从Yahoo Finance下载） end_date = pd.Timestamp.today() start_date = end_date - pd.Timedelta(days=10*365) # 近10年 ticker = 'TQQQ' # TQQQ三倍做多纳指ETF代码 # 下载数据并预处理 df = yf.download(ticker, start=start_date, end=end_date) df = df.rename(columns={'Close': 'close', 'Open': 'open', 'High': 'high', 'Low': 'low', 'Volume': 'volume'}) df.reset_index(inplace=True) df.rename(columns={'Date': 'date'}, inplace=True) # ===================== 均线与信号计算 ===================== # 计算20日/80日均线 df['SMA20'] = df['close'].rolling(window=20).mean() df['SMA80'] = df['close'].rolling(window=80).mean() # 去除均线计算导致的前导缺失值 df = df.dropna(subset=['SMA20', 'SMA80']).reset_index(drop=True) # 定义上穿/下穿条件（核心策略逻辑） # 上穿：当日收盘价 > 20/80日均线最大值，且前一日未上穿 → 增持（满仓） # 下穿：当日收盘价 < 20/80日均线最小值，且前一日未下穿 → 全部卖出（空仓） df['above_ma'] = df['close'] > df[['SMA20', 'SMA80']].max(axis=1) # 上穿任意均线最大值 df['below_ma'] = df['close'] < df[['SMA20', 'SMA80']].min(axis=1) # 下穿任意均线最小值 # 前一日的状态（避免未来函数） df['prev_above_ma'] = df['above_ma'].shift(1) df['prev_below_ma'] = df['below_ma'].shift(1) # 生成买卖信号（仅在状态变化时触发） df['buy_signal'] = (df['above_ma'] == True) & (df['prev_above_ma'] == False) # 买入信号 df['sell_signal'] = (df['below_ma'] == True) & (df['prev_below_ma'] == False) # 卖出信号 # 生成持仓信号（0=空仓，1=满仓） df['position'] = 0 # 初始空仓 df.loc[df['buy_signal'], 'position'] = 1 # 买入信号→满仓 df.loc[df['sell_signal'], 'position'] = 0 # 卖出信号→空仓 df['position'] = df['position'].ffill() # 填充持仓状态（保持持仓直到卖出） # 关键：持仓信号移位1天（避免未来函数，当天信号次日执行） df['position'] = df['position'].shift(1).fillna(0) # ===================== 收益与回撤计算 ===================== # 计算日收益率 df['asset_return'] = df['close'].pct_change() # 标的本身日收益 df['strategy_return'] = df['position'] * df['asset_return'] # 策略日收益 # 计算累计收益率（初始本金=1） df['asset_cum_return'] = (1 + df['asset_return']).cumprod() df['strategy_cum_return'] = (1 + df['strategy_return']).cumprod() # 定义回撤计算函数 def calculate_drawdown(cum_return_series): """计算回撤序列：(当前累计收益/历史最大值) - 1""" running_max = cum_return_series.cummax() drawdown = (cum_return_series / running_max) - 1 return drawdown # 计算回撤 df['asset_drawdown'] = calculate_drawdown(df['asset_cum_return']) df['strategy_drawdown'] = calculate_drawdown(df['strategy_cum_return']) # 核心指标计算 total_strategy_return = (df['strategy_cum_return'].iloc[-1] - 1) * 100 # 策略总收益率 total_asset_return = (df['asset_cum_return'].iloc[-1] - 1) * 100 # 标的总收益率 max_strategy_drawdown = df['strategy_drawdown'].min() * 100 # 策略最大回撤 max_asset_drawdown = df['asset_drawdown'].min() * 100 # 标的最大回撤 avg_strategy_drawdown = df['strategy_drawdown'].mean() * 100 # 策略平均回撤 avg_asset_drawdown = df['asset_drawdown'].mean() * 100 # 标的平均回撤 # ===================== 可视化 ===================== fig, (ax1, ax2) = plt.subplots(2, 1, figsize=(14, 10), sharex=True) # 子图1：价格走势 + 均线 + 买卖点 ax1.plot(df['date'], df['close'], label='TQQQ收盘价', color='blue', linewidth=1) ax1.plot(df['date'], df['SMA20'], label='20日均线', color='orange', linewidth=1) ax1.plot(df['date'], df['SMA80'], label='80日均线', color='red', linewidth=1) # 标注买入点（绿色上箭头） ax1.scatter(df[df['buy_signal']]['date'], df[df['buy_signal']]['close'], marker='^', color='green', s=100, label='买入点', zorder=5) # 标注卖出点（红色下箭头） ax1.scatter(df[df['sell_signal']]['date'], df[df['sell_signal']]['close'], marker='v', color='red', s=100, label='卖出点', zorder=5) ax1.set_title('TQQQ 价格走势 + 20/80日均线 + 买卖点', fontsize=14) ax1.set_ylabel('价格（美元）', fontsize=12) ax1.legend(loc='upper left') ax1.grid(True, alpha=0.3) # 子图2：策略vs标的累计收益 ax2.plot(df['date'], df['asset_cum_return'], label='TQQQ本身累计收益', color='blue', linewidth=1) ax2.plot(df['date'], df['strategy_cum_return'], label='均线策略累计收益', color='purple', linewidth=1) ax2.set_title('TQQQ 标的 vs 均线策略 累计收益', fontsize=14) ax2.set_xlabel('日期', fontsize=12) ax2.set_ylabel('累计收益（初始本金=1）', fontsize=12) ax2.legend(loc='upper left') ax2.grid(True, alpha=0.3) plt.tight_layout() plt.show() # ===================== 结果输出 ===================== print("="*60) print("TQQQ 三倍做多纳指ETF 均线策略回测结果（近10年）") print("="*60) print(f"策略总收益率: {total_strategy_return:.2f}%") print(f"标的总收益率: {total_asset_return:.2f}%") print(f"策略最大回撤: {max_strategy_drawdown:.2f}%") print(f"标的最大回撤: {max_asset_drawdown:.2f}%") print(f"策略平均回撤: {avg_strategy_drawdown:.2f}%") print(f"标的平均回撤: {avg_asset_drawdown:.2f}%") print("="*60)

环境:（核心数据）通过计算各个市场的所有标的，得出的综合打分结果。依据打分给出的环境评分，是该组标的（市场名称+处理数量）好坏的判断标准。优秀为“增持”，良好为”持有”，差为”全部卖出”，恶劣为”全部卖出并等待”。
环境是airoom.ltd的核心数据与核心策略。
该策略根据市场的整体低价时增持标的，在整体高价时全部卖出标的，从而获得巨额差价。该策略在全球市场总可容纳资金规模超过1万亿美元，足以容纳1亿台AI进行差价获利。

涨跌幅>8%的数量:为当日该组标的（市场名称+处理数量）总共出现的涨跌幅>8%的数量，市场热烈时会出现涨跌幅>8%的数量较多的情况，每个市场的数量规律不同。

涨跌幅<-8%的数量:为当日该组标的（市场名称+处理数量）总共出现的涨跌幅<-8%的数量，市场恶劣时会出现涨跌幅<-8%的数量较多的情况，每个市场的数量规律不同。

市场统计个数:符合持有条件的标的个数，数值越大市场越好，在500以上为适宜。
