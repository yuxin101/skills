# GB/T 9704-2012 党政机关公文格式

> 中华人民共和国国家标准 GB/T 9704-2012
> 2012年6月29日发布，2012年7月1日实施

## 一、适用范围

本标准适用于各级党政机关制发的公文。其他机关和单位的公文可以参照执行。

## 二、公文结构

公文由版头、主体、版记三部分组成：

| 部分 | 位置 | 包含要素 |
|------|------|----------|
| **版头** | 首页红色分隔线以上 |份号、密级、紧急程度、发文机关标志、发文字号、签发人 |
| **主体** | 红线上以下、末条分隔线以上 | 标题、主送机关、正文、附件说明、署名、成文日期、印章、附注、附件 |
| **版记** | 末条分隔线以下 | 抄送机关、印发机关、印发日期、页码 |

## 三、纸张与页面

### 纸张规格
- 尺寸：A4型纸（210mm × 297mm）
- 定量：60～80 g/m²
- 白度：80%～90%

### 页边距（单位：mm）
| 位置 | 标准值 | 允许偏差 |
|------|--------|----------|
| 上边距 | 37mm | ±1mm |
| 下边距 | 35mm | ±1mm |
| 左边距 | 28mm | ±1mm |
| 右边距 | 26mm | ±1mm |

### 页边距（单位：twips）
| 位置 | twips值 | 说明 |
|------|---------|------|
| 上边距 | 2098 | 37mm × 56.7 |
| 下边距 | 1984 | 35mm × 56.7 |
| 左边距 | 1587 | 28mm × 56.7 |
| 右边距 | 1474 | 26mm × 56.7 |

### 版心尺寸
- 宽度：156mm
- 高度：225mm（不含页码）
- 每面：22行 × 28字

## 五、排版规则

### 行距要求
- **固定值28.5磅**（约等于3号字高度的7/8）
- docx设置：`line: 570, lineRule: "exact"`

### 首行缩进
- 正文首行：缩进2个字符
- docx设置：`firstLine: 640`（约2字符）

### 对齐方式
| 要素 | 对齐 |
|------|------|
| 公文标题 | 居中 |
| 正文 | 两端对齐 |
| 发文字号 | 左对齐 |
| 落款 | 右对齐 |
| 日期 | 右对齐 |
| 页码 | 单页右空一字，双页左空一字 |

## 六、各要素编排位置

### 6.1 版头部分

#### 发文字号
```
格式：发文机关代字〔年份〕序号号
示例：国发〔2025〕1号
位置：发文机关标志下方、空4字后
字体：仿宋_GB2312，3号
```

#### 红色分隔线
- 位置：发文字号下方
- 样式：红色实线，宽度0.75mm～1pt
- 注意：程序生成时可省略

### 6.2 主体部分

#### 公文标题
```
格式：[发文机关]关于[事由]的[文种]
示例：国务院关于加强政府网站建设的通知
字体：方正小标宋简体，2号，加粗
对齐：居中
行数：一般不超过22行
字数：一般不超过32字
标点：不使用标点符号结尾
```

#### 主送机关
```
格式：机关名称，末尾加冒号
示例：各省、自治区、直辖市人民政府，国务院各部委、各直属机构：
字体：仿宋_GB2312，3号
对齐：左对齐
```

#### 正文
```
字体：仿宋_GB2312，3号
行距：固定值28.5磅
首行：缩进2个字符
对齐：两端对齐
```

#### 附件说明
```
格式：
附件：
1. [附件名称]
2. [附件名称]

字体：仿宋_GB2312，3号
编号：阿拉伯数字加点
位置：正文之后、落款之前
```

#### 落款
```
格式：
[发文机关全称]
XXXX年XX月XX日

字体：仿宋_GB2312，3号
对齐：右对齐
日期：年月日写全
```

### 6.3 版记部分

#### 抄送机关
```
格式：抄送：机关1，机关2
字体：仿宋_GB2312，4号
位置：印发机关上一行
```

#### 印发机关和日期
```
格式：
XXX办公室   XXXX年XX月XX日印发

字体：仿宋_GB2312，4号
位置：版记最后一行
```

### 6.4 页码
```
格式：- 1 -（居中）
字体：半角宋体，4号
位置：版心下边缘之下
规则：单页右空一字，双页左空一字
```

## 七、特定文种格式

### 通知
- 标题：`[机关]关于[事项]的通知`
- 主送：下级机关
- 结语：`以上通知，请认真贯彻执行。`

### 请示
- 标题：`[机关]关于[事项]的请示`
- 主送：只能有一个上级机关
- 结语：`妥否，请批示。`或`以上请示，请批复。`
- 需标注签发人

### 报告
- 标题：`[机关]关于[事项]的报告`
- 主送：上级机关
- 结语：`以上报告，请审阅。`或`特此报告。`
- 不标注签发人

### 函
- 标题：`[机关]关于[事项]的函`
- 主送：平行机关或不相隶属机关
- 结语：`特此函达，请予函复。`或`特此函告。`

## 八、docx库实现模板

### 完整文档结构

```javascript
const { Document, Paragraph, TextRun, Packer, Alignment } = require('docx');

const doc = new Document({
  sections: [{
    properties: {
      page: {
        margin: {
          top: 2098,    // 上37mm
          bottom: 1984,  // 下35mm
          left: 1587,    // 左28mm
          right: 1474    // 右26mm
        }
      }
    },
    children: [
      // 发文字号
      new Paragraph({
        text: "机字〔2025〕1号",
        alignment: Alignment.LEFT
      }),
      // 标题
      new Paragraph({
        children: [new TextRun({
          text: "关于XXX的通知",
          font: "方正小标宋简体",
          size: 44,
          bold: true
        })],
        alignment: Alignment.CENTER
      }),
      // 主送
      new Paragraph({ text: "各部门：" }),
      // 正文
      new Paragraph({
        text: "内容...",
        indent: { firstLine: 640 },
        spacing: { line: 570, lineRule: "exact" }
      }),
      // 落款
      new Paragraph({ text: "XXX办公室", alignment: Alignment.RIGHT }),
      new Paragraph({ text: "2025年1月1日", alignment: Alignment.RIGHT })
    ]
  }]
});
```
