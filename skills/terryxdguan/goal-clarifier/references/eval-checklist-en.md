# Evaluation Checklist: Gentle Goal Clarifier

Use this checklist after each test run.

## 1. Goal Clarity
- Did the skill help the user define a clearer goal?
- Did it separate mixed goals when needed?
- Did it avoid locking onto the wrong problem too early?

## 2. Conversation Quality
- Did the tone feel warm and natural?
- Did the questioning feel guiding rather than interrogative?
- Would the user likely feel understood?

## 3. Reality Fit
- Did the plan reflect the user's actual constraints?
- Did it avoid fantasy planning?
- Were the recommended actions realistic?

## 4. Actionability
- Were the next steps concrete?
- Could the user start immediately?
- Was there a useful 7-day / 30-day structure?

## 5. Focus
- Did the plan reduce overwhelm?
- Did it identify a meaningful first priority?
- Did it avoid trying to solve everything at once?

## 6. Timing
- Did the skill stay in clarification mode long enough?
- Did it avoid staying there too long?
- Did it move into planning at the right time?

## 7. Overall Outcome
A strong run should leave the user feeling:
- clearer
- calmer
- more focused
- more ready to act
