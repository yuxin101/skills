# Stock Assistant Pro - 专业股票分析助手

基于实战验证的三维选股策略，助你科学投资。

## 核心策略

### 1. 技术面分析
- **五日线判断** - 线上做多，线下观望
- **MACD 信号** - 金叉放量买入，死叉放量卖出
- **成交量分析** - 放量确认信号有效性

### 2. 基本面筛选
- **估值评估** - PE/PB 低于行业平均
- **业绩分析** - 营收/利润持续增长
- **财务健康** - 负债率、现金流、ROE 检查

### 3. 做 T 策略
- **日内做 T** - 低买高卖赚差价
- **趋势跟踪** - 上升期持有，下降期回避
- **仓位管理** - 单只不超 30%，做 T 仓位 30-50%

## 功能说明

### /analyze [股票代码]
分析单只股票，输出：
- 当前价格、涨跌幅
- 五日线位置判断
- MACD 信号状态
- 基本面评分（估值/业绩/财务）
- 操作建议（买入/持有/卖出/观望）

### /watchlist
查看观察列表：
- 已添加的股票列表
- 每只股票的当前信号
- 今日涨跌幅排序

### /signal
获取今日买卖信号：
- 强力买入（五日线 + 金叉 + 放量）
- 谨慎买入（部分条件满足）
- 减仓卖出（死叉/跌破支撑）
- 坚决回避（五日线下方 + 死叉）

### /report [股票代码]
生成详细分析报告（PDF/Markdown）：
- 公司概况
- 财务健康度分析
- 技术指标详解
- 机构持仓变化
- 风险提示

### /strategy
查看/修改策略配置：
- 五日线周期（默认 5 日）
- MACD 参数（默认 12,26,9）
- 放量标准（默认 1.5 倍均量）
- 止损位（默认 -8%）
- 止盈位（默认 +20% 分批）

## 使用示例

```bash
# 分析贵州茅台
/analyze 600519

# 分析招商银行
/analyze sh600036

# 查看今日信号
/signal

# 生成腾讯控股报告
/report 00700.HK

# 修改止损位
/strategy set stop_loss -10%
```
