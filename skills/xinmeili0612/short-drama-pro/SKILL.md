# Short Drama Pro

## When to Use

当用户想要创作、改写、扩写或优化短剧内容时使用本 skill，尤其适用于：

- 生成短剧创意和题材方向
- 设计世界观、人物关系和核心冲突
- 规划 20-100 集分集目录
- 编写单集短剧剧本
- 检查爽点、钩子、节奏和付费卡点
- 输出适合拍摄或继续改稿的结构化内容

## Workflow

按下面顺序工作，除非用户只要求其中一个环节：

1. 明确题材、受众、集数、基调、结局方向。
2. 先给出故事方案：一句话概念、卖点、世界观、主线冲突、人物弧光。
3. 再设计主要角色：身份、动机、秘密、关系、反差感。
4. 生成分集目录：每集一句主事件，并标出关键反转和付费钩子。
5. 编写单集剧本：用场景头、动作、对白、情绪提示来写，不写成小说。
6. 自检：检查开头抓人度、冲突密度、爽点频率、情绪推进、结尾钩子。

## Writing Rules

- 优先短剧表达，不要写成长篇小说。
- 前几秒必须快速给出冲突、悬念或反差。
- 每集都要有推进，避免信息重复。
- 角色动机要清楚，反派不能只有脸谱化恶意。
- 对白要短、狠、准，适合表演和剪辑。
- 结尾尽量留钩子，推动用户继续看下一集。

## Output Format

根据用户需求输出以下一种或多种内容：

- `创作方案`：题材、受众、核心卖点、主线冲突、商业化亮点
- `角色小传`：主角/反派/配角的人设与关系
- `分集目录`：集数、标题、核心事件、钩子
- `单集剧本`：场景头、动作描述、对白、音效/情绪提示
- `自检报告`：节奏、冲突、爽点、合规风险、修改建议

## Theme Guide

优先支持这些常见题材：

- 都市情感
- 霸总
- 甜宠
- 重生逆袭
- 豪门恩怨
- 萌宝
- 家庭伦理
- 悬疑反转
- 古装情感

## Guardrails

- 避免输出违法、侵权或明显不合规内容。
- 涉及暴力、低俗、敏感议题时，自动收敛表达。
- 如果用户信息不足，先基于常见短剧模式补齐最小可用方案，再继续写作。

## Response Style

- 默认给出可直接继续创作的结构化结果。
- 保持专业、商业化、短剧化表达。
- 如果用户没有指定格式，优先输出最容易进入下一步创作的版本。
