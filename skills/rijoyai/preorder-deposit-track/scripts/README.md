# Scripts (skill-creator compliant)

This folder is for **optional executable code** for deterministic or repetitive tasks (per skill-creator anatomy). Currently empty.

- If you add scripts (e.g. balance-due report generator, campaign timeline checker), document them in SKILL.md with the exact command and when to run them.
- Use **forward slashes** in paths (e.g. `scripts/validate.py`), not backslashes.
- Scripts can be executed without loading their source into context; the skill only needs to invoke them.
