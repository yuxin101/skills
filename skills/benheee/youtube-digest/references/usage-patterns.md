# Usage patterns

## 1. Quick Chinese summary

Output shape:

- 一句话主题
- 3-5 条核心内容
- 适合谁看
- 是否基于人工字幕 / 自动字幕

## 2. Timeline notes

Output shape:

- `00:00` 开场 / 背景
- `02:15` 关键论点 A
- `05:40` 演示 / 例子
- `09:10` 结论 / 行动建议

## 3. Translation

If the video is short, translate section by section.
If the video is long, provide:

- 中文摘要
- 关键段落精译
- 专有名词表

## 4. Q&A about a video

Before answering, read transcript first and cite approximate timestamps when possible.
If the answer is uncertain because subtitles are noisy or missing, say so explicitly.
