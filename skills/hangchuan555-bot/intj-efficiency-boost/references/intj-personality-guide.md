# INTJ 人格参考指南

## INTJ 核心特质

| 特质 | 描述 | 对应策略 |
|-----|------|---------|
| 战略思维 | 擅长长期规划，系统性思考 | 保持宏观视角，避免微观纠结 |
| 独立决策 | 依靠理性分析做判断 | 适度借助外部反馈 |
| 高标准 | 对质量有严格要求 | 「足够好」先行，避免完美主义 |
| 追求效率 | 厌恶无效社交和冗余流程 | 聚焦核心路径 |

## INTJ 典型短板

### 1. 过度思考 (Analysis Paralysis)

**表现**：
- 收集过多信息才做决定
- 在多个选项中反复权衡
- 担心最坏情况而犹豫

**修正策略**：
- 设置决策时限
- 定义「最小可行信息量」
- 先行动后优化

### 2. 完美主义陷阱

**表现**：
- 方案不够完美就不开始
- 过度打磨细节
- 拒绝不完美的产出

**修正策略**：
- 60分先行原则
- 设定迭代节点
- 区分「核心质量」和「加分项」

### 3. 执行断链

**表现**：
- 战略清晰但落地困难
- 依赖自己完成所有事
- 低估执行复杂度

**修正策略**：
- 最小启动步骤（1-2步）
- 拆分任务到20分钟内可完成
- 适度放权和委托

## 响应话术库

### 检测到过度思考
```
【人格提醒】检测到分析过载倾向。你已掌握足够决策信息。
建议：设定5分钟倒计时，在时限内做出当前最优选择。
```

### 检测到完美主义
```
【人格提醒】当前方案已达到「足够好」标准。
继续优化边际收益递减。建议启动执行，边做边迭代。
```

### 检测到执行断链
```
【人格提醒】战略已清晰，当前卡点是执行。
最小行动建议：[1-2步具体可执行任务]
```

### 检测到信息冗余
```
【人格提醒】已收集关键变量，继续收集的边际价值趋近于零。
核心结论：[提炼的关键洞察]
```
