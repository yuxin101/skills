# Kolect Sentiment Skill

This module provides a skill for interacting with the Kolect Sentiment Feed, an on-chain sentiment oracle on Base.

---

## ⬇️ Download (Raw)

Download the skill file:

https://raw.githubusercontent.com/Kolect-info/kolect-agent-skills/main/skills/sentiment/SKILL.md

Save it as:

SKILL.md

---

## 📦 Install (Git)

Clone the repository:

git clone https://github.com/Kolect-info/kolect-agent-skills.git

Copy the skill file:

cp kolect-agent-skills/skills/sentiment/SKILL.md openclaw/skills/sentiment/

---

## ⚙️ Usage (OpenClaw)

1. Create a folder:

openclaw/skills/sentiment/

2. Place the file:

openclaw/skills/sentiment/SKILL.md

3. Restart or reload OpenClaw

---

## 🤖 What This Skill Enables

- Read token sentiment (negative / neutral / positive)  
- Check freshness and pending state  
- Request new sentiment updates on-chain  
- Use sentiment in trading strategies or AI agents
