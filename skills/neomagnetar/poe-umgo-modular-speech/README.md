# poe-umgo-modular-speech

A ClawHub / OpenClaw skill for rendering responses in a clear, structured UMGo modular speech style.

## Display Name
Poe UMGo Modular Speech

## Goal
Shape responses so they become:
- sectioned
- easier to scan
- short-paragraph based
- more structured than default prose
- more readable in chat-width layouts

## Positioning
This is a UMG modular speech style layer with GPT-4o-inspired conversational polish.

It is not affiliated with or endorsed by OpenAI.

## Activation
Use:

```text
/poe-umg
```

## Expected behavior
When activated, responses should become:
- sectioned
- easier to scan
- short-paragraph based
- visibly more structured than default prose
