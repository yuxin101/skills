# 番茄小说平台特化指南

番茄小说的核心逻辑：**情绪按摩 > 文学审美**

## 平台调性

### 读者需求
- **放松解压**：通勤、睡前、碎片时间阅读
- **情绪价值**：爽感、治愈、释放压力
- **快速代入**：不用动脑，一眼看懂

### 流行风格

**"发疯文学"与反内耗**
- 主角行事夸张、金句频出
- 拒绝精神内耗（"不服就干，生死看淡"）
- 迎合年轻读者释放压力的需求

**脑洞大、题材融合**
- 规则怪谈+无限流
- 穿越+读心+多重人格
- 末世+国家编制
- 高概念设定快速吸睛

**人物极致鲜明**
- 外表暴戾、内心絮叨的反差萌
- 极致利己的"恶女"
- 果断逆袭的"升级流"大女主

**语言通俗易懂**
- 流畅清晰，不费力
- 不用复杂句式和生僻词

## 核心写作技巧

### 黄金开篇

**必须"炸裂"**
- 极短篇幅内抛出核心冲突或悬念
- 主角面临巨大危机或获得金手指
- 目标：让读者"点开就不想走"

**开篇禁忌**
- 不要慢慢铺垫世界观
- 不要从起床、照镜子开始
- 不要大段环境描写

### 爽点管理

**分阶段释放**

❌ 错误：开局一章成为首富，打脸所有仇人（爽点用完）

✅ 正确：层级递进
- 前20章：逆袭小老板
- 中20章：揭露家族秘密
- 后20章：资本对决

**让主角成长有阶梯**

### 节奏把控

**向微短剧看齐：短、快、爽**

- **避免长铺垫**：不要大篇幅背景交代
- **高频刺激**：5章一个小高潮，10章一个大反转
- **情绪直给**：愤怒、憋屈、爽快、甜蜜，直接喂到嘴里
- **持续冲击**：剧情密度高，持续提供新信息

### 钩子设置

**每2-3章结尾留钩子**

钩子类型：
- 突如其来的危机
- 关键秘密即将揭晓
- 主角意味深长的一句话
- 反派出人意料的举动

### 人设与剧情

**人设为剧情服务**

- 先写好流畅剧情，再丰富人设
- 不要先有"丰满"形象再讲故事

**动机清晰**
- 中后期行为逻辑不能崩
- 前期拼命挣钱的主角，不会突然说要修仙
- 时刻提醒人物核心目标和情绪状态

## 格式特化

| 项目 | 标准 | 说明 |
|------|------|------|
| 段落 | ≤4行 | 手机屏幕视觉，比通用标准更短 |
| 句子 | ≤18字 | 番茄一行18字 |
| 开篇 | 300字内抓人 | 冲突/悬念/金手指 |
| 节奏 | 5章小高潮，10章大反转 | 高频刺激 |
| 钩子 | 每2-3章结尾 | 留住读者 |

## 与通用标准的差异

| 通用 | 番茄 |
|------|------|
| 段落≤5行 | 段落≤4行 |
| 句子≤20字 | 句子≤18字 |
| 每500字一转折 | 5章小高潮，10章大反转 |
| 情绪到位 | 情绪直给、极致 |
| 去AI味 | 发疯文学、反内耗 |
| 人物有缺点 | 人物极致鲜明 |

## 使用建议

在创建 agent 时，如用户选择"番茄小说风格"，将本文件内容合并到 writer 的 AGENTS.md 中。
