# 十天干克应（天地盘天干组合）

天盘天干在上，地盘天干在下。格式：**天盘X加地盘Y**。

> 注：甲在奇门中隐遁不露，通常以甲子戊、甲戌己等六仪代替。但在分析天地盘组合时，仍需列出甲的组合作为参考。实际排盘中天盘地盘出现的是乙丙丁戊己庚辛壬癸九个天干。

## 十一、组合速查规律

### 吉组合特征
1. **三奇互配**：乙丙、乙丁、丙丁组合多吉
2. **天干相合**：甲己、乙庚、丙辛、丁壬、戊癸相合多有转机
3. **三奇配戊**：丙加戊（飞鸟跌穴）、丁加戊（青龙转光）为大吉
4. **壬加戊/壬加乙**：水生木为吉

### 凶组合特征
1. **庚加任何干**：庚为凶星，多数组合为凶
2. **癸加火（丙丁）**：水克火为凶
3. **三奇被克**：庚加乙、壬加丙、己加丁等
4. **同干伏吟**：任何天干自加自为伏吟

### 克应强弱
- **天盘克地盘**：外来之力克内在，主被动受制
- **地盘克天盘**：内在之力克外来，主主动制人
- **天盘生地盘**：外力生内在，主有助力
- **地盘生天盘**：内在泄气予外来，主消耗
