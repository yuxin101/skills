# 用神取法系统

## 一、概述

奇门遁甲断事的核心在于**取用神**。根据所问事项的类型，确定代表该事物的用神（天干、八门、九星、八神、宫位等），然后分析用神的旺衰、落宫、与其他盘元素的生克关系，最终判断吉凶成败。

### 通用取用原则

1. **日干**代表问事者本人（求测者）
2. **时干**代表所问之事
3. **年干**代表长辈、上级、大环境
4. **月干**代表同辈、平级、月运
5. **值符**（九星）代表上级、领导、贵人
6. **值使**（八门）代表执行、结果

### 2.1 求财（一般）

| 用神 | 说明 |
|---|---|
| **日干** | 求测者本人 |

**断法**：
1. 看日干落宫与生门落宫的关系（生门生日干宫→有财）

4. 生门入墓、落空亡→求财无望
5. 日干宫克戊宫→可得财（我克者为财）
6. 日干宫生戊宫→财泄我力（付出大于收获）

## 十四、旺衰判断通则

### 14.1 用神旺相判断
1. **得令**：用神五行与当前月令（季节）相合，为旺
2. **得地**：用神落在与其五行相生/比和的宫位
3. **得奇**：用神宫位有三奇（乙丙丁）临之
4. **得门**：用神宫位有吉门（开休生）临之
5. **得星**：用神宫位有吉星临之

### 14.2 用神衰弱判断
1. **失令**：五行与月令相克
2. **失地**：落在被克的宫位
3. **入墓**：落入墓库宫位
4. **逢空**：落在空亡宫位
5. **受制**：被凶星凶门凶神夹击

### 14.3 成败逻辑
- **用神旺相 + 吉门吉星**→事可成
- **用神旺相 + 凶门凶星**→事可成但有波折
- **用神衰弱 + 吉门吉星**→有贵人助但力不足
- **用神衰弱 + 凶门凶星**→事难成
- **用神逢空**→事为虚，暂不可得（但空亡也有"空而后实"之说）
- **用神入墓**→事被困，需等出墓之时
