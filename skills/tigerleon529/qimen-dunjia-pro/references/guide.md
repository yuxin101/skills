# 奇门遁甲初学者使用指南

## 什么是奇门遁甲？

奇门遁甲是中国古代最高层次的预测学之一，被誉为"帝王之学"。与太乙神数、大六壬并称为古代"三式"（三大绝学）。

**"奇"** = 乙、丙、丁三奇（日奇、月奇、星奇）
**"门"** = 开、休、生、伤、杜、景、死、惊八门
**"遁"** = 隐藏
**"甲"** = 十天干之首，甲为统帅

合起来就是：**天干甲隐藏（遁）在六仪之下，通过三奇和八门来预测吉凶**。

## 奇门遁甲能做什么？

### ✅ 适合的事情
- **择时**：选择做某事的最佳时间
- **择方**：选择有利的方位
- **占事**：预测某件事的成败
- **决策**：帮助在两难中做出选择
- **出行**：看出行是否顺利
- **求财**：看某笔投资/生意的前景
- **考试**：看能否考上
- **官司**：看诉讼结果
- **婚姻**：看姻缘走向
- **健康**：看病情发展
- **失物**：找丢失的东西
- **风水**：看住宅吉凶

### ❌ 不适合的事情
- 不占违法之事
- 不占害人之事
- 不占已知结果之事
- 不可反复占同一事（一事一占）
- 不可心存戏谑

## 基本概念（通俗版）

### 九宫格
想象一个 3×3 的格子（就像井字棋），每格代表一个方位：

```
┌──────┬──────┬──────┐
│ 东南 │ 正南 │ 西南 │
│ 巽4  │ 离9  │ 坤2  │
├──────┼──────┼──────┤
│ 正东 │ 中央 │ 正西 │
│ 震3  │ 中5  │ 兑7  │
├──────┼──────┼──────┤
│ 东北 │ 正北 │ 西北 │
│ 艮8  │ 坎1  │ 乾6  │
└──────┴──────┴──────┘
```

### 四盘（四层信息）

每个格子里有四层信息，就像四层楼：

| 楼层 | 名称 | 代表 | 比喻 |
|------|------|------|------|
| 4层 | **神盘**（八神） | 不可控的运气 | 天意、运势 |
| 3层 | **天盘**（九星） | 天时 | 大环境、趋势 |
| 2层 | **人盘**（八门） | 人事 | 人的选择、行动 |
| 1层 | **地盘**（天干） | 地利 | 基础条件 |

### 天干（10个）
甲、乙、丙、丁、戊、己、庚、辛、壬、癸

- **三奇**：乙（日奇）、丙（月奇）、丁（星奇）→ 代表吉祥
- **六仪**：戊、己、庚、辛、壬、癸 → 甲将军藏在里面

### 九星（9颗星）
天蓬、天芮、天冲、天辅、天禽、天心、天柱、天任、天英

- 🟢 吉星：天心、天任、天辅、天禽、天冲
- 🔴 凶星：天蓬、天芮、天柱
- 🟡 平星：天英

### 八门（8扇门）
开、休、生、伤、杜、景、死、惊

- 🟢 三吉门：**开门**（事业）、**休门**（休养）、**生门**（财运）
- 🟡 中平门：景门（文书）、杜门（隐藏）
- 🔴 三凶门：**伤门**（受伤）、**死门**（死亡）、**惊门**（惊恐）

### 八神（8位神灵）
值符、腾蛇、太阴、六合、白虎、玄武、九地、九天

- 🟢 吉神：值符（贵人）、太阴（暗中帮助）、六合（合作）、九天（光明）
- 🔴 凶神：腾蛇（纠缠）、白虎（凶险）、玄武（欺骗）
- 🟡 中平：九地（稳定）

## 如何起卦？

### 方法一：按时间（最常用）
直接用当前时间排盘。心中想着要问的问题，然后排盘。

```bash
# 当前时间排盘
node scripts/main.js

# 指定时间
node scripts/main.js --time "2026-03-25 14:30"
```

### 方法二：报数
心中想一个数字（1-9），以此数为局数排盘。

### 方法三：特定时刻
当某个重要事件发生的时刻（比如接到电话、收到消息），以该时刻排盘。

**原则**：**念动则占** — 心中产生疑问的那一刻，就是最佳起卦时间。

## 如何看懂盘面？

### 第一步：看基本信息
```
【四柱】丙午年 辛卯月 戊戌日 庚申时
【节气】春分
【局制】阳遁 9 局
```
- 四柱 = 年月日时的干支
- 阳遁/阴遁 = 当前时段的能量方向
- 局数 = 排盘的基准

### 第二步：看值符值使
```
【值符】天心星（落6宫）
【值使】开门（落8宫）
```
- 值符 = 当班的星，代表贵人、领导
- 值使 = 当班的门，代表事情的发展方向

### 第三步：看九宫格
每个宫里有：
- 天盘干 + 地盘干 → 组成"天干组合"（决定吉凶）
- 九星 → 天时好坏
- 八门 → 人事吉凶
- 八神 → 运气加持

### 第四步：看格局
- 🟢 吉格（龙回首、鸟跌穴等）→ 好兆头
- 🔴 凶格（伏吟、反吟等）→ 需小心
- 吉格多 > 凶格 → 事情顺利
- 凶格多 > 吉格 → 事情有阻碍

### 第五步：看用神
根据你问的问题，找到对应的"用神"：
- 问财运 → 看**生门**和**戊**在哪个宫
- 问事业 → 看**开门**和**天心**在哪个宫
- 问婚姻 → 看**六合**在哪个宫

用神所在宫位的好坏 = 事情的结果

### 第六步：看评分和建议
系统会自动给出：
- 📊 综合评分（0-100）
- 🧭 吉方（去哪个方向好）
- 🎨 吉色（穿什么颜色好）
- ⏰ 建议时间

## 常见问题 FAQ

### Q: 同一件事可以反复排盘吗？
**A**: 不可以。"初筮告，再三渎，渎则不告"。第一次的盘最准，反复排只会越来越不准。

### Q: 排盘时间是用北京时间还是真太阳时？
**A**: 奇门遁甲用的是当地真太阳时。但实际操作中，用北京标准时间误差在可接受范围内，除非在新疆等偏远地区。

### Q: 晚上23点以后算第二天吗？
**A**: 子时分早子时（23:00-0:00）和晚子时（0:00-1:00）。传统上23:00开始就算次日了。

### Q: 结果说凶，怎么办？
**A**: 
1. 凶不代表一定坏，可能是"有阻碍但最终能成"
2. 看具体凶格，有些只是提醒注意
3. 趋吉避凶：往吉方走、穿吉色、选吉时
4. 重大事情建议多方面综合判断

### Q: 奇门遁甲和紫微斗数有什么区别？
**A**: 
- **紫微斗数** = 看人的一生命运（固定的）
- **奇门遁甲** = 看某一件事的吉凶（随时变化的）
- 紫微看"命"，奇门看"事"

### Q: 准确率有多高？
**A**: 奇门遁甲的准确率取决于：
1. 排盘是否精确（本系统已保证）
2. 解盘者的水平
3. 起卦时的诚心程度
传统说法是"十占七八"（70-80%），但仅供参考。

## 注意事项

1. **心诚** - 不诚不占，心存戏谑不准
2. **一事一占** - 一次只问一个问题
3. **时机** - 心中产生疑问的那一刻最准
4. **客观** - 盘面只是参考，不要过度依赖
5. **保密** - 占卜结果不宜到处说
6. **正面** - 用于趋吉避凶，不用于害人

## 推荐学习路径

1. **入门**：先理解九宫、八门、九星的基本含义
2. **初级**：学会看天干组合的吉凶
3. **中级**：理解用神取法，能针对具体问题解盘
4. **高级**：掌握格局判断、五行旺衰、综合断事
5. **精通**：活盘活断，融会贯通
