# 奇门遁甲基础

## 一、起源与核心原理

### 1.1 起源

奇门遁甲与太乙神数、六壬神课并称"三式"，是中国古代最高层次的预测术和决策术。传说起源于黄帝战蚩尤时期，由九天玄女传授，经姜太公、张良、诸葛亮等历代高人传承发展。

- **黄帝时期**：传说始创，最初有4320局
- **姜太公**：简化为72局
- **张良**：最终简化为18局（阳遁9局 + 阴遁9局）

### 1.2 核心原理

奇门遁甲以**天、地、人、神**四盘合一，结合**时间**（干支历法）和**空间**（九宫八卦），模拟宇宙时空的运行规律，用以预测事物的发展趋势和选择最佳行动方案。

核心公式：**时间（干支）+ 空间（九宫）+ 天（九星）+ 地（地盘）+ 人（八门）+ 神（八神）= 局**

### 1.3 "奇门遁甲"名称含义

- **奇**：三奇，即乙、丙、丁
- **门**：八门，即休、生、伤、杜、景、死、惊、开
- **遁**：隐遁、藏匿
- **甲**：十天干之首，甲为统帅，在奇门中隐遁于六仪之下

甲为十干之首、为元帅，在奇门中不直接出现在天盘/地盘上，而是隐遁在六仪（戊、己、庚、辛、壬、癸）之下。故称"遁甲"。

## 四、阴阳遁与十八局

### 4.1 阴阳遁划分

- **阳遁**：冬至后到夏至前（阳气上升期），共9局

## 六、定局方法：拆补法与置润法

### 6.1 置润法

**原理**：严格按照节气交接来定局，每个节气固定管15天，分上中下三元各5天。

**算法**：
1. 确定当前时间所属的节气
2. 计算距离该节气交接的天数
3. 第1-5天为上元，第6-10天为中元，第11-15天为下元
4. 查表得到对应局数

**优点**：简单直接
**缺点**：实际节气间隔不总是恰好15天，会导致"超神"（节气未到已用完三元）和"接气"（节气已到但前元未用完）的问题。置润法通过设置"闰奇"来解决。

**闰奇处理**：当芒种或大雪节气有多余天数时，重复使用该节气的下元局数。

### 6.2 拆补法（目前最主流）

**原理**：严格按照符头（甲、己日）来分元，不管节气是否已交。

**算法**：
1. 确定当前时间的日干支
2. 找到最近的**节气交接准确时间**
3. 找到节气后第一个符头日（天干为甲或己的日子）
4. 节气交接时刻到符头日之间的时辰，用**上一个节气的下元局数**（这就是"拆"——把上一节气的尾巴拆过来）
5. 从符头日起，按上中下三元各5天排列（这就是"补"——补足本节气的三元）

**具体步骤**：
1. 查找该时刻所属节气及准确交节时间
2. 判断当前日干支是否为符头日
3. 如果节气已交但还未到符头日→用上一节气下元的局数
4. 如果已过符头日→按正常上中下元计算

**拆补法示例**：
假设春分交节在3月20日辰时，但该日天干为丙，最近的符头日（甲或己）是3月22日甲寅日。
- 3月20日辰时至3月22日之前：仍用雨水下元（阳3局）
- 3月22日起：用春分上元（阳3局）
- 3月27日起：用春分中元（阳9局）
- 4月1日起：用春分下元（阳6局）

### 6.3 两种方法的选择

| 比较项 | 拆补法 | 置润法 |
|---|---|---|
| 主流程度 | **目前最主流** | 古法，使用者较少 |
| 核心逻辑 | 以符头（甲己日）定元 | 以节气交接定元 |
| 闰的处理 | 自然拆补，无需特别处理 | 需要设闰奇 |
| 准确性 | 多数实践者认为更准 | 有争议 |

> **本技能默认使用拆补法。**

## 七、超神接气

### 7.1 概念

- **超神**：符头日在节气交接之前到来，即"神（符头）超过了气（节气）"。这时符头起的三元，实际上跑到了下一个节气的范围里。
- **接气**：节气交接时恰好遇到符头日，或者符头日在节气交接之后，即"气等着神来接"。

### 7.2 在拆补法中的处理

拆补法天然解决了超神接气问题：
- 节气交接后，如果还没到新的符头日，就继续用前一个节气的局数（"拆"前节气的尾巴）
- 到了符头日，才正式开始新节气的三元（"补"上新节气的头）

### 7.3 在置润法中的处理

置润法需要通过"置闰"来处理：
- 当超神积累到一定程度（通常超过9天），就在芒种/大雪后加一个闰局
- 闰局重复使用芒种/大雪的下元局数

## 八、时家奇门的排盘总流程

1. **确定时间**：将公历时间转换为干支历（年月日时四柱）
2. **定阴阳遁**：根据节气判断是阳遁还是阴遁
3. **定局数**：用拆补法（或置润法）确定具体局数（1-9）
4. **排地盘**：根据局数排列三奇六仪在九宫中的位置
5. **排天盘**：根据时干找到值符星（旬首所在宫的星），再根据时干应到的宫位确定天盘
6. **排人盘**：根据时干找到值使门（旬首所在宫的门），按同样规则转动
7. **排神盘**：根据值符星所在宫位，按阳遁顺排/阴遁逆排八神
8. **标注特殊状态**：空亡、马星、暗干等

### 8.1 值符值使的确定

1. 找到时辰干支所属的**旬首**（六甲旬头）
2. 旬首所遁的天干（六仪之一）在**地盘**上所在的宫位
3. 该宫位原有的**九星**即为**值符**（星）
4. 该宫位原有的**八门**即为**值使**（门）

### 8.2 天盘转动

1. 确定值符星
2. 将值符星从原宫位转到**时干**所在的地盘宫位
3. 其余八星随之转动（保持相对位置不变，按顺序排列）

### 8.3 人盘转动

1. 确定值使门
2. 将值使门从原宫位转到**时干**所在的地盘宫位（但需考虑时干在第几宫）
3. 阳遁时辰干支的地支按**顺时针**排列推算值使门移动步数
4. 阴遁时辰干支的地支按**逆时针**排列推算值使门移动步数

> 注：人盘（八门）的转动方式与天盘略有不同。值使门的转动是根据**时干支**相对于**旬首**的宫位差来确定移动步数。具体为：从值使门原位出发，阳遁顺行、阴遁逆行，行进的宫数等于时干在旬中的序数（甲为0不动，乙为1，丙为2...癸为9）。但遇中五宫要跳过。

### 8.4 神盘排布

1. 值符星最终落入的宫位放**值符**（神）
2. 阳遁：按**顺时针**依次排其余七神
3. 阴遁：按**逆时针**依次排其余七神
4. 中五宫跳过

### 9.1 旬空（空亡）

| 旬首 | 空亡地支 |
|---|---|
| 甲子旬 | 戌、亥 |
| 甲戌旬 | 申、酉 |
| 甲申旬 | 午、未 |
| 甲午旬 | 辰、巳 |

| 甲寅旬 | 子、丑 |

### 9.2 马星（驿马）

驿马取法按"申子辰马在寅，寅午戌马在申，巳酉丑马在亥，亥卯未马在巳"。

日支或时支三合局对应驿马：
| 三合局 | 驿马 |
|---|---|
| 申子辰 | 寅 |
| 寅午戌 | 申 |
| 巳酉丑 | 亥 |
| 亥卯未 | 巳 |

### 9.3 时干入墓

| 天干 | 墓库 | 所在宫 |
|---|---|---|
| 甲/乙（木） | 未 | 坤二宫 [待验证：有说法为六宫] |
| 丙/丁（火） | 戌 | 乾六宫 |
| 戊/己（土） | 戌 | 乾六宫 [待验证：有说法为不同] |
| 庚/辛（金） | 丑 | 艮八宫 |
| 壬/癸（水） | 辰 | 巽四宫 |

> 注：天干入墓的宫位取法，不同派别有细微差异，此处以多数派为准。
