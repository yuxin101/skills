# 格局大全

## 一、吉格

### 1. 乙奇得使（日奇得使）
- **条件**：天盘乙 + 开门同宫
- **吉凶**：大吉
- **含义**：日奇配开门，万事通达，利于求见贵人、开业、求官、出行

### 2. 丙奇得使（月奇得使）
- **条件**：天盘丙 + 休门同宫
- **吉凶**：大吉
- **含义**：月奇配休门，光明正大，利于求财、贵人相助、升迁

### 3. 丁奇得使（星奇得使）
- **条件**：天盘丁 + 生门同宫
- **吉凶**：大吉
- **含义**：星奇配生门，聪明智慧生财，利于求财、考试、谋划

### 4. 三奇得使
- **条件**：乙+开门，丙+休门，丁+生门（以上三者统称）
- **吉凶**：大吉
- **含义**：三奇各配其最佳门，万事亨通

### 5. 龙回首（青龙回首）
- **条件**：天盘乙落在地盘甲子戊所在宫，且该宫临吉门（开、休、生）
- **吉凶**：大吉
- **含义**：乙为甲之妹，回到甲身边相护。主贵人返，失物可回，利于团聚

### 6. 鸟跌穴（朱雀投江/凤凰入巢）
- **条件**：天盘丁落在地盘甲子戊所在宫，且该宫临吉门
- **吉凶**：大吉
- **含义**：丁火（文昌星）投到甲的宫位，主文章大成、功名可就

### 7. 虎猖狂（白虎猖狂）→ 此为凶格，移至凶格部分
- [此处为纠正说明]

### 7. 玉女守门（仙人守门）
- **条件**：天盘丁落六宫（乾），且临休门或开门
- **吉凶**：大吉
- **含义**：丁为玉女（星奇），乾为天门，玉女守天门，主贵人暗助、机密事可成

### 8. 天遁
- **条件**：天盘丙 + 地盘戊 + 生门同宫（或天盘丙+天盘生门同宫落于戊地）
- **吉凶**：大吉
- **含义**：天时地利人和俱全，利于升迁、求官、大事谋划

### 9. 地遁
- **条件**：天盘乙 + 地盘己 + 开门同宫（乙己合，开门为吉门）
- **吉凶**：大吉
- **含义**：六合相配开门，利于藏匿、防守、避难、求地产

### 10. 人遁
- **条件**：天盘丁 + 地盘己 + 休门同宫（或太阴同宫）
- **吉凶**：大吉
- **含义**：丁己暗合配休门，利于求人、交际、暗中谋事

### 11. 风遁
- **条件**：天盘乙 + 开门同落四宫（巽宫），巽为风
- **吉凶**：吉
- **含义**：利于经商、出行、传播消息、行事快速

### 12. 云遁
- **条件**：天盘乙 + 开门同落六宫（乾），乾为天，或天盘乙+六合在六宫
- **吉凶**：吉
- **含义**：云行天下，利于出行、远行、化险为夷

### 13. 龙遁
- **条件**：天盘乙 + 休门同落一宫（坎），坎为水，乙为龙
- **吉凶**：吉
- **含义**：龙归大海，利于涉水、航运、求财

### 14. 虎遁
- **条件**：天盘辛 + 开门或生门同落八宫（艮），或天盘辛+九地在艮宫
- **吉凶**：吉
- **含义**：虎归山林，利于防守、营建、镇宅

### 15. 神遁
- **条件**：天盘丙 + 九地 + 生门同宫
- **吉凶**：大吉
- **含义**：神助加持，百事皆宜

### 16. 鬼遁
- **条件**：天盘丁 + 九天 + 休门同宫（或丁+太阴+生门）
- **吉凶**：吉（利于阴事）
- **含义**：鬼神暗助，利于密谋、夜行、暗中行事

### 17. 天三门
- **条件**：开门、休门、生门所落的宫位
- **吉凶**：吉
- **含义**：天上三吉门所在方位，宜向此方向行动。**真天三门**要求三门不受刑克

### 18. 地四户
- **条件**：开门、休门、生门、杜门所落的四个宫位
- **吉凶**：吉
- **含义**：地上四吉户，利于遁走、避难、藏匿。**杜门**在此处属于"利于隐匿"

### 19. 三奇贵人升殿
- **条件**：三奇（乙丙丁）落在各自旺相之地，且临吉门吉星
- **吉凶**：大吉
- **含义**：三奇得地得门得星，主大贵、大吉、事业腾达

### 20. 奇仪顺遂（天显时格）
- **条件**：天盘三奇落宫得地盘生合，且临吉门、吉星、吉神
- **吉凶**：上吉
- **含义**：天时、地利、人和、神助俱全，百事无忧

### 21. 三奇游六仪
- **条件**：天盘三奇加地盘六仪，呈相生关系
- **吉凶**：吉
- **含义**：奇仪相配得宜，事有助力

### 22. 飞鸟跌穴
- **条件**：天盘丙落在地盘甲子戊所在宫，临吉门
- **吉凶**：大吉
- **含义**：丙火光辉照甲，主功名显赫、一飞冲天

### 23. 真诈二门
- **条件**：开门临六宫（乾，原位），或休门临一宫（坎，原位）
- **吉凶**：吉（真门为吉，诈门须辨）
- **含义**：门归原位为"真"，力量最强。开门归乾、休门归坎为真门

### 24. 门合星
- **条件**：吉门（开休生）落在吉星（天心天辅天禽）所在宫
- **吉凶**：吉
- **含义**：人事与天时相合，行动顺利

## 二、凶格

### 1. 伏吟（天盘伏吟）
- **条件**：值符星回到自己原宫（天盘九星不动），即时干落在旬首所在宫位
- **判断方法**：天盘各星均在各自原宫位置不动
- **吉凶**：大凶
- **含义**：主事滞不前、进退两难、忧虑重重。"伏吟"如人伏地哭泣

### 2. 反吟（天盘反吟）
- **条件**：值符星落到自己原宫的对冲宫位（天盘各星均到对冲位）
- **对冲关系**：1↔9、2↔8、3↔7、4↔6（中五无对冲）
- **吉凶**：大凶
- **含义**：主反复无常、事情颠倒、出尔反尔。"反吟"如人反复折腾

### 3. 门伏吟
- **条件**：八门各回原宫（值使门不动）
- **吉凶**：凶
- **含义**：人事停滞、行事不动

### 4. 门反吟
- **条件**：八门各到对冲宫位
- **吉凶**：凶
- **含义**：人事反复、计划变动

### 5. 六仪击刑
- **条件**：六仪所在宫位的地支与该宫位所属地支形成**自刑**或**相刑**关系
- **具体条件**：
  - 甲子(戊)在一宫(坎/子)→子刑卯？不对。正确为：
  - **戊落三宫(震/卯)**：甲子之子刑卯（子卯相刑）
  - **己落六宫(乾/戌)**：甲戌之戌刑未？实为戌刑丑... [以下为正确版本]
  
**标准六仪击刑表**：

| 六仪 | 遁甲 | 地支 | 击刑宫位 | 说明 |
|---|---|---|---|---|
| 戊 | 甲子 | 子 | 三宫(震) | 子卯相刑 |
| 己 | 甲戌 | 戌 | 六宫(乾) | 戌刑未→但乾宫配戌... [待验证] 实际上：戌自刑在六宫（乾宫地支含戌亥） |
| 庚 | 甲申 | 申 | 八宫(艮) | 申刑寅（寅申相刑）→艮宫含寅 [待验证] |
| 辛 | 甲午 | 午 | 九宫(离) | 午自刑→午午自刑，离宫含午 |
| 壬 | 甲辰 | 辰 | 四宫(巽) | 辰自刑→辰辰自刑，巽宫含辰巳 |
| 癸 | 甲寅 | 寅 | 三宫(震) | 寅刑巳→... [待验证] 寅巳相刑 |

> **注意**：六仪击刑的具体对应在不同资料中有出入，需以实际使用的传统定义为准。[部分待验证]

- **吉凶**：凶
- **含义**：主刑伤、官非、灾祸。用事不利，容易受到惩罚或伤害

### 6. 三奇入墓
- **条件**：三奇落入墓库所在宫位
  - **乙入墓**：乙（木）墓在未→落坤二宫（坤宫含未申）
  - **丙入墓**：丙（火）墓在戌→落乾六宫（乾宫含戌亥）
  - **丁入墓**：丁（火）墓在戌→落乾六宫
- **吉凶**：凶
- **含义**：奇被困入墓中，力量全失。主智慧受困、贵人不显、谋事不成

### 7. 门迫（门受迫）
- **条件**：八门落入的宫位五行克该门的五行
- **示例**：
  - 开门(金)落三宫(震/木)→被木泄？不对。门克宫为"门迫"：
  - **正确定义**：门所在宫位的五行克门的五行→"门受制"，即宫克门
  - 休门(水)落二宫(坤/土)→土克水→休门受迫
  - 开门(金)落九宫(离/火)→火克金→开门受迫
  - 生门(土)落三宫(震/木)→木克土→生门受迫
- **吉凶**：凶（吉门受迫减吉，凶门受迫减凶）
- **含义**：人事受到环境压制，行动不自由，计划受阻

### 8. 大格（庚格/太白格）
- **条件**：天盘庚 + 地盘癸（庚加癸）
- **吉凶**：大凶
- **含义**：庚为天狱格中的白虎，癸为地网，上有天狱下有地网。主阻隔、行人不归、事情阻塞

### 9. 小格
- **条件**：天盘庚 + 地盘壬（庚加壬）
- **吉凶**：凶
- **含义**：格而不通，稍逊于大格。主事有阻碍但程度较轻

### 10. 刑格（上格/刑俱格）
- **条件**：天盘庚 + 地盘甲子戊（庚加戊）
- **吉凶**：凶
- **含义**：庚为七煞克甲木，甲遁戊下被庚金所克。主上级施压、官灾、被克制

### 11. 五不遇时
- **条件**：时干克日干（时干五行克日干五行）
- **判断**：
  - 甲乙日遇庚辛时（金克木）
  - 丙丁日遇壬癸时（水克火）
  - 戊己日遇甲乙时（木克土）
  - 庚辛日遇丙丁时（火克金）
  - 壬癸日遇戊己时（土克水）
- **吉凶**：凶
- **含义**：时辰与日主相冲克，主行事不顺、出行不利、谋事难成

### 12. 时干入墓
- **条件**：时干落入自身墓库所在的宫位
  - 甲乙(木)墓在未→二宫(坤)
  - 丙丁(火)墓在戌→六宫(乾)
  - 戊己(土)墓在戌→六宫(乾) [待验证：有说戊墓在戌、己墓在丑]
  - 庚辛(金)墓在丑→八宫(艮)
  - 壬癸(水)墓在辰→四宫(巽)
- **吉凶**：凶
- **含义**：时运入墓，做事不明朗，被困、受阻

### 13. 奇格入墓（三奇入墓的扩展）
- **条件**：三奇六仪中任何天干落入对应墓库宫位
- **吉凶**：凶
- **含义**：力量被困，该天干所代表的事物受到极大限制

### 14. 荧入白（荧入太白）
- **条件**：天盘丙 + 地盘庚（丙加庚）
- **吉凶**：凶（用于战争则利客不利主）
- **含义**：火入金乡，贼来侵犯。主盗贼、竞争激烈、敌方来攻

### 15. 白入荧（太白入荧）
- **条件**：天盘庚 + 地盘丙（庚加丙）
- **吉凶**：凶（用于战争则利主不利客）
- **含义**：金入火地，自投罗网。主敌人自来送死、官司可胜

### 16. 飞宫格（天飞宫格/天网格）
- **条件**：天盘甲子戊加地盘庚（戊加庚），即戊落在庚的地盘宫上
- **吉凶**：凶
- **含义**：甲被庚金七杀所追，主官灾、口舌、牢狱

### 17. 年格（岁格）
- **条件**：天盘庚 + 地盘甲子戊上（同刑格/上格）
- **吉凶**：凶
- **含义**：同"刑格"，但特指年命受克

### 18. 月格
- **条件**：天盘庚 + 地盘丙（庚加丙），同"白入荧"
- **吉凶**：凶
- **含义**：月奇被庚克，主月事不利

### 19. 日格
- **条件**：天盘庚 + 地盘乙（庚加乙）
- **吉凶**：凶
- **含义**：日奇被庚克，主日事不利、阴人受害

### 20. 悖格（天悖格）
- **条件**：天盘丙 + 地盘甲子戊上（丙加戊），六仪击刑时更凶
- **吉凶**：凶
- **含义**：主背悖、事情颠倒、忘恩负义

### 21. 地私门（私门/鬼私门）
- **条件**：杜门 + 地盘六仪击刑，或杜门临凶星凶神
- **吉凶**：凶
- **含义**：主暗中害人、阴私事败露

### 22. 天牢（天狱）
- **条件**：天盘庚落宫旺相，且克制其他宫位
- **吉凶**：大凶
- **含义**：主牢狱之灾、被困被拘

### 23. 地网
- **条件**：时干或日干入墓，且逢空亡
- **吉凶**：凶
- **含义**：天罗地网，动弹不得

### 24. 墓迫
- **条件**：门入墓 + 门迫同时出现（门落入克门的宫位，且该宫恰好是门的墓地）
- **吉凶**：大凶
- **含义**：双重不利，极为困难

### 25. 奇墓格
- **条件**：三奇入墓 + 又逢凶门凶星
- **吉凶**：凶上加凶
- **含义**：贵人全无、智慧受困、雪上加霜

### 26. 天干相冲（天地盘相冲）
- **条件**：天盘天干与地盘天干五行相克
- **吉凶**：不利（但需看具体组合）
- **含义**：上下不和、内外矛盾

## 三、特殊格局

### 1. 值符落空
- **条件**：值符星所在宫位逢空亡
- **吉凶**：大凶
- **含义**：领导不力、靠山全无、大事不可为

### 2. 值使落空
- **条件**：值使门所在宫位逢空亡
- **吉凶**：凶
- **含义**：执行力缺失、事情虚而不实、计划落空

### 3. 空亡宫
- **条件**：某宫位的地支属于当前时辰旬空
- **吉凶**：视情况（空亡宫的事物不实、力量大减）
- **含义**：该宫代表的事物为虚、不可靠、但也可能逢空则不受克

### 4. 马星动
- **条件**：驿马星所在宫位有吉门吉星
- **吉凶**：吉
- **含义**：利于出行、变动、跳槽、搬迁

### 5. 奇门遇合
- **条件**：天地盘天干形成六合关系（甲己合、乙庚合、丙辛合、丁壬合、戊癸合）
- **吉凶**：吉（主和合）
- **含义**：事物和谐、有合作、有助力

### 6. 青龙逃走
- **条件**：天盘乙落到地盘庚上（乙加庚），且逢凶门
- **吉凶**：凶
- **含义**：甲之妹乙被庚金所克，贵人逃走、失去靠山

### 7. 白虎猖狂
- **条件**：天盘辛落到地盘乙上（辛加乙），即地盘乙上天盘辛
- **吉凶**：凶
- **含义**：白虎欺凌青龙，主恶人猖狂、好人受欺

### 8. 螣蛇夭矫
- **条件**：天盘丁 + 螣蛇（神）同宫
- **吉凶**：视情况（丁为奇但螣蛇为凶神）
- **含义**：虚惊、怪梦、但有谋略之象

## 四、格局判定优先级

1. **先看伏吟、反吟**（大局面）
2. **再看五不遇时**（时间层面的凶）
3. **看值符值使落宫状态**（空亡、入墓、击刑）
4. **看天地盘干组合**（吉格凶格）
5. **看门迫、三奇入墓等**
6. **综合判断**：多个格局叠加时，吉凶需综合权衡，非简单相加
