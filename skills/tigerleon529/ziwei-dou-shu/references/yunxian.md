# 大限流年分析

## 运限系统

### 层级关系

```
生年四化（先天） → 大限（10 年） → 流年（1 年） → 流月（1 月） → 流日（1 日） → 流时（1 时辰）
```

---

## 大限（10 年运势）

### 起运年龄

| 五行局 | 起运年龄 |
|--------|----------|
| 水二局 | 2 岁 |
| 木三局 | 3 岁 |
| 金四局 | 4 岁 |
| 土五局 | 5 岁 |
| 火六局 | 6 岁 |

### 大限顺逆

- **阳男阴女**：顺行（命宫→兄弟→夫妻...）
- **阴男阳女**：逆行（命宫→父母→福德...）

### 大限宫位

每个大限走一个宫位，10 年一换。

### 大限四化

每个大限有自己的天干，产生该大限的四化。

**例**：34-43 岁大限，天干为庚 → 庚干四化（阳武府同）

### 大限分析要点

1. **大限命宫** - 该 10 年的重点
2. **大限四化** - 该 10 年的运势走向
3. **叠宫** - 大限宫位与本命宫位重迭
4. **冲照** - 大限四化冲照本命宫位

---

## 流年（1 年运势）

### 流年命宫

- 以生肖定流年命宫位置
- 子年命宫在子，丑年命宫在丑，以此类推

### 流年四化

- 以流年天干定四化
- 甲年：廉破武阳
- 乙年：机梁紫阴
- 丙年：同机昌廉
- 丁年：阴同机巨
- 戊年：贪阴右机
- 己年：武贪梁曲
- 庚年：阳武府同
- 辛年：巨阳曲昌
- 壬年：梁紫左武
- 癸年：破巨阴贪

### 流年分析要点

1. **流年命宫** - 该年的重点
2. **流年四化** - 该年的运势走向
3. **重迭** - 流年宫位与大限/本命重迭
4. **应事** - 流年四化引动本命/大限四化

---

## 流月、流日、流时

### 流月
- 以农历月份定流月命宫
- 正月在寅，二月在卯，以此类推

### 流日
- 以日期定流日命宫
- 初一在命宫，初二在兄弟，以此类推

### 流时
- 以时辰定流时命宫
- 子时在命宫，丑时在兄弟，以此类推

---

## 重迭原则

### 宫位重迭

| 重迭类型 | 含义 |
|----------|------|
| 流年命宫 = 大限命宫 | 该年是大限重点年 |
| 流年命宫 = 本命命宫 | 该年是本命重点年 |
| 流年财帛 = 大限财帛 | 该年财运是大限重点 |

### 四化重迭

| 重迭类型 | 含义 |
|----------|------|
| 流年禄 = 大限禄 | 该年财运/机遇加强 |
| 流年忌 = 大限忌 | 该年阻碍/压力加强 |
| 流年忌冲大限禄 | 该年有破耗 |
| 流年禄会大限忌 | 该年有解 |

---

## 应事判断

### 时间应期

1. **大限应事** - 大限四化引动本命四化
2. **流年应事** - 流年四化引动本命/大限四化
3. **流月应事** - 流月四化引动流年四化

### 应事强弱

| 情况 | 强度 |
|------|------|
| 生年 + 大限 + 流年三重 | 最强 |
| 生年 + 大限 | 强 |
| 生年 + 流年 | 中 |
| 单一大限/流年 | 弱 |

### 应事类型

- **禄** - 财运、机遇、喜事
- **权** - 权力、竞争、事业
- **科** - 名声、学业、贵人
- **忌** - 阻碍、欠债、执着

---

## 运势分析流程

### 1. 定当前大限
- 根据年龄确定当前大限
- 查看大限宫位、天干、四化

### 2. 看大限与本命关系
- 大限宫位是本命何宫
- 大限四化引动本命何宫

### 3. 分析流年
- 流年命宫位置
- 流年四化
- 与大限/本命的重迭

### 4. 判断应事
- 四化重迭情况
- 宫位重迭情况
- 应事时间和类型

---

## 实用技巧

### 1. 抓重点
- 先看来因宫
- 再看生年四化
- 最后看大限流年

### 2. 分主次
- 生年四化是先天，最重要
- 大限四化是 10 年运势，次重要
- 流年四化是 1 年运势，再次

### 3. 看重迭
- 重迭越多，应事越明显
- 三重以上，必有事发生

### 4. 辨吉凶
- 禄权科多为吉
- 忌多需注意
- 但要结合宫位和星情

## 常见运势组合

### 事业运
- 官禄宫 + 财帛宫 + 迁移宫
- 看大限流年四化是否引动

### 财运
- 财帛宫 + 田宅宫 + 福德宫
- 禄存、化禄是否汇聚

### 婚姻运
- 夫妻宫 + 福德宫 + 子女宫
- 桃花星曜是否汇聚

### 健康运
- 疾厄宫 + 命宫 + 福德宫
- 煞星是否汇聚，化忌是否冲照
