# 北派四化飞星理论

## 核心原则

### 解象次序（严格遵循）

1. **生年四化** - 先天能量，最先存在
2. **向心自化** - 先有后无，向心收敛
3. **离心自化** - 先无后有，离心发散
4. **飞宫碰撞** - 宫干飞星产生的交会

> 口诀：先生年，后自化；先向心，后离心

---

## 四化基本含义

| 四化 | 含义 | 特性 | 时间象 |
|------|------|------|--------|
| **禄** | 福气、财富、顺遂、人缘 | 给予、增加、开始 | 起因 |
| **权** | 权力、能力、掌控、竞争 | 强化、争夺、过程 | 发展 |
| **科** | 名声、学业、贵人、温和 | 缓和、化解、修饰 | 结果 |
| **忌** | 执着、欠债、阻碍、专注 | 收敛、结束、因果 | 结果 |

---

## 禄随忌走 vs 忌随禄走

### 禄随忌走
当化禄与化忌同宫或产生交会时，禄气会跟随忌的轨迹流动。

**常见场景**：
- 迁移宫与财帛宫互动
- 禄在 A 宫，忌在 B 宫 → 禄的能量流向 B 宫

**例**：财帛宫化禄，迁移宫化忌 → 为求财外出奔波

### 忌随禄走
化忌跟随化禄的方向，表示对禄的追求和执着。

**例**：事业宫化禄，财帛宫化忌 → 为事业赚钱而执着

## 宫位飞星规则

### 十二宫互串

每个宫位的天干可以飞出四化到任一宫位，形成 12×12×4 = 576 种组合。

**我宫 vs 他宫**：
- **我宫**：命宫、疾厄、福德、财帛、官禄、田宅（六内宫）
- **他宫**：兄弟、夫妻、子女、迁移、交友、父母（六外宫）

**飞星原则**：
- 我宫飞向他宫 → 我付出、我给予
- 他宫飞向我宫 → 他给予、我得到

### 飞宫遇生年/自化

| 情况 | 结果 |
|------|------|
| 飞宫遇生年 | 现象定会发生 |
| 飞宫遇自化 | 往往也会成局 |
| 飞宫无遇 | 力量较弱，需看三方四正 |

## 来因宫（北派核心）

### 什么是来因宫

来因宫是生年天干所在的宫位，是能量最活跃的宫位，决定一生的重点和因果。

### 查找方法

1. 找到生年天干（甲乙丙丁戊己庚辛壬癸）
2. 看该天干落在哪个宫位
3. 该宫位即为来因宫

### 十二来因宫含义

| 来因宫 | 人生重点 |
|--------|----------|
| 命宫 | 自我发展，命运自主 |
| 兄弟 | 手足朋友，合作发展 |
| 夫妻 | 婚姻感情，配偶影响 |
| 子女 | 子女缘分，创意投资 |
| 财帛 | 财富追求，一生为财 |
| 疾厄 | 健康关注，医疗养生 |
| 迁移 | 外出发展，变动机遇 |
| 交友 | 人脉社交，朋友助力 |
| 官禄 | 事业成就，工作核心 |
| 田宅 | 家庭房产，不动产缘 |
| 福德 | 精神追求，晚年运势 |
| 父母 | 家庭背景，长辈上级 |

## 用神选取

### 优先级

1. **来因宫** - 能量最活跃
2. **命宫** - 自我核心
3. **疾厄宫** - 身体健康
4. **福德宫** - 精神福气
5. **官禄宫** - 事业成就
6. **财帛宫** - 财富收入

### 男女有别

- **男命**：优先看阳星（太阳、天机、武曲等）
- **女命**：优先看阴星（太阴、天同、天府等）

### 有生年四化的主星优先

用神应在有生年四化的主星中寻找，能量更强。

## 自化分析

### 向心自化 vs 离心自化

| 类型 | 方向 | 含义 | 时间 |
|------|------|------|------|
| 向心自化 | 向内收敛 | 先有后无，得到后失去 | 前期 |
| 离心自化 | 向外发散 | 先无后有，失去后得到 | 后期 |

### 自化优先级

离心自化 > 向心自化

### 自化在命宫

命宫有自化，优先看自化：
- 离心化忌优先
- 表示自我变化、自我执着

---

## 大限四化

### 计算方法

每个大限有自己的天干，产生该大限的四化。

### 叠加原则

- 大限四化 + 生年四化 = 该大限的重点
- 大限忌冲生年禄 → 该大限有破耗
- 大限禄会生年忌 → 该大限有解

---

## 流年四化

### 层级关系

生年四化（先天） → 大限四化（10 年） → 流年四化（1 年） → 流月四化（1 月）

### 重迭原则

- 流年宫位与大限宫位重迭 → 该宫位能量加强
- 流年四化与大限四化相同 → 该年应事明显

---

## 解盘流程总结

1. **定来因宫** - 找一生重点
2. **看生年四化** - 先天能量分布
3. **查自化** - 向心/离心分析
4. **飞星交会** - 宫干飞星碰撞
5. **大限叠加** - 当前运势分析
6. **流年应事** - 具体年份判断
7. **回答问题** - 针对性解读

---

## 注意事项

1. 不可单宫论断，需看三方四正
2. 星情 + 四化结合，不可偏废
3. 时间象很重要（先后顺序）
4. 男女有别，用神不同
5. 重迭宫位能量加强
