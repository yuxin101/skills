# 北派紫微斗数技能

## 核心功能

1. **排盘计算** - 12 宫位、14 主星、辅星、四化
2. **真太阳时校正** - 根据出生地点自动校正
3. **北派解盘** - 四化飞星、来因宫、运势分析
4. **多输出格式** - 文本命盘、飞书文档、图片（需浏览器技能）

## 使用流程

### 1. 获取用户信息

必需信息：
- 出生年月日时（阳历或农历）
- 出生地点（省市县，用于真太阳时校正）
- 性别

可选信息：
- 想看的具体问题（事业、财运、婚姻、健康等）

### 2. 执行排盘

```bash
# 调用计算脚本
node scripts/calculate.js --birth="1990-05-15 14:30" --location="北京市朝阳区" --gender="男"
```

### 3. 北派解盘步骤

**解象次序**（严格遵循北派理论）：
1. 先解释生年四化
2. 解释向心自化
3. 解释离心自化
4. 飞宫遇生年/自化的碰撞

**核心宫位优先级**：
1. 来因宫（能量最活跃）
2. 命宫
3. 疾厄宫
4. 福德宫
5. 官禄宫
6. 财帛宫

**四化分析**：
- 禄随忌走、忌随禄走
- 宫干飞星到 12 宫
- 大限四化、流年四化叠加

### 4. 输出格式

- **文本命盘** - 飞书消息友好
- **飞书文档** - 详细报告（使用 feishu-create-doc）
- **图片** - 命盘图（使用 stealth-browser 截图）

## 注意事项

1. **真太阳时** - 必须根据出生地点校正时辰
2. **北派特色** - 重四化、轻星情，与南派不同
3. **飞星逻辑** - 宫干飞星是活动性，生年四化是独立性
4. **解盘顺序** - 先生年→自化→飞宫，不可颠倒

## 外部依赖

- **iztro** - `npm install iztro`（紫微斗数排盘库）
- **node-fetch** - 获取地理位置数据
- **timezone** - 时区校正
