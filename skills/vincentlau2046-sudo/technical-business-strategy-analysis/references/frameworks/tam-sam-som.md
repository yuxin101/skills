# TAM/SAM/SOM 市场规模估算指南

## 核心概念

- **TAM (Total Addressable Market)**: 总可寻址市场
- **SAM (Serviceable Addressable Market)**: 可服务可寻址市场  
- **SOM (Serviceable Obtainable Market)**: 可服务可获得市场

## 计算方法

### Bottom-up 方法（推荐）
```
TAM = 总潜在客户数 × 平均客单价
SAM = TAM × 地理/细分市场覆盖率
SOM = SAM × 市场份额预期
```

### Top-down 方法
```
TAM = 宏观市场规模 × 行业渗透率
SAM = TAM × 目标客户占比
SOM = SAM × 竞争力系数
```

## 数据源优先级

1. **一手调研**: 客户访谈、问卷调查
2. **行业报告**: Statista, IBISWorld, Gartner
3. **公开数据**: 上市公司财报、政府统计
4. **网络搜索**: Tavily, Google Trends

## 质量控制

- **数据时效性**: 优先使用最近12个月数据
- **来源验证**: 至少2个独立来源交叉验证
- **置信度评分**: 基于数据质量和一致性
