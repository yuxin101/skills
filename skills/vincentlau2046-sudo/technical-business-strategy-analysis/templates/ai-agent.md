# AI Agent 行业分析模板

## 核心分析维度

### 1. 能力矩阵（量化评分 1-10 分）
- **自然语言理解**：上下文长度、多轮对话能力、意图识别准确率
- **推理能力**：逻辑推理、数学计算、代码生成准确率
- **工具集成**：支持的工具数量、API 调用成功率、工具多样性
- **多模态支持**：图像理解、语音识别、视频分析能力评分
- **部署复杂度**：硬件要求、内存占用、延迟指标（ms）

### 2. 工具集成生态
- **内置工具数量**：具体数值统计
- **第三方工具兼容性**：支持的平台/API 数量
- **自定义工具开发难度**：开发者体验评分（1-5分）

### 3. 多模态支持能力
- **输入模态支持**：文本、图像、音频、视频（是/否 + 质量评分）
- **输出模态支持**：文本、图像生成、语音合成（是/否 + 质量评分）
- **跨模态理解**：图文理解、音视频同步等能力评分

### 4. 部署复杂度评估
- **硬件要求**：最低 GPU 显存（GB）、CPU 核心数、内存（GB）
- **软件依赖**：Python 版本、框架版本、依赖库数量
- **部署时间**：从零到运行的平均时间（分钟）
- **运维复杂度**：监控指标数量、故障恢复时间（分钟）

## 数据源优先级
- **技术指标**：官方基准测试、MLPerf、学术论文
- **用户反馈**：GitHub Issues、Reddit、Twitter
- **商业信息**：Crunchbase、公司官网、融资新闻
- **开源生态**：GitHub Stars、Forks、Contributors
