# AI Infra 行业分析模板

## 核心分析维度

### 1. 芯片性能指标（量化）
- **计算性能**: TFLOPS/GFLOPS (FP16/FP32/INT8)
- **内存带宽**: GB/s
- **能效比**: TOPS/Watt
- **互连带宽**: NVLink/Infiniband 带宽 Gb/s
- **基准测试**: MLPerf 训练/推理分数
- **置信区间**: ±X% (基于多个测试结果)

### 2. 软件生态评估（量化+定性）
- **框架支持度**: TensorFlow/PyTorch/JAX 支持评分 (1-5分)
- **开发者工具**: SDK 完整性评分 (1-5分)
- **社区活跃度**: GitHub stars, commits/月, issues响应时间
- **文档质量**: 完整性、示例丰富度评分 (1-5分)
- **定性分析**: 生态系统战略优势/劣势

### 3. 客户案例分析（量化）
- **头部客户数量**: Fortune 500/独角兽公司采用数量
- **行业渗透率**: 各行业采用百分比
- **客户留存率**: 年度续约率 %
- **平均合同价值**: ACV (美元)
- **部署规模**: 平均集群规模 (GPU数量)

### 4. 市场竞争格局
- **市场份额**: 按收入/出货量计算的百分比
- **价格策略**: 每TFLOPS成本 ($/TFLOPS)
- **技术路线图**: 未来2-3年产品规划评分 (1-5分)
- **供应链安全**: 代工依赖度、地缘政治风险评分

## 数据源优先级
1. **官方数据**: NVIDIA/AMD/Intel 官方规格、财报
2. **基准测试**: MLPerf.org, academic papers
3. **客户信息**: Crunchbase, 公司新闻稿, SEC filings
4. **技术分析**: AnandTech, Tom's Hardware, academic conferences
5. **市场数据**: IDC, Gartner, Statista

## 输出要求
- 所有性能数据必须包含具体数值和测试条件
- 软件生态评分需附置信区间
- 客户案例需提供具体公司名称和使用场景
- 竞争分析需包含量化对比表格
