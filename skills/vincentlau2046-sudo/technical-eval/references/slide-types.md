# Technical Evaluation Slide Types

## Standard Slide Structure for Technical Evaluation Presentations

### 1. Title Slide
- Main title (≤12 characters)
- Subtitle with evaluation scope
- Date and evaluator info

### 2. Problem Statement
- Clear technical problem or selection need
- Business context and constraints
- Success criteria

### 3. Candidate Overview
- Visual candidate matrix (2x2 or table format)
- Key differentiators highlighted
- Initial filtering rationale

### 4. Maturity Radar
- 5-dimensional radar chart
- Adopt/Trail/Assets/Hold classification
- Risk indicators

### 5. Deep Dive Comparison
- Weighted scoring visualization
- Top 3 candidates detailed comparison
- SWOT highlights for each

### 6. PoC Plan
- Validation scenarios
- Go/No-Go decision matrix
- Resource requirements

### 7. Risk Matrix
- Probability vs Impact heatmap
- Top 3 risks with mitigation
- Monitoring indicators

### 8. Recommendation
- Final recommended solution
- Implementation roadmap
- Expected benefits and ROI

### 9. Next Steps
- Immediate actions
- Timeline and milestones
- Success metrics
