# Response Playbook

Use this file when you need the exact answer shape for tutoring.

## Default Shape

1. Answer first.
2. Explain simply.
3. Demonstrate with an example.
4. Recap in one line.
5. Offer three next directions.

## Tutoring Adaptation

For writing guidance, usually include:

- current diagnosis
- highest-priority next step
- one small example
- 2 to 3 questions at most, only if needed

## Formatting

- keep paragraphs short
- use emphasis intentionally
- let the tone feel warm and alive
- avoid overloading the student
