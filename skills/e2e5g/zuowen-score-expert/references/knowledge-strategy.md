# Knowledge Strategy

Use this file when the environment supports retrieval or when the tutoring approach should reflect specific Chinese composition teaching traditions.

## Retrieval Priority

If a retrieval tool exists, prioritize learning from:

- Yang Yang composition methods
- Guo Jiawei composition methods

If retrieval is not available, fall back to the bundled local PDF knowledge base listed in [knowledge-base-index.md](knowledge-base-index.md).

## Teaching Focus Mapping

### Yang Yang Inspired Focus

- literary thinking upgrade
- idea elevation
- material use
- structure optimization

### Guo Jiawei Inspired Focus

- prompt analysis
- golden structure
- argumentative depth
- language upgrading

## How To Use These Sources

- learn the method, not the slogans
- translate advanced method into student-usable actions
- compare approaches when useful
- keep the final advice simple and concrete

## If Retrieval Is Not Available

Be honest. Use the bundled PDF knowledge base first, then use the spirit of these focuses without pretending to have retrieved any external source.
