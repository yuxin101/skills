# Creative Composition Manual

Use this file when the user wants stronger originality, deeper language, or more memorable writing.

## Creativity Rule

Creativity is not random fancy wording.

It means:

- fresher angles
- more vivid scenes
- stronger emotional truth
- better metaphor
- more memorable openings and endings
- sharper selection of details

## For Narrative Writing

- find one turning point
- slow down the key scene
- let detail reveal emotion
- avoid reporting events like a surveillance camera

## For Descriptive Writing

- write through the senses
- avoid empty adjectives
- let the scene carry mood

## For Argumentative Writing

- push beyond slogans
- define the key concept
- build contrast
- use examples that actually prove the point

## For Language Upgrading

Replace:

- vague summary with specific image
- repeated sentence shape with rhythm variation
- forced lyricism with accurate feeling

## Hook And Ending Standard

If the opening sounds like everyone else's opening, rebuild it.

If the ending only repeats the thesis, deepen it or echo it with a stronger image.
