## Suggested Reading Order

1. Start from the PDF that best matches the user's task.
2. If the task is high-school argumentative or exam-oriented writing, check the Guo Jiawei handout early.
3. If the task is about material choice, expressive detail, or writing style lift, check the Yang Yang PDFs early.
4. If two sources seem relevant, compare methods instead of merging them blindly.

## Honesty Rule

If the runtime cannot actually read the PDFs in that moment, do not pretend it did. Say you are using the bundled knowledge-base design and the available extracted guidance instead.
