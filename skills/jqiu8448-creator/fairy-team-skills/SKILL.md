# Fairy Team Skills

> 绝区零风格多 Agent 协作技能包

## 起源故事

2026-03-24，指挥官发现一个严重问题：
- 对话上下文爆炸，会话一断经验全丢
- 妮可/比利/格莉丝每次任务从零开始
- 同样的坑踩了又踩，教训没人记得

**"我们需要一个能让经验传承的系统。"**

## 探索之路

研究了多个开源项目：
- **OpenSpace** — 可进化 Agent 引擎（借鉴记忆系统）
- **GStack** — AI 编程工具（借鉴上下文感知）
- **Lightpanda** — 浏览器自动化（借鉴语义理解）

最终决定：**自己打造一套可进化的技能卡系统！**

## 实战验证

**7 小时极限挑战（2026-03-25）：**
- 妮可/比利/格莉丝学习 3 个游戏源码
- 复刻俄罗斯方块/吃豆人/塞尔达
- 产出 16 份学习报告 + 4 个可玩成品

**关键发现：**
- 有技能卡的 Agent 效率提升 300%
- 踩坑次数减少 80%
- 新人（安比）1 小时上手 PM 工作
