# Badman — Agent Rental

> "The AI that runs your business while you're busy living your life."

Badman is a production-ready autonomous agent built on Base44 and deployable via OpenClaw.
He handles the repetitive, time-consuming tasks that eat your day.

## Quick Install

```bash
openclaw skills install badman-agent-rental
```

## Features

| Feature | Description |
|---|---|
| 📧 Email Manager | Monitors Gmail, drafts replies, sends with CV |
| 📱 Content Gen | Weekly posts for Twitter, TikTok, LinkedIn |
| 🔍 Prospector | Finds and qualifies leads automatically |
| 🤖 Auto-Reply | Responds to messages when you're away |
