# debate-learning-workflow

Evidence-backed, multi-agent debate system for reasoning improvement.

## What changed in v1.0.1
- Added mandatory opening evidence format (Claim/Evidence/Source)
- Added Judge-gated 20-40 round policy
- Added universal lesson quality gate (actionable + measurable + transfer examples)

## Debate loop
1. Topic definition
2. A/B/Opponent3 opening with evidence
3. Round-by-round rebuttal + loophole analysis
4. Judge continuation/stop decision
5. Lesson extraction and universalization
