# Psyke Audit — Slide Structure

Every audit follows this exact slide order. No improvisation on structure.

---

## Fixed Slide Order (12-14 slides)

### Slide 1: Title
- Psyke logo (top-left, text only: "PSYKE")
- Client name (large, H1)
- Subtitle: "SEO & GEO Audit — [Month Year]"
- Client URL
- Glow orb decorations (purple top-right, lime bottom-left)

### Slide 2: Dual Score
- Split layout
- Left: SEO Health score ring (purple stroke) + summary sentence
- Right: GEO Readiness score ring (lime stroke) + summary sentence
- Bottom: Score breakdown grid (Crawlability, Technical, On-Page, Structured Data, Content for SEO; Citability, Visibility, Architecture, Competitive for GEO)

### Slide 3: What's Working
- Split layout
- Left: "Positives" label (green), heading, brief context
- Right: Findings list with evidence bullets

### Slides 4-7: Issues (3-4 slides)
- Each issue gets one slide
- Split layout: left = badge + issue title + explanation, right = evidence + fix
- Order by severity: Critical → High → Medium
- Typical issues: missing H1s, no meta descriptions, schema gaps, broken sitemap, no OG tags, soft 404s
- Maximum 4 issue slides (consolidate medium issues if needed)

### Slide 8: GEO Gap (full-width dark slide)
- Centre-aligned
- Pull quote or headline about the content/AI visibility gap
- Supporting paragraph
- Keyword badges showing missed opportunities

### Slide 9: Competitor Benchmark (if Ahrefs data available)
- Split layout or full-width
- Competitor grid: Client + 2-3 competitors
- Columns: DR, Organic Keywords, Monthly Traffic, AI Overview Citations, ChatGPT Citations
- Key insight callout below the table

### Slide 10: AI Visibility Test (if performed)
- Screenshots or quoted responses from ChatGPT/Perplexity queries
- "We asked [query] — here's who gets cited" format
- Show client presence (or absence)

### Slide 11: Quick Wins
- Split layout
- Left: "Action Plan — This Week" heading
- Right: Numbered priority list (7-8 items)

### Slide 12: Growth Roadmap
- Split layout
- Left: "Next 2-4 Weeks" heading
- Right: Numbered priority list (5-6 items)
- Include content strategy and GEO recommendations here

### Slide 13: How Psyke Helps (CTA slide)
- Dark slide with gradient text
- 3-4 bullet points on Psyke's services mapped to the findings
- "Book a Discovery Call" CTA
- Contact details / Calendly link

### Slide 14: Closing
- Dark slide
- Both scores displayed
- One-sentence summary
- Client URL + "Prepared by Psyke — [Month Year]"

---

## Design Rules

1. **Always dark mode** — every slide uses the dark Psyke palette
2. **Split layouts are the default** — left = context/heading, right = data/evidence
3. **Full-width dark slides** for emphasis moments only (GEO gap, closing)
4. **Maximum 4 issue detail slides** — if more than 4 issues, consolidate the medium ones
5. **Glow orbs** on title and closing slides only — don't overuse
6. **Grid background** optional, use on max 2-3 slides
7. **Score rings** always use purple stroke for SEO, lime stroke for GEO
8. **Badges** use the severity colours consistently: critical=coral, high=orange, medium=lime, good=cyan
9. **No images** unless specifically provided — these are data-driven decks
10. **Psyke branding** on first and last slides only — not on every slide (it's not a PowerPoint template)
