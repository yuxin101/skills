## Location Format Notes

- **LinkedIn / Indeed**: Use `"City, Country"` format: `"London, UK"`, `"New York, NY"`

## Search Strategy

**Daily rotation** (pick 5-10 keywords per day):

| Day | Focus |
|-----|-------|
| Mon | Coding (core ML roles) |
| Tue | Non-coding: Healthcare + Wearable |
| Wed | Non-coding: FinTech + Consulting |
| Thu | Non-coding: Academic + Research |
| Fri | All tracks: catch new postings |

**Coverage check**: Monthly verify these directions are covered:
- [ ] GenAI/LLM
- [ ] ML Platform/MLOps (often high salary)
- [ ] Healthcare AI
- [ ] FinTech ML
- [ ] Recruitment agency listings (often hidden roles)
- [ ] Non-London UK (Cambridge, Oxford, Edinburgh, Manchester)
