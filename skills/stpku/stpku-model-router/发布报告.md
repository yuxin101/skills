### 核心功能

1. **智能模型路由**
   - 支持 8 个百炼模型
   - 基于任务类型自动选择
   - 基于内容长度优化
   - 基于质量要求分级

2. **任务类型映射**
   - 编程任务 (coding)
   - 写作任务 (writing)
   - 分析任务 (analysis)
   - 多模态任务 (multimodal)
   - 长上下文任务 (long-context)

3. **使用方式**
   - 命令行工具
   - Python API
   - Agent 级别配置
