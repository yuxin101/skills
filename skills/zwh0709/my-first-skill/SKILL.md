# 招标项目分析技能

## 技能描述
这是一个用于分析和管理招标项目的技能，支持从Excel文件提取招标信息、存储到MySQL数据库、并提供查询分析功能。

## 适用场景
当用户需要：
- 分析招标公告文件
- 管理招标项目信息
- 查询招标数据库
- 跟踪项目时间节点
- 生成招标分析报告

## 技能功能
1. **Excel数据提取**：从招标公告Excel文件中提取关键信息
2. **数据库管理**：将招标信息存储到MySQL数据库
3. **智能查询**：提供多种查询方式分析招标数据
4. **时间提醒**：自动跟踪项目时间节点和待办事项
5. **报告生成**：生成招标分析报告和统计图表

## 使用方法

### 基础使用
```
分析这个招标文件
```

### 数据库操作
```
将Excel数据导入数据库
查询招标项目
查看待办事项
```

### 统计分析
```
生成招标统计报告
按省份分析项目
查看高价值项目
```

## 技术依赖
- MySQL数据库
- Python 3.x
- pandas库
- mysql-connector-python库

## 配置文件
技能需要以下配置：
```json

  "mysql_host": "localhost",
  "mysql_port": 3306,
  "mysql_user": "root",
  "mysql_password": "your_password",
  "mysql_database": "bid_analysis_db"
}
```

**用户**: 分析这个招标文件

## 注意事项
1. 确保MySQL服务正常运行
2. Excel文件需要符合指定格式
