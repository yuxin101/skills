# NextSteps Backlog
## Format: - [DATE] [STATUS] brief-description (context: where-mentioned)
## Statuses: OPEN, IN-PROGRESS, DONE, DISMISSED
## Max active items: 30 (archive DONE/DISMISSED items when overflow)

## Examples
- [2026-03-20] OPEN Add error handling to payment service (context: mentioned during API review)
- [2026-03-22] IN-PROGRESS Implement rate limiting middleware (context: security discussion)
- [2026-03-24] DONE Set up CI/CD pipeline (context: deployment planning)
