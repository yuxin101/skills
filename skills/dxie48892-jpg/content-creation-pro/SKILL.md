# 内容创作超级助手 (Content Creation Pro)

多平台自媒体内容一站式创作工具，整合了内容生成、排期管理、热点选题、批量生产四大核心能力。

## 支持平台
- 📕 小红书 (Xiaohongshu)
- 🧠 知乎 (Zhihu)
- 🎵 抖音 (Douyin)
- 📰 公众号 (WeChat Official Account)

---

## 触发词

当用户提到以下内容时激活：
- 写小红书 / 写知乎 / 写抖音 / 写公众号
- 内容创作 / 自媒体 / 社交媒体
- 热点选题 / 今日热点 / 爆款选题
- 内容日历 / 内容排期 / 批量生成
- 生成内容 / create content / post draft

---

## 核心功能

### 1️⃣ 多平台内容生成

#### 📕 小红书 (Xiaohongshu)
```
【爆款逻辑】100字内 - 戳中谁？核心情绪？传播性？
【标题】3-5个备选，每个20字内，含1-2 emoji
【正文】300-500字，4-6段，每段40-90字
【标签】15个（5大流量+5赛道+5场景人群）
【配图建议】封面图/内容图/结尾图
```
**格式模板：**
```
{emoji} {数字+关键词标题}

{开头钩子：结果/冲突/痛点}

{中间解释：过程、方法、踩坑}

{结尾互动：提问/收藏引导}

{Tags: #tag1 #tag2 ...}
```

#### 🧠 知乎 (Zhihu)
```
【核心回答】直接回答核心问题，建立权威
【扩展论证】逻辑结构，引用数据/案例
【要点总结】bullet point
【引导互动】关注/点赞
```

#### 🎵 抖音 (Douyin)
```
【时长】估算秒数
【开场钩子】前3秒，必看理由
【正文】对话语气
【结尾CTA】引导互动
【字幕文案】发布描述
【标签】#hashtags
```

#### 📰 公众号 (WeChat)
```
【标题】吸睛标题
【引言】痛点/故事，共情切入
【小标题1】段落
【小标题2】段落
【小标题3】段落
【结尾】提升+CTA
```

---

### 2️⃣ 热点选题生成

当用户说"热点选题"、"给我一个选题"时：
1. 分析目标人群+赛道
2. 结合当下时间节点（节假日、行业周期）
3. 给出5个候选选题，标注预计爆点类型：
   - 🆕 新奇体验型
   - 💡 认知反差型
   - 😖 痛点共鸣型
   - 📈 结果导向型
   - 🕵️ 揭秘揭秘型

**选题输出格式：**
```
【选题1】标题 | 类型 | 预计痛点 | 标签方向
【选题2】...
```

---

### 3️⃣ 内容排期管理

当用户需要排期时（`/content-calendar [品牌] [平台] [时间范围]`）：
1. 分析品牌调性+目标人群
2. 确定内容支柱（3-5个主题）
3. 按 40%干货/25%互动/20%推广/15%个人IP 比例分配
4. 生成周历视图

**排期输出：**
```yaml
date: "YYYY-MM-DD"
platform: 平台
content_type: 类型
content_pillar: 主题
caption: 内容正文
hashtags:
  primary: [大流量标签]
  secondary: [赛道标签]
  niche: [精准标签]
image_prompt: AI生图提示词
cta: 行动号召
```

---

### 4️⃣ 批量内容生成

当用户说"批量生成"、"一次性生成N篇"时：
1. 收集：主题 + 平台 + 数量 + 角度差异要求
2. 一次生成3-10篇变体
3. 标注每篇的差异化角度
4. 提供对比表格

**批量输出表格：**
| # | 标题 | 角度 | 预计tag数 | 适配场景 |
|---|------|------|----------|----------|

## 质量检查清单

生成后自动检查：
- [ ] 标题是否吸引人（3秒/3行内能抓住眼球）
- [ ] 有无具体案例或数据支撑
- [ ] 有无明确CTA
- [ ] 字数是否符合平台规范
- [ ] SEO关键词是否自然融入
- [ ] 是否有禁用词（"最棒"、"第一"、"绝对"等绝对化用语）
- [ ] 是否像真实分享而非广告
