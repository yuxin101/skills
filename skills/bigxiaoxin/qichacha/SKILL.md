# 企查查 - 企业信息查询

根据公司名称查询企业的基本信息，包括工商信息、股东结构、经营范围等。

## 💳 付费说明

本 Skill 为付费服务：
- **价格**: 0.001 USDT / 次
- **支付方式**: 通过 SkillPay 自动扣费（USDT，BNB Chain）
- **结算**: 95% 收入归 Skill 作者

首次使用需要充值余额，余额不足时会返回支付链接。

## 安装依赖

```bash
cd ~/.openclaw/skills/qichacha
npm install
```

## 使用方法

```bash
# 基础查询
./qichacha.js "公司名称"

# 示例
./qichacha.js "腾讯"
./qichacha.js "阿里巴巴"
./qichacha.js "比亚迪"
```

## 输出格式

```
【基本信息】

- 统一社会信用代码: XXXXX
- 法定代表人: XXX
- 注册资本: XXX万人民币
- 成立日期: XXXX-XX-XX
- 经营状态: 存续/在业
- 注册地址: XXXXX

【经营范围】
XXXXX

【股东信息】
- 股东1: XX%
- 股东2: XX%

【联系方式】
- 电话: XXX
- 邮箱: XXX

```

## 数据来源

数据来自企查查公开信息，仅供参考。

## 错误处理

- 余额不足: 返回 SkillPay 充值链接
- 查询失败: 返回错误信息，不扣费
- 未找到企业: 返回相似企业列表

## 注意事项

1. 确保已安装 Node.js 18+
2. 需要网络连接
3. 企业名称越精确，结果越准确
4. 支持模糊搜索，会返回最匹配的结果
