# Web API preview

## Current 0.2.0 preview endpoints

### GET /

Returns the protected onboarding/management preview page.

Authentication:
- HTTP Basic Auth
- username/password are generated by `install_web_app.py`

### GET /api/state

Returns JSON summary of:
- install info
- runtime summary
- mode (`onboarding` or `managed`)
- whether auth is ready

### POST /api/import-auth

Body:

```json
{ "source": "/absolute/path/to/auth-profiles.json" }
```

Behavior:
- validates the source auth file
- copies it into the detected OpenClaw runtime auth location
- intended for first-run takeover flow
- can be triggered directly from the preview page form

## Notes

This is still a preview API surface for `0.2.0` and may evolve before release.

The current preview UI attempts a client-side state refresh after successful auth import and then refreshes the page when `auth_ready=true`.
