# 医疗咨询技能 (Medical Advice Skill)

一个集成权威医学数据库的智能医疗咨询技能，提供症状分析、就医建议、医学文献查询和药品信息查询。

## 功能特性

### 🩺 症状分析与就医建议
- 根据用户描述的症状进行智能分析
- 提供可能的原因判断
- 推荐合适的就诊科室
- 给出家庭护理建议

### 📚 PubMed 医学文献查询
- 集成 NIH PubMed 数据库
- 搜索权威医学文献
- 支持关键词搜索、年份过滤
- 返回文献标题、摘要、链接

### 💊 OpenFDA 药品信息查询
- 集成 FDA 官方药品数据库
- 查询药品适应症、用法用量
- 获取药品警告信息
- 查询不良事件报告

## 使用示例

### 症状咨询
```
用户：最近喉咙一直咳嗽，为什么？
AI：【症状分析】根据描述，可能的原因包括...
```

### 药品查询
```
用户：布洛芬有什么副作用？
AI：【药品信息】根据 OpenFDA 数据库...
```

### 医学文献查询
```
用户：查找关于咳嗽治疗的研究
AI：【权威文献】根据 PubMed...
```

## 技术细节

- **PubMed API**: https://eutils.ncbi.nlm.nih.gov/
- **OpenFDA API**: https://api.fda.gov/
- 无需 API Key，完全免费
- 支持中文界面输出

## 注意事项

⚠️ 本技能提供的信息仅供参考，不能替代专业医疗诊断。如有严重症状，请及时就医。
