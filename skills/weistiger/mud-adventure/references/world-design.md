# 世界设计指南

## 1. 确定题材与基调

| 题材 | 适合风格 | 关键词 |
|------|---------|--------|
| 奇幻冒险 | 史诗、解谜 | 魔法、龙、城堡 |
| 科幻探索 | 悬疑、生存 | 飞船、异星、危机 |
| 武侠江湖 | 情义、复仇 | 门派、秘籍、江湖 |
| 恐怖生存 | 压抑、紧张 | 怪物、逃亡、抉择 |
| 日常温馨 | 轻松、治愈 | 村庄、好友、生活 |

## 2. 世界观设定模板

```
【世界名称】
- 世界观概述：（2-3句话描述核心设定）
- 历史背景：（可选，3-5句关键历史事件）
- 核心冲突：（推动故事发展的主要矛盾）
```

## 3. 故事结构（三幕法）

### 第一幕：序章（玩家进入）
- 起始地点：安全、熟悉
- 触发事件：引导玩家进入主线
- 预期时长：3-5 个房间

### 第二幕：探索（核心体验）
- 主线任务：明确目标
- 支线内容：可选深度
- 挑战递增：战斗难度提升
- 预期时长：8-15 个房间

### 第三幕：高潮与结局
- 终极挑战：BOSS 战或终极谜题
- 多个结局：善恶/成败/隐藏结局
- 预期时长：3-5 个房间

## 4. 房间设计原则

每个房间需要包含：

```
name        : 房间名称（标题）
desc        : 房间描述（2-4句话，营造氛围）
exits       : 出口映射 {方向: 目标房间}
items       : 初始物品列表
npcs        : 初始NPC列表
locked      : 是否上锁（可选）
special     : 特殊事件标记（可选）
```

### 房间密度建议

- 主线房间：每 2-3 个房间有 1 个战斗或谜题
- 支线房间：每 5-6 个房间有 1 个隐藏房间
- NPC 房间：每 4-5 个房间有 1 个可交互 NPC
- 物品房间：每 3-4 个房间有 1 个可拾取物品

## 5. 地图连通设计

### 星型结构（适合线性故事）
```
        [中央大厅]
       /    |    \
    [东翼] [北翼] [西翼]
```
适合：关卡式冒险

### 网状结构（适合开放世界）
```
[A]—[B]—[C]
|   |   |
[D]—[E]—[F]
```
适合：自由探索

### 树形结构（适合线性叙事）
```
[起点]
   |
[章节1]
   |
[章节2]
   |
[结局]
```
适合：剧情驱动游戏

## 6. 氛围营造技巧

- **视觉**：用颜色词（漆黑、金光、血红）
- **听觉**：静谧回响、远处低语、脚步声
- **触觉**：湿冷石墙、粗糙树皮、冰冷金属
- **嗅觉**：腐叶、焚香、血腥
- **情绪暗示**：环境描写同时暗示危险程度
