# 反转剧情模板

## 结构

```
[Setup] 20-30% 时长
  └─ 建立正常叙事
  └─ 让观众产生预期

[Build] 20-30% 时长
  └─ 继续强化预期
  └─ 埋下伏笔（可选）

[Twist] 3-5s
  └─ 意外反转
  └─ 打破预期

[Reveal] 30-40% 时长
  └─ 揭示真相/产品
  └─ 产品演示（demo_insert: true）

[CTA] 2-3s
  └─ 行动号召
```

## 反转类型

### 身份反转
- Setup: "我老婆问我为什么天天加班..."
- Twist: "因为我在偷偷学 AI 帮她做副业"

### 结果反转
- Setup: "朋友说我花3000请设计师太贵了"
- Twist: "其实我只花了3块钱用 AI 生成的"

### 期望反转
- Setup: "我要教你一个月入过万的方法"
- Twist: "就是好好上班（但用 AI 提效）"

### 角色反转
- Setup: "老板说这个方案不行"
- Twist: "然后他发现是 AI 5分钟写的"

## Twist 节奏

- 音乐突然停止/变化
- 画面定格 0.5s
- 特效强调（放大、闪烁）
- 表情特写

## 使用技巧

1. Setup 要足够"正常"，让观众放松警惕
2. Twist 要够意外但合理
3. Reveal 要自然过渡到产品
4. 不要为了反转而反转

## 风险提示

- 反转太牵强会显得尬
- 适合娱乐向/种草向内容
- 严肃产品慎用
