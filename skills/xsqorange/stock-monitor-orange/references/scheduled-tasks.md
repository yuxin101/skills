# 定时任务说明

## 概述

stock-monitor 技能通过 OpenClaw cron 系统执行定时任务，实现自动化的股票监控报告推送。

## Cron 表达式说明

### 格式
```
分 时 日 月 周
│  │  │  │  │
│  │  │  │  └─── 周 (1-7, 周一=1)
│  │  │  └────── 月 (1-12)
│  │  └───────── 日 (1-31)
│  └──────────── 时 (0-23)
└──────────────── 分 (0-59)
```

### 特殊字符
- `*` — 匹配任意值
- `,` — 列表分隔符 (如 `5,35` 表示5分和35分)
- `-` — 范围 (如 `10-11` 表示10点和11点)

### 示例
| 表达式 | 含义 |
|--------|------|
| `0 9 * * 1-5` | 每周一至周五 9:00 |
| `5,35 10-11 * * 1-5` | 每周一至周五 10:05 和 10:35 |
| `10,40 13-15 * * 1-5` | 每周一至周五 13:10, 13:40, 14:10, 14:40 |

## 执行流程

### 标准定时任务执行步骤

```
┌─────────────────────────────────────────────────────────────┐
│  1. 获取大盘指数                                              │
│     python stock_monitor.py index                            │
│     → 上证、深证、创业板、沪深300 等实时数据                    │
└─────────────────────────────────────────────────────────────┘
                            ↓
┌─────────────────────────────────────────────────────────────┐
│  2. 读取股票池配置                                            │
│     读取 ~/.openclaw/stock-pool.json                         │
│     → 获取股票列表（A股/港股分类）                             │
└─────────────────────────────────────────────────────────────┘
                            ↓
┌─────────────────────────────────────────────────────────────┐
│  3. 获取技术面数据                                            │
│     python stock_monitor.py monitor-a   (A股)                │
│     python stock_monitor.py monitor-hk  (港股)               │
│     → 现价、涨跌幅、MACD、KDJ、RSI、BOLL等技术指标             │
└─────────────────────────────────────────────────────────────┘
                            ↓
┌─────────────────────────────────────────────────────────────┐
│  4. 获取最新资讯 (可选)                                       │
│     web_fetch 抓取百度搜索结果                                │
│     关键词：股票名+代码+最新消息+2026                          │
│     → 资讯情感分析、重大公告                                   │
└─────────────────────────────────────────────────────────────┘
                            ↓
┌─────────────────────────────────────────────────────────────┐
│  5. 生成分析报告                                              │
│     结合技术指标 + 资讯分析 → 综合报告                         │
│     【重要】价格数据必须使用步骤3脚本返回的数据                  │
└─────────────────────────────────────────────────────────────┘
                            ↓
┌─────────────────────────────────────────────────────────────┐
│  6. 推送报告                                                  │
│     发送至社交群         │
│     使用 message 工具，mode=announce                          │
└─────────────────────────────────────────────────────────────┘
```

## 配置文件

### 任务配置文件
```
~/.openclaw/cron/jobs.json
```

### 股票监控配置
| 文件 | 路径 | 说明 |
|------|------|------|
| 股票池 | `~/.openclaw/stock-pool.json` | 监控的股票列表 |
| 持仓 | `~/.openclaw/stock-positions.json` | 当前持仓 |
| 交易记录 | `~/.openclaw/stock-trades.json` | 交易历史 |
| 预警配置 | `~/.openclaw/stock-alerts.json` | 预警规则 |

## 执行命令参考

### 手动触发定时任务

定时任务由 OpenClaw cron 系统自动触发，无需手动执行。

如需手动测试，可直接执行任务中的命令：

```bash
# 1. 获取大盘指数
python C:\Users\Administrator\.openclaw\skills\stock-monitor\scripts\stock_monitor.py index

# 2. 获取A股技术面
python C:\Users\Administrator\.openclaw\skills\stock-monitor\scripts\stock_monitor.py monitor-a

# 3. 获取港股技术面
python C:\Users\Administrator\.openclaw\skills\stock-monitor\scripts\stock_monitor.py monitor-hk

# 4. 生成完整报告
python C:\Users\Administrator\.openclaw\skills\stock-monitor\scripts\stock_monitor.py report
```

## 任务执行配置

### 当前配置参数

```json
{
  "sessionTarget": "isolated",
  "wakeMode": "now",
  "agentId": "analysis",
  "delivery": {
    "mode": "announce",
    "channel": "feishu",
    "to": "",
    "bestEffort": false
  }
}
```

| 参数 | 说明 |
|------|------|
| sessionTarget | `isolated` - 在独立会话中执行 |
| wakeMode | `now` - 立即唤醒执行 |
| agentId | `analysis` - 使用 analysis agent 执行 |
| delivery.mode | `announce` - 公告模式推送 |
| delivery.channel | `feishu` - 飞书频道 |
| delivery.to | 群组 ID |

## 报告类型

| 报告类型 | 触发时间 | 内容 |
|----------|----------|------|
| **早报** | 9:00 / 9:20 | 大盘开盘 + 隔夜资讯 + 持仓动态 |
| **盘中监控** | 10:00-14:40 | 实时异动 + 技术信号 |
| **午报** | 12:00 / 12:10 | 上午回顾 + 午后展望 |
| **晚报** | 15:20 / 16:10 | 收盘总结 + 明日展望 |

## 故障排查

### 任务未执行
1. 检查 `openclaw gateway status` 是否正常
2. 检查 cron 任务是否 enabled
3. 查看 `~/.openclaw/logs/` 日志

### 报告推送失败
1. 检查飞书群 ID 是否正确
2. 检查机器人是否在群中
3. 确认 `bestEffort: false` 模式

### 数据获取失败
1. 检查网络连接
2. 验证股票代码是否正确
3. 查看接口是否维护
