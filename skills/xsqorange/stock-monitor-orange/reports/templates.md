# 报告模板

## 报告结构规范

所有报告必须包含以下核心模块：

1. **标题栏** - 报告类型 + 日期时间
2. **大盘指数** - 核心指数实时涨跌
3. **市场情绪** - 涨跌统计 + 资讯情感
4. **持仓分析** - 持仓股涨跌 + 盈亏
5. **技术信号** - 关键指标信号
6. **资讯摘要** - 重要新闻
7. **操作建议** - 基于分析的结论

## 综合评级模板

```markdown
【📋 综合评级】

| 排名 | 股票 | 现价 | 涨跌 | 信号 | 差值 |
|------|------|------|------|------|------|
| 1 | {名称}({代码}) | xx.xx | ±x.xx% | 强烈买入 ✓✓ | +x |
| 2 | {名称}({代码}) | xx.xx | ±x.xx% | 建议买入 ✓ | +x |
| 3 | {名称}({代码}) | xx.xx | ±x.xx% | 观望 | 0 |
| ... | ... | ... | ... | ... | ... |

【信号说明】
• ✓✓ 强烈买入：多个正向指标共振
• ✓ 建议买入：1-2个正向指标
• ✗ 建议卖出：1-2个负向指标
• ✗✗ 强烈卖出：多个负向指标共振
```
