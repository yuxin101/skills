# purchase_record 技能

## 功能描述

这是一个采购记录管理工具，可以自动将您输入的采购信息写入 Excel 表格。

## 使用方法

### 命令格式

```
采购 <日期> <物品名称> <价格>
```

- **日期**: MMDD 格式，例如 0312 表示 3 月 12 日
- **物品名称**: 直接输入物品的完整名称
- **价格**: 输入数字即可（系统会自动处理"元"、"$"等单位）

### 示例

```
采购 0312 螺丝 3 元
采购 0317 小锂 3125 手电钻 290 元
采购 0401 USB 数据线 15.5
```

## Excel 表格结构

- **A 列**: 日期（MM-DD 格式）
- **B 列**: 物品名称
- **C 列**: 价格（纯数字）

数据会自动追加到第一非空白行。
