# `AccountData`

## `AccountData`

账户数据类

| 属性名 | 类型 | 说明 |
|---|---|---|
| `investor` | `str` | 投资者账号 |
| `account` | `str` | 资金账号 |
| `balance` | `float` | 结算准备金 |
| `pre_balance` | `float` | 上次结算准备金 |
| `available` | `float` | 可用资金 |
| `pre_available` | `float` | 上日可用资金 |
| `close_profit` | `float` | 平仓盈亏 |
| `position_profit` | `float` | 持仓盈亏 |
| `dynamic_rights` | `float` | 动态权益 |
| `commission` | `float` | 手续费 |
| `margin` | `float` | 占用保证金 |
| `frozen_margin` | `float` | 冻结保证金 |
| `risk` | `float` | 风险度 |
| `deposit` | `float` | 入金金额 |
| `withdraw` | `float` | 出金金额 |
