# Boundaries

## Hard stop boundaries
- budget exhausted
- safety boundary reached
- human-only decision with no meaningful bypass path
- repeated no-progress cycles
- no clear next step across multiple wake cycles

## Soft block boundaries
- external approval pending
- repository creation blocked
- publication blocked
- one work branch blocked while another local branch remains

## Principle
A soft block should not stop the whole project if the system can still make meaningful local progress.
