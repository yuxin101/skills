# Continuation Integrity

## Core rule
No artifact, no progress.

## False continuation
False continuation means the agent claims it continued the project, but no durable project artifact changed.

## Durable artifacts
Examples:
- `STATE.md`
- `TODO.md`
- `PENDING-DECISIONS.md`
- `HEARTBEAT-LOG.md`
- new review / decision / doc artifacts
- a clearly tracked child task

## Escalation rule
If multiple cycles in a row produce no durable artifacts, treat that as evidence of stagnation, not steady progress.
