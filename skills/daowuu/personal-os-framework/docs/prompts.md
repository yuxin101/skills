# AI Collaboration Prompts

Copy and paste these prompts to start using personal-os with your AI.

---

## Start a New Project

```
I want to track my new project [name]. 
Create the project structure in my personal-os. 
Add it to STATE.md and set up the initial TODO items.
```

---

## Log a Decision

```
I just made a decision: [what you decided].
Log this in DECISIONS.md with:
- The decision
- Why you made this choice
- Alternatives you considered
- Expected consequences
- Follow-up actions needed
```

---

## End of Day Review

```
Run a quick daily review:
1. Check STATE.md for project updates
2. Check TODO.md for completed tasks
3. Update any stale items
4. Log any new decisions made today
5. Suggest priorities for tomorrow
```

---

## Weekly Review

```
Generate a weekly review for me:
1. Read through all DECISIONS.md entries from this week
2. Check STATE.md and TODO.md for overall status
3. Surface any stale projects (not updated in 14+ days)
4. Identify blockers or pending decisions
5. Draft priorities for next week
Write the review to Chronicle/reviews/[date]-weekly.md
```

---

## Capture and Route

```
I captured this note: [your note]
Classify it and route it:
- If it's a task → add to TODO.md
- If it's a decision → log in DECISIONS.md
- If it's reference material → save to Knowledge/
- If it's temporary → save to Inbox/
```

---

## Task Status Update

```
I completed the task: [task name]
Update its status to done in TODO.md.
Check if there are any follow-up actions.
```

---

## Stuck on Something

```
I'm stuck on [what you're stuck on].
Log this as a blocked item in PENDING-DECISIONS.md.
Note what's blocking it and what would unblock it.
Suggest any workarounds.
```

---

## Project Check-in

```
Run a quick project check-in for [project name]:
1. What's the current status?
2. What got done since last update?
3. What's the next action?
4. Any blockers?
Update STATE.md with the findings.
```

---

## Build a Decision Argument

```
I need to make a decision about [topic].
Help me build a structured argument by:
1. Listing all options
2. For each option, noting pros and cons
3. Identifying what information would help decide
4. Recommending a choice based on what matters most to me
Once I decide, log it in DECISIONS.md.
```

---

## Memory Dump

```
I want to do a memory dump. 
I'm going to talk about [topic/project/period] 
and I want you to:
1. Listen and take notes
2. Organize the notes into categories
3. Route each note to the right place (DECISIONS/TODO/Knowledge/Inbox)
4. Identify any decisions that should be logged
```

---

## Monthly Reflection

```
Generate a monthly reflection:
1. Review all weekly reviews from this month
2. Identify patterns in decisions
3. Look at what got done vs what was planned
4. Note any major learnings
5. Set intentions for next month
Write to Chronicle/reviews/[date]-monthly.md
```

---

## Quick Status

```
Give me a quick status report:
1. All active projects and their status
2. Tasks due soon
3. Blocked items waiting for me
4. Decisions pending follow-up
```

---

## How to Use These Prompts

1. Copy the prompt that matches what you need
2. Paste it into your AI conversation
3. Replace the bracketed text with your specific situation
4. The AI will handle the rest

You don't need to remember all the file structures. Just use the right prompt and the AI maintains the system.
