# PPT 设计规范

## 幻灯片尺寸

- 宽：10 英寸
- 高：5.625 英寸（16:9 比例）

## 配色方案

| 变量名 | 用途 | HEX | RGB |
|--------|------|-----|-----|
| primary | 标题栏背景、文字主色 | #1E2761 | (30, 39, 97) |
| secondary | 副标题文字 | #CADCFC | (202, 220, 252) |
| accent | 卡片背景（白色） | #FFFFFF | (255, 255, 255) |
| dark | 封面背景 | #0F1631 | (15, 22, 49) |
| light | 内容页背景 | #F5F7FA | (245, 247, 250) |
| text | 卡片文字 | #1E2761 | (30, 39, 97) |
| muted | 英文名、次要文字 | #6B7280 | (107, 114, 128) |

## Tier 色条颜色

| Tier | 含义 | HEX | RGB |
|------|------|-----|-----|
| 1 | 全球顶级 | #DC2626 | (220, 38, 38) 红 |
| 2 | 行业领先 | #F59E0B | (245, 158, 11) 橙 |
| 3 | 区域知名 | #10B981 | (16, 185, 129) 绿 |
| 4 | 其他客户 | #6B7280 | (107, 114, 128) 灰 |

## 封面页布局

```
┌─────────────────────────────────────┐
│                                     │
│         Licensed Customers          │  ← 44pt Bold 白色，垂直居中

```

```
┌──────────────────────────── 标题栏 (0.8英寸高，primary色) ────┐
│  Tier N | 全球顶级知名公司                                      │
│  Global Top Tier Companies                                     │
├──────────────────────────────────────────────────────────────┤
│  ┌──────────────────┐  ┌──────────────────┐                  │
│  │ █ [LOGO] 公司名  │  │ █ [LOGO] 公司名  │  ← 第1行         │
│  │        英文名    │  │        英文名    │                   │
│  └──────────────────┘  └──────────────────┘                  │
│  ┌──────────────────┐  ┌──────────────────┐                  │
│  │ █ [LOGO] 公司名  │  │ █ [LOGO] 公司名  │  ← 第2行         │
│  └──────────────────┘  └──────────────────┘                  │
│  ┌──────────────────┐  ┌──────────────────┐                  │
│  │ █ [LOGO] 公司名  │  │ █ [LOGO] 公司名  │  ← 第3行         │

```

**卡片尺寸**：4.3 × 1.4 英寸
**卡片排列**：2列，每页最多6个（3行）
**起始坐标**：(0.5, 1.0)
**列间距**：0.4 英寸
**行间距**：0.2 英寸

## 卡片内部布局

```
┌──────────────────────────────────────┐
│█ [    LOGO    ] 公司中文名           │
│                 Company English Name  │
└──────────────────────────────────────┘
```

| 元素 | X偏移 | Y偏移 | 宽 | 高 | 字号 |
|------|-------|-------|----|----|------|
| Tier色条 | +0 | +0 | 0.07 | card_h | - |
| Logo图片 | +0.2 | +0.2 | 1.0 | 1.0 | - |
| 中文名 | +1.35 | +0.25 | 2.8 | 0.50 | 13pt Bold |
| 英文名 | +1.35 | +0.72 | 2.8 | 0.55 | 7pt Normal |

## 公司输入 JSON 格式

```json
[
  {
    "en": "Taiwan Semiconductor Manufacturing Company Limited",
    "cn": "台积电",
    "tier": 1,
    "file": "tsmc.png",
    "official": "https://www.tsmc.com"
  },
  ...
]
```

字段说明：
- `en`：公司英文全名（必须）
- `cn`：公司中文名（必须）
- `tier`：1~4（必须）
- `file`：Logo 文件名（可选，若无则显示占位符）
- `official`：官网 URL（可选，供 Logo 下载备用）
- `query`：百度搜索关键词（可选，若无则自动生成）
