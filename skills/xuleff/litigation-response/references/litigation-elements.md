## 概览

本文档提供常见民事案由的构成要件、法律依据及举证责任分配，用于快速识别案件性质和确定答辩方向。

## 合同纠纷类

### 买卖合同纠纷

**构成要件**：
1. 双方存在买卖合同关系
2. 出卖人交付标的物
3. 买受人支付价款
4. 存在违约行为或争议

**法律依据**：
- 《中华人民共和国民法典》第三编第九章
- 《最高人民法院关于审理买卖合同纠纷案件适用法律问题的解释》

**常见争议焦点**：
- 合同效力（是否成立、是否无效、是否可撤销）
- 标的物质量是否符合约定
- 价款支付争议
- 违约责任认定与违约金调整

**举证责任**：
- 买受人主张质量问题 → 买受人承担举证责任
- 出卖人主张价款未付 → 出卖人承担举证责任

### 借款合同纠纷

**构成要件**：
1. 双方存在借款合同关系
2. 出借人交付借款
3. 借款人应按约定期限返还借款
4. 存在未还款事实

**法律依据**：
- 《中华人民共和国民法典》第三编第十二章
- 《最高人民法院关于审理民间借贷案件适用法律若干问题的规定》

**常见争议焦点**：
- 借款本金数额
- 利息约定（利率标准、利息计算）
- 还款期限与还款事实
- 担保责任

**举证责任**：
- 出借人主张借款关系存在 → 出借人举证借款合同、转账凭证等
- 借款人主张已还款 → 借款人承担举证责任

### 租赁合同纠纷

**构成要件**：
1. 双方存在租赁合同关系
2. 出租人交付租赁物
3. 承租人支付租金
4. 存在违约行为或争议

**法律依据**：
- 《中华人民共和国民法典》第三编第十四章

**常见争议焦点**：
- 租赁物交付状态
- 租金支付争议
- 租赁物维修责任
- 提前解约责任

## 侵权责任纠纷类

### 机动车交通事故责任纠纷

**构成要件**：
1. 存在交通事故
2. 存在损害后果
3. 行为人存在过错
4. 过错与损害之间存在因果关系

**法律依据**：
- 《中华人民共和国民法典》侵权责任编
- 《中华人民共和国道路交通安全法》
- 《最高人民法院关于审理道路交通事故损害赔偿案件适用法律若干问题的解释》

**常见争议焦点**：
- 责任划分
- 损害赔偿项目与数额
- 保险理赔范围

**举证责任**：
- 受害人证明侵权行为、损害后果、因果关系
- 加害人主张免责或减责事由 → 加害人承担举证责任

### 医疗损害责任纠纷

**构成要件**：
1. 医患关系存在
2. 患者受到损害
3. 医疗机构及医务人员存在过错
4. 过错与损害之间存在因果关系

**法律依据**：
- 《中华人民共和国民法典》第一千二百一十八条至第一千二百二十七条
- 《最高人民法院关于审理医疗损害责任纠纷案件适用法律若干问题的解释》

**常见争议焦点**：
- 医疗过错认定
- 因果关系认定
- 损害后果认定
- 责任比例划分

**举证责任**：
- 患者证明医患关系、损害后果
- 医疗机构主张诊疗符合规范 → 医疗机构承担举证责任（过错推定情形）

### 产品责任纠纷

**构成要件**：
1. 产品存在缺陷
2. 存在损害后果
3. 缺陷与损害之间存在因果关系

**法律依据**：
- 《中华人民共和国民法典》第一千二百零二条至第一千二百零七条
- 《中华人民共和国产品质量法》

**常见争议焦点**：
- 产品是否存在缺陷
- 损害是否因产品缺陷导致
- 赔偿责任主体（生产者 vs 销售者）

**举证责任**：
- 受害人证明产品缺陷、损害后果、因果关系
- 生产者主张法定免责事由 → 生产者承担举证责任

## 劳动争议类

### 劳动合同纠纷

**构成要件**：
1. 双方存在劳动关系
2. 存在劳动争议事项
3. 经过劳动仲裁前置程序

**法律依据**：
- 《中华人民共和国劳动合同法》
- 《中华人民共和国劳动争议调解仲裁法》
- 《最高人民法院关于审理劳动争议案件适用法律问题的解释》

**常见争议焦点**：
- 劳动关系是否成立
- 劳动合同解除的合法性
- 工资、加班费、经济补偿金计算
- 年休假、社会保险等待遇

**举证责任**：
- 用人单位作出的开除、除名、辞退、解除劳动合同等决定 → 用人单位承担举证责任
- 工资支付情况 → 用人单位承担举证责任（工资支付凭证属于用人单位掌握管理）
- 加班费 → 劳动者承担举证责任（用人单位掌握考勤记录的除外）

## 婚姻家庭纠纷类

### 离婚纠纷

**构成要件**：
1. 存在合法婚姻关系
2. 夫妻感情确已破裂

**法律依据**：
- 《中华人民共和国民法典》第五编第四章
- 《最高人民法院关于适用<中华人民共和国民法典>婚姻家庭编的解释（一）》

**常见争议焦点**：
- 是否符合离婚条件
- 子女抚养权归属
- 夫妻共同财产分割
- 债务承担

**举证责任**：
- 主张离婚的一方证明夫妻感情确已破裂
- 主张财产为个人财产的一方承担举证责任

## 举证责任分配规则

### 基本原则

1. **谁主张，谁举证**
   - 当事人对自己提出的诉讼请求所依据的事实或反驳对方诉讼请求所依据的事实，应当提供证据加以证明

2. **举证责任倒置**
   - 法律明确规定由对方当事人举证的情形
   - 如：环境污染侵权、建筑物搁置物悬挂物致害、医疗损害责任（过错推定）等

3. **举证责任分配的特殊规则**
   - 举证妨碍：有证据证明一方当事人持有证据无正当理由拒不提供，对方当事人主张该证据的内容不利于证据持有人，可以推定该主张成立
   - 优势证据标准：民事诉讼的证明标准为高度盖然性

### 常见案件的举证责任分配表

| 案件类型 | 原告举证责任 | 被告举证责任 |
|---------|-------------|-------------|
| 买卖合同 | 合同关系、交付货物、对方未付款 | 已付款、货物存在质量问题等抗辩事由 |
| 借款合同 | 借款合同关系、已交付借款 | 已还款、债务已抵销等抗辩事由 |
| 交通事故 | 事故发生、损害后果、因果关系 | 免责事由、减责事由 |
| 劳动争议（解除劳动关系） | - | 解除决定的合法性与事实依据 |
