# 应诉思路梳理

## 任务目标
- 本 Skill 用于：解析原告起诉状，提炼争议焦点，制定应诉策略
- 能力包含：起诉状解析、法律关系识别、争议焦点提炼、应诉策略制定、风险评估
- 触发条件：收到原告起诉状，需要梳理应诉思路

## 操作步骤

### 步骤1：起诉状内容解析

**目标**：全面提取起诉状中的关键信息

**执行要点**：
1. 原告信息提取
   - 原告主体资格（自然人/法人/其他组织）
   - 原告基本信息（姓名、身份证号/统一社会信用代码、住址、联系方式）
   - 代理人信息（如有）

2. 诉讼请求解析
   - 明确各项诉讼请求的具体内容
   - 识别诉讼请求的类型（给付之诉、确认之诉、形成之诉）
   - 标注诉讼请求数额及计算依据
   - 判断诉讼请求是否明确、具体

3. 事实与理由梳理
   - 原告主张的事实经过
   - 原告引用的法律依据
   - 原告的逻辑推理链条

4. 证据清单识别
   - 原告提交的证据名称、来源
   - 证据的证明目的
   - 证据的形式（原件/复印件）

**输出**：起诉状内容结构化摘要

### 步骤2：法律关系识别

**目标**：准确定性案件性质，明确法律适用

**执行要点**：
1. 案由判定
   - 参考 `references/litigation-elements.md` 中常见案由的构成要件
   - 根据原告主张的事实和法律关系确定案由
   - 注意识别是否存在多个法律关系的竞合

2. 法律关系分析
   - 明确法律关系的主体、客体、内容
   - 识别法律关系的成立、变更、消灭的法律事实
   - 确定适用的法律法规（法律、行政法规、司法解释等）

3. 管辖与时效审查
   - 受诉法院是否有管辖权
   - 是否存在管辖权异议的法定事由
   - 诉讼时效是否届满

**输出**：案件性质分析报告

### 步骤3：争议焦点提炼

**目标**：明确双方争议的核心问题

**执行要点**：
1. 事实争议焦点
   - 双方对事实陈述的差异
   - 需要证据证明的关键事实
   - 事实真伪不明时的举证责任分配

2. 法律争议焦点
   - 法律适用的争议（法律条文解释、适用范围）
   - 法律关系性质的争议
   - 法律后果的争议

3. 证据争议焦点
   - 证据真实性、合法性、关联性的争议
   - 证明责任与证明标准的争议
   - 证据效力的争议

**输出**：争议焦点清单（核心焦点 vs 次要焦点）

### 步骤4：应诉策略制定

**目标**：制定系统化的应诉方案

**执行要点**：
1. 程序性事项处理
   - 是否提出管辖权异议
   - 是否申请追加当事人
   - 是否申请鉴定、调查取证
   - 答辩期限与举证期限管理

2. 实体答辩方向
   - 对诉讼请求的认可/部分认可/否认
   - 针对各争议焦点的答辩意见
   - 抗辩权/反诉的考量（抵销权、同时履行抗辩权、诉讼时效抗辩等）

3. 证据准备计划
   - 参考 `references/evidence-guide.md` 中的证据规则
   - 确定需要收集的证据清单
   - 证据的证明目的与证明力评估
   - 举证责任分配与举证策略

4. 调解与和解考量
   - 评估调解/和解的可能性与必要性
   - 确定调解方案的可接受范围

**输出**：应诉策略方案

### 步骤5：风险评估与建议

**目标**：评估诉讼风险，提供决策建议

**执行要点**：
1. 胜诉可能性评估
   - 证据充分性分析
   - 法律适用确定性分析
   - 司法实践倾向分析

2. 成本效益分析
   - 诉讼成本（律师费、诉讼费、鉴定费等）
   - 时间成本与机会成本
   - 执行难度评估

3. 决策建议
   - 是否应诉的建议
   - 调解/和解的时机与策略
   - 其他替代性纠纷解决方式的建议

**输出**：风险评估报告与决策建议

## 输出要求

按照 `assets/response-strategy-template.md` 的模板格式，生成完整的应诉思路文档，包括：

1. 案件基本信息
2. 原告诉讼请求分析
3. 争议焦点提炼
4. 应诉策略方案
5. 证据准备清单
6. 风险评估与建议

## 注意事项

- 起诉状是原告单方主张，需结合证据客观分析，不可直接采信
- 争议焦点的提炼应紧扣诉讼请求，避免偏离主题
- 应诉策略应具有可操作性，明确责任人和时间节点
- 证据准备应遵循举证期限，注意证据的合法性
- 风险评估应全面、客观，避免盲目乐观或过度悲观
- 注意保守案件秘密，遵守律师职业道德
