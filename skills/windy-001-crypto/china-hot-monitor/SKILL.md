### 1. 全网热点总览

### 2. 选题分析

根据多平台热点，输出内容选题建议：

```
🎯 今日内容选题

高热度话题（适合追热点）：
- 特斯拉降价 → "特斯拉又降价了！等等党永不吃亏？"
- 国足世预赛 → "中国足球还有希望吗？"
- OpenAI新模型 → "AI替代人类工作的最新进展"

持续热门（适合深度内容）：
- 新能源车购置税 → "2026新能源补贴政策解读"
- 养老金账户 → "个人养老金怎么开户最划算"

蓝海话题（竞争少）：
- 考研成绩 → "考研出分后该做什么"
```

### 3. 趋势对比

追踪同一话题在不同平台的热度：

```
📈 话题热度对比：「淄博烧烤」

抖音：🔺 热度 1000万+ (上升中)
微博：🔺 热度 5.2亿 (持续发酵)
小红书：🔺 相关笔记 50万+篇
百度：🔺 搜索指数 920万

趋势判断：🔥 爆发期，建议48小时内追热点
建议角度：美食探店/旅游攻略/情绪共鸣
```

### 4. 热点预警

监控特定话题热度变化：

```
🚨 热点预警

【突发】某明星塌房
- 微博: 🔺🔺🔺 5分钟内冲上热搜第1
- 抖音: 🔺🔺 相关视频爆发中
- 小红书: 🔺🔺 笔记开始出现

【预测】清明节临近
- 预计1-2天内：旅游/祭扫/出行相关内容将爆发
- 建议提前准备内容

【衰退】春节档电影
- 各平台热度开始下滑
- 不建议再追
```

## 使用场景

### 场景1：每日选题

```
触发：用户说"今天发什么内容好"
操作：
1. 获取7平台实时热搜
2. 筛选与账号定位相关话题
3. 输出3-5个可操作选题
4. 提供爆款标题模板
```

### 场景2：追热点

```
触发：突发热点事件
操作：
1. 判断热度级别（🔥🔥🔥/🔥🔥/🔥）
2. 评估与账号相关性
3. 快速产出内容框架
4. 建议发布时间（越快越好）
```

### 场景3：竞品监控

```
触发：监控竞品账号动态
操作：
1. 追踪行业热点
2. 对比同类内容数据
3. 分析爆款规律
4. 输出差异化建议
```

## 内容创作模板

### 热点借势模板

```
标题公式：
[热点词] + [我的角度] + [情绪词]

示例：
"特斯拉降价了！等等党这次赢麻了吗？"
"淄博烧烤重现！我去吃了3家总结这篇"
"OpenAI又放大招，普通人怎么抓住机会"

正文结构：
1. 热点引入（1-2段）
2. 我的观点/分析（3-4段）
3. 实用干货（2-3段）
4. 互动引导（结尾）
```

### 情绪共鸣模板

```
开头：描述一个普遍场景
"你是不是也有这样的经历..."

中间：讲故事/举例子
"我一个朋友..."

高潮：反转/洞察
"但其实..."

结尾：总结金句
"所以我说..."
```
