## 展厅接待SOP

### 客户进门第一步

```
客户进门 →
  微笑点头示意 (3秒内) →
  "您好，欢迎光临！请问您是第一次来吗？" →
  [如是] → 引导参观展厅
  [否] → "很高兴您再次光临，今天想看哪款车型？"
```

### 需求挖掘话术

```
探测需求问题：
1. "您今天是想了解哪方面的车型呢？"
2. "您选车最看重哪些方面？安全性？续航？还是智能科技？"
3. "您平时主要是自己开还是家用为主？"
4. "有没有考虑过换新能源车？"

倾听要点：
- 价格预算
- 用车场景
- 品牌偏好
- 决策时间
```
