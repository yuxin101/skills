## 微头条创作模板

### 爆款微头条结构

```
【开头痛点/冲突】（引发好奇）
例如："我发现一个扎心的真相：大多数人努力一辈子，也没能赚到第一个100万。"

【中间故事/数据】（建立共鸣）
例如："我研究了100个普通人的赚钱路径，发现他们都在用这3种方法..."

【结尾引导互动】（激发评论）
例如："你怎么看？你用的是哪种方法？评论区告诉我！"
```

### 领域爆款关键词

| 领域 | 关键词 |
|------|--------|
| 财经 | 赚钱、存钱、负债、被动收入、复利 |
| 职场 | 加薪、辞职、领导、升职、打工 |
| 情感 | 婚姻、恋爱、离婚、婆媳、出轨 |
| 育儿 | 成绩、方法、习惯、家长、教育 |

## 小红书笔记模板

### 爆款封面标题公式

```
公式1：[数字] + [效果] + [方法]
例如：3个动作，让你狠狠提升表达力！

公式2：[痛点] + [解决方案]
例如：不会聊天？教你3招秒变社交达人

公式3：[对比] + [结果]
例如：30天前的我 vs 现在的我，差距在哪里？
```

### 正文结构

```
1. 开头：痛点共鸣（30字内）
   "你是不是也有这样的困扰..."

2. 中间：干货内容（分段清晰）
   - 要点1：[具体方法]
   - 要点2：[具体方法]
   - 要点3：[具体方法]

3. 结尾：行动号召
   "收藏起来慢慢看~"
   "评论区告诉我你的想法"
   "关注我，每天分享干货"
```

## 抖音文案模板

### 黄金3秒开场

```
类型1：痛点开场
"你知道为什么你赚不到钱吗？"

类型2：惊人数据
"90%的人都不知道，睡觉能减肥！"

类型3：身份反差
"作为一个从业10年的程序员，我为什么劝你别卷了？"

类型4：直接承诺
"这个方法，坚持7天，你也能写出爆款文案！"
```

### 脚本结构

```
[开场 3秒] - 抓住注意力
[痛点 5秒] - 建立共鸣
[方案 20秒] - 给出干货
[钩子 7秒] - 引导互动
[结尾 5秒] - 行动号召
```

## 网文开头模板

### 男频爆款开头

```
第一章：觉醒/转折
主角原本处于困境（没钱/被欺负/废物）
获得金手指（系统/传承/重生记忆）
第一次小成功，树立信心

模板：
"三十岁生日那天，张伟收到了公司的裁员通知。
走出公司大门，他收到了那条改变命运的消息..."
```

### 女频爆款开头

```
第一章：穿越/重生
主角发现自己穿越到某个节点
有前世记忆，知道未来走向
用信息差获得先机

模板：
"林晚晚睁开眼，发现自己回到了十年前。
前世，她在这一年嫁给了那个男人，从此走进了地狱..."
```

### 开头公式

```
1. 强设定：穿越/系统/重生立刻交代
2. 强冲突：立刻有一个困境或敌人
3. 强钩子：立刻有一个悬念或爽点
4. 目标明确：主角目标立刻清晰
```

## 快速创作工作流

```
1. 确定今日选题（根据热点/领域）
2. 选择内容形式（微头条/小红书/文案）
3. 调用对应模板生成初稿
4. 人工优化（加入个人特色/真实案例）
5. 添加标题/封面元素
6. 发布并观察数据
7. 根据数据反馈优化
```

## 爆款内容规律

| 规律 | 说明 |
|------|------|
| 情绪共鸣 | 引发强烈情绪（愤怒/焦虑/惊喜） |
| 实用价值 | 提供可直接使用的信息 |
| 身份认同 | 让读者觉得"说的就是我" |
| 认知反差 | 打破常规认知 |
| 互动引导 | 引发评论和分享 |

## 禁忌事项

- 避免绝对化表述（"一定"、"必须"、"绝对"）
- 避免虚假信息（数据要有来源）
- 避免过度营销（先给价值再引导）
- 避免敏感话题（政策/黄赌毒/政治）
