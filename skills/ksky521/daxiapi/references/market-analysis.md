本文档详细说明各项指标的分析方法和使用框架。

## 一、CS指标与市场热力图分析

### CS指标定义

CS（Close-Short MA）是短期动量指标，衡量股价与20日均线的乖离程度。

**公式：**
```
CS = (ln(Close) - ln(EMA(ln(Close), 20))) × 100
```

### CS阈值判断

| CS值范围 | 状态 | 操作建议 |
|----------|------|----------|
| > 15 | 偏离度过高 | 不追高 |
| 5 ~ 15 | 偏离度适中 | 适合入场 |
| 0 ~ 5 | 均线附近 | 正常波动 |
| -5 ~ 0 | 刚跌穿均线 | 建议坚守 |
| < -5 | 大幅跌穿均线 | 建议离场 |

**核心原则：**
- 入场时：不追在偏离度过高（>15），追在偏离度适中（5-15）
- 止损时：均线上方（CS>0）无需担忧
- 跌穿均线：偏离度-5~0建议坚守，<-5建议离场

### 市场宽度分析四维度

使用 `get_sector_data` 获取板块热力图后，按以下维度分析：

#### 1. 板块热度延续与扩散

**分析要点：**
- 对比今日与昨日领涨板块
- 判断热度延续性：昨日领涨板块是否持续走强
- 判断热度扩散：上涨板块数量是否增多
- 分析扩散路径：是否扩散到关联板块
- 评估速度：上涨速度是否加速

#### 2. 找领涨行业

**分析要点：**
- 依据热力图动量指标变化确定领涨行业
- 分析持续性：是否持续走强或走弱
- 判断反转可能：是否存在反转信号
- 分析关联性：领涨行业板块之间的关联性

#### 3. 找上升行业

**分析要点：**
- 找出连续多日动量持续上升的行业
- 评估持续性：动量值是否持续上升
- 分析变化速率：动量值的变化速率
- 排名上升速度：推测未来领涨潜力
- 分析关联性：上升行业间的关联性

#### 4. 找反转行业

**分析要点：**
- 关注热力图底部区域行业
- 判断反转可能：是否走出尾部区域
- 观察向上动能：是否展现向上动能
- 识别交易机会：有反转交易机会的行业
- 分析关联性：反转行业间的关联性

## 二、大小盘风格轮动分析

### 指标定义

**大小盘波动差值** = 中证2000涨幅 - 沪深300涨幅

- **沪深300**：大盘股代表
- **中证2000**：小盘股标杆

### 差值解读

| 差值范围 | 含义 | 判断 |
|----------|------|------|
| > 0 | 中证2000强于沪深300 | 小盘股占优 |
| < 0 | 中证2000弱于沪深300 | 大盘股占优 |
| ≥ 10% | 小盘股大幅领先 | 强烈看空小盘股，注意均值回归 |
| ≤ -10% | 大盘股大幅领先 | 强烈看空大盘股，注意均值回归 |

### 均值回归信号

- 差值达到 **±10%** 附近时，可能出现风格切换
- 例：差值达到 -10%，代表小盘股弱于大盘股 10%，可能出现反转，小盘股将走强

### 运动方向判断

| 差值方向 | 含义 |
|----------|------|
| 向下运动 | 大盘股在未来表现更好 |
| 向上运动 | 小盘股在未来表现更好 |

### 机构vs游资活跃度

| 风格占优 | 市场特征 |
|----------|----------|
| 大盘股占优 | 机构更加积极 |
| 小盘股占优 | 游资更加积极 |

## 三、市场温度指标

### 3.1 估值温度

**定义：** 市场整体市盈率的历史百分位（0-100）

| 温度值 | 估值状态 | 投资建议 |
|--------|----------|----------|
| < 20 | 低估 | 入场好时机，可考虑定投 |
| 20-70 | 中性 | 正常投资 |
| > 70 | 高估 | 市场可能存在泡沫，需谨慎 |

**重要提示：** 估值温度是**长期指标**，对短期操作指导意义有限。低估时考虑定投，高估时考虑分批卖出。

**API调用：** `get_market_value_data` 获取各指数估值温度

---

### 3.2 恐贪指数

**定义：** 衡量市场短期贪婪和恐惧程度的指标（0-100）

| 指数范围 | 状态 | 市场特征 | 投资建议 |
|----------|------|----------|----------|
| 0-10 | 极度恐惧 | 情绪极度悲观，潜在阶段性底部 | 分批买入或定投 |
| 10-30 | 恐惧 | 市场情绪低落 | 观察 or 小额试探性买入 |
| 30-70 | 中性 | 情绪平稳 | 正常投资，保持策略 |
| 70-90 | 贪婪 | 情绪乐观，警惕过热 | 谨慎投资，控制仓位 |
| 90-100 | 极度贪婪 | 情绪亢奋，潜在阶段性顶部 | 分批卖出或止盈 |

**API调用：** `get_market_temp` 获取恐贪指数

---

### 3.3 趋势温度

**定义：** 市场中股价高于60日均线的占比（0-100）

| 数值范围 | 状态 | 市场特征 | 投资建议 |
|----------|------|----------|----------|
| 0-20 | 极度低迷区 | 少数股票站上MA60，空头主导 | 风险大于机会 |
| 20-40 | 弱势震荡区 | 部分股票尝试突破，多空过渡初期 | 结构性机会，警惕反复 |
| 40-60 | 中性平衡区 | 多空均衡，震荡或转换期 | 高抛低吸，避免追涨杀跌 |
| 60-80 | 强势觉醒区 | 多数股票中期上升，多头主导 | 顺势参与主流题材 |
| 80-100 | 过热泡沫区 | 几乎全部突破MA60，情绪亢奋 | 中期趋势过度延伸，风险积累 |

**API调用：** `get_market_data` 返回的 `above_ma200_ratio` 类似概念

---

### 3.4 动量温度

**定义：** 市场股票动量值中位数的历史百分位（0-100）

| 动量范围 | 状态 | 市场特征 | 投资建议 |
|----------|------|----------|----------|
| < 20 | 动量太弱 | 股票超跌 | 关注超跌反弹机会 |
| 20-40 | 偏弱 | 动量不足 | 谨慎观望 |
| 40-60 | 正常 | 正常范围 | 正常交易 |
| 60-80 | 偏强 | 动量较强 | 可适当参与 |
| > 80 | 动量强势 | 可能离市场高潮不远 | 注意风险，考虑减仓 |

## 四、综合分析方法

### 市场整体分析流程

```
1. get_market_data      → 市场宽度、主要指数涨跌
2. get_market_degree    → 市场温度冷热判断
3. get_market_style     → 大小盘风格轮动
4. get_market_temp      → 估值/恐贪/趋势温度
5. get_market_value_data → 指数估值水平
```

### 板块分析流程

```
1. get_sector_data      → 板块热力图，CS动量排序
2. 分析四维度：延续/领涨/上升/反转
3. get_sector_rank_stock → 行业内龙头股筛选
```

### 个股分析流程

```
1. query_stock_data     → 搜索股票代码
2. get_stock_data       → 获取个股CS/SM/ML/RPS等指标
3. 结合形态标签：VCP/SOS/LPS/Spring/信号K
```

## 五、投资建议规范

### 回复原则

1. **模糊的正确**：推荐行业要精准但不给具体仓位
2. **不推荐个股**：只分析数据和趋势
3. **逻辑严密**：不让用户找到逻辑漏洞
4. **聚焦A股**：只围绕A股市场话题

### 分析框架

- 结合差值的具体数值、变化趋势等量化指标
- 依据差值偏离均值的程度判断风格切换可能性
- 根据差值变动速率评估短期和中期走向
- 多指标交叉验证，避免单一指标决策

### 语言风格

- 客观理性、通俗易懂
- 不得包含脏话和低俗表述
- 符合专业投资者形象

## 六、个股技术面分析

### 核心动量指标

| 指标 | 类型 | 说明 | 关键阈值 |
|------|------|------|----------|
| **CS** | 短期动量（主要） | 收盘价与EMA20乖离率，**默认动量指标** | 个股>30过热，行业>20过热 |
| **SM** | 中期动量 | EMA20与EMA60乖离率，判断中期趋势 | >0多头排列 |
| **ML** | 长期动量 | EMA60与EMA120乖离率，判断长期趋势 | >0长期向上 |
| **RPS** | 相对强度 | 欧奈尔市场排名百分比 | >80为强势股 |
| **SCTR** | 技术排名 | 短中长期加权排名，**比RPS更敏感，A股更有效** | 60=强于60%股票 |

### 涨跌幅指标

| 指标 | 说明 |
|------|------|
| **zdf** | 当日涨跌幅 |
| **zdf5 / zdf_5d** | 5日累计涨幅 |
| **zdf10 / zdf_10d** | 10日累计涨幅 |
| **zdf20 / zdf_20d** | 20日累计涨幅 |
| **zdf30 / zdf_30d** | 30日累计涨幅 |

### 成交量指标

| 指标 | 说明 | 判断方法 |
|------|------|----------|
| **VCS** | 成交量与EMA5距离 | >0放量，值越大放量越大 |
| **VMA5** | 成交量5日均线 | VOL>VMA5代表放量 |
| **VOL** | 当日成交量 | - |
| **VOL1** | 昨日成交量 | 对比判断放量/缩量 |

### 技术形态标签

调用 `get_stock_data` 返回的形态标签：

| 形态 | 字段 | 说明 |
|------|------|------|
| VCP | `isVCP` | 波动收缩模式，供应吸收后可能上涨 |
| SOS | `isSOS` | Wyckoff强势走势行为 |
| LPS | `isLPS` | 最后支撑点，突破后可买入 |
| Spring | `isSpring` | 弹簧形态，假破位后快速回升 |
| 箱体突破 | `isCrossoverBox` | 突破整理区间 |
| IB | `isIB` | Inside Bar内包线 |
| NR4/NR7 | `isNR4/isNR7` | 振幅收窄4/7天 |

### 技术指标补充

| 指标 | 用途 |
|------|------|
| **鳄鱼线** | 趋势判断，乖离率cg/cr/cb |
| **RSI** | 超买超卖判断 |
| **量比** | 成交量异动 |

### CS与RSI组合判断

| RSI状态 | CS状态 | 信号含义 |
|---------|--------|----------|
| 超买 | CS过高 | 回调风险，谨慎 |
| 超卖 | CS较低（<-10） | 潜在买入机会 |

### 技术面分析步骤

**步骤1：获取数据**
```
调用 get_stock_data 获取 momInfo（动量数据）
```

**步骤2：动量分析（优先CS）**
- 分析CS短期动量强弱
- 判断是否过热（个股>30，行业>20）
- 结合SM/ML判断中期长期趋势

**步骤3：市场排名分析**
- 使用SCTR指标分析全市场排名
- SCTR比RPS更敏感，A股推荐优先使用

**步骤4：涨跌幅与成交量**
- 查看累计涨跌幅（zdf5/10/20/30）
- 判断放量或缩量（VCS/VMA5）
- 放量程度与缩量大小

**步骤5：趋势与位置**
- 分析趋势方向
- 识别短期压力和支撑
- 判断Price Action阶段：突破/趋势/交易区间

**步骤6：技术形态确认**
- 检查形态标签：VCP/SOS/LPS/Spring等
- 结合鳄鱼线、RSI等辅助指标
- 综合给出技术面投资建议

## 七、个股基本面分析

### 数据来源

调用 `get_stock_data` 获取 `finInfo`（财务数据）

### 分析维度

#### 1. 估值指标

| 指标 | 说明 |
|------|------|
| **PE / pe_ttm** | 市盈率TTM |
| **PB** | 市净率 |
| **ROE** | 净资产收益率 |
| **股息率** | 分红收益率 |

#### 2. 财务分析

- 分析财报数据质量
- 评估财务健康程度
- 判断市场潜力

#### 3. 市值与流动性

| 指标 | 说明 |
|------|------|
| **shizhi_lt** | 流通市值 |
| **vol** | 成交量（流动性） |

### 基本面分析要点

1. **估值合理性**：PE/PB是否处于合理区间
2. **盈利能力**：ROE是否稳定且较高
3. **财务健康**：负债率、现金流等
4. **成长性**：营收和利润增长趋势

## 八、综合评估报告输出

### 报告结构

```
一、技术面分析
   1. 动量指标分析（CS/SM/ML）
   2. 市场排名（SCTR/RPS）
   3. 涨跌幅与成交量
   4. 趋势与位置判断
   5. 技术形态确认
   6. 技术面结论

二、基本面分析
   1. 估值分析（PE/PB/ROE）
   2. 财务状况评估
   3. 基本面结论

三、综合投资建议
   - 风险提示
   - 操作建议
```

### 分析原则

1. **财务预警**：如财务状况有明显问题，给出明确提示
2. **合规底线**：不回答违法或超出认知的问题
3. **模糊正确**：不给具体仓位，推荐行业要精准
4. **风险披露**：提示数据延迟，不构成投资建议
