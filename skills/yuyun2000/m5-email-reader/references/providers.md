# 常见邮箱 POP3 服务器配置参考

## 163 邮箱
- POP3_SERVER: pop.163.com
- POP3_PORT: 995
- 密码类型: 授权码（非登录密码）
- 开启路径: 设置 → POP3/SMTP/IMAP → 开启 POP3 → 获取授权码

## 126 邮箱
- POP3_SERVER: pop.126.com
- POP3_PORT: 995
- 密码类型: 授权码

## QQ 邮箱
- POP3_SERVER: pop.qq.com
- POP3_PORT: 995
- 密码类型: 授权码（非QQ密码）
- 开启路径: 设置 → 账户 → POP3/SMTP 服务 → 开启 → 获取授权码

## Gmail
- POP3_SERVER: pop.gmail.com
- POP3_PORT: 995
- 密码类型: 应用专用密码（需开启两步验证后生成）
- 开启路径: Gmail 设置 → 转发和 POP/IMAP → 启用 POP

## Outlook / Hotmail
- POP3_SERVER: outlook.office365.com
- POP3_PORT: 995
- 密码类型: 账户密码（或应用密码，视账户安全设置）

## 企业微信邮箱 / 腾讯企业邮
- POP3_SERVER: pop.exmail.qq.com
- POP3_PORT: 995
- 密码类型: 邮箱密码

## 阿里云企业邮箱
- POP3_SERVER: pop3.mxhichina.com
- POP3_PORT: 995
- 密码类型: 邮箱密码
