# 血糖管理助手

一个综合性的血糖管理技能，帮助用户记录、分析和预测血糖数据，并提供个性化的健康建议。

## 功能特点

- 📝 **血糖记录** - 支持手动录入和文件导入（CSV/Excel）
- 📊 **趋势分析** - 生成日/周/月统计报告和可视化图表
- 🔮 **血糖预测** - 基于历史数据预测未来血糖走势
- 💡 **健康建议** - 个性化的饮食、运动和就医建议
- 🔒 **隐私保护** - 所有数据本地存储，不上传云端

## 使用方法

### 记录血糖
```
用户: 我血糖7.2
用户: 今天早餐前血糖6.9
用户: 刚吃完午饭两小时，血糖9.5
```

### 查看报告
```
用户: 看看这周的血糖报告
用户: 最近血糖趋势怎么样
用户: 这周和上周比怎么样
```

### 获取建议
```
用户: 晚餐吃什么比较好
用户: 什么时候运动比较好
用户: 今晚会不会低血糖
```

## 数据存储

所有血糖数据存储在本地：`~/.workbuddy/data/glucose_readings.json`

## 隐私说明

- ✅ 所有数据存储在用户本地设备
- ✅ 不向外部服务器传输任何数据
- ✅ 用户对自己的健康数据拥有完全控制权
