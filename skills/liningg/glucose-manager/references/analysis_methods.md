# 血糖分析方法说明

## 数据统计分析方法

### 1. 基础统计指标

#### 平均血糖（Mean Glucose）
**计算方法**：
```
平均血糖 = Σ 所有血糖值 / 血糖值数量
```

**临床意义**：
- 反映总体血糖控制水平
- 与糖化血红蛋白（HbA1c）相关
- 目标：接近个体化目标范围

**注意事项**：
- 极端值会影响平均值
- 需结合其他指标综合判断
- 分时段分析更有意义（空腹平均、餐后平均）

#### 标准差（Standard Deviation, SD）
**计算方法**：
```
SD = √[Σ(血糖值 - 平均血糖)² / (n-1)]
```

**临床意义**：
- 衡量血糖波动程度
- 数值越大，波动越大
- 目标：< 1.4 mmol/L（理想）

**解释**：
- SD < 1.4 mmol/L：血糖稳定
- SD 1.4-2.0 mmol/L：血糖波动中等
- SD > 2.0 mmol/L：血糖波动大

#### 变异系数（Coefficient of Variation, CV）
**计算方法**：
```
CV = (标准差 / 平均血糖) × 100%
```

**临床意义**：
- 标准化的血糖波动指标
- 不受平均血糖水平影响
- 国际共识目标：< 36%

**解释**：
- CV < 36%：血糖稳定性好 ✅
- CV 36-50%：血糖稳定性中等
- CV > 50%：血糖稳定性差

### 2. 血糖范围分析

#### 时间范围内（Time in Range, TIR）
**定义**：血糖在目标范围内的时间百分比

**标准范围**：
- 目标范围：3.9-10.0 mmol/L（70-180 mg/dL）
- 低于范围：< 3.9 mmol/L（< 70 mg/dL）
- 高于范围：> 10.0 mmol/L（> 180 mg/dL）

**计算方法**：
```
TIR = (目标范围内读数数量 / 总读数数量) × 100%
```

**国际目标**：
- TIR（3.9-10.0 mmol/L）：> 70%
- TBR（< 3.9 mmol/L）：< 4%
- TAR（> 10.0 mmol/L）：< 25%

#### 达标率
**定义**：血糖值达到个体化目标的百分比

**计算方法**：
```
达标率 = (达标读数数量 / 总读数数量) × 100%
```

**个体化目标示例**：
- 空腹：4.4-7.0 mmol/L
- 餐后：4.4-10.0 mmol/L
- 睡前：6.0-8.0 mmol/L

### 3. 血糖模式识别

#### 昼夜节律分析
**方法**：按时间段分组统计

**时间段划分**：
- 夜间（00:00-06:00）
- 早晨（06:00-12:00）
- 下午（12:00-18:00）
- 晚间（18:00-24:00）

**分析内容**：
- 各时段平均血糖
- 各时段血糖波动
- 识别异常时段

#### 餐前餐后对比
**方法**：配对分析餐前和餐后血糖

**计算指标**：
- 餐后血糖增幅 = 餐后血糖 - 餐前血糖
- 平均餐后增幅
- 异常增幅频率（> 3.0 mmol/L）

**临床意义**：
- 评估饮食和药物效果
- 识别餐后高血糖问题

#### 周内变异分析
**方法**：按星期几分组统计

**分析内容**：
- 工作日 vs 周末
- 各天平均血糖
- 识别生活方式影响

### 4. 趋势分析方法

#### 移动平均
**方法**：计算指定时间窗口内的平均值

**常用窗口**：
- 7天移动平均
- 14天移动平均
- 30天移动平均

**应用**：
- 平滑日常波动
- 识别长期趋势
- 评估治疗效果

**计算示例**（7天移动平均）：
```
MA7 = (Day1 + Day2 + ... + Day7) / 7
```

#### 线性回归趋势
**方法**：拟合血糖随时间变化的直线

**公式**：
```
血糖 = a × 天数 + b
```

**斜率解释**：
- 斜率 > 0：血糖上升趋势
- 斜率 = 0：血糖稳定
- 斜率 < 0：血糖下降趋势

**临床应用**：
- 评估治疗效果
- 预测未来血糖走向
- 早期发现问题

### 5. 异常检测

#### 低血糖检测
**定义标准**：
- 一级低血糖：< 3.9 mmol/L
- 二级低血糖：< 3.0 mmol/L
- 三级低血糖：需要他人帮助

**统计指标**：
- 低血糖发生次数
- 低血糖发生率
- 低血糖发生时段
- 平均持续时间

#### 高血糖检测
**定义标准**：
- 轻度高血糖：> 10.0 mmol/L
- 中度高血糖：> 13.9 mmol/L
- 重度高血糖：> 22.0 mmol/L

**统计指标**：
- 高血糖发生次数
- 高血糖发生率
- 高血糖发生时段
- 极端高血糖（> 22.0 mmol/L）

#### 异常值识别
**统计方法**：
- Z-score 方法：|Z| > 3 标记为异常
- IQR 方法：超过 Q3 + 1.5×IQR 或低于 Q1 - 1.5×IQR

**Z-score 计算**：
```
Z = (血糖值 - 平均血糖) / 标准差
```

### 6. 风险评估模型

#### 低血糖风险评分
**考虑因素**：
1. 历史低血糖频率（权重：30%）
2. 平均血糖水平（权重：25%）
3. 血糖波动性（CV）（权重：25%）
4. 最近一次低血糖时间（权重：20%）

**评分公式**：
```
风险评分 = Σ (因素得分 × 权重)
```

**风险等级**：
- 0-30：低风险
- 31-60：中等风险
- 61-100：高风险

#### 高血糖风险评分
**考虑因素**：
1. 历史高血糖频率（权重：30%）
2. 平均餐后血糖（权重：30%）
3. 空腹血糖水平（权重：20%）
4. 最近血糖趋势（权重：20%）

### 7. 预测分析方法

#### 短期预测（2-4小时）
**方法**：基于最近读数的趋势外推

**算法**：
1. 提取最近2-4小时血糖数据
2. 计算变化速率（mmol/L/小时）
3. 假设速率保持不变
4. 推算未来血糖

**公式**：
```
预测血糖 = 当前血糖 + 变化速率 × 预测时间
```

**局限性**：
- 仅适用于短期
- 未考虑饮食、运动等变量
- 准确性随时间降低

#### 日模式预测
**方法**：基于历史同时段平均值

**算法**：
1. 提取过去7-14天同一时段数据
2. 计算平均值和标准差
3. 预测范围：平均值 ± 标准差

**示例**：
```
明日空腹预测 = 过去7天空腹平均 ± 过去7天空腹标准差
```

#### 周模式预测
**方法**：考虑星期几的模式

**算法**：
1. 按星期几分组统计历史数据
2. 识别工作日 vs 周末差异
3. 预测特定星期几的血糖

### 8. 相关性分析

#### 血糖与时间的相关性
**方法**：计算皮尔逊相关系数

**公式**：
```
r = Σ[(xi - x̄)(yi - ȳ)] / √[Σ(xi - x̄)² × Σ(yi - ȳ)²]
```

**解释**：
- r 接近 1：强正相关
- r 接近 0：无线性相关
- r 接近 -1：强负相关

#### 血糖影响因素分析
**潜在因素**：
- 饮食（碳水化合物摄入）
- 运动
- 睡眠质量
- 压力水平
- 药物使用

**分析方法**：
- 多元线性回归
- 时间序列分析
- 机器学习方法（如需要）

### 9. 报告生成

#### 周报告内容
1. **数据概览**
   - 总监测次数
   - 监测天数
   - 平均血糖
   - 血糖范围

2. **血糖控制评估**
   - 达标率
   - 时间范围内（TIR）
   - HbA1c 估算值

3. **血糖波动评估**
   - 标准差
   - 变异系数
   - 最大值、最小值

4. **异常事件**
   - 低血糖次数
   - 高血糖次数
   - 极端值提醒

5. **趋势分析**
   - 与上周对比
   - 7天移动平均
   - 改善或恶化趋势

6. **个性化建议**
   - 饮食建议
   - 运动建议
   - 监测建议

#### 月报告内容
在周报告基础上增加：
1. 月度趋势图
2. 餐前餐后分析
3. 时段分析
4. HbA1c 估算准确性验证
5. 治疗效果评估

### 10. HbA1c 估算

#### ADAG 公式
**方法**：从平均血糖估算 HbA1c

**公式**：
```
HbA1c (%) = (平均血糖 mmol/L + 1.59) / 1.59
```

或：
```
HbA1c (%) = (平均血糖 mg/dL + 46.7) / 28.7
```

**注意**：
- 仅适用于平均血糖稳定时
- 个体差异可达 ±0.5%
- 不能替代实验室检测

## 数据可视化方法

### 1. 时间序列图
**用途**：展示血糖随时间变化

**要素**：
- X轴：时间
- Y轴：血糖值
- 目标范围阴影区域
- 标注餐前餐后

### 2. 血糖分布直方图
**用途**：展示血糖值分布

**要素**：
- X轴：血糖范围
- Y轴：频率
- 标注目标范围

### 3. 饼图
**用途**：展示时间范围内（TIR）

**要素**：
- 低于范围（红色）
- 目标范围（绿色）
- 高于范围（黄色）

### 4. 箱线图
**用途**：展示各时段血糖分布和变异

**要素**：
- 中位数、四分位数
- 异常值

### 5. 热力图
**用途**：展示血糖模式

**要素**：
- X轴：日期
- Y轴：时段
- 颜色：血糖水平

## 数据质量要求

### 最小数据量
- 周报告：至少7天，每天≥3次
- 月报告：至少20天
- 趋势分析：至少14天
- 预测分析：至少30天

### 数据分布要求
- 包含空腹和餐后读数
- 覆盖不同时段
- 避免连续缺失数据

**重要提示**：本分析方法基于统计学原理，仅供参考，不能替代医疗诊断。实际应用应结合临床判断和个体情况。
