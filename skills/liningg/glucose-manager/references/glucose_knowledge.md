# 血糖管理知识库

本文档提供血糖管理的医学知识和参考信息，用于支持技能的建议生成。

## 一、血糖基础知识

### 1.1 什么是血糖

血糖是指血液中的葡萄糖浓度，是人体主要的能量来源。正常情况下，人体通过激素调节（主要是胰岛素和胰高血糖素）维持血糖在相对稳定的范围内。

### 1.2 血糖调节机制

**胰岛素的作用：**
- 促进细胞摄取葡萄糖
- 促进糖原合成
- 抑制糖原分解
- 促进脂肪合成

**胰高血糖素的作用：**
- 促进糖原分解
- 促进糖异生
- 升高血糖

### 1.3 血糖单位换算

- **mmol/L**（国际单位）：中国常用
- **mg/dL**（美国单位）：美国等地区常用

换算公式：
- mmol/L × 18 = mg/dL
- mg/dL ÷ 18 = mmol/L

## 二、血糖目标范围

### 2.1 糖尿病患者血糖目标

根据《中国2型糖尿病防治指南》：

| 测量类型 | 目标范围 (mmol/L) |
|---------|------------------|
| 空腹血糖 | 4.4-7.0 |
| 餐后2小时血糖 | <10.0 |
| 睡前血糖 | 6.0-8.0 |
| HbA1c | <7.0% |

### 2.2 血糖分类

| 分类 | 血糖范围 (mmol/L) | 说明 |
|-----|------------------|------|
| 低血糖 | <3.9 | 需要立即处理 |
| 正常血糖 | 3.9-7.0 (空腹) | 健康范围 |
| 糖尿病前期 | 6.1-6.9 (空腹) | 需要干预 |
| 糖尿病诊断标准 | ≥7.0 (空腹) 或 ≥11.1 (餐后2小时) | 需要治疗 |

### 2.3 个体化目标

血糖目标应根据个体情况调整：

**更严格目标（HbA1c <6.5%）：**
- 年轻、病程短
- 无心血管疾病
- 无低血糖风险

**较宽松目标（HbA1c <8.0%）：**
- 老年人
- 有心血管疾病
- 频发低血糖
- 预期寿命有限

## 三、血糖监测

### 3.1 监测频率建议

**胰岛素治疗患者：**
- 每日至少3-4次（空腹+三餐后）
- 调整剂量时可能需要更频繁

**口服降糖药患者：**
- 每周2-4次（空腹+餐后各1-2次）
- 血糖稳定时可减少频率

**仅生活方式干预：**
- 每周1-2次
- 空腹和餐后交替

### 3.2 监测时间点

| 时间点 | 意义 |
|-------|------|
| 空腹 | 反映基础血糖水平 |
| 餐前 | 了解餐前血糖状态 |
| 餐后2小时 | 评估餐后血糖控制 |
| 睡前 | 预防夜间低血糖 |
| 凌晨2-3点 | 排除夜间低血糖 |
| 运动前后 | 评估运动影响 |

### 3.3 影响血糖的因素

**升高血糖的因素：**
- 食物（尤其是碳水化合物）
- 应激、感染、疾病
- 某些药物（糖皮质激素等）
- 黎明现象
- 缺乏运动

**降低血糖的因素：**
- 胰岛素和降糖药物
- 运动
- 酒精（延迟效应）
- 热量摄入不足

## 四、饮食管理

### 4.1 碳水化合物计数

**基本概念：**
- 1份碳水 = 15克碳水化合物
- 一般成人每餐需要3-4份碳水（45-60克）
- 根据个体情况调整

**碳水计数应用：**
```
餐前胰岛素剂量 = 总碳水克数 ÷ 胰岛素/碳水比
```

### 4.2 血糖指数（GI）

**GI分类：**
- 低GI：<55
- 中GI：55-70
- 高GI：>70

**常见食物GI值：**

| 食物 | GI值 | 分类 |
|-----|------|------|
| 葡萄糖 | 100 | 高 |
| 白米饭 | 83 | 高 |
| 白面包 | 75 | 高 |
| 全麦面包 | 53 | 低 |
| 燕麦 | 55 | 中 |
| 荞麦 | 46 | 低 |
| 苹果 | 36 | 低 |
| 香蕉 | 51 | 低 |
| 牛奶 | 27 | 低 |
| 豆类 | 30-40 | 低 |

### 4.3 饮食原则

**均衡饮食：**
- 碳水化合物：50-60%
- 蛋白质：15-20%
- 脂肪：25-30%

**食物选择建议：**

| 推荐食物 | 限制食物 |
|---------|---------|
| 全谷物 | 精制谷物 |
| 蔬菜 | 含糖饮料 |
| 水果（适量） | 甜点 |
| 瘦肉、鱼、蛋 | 肥肉 |
| 豆类 | 油炸食品 |
| 低脂奶制品 | 高糖水果 |

## 五、运动管理

### 5.1 运动对血糖的影响

**有氧运动：**
- 即时降低血糖
- 增加胰岛素敏感性
- 效果持续24-48小时

**抗阻运动：**
- 增加肌肉量
- 提高基础代谢
- 长期改善血糖控制

### 5.2 运动建议

**频率：**
- 每周至少150分钟中等强度有氧运动
- 每周2-3次抗阻运动

**时机：**
- 餐后1-2小时最佳
- 避免空腹运动（防低血糖）
- 避免血糖>13.9时剧烈运动

**运动强度判断：**
- 低强度：可以轻松交谈
- 中等强度：可以交谈但不能唱歌
- 高强度：说话困难

### 5.3 运动注意事项

**运动前：**
- 血糖<5.6：补充碳水
- 血糖>13.9：暂缓运动
- 血糖>16.7+酮症：禁止运动

**运动中：**
- 随身携带糖果
- 每30分钟补充水分
- 注意低血糖症状

**运动后：**
- 监测血糖变化
- 注意延迟性低血糖
- 适当补充碳水

## 六、药物管理

### 6.1 常用口服降糖药

| 类别 | 代表药物 | 作用机制 | 主要副作用 |
|-----|---------|---------|-----------|
| 双胍类 | 二甲双胍 | 减少肝糖输出 | 胃肠道反应 |
| 磺脲类 | 格列美脲 | 促进胰岛素分泌 | 低血糖 |
| 格列奈类 | 瑞格列奈 | 促进胰岛素分泌 | 低血糖 |
| α-糖苷酶抑制剂 | 阿卡波糖 | 延缓碳水吸收 | 腹胀 |
| DPP-4抑制剂 | 西格列汀 | 增加肠促胰素 | 较少 |
| SGLT-2抑制剂 | 达格列净 | 促进尿糖排出 | 尿路感染 |
| TZD | 吡格列酮 | 改善胰岛素抵抗 | 水肿、体重增加 |

### 6.2 胰岛素类型

| 类型 | 起效时间 | 峰值时间 | 持续时间 | 使用时机 |
|-----|---------|---------|---------|---------|
| 速效 | 10-15分钟 | 1-2小时 | 4-6小时 | 餐前即刻 |
| 短效 | 30分钟 | 2-4小时 | 6-8小时 | 餐前30分钟 |
| 中效 | 1-2小时 | 4-10小时 | 12-18小时 | 每日1-2次 |
| 长效 | 2-4小时 | 无明显峰值 | 20-24小时 | 每日1次 |
| 预混 | 10-30分钟 | 双峰 | 12-24小时 | 每日2次 |

### 6.3 用药原则

- 从小剂量开始
- 逐步调整剂量
- 注意低血糖风险
- 定期评估疗效
- 联合用药需谨慎

## 七、低血糖管理

### 7.1 低血糖定义

- **低血糖**：血糖 <3.9 mmol/L
- **严重低血糖**：需要他人协助

### 7.2 低血糖症状

| 早期症状 | 中期症状 | 严重症状 |
|---------|---------|---------|
| 饥饿感 | 视力模糊 | 意识模糊 |
| 心慌 | 行为异常 | 昏迷 |
| 出汗 | 步态不稳 | 抽搐 |
| 手抖 | 说话含糊 | |
| 头晕 | | |

### 7.3 低血糖处理

**15-15原则：**
1. 补充15克快速吸收的碳水
2. 等待15分钟
3. 复测血糖
4. 如仍<4.0，重复步骤1-3

**15克碳水来源：**
- 葡萄糖片：3-4片
- 果汁：150ml
- 糖果：4-5颗
- 蜂蜜：1-2汤匙

**严重低血糖（意识不清）：**
- 不要喂食任何东西
- 立即就医
- 注射胰高血糖素（如有）

## 八、高血糖管理

### 8.1 高血糖原因

- 饮食过量
- 药物不足或漏服
- 应激、感染
- 运动不足
- 黎明现象
- 苏末效应

### 8.2 高血糖处理

**轻度升高（10-13.9 mmol/L）：**
- 多饮水
- 适当运动
- 检查饮食
- 下餐减少碳水

**中度升高（13.9-16.7 mmol/L）：**
- 监测血糖变化
- 检查药物是否漏服
- 排除感染等诱因
- 必要时咨询医生

**重度升高（>16.7 mmol/L）：**
- 检测尿酮/血酮
- 如有酮症：立即就医
- 如无酮症：咨询医生是否调整药物
- 注意补水

### 8.3 酮症酸中毒警示

**症状：**
- 明显口渴
- 多尿
- 恶心呕吐
- 呼吸深快
- 腹痛
- 意识模糊

**处理：**
- 立即就医
- 不要运动
- 补充水分

## 九、HbA1c与血糖控制评估

### 9.1 HbA1c的意义

- 反映最近2-3个月平均血糖
- 不受短期血糖波动影响
- 糖尿病控制的金标准

### 9.2 HbA1c与平均血糖关系

| HbA1c (%) | 平均血糖 (mmol/L) |
|-----------|------------------|
| 5 | 5.4 |
| 6 | 7.0 |
| 7 | 8.6 |
| 8 | 10.2 |
| 9 | 11.8 |
| 10 | 13.4 |

**计算公式：**
```
平均血糖 (mmol/L) = (HbA1c × 1.59) - 2.59
```

### 9.3 Time in Range（目标范围内时间）

| 指标 | 目标 |
|-----|------|
| 目标范围内 (3.9-10.0) | >70% |
| 低于目标 (<3.9) | <4% |
| 高于目标 (>10.0) | <25% |

## 十、特殊情况

### 10.1 黎明现象

**定义：** 清晨血糖升高

**原因：** 夜间激素分泌增多

**处理：**
- 调整睡前胰岛素剂量
- 调整长效胰岛素注射时间
- 咨询医生

### 10.2 苏末效应

**定义：** 夜间低血糖后清晨血糖反跳性升高

**处理：**
- 睡前适当加餐
- 减少晚餐前或睡前胰岛素
- 监测夜间血糖

### 10.3 生病日管理

- 增加血糖监测频率
- 维持足够的水分和热量
- 不要停用胰岛素
- 监测尿酮
- 何时就医：
  - 持续呕吐腹泻
  - 血糖>16.7且持续
  - 尿酮阳性
  - 意识改变

## 十一、并发症预防

### 11.1 定期检查项目

| 检查项目 | 频率 |
|---------|------|
| HbA1c | 每3-6个月 |
| 眼底检查 | 每年 |
| 尿微量白蛋白 | 每年 |
| 足部检查 | 每次就诊 |
| 血脂 | 每年 |
| 血压 | 每次就诊 |

### 11.2 足部护理

- 每日检查足部
- 保持足部清洁干燥
- 穿合适的鞋袜
- 及时处理足部伤口
- 避免烫伤

## 十二、紧急情况

### 12.1 需要立即就医的情况

- 严重低血糖（意识不清）
- 酮症酸中毒
- 高渗高血糖状态
- 严重感染
- 心脑血管事件
- 严重脱水

### 12.2 急救联系

- 急救电话：120
- 内分泌科急诊
- 随身携带糖尿病识别卡
