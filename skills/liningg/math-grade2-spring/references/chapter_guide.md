## 一、除法

### 1.1 分苹果（P2-3）

**知识点详解：**

平均分是除法的基础概念，核心是"每份分得同样多"。

**两种平均分的方法：**

1. **按份数分（等分）**
   - 把物品分成指定的几份，每份分得同样多
   - 例如：有12个苹果，平均分成3份，每份4个

2. **按每份数分（包含除）**
   - 按照每份几个来分，能分成几份
   - 例如：有12个苹果，每份4个，能分成3份

**例题讲解：**

例1：有6个苹果，平均分给3个人，每人分得几个？
```
解答：6 ÷ 3 = 2（个）
      ÷ 表示平均分
      6 是被分的东西
      3 是要分给的人数
      2 是每人分得的个数
```

**重点难点：**
- 重点：理解"平均分"就是每份分得同样多
- 难点：分清两种平均分的方法

### 1.4 分草莓（P8-9）

**知识点详解：**

认识除法的初步理解。

**除法的含义：**
- 除法是解决平均分问题的运算
- 表示把一些物品平均分成若干份

**除法算式的各部分：**
```
12 ÷ 3 = 4
│  │  │
│  │  └─ 商
│  └──── 除号（平均分）
└─────── 被除数（总数）
        └─ 除数（平均分成几份）
```

**例题讲解：**

例1：有12个草莓，平均分给4个小朋友，每人分得几个？
```
12 ÷ 4 = 3（个）
表示：把12个草莓平均分成4份，每份3个
```

**重点难点：**
- 重点：理解除法的含义
- 难点：理解除法算式的意义

### 1.5 分橘子（P10-11）

**知识点详解：**

除法算式的各部分名称。

**除法算式的读写：**
```
读法：12除以4等于3
写法：12 ÷ 4 = 3
```

**各部分名称：**
```
被除数 ÷ 除数 = 商
  12   ÷   4   =   3
```

**例题讲解：**

例1：读出算式：8 ÷ 2 = 4
```
读作：8除以2等于4
```

例2：写出算式：10除以5等于2
```
10 ÷ 5 = 2
```

**重点难点：**
- 重点：除法算式的读写
- 难点：区分被除数和除数

### 1.6 分一分（P12-13）

**知识点详解：**

**两种分法的比较：**

1. **按份数分**

   - 算式：12 ÷ 4 = 3

2. **按每份数分**

    → 按每份数分
```

**重点难点：**
- 重点：两种平均分方法的区别
- 难点：选择合适的方法

## 二、混合运算

### 2.1 小熊购物（P16-17）

**知识点详解：**

加减乘除混合运算的顺序。

**运算顺序：**
1. 有括号的，先算括号里面的
2. 没有括号的，先算乘除，后算加减

**例题讲解：**

例1：3 + 2 × 4
```
先算乘法：2 × 4 = 8
再算加法：3 + 8 = 11
```

例2：（3 + 2）× 4
```
先算括号里面：3 + 2 = 5
再算乘法：5 × 4 = 20
```

**重点难点：**
- 重点：先乘除后加减
- 难点：带有括号的运算

### 2.3 周长（P20-21）

**知识点详解：**

认识周长，计算图形的周长。

**周长的概念：**
- 周长是封闭图形一周的长度

**长方形周长：**
```
周长 = （长 + 宽）× 2
```

**正方形周长：**
```
周长 = 边长 × 4
```

**例题讲解：**

例1：一个长方形，长6厘米，宽4厘米，周长是多少？
```
（6 + 4）× 2
= 10 × 2
= 20（厘米）

答：周长是20厘米。
```

**重点难点：**
- 重点：计算长方形、正方形的周长
- 难点：周长公式的应用

(3) 比直尺的角小的角 → 锐角
(4) 比直尺的角大的角 → 钝角
```

**重点难点：**
- 重点：直角、锐角、钝角的区分
- 难点：角的大小比较

### 6.2 认识长方形和正方形（P48-49）

**知识点详解：**

认识长方形和正方形的特征。

**长方形特征：**
- 有4条边
- 对边相等
- 有4个直角

**正方形特征：**
- 有4条边
- 四条边都相等
- 有4个直角

**区别：**
- 长方形：对边相等
- 正方形：四条边都相等

**例题讲解：**

例1：判断下面的图形是长方形还是正方形？
```
(1) 长是6厘米，宽是4厘米 → 长方形
(2) 四条边都是5厘米 → 正方形
```

**重点难点：**
- 重点：长方形和正方形的特征
- 难点：长方形和正方形的区别
