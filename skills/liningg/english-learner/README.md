1. **首次使用**：让助手评估您的英语水平，制定个性化学习计划
2. **日常学习**：每天坚持 30-60 分钟，涵盖听说读写
3. **词汇学习**：结合语境记忆，定期复习
4. **语法学习**：理解规则后立即练习
5. **听说训练**：多听多模仿，大胆开口
6. **写作训练**：从简单句开始，逐步提高复杂度
