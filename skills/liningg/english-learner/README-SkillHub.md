## 🎯 技能简介

这是一个全功能的英语学习辅助技能，帮助用户系统性地提升英语能力，涵盖听说读写四个维度，提供个性化的学习方案和进度追踪。

**适用人群：**
- 英语初学者（A1-A2 级别）
- 英语进阶者（B1-B2 级别）
- 英语高级者（C1-C2 级别）
- 备考学生（托福、雅思、四六级等）
- 职场人士（商务英语）
- 英语爱好者（持续学习）

- ✅ 分级阅读材料推荐

- ✅ 写作批改和优化建议

### 6. 翻译助手
- ✅ 中英互译

### 7. 学习管理

## 🚀 安装方法

### 方法 1：通过 SkillHub 安装（推荐）

1. 下载 `english-learner.zip` 文件
2. 打开 WorkBuddy
3. 进入技能市场（Skills/技能）
4. 点击"导入技能"
5. 选择下载的 `english-learner.zip` 文件
6. 点击"安装"
7. 安装完成后即可使用

### 方法 2：手动安装

1. 下载并解压 `english-learner.zip` 文件
2. 将解压后的 `english-learner` 文件夹复制到：
   - 用户技能：`~/.workbuddy/skills/`
   - 项目技能：`.workbuddy/skills/`
3. 重启 WorkBuddy
4. 技能自动加载完成

## 📊 使用场景示例

### 场景 1：制定学习计划
```
用户：我是一个英语初学者，请帮我制定学习计划
助手：✅ 已为您制定 30 天个性化学习计划
      • 当前水平：A1-A2
      • 目标：A2-B1
      • 词汇量目标：1000 词
      • 每日任务：词汇 15分钟 + 语法 15分钟 + 综合练习 20-30分钟
```

### 场景 2：背单词
```
用户：帮我背雅思核心词汇
助手：📚 雅思核心词汇学习计划
      • 今日目标：20 个单词
      • 复习：15 个单词
      • 新词：5 个单词
      
      第 1 个：abandon
      • 音标：/əˈbændən/
      • 词义：v. 放弃，抛弃
      • 例句：He abandoned his car in the snow.
      • 搭配：abandon hope（放弃希望）
```

### 场景 3：写作批改
```
用户：帮我改一下这段英文
用户原文：I very like English, but my speaking is not good.
助手：✍️ 写作批改
      
      修改建议：
      • "very like" → "really like" 或 "really enjoy"
      • "is not good" → "is not very good" 或 "needs improvement"
      
      更好表达：
      "I really enjoy English, but my speaking needs improvement."
```

### 场景 4：翻译
```
用户："这个项目进展顺利" 翻译成英语
助手：🌐 翻译：中文 → 英文
      
      翻译选项：
      1. The project is proceeding smoothly.（常用）
      2. The project is making good progress.（地道）
      3. The project is going well.（简洁）
      
      建议使用选项 2，更符合商务英语表达习惯。
```

## 📞 技术支持

### 常见问题

**Q: 技能安装后无法使用？**
A: 请检查 SKILL.md 文件是否完整，确保 YAML frontmatter 格式正确。

**Q: 想要添加自定义词汇怎么办？**
A: 可以在 references/ 目录添加自定义词汇表，或直接告诉我想要添加的单词。

**Q: 能否修改学习计划？**
A: 可以！随时告诉我您的需求，我会为您调整学习计划。

**Q: 学习数据保存在哪里？**
A: 所有学习数据保存在您本地（`~/.workbuddy/memory/` 或 `.workbuddy/memory/`），不会上传到云端。
