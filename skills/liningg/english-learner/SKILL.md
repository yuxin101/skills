# 英语学习助手

## 概述

这是一个全功能的英语学习辅助技能，帮助用户系统性地提升英语能力，涵盖听说读写四个维度，提供个性化的学习方案和进度追踪。

## 核心功能

### 1. 词汇学习

**单词记忆**
- 智能记忆曲线：根据遗忘曲线安排复习间隔
- 词源解析：词根、前缀、后缀分析
- 例句展示：真实语境中的应用
- 搭配推荐：常见词组搭配和用法
- 同义词/反义词/近义词扩展

**单词测试**
- 英译中/中译英测试
- 填空练习
- 听音选词
- 错题记录和强化训练

**词汇管理**
- 按主题分类（商务、日常、学术等）
- 按难度分级（CEFR A1-C2）
- 按词频分级（高频词、核心词、扩展词）
- 导入/导出单词本（支持 Excel/CSV）

### 2. 语法学习

**语法点讲解**
- 时态、语态、从句等核心语法
- 图表化展示，易于理解
- 对比分析（相似语法点对比）
- 常见错误提示

**语法练习**
- 填空练习
- 改错练习
- 句型转换
- 语法综合测试

**语法诊断**
- 分析用户句子中的语法问题
- 提供修改建议和解释
- 追踪常见错误模式

### 3. 阅读训练

**阅读材料推荐**
- 根据英语水平推荐文章
- 多领域文章（新闻、科技、文化等）
- 可分级阅读材料
- 带注释的原文

**阅读理解**
- 文章理解测试
- 段落主旨分析
- 推理判断题
- 词汇在语境中的含义

**精读训练**
- 逐句解析
- 长难句拆解
- 词汇积累
- 写作手法分析

### 4. 听说训练

**听力练习**
- 英式/美式发音
- 不同语速和口音
- 听写练习
- 对话和独白材料

**口语练习**

- 发音指导（音标、重音、语调）

**发音评测**
- 语音识别分析
- 对比标准发音

### 5. 写作训练

**写作批改**

### 6. 翻译助手

### ✍️ 写作训练场景

**场景 9：写作批改**
```
用户: 帮我改一下这段英文
用户原文：I very like English, but my speaking is not good.
助手: ✍️ 写作批改
      
      原文：I very like English, but my speaking is not good.
      
      修改建议：
      • "very like" → "really like" 或 "really enjoy"
      • "is not good" → "is not very good" 或 "needs improvement"
      
      更好表达：
      "I really enjoy English, but my speaking needs improvement."
      或
      "I'm very interested in English, though my speaking skills could use some work."
```

**场景 10：写作模板**

### 🌐 翻译场景

**场景 11：中译英**
```
用户: "这个项目进展顺利" 翻译成英语
助手: 🌐 翻译：中文 → 英文
      
      原文：这个项目进展顺利
      
      翻译选项：
      1. The project is proceeding smoothly.（常用）
      2. The project is making good progress.（地道）
      3. The project is going well.（简洁）
      
      建议使用选项2，更符合商务英语表达习惯。
```

**场景 12：英译中**
```
用户: "What brings you here today?" 翻译成中文
助手: 🌐 翻译：英文 → 中文
      
      原文：What brings you here today?
      
      翻译：
      • "您今天为什么来这里？"（直译）
      • "您今天有什么事？"（口语）
      • "请问您今天有何贵干？"（正式）
      
      根据语境选择合适的表达。
```

## 工作流程

1. **初始评估** - 首次使用时评估用户英语水平（词汇量、语法基础等）
2. **个性化规划** - 根据评估结果制定学习计划和目标
3. **日常学习** - 按计划进行词汇、语法、听说读写训练
4. **进度追踪** - 记录学习数据，分析进步情况
5. **调整优化** - 根据学习效果调整学习策略
6. **定期测试** - 周期性进行综合测试，检验学习成果

## 使用建议

**日常学习**
- 每天坚持30-60分钟学习时间
- 新学习和复习比例1:2
- 听说读写均衡发展

**词汇学习**
- 结合语境记忆单词
- 定期复习（艾宾浩斯遗忘曲线）
- 在阅读和听力中巩固词汇

**语法学习**
- 理解语法规则后立即练习
- 在写作中应用语法知识
- 记录常见错误并强化

**听说训练**
- 多听不同口音和语速的材料
- 模仿标准发音
- 大胆开口练习

**写作训练**
- 从简单句开始，逐步提高复杂度
- 多读范文，学习表达方式
- 重视批改反馈

## 重要说明

**免责声明**
- 本技能是学习辅助工具，不能替代系统性的英语课程
- AI批改和建议仅供参考，最终由用户判断
- 真实交流中建议咨询专业英语教师

**学习效果**
- 学习效果取决于学习频率和投入时间
- 建议每天坚持学习，避免长时间中断
- 结合实际应用（阅读英文资料、观看英文视频等）效果更佳

**数据隐私**
- 学习数据存储在用户本地
- 不向外部服务器传输个人信息
- 用户对自己的学习数据拥有完全控制权
