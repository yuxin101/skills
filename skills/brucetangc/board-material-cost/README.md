# 艾威板材费用计算器

## 📋 功能说明

从激光切割 PDF 文件中提取 **Sheet dimension** 数据，**独立生成**艾威格式 Excel（**不依赖模板文件**）。

## 🚀 使用方法

### 基础用法
```bash
python3 calculator.py <PDF 文件夹> [输出文件] [废料单价] [价格表] [月份]
```

### 示例
```bash
# 基础用法
python3 calculator.py /tmp/060_processing /tmp/060_艾威板材统计.xlsx

# 带废料单价（2%）
python3 calculator.py /tmp/060_processing /tmp/060_艾威板材统计.xlsx 0.02

# 带价格表（自动填充 M 列单价）
python3 calculator.py /tmp/060_processing /tmp/060_艾威板材统计.xlsx 0.02 /tmp/2026 年 3 月价格表.xlsx 2026-03

# 只带价格表（当前月份）
python3 calculator.py /tmp/060_processing /tmp/060_艾威板材统计.xlsx 0 0 /tmp/价格表.xlsx
```

## 📊 完整逻辑流程

### 1️⃣ 从文件名提取厚度和材质

| PDF 文件名 | 提取逻辑 | 结果 |
|-----------|---------|------|
| `1.5mm.pdf` | 数字+mm | 厚度=1.5mm, 材质=Q235 |
| `3Mn.pdf` | 数字+Mn | 厚度=3mm, 材质=Q345 |
| `10mm.pdf` | 数字+mm | 厚度=10mm, 材质=Q235 |

### 2️⃣ 厚度映射（负公差）

| PDF 厚度 | 模板厚度 | 说明 |
|---------|---------|------|
| 1.5mm | 1.5mm | - |
| 2mm | 2mm | - |
| 3mm | 2.75mm | 负公差 |
| 4mm | 3.75mm | 负公差 |
| 5mm | 4.75mm | 负公差 |
| 6mm | 5.75mm | 负公差 |
| 8mm | 7.75mm | 负公差 |
| 10mm | 9.75mm | 负公差 |
| 12mm | 11.75mm | 负公差 |

### 3️⃣ 材质显示规则

| 厚度 | 材质 | 显示 |
|------|------|------|
| ≤2mm | Q235 | Q235 冷板 |
| >2mm | Q235 | Q235 |
| 任意 | Q345 | Q345 |
| 任意 | SUS | SUS |

### 4️⃣ 从 PDF 提取 Sheet dimension

**提取流程**：
```
1. 查找 "Sheet dimension" 表头
   ↓
2. 从表头下一行开始遍历
   ↓
3. 查找尺寸数据（格式：数字 x 数字 mm）
   ↓
4. 检查前面是否有另一个尺寸（Plan dimension）
   - 有 → 这是 Sheet dimension，继续提取
   - 没有 → 跳过（可能是 Plan dimension）
   ↓
5. 向后查找 Cycles（纯数字）
   ↓
6. 向后查找 Waste（百分比）
   ↓
7. 保存完整数据（长×宽×数量×废料率）
   ↓
8. 去重（相同长×宽×数量×废料率只保留一次）
```

**PDF 数据结构示例**：
```
行 8:  Sheet dimension     ← 表头
行 9:  Cycles              ← 表头
行 10: Cutting time        ← 表头
行 11: Waste               ← 表头
行 12: Number of parts     ← 表头
行 13: 2318x 1033mm        ← Plan dimension（忽略）
行 14: 2330x 1050mm        ← Sheet dimension（提取）✓
行 15: 1                   ← Cycles（提取）✓
行 16: 5.09min             ← Cutting time（跳过）
行 17: 14.35%              ← Waste（提取）✓
行 18: 5                   ← Number of parts（跳过）
```

### 5️⃣ Excel 生成（无模板）

**15 列结构**：

| 列号 | 字母 | 字段名 | 数据来源 | 公式 |
|------|------|--------|---------|------|
| 1 | A | 序号 | 自动生成 | 1,2,3... |
| 2 | B | 日期 | 空 | - |
| 3 | C | 订单号 | 空 | - |
| 4 | D | 材质 | PDF 文件名 | Q235 冷板/Q235/Q345 |
| 5 | E | 长（mm） | PDF 提取 | - |
| 6 | F | 宽（mm） | PDF 提取 | - |
| 7 | G | 厚（mm） | 厚度映射 | 1.5/2/2.75... |
| 8 | H | 数量 | PDF 提取 | - |
| 9 | I | 重量（单重 Kg） | 公式 | `=E×F×G/1000000×7.85` |
| 10 | J | 合计重量（Kg） | 公式 | `=I×H` |
| 11 | K | 板材利用率 | 公式 | `=1-O` |
| 12 | L | 废料重量（Kg） | 公式 | `=J×(1-K)×0.85` |
| 13 | M | 单价（元/Kg） | 手动填写 | - |
| 14 | N | 板材价格（元） | 公式 | `=J×M` |
| 15 | O | 废料率% | PDF 提取 | - |

### 6️⃣ 合计行结构

| 行号 | A 列 | L 列 | 说明 |
|------|------|------|------|
| N+1 | 合计 | `=SUM(L2:LN)` | 废料重量合计 |
| N+2 | 废料单价 | 空或用户指定 | 废料单价（元/Kg） |
| N+3 | 废料价格 | `=L(N+1)×L(N+2)` | 废料价格 |

**说明**：
- N = 最后数据行号
- 废料单价：如果用户提供则填写，否则留空
- 废料价格：公式自动计算（合计废料重量 × 废料单价）

### 7️⃣ 样式设置

**表头（第 1 行）**：
- 行高：54
- 背景：灰色（#DDDDDD）
- 字体：加粗
- 对齐：居中
- 边框：细边框

**数据行（第 2-N 行）**：
- 对齐：居中
- 边框：细边框
- K 列、O 列：百分比格式（0.00%）
- I、J、L、N 列：小数两位（0.00）

**合计行 + 废料单价 + 废料价格**：
- 字体：加粗（合计行和废料价格）
- 对齐：居中
- 边框：细边框（到 O 列）

## 🎯 使用场景

1. **没有模板文件时**：独立生成 Excel
2. **批量处理**：自动处理整个文件夹的 PDF
3. **自定义样式**：可调整列宽、行高、格式等

## ⚠️ 注意事项

1. **废料单价**：
   - 如果提供：填写到 L 列
   - 如果不提供：留空，等待用户手动填写

2. **去重逻辑**：
   - 相同 长×宽×数量×废料率 只保留一次
   - 避免重复数据

3. **厚度映射**：
   - 使用负公差标准（3mm→2.75mm）
   - 符合艾威模板规范
