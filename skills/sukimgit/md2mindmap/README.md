# Md2Mindmap - Markdown 思维导图工具

[![ClawHub Skill](https://img.shields.io/badge/ClawHub-Skill-blue)](https://clawhub.com)
[![Version](https://img.shields.io/badge/version-1.0.0-green)](https://clawhub.com)

**一句话介绍：** 将 Markdown 文档一键转换为可交互的思维导图，支持 HTML + PDF 输出！

---

## 快速开始

### 安装

```bash
clawhub install md2mindmap
```

### 使用

```bash
# 生成 HTML
python markmap_generator.py --file input.md --output output.html

# 生成 HTML + PDF
python markmap_generator.py --file input.md --output output.html --pdf output.pdf
```

---

## 功能特性

- ✅ Markdown 标题层级自动识别
- ✅ 可交互思维导图（缩放、拖拽、展开/折叠）
- ✅ PDF 输出支持
- ✅ 暗色主题支持
- ✅ 完美中文支持
