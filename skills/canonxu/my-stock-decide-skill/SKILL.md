# My Stock Decide Skill (美股交易决策技能)

## 1. 核心规则与触发条件
- **触发条件**：仅在用户要求“做股票或交易决策”，并且“提供持仓列表（通过文本或图片）”的情况下触发。
- **决策范围**：仅限于**当前持仓标的**，以及通过 `my_stock_watchlist_skill` 等途径获取的**观察仓列表**标的。

## 2. 仓位管理与分类体系
做好整体仓位控制，最终**总持仓不得超过总资产的 90%**。资产需分为以下三层梯队：
- **T1 核心 (Core)**：压舱石，精选优质蓝筹。建议仓位：~30%
- **T2 趋势 (Growth)**：趋势增长，聚焦高增速、高Beta标的，捕捉起涨点动能。建议仓位：~20%
- **T3 卫星 (Satellite)**：卫星博弈，高弹性题材与事件驱动标的。建议仓位：~20%

## 3. 宏观压力测试打分引擎 (Macro Stress Test Score)
必须结合当前的短期宏观形势（如 CPI/通胀、利率预期/流动性、地缘政治/风险偏好等）进行分析并**打分（范围：-5 到 +5）**。

### 动态阈值与仓位调控规则：
- **总分 > 3（环境极好）**：
  总持仓上限可达 90%，允许 T2/T3 比例向上浮动（例如 T2 提升至 25%）。
- **总分 < -2（环境恶劣）**：
  进入防守模式，强制缩减 T3 仓位至 5%-10%，甚至将总持仓上限大幅降至 40%-50%。
- **其他分数**：
  维持默认的稳健配置比例（T1 30%, T2 20%, T3 20%）。

## 4. 工作流程

1. **环境解析**：
   - 识别用户提供的图片或文字中的当前持仓列表。
   - 联动 `my_stock_watchlist_skill` 获取观察仓的标的列表。
2. **宏观研判**：
   - 获取近期核心宏观数据（如最新的美国CPI数据、美联储利率预期/点阵图、重大地缘冲突或黑天鹅事件）。
   - 输出宏观压力测试分析并得出具体分数。
3. **标的决策**：
   - 对持仓和观察仓中的每个标的进行独立评估，结合宏观环境和个股基本面/技术面给出决策。

## 5. 最终输出规范
输出的最终决策**必须**涵盖决策范围内的每个标的，且每个标的的决策信息必须包含以下5个要素：
- **标的简码 (Ticker)**：如 AAPL, TSLA 等。
- **操作方向 (Action)**：BUY / SELL / HOLD （对于明确且强烈的动作，使用 STRONG BUY 或 STRONG SELL）。
- **操作情绪 (Sentiment)**：普通 / 强烈。
- **操作价格 (Target Price)**：给出明确的买入/卖出/止损目标价位区间。
- **操作原因 (Rationale)**：**必须充分详尽**，阐述为何在此宏观背景及个股结构下做出该方向的判定。
