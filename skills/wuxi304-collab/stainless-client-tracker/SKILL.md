# stainless-client-tracker - 不锈钢客户跟进管理系统

## 技能描述

专为不锈钢行业设计的客户跟进管理工具，追踪从初次接触到成交的全流程。

## 核心功能

### 1️⃣ 客户分类管理
- **行业细分**：封头/压力容器/造纸设备/化工设备/环保设备等
- **规模分级**：大型（年产值>1 亿）/中型（5000 万 -1 亿）/小型（<5000 万）
- **地域分布**：按省份/城市聚类
- **产品需求**：304/316L/2205/2507 等牌号偏好

### 2️⃣ 跟进阶段追踪
```
线索 → 初次接触 → 需求确认 → 报价 → 样品 → 谈判 → 成交 → 复购
```

每个阶段记录：
- 跟进日期
- 联系人/职位
- 沟通要点
- 下一步行动
- 预计成交金额

### 3️⃣ 拜访记录管理
- **拜访日期**：自动关联天气/交通
- **参与人员**：客户方 + 我方
- **讨论议题**：技术/价格/交期/服务
- **客户反馈**：正面/负面/中性
- **待办事项**：自动提醒

### 4️⃣ 商机漏斗分析
- **Pipeline 可视化**：各阶段客户数量/金额
- **转化率统计**：阶段间转化比例
- **预测分析**：基于历史数据的成交概率
- **业绩目标**：月度/季度/年度达成率

### 5️⃣ 自动提醒
- **跟进提醒**：超过 7 天未联系的客户
- **生日/节日**：客户关键日期关怀
- **合同到期**：续约/复购提醒
- **价格波动**：镍价/钢价大幅变动时通知重点客户

### 6️⃣ 报表生成
- **周报**：本周跟进客户/新增商机/预计成交
- **月报**：业绩达成/客户增长/问题分析
- **季度复盘**：成功/失败案例总结

## 使用示例

```javascript
// 新增客户
await tracker.addClient({
  名称："万福压力容器",
  行业："压力容器",
  地址："江苏省扬州市广陵区泰安镇"
});

// 记录跟进
await tracker.logFollowUp({
  客户 ID："CLI-2026-002",
  日期："2026-03-26",
  方式："电话",
  要点：["确认技术参数", "约定下周拜访"]
});

// 生成周报
const report = await tracker.generateWeeklyReport({
  日期范围：["2026-03-24", "2026-03-30"]
});
```

## 集成能力

- **飞书集成**：自动同步客户信息到飞书多维表格
- **微信集成**：客户生日/节日自动发送祝福
- **邮件集成**：报价单/合同自动发送
- **日历集成**：拜访行程自动添加到日历
