# stainless-chain-analyzer - 不锈钢产业链分析工具

## 技能描述

深度分析不锈钢产业链上下游关系，识别商机和风险评估。

## 核心功能

### 1️⃣ 产业链图谱
```
上游                    中游                    下游
─────────────────────────────────────────────────────
镍矿 → 镍铁 → 不锈钢坯 → 热轧/冷轧 → 制品加工 → 终端应用
铬矿 → 铬铁              ↓
废钢回收              贸易商/加工厂           封头/压力容器/化工设备
                                                    造纸设备/环保设备
                                                    食品机械/医疗器械
```

### 2️⃣ 客户关联度评级
- **直接消耗**：封头厂/压力容器厂（板材直接加工）→ ⭐⭐⭐⭐⭐
- **间接消耗**：化工设备厂（采购半成品）→ ⭐⭐⭐⭐
- **终端用户**：造纸厂/食品厂（采购成品设备）→ ⭐⭐⭐

### 3️⃣ 产品匹配分析
根据客户类型推荐不锈钢牌号：

| 客户类型 | 推荐牌号 | 理由 |
|----------|----------|------|
| 封头制造 | 304/316L | 通用性强，需求量大 |
| 压力容器 | 304/316L/2205 | 需耐压耐腐蚀 |
| 造纸设备 | 316L/2507 | 耐酸碱腐蚀 |
| 化工设备 | 316L/2205/2507 | 强腐蚀环境 |
| 食品机械 | 304/316L | 食品级要求 |
| 医疗器械 | 316L/317L | 医用级要求 |

### 4️⃣ 价格趋势分析
- **镍价追踪**：LME 镍价/沪镍主力
- **钢价指数**：无锡/佛山市场 304/316L 价格
- **成本测算**：基于镍价推算不锈钢成本
- **报价建议**：根据成本 + 合理利润给出报价区间

### 5️⃣ 竞品分析
- **主要钢厂**：太钢/宝新/张浦/青山等
- **产品特点**：各钢厂优势牌号/规格
- **价格对比**：市场主流报价
- **交期对比**：钢厂排产情况

### 6️⃣ 风险评估
- **客户风险**：工商异常/诉讼/失信记录
- **行业风险**：产能过剩/环保限产/出口限制
- **价格风险**：镍价波动/汇率变化
- **账期风险**：客户付款记录/信用评级

## 分析方法

### 关联度评分模型
```
关联度 = (直接消耗系数 × 0.4) + (采购规模 × 0.3) + (合作潜力 × 0.3)

直接消耗系数:
- 封头/压力容器厂：1.0
- 化工/造纸设备厂：0.7
- 终端用户：0.4

采购规模:
- >1000 吨/年：1.0
- 500-1000 吨/年：0.7
- 100-500 吨/年：0.4
- <100 吨/年：0.2

合作潜力:
- 已合作且增长：1.0
- 已合作稳定：0.7
- 新接触有意向：0.5
- 初步接触：0.3
```

### 商机识别规则
1. **扩产信号**：客户新厂房/新设备招标
2. **替代机会**：客户使用竞品但满意度低
3. **新品需求**：客户开发新产品需要特殊材质
4. **供应链风险**：客户现有供应商出现问题

  年采购量："500 吨"
