# stainless-quote-generator - 不锈钢快速报价单生成器

## 技能描述

为不锈钢销售场景快速生成专业报价单，含成本测算、运费计算、利润分析。

## 核心功能

### 1️⃣ 成本测算
```
成本 = 材料成本 + 加工费 + 运费 + 其他费用

材料成本 = 钢厂基价 + 材质差价 + 规格差价 + 表面差价
加工费 = 开平费 + 分条费 + 切割费 + 表面处理费
运费 = 干线运输 + 同城配送 + 装卸费
其他 = 包装费 + 保险费 + 资金成本
```

### 2️⃣ 价格数据库
- **钢厂基价**：太钢/宝新/张浦/青山主流牌号
- **材质差价**：304/316L/2205/2507 价差
- **规格差价**：不同厚度/宽度加价
- **表面差价**：2B/BA/No.1/拉丝/镜面

### 3️⃣ 运费计算
- **干线运输**：按距离/重量/车型计算
- **同城配送**：无锡/佛山/天津等主要市场
- **快递小包**：样品快递费用

### 4️⃣ 利润分析
- **毛利率**：(报价 - 成本) / 报价
- **吨钢利润**：元/吨
- **总利润**：元/单
- **底价提醒**：低于成本价时预警

### 5️⃣ 报价单模板
支持多种格式：
- **PDF**：正式报价单（盖章版）
- **Excel**：可编辑版本
- **图片**：微信快速发送
- **飞书文档**：在线协作

### 6️⃣ 报价历史
- **客户报价记录**：历史报价/成交价
- **价格趋势**：同一客户价格变化
- **竞品对比**：市场均价参考

## 使用示例

```javascript
// 生成报价单
const quote = await generator.createQuote({
  客户：{
    名称："新乡七里营封头厂",
    联系人："张总",
    电话："138****1234",
    地址："河南省新乡市新乡县七里营镇"
  },
  产品：[
    { 材质："304", 规格："2.0×1219×C", 表面："2B", 数量：5, 单位："吨" },
    { 材质："316L", 规格："2.0×1219×C", 表面："2B", 数量：2, 单位："吨" }
  ],
  发货地："无锡",
  目的地："新乡",
  毛利率目标：0.05
});

// 输出格式：pdf | excel | image | doc
await quote.export('pdf');
```
