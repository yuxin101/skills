# Short Video Template

Use for Shorts / Reels / TikTok / X video.

## Length
- 30–60 seconds default
- 60–90 seconds when the idea needs more explanation

## Structure

### Hook (0–5s)
Choose one:
- “This OpenClaw video is great, but here’s what beginners actually need.”
- “Too long, didn’t watch? Here’s the part that matters.”
- “Most people miss this point when they watch OpenClaw tutorials.”

### Insight 1 (5–20s)
Explain the first key idea in plain English.

### Insight 2 (20–35s)
Explain the most practical implication.

### Insight 3 / reframing (optional) (35–45s)
Add beginner framing or setup-friction context.

### ClawLite bridge (last 10–15s)
Use a soft bridge:
- “If you want to try this without the setup pain, start with ClawLite.”
- “This is where ClawLite helps beginners get to the first working agent faster.”

### CTA
- “Try ClawLite.”
- “Start with ClawLite.”
- “Get your first agent running faster with ClawLite.”

## Style rules
- educational first
- product second
- do not sound like a hard ad in the first half
- keep short sentences for subtitles
- prefer 1 idea per sentence
