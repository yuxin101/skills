# Content Pack Template

Generate these assets from one source note.

## 1. X Thread
Structure:
- hook
- source context
- 3 takeaways
- beginner framing
- ClawLite bridge
- source attribution

## 2. LinkedIn / Facebook post
Structure:
- what the source video is about
- why it matters
- practical takeaway
- ClawLite bridge
- source attribution

## 3. Short blog summary
Structure:
- title
- intro
- 3 lessons
- beginner takeaway
- ClawLite bridge
- source attribution

## 4. Optional newsletter blurb
Structure:
- one paragraph summary
- 1 key lesson
- 1 action item
- 1 CTA

## Packaging rule
All outputs should come from the same structured source note to keep messaging consistent.
