# Angle Framework

Pick one primary angle per source video.

## 1. Beginner translation
Use when the source is strong but too dense/technical.

Prompt shape:
- “This is what beginners actually need to know.”

## 2. 3 takeaways
Use when the source has several useful insights.

Prompt shape:
- “3 useful takeaways from this OpenClaw video.”

## 3. Practical summary
Use when the source is long and users need a concise operational version.

Prompt shape:
- “Too long, didn’t watch? Here’s the practical version.”

## 4. Setup-friction bridge
Use when the source concept is valid but real-world onboarding is the blocker.

Prompt shape:
- “The idea is strong. The real blocker is setup friction.”

## 5. Why this matters
Use when the source introduces a concept people underestimate.

Prompt shape:
- “Why this OpenClaw lesson matters more than it looks.”

## Selection rule
Default order:
1. beginner translation
2. practical summary
3. setup-friction bridge
4. 3 takeaways
5. why this matters
