# Asset Promotion Guide

Use this guide after building a JK marketing asset from a source video.

## Purpose
A source-specific asset folder is useful, but it is not enough.
To make the system compound over time, promote reusable lines into the shared `marketing-assets` layer.

## Promotion rule
- `video-content/<videoId>/...` = source-specific working pack
- `marketing-assets/inbox/...` = intermediate structured note
- `marketing-assets/<category>/...` = durable shared substrate

Do not stop at the inbox layer when the source contains reusable value.

## Typical destinations

### Pain points
Promote to:
- `02-pain-points/`

Good candidates:
- repeated user pain
- activation blockers
- misconceptions that affect conversion

### Hooks
Promote to:
- `01-hooks/`

Good candidates:
- short sharp lines
- thread openers
- beginner-oriented reframes

### Angles
Promote to:
- `06-angles/`

Good candidates:
- positioning frames
- reframing logic
- why-this-matters lines
- setup-friction / onboarding bridges

### Proof points / source links
Promote to:
- `03-proof-points/`

Good candidates:
- source-backed claims
- reusable attribution lines
- links that support positioning

### CTA
Promote to:
- `07-cta/`

Good candidates:
- source-aligned bridge lines
- non-spammy product transitions

## Minimum extraction checklist
At minimum, extract these from every strong source:
- 2–5 reusable pain lines
- 2–5 hooks
- 1–3 angle lines
- 1–3 proof/source lines

## Example
If a YouTube source reinforces setup friction as the main beginner blocker:
- add reusable pain language to `02-pain-points/setup-friction.md`
- add strong setup-friction openers to `01-hooks/x-hooks.md`
- add “first useful workflow” framing to `06-angles/one-click-install.md`
- add the source URL to `03-proof-points/source-links.md`

## Goal
The purpose of promotion is to make each source strengthen the whole marketing system.
The source should not stay trapped in one campaign pack forever.
