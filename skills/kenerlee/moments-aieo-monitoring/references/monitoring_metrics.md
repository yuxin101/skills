# AIEO 监控指标详细定义

## 一、可见性指标 (Visibility Metrics)

### V-1 AI提及率 (AI Mention Rate)

**定义**: 在测试问题中，品牌被 AI 提及的问题比例

**计算公式**:
```
AI提及率 = 品牌被提及的问题数 / 测试问题总数 × 100%
```

**判断标准**:
- ✅ 提及: AI 回答中出现品牌名称
- ❌ 未提及: AI 回答中完全未出现品牌名称

**评估等级**:
| 得分 | 评级 | 建议 |
|------|------|------|
| 0-20% | 🔴 危险 | 紧急启动AIEO优化 |
| 21-50% | 🟠 待改进 | 需要系统性内容布局 |
| 51-70% | 🟡 一般 | 有基础，加强重点场景 |
| 71-90% | 🟢 良好 | 保持并持续优化 |
| 91-100% | ⭐ 优秀 | 行业领先，维护为主 |

---

### V-2 AI首推率 (AI First Recommendation Rate)

**定义**: 在测试问题中，品牌被 AI 首先/优先推荐的比例

**计算公式**:
```
AI首推率 = 品牌被首推的问题数 / 测试问题总数 × 100%
```

**判断标准**:
- ✅ 首推: 品牌在 AI 回答中首个被提及，或被明确推荐为"首选"/"最佳"
- ⚠️ 提及: 品牌被提及但非首选
- ❌ 未提及: 未出现

**评估等级**:
| 得分 | 评级 | 说明 |
|------|------|------|
| 0-10% | 🔴 危险 | 品牌几乎不被优先推荐 |
| 11-25% | 🟠 待改进 | 偶尔被首推 |
| 26-40% | 🟡 一般 | 部分场景有首推优势 |
| 41-60% | 🟢 良好 | 多数核心场景首推 |
| 61%+ | ⭐ 优秀 | 行业首选品牌 |

---

### V-3 平台覆盖率 (Platform Coverage Rate)

**定义**: 品牌在各 AI 平台的覆盖情况

**计算公式**:
```
平台覆盖率 = 有提及的平台数 / 测试平台总数 × 100%
```

**评估等级**:
| 覆盖率 | 评级 | 说明 |
|--------|------|------|
| 1-2/6 | 🔴 低 | 仅少数平台有存在 |
| 3-4/6 | 🟡 中 | 部分平台有覆盖 |
| 5-6/6 | 🟢 高 | 全平台覆盖 |

---

### V-4 竞品提及率 (Competitor Mention Rate)

**定义**: 在同一问题中，竞品被同时提及的比例（用于竞争分析）

**计算公式**:
```
竞品提及率 = 竞品被提及的问题数 / 测试问题总数 × 100%
```

**用途**:
- 监测竞品 AI 可见性变化
- 评估相对竞争位置
- 发现竞品优化动作

---

## 二、质量指标 (Quality Metrics)

### Q-1 内容准确率 (Content Accuracy Rate)

**定义**: AI 回答中品牌相关信息的准确性

**评估维度**:
| 维度 | 检查内容 |
|------|---------|
| 基础信息 | 品牌名称、归属、成立时间 |
| 产品信息 | 产品名称、特点、价格区间 |
| 数据信息 | 市场份额、排名、获奖 |
| 历史信息 | 发展历程、重大事件 |

**评分标准**:
```
准确率 = 准确信息点数 / 总信息点数 × 100%
```

**评估等级**:
| 准确率 | 评级 | 行动 |
|--------|------|------|
| <70% | 🔴 严重 | 紧急修正错误信息 |
| 70-85% | 🟡 待改进 | 补充/修正关键信息 |
| 86-95% | 🟢 良好 | 持续优化细节 |
| >95% | ⭐ 优秀 | 维护现状 |

---

### Q-2 定位一致率 (Positioning Consistency Rate)

**定义**: AI 对品牌的描述与目标定位声明的一致程度

**评估方式**:

1. 对照定位声明的核心要素:
   - 品类归属
   - 差异化价值
   - 目标人群
   - 核心卖点

2. 逐项检查 AI 回答是否体现

**评分标准**:
```
定位一致率 = 体现的定位要素数 / 总定位要素数 × 100%
```

**评估等级**:
| 一致率 | 评级 | 说明 |
|--------|------|------|
| <50% | 🔴 不一致 | AI认知与目标定位严重偏离 |
| 50-70% | 🟡 部分一致 | 核心定位未完全传达 |
| 71-85% | 🟢 基本一致 | 主要定位已被AI认知 |
| >85% | ⭐ 高度一致 | 定位成功植入AI |

---

### Q-3 情感正面率 (Positive Sentiment Rate)

**定义**: AI 回答中对品牌描述的情感倾向

**判断标准**:
| 情感 | 标识 | 示例 |
|------|------|------|
| 正面 | 😊 | "优秀"、"领先"、"推荐"、"高品质" |
| 中性 | 😐 | 客观描述，无明显倾向 |
| 负面 | 😞 | "问题"、"争议"、"不如"、"缺点" |

**计算公式**:
```
情感正面率 = (正面回答数 + 中性回答数×0.5) / 总回答数 × 100%
```

**评估等级**:
| 正面率 | 评级 | 行动 |
|--------|------|------|
| <50% | 🔴 负面 | 紧急处理负面信息 |
| 50-70% | 🟡 中性 | 增加正面内容输入 |
| 71-85% | 🟢 正面 | 保持并强化 |
| >85% | ⭐ 高度正面 | 维护品牌形象 |

---

### Q-4 信息完整率 (Information Completeness Rate)

**定义**: AI 回答对品牌核心信息点的覆盖程度

**核心信息点清单**（根据定位报告确定）:
```
示例（百威）:
□ 品牌定位（高端啤酒）
□ 历史传承（149年/1876年）
□ 核心工艺（榉木酿造）
□ 市场地位（全球销量领先）
□ 适用场景（商务宴请、聚会）
```

**计算公式**:
```
信息完整率 = 被AI提及的核心信息点数 / 总核心信息点数 × 100%
```

## 三、业务指标 (Business Metrics)

### B-1 品牌搜索量 (Brand Search Volume)

**数据来源**:
- Google Trends（国际）
- 百度指数（国内）
- 微信指数（社交）

**监测方式**:
```
1. 记录基准期搜索指数
2. 每周记录当期搜索指数
3. 计算环比变化率
4. 分析与AIEO动作的相关性
```

**关联分析**:
| 现象 | 可能原因 |
|------|---------|
| 搜索量上升 | AIEO生效，用户主动搜索品牌 |
| 搜索量下降 | 需检查AI回答是否有负面 |
| 搜索量平稳 | AIEO可能尚未产生明显影响 |

---

### B-2 官网流量 (Website Traffic)

**数据来源**:
- GA4（Google Analytics 4）
- 百度统计

**关键指标**:
| 指标 | 说明 |
|------|------|
| 总访问量 | 整体流量变化 |
| 自然搜索流量 | 可能包含AI驱动的搜索 |
| 直接访问流量 | 用户主动输入URL |
| 品牌词流量 | 搜索品牌词进入 |

---

### B-3 AI来源流量 (AI-Referred Traffic)

**追踪方法**:

1. **UTM 标记法**（适用于可控内容）:
```
在AI可能引用的内容中嵌入UTM参数
?utm_source=ai&utm_medium=aieo&utm_campaign=monitoring
```

2. **Referrer 分析法**:
```
GA4 中筛选来自 AI 平台域名的流量:
- chat.openai.com
- perplexity.ai
- doubao.com
等
```

3. **行为推断法**:
```
分析用户行为特征:
- 问答式搜索词进入
- 品牌词搜索后直接进入
- 特定FAQ页面访问
```

---

### B-4 品牌词转化 (Brand Search Conversion)

**数据来源**:
- 电商后台（天猫、京东）
- CRM 系统
- 广告后台

**监测指标**:
| 指标 | 说明 |
|------|------|
| 品牌词搜索量 | 电商平台内品牌词搜索 |
| 品牌词转化率 | 搜索→购买转化 |
| 品牌词GMV | 品牌词带来的销售额 |

## 四、综合评分模型

### AIEO 健康度评分

**权重分配**:
| 指标 | 权重 | 说明 |
|------|------|------|
| AI提及率 (V-1) | 25% | 基础可见性 |
| AI首推率 (V-2) | 20% | 竞争优势 |
| 定位一致率 (Q-2) | 20% | 定位传达效果 |
| 情感正面率 (Q-3) | 15% | 品牌形象 |
| 内容准确率 (Q-1) | 10% | 信息质量 |
| 信息完整率 (Q-4) | 10% | 内容覆盖 |

**综合评分计算**:
```
AIEO健康度 = Σ(各指标得分 × 权重)
```

**评级标准**:
| 综合得分 | 评级 | 状态 |
|---------|------|------|
| 0-40 | 🔴 D级 | 需紧急优化 |
| 41-60 | 🟠 C级 | 需系统优化 |
| 61-75 | 🟡 B级 | 基础良好 |
| 76-90 | 🟢 A级 | 表现优秀 |
| 91-100 | ⭐ S级 | 行业领先 |

---

## 五、趋势分析指标

### 环比变化率

**计算公式**:
```
环比变化率 = (本期值 - 上期值) / 上期值 × 100%
```

### 趋势判断

| 变化率 | 趋势符号 | 说明 |
|--------|---------|------|
| >+10% | ↑↑ | 显著上升 |
| +5%~+10% | ↑ | 上升 |
| -5%~+5% | → | 稳定 |
| -10%~-5% | ↓ | 下降 |
| <-10% | ↓↓ | 显著下降 |

### 预警机制

| 情况 | 预警级别 | 建议行动 |
|------|---------|---------|
| 提及率连续两周下降 | 🟡 关注 | 分析原因，准备内容 |
| 首推率下降超过10% | 🟠 警告 | 检查竞品动态，加强内容 |
| 出现负面情感 | 🔴 紧急 | 立即排查，制定应对方案 |
| 信息错误被引用 | 🔴 紧急 | 紧急发布修正内容 |
