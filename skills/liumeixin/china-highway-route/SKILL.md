# 中国高速路线规划 Skill

智能规划驾车路线，充分利用各地绕城高速免费政策为车主节省过路费。

## 功能特性

- 🔍 起点到终点路线规划
- 💰 智能计算高速费用（利用绕城免费）
- 🚗 限行政策查询（目的地限行提醒 + 用户车牌限号查询）
- 🔢 限号查询（用户车牌在目的地的今明两天是否限号）
- 🎯 推荐最优路线（利用绕城免费 + 避开限行）
- 📱 生成高德地图导航链接

## 触发条件

用户表达以下意图时：
- 询问从 A 地到 B 地的驾车路线
- 询问高速费用、过路费
- 包含"路线"、"怎么走"、"多少高速费"等关键词

## 使用流程

### 第一步：确认车牌信息

**首次使用必须询问用户**：
- "请问你的车牌是哪个城市的？"

如果用户未提供车牌，按以下优先级处理：
1. 直接询问车牌所在地
2. 如已知用户默认车牌（郑州/洛阳等），使用已知信息并告知用户

### 第二步：分析路线

1. 获取起点、终点的经纬度坐标
2. 查询高速路线方案
3. 分析是否可以利用绕城高速免费政策
4. 计算费用对比

### 第三步：输出方案

输出内容包括：
- 路线分段详情（距离、费用）
- 高速费总计
- 节省金额
- 如有优化方案，说明节省多少

## 规划思路

### 核心原则

1. **查询车牌所在地**：先确认用户车牌
2. **查找绕城高速**：确认该城市是否有绕城免费政策
3. **设计换乘方案**：在绕城高速两端下高速再上，利用免费段
4. **费用对比**：对比全程高速 vs 优化方案

### 政策缓存机制

为提高效率，使用政策缓存机制：

1. **首次搜索**：规划路线时，必须搜索途径城市的高速免费政策
2. **记录日期**：将搜索日期（YYYY-MM-DD）和搜索结论保存到本地文件
3. **缓存有效期**：下次再用时，对比当前日期与缓存日期：
   - 超过1个月：重新搜索最新政策
   - 不超过1个月：直接使用缓存结论

### 限行政策查询

规划路线时，还需要查询目的地的限行政策：

1. **获取车牌**：如果用户提供了车牌，直接使用
2. **查询限行**：搜索目的地城市的限行政策
3. **输出提醒**：告知用户限行规则

## 高德 API 调用

使用高德 Web 服务 API 进行路线规划：

### 地理编码
```
https://restapi.amap.com/v3/geocode/geo?address={地址}&key={API_KEY}
```

### 驾车路线
```
https://restapi.amap.com/v3/direction/driving?origin={经度,纬度}&destination={经度,纬度}&key={API_KEY}
```

### 途径点路线
```
https://restapi.amap.com/v3/direction/driving?origin={起点}&destination={终点}&waypoints={途径点}&key={API_KEY}
```

## 配置

如需配置高德 API Key：
- 环境变量：`AMAP_WEBSERVICE_KEY`
- 或直接在代码中使用

## 注意事项

- 绕城免费政策可能有变化，建议以最新官方信息为准
- 免费政策通常只针对本地车牌，外地车牌不享受
- 规划时优先考虑用户体验，不仅看费用
