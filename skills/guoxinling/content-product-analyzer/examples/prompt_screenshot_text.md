# Example Prompt (Screenshot + Pasted Text, User-only)

我不希望你联网。请只基于我提供的内容做拆解（User-provided only）。

我把一个 landing page 的文案贴给你：
- 标题：xxxx
- 副标题：xxxx
- 亮点 bullet：1) ... 2) ... 3) ...
- 定价：Free / Pro $xx/mo / Business $xx/mo
- 社会证明：xx users, xx reviews, SOC2, GDPR

同时我会上传 2 张截图（包含 hero、pricing、testimonials）。

请输出：
- Structured summary
- Detailed teardown
- 明确列出：Observed facts / Supported inferences / Unknowns
- 最后给 5 条最重要的改版建议（按“提升转化”的优先级排序）
