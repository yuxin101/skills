# 政治评论员 - AI 技能

一个专为 OpenClaw AI 助手设计的智能政治新闻评论技能。帮助用户收集热门政治新闻、分析时事，并生成多格式的评论内容。

## 🌟 功能特色

### 阶段一：新闻收集
- 从多个来源抓取热门政治新闻（台湾、中国、国际）
- 支持 20+ 新闻来源，包括中央社、BBC、CNN、路透社、新华社等
- 智能过滤与去重
- 按热度排序

### 阶段二：兴趣选择
- 互动式选择界面
- 按标签筛选：#两岸 #选举 #外交 #经济
- 关键词搜索
- 用户偏好记忆

### 阶段三：深度分析
- 5W1H 分析框架
- 多角度观点分析
- 利益相关方分析
- 趋势预测（短/中/长期）

### 阶段四：内容输出
- 📄 完整评论文章（1500-3000字）
- 🎙️ 视频/播客脚本
- 📱 社媒帖文（Twitter/IG/Facebook）
- 📊 可视化分析卡片

### 评论风格
- ⚖️ **中立客观型** — 陈述事实，呈现多方观点
- 🔍 **批判分析型** — 带有批判视角的深度分析
- 😂 **讽刺幽默型** — 以幽默讽刺方式评论
- 📚 **深度学术型** — 学术化、引用理论框架

## 🚀 使用方式

### 触发指令
- 「帮我评论今天的政治新闻」
- 「最近两岸有什么大事？」
- 「我想写一篇关于中美关系的文章」
- 「帮我分析这个新闻」+ 贴上链接

### 工作流程
1. AI 展示今日新闻列表
2. 用户选择感兴趣的新闻
3. AI 进行深度分析
4. AI 生成评论内容
5. 用户可修改风格/格式后重新生成

## 📝 输出格式说明

### 完整文章
```
标题：[角度] + [核心事件] + [吸引力标题]

结构：
- 导言（150-300字）
- 背景（300-500字）
- 多方观点（400-600字）
- 深度分析（500-800字）
- 趋势预测（300-500字）
- 结论（150-300字）
```

### 社媒帖文
```
开场钩子（20-40字）
↓
核心论点（50-80字）
↓
支撑论据（50-100字）
↓
互动引导（20-30字）
```

### 视频脚本
```
开场钩子（0-30秒）→ 背景（1-3分钟）→
多方观点（2-4分钟）→ 深度分析（3-5分钟）→
总结（1-2分钟）→ 行动呼吁（30-60秒）
```
