# 社群短評模板

## Twitter/X 格式（280字內）

### 結構
```
[犀利觀點/ hook]（20-40字）
↓
[核心論點]（50-80字）
↓
[支撐論據]（50-100字）
↓
[互動引導/ call to action]（20-30字）
```

### 範例

**時事評論型：**
```
今天蔡英文與麥卡錫會面的消息傳出，我的看法是：

這不是單純的「過境外交」，而是美中台三方在晶片戰略下的最新棋子。

你覺得這對兩岸關係意味著什麼？👇
```

**數據驅動型：**
```
最新民調顯示：67% 民眾對政府不滿意。

但問題不在於數字本身，而在於——
當不滿累積到臨界點，會以什麼形式爆發？

#台灣政治 #時事分析
```

**對比諷刺型：**
```
北京：「台灣是中國不可分割的一部分」
現實：70年來雙方政治制度、經濟模式、人民生活差距越來越大

統一？說的是哪個平行宇宙的統一？
```

**自問自答型：**
```
問：為什麼台灣年輕人越來越「天然獨」？
答：不是因為課本，而是因為抖音、小紅書、淘寶。
中國的「銳實力」正在反向塑造台灣青年對中國的認同。
```

## Instagram / Facebook 格式

### 結構
```
[吸睛標題/問題]
[1-2句背景]
[3個重點（用 bullet points）]
[引導互動]
[Hashtags]
```

### 範例
```
台灣是否應該恢復徵兵制？

背景：近年兩岸關係緊張，兵役議題再次浮上檯面。

✅ 支持方觀點：
• 強化國防有助嚇阻
• 年輕人需有基本戰技

❌ 反對方觀點：
• 影響就業與學業
• 微型志願役更有效率

你支持哪一邊？留言告訴我 👇

#台灣國防 #兵役改革 #兩岸議題
```

## Threads / 微博格式

### 特點
- 更短的段落
- 更直接的觀點
- 可用 emoji 增加活潑性
- 適合跟風熱點

### 範例
```
昨天那場記者會，我只記住一個畫面：

XXX 站在台上，面對記者的尖銳提問，他的眼神飄向了 XXX。

那一刻，我讀到的是：_____ 🔍

#時事 #觀察
```
