# 完整評論文章模板

## 標題格式
```
[立場/角度] + [核心事件] + [吸引力標題]

示例：
「從中美博弈看台灣在半導體產業的戰略抉擇」
「蔡英文過境美國：一次「低調」卻「高調」的外交突破」
```

---

## 結構模板

### 導言（150-300字）
```
[吸睛開場：一個場景/問題/數據]
[過渡到核心議題]
[本文觀點預告]
```

### 正文（3-5段落）

#### 第一段：事件背景（300-500字）
```
[事件發生經過]
[關鍵人物/機構]
[為什麼重要]
```

#### 第二段：多方觀點呈現（400-600字）
```
[官方/主流立場]
[反對/批評聲音]
[專家/學者觀點]
[國際反應]
```

#### 第三段：深度分析（500-800字）
```
[利益博弈分析]
[結構性因素]
[歷史脈絡對照]
```

#### 第四段：趨勢預測（300-500字）
```
[短期走向]
[中期影響]
[長期展望]
```

### 結論（150-300字）
```
[核心觀點總結]
[對讀者的呼籲/思考]
[預告後續發展]
```

---

## 標題範例

### 吸睛型
- 「當習近平和拜登坐下來談判時，桌下藏著什麼？」
- 「一個數字看懂台灣選舉經費」

### 疑問型
- 「ECFA終結後，台灣經濟還剩什麼？」
- 「美國會出兵保衛台灣嗎？」

### 對比型
- 「從2019到2024：香港劇變的五年」
- 「蔡英文 vs 馬英九：兩岸政策的十字路口」

### 數據型
- 「65%支持統一？民調數字背後的真相」
- 「一張圖看懂美中貿易戰全貌」
