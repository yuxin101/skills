# 口播稿模板

## 影片/播客腳本格式

### 時長建議
- 短影片（1-3分鐘）：800-1000字
- 中影片（5-10分鐘）：1500-2500字
- 長影片（15分鐘+）：4000字以上

---

## 開場結構

### Hook（0-30秒）
```
[一個吸引注意力的陳述/問題/數據]

範例：
「你知道嗎？過去一個月，台灣周邊海域的外國軍機出現次數增加了300%。」
「今天我要告訴你一個連很多台灣人都不知道的事實——」
```

### 自我介紹（可選，0-30秒）
```
嗨，我是XXX，今天要來聊聊_____。

如果你對_____感興趣，訂閱我們，別忘了開啟小鈴鐺。
```

### 過渡（30-60秒）
```
「好，讓我們直接進入今天的主題——_____。」
```

---

## 正文結構

### 第一部分：背景交代（1-3分鐘）
```
讓我們先把時間倒回_____。

[事件背景]
[關鍵人物]
[為什麼重要]

這就是這次事件的起因。
```

### 第二部分：多方觀點（2-4分鐘）
```
那麼，針對這件事，不同陣營是怎麼看的呢？

首先，_____的立場是_____
其次，_____認為_____
最後，國際社會_____ 

這裡有一個很有趣的對比——
```

### 第三部分：深度分析（3-5分鐘）
```
但我今天真正想聊的，不是表面的口水仗，
而是這件事背後，真正在運作的邏輯。

我認為，有三個關鍵點——
第一，_____
第二，_____
第三，_____
```

---

## 結尾結構

### 總結（1-2分鐘）
```
好，今天聊了這麼多，讓我們快速回顧一下：
- _____
- _____
- _____

歸結到最後，這件事告訴我們_____
```

### Call to Action（30-60秒）
```
如果你認同今天的分析，請幫我按個讚。
如果你有不同的看法，歡迎在留言區告訴我，我們可以討論。

我是XXX，我們下次見。
```

---

## 口播技巧

### 語速
- 標準語速：每分鐘150-180字
- 重要內容：適度放慢
- 數據列舉：稍快但清晰

### 停頓
- 每個段落結束：停2-3秒
- 重要觀點後：停1-2秒
- 製造懸念時：停3-5秒

### 強調
- 關鍵詞加重語氣
- 數據明確說出
- 對比用「不是...而是...」
