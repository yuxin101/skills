# 政治評論員 - AI 技能

一個專為 OpenClaw AI 助手設計的智能政治新聞評論技能。幫助用戶收集熱門政治新聞、分析時事，並生成多格式的評論內容。

## 🌟 功能特色

### 階段一：新聞收集
- 從多個來源抓取熱門政治新聞（台灣、中國、國際）
- 支援 20+ 新聞來源，包括中央社、BBC、CNN、路透社、新華社等
- 智能過濾與去重
- 依熱度排序

### 階段二：興趣選擇
- 互動式選擇介面
- 依標籤篩選：#兩岸 #選舉 #外交 #經濟
- 關鍵字搜尋
- 用戶偏好記憶

### 階段三：深度分析
- 5W1H 分析框架
- 多角度觀點分析
- 利益相關方分析
- 趨勢預測（短/中/長期）

### 階段四：內容輸出
- 📄 完整評論文章（1500-3000字）
- 🎙️ 影片/播客腳本
- 📱 社群貼文（Twitter/IG/Facebook）
- 📊 視覺化分析圖卡

### 評論風格
- ⚖️ **中立客觀型** — 陳述事實，呈現多方觀點
- 🔍 **批判分析型** — 帶有批判視角的深度分析
- 😂 **諷刺幽默型** — 以幽默諷刺方式評論
- 📚 **深度學術型** — 學術化、引用理論框架

## 🚀 使用方式

### 觸發指令
- 「幫我評論今天的政治新聞」
- 「最近兩岸有什麼大事？」
- 「我想寫一篇關於中美關係的文章」
- 「幫我分析這個新聞」+ 貼上連結

### 工作流程
1. AI 展示今日新聞列表
2. 用戶選擇感興趣的新聞
3. AI 進行深度分析
4. AI 生成評論內容
5. 用戶可修改風格/格式後重新生成

## 📝 輸出格式說明

### 完整文章
```
標題：[角度] + [核心事件] + [吸引力標題]

結構：
- 導言（150-300字）
- 背景（300-500字）
- 多方觀點（400-600字）
- 深度分析（500-800字）
- 趨勢預測（300-500字）
- 結論（150-300字）
```

### 社群貼文
```
開場鉤子（20-40字）
↓
核心論點（50-80字）
↓
支撐論據（50-100字）
↓
互動引導（20-30字）
```

### 影片腳本
```
開場鉤子（0-30秒）→ 背景（1-3分鐘）→
多方觀點（2-4分鐘）→ 深度分析（3-5分鐘）→
總結（1-2分鐘）→ 行動呼籲（30-60秒）
```
