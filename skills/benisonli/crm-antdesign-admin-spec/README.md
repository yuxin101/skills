# crm-antdesign-admin-spec

A reusable skill for designing and generating B2B CRM/admin pages with an Ant Design-style system.

## What it helps with

This skill helps with:
- page taxonomy
- admin layout decisions
- token guidance
- component mapping
- page design constraints
- AI-generation prompt templates
- design review for CRM/admin pages

## Best-fit scenarios

- statistics dashboards
- AI analytics pages
- customer or operational list pages
- configuration / tool pages
- SaaS admin interfaces with Ant Design-like implementation goals

## Included files

- `SKILL.md`
- `references/admin-guidelines.md`
- `references/token-and-components.md`
- `references/prompt-templates.md`
- `references/examples.md`

## Design point of view

This skill assumes:
- admin pages should be structured and implementation-friendly
- not every page should look like a dashboard
- AI-enhanced pages still need admin realism
- consistency is more valuable than decorative novelty

## Not intended for

- marketing websites
- landing pages
- mobile-first product UI
- highly expressive art-direction work

## Status

Validated and packaged as a reusable skill.
