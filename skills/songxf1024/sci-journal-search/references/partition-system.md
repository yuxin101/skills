# 期刊分区体系参考

## 中科院分区（CAS Partition）

中国科学院文献情报中心期刊分区表，是中国常用的期刊评价体系。

### 分区标准

| 分区 | 说明 | 占比 |
|------|------|------|
| 1 区 | 顶级期刊 | 前 5% |
| 2 区 | 高水平期刊 | 6%-20% |
| 3 区 | 中等水平期刊 | 21%-50% |
| 4 区 | 一般期刊 | 51%-100% |

### 大类与小类

- **大类分区**：按学科门类划分（如医学、工程技术、理学等）
- **小类分区**：按具体学科方向划分（如肿瘤学、计算机科学与工程等）

## 新锐分区（XinRui Ranking）

新锐 XinRui Journal Ranking 是 2026 年推出的期刊评价体系。

### 分区标识

| 标识 | 对应中科院分区 | 说明 |
|------|----------------|------|
| T1 | 1 区 | 顶级期刊 |
| T2 | 2 区 | 高水平期刊 |
| T3 | 3 区 | 中等水平期刊 |
| T4 | 4 区 | 一般期刊 |

### Top 期刊标识

- **Top**：表示该期刊在其学科领域内具有较高学术影响力
- 通常对应 1 区和部分 2 区期刊

## JCR 分区（Journal Citation Reports）

科睿唯安（Clarivate）发布的期刊引证报告分区。

### Q 分区

| 分区 | 占比 |
|------|------|
| Q1 | 前 25% |
| Q2 | 25%-50% |
| Q3 | 50%-75% |
| Q4 | 75%-100% |

## 分区体系对比

| 特征 | 中科院分区 | 新锐分区 | JCR 分区 |
|------|------------|----------|----------|
| 发布机构 | 中科院文献情报中心 | 新锐学者 | 科睿唯安 |
| 分区数量 | 4 区 | 4 区 (T1-T4) | 4 区 (Q1-Q4) |
| Top 标识 | 有 | 有 | 无 |
| 更新频率 | 年度 | 年度 | 年度 |
| 主要用途 | 中国科研评价 | 综合学术评价 | 国际学术评价 |

## 使用建议

1. **国内评价**：优先参考中科院分区
2. **国际交流**：可同时提供 JCR 分区
3. **投稿选刊**：综合参考多个分区体系
4. **数据核实**：重要决策前请核实官方最新数据
