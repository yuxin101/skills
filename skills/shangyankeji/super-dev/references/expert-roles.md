# Super Dev 10 专家角色

## PM（产品经理）

- 需求分析与优先级排序
- PRD 文档生成
- 用户画像与场景梳理
- 验收标准定义

调用: `super-dev expert PM "你的问题"`

## ARCHITECT（架构师）

- 系统架构设计
- 技术栈选型与评估
- API 契约设计
- 模块边界划分
- 可扩展性规划

调用: `super-dev expert ARCHITECT "你的问题"`

## UI（视觉设计师）

- 视觉设计规范
- 设计系统（色彩、字体、间距）
- 组件库选型
- 页面布局蓝图

调用: `super-dev expert UI "你的问题"`

## UX（交互设计师）

- 用户体验设计
- 交互流程设计
- 信息架构
- 可用性优化

调用: `super-dev expert UX "你的问题"`

## SECURITY（安全专家）

- OWASP Top 10 审查
- 依赖漏洞扫描
- 认证授权方案
- 数据安全合规

调用: `super-dev expert SECURITY "你的问题"`

## CODE（代码专家）

- 代码实现与重构
- Code Review
- 性能优化
- 技术债务管理

调用: `super-dev expert CODE "你的问题"`

## DBA（数据库专家）

- 数据模型设计
- 迁移脚本生成
- 索引优化
- 查询性能调优

调用: `super-dev expert DBA "你的问题"`

## QA（测试专家）

- 测试策略制定
- 测试用例设计
- 覆盖率分析
- 质量门禁执行

调用: `super-dev expert QA "你的问题"`

## DEVOPS（运维专家）

- CI/CD 流水线设计
- 容器化与编排
- 监控告警策略
- 部署自动化

调用: `super-dev expert DEVOPS "你的问题"`

## RCA（根因分析专家）

- 故障排查与诊断
- 根因分析
- 性能瓶颈定位
- 修复方案建议

调用: `super-dev expert RCA "你的问题"`
