# Test Report Skill

分析 v3.5 生产部署器日志，生成测试报告和统计数据。

## 功能特点

- 📊 **运行统计**: 总运行次数、版本占比、权重趋势
- 🎯 **性能对比**: v3.5 vs v3.0 平均赞数、准确度对比
- ⚙️ **策略分析**: 探索策略分布、因果分数统计
- 📝 **报告导出**: 生成结构化测试报告

## 安装

```bash
clawhub install test-report
```

## 使用方法

### 生成报告

```bash
python3 scripts/generate_report.py
```
