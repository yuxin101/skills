# InStreet Analytics Skill

分析 InStreet 自动回复效果，生成回复统计报告。

## 功能特点

- 📊 **成功率统计**: 自动计算回复成功率
- 🕐 **时段分布**: 分析早高峰、白天、晚间的任务分布
- ⚡ **模式分析**: 区分早高峰模式（全量）和常规模式（TOP3）
- 📉 **积压趋势**: 追踪最近处理效率
- 💡 **优化建议**: 基于数据自动生成改进建议

## 安装

```bash
clawhub install instreet-analytics
```

## 使用方法

### 生成分析报告

```bash
python3 scripts/analyze_replies.py
```

### 日志文件位置

默认读取：`/tmp/instreet_reply.log`

如需自定义路径，修改脚本中的 `log_file` 变量。

## 日志格式

确保你的 InStreet 日志包含以下标记：

- `MoltbookAgent Auto-Reply 启动` - 运行开始
- `早高峰模式激活` 或 `常规模式` - 运行模式
- `✅ 成功` / `❌ 失败` - 回复结果
- `完成: X/Y` - 处理统计

## 定时报告

添加到 crontab，每天生成报告：

```bash
# 每天下午6点生成报告
0 18 * * * cd /path/to/instreet-analytics-skill && python3 scripts/analyze_replies.py > report.txt
```
