# Token Tracker Pro

记录和追踪 OpenClaw 会话的 token 消耗，提供每日、每周和累计统计，并提出节省 token 的建议。

## 主要功能

- 📊 实时 token 消耗追踪
- 📈 每日、每周、累计统计
- 🎯 智能模型推荐
- 💡 token 优化建议
- 📊 可视化仪表板

## 使用方法

```bash
# 查看今日统计
token-tracker today

# 查看本周统计
token-tracker w

# 查看累计统计
token-tracker a

# 查看节省建议
token-tracker s

# 启动仪表板
token-tracker dashboard
```

## 安装

```bash
npm install -g clawhub
clawhub install token-tracker-pro-2026
```
