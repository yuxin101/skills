# 决策思维框架参考手册

本文档包含 15 个经过验证的决策思维框架，按四个层级组织。在决策引导过程中，根据决策类型和阶段选择合适的框架组合使用。

---

## 第一层：元认知层（认清决策本身）

在深入分析之前，先帮助求助者理解这个决策的本质。

### 1. 可逆 vs 不可逆决策（Bezos Type 1/Type 2）

**核心思想**：Jeff Bezos 将决策分为两类：
- **Type 1（不可逆）**：单向门，走过去就回不来。需要深思熟虑、充分分析。例如：卖掉房子、放弃国籍。
- **Type 2（可逆）**：双向门，走过去不满意还能回来。应该快速决策，不要过度分析。例如：换一份工作（还能再换）、搬到另一个城市试住。

**引导问题**：
- "如果这个决定做错了，最坏的结果是什么？能挽回吗？"
- "这个决定是单向门还是双向门？"
- "你是否在用 Type 1 的谨慎对待一个 Type 2 的决策？"

**适用场景**：所有决策的起点。帮助校准投入的分析精力。

### 2. 第一性原理（First Principles Thinking）

**核心思想**：剥离社会期望、他人意见、惯性思维，回到最基本的事实和自身真实需求。不是"别人都这么做所以我也应该"，而是"我到底要什么，为什么"。

**引导问题**：
- "抛开所有人的看法，你内心真正想要的是什么？"
- "如果没有任何外在限制（钱、面子、家人期望），你会怎么选？"
- "你做这个决定是因为自己想要，还是因为觉得'应该'？"
- "这个选择背后，你真正在追求的是什么？（安全感？自由？成长？认可？）"

**适用场景**：当求助者明显受到外界压力或社会比较的影响时。是整个决策过程的根基。

## 第二层：探索层（挖出真实想法）

帮助求助者发现自己尚未意识到的偏好、假设和价值观。

### 3. 苏格拉底式提问（Socratic Questioning）

**核心思想**：不直接给答案，通过层层追问帮助对方自己发现真相。六类核心问题：
1. **澄清类**："你说的'稳定'具体指什么？"
2. **假设探究**："你为什么认为去大公司就一定更稳定？"
3. **证据追问**："有什么证据支持这个想法？"
4. **视角转换**："如果你最好的朋友面临同样的选择，你会怎么建议？"
5. **后果推演**："如果这个假设是错的，会怎样？"
6. **反思元问题**："你为什么会问这个问题？这背后你真正担心的是什么？"

**引导技巧**：
- 每次只问一个问题，等待回答后再深入
- 不要急于纠正，让对方自己发现矛盾
- 当对方说"我不知道"时，换个角度重新提问

**适用场景**：贯穿整个决策过程，尤其是求助者思路混乱或自相矛盾时。

### 4. 价值观排序法

**核心思想**：人在面临选择时往往同时追求多个目标，但目标之间存在冲突。通过强制排序，让求助者看清自己真正的优先级。

**常见人生价值维度**：
- 经济收入 / 财务安全
- 职业成长 / 学习机会
- 自由度 / 灵活性
- 家庭 / 亲密关系
- 地理位置 / 生活环境
- 社会认可 / 地位
- 健康 / 生活质量
- 使命感 / 意义感
- 社交圈 / 归属感
- 冒险 / 新鲜感

**引导方式**：
- "在这些维度中，如果只能保留 3 个，你会选哪 3 个？"
- "如果 A 和 B 只能选一个，你选哪个？"（两两比较法）
- "给这些维度打分（1-10），然后分配权重（总和 100%）"

**适用场景**：求助者说"两个都想要"或者无法取舍时。

### 5. 预设遗憾测试（Regret Minimization Framework）

**核心思想**：Jeff Bezos 用来做辞职创业决策的框架。想象自己 80 岁回顾人生，哪个选择会让你更后悔？

**引导问题**：
- "想象你 80 岁坐在摇椅上回忆人生，你会更后悔做了这件事，还是更后悔没做？"
- "10 年后的你，会怎么看今天的这个选择？"
- "你更害怕失败的痛苦，还是错过的遗憾？"

**适用场景**：在"安全选项"和"冒险选项"之间犹豫时特别有效。

### 6. 硬币测试

**核心思想**：抛一枚硬币，正面选 A，反面选 B。关键不是硬币的结果，而是硬币在空中旋转的那一刻，你心里希望它落在哪一面。

**引导方式**：
- "我们来做个小实验：假设正面选 A，反面选 B。硬币抛出去的那一瞬间，你心里希望是哪面？"
- "如果结果是 B，你是松了口气，还是有点失望？"

**适用场景**：理性分析势均力敌时，用来揭示潜意识偏好。通常作为辅助验证工具。

## 第三层：分析层（结构化评估）

将模糊的感觉转化为可比较的结构化信息。

### 7. 加权决策矩阵

**核心思想**：将决策标准和选项量化为一个评分表，每个标准赋予权重，每个选项打分，计算加权总分。

**操作步骤**：
1. 列出所有决策维度（来自价值观排序）
2. 为每个维度分配权重（总和 100%）
3. 对每个选项在每个维度上打分（1-10）
4. 计算加权总分 = Σ(权重 × 分数)

**示例**：

| 维度 | 权重 | 选项A: 北京 | 选项B: 成都 |
|------|------|-------------|-------------|
| 收入 | 30% | 9 | 6 |
| 生活成本 | 20% | 3 | 8 |
| 生活质量 | 25% | 5 | 8 |
| 职业发展 | 25% | 9 | 6 |
| **加权总分** | | **6.45** | **6.90** |

**注意**：数字不是最终答案，而是促进思考的工具。如果算出来的结果和直觉不符，说明权重或打分需要调整——这本身就是发现。

**适用场景**：有多个明确选项需要比较时。

### 8. SWOT 分析

**核心思想**：从四个维度全面评估每个选项：
- **S（Strengths 优势）**：这个选项本身的优点
- **W（Weaknesses 劣势）**：这个选项本身的缺点
- **O（Opportunities 机会）**：外部环境带来的正面可能
- **T（Threats 威胁）**：外部环境带来的负面风险

**引导问题**：
- "选择 A 最大的优势是什么？最大的劣势呢？"
- "这个选择在未来 3-5 年有什么机会？有什么潜在的风险？"
- "你的优势和这个机会匹配吗？"

**适用场景**：职业选择、创业方向、搬迁决策等需要评估内外部因素的场景。

### 9. 10/10/10 法则

**核心思想**：Suzy Welch 提出的时间透视法。分别从三个时间尺度审视决策：
- **10 分钟后**你会怎么想？（情绪反应）
- **10 个月后**你会怎么想？（中期影响）
- **10 年后**你会怎么想？（长期价值）

**引导问题**：
- "做完这个决定，10 分钟后你的感受是什么？"
- "10 个月后，这个决定对你的生活会产生什么影响？"
- "10 年后回看，这个决定重要吗？你会怎么评价？"

**适用场景**：被短期情绪驱动、或者过于焦虑长远的求助者。帮助校准时间视角。

### 10. 二阶思维（Second-Order Thinking）

**核心思想**：Howard Marks 强调的思维方式。不仅想"然后会怎样"，还要想"然后的然后会怎样"。大多数人只想到一阶后果，忽略了连锁反应。

**引导方式**：
- 一阶："如果你选了 A，直接结果是什么？"
- 二阶："这个结果又会导致什么？"
- 三阶："再往后呢？"

**示例**：
- 一阶：去大厂 → 收入翻倍
- 二阶：收入翻倍 → 996 → 没时间陪家人
- 三阶：没时间陪家人 → 关系紧张 → 影响生活幸福感

**适用场景**：求助者只关注最直接的好处/坏处时。

### 11. 机会成本分析

**核心思想**：每个选择都意味着放弃其他可能性。真正的成本不是你付出了什么，而是你放弃了什么。

**引导问题**：
- "选了 A，你放弃的最大东西是什么？"
- "你愿意为 A 付出这个代价吗？"
- "有没有第三种选择，能让你不用放弃那么多？"

**适用场景**：求助者只看到选项的好处、没有考虑代价时。

### 12. WRAP 框架

**核心思想**：Chip Heath 和 Dan Heath 在《Decisive》中提出的四步决策框架，专门针对常见的决策偏误设计：
- **W（Widen Your Options 拓宽选项）**：是否陷入了"非此即彼"的狭隘框架？有没有第三种可能？
- **R（Reality-Test Your Assumptions 验证假设）**：你的假设有证据支持吗？能不能小规模试一试？
- **A（Attain Distance Before Deciding 保持距离）**：跳出当下情绪，用 10/10/10 或第三方视角来看
- **P（Prepare to Be Wrong 准备应对错误）**：设定触发条件，万一判断错了，什么时候转向？

**适用场景**：作为完整决策流程的检查清单，尤其是防止思维盲区。

### 13. Eisenhower 矩阵（重要-紧迫矩阵）

**核心思想**：按"重要性"和"紧迫性"两个维度将事项分为四象限：
- **重要且紧迫**：立即行动
- **重要不紧迫**：安排计划（最容易被忽视的象限）
- **紧迫不重要**：委托他人或快速处理
- **不重要不紧迫**：放弃

**引导问题**：
- "这个决策真的紧急吗？还是你只是感觉焦虑？"
- "你是在被紧迫感裹挟，还是在处理真正重要的事？"
- "如果不急着决定，你还会这么焦虑吗？"

**适用场景**：求助者感到"必须马上决定"的时间压力时。帮助区分真正的紧迫和制造出来的焦虑。

## 第四层：压力测试层（验证决策）

对初步结论进行反向检验，确保决策经得起质疑。

### 14. 恶魔代言人（Devil's Advocate）

**核心思想**：故意站在反对立场，对当前最优选项提出最强的反驳。如果反驳不了，说明决策经得起考验。

**引导方式**：
- "我来扮演一个反对者。你选了 A，但我认为 A 有这些问题...你怎么回应？"
- "如果有人坚决反对你这个选择，他最可能的理由是什么？"
- "这个选择最大的隐患是什么？你准备怎么应对？"

**适用场景**：求助者已有明确倾向时，用来检验决策的稳健性。

### 15. 前验尸法（Pre-mortem）

**核心思想**：Gary Klein 提出。假设已经做了这个决定，一年后发现失败了。回溯思考：最可能的失败原因是什么？

**引导方式**：
- "假设你选了 A，一年后发现这是个错误决定。你觉得最可能是因为什么？"
- "如果失败了，你有 Plan B 吗？"
- "知道了这些风险，你还愿意选 A 吗？"

**适用场景**：作为最终决策前的"安全检查"。

## 框架选择指南

| 决策场景 | 推荐框架组合 |
|---------|-------------|
| 职业选择（去哪家公司） | 第一性原理 + 价值观排序 + SWOT + 加权矩阵 + 前验尸 |
| 城市选择（去哪里生活） | 价值观排序 + 加权矩阵 + 10/10/10 + 二阶思维 |
| 感情决策（要不要在一起/分手） | 第一性原理 + 苏格拉底提问 + 预设遗憾 + 硬币测试 |
| 要不要冒险（创业/转行/出国） | 可逆性判断 + 预设遗憾 + 前验尸 + 机会成本 |
| 多选一纠结 | WRAP + 加权矩阵 + 恶魔代言人 |
| 被催着做决定 | Eisenhower + 可逆性判断 + 10/10/10 |

## 常见决策偏误提醒

在引导过程中注意识别以下偏误：

1. **锚定效应**：过度受第一个信息影响。→ 提醒求助者考虑更多参照点
2. **确认偏误**：只关注支持自己已有倾向的信息。→ 用恶魔代言人对抗
3. **损失厌恶**：对失去的痛苦远大于获得的快乐。→ 用第一性原理还原真实价值
4. **从众心理**：因为别人都这么做所以跟随。→ 用第一性原理回到自我
5. **现状偏差**：倾向于维持现状，害怕改变。→ 用预设遗憾和机会成本挑战
6. **沉没成本谬误**：因为已经投入了所以不愿放弃。→ 提醒"过去的投入不应影响未来的决策"
7. **过度自信/过度悲观**：对结果概率的错误估计。→ 用前验尸和现实检验纠正
8. **框架效应**：问题的呈现方式影响判断。→ 用 WRAP 的拓宽选项打破框架
