## 中国财政数据收集

### 主要目的

根据中国财政部官网在信息公开-财政数据发布的公告，收集各种财政数据，具体指标包括但不限于：财政支出、财政收入、财政赤字、政府债务。

### 收集来源

https://www.mof.gov.cn/gkml/caizhengshuju/

### 指标

#### 财政收支情况

1. 分指标收集，数据项为月份、指标值。需要根据每次收集总和的数据，推导当月数据。

##### 示例

收集了以下信息：

1. 
2025年1-10月财政收支情况
2025年11月17日 来源：国库司

　　 一、全国一般公共预算收支情况

　　（一）一般公共预算收入情况。 

　　1-10月，全国一般公共预算收入186490亿元，同比增长0.8%。其中，全国税收收入153364亿元，同比增长1.7%；非税收入33126亿元，同比下降3.1%。分中央和地方看，中央一般公共预算收入81856亿元，同比下降0.8%；地方一般公共预算本级收入104634亿元，同比增长2.1%。 

　　主要税收收入项目情况如下： 

　　1.国内增值税58858亿元，同比增长4%。 

　　2.国内消费税14390亿元，同比增长2.4%。 

2. 2025年前三季度财政收支情况
2025年10月17日 来源：国库司

　　一、全国一般公共预算收支情况

　　（一）一般公共预算收入情况。 

　　前三季度，全国一般公共预算收入163876亿元，同比增长0.5%。其中，全国税收收入132664亿元，同比增长0.7%；非税收入31212亿元，同比下降0.4%。分中央和地方看，中央一般公共预算收入70837亿元，同比下降1.2%；地方一般公共预算本级收入93039亿元，同比增长1.8%。 

　　主要税收收入项目情况如下： 

　　1.国内增值税52271亿元，同比增长3.6%。 

　　2.国内消费税12934亿元，同比增长2.2%。 

---

则可以计算出：

全国一般公共预算收入 = 186490亿元 - 163876亿元 = [计算结果]
全国税收收入 = 153364亿元 - 132664亿元 = [计算结果]
...
主要税收收入项目情况：
1.国内增值税 = 58858亿元 - 52271亿元 = [计算结果]

将各个指标数据项计算结果计入 10月 的指标值。
