# 🎉 Fiesta Agents

A complete AI agent team for OpenClaw — 61 specialized agents across 8 departments with a central orchestrator.

## What Is This?

Fiesta Agents is a skill that gives your OpenClaw assistant access to a full virtual team. Need a frontend component built? Call the frontend-dev. Need a product launch planned and executed? Let the orchestrator coordinate engineering, design, marketing, and QA.

## Departments

- **💻 Engineering** (7 agents) — Frontend, backend, mobile, AI/ML, DevOps, prototyping, senior review
- **🎨 Design** (7 agents) — UI, UX research, UX architecture, brand, visuals, interaction, prompt art
- **📢 Marketing** (8 agents) — Growth, content, Twitter, TikTok, Instagram, Reddit, ASO, social lead
- **📊 Product** (3 agents) — Sprint planning, market analysis, feedback analysis
- **🎬 Project Management** (5 agents) — Program direction, coordination, operations, experiments, senior PM
- **🧪 QA & Testing** (7 agents) — Visual QA, release gates, test analysis, performance, API QA, tool audit, process optimization
- **🛟 Operations** (6 agents) — Support, data analysis, finance, infrastructure, compliance, executive reporting
- **🎯 Specialist** (6 agents) — Orchestration, BI, code intelligence, data extraction, data integration, report automation

## Installation

Place the `fiesta-agents` folder in your OpenClaw skills directory:
```
~/.openclaw/workspace/skills/fiesta-agents/
```

## Usage

### Single Agent
```
Use the backend-architect agent to design a REST API for user management.
```

### Multi-Agent via Orchestrator
```
Use the fiesta-agents orchestrator to execute a complete redesign of our landing page.
```
