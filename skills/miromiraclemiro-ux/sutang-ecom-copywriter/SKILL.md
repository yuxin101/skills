## 生成框架（内部逻辑）

### 淘宝详情页文案结构

```
【首图吸睛文案】
→ 痛点开场 + 产品核心承诺（10字以内）

【痛点共鸣区】
→ 描述用户买这类产品前的3个常见困扰
→ 用"你是不是也..."句式引发共鸣

【解决方案区】
→ 产品如何解决上述痛点
→ 列出3个核心卖点，每个配场景描述

【信任背书区】
→ 材质/成分/工艺说明（让用户觉得值）
→ 用户评价/销量数据（让用户觉得安全）

【行动号召区】
→ 限时优惠/库存紧张（制造紧迫感）
→ 售后保障/退换承诺（消除顾虑）

【关联销售区】
→ 搭配推荐，提升客单价
```

### 抖音带货脚本结构（15秒版）

```
0-2秒：开场钩子（痛点/好奇/冲突）
2-10秒：产品展示 + 核心卖点（场景化）
10-13秒：价格对比/优惠信息
13-15秒：行动号召（点击购买/左下角）
```

### 抖音带货脚本结构（60秒版）

```
0-5秒：黄金开场（制造悬念或引发共鸣）
5-20秒：痛点放大（让用户觉得"说的就是我"）
20-35秒：产品介绍（场景化展示，不是念参数）
35-45秒：使用体验/对比效果（真实感）
45-55秒：价格/优惠说明（限时限量制造紧迫感）
55-60秒：收尾号召（信任背书 + 行动引导）
```

### 小红书种草笔记结构

```
【标题】：痛点+解决方案+情绪词（emoji加分）
【开头】：真实使用场景，引发共鸣
【正文】：个人真实体验，包含：
  - 使用前/使用后对比
  - 3个真实使用场景
  - 1个意外惊喜/缺点诚实说
【结尾】：互动引导（评论区见/私信问） + 相关话题标签
```

### 拼多多商品描述结构

```
【第一行】：核心卖点 + 价格锚点（让人觉得值）
【第二行】：材质/规格/数量（核心参数）
【第三行】：发货/售后保障
【第四行】：促销信息（如有）
```

## 转化心理学要点

生成文案时自动遵循以下原则：

1. **痛点→快乐**：先描述不买会怎样痛苦，再给解决方案
2. **具体>模糊**："3秒上手" > "简单易用"
3. **数字说服**："已帮助1000+卖家" > "很多人用"
4. **场景化**："做饭时单手开盖" > "设计人性化"
5. **紧迫感**："仅剩30件" > "库存有限"
6. **消除顾虑**："30天无理由退换" > "品质保证"
