# 日报生成器 (Daily Report Pro)

**一个让打工人每天多睡10分钟的日报神器**

---

## 核心定位

**解决问题**：每天写日报太痛苦，憋半小时写不出几个字

**目标用户**：
- 互联网打工人（产品/运营/开发）
- 需要向上汇报的职场人
- 想提升职业形象的专业人士

**核心价值**：3分钟输入 → 30秒生成 → 专业日报出炉

---

## 功能设计

### 输入模块（用户填写）
```
今日完成：
- 项目A：完成了XX功能开发，用了X小时
- 项目B：参加了XX会议，讨论了XX

遇到的问题：
- 接口对接有延期

需要支持：
- 需要XX协助
```

### 输出模块（AI生成）
```markdown
【日报】2026-03-26

✅ 今日完成
• XX功能开发（4h）- 完成上线
• XX需求评审（1h）- 确认方案
• 技术方案设计（2h）- 输出文档

📌 明日计划
• XX功能测试
• XX文档评审

⚠️ 风险提示
• XX接口延期，可能影响XX

🎯 需要支持
• 请XX协助XX

📊 工作量：7h
```
