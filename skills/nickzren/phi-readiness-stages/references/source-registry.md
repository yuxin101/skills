## Verification rules

- Use 45 CFR Part 164 regulation text when the exact legal requirement matters.
- Use official HHS/OCR, Federal Register, NIST, or regulation text sources for authority.
- Some guidance pages carry current-law limitations or court-order qualifiers. Read those live and preserve the qualifier in the assessment when relevant.
- For `Access right, health apps, and APIs`, also review the linked right-of-access court-order notice and rely on the page only to the extent the live HHS page says it remains in effect.
- Use non-official sources only for supplemental context, never as the authority for a stage requirement.
- Record access date in each assessment.
- If a source appears to have changed, prefer the current official page over this registry and update the registry later.

## Source hierarchy

1. Effective regulation text
2. Official HHS/OCR guidance and FAQs
3. Official NIST implementation guidance
4. Supplemental non-authoritative commentary only if clearly labeled as such
