# 2024 NPRM Tracker

Subject: HIPAA Security Rule Notice of Proposed Rulemaking published December 27, 2024.

Official tracking source: https://www.hhs.gov/hipaa/for-professionals/security/hipaa-security-rule-nprm/index.html

Status last checked: March 17, 2026.

Next check date: June 17, 2026.

## Current rule for PRS

The NPRM remains proposed. PRS stage assignment must continue to use the currently effective Security Rule and current official guidance as the mandatory baseline unless the user explicitly requests a forward-looking or proposed-state analysis.

## How to use this tracker

- capture future-state gaps separately from current-stage blockers
- do not mix proposed requirements into the current mandatory rubric
- if requested, produce a separate `proposed-state delta` section instead of changing the assigned current PRS stage

## Reporting rule

When the NPRM is discussed in an assessment:

- say that it is proposed
- state the access date
- keep the current-stage decision anchored to the effective rule
