## How to use these references

- Start with `framework/stage-rubric.md` to determine which domains are required for the candidate stage.
- Read only the domain files needed for blockers, unclear evidence, or recommended actions.
- Treat these files as companion review rules, not as a substitute for live source verification from `references/source-registry.md`.
