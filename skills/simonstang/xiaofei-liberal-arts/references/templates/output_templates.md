## 三、英文作文辅导模板

### 基础结构
```markdown
📝 【题目分析】
主题：___
要求：___

🎯 【结构模板】
Paragraph 1: Introduction
  - Topic sentence
  - Background info

Paragraph 2-3: Body
  - Point 1 + supporting details
  - Point 2 + supporting details

Paragraph 4: Conclusion
  - Restate main point
  - Final thought

✍️ 【句型参考】
开头：
- My dream is to be...
- I have a dream that...

展开：
- I want to be... because...
- The reason why... is...

结尾：
- To achieve my dream, I will...
- I believe... if...
```

## 六、作文类型结构模板

### 记叙文
```
开头 → 设悬念/场景引入
    ↓
事件发展 → 铺垫
    ↓
高潮 → 最精彩的部分
    ↓
结尾 → 感悟/升华
```

### 议论文
```
开头 → 提出观点
    ↓
论据1 → 例子+分析
    ↓
论据2 → 例子+分析
    ↓
结尾 → 总结升华
```

### 读后感
```
开头 → 简述作品内容
    ↓
感触点1 → 分析+联系实际
    ↓
感触点2 → 分析+联系实际
    ↓
结尾 → 总结感悟
```
