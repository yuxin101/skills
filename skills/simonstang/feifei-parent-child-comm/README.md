# 菲菲老师亲子沟通专家 🌸

> 让孩子愿意开口，让家长懂得孩子

## 简介

基于青少年发展心理学博士+UCL教育学讲师背景，专注解决中国家庭亲子沟通难题。

**核心价值：** 打通亲子沟通障碍，建立信任关系

## 目标用户

- 小学4年级-初二学生家长
- 焦虑但无从下手的家长
- 孩子有沟通问题的家庭

## 核心功能

### 1. 孩子对话引导
- 每日一聊开场白库
- 打开话匣子技巧
- 深度对话引导框架
- 敏感话题应对

### 2. 家长指导
- 沟通方式自测
- 黄金法则
- 不同年龄段策略
- 场景话术库

### 3. 亲子诊断
- 关系自测问卷
- 家庭氛围观察表

### 4. 情绪管理
- 情绪识别训练
- 情绪调节方法
- 崩溃应对指南

### 5. 手机管理
- 沉迷原因分析
- 管理方案
- 家庭公约模板

## 适用场景

- 孩子不说话/关门
- 青春期叛逆
- 学习冲突
- 沉迷手机
- 情绪问题
- 亲子关系紧张

将 `feifei-parent-child-comm` 文件夹复制到 OpenClaw skills 目录即可使用。
