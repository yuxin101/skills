# 🦞 波龙菠萝蜜 使用教程

## 简介

**波龙菠萝蜜** 是一款专业的AI选股报告系统。

- 基于V2.0量化选股引擎
- 提供个性化选股、荐股、点评、复盘服务
- 让你的选股报告看起来专业、高端、可传播

## 数据来源

波龙选股数据均来自**规范信息渠道**：
- 📊 新浪财经 - A股实时行情
- 📈 金十数据 - 财经资讯与市场情绪
- 💹 万得资讯 - 专业金融数据
- 🏦 券商研报 - 上市公司研究
- 📰 财经媒体 - 东方财富、同花顺等公开信息

⚠️ 所有内容仅供参考，不构成投资建议！

## 使用方法

### 触发技能

在任意支持OpenClaw的场景中，输入以下任一触发词：

- `波龙菠萝蜜`
- `波龙推荐`
- `菠萝蜜选股`

### 功能命令

| 命令 | 功能说明 |
|:---|:---|
| `波龙菠萝蜜` | 生成当日选股报告 |
| `波龙菠萝蜜 荐股` | 推荐最佳1-3只股票 |
| `波龙菠萝蜜 点评 [代码]` | 深度分析单只股票 |
| `波龙菠萝蜜 复盘` | 查看历史战绩 |

## 核心特色

### 1. 个性化报告

- **用户昵称显示**：报告上显示你的名字
- **波龙等级**：越用等级越高
- **契合度**：波龙越来越懂你

### 2. 专业分析

- 技术面：趋势、支撑、量能
- 基本面：估值、财报、行业
- 情绪面：市场热度、资金流向

### 3. 战绩追踪

- 自动记录每次推荐
- 跟踪T+1/T+5/T+10收益
- 统计成功率
