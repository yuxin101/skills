# 巧未来·波龙股神 - 收款码配置

## 收款码放置位置

请将您的QQ收款码图片放在以下位置：

```
assets/
├── bolong-brand.jpg          # 品牌图片（已有）
└── qrcode-payment.png        # QQ收款码（待添加）
```

## 收款码规格

| 项目 | 建议 |
|------|------|
| 格式 | PNG 或 JPG |
| 尺寸 | 400x400 像素以上 |
| 背景 | 白色或透明 |
| 内容 | QQ收款码 |

## 获取QQ收款码方法

1. 打开 QQ App
2. 点击右上角「+」
3. 选择「收付款」
4. 点击「收款码」
5. 截图保存
6. 上传到 `assets/qrcode-payment.png`

## 收款码文案

图片上建议加上文字：

```
═══════════════════
   🦞 波龙寺香火箱
═══════════════════

   [收款码]

 💰 有钱就有未来 💰

═══════════════════
   巧未来AI
  有钱就有未来[得意]
═══════════════════
```

## 配置完成后

修改 `scripts/bolong_picker_v2.py` 中的收款码引用：

```python
# 将占位符替换为实际路径
QRCODE_PATH = "assets/qrcode-payment.png"
```
