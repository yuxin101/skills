# 🦞 波龙菠萝蜜

**专业选股报告系统 | 基于V2.0选股引擎**

---

## 功能概览

| 功能 | 描述 | 命令 |
|------|------|------|
| 🔍 **选股** | 基于V2.0多因子系统选股 | `波龙菠萝蜜` |
| ⭐ **荐股** | 选出最佳1-3只重点推荐 | `波龙菠萝蜜 荐股` |
| 📊 **点评** | 技术面+基本面+情绪面分析 | `波龙菠萝蜜 点评 [股票代码]` |
| 📈 **复盘** | 往期选股表现回顾 | `波龙菠萝蜜 复盘` |
| 💰 **香火** | 随喜添加香火钱 | `今日香火 XX` |
| ☕ **奶茶** | 波龙请你喝奶茶 | `波龙请客` |
| 📞 **电话** | 波龙打电话安慰 | `波龙打电话给我` |

---

## 快速开始

### 首次使用

输入 `波龙菠萝蜜` 即可生成当日选股报告。

### 添加香火

输入 `今日香火 88` 即可添加88元香火钱。

### 奶茶关怀

输入 `波龙请客` 获取奶茶关怀。

### 电话安慰

输入 `波龙打电话给我` 让波龙给你打电话。

## 核心特性

### 1. 游戏化等级

- **香客龙** → **知客龙** → **典座龙** → **维那龙** → **主持龙** → **方丈龙**
- 根据香火金额自动升级

### 2. 自愿香火

- 输入 `今日香火 XX` 随喜添加
- 一分不嫌少，几百不嫌多
- 心诚则灵，有钱就有未来

### 3. 关怀系统

- **奶茶关怀**：亏钱时波龙请你喝奶茶
- **电话安慰**：波龙打电话安慰你
- **随机奖励**：达到等级随机送波龙周边

## 数据来源

波龙选股数据均来自以下**规范信息渠道**：
- 📊 **新浪财经** - A股实时行情数据
- 📈 **金十数据** - 财经资讯与市场情绪
- 💹 **万得资讯** - 专业金融数据支持
- 🏦 **券商研报** - 上市公司研究报告
- 📰 **财经媒体** - 东方财富、同花顺等公开信息

⚠️ **免责声明**：
- 本技能内容来源于公开网络数据，仅供参考交流
- **不构成任何投资建议**，股市有风险，投资需谨慎
- 历史表现不代表未来收益，价格波动可能导致本金亏损
- 请根据自身风险承受能力和独立判断做决策
