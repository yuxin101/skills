### 函数类型速查

| 函数类型 | 一般形式 | 图像特征 | 实际应用 |
|----------|----------|----------|----------|
| 一次函数 | y = kx + b | 直线 | 匀速运动、成本计算 |
| 二次函数 | y = ax² + bx + c | 抛物线 | 抛体运动、利润最大化 |
| 反比例函数 | y = k/x | 双曲线 | 工作效率、密度问题 |
| 指数函数 | y = a^x | 快速增长/衰减 | 复利、人口增长 |
| 对数函数 | y = logₐx | 缓慢增长 | 地震等级、声音分贝 |
| 三角函数 | sin/cos/tan | 周期性波动 | 波浪、振动、交流电 |

### 重要公式

**代数**
- 平方差：a² - b² = (a+b)(a-b)
- 完全平方：(a±b)² = a² ± 2ab + b²
- 一元二次方程求根：x = (-b ± √Δ) / 2a

**几何**
- 勾股定理：a² + b² = c²
- 圆面积：S = πr²
- 球体积：V = (4/3)πr³

**三角函数**
- sin²θ + cos²θ = 1
- 正弦定理：a/sinA = b/sinB = c/sinC = 2R
- 余弦定理：c² = a² + b² - 2ab·cosC

## 二、物理核心概念

### 力学基础

**牛顿三定律**
1. **惯性定律**：物体保持静止或匀速直线运动，除非受外力
2. **F=ma**：加速度与合外力成正比，与质量成反比
3. **作用力与反作用力**：大小相等、方向相反

**能量守恒**
- 动能：Ek = ½mv²
- 重力势能：Ep = mgh
- 机械能守恒：Ek₁ + Ep₁ = Ek₂ + Ep₂

### 电磁学基础

**欧姆定律**
- I = U/R（电流 = 电压 / 电阻）

**电功率**
- P = UI = I²R = U²/R

**电路类型**
| 连接方式 | 电流特点 | 电压特点 | 电阻特点 |
|----------|----------|----------|----------|
| 串联 | I = I₁ = I₂ | U = U₁ + U₂ | R = R₁ + R₂ |
| 并联 | I = I₁ + I₂ | U = U₁ = U₂ | 1/R = 1/R₁ + 1/R₂ |

### 光学基础

**光的折射**
- n₁sinθ₁ = n₂sinθ₂（斯涅尔定律）
- 光从光疏→光密介质：向法线偏折
- 光从光密→光疏介质：远离法线偏折

**全反射条件**
1. 光从光密→光疏介质
2. 入射角 > 临界角

## 三、化学核心概念

### 原子结构

```
原子
├── 原子核
│   ├── 质子（带正电，决定元素种类）
│   └── 中子（不带电，影响质量）
└── 核外电子（带负电，决定化学性质）
```

- 质子数 = 原子序数 = 核外电子数（中性原子）
- 质量数 = 质子数 + 中子数

### 化学键类型

| 键类型 | 形成方式 | 特点 | 例子 |
|--------|----------|------|------|
| 离子键 | 电子转移 | 金属+非金属 | NaCl |
| 共价键 | 电子共用 | 非金属+非金属 | H₂O, CO₂ |
| 金属键 | 自由电子 | 金属原子间 | Fe, Cu |

### 常见化学反应类型

1. **化合反应**：A + B → AB
2. **分解反应**：AB → A + B
3. **置换反应**：A + BC → AC + B
4. **复分解反应**：AB + CD → AD + CB

### 溶液浓度计算

- 质量分数 = (溶质质量 / 溶液质量) × 100%
- 物质的量浓度 = 溶质物质的量 / 溶液体积 (mol/L)

## 五、家庭实验方案库

### 物理实验

#### 1. 浮力实验
**材料**：鸡蛋、玻璃杯、盐、水
**步骤**：
1. 清水中放入鸡蛋 → 下沉
2. 加盐搅拌 → 慢慢浮起
**原理**：液体密度增加，浮力增大

#### 2. 光的折射实验
**材料**：激光笔、水杯、水
**步骤**：
1. 激光斜射入水中
2. 观察光线的偏折
**原理**：光从光疏→光密介质向法线偏折

#### 3. 电路实验
**材料**：电池、导线、小灯泡、开关
**步骤**：
1. 串联：电池→灯泡→开关→电池
2. 并联：两个灯泡分别连接电池两端
**观察**：串联亮度变化 vs 并联亮度

### 化学实验

#### 1. 密度分层
**材料**：蜂蜜、水、油、透明玻璃杯
**步骤**：依次倒入蜂蜜→水→油
**原理**：密度大的在下，密度小的在上

#### 2. 小苏打火山
**材料**：小苏打、醋、洗洁精、颜料
**步骤**：小苏打+洗洁精，倒入醋
**原理**：酸碱反应产生CO₂气体
