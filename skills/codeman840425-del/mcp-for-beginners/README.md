# MCP for Beginners

Microsoft 出品的 Model Context Protocol (MCP) 入门课程。

## 概述

开源课程，通过 .NET、Java、TypeScript、JavaScript、Rust 和 Python 的跨语言实例，帮助开发者学习 MCP 基础并构建模块化、可扩展的 AI 工作流。

## 快速开始

```bash
# Fork 并克隆仓库
git clone https://github.com/microsoft/mcp-for-beginners.git

# 加入 Discord 社区
https://discord.gg/nTYy5BXMWG
```

## 学习路径

- 会话配置基础
- MCP 核心概念
- 多语言实现对比
- 服务编排实践
