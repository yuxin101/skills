# freelance-proposal-wizard

Generate professional freelance proposals in seconds from a job posting or client brief.

## Usage

```
Write a proposal for this job: [paste job description]
Generate proposal for: [client name] - [project description] - budget [X USD] - deadline [date]
```

## What It Generates

A complete freelance proposal with:
- **Subject line** — attention-grabbing, professional
- **Opening paragraph** — personalized hook referencing client's specific need
- **Understanding** — shows you understand their problem
- **Proposed solution** — clear approach, no jargon
- **Timeline** — realistic milestones
- **Pricing** — itemized breakdown
- **Why me** — your relevant experience / social proof
- **Call to action** — next step, invite to call
- **Signature** — closing with availability

## Input Fields

| Field | Description | Required |
|-------|-------------|---------|
| job_description | The job posting or client brief | Yes |
| your_name | Your name | No |
| your_skills | What you actually do well | No |
| budget | Fixed / hourly / range | No |
| deadline | Required by date | No |
| portfolio_links | Relevant past work | No |

## Tips for Better Proposals

- Always personalize the opening — reference something specific from their post
- Price slightly below market for first-time clients
- Include 1-2 relevant portfolio items
- Keep it under 400 words — clients scan, don't read

## Notes

- Generates in markdown — paste directly into Upwork, Freelancer, Fiverr, or email
- Tone: confident but not arrogant, technical but accessible
- Great for: web dev, mobile dev, automation, API work, data projects, writing
