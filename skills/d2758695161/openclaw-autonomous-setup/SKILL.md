## Setup Steps

### 1. Install OpenClaw
```bash
npm install -g openclaw
openclaw gateway start
```

### 2. Install Core Skills
```bash
openclaw skills install proactive-agent
openclaw skills install sendclaw-email
openclaw skills install crypto-degen-toolkit
```

### 3. Configure Memory
Create `SESSION-STATE.md` with your business context.

### 4. Set Up Payment Monitor
Use Blockscout API (free, no key):
```
GET https://eth.blockscout.com/api?module=account&action=balance&address=YOUR_WALLET&tag=latest
```

### 5. Configure Cron Jobs
- Payment check: every 30 min
- Daily report: 09:00 daily
- Email check: every 15 min
