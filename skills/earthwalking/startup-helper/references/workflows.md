# 创业启动助手 - 工作流程参考

## 场景一：科技创业公司（APP/网站）

```
适用场景：互联网创业者，计划做APP或网站

推荐流程：
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

阶段1：公司注册（Day 1-15）
├── 1. 企业名称核准（e-dengji.com）
├── 2. 准备注册材料
├── 3. 提交注册申请（全程电子化）
├── 4. 领取营业执照
└── 5. 刻章

阶段2：银行开户 + 税务（Day 16-30）
├── 6. 预约银行开户
├── 7. 办理基本户
├── 8. 税务登记
├── 9. 购买金税盘
└── 10. 开通电子税务局

阶段3：ICP备案（Day 1-60，并行）
├── 11. 购买域名（阿里云/腾讯云）
├── 12. 购买云服务器
├── 13. 提交ICP备案
├── 14. 等待管局审核
└── 15. 完成备案

阶段4：知识产权（Day 30-90）
├── 16. 商标查询
├── 17. 提交商标申请（9-12类）
└── 18. 软件著作权登记（可选）

阶段5：社保开户（Day 30-45）
├── 19. 办理社保登记
└── 20. 添加员工信息

完成！🎉
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
预计总周期：2-3个月
预计总费用：8000-15000元（首年）
```

## 场景二：电商创业公司

```
适用场景：计划做电商平台或在线销售

推荐流程：
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

阶段1：公司注册（Day 1-15）
├── 1. 企业名称核准
├── 2. 准备注册材料（含电商经营范围）
├── 3. 提交注册
├── 4. 领取营业执照
└── 5. 刻章

阶段2：银行 + 税务（Day 16-30）
├── 6. 银行开户
├── 7. 税务登记
├── 8. 申请电商平台入驻资格
└── 9. 开通支付接口（支付宝/微信）

阶段3：特殊资质（如需）
├── 10. 食品经营许可（卖食品）
├── 11. 出版物经营许可（卖图书）
└── 12. 医疗器械许可（卖医疗用品）

阶段4：ICP备案（Day 1-45）
├── 13. 购买域名+服务器
├── 14. 提交ICP备案
└── 15. 完成备案

阶段5：电商平台入驻
├── 16. 淘宝/天猫/京东入驻
├── 17. 申请特殊类目资质
└── 18. 设置支付和物流

完成！🎉
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
预计总周期：1-2个月
预计总费用：5000-10000元（首年）
```

## 决策树：什么时候需要做什么

```
你是哪种创业者？

├── 我要做APP/网站
│   └── 必须：ICP备案 + 公司注册 + 商标（9类/42类）
│
├── 我要做电商平台
│   └── 必须：公司注册 + ICP备案 + 特殊资质（根据品类）
│
├── 我要做线下生意（餐饮/零售）
│   └── 必须：公司注册 + 食品经营许可/营业执照

│   └── 通常只需：公司注册 + 行业资质（如需）
│

    └── 可选：个体工商户（简单但无限责任）
```

## 第三方平台选择

### 阿里云 vs 腾讯云

| 服务 | 阿里云 | 腾讯云 |
|------|--------|--------|
| 备案服务 | 优（备案速度快） | 优（小程序生态好） |
| 服务器 | 稳定，覆盖广 | 游戏/小程序优化 |
| 价格 | 适中 | 略低 |
| 创业扶持 | 阿里云创业者计划 | 腾讯云创业扶持 |
| 适合场景 | APP/网站/电商 | 微信生态/游戏 |

**建议**：
- APP/网站类 → 选阿里云
- 小程序类 → 选腾讯云
- 两者都用 → 根据主业务选择

## 常见问题流程

### Q：没有注册地址怎么办？
```
解决方案：
├── 1. 创业园区/孵化器地址（推荐）
├── 2. 集中办公区地址
├── 3. 第三方地址挂靠服务
└── 费用：500-3000元/年
```

### Q：ICP备案需要多久？
```
时间线：
├── 阿里云初审：1-2天
├── 短信核验：1天
├── 管局审核：7-20天（省差异）
└── 总计：15-30天

加速技巧：
├── 材料准备充分
├── 法人手机畅通
└── 选择审核快的省份（如广东、上海）
```

### Q：先注册公司还是先买服务器？
```
推荐顺序：
├── 1. 先注册公司（公司名可用于域名）
├── 2. 同时购买域名（锁定品牌名）
└── 3. 注册完成后立即提交ICP备案

域名建议：
├── 先注册.com/.cn等主流域名
└── 域名具有唯一性，抢注风险大
```
