# Upwork Gig Parsing Patterns

## Gig Score Criteria
- Budget match: High if within +/-20% your rate.
- Urgency: 'urgent', 'ASAP', 'quick'.
- Fit: Keywords in niche 3+.

## Proposal Template Vars
- [client]: From gig owner.
- [budget]: Extract $XXX.
- Timeline: 24-72hrs.

## Search Queries
- 'niche' + 'logo' 'design' 'urgent' etc.
