# A/B Test Calculator Tips

## 核心概念

1. **统计显著性** — p值 < 0.05 通常认为结果显著（95%置信度）
2. **样本量** — 样本越大，检测到小差异的能力越强
3. **最小可检测效应(MDE)** — 你关心的最小变化量，越小需要的样本越多
4. **统计功效(Power)** — 通常设为80%，表示有80%概率检测到真实差异

## 实验设计要点

- **先算样本量，再开始实验** — 避免过早停止或跑太久
- **一次只测一个变量** — 多变量测试需要更复杂的设计
- **随机分配** — 确保A/B组用户特征一致
- **跑完整周期** — 至少覆盖一个完整业务周期（通常7天）
- **避免偷看** — 中途看结果容易做出错误决策

## 常见陷阱

| 陷阱           | 影响              | 规避方法            |
|---------------|-------------------|-------------------|
| 样本不足        | 假阳性/假阴性      | 提前计算样本量       |
| 过早停止        | 结果不可靠         | 等到预定样本量       |
| 多重比较        | p值膨胀            | Bonferroni校正     |
| 辛普森悖论      | 混杂因素           | 分层分析            |
| 新奇效应        | 短期偏高           | 延长实验时间        |

## 贝叶斯 vs 频率派

- **频率派**：回答"如果没有差异，观察到这个结果的概率"
- **贝叶斯**：直接回答"B比A好的概率是多少"
- 小样本用贝叶斯更直观，大样本两者趋同
