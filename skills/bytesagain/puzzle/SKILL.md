# puzzle

Brain teasers, riddles, and logic puzzles with streak tracking. Use when exercising your brain.

## Commands

### `riddle`

```bash
scripts/script.sh riddle
```

### `math`

```bash
scripts/script.sh math
```

### `word`

```bash
scripts/script.sh word
```

### `logic`

```bash
scripts/script.sh logic
```

### `daily`

```bash
scripts/script.sh daily
```

### `streak`

```bash
scripts/script.sh streak
```

## Data Storage

Data stored in `~/.local/share/puzzle/`.
