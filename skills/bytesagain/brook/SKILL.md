# Brook

A cross-platform programmable network tool ## Commands

- `help` - Help
- `run` - Run
- `info` - Info
- `status` - Status

## Features

- Core functionality from txthinking/brook

## Usage

Run any command: `brook <command> [args]`

## Examples

```bash
# Show help
brook help

# Run
brook run
```

- Run `brook help` for commands
- No API keys needed

- Run `brook help` for all commands

- Run `brook help` for all commands

## When to Use

- when you need quick brook from the command line
- to automate brook tasks in your workflow

## Output

Returns reports to stdout. Redirect to a file with `brook run > output.txt`.

## Configuration

Set `BROOK_DIR` to change data directory. Default: `~/.local/share/brook/`
