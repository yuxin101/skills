# WordPress 运营最佳实践

> 从建站到运营，WordPress 站长的全方位指南。

---

## 一、SEO 优化最佳实践

### 1. 基础配置
- **固定链接**：设置为 `/%postname%/`，简短且含关键词
- **XML Sitemap**：安装 Yoast SEO 或 Rank Math，自动生成 sitemap
- **Google Search Console**：提交 sitemap，监控索引状态
- **robots.txt**：确保不屏蔽重要页面

### 2. 页面优化
- **标题**：30-60 字符，含核心关键词，放在前面
- **Meta Description**：120-160 字符，包含行动号召
- **H1 标签**：每页只用 1 个 H1，包含主关键词
- **图片 Alt**：每张图都要写描述性 alt 文本
- **内链**：每篇文章 3-5 个内链，用锚文本链接相关文章
- **URL**：简短、用连字符分隔、包含关键词

### 3. 内容策略
- **关键词密度**：自然出现，1%-3% 为宜
- **内容长度**：深度文章 1500-3000 字效果最佳
- **更新频率**：每周至少 1 篇新文章
- **内容更新**：定期更新旧文章的数据和链接

### 4. 技术 SEO
- **Core Web Vitals**：LCP < 2.5s, FID < 100ms, CLS < 0.1
- **Mobile-First**：确保移动端体验优秀
- **结构化数据**：使用 Schema.org 标记获取富摘要
- **Canonical 标签**：避免重复内容被重复收录

## 二、安全加固清单

### 1. 第一天必做
- [ ] 修改默认 admin 用户名
- [ ] 设置强密码（16位+大小写+数字+特殊字符）
- [ ] 安装安全插件（Wordfence / Sucuri）
- [ ] 启用 2FA 双因素认证
- [ ] 全站强制 HTTPS

### 2. 每周检查
- [ ] 更新 WordPress 核心、插件、主题
- [ ] 检查登录日志，排查异常登录
- [ ] 扫描恶意软件
- [ ] 验证备份是否正常运行

### 3. 防御配置
- 修改登录 URL（WPS Hide Login）
- 限制登录尝试次数（5次锁定30分钟）
- 禁用 XML-RPC（如果不需要）
- 隐藏 WordPress 版本号
- 设置 HTTP 安全头（X-Frame-Options, CSP 等）
- wp-config.php 权限设为 400

## 三、速度优化路线

### 优化优先级（ROI 从高到低）

| 优先级 | 措施 | 预期提升 | 难度 |
|--------|------|----------|------|
| 1 | 启用页面缓存 | 50-80% | ⭐ |
| 2 | 接入 CDN | 30-60% | ⭐⭐ |
| 3 | 压缩图片 | 20-40% | ⭐ |
| 4 | 合并+延迟 JS/CSS | 15-30% | ⭐⭐ |
| 5 | 升级 PHP 版本 | 10-20% | ⭐⭐ |
| 6 | 数据库优化 | 5-15% | ⭐⭐ |
| 7 | 更换主机 | 30-70% | ⭐⭐⭐ |

### 推荐插件组合
- **缓存**：WP Super Cache（简单）或 W3 Total Cache（全面）
- **图片**：ShortPixel（压缩+WebP转换+懒加载一体）
- **代码**：Autoptimize（合并压缩CSS/JS）
- **数据库**：WP-Optimize（清理+优化）
- **监控**：Query Monitor（找出慢查询插件）

### 测速工具
- Google PageSpeed Insights（免费，含 Core Web Vitals）
- GTmetrix（详细瀑布图分析）
- WebPageTest（多地点多浏览器测试）

## 四、内容运营策略

### 1. 内容规划
- **内容日历**：提前规划 1-2 周的发布计划
- **支柱内容**：每月写 1-2 篇深度长文（2000+ 字）
- **集群策略**：围绕核心关键词创建内容集群
- **常青内容**：写不过时的内容，持续获取流量

### 2. 分发渠道
- RSS 订阅
- 社交媒体自动分享（Jetpack Social / Buffer）
- 邮件列表（Newsletter）
- SEO 自然搜索

### 3. 数据分析
- **Google Analytics**：监控流量来源和用户行为
- **Google Search Console**：监控搜索表现和关键词排名
- **关键指标**：跳出率、平均停留时间、转化率

### 4. 用户体验
- 文章目录（Table of Contents）
- 相关文章推荐
- 面包屑导航
- 搜索功能优化
- 评论管理（Akismet 防垃圾）

## 五、备份策略

### 3-2-1 备份法则
- **3** 份备份副本
- **2** 种不同存储介质
- **1** 份异地存储

### 备份频率建议
| 频率 | 内容 | 保留期 |
|------|------|--------|
| 每日 | 数据库 | 30天 |
| 每周 | 完整站点 | 12周 |
| 每月 | 完整+配置 | 12月 |

### 恢复测试
- 每季度至少测试一次备份恢复
- 记录恢复步骤和时间
- 保持恢复文档更新

## 六、常见问题速查

| 问题 | 解决方案 |
|------|----------|
| 白屏 / 500 错误 | 禁用所有插件，逐个排查 |
| 登录被锁定 | 通过 phpMyAdmin 重置密码 |
| 网站被黑 | Wordfence 扫描 → 清除 → 加固 |
| 速度变慢 | Query Monitor 排查 → 缓存+CDN |
| 数据库太大 | WP-Optimize 清理修订和垃圾数据 |
| 图片加载慢 | ShortPixel 压缩 + CDN |
| SEO 排名下降 | Search Console 诊断 → 内容更新 |
