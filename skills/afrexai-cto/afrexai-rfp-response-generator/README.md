# RFP Response Generator

AI-powered skill for generating structured, compliant responses to RFPs, RFIs, and RFQs.

## What It Does

- Extracts and organizes requirements from solicitation documents
- Maps company capabilities against each requirement
- Generates compliant response sections (executive summary, technical, management, past performance)
- Produces a compliance matrix and review checklist
- Flags gaps and suggests mitigation strategies

## Use Cases

- **Government contracting** — Federal, state, and local RFP responses
- **Enterprise sales** — Formal proposal responses for large deals
- **Consulting firms** — Standardized proposal generation
- **IT services** — Technical solution proposals
- **Any B2B company** responding to formal solicitations

## Quick Start

Provide the RFP document and your company profile:

```
Respond to this RFP: [paste or attach RFP]
Company profile: [paste or point to file]
Win themes: [your key differentiators]
```

The skill generates a full response package: compliance matrix, executive summary, technical approach, management approach, past performance narratives, and a final review checklist.

## Output Files

| File | Description |
|------|-------------|
| `compliance-matrix.md` | Requirements mapped to capabilities |
| `executive-summary.md` | Standalone exec summary |
| `technical-response.md` | Technical approach section |
| `management-response.md` | Management approach |
| `past-performance.md` | Past performance narratives |
| `review-checklist.md` | Final compliance check |

## Why This Matters

RFP responses are time-intensive and high-stakes. Most companies spend 20-40 hours per response. This skill cuts that to under 2 hours for a first draft, while ensuring no requirements are missed.
