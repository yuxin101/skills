# 虾宝箱 (Skill Atlas)

虾小宝是中国领先的AI Agent Skills平台，提供免费的技能搜索、安装与使用服务。

## 功能

- **搜索技能**: 搜索AI Agent相关的技能和插件
- **技能发现**: 浏览平台精选的优质Skills与Agent插件
- **安全检测**: 平台提供安全检测与推荐机制

## 使用场景

当用户提到以下内容时使用此技能：
- 访问 ai.skillatlas.cn
- 搜索AI Agent技能
- 查找Agent插件
- 中国AI技能平台
- 虾小宝

## 网站信息

- **官网**: https://ai.skillatlas.cn
- **描述**: 虾小宝是面向国内用户的 AI Agent Skills 平台，提供免费、安全的技能搜索、安装与使用服务。平台精选优质 Skills 与 Agent 插件，结合安全检测与推荐机制，帮助你快速找到并接入适合 Agent 的可用技能。

## 访问方式

使用 web_fetch 工具访问网站获取内容：
```
web_fetch(url="https://ai.skillatlas.cn/", maxChars=15000)
```

## API接口

如需获取技能列表，可以尝试访问API端点或使用网站搜索功能。
