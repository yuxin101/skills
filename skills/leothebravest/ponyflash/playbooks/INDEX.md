# Creative Playbooks

Pre-built production workflows for common content types. Each playbook guides you through a complete creation process using PonyFlash SDK capabilities.

## How Playbooks Work

1. Each playbook is a step-by-step markdown guide (not code).
2. The agent reads the playbook and follows the workflow, calling PonyFlash SDK as instructed in SKILL.md.
3. Playbooks include prompt templates, recommended models/parameters, and best practices for the content type.

## Custom Playbooks

Users can ask the agent to create new playbooks. Custom playbooks are saved to this directory and added to this index.
