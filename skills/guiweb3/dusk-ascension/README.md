# 🌅 Dusk Ascension — OpenClaw Image Generation Skill

Generate stunning product and concept hero shots in a dreamy ethereal dusk aesthetic.

## ✨ What It Does

Turns any subject into a high-fashion editorial hero shot with:

- **1990s film photography** aesthetic — analog grain, chromatic aberration, dream-haze
- **Dusk-to-dawn gradient** background — midnight blue → steel blue → warm amber
- **Starburst lens flares** on metallic highlights
- **Floating, centered composition** — no stands, no pedestals, pure product focus

## 📦 Installation

### Via ClawHub (recommended)
```bash
clawhub install dusk-ascension
```

### Manual
Copy `SKILL.md` to your OpenClaw skills directory:
```bash
mkdir -p ~/.openclaw/workspace/skills/dusk-ascension
cp SKILL.md ~/.openclaw/workspace/skills/dusk-ascension/
```

## 🚀 Usage

Just ask your OpenClaw agent:

```
Generate a dusk ascension image of [your subject]
```

Or use the prompt template directly with any image generation model. The skill works best with models that support detailed style prompts (GPT Image, Gemini, etc).

### Examples

- "A pair of premium headphones in dusk ascension style"
- "A hardware wallet floating in ethereal dusk light"
- "A luxury watch in 1990s film photography aesthetic"

## 🔧 Technical Details

- **Aspect Ratio:** 4:5 (portrait/Instagram-friendly)
- **Best Models:** gpt-image-1, gemini-2.0-flash (image generation)
- **Key Features:** Film grain is a feature, not a bug. The analog imperfection is the aesthetic.
