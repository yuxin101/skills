# Ecommerce Copy Humanizer ZH

把“AI 味很重”的中文电商文案，改成更像真人运营/卖货团队会写的表达，同时保留卖点、转化意图和合规边界。

这不是普通润色器，也不是文学改写器。

它的目标是：
**把生硬、模板化、空泛承诺式的中文电商文案，改成更自然、更可信、更像真人写的版本，而且仍然能卖货。**

## Solves

中文电商文案常见的失败点不是“信息不够”，而是：
- 一眼看出是 AI 写的，语言像模板拼接；
- 语气太硬、太满、太浮夸，降低信任；
- 太像通用营销文案，不像真实商家/运营在说话；
- 改得更自然之后，反而把卖点、CTA、促销重点改没了；
- 平台、客服、直播、短视频、详情页需要的语气完全不同，但文案还在一把梭。

目标：
**在“自然表达、转化效率、平台语境、风险控制”之间找到平衡。**

## Use when

Use when the user wants to make Chinese ecommerce copy sound less AI-generated and more commercially credible.

Typical cases:
- 商品详情页文案润色
- 广告文案/投流素材改写
- 评论回复、私信话术优化
- 直播脚本、口播台词、短视频字幕去 AI 味
- 促销文案、活动文案本地化
- 将英文或 AI 初稿改成更自然的中文电商表达
- 让“听起来很会写，但不太会卖”的文案变成更能转化的版本

## Do not use when

Do not use this skill when:
- 用户要的是纯翻译，而不是转化导向改写
- 用户要的是文学创作、品牌 manifesto、长篇散文风格写作
- 用户要的是法律、医疗、金融等高风险专业合规审校
- 输入信息太少，无法判断卖点、场景和风险边界
- 任务本质是从零写完整营销策略，而不是优化已有文案

## Inputs

Ask for the minimum useful context:
- 原始中文文案或 AI 初稿
- 平台 / 场景（详情页、广告、评论区、直播、短信、私信等）
- 产品类型
- 目标受众
- 目标动作（点击、下单、咨询、收藏、回复）
- 语气偏好（自然、克制、种草、强卖点、客服式、直播式）
- 风险边界 / 禁用词 / 禁止承诺

## Workflow

### 1. Diagnose why the copy feels AI-written
Check for:
- 套话、空话、过度修辞
- 句子结构太整齐、太平均
- 卖点很完整但没有“人味”
- 夸大承诺、假大空表达
- 语言像 PPT，不像真实运营在说话
- 促销信息堆叠但缺乏重点

### 2. Identify the commercial job of the copy
Before rewriting, clarify the job:
- 是为了提高点击？
- 是为了促成转化？
- 是为了降低疑虑？
- 是为了回应用户异议？
- 是为了让内容更适合评论区/直播/短视频场景？

### 3. Rewrite for naturalness without losing selling intent
When rewriting:
- 保留核心卖点、利益点、使用场景、CTA
- 把生硬表达改成更像真人会说的话
- 去掉空泛承诺，换成更具体、更可信的说法
- 保持中文平台语境，不要写成翻译腔
- 根据场景调整语气，不同平台不要混写

### 4. Check risk and compliance edges
Review for:
- 绝对化承诺
- 容易踩平台规则的词
- 夸张疗效/效果表述
- 容易引发误导的比较或保证
- 过度营销、低信任措辞

## Output

Return a practical rewrite package:

1. **Problem diagnosis**
   - 文案里最明显的 AI 痕迹 / 信任问题 / 转化问题

2. **Primary rewrite**
   - 最推荐版本，可直接使用或稍作调整

3. **Alternative rewrite(s)**
   - 更克制版 / 更种草版 / 更转化版（按场景给 1–2 个）

4. **Risk replacements**
   - 哪些词建议替换，替换成什么更稳

5. **Usage notes**
   - 更适合详情页 / 广告 / 评论区 / 直播哪种场景

## Quality bar

A strong output should:
- 读起来像真人运营，不像模型模板
- 保留商业目的，而不是只追求“自然”
- 更像真实卖货表达，而不是空洞文案腔
- CTA 清晰，但不过度 push
- 能直接用于发布、回复、投放或脚本改稿
- 明确指出风险边界，而不是盲目“润色”

## What “better” looks like

Good output should feel like:
- 更可信
- 更具体
- 更接近真实中国电商语境
- 更适合平台和场景
- 更容易转化，而不是更像作文
