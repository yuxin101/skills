# Sideboard

Low-height dining storage — sideboards, buffets, servers. Like a wide, short cabinet. Typically 30–36" tall (counter height), with a combination of doors and drawers.

## Components

| Component | Required | Role |
|-----------|----------|------|
| Case | Yes | Top, bottom, sides |
| Top | Yes | Serving surface (often with overhang) |
| Doors | Yes | Panels closing lower compartments |
| Drawers | Optional | Upper drawer tier |
| Dividers | Optional | Vertical partitions creating sections |
| Shelves | Optional | Inside compartments behind doors |
| Base/Kick | Yes | Feet or plinth base |
| Back | Yes | Plywood back panel |

### Component relationships

```
Same as cabinet but wider and shorter.
Top has serving overhang (1–2" front and sides)
Typically 2–4 door sections with optional upper drawer tier
```

Refer to `types/cabinet.md` for detailed connections, hardware, build order, and common mistakes. Key differences:

- **Proportions**: wide (48–72") and short (30–36")
- **Top overhang**: serving surface extends beyond case
- **Sections**: typically 2–4 compartments with doors
- **Upper drawers**: common — one drawer per section above door
