# 🔥 微博热搜日报

每日自动抓取微博热搜榜，支持AI摘要和多渠道推送。

## ✨ 功能

- 📊 自动抓取微博热搜 Top 50
- 🤖 AI 智能分类摘要
- 📱 多渠道推送
- 📈 数据统计

## 📦 安装

```bash
npx clawhub@latest install weibo-hot-daily
```

## 🚀 使用

```bash
python3 fetch_hot.py --top 10
python3 fetch_hot.py --top 50 --summary --output hot.json
```
