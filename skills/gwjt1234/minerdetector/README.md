# MinerDetector (publishable folder)

This folder is ready for `clawhub publish`.

## Publish command

```bash
clawhub publish ./MinerDetector_publishable --slug minerdetector --name "MinerDetector" --version 1.0.0 --tags latest
```

## What is inside

- `SKILL.md`
- `scripts/miner_detector.py`
- `indicators/*.txt` (4 bundled libraries)
- `.clawhubignore`

## Important

This skill does not scan or detect anything. It only bills and exports the 4 bundled text libraries.
