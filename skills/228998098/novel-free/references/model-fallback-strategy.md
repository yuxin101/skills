# 模型故障/拒绝回退策略

> **核心原则：预防优于回退。Coordinator 在分发任务前主动预判敏感度。**
> **回退触发条件：** 模型返回"无法生成"、"拒绝"、"内容敏感"、超时（>120s）

---

## 模型回退链

| 原 Agent | 触发条件 | 回退目标 | 回退后处理 |
|----------|----------|----------|------------|
| MainWriter | 拒绝/故障 | 备选写作模型 | 担任主笔+润色（合并执行） |
| FinalReviewer | 拒绝 | OOCGuardian模型 | 担任终审；若也拒绝 → Coordinator 自行终审 |
| BattleAgent | 拒绝 | MainWriter | 写战斗段落，Prompt附加"动作描写+专业术语" |
| OOCGuardian | 拒绝 | Coordinator | 自行一致性检查，输出简化版OOC报告 |
| ReaderSimulator | 拒绝 | 跳过 | 跳过读者模拟步骤，不阻断写作 |
| Worldbuilder | 拒绝 | Coordinator | 自行处理 |
| CharacterDesigner | 拒绝 | Coordinator | 自行处理 |
| OutlinePlanner | 拒绝 | Coordinator | 自行处理 |
| RollingSummarizer | 拒绝 | Coordinator | 自行生成简化摘要 |

## 敏感度预判矩阵

Coordinator 在分发任务前，根据章节内容预判敏感度：

| 敏感等级 | 场景类型 | Prompt约束 | 模型选择 |
|----------|----------|------------|----------|
| **高敏感** | 亲密场景/情感爆发/R18相关 | 强制附加文学性/含蓄/情感约束 | 优先审核宽松模型 |
| **中敏感** | 暴力场面/悬疑/心理压迫 | 附加"适度描写，避免细节过多" | 任意模型 |
| **低敏感** | 日常/战斗/信息交代 | 标准 Prompt，无额外约束 | 任意模型 |

---

## 高敏感 Prompt 强制附加

当涉及亲密/情感场景时，**所有写作 Agent 的 prompt 必须附加**：

```
【敏感度约束】
- 必须：文学性描写，含蓄隐晦，注重情感而非动作
- 必须：使用留白、暗示、情绪侧面烘托
- 禁止：直接描写身体接触/性暗示/过于直白的欲望表达
- 禁止：R18相关词汇或暗示
- 风格参考：村上春树的隐晦笔触、金陵雪的克制抒情
```

---

## 连续拒绝处理流程

```
首次拒绝 → 记录日志，尝试回退模型
    ↓ 第二次拒绝（同一任务）
回退模型也失败 → Coordinator 自行处理（降级输出）
    ↓ Coordinator 也无法完成
跳过该段落（保留占位）→ 汇报中标注"ChXX 第Y段需用户确认后重写"
```

## 回退记录格式

每次触发回退，必须在 `archive/archive.md` 中记录：

```markdown
## 回退记录 ChXX

| 时间 | Agent | 原任务 | 回退目标 | 结果 |
|------|-------|--------|----------|------|
| 2026-03-23 16:30 | MainWriter | 初稿 | 备选模型 | 成功 |
```
