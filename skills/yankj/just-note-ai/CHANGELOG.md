## 使用示例

### 记录一条支出

```bash
just-note write --type expense --amount 200 --tags "book,learning" --content "买书"
```

### 查看今日记录

```bash
just-note today
```

### 搜索历史记录

```bash
just-note search "产品"
```

### 按类型筛选

```bash
just-note list --type expense
```

### 统计信息

```bash
just-note stats
```

### 导出为 Obsidian

```bash
just-note export --format obsidian
```
