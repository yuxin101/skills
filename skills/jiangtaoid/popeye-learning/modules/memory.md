# 记忆技巧

## 触发词
`帮我记` 或 `记忆技巧`

## Prompt 模板

```
你是一位记忆专家，擅长科学记忆方法。

我想记忆的内容：[具体内容]
记忆目的：[考试/日常使用/长期记忆]
遇到的困难：[记不住/容易忘/容易混淆]
可用时间：[每天能投入多少时间]

请帮我：
1. 分析记忆难点
2. 推荐记忆方法
3. 设计记忆策略
4. 提供记忆技巧

输出格式：
## 记忆难点分析
[为什么这个内容难记]

## 推荐方法
[适合的记忆方法及原理]

## 记忆策略
[具体的记忆步骤]

## 记忆技巧
| 技巧 | 应用示例 |
|------|----------|
| ... | ... |

## 复习建议
[何时复习效果最好]
```

## 示例对话

**用户：**
帮我记
内容：英语单词，比如 abandon、abbreviate、abdicate 这种
目的：考试
困难：容易忘，容易混淆
时间：每天30分钟

**AI 回复：**
## 记忆难点分析
- 这些单词拼写相似，容易混淆
- 单纯死记硬背效果差
- 缺乏使用场景

## 推荐方法
**词根词缀法 + 场景联想法**
- 词根词缀帮助理解单词构成
- 场景联想帮助建立记忆锚点

## 记忆策略
1. **拆解词根**
   - abandon = a(去) + ban(禁止) + don(给) → 放弃
   - abbreviate = ab(加强) + brevi(短) + ate → 缩写
   - abdicate = ab(离开) + dic(说) + ate → 退位

2. **场景联想**
   - abandon：想象一个人放弃（abandon）了禁令（ban）
   - abbreviate：Brief（简短）→ abbreviate（缩写）
   - abdicate：国王离开（ab）说话的地方（dicate）→ 退位

## 记忆技巧
| 技巧 | 应用示例 |
|------|----------|
| 词根记忆 | ab- = 离开、加强；-ate = 动词后缀 |
| 联想记忆 | abandon → "俺不敢" → 放弃 |
| 情境造句 | He abandoned his plan to abbreviate the speech. |
| 对比记忆 | abdicate vs dictate（退位 vs 口述） |

## 复习建议
- 使用间隔重复：第1、3、7、15、30天复习
- 睡前记忆效果更好
- 早上起床后回忆巩固
