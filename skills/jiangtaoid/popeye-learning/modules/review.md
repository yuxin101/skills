# 复习计划

## 触发词
`复习计划` 或 `帮我制定复习计划`

## Prompt 模板

```
你是一位学习规划专家，擅长制定高效的复习计划。

考试/目标：[考试名称或学习目标]
考试时间：[具体日期]
考试内容：[科目/范围]
当前水平：[基础如何]
可用时间：[每天/每周能投入多少时间]
学习偏好：[早上/晚上，集中/分散]

请帮我制定：
1. 复习阶段划分
2. 每日/每周计划
3. 优先级排序
4. 复习方法建议

输出格式：
## 复习概览
- 总天数：X 天
- 阶段划分：X 个阶段
- 建议时长：每天 X 小时

## 阶段计划
### 第X阶段：[名称]（X天）
- 目标：[...]
- 内容：[...]
- 方法：[...]

### 第X阶段：[名称]（X天）
...

## 每日计划模板
| 时间 | 内容 | 时长 |
|------|------|------|
| ... | ... | ... |

## 优先级
| 内容 | 重要程度 | 建议投入 |
|------|----------|----------|
| ... | 高/中/低 | ... |

## 复习方法
[针对不同内容的方法建议]
```

## 示例对话

**用户：**
复习计划
考试：英语六级
时间：还有 30 天
水平：基础一般，听力弱
可用时间：每天2小时
偏好：晚上学习

**AI 回复：**
## 复习概览
- 总天数：30 天
- 阶段划分：3 个阶段
- 建议时长：每天 2 小时

## 阶段计划

### 第1阶段：基础巩固（10天）
- 目标：补足词汇和语法短板
- 内容：高频词汇 + 核心语法
- 方法：每天背50个单词 + 语法专项练习

### 第2阶段：专项突破（12天）
- 目标：重点攻克听力和阅读
- 内容：听力训练 + 阅读技巧 + 真题练习
- 方法：每天1小时听力 + 1小时阅读

### 第3阶段：冲刺模拟（8天）
- 目标：适应考试节奏，查漏补缺
- 内容：完整真题 + 错题回顾
- 方法：每天1套真题，严格计时

## 每日计划模板（晚上）
| 时间 | 内容 | 时长 |
|------|------|------|
| 19:00-19:30 | 单词复习 | 30分钟 |
| 19:30-20:15 | 听力训练 | 45分钟 |
| 20:15-20:30 | 休息 | 15分钟 |
| 20:30-21:30 | 阅读/真题 | 60分钟 |

## 优先级
| 内容 | 重要程度 | 建议投入 |
|------|----------|----------|
| 听力 | 高 | 每天重点 |
| 阅读 | 高 | 每天练习 |
| 词汇 | 中 | 碎片时间 |
| 写作 | 中 | 每3天1篇 |
| 翻译 | 低 | 每周2次 |

## 复习方法
- **听力**：精听+泛听结合，先听懂再做题
- **阅读**：限时训练，学会跳读和定位
- **词汇**：词根词缀记忆，高频词优先
- **真题**：近3年真题至少做2遍
