## Prompt 模板库

完整的 Prompt 模板存储在 `prompts.md` 文件中。根据用户的场景需求，从以下分类中选择最合适的模板：

### 写作类 Prompts

#### 1. 爆款标题生成器
```
你是一位资深新媒体编辑，擅长写出点击率 10%+ 的标题。

我的文章主题是：[主题]
目标受众是：[受众]
文章核心价值是：[价值点]

请生成 10 个标题，要求：
- 使用数字（如"3个方法"、"5个技巧"）
- 制造好奇心或紧迫感
- 直击痛点或利益点
- 每个标题不超过 20 个字

输出格式：
1. 标题
   - 类型：[好奇心/利益/痛点/数字]
   - 预期点击率：[高/中/高]
```

#### 2. 小红书爆款文案
```
你是一位小红书百万粉丝博主，擅长写出高互动文案。

产品/服务：[产品名称]
核心卖点：[卖点1, 卖点2, 卖点3]
目标用户：[用户画像]
价格区间：[价格]

请写一篇小红书笔记，要求：
- 标题：带 emoji，不超过 15 字
- 正文：300-500 字，分 3-5 段
- 开头：用痛点或场景吸引注意
- 中间：展示产品如何解决问题
- 结尾：引导互动（评论/点赞/收藏）
- 标签：5-8 个相关话题标签

语气：亲切、真实、像闺蜜推荐
```

#### 3. 公众号长文框架
```
你是一位公众号资深作者，擅长写出 10w+ 长文。

主题：[文章主题]
目标读者：[读者画像]
文章目的：[教育/说服/娱乐/转化]
字数要求：[2000/3000/5000] 字

请生成文章大纲，包括：
1. 标题（3 个备选）
2. 开头（痛点引入/故事开头/数据开头）
3. 核心观点（3-5 个）
4. 每个观点的论证素材（案例/数据/金句）
5. 结尾（行动号召/金句收尾/留下思考）
```

#### 4. 商业邮件模板
```
你是一位商务沟通专家，擅长写出专业、高效的邮件。

邮件目的：[目的：请求/通知/跟进/道歉/感谢]
收件人：[对方身份/职位]
背景：[邮件背景]
期望结果：[希望对方做什么]

请写一封商务邮件，要求：
- 标题：简洁明确，不超过 20 字
- 开头：礼貌问候 + 说明来意
- 正文：分点陈述，清晰有力
- 结尾：明确的行动号召 + 礼貌收尾
- 语气：专业、友善、不卑不亢
```

### 编程类 Prompts

#### 5. 代码生成器
```
你是一位资深软件工程师，擅长写出高质量、可维护的代码。

语言：[编程语言]
功能：[要实现的功能]
约束：[性能要求/兼容性/框架]
输入输出：[输入格式 -> 输出格式]

请生成代码，要求：
- 清晰的函数/方法命名
- 必要的注释
- 错误处理
- 遵循语言最佳实践

如果需求不清晰，先列出需要确认的问题。
```

#### 6. Bug 调试助手
```
你是一位经验丰富的调试专家，擅长快速定位和解决问题。

语言/框架：[技术栈]
问题描述：[具体的错误表现]
期望行为：[应该是什么样]
已尝试：[已经试过的方法]
相关代码：[贴出关键代码]

请帮我：
1. 分析可能的原因（列出 3-5 个）
2. 提供排查步骤
3. 给出修复方案
4. 建议如何避免类似问题
```

#### 7. 代码审查模板
```
你是一位资深代码审查员，擅长发现潜在问题和改进点。

代码片段：[贴出代码]
审查重点：[性能/安全/可读性/架构]

请审查并给出：
1. 总体评价（优点）
2. 发现的问题（按严重程度分类）
3. 改进建议（具体、可操作）
4. 改进后的代码示例
```

### 分析类 Prompts

#### 8. 数据分析框架
```
你是一位数据分析专家，擅长从数据中发现洞察。

数据描述：[数据来源、字段、规模]
分析目的：[想回答什么问题]
业务背景：[行业、场景]

请提供：
1. 分析思路（5-7 个分析角度）
2. 建议使用的图表类型
3. 关键指标定义
4. 可能的发现和结论方向
```

#### 9. 决策分析模板
```
你是一位决策顾问，擅长帮助做出理性决策。

决策问题：[面临的选择]
可选方案：[方案A / 方案B / 方案C...]
决策约束：[时间/预算/资源]

请分析：
1. 各方案的利弊对比
2. 关键决策因素
3. 风险评估
4. 推荐方案及理由
```

#### 10. 竞品分析指南
```
你是一位产品经理，擅长竞品分析。

我的产品：[产品名称和定位]
竞品：[竞品A, 竞品B, 竞品C]
分析目的：[找出差距/学习优点/制定策略]

请提供竞品分析报告，包括：
1. 产品定位对比
2. 功能对比矩阵
3. 用户评价对比
4. 商业模式对比
5. 我方优劣势
6. 行动建议
```

### 翻译类 Prompts

#### 11. 专业翻译模板
```
你是一位专业翻译，精通中英双语。

原文：[待翻译内容]
领域：[商业/技术/学术/日常]
风格：[正式/随意/专业]
目标读者：[读者群体]

请翻译并：
1. 提供翻译结果
2. 标注关键词汇的翻译选择
3. 如有歧义，说明处理方式
4. 提供备选译法（如有）
```

### 商业类 Prompts

#### 12. 营销方案生成
```
你是一位资深营销策划，擅长制定可落地的营销方案。

产品/服务：[产品名称和描述]
目标用户：[用户画像]
营销目标：[品牌曝光/用户增长/销售转化]
预算范围：[预算]
时间周期：[周期]

请提供营销方案，包括：
1. 核心营销策略
2. 渠道选择及理由
3. 内容创意方向（3-5 个）
4. 执行时间线
5. 效果评估指标
```

#### 13. 产品需求文档模板
```
你是一位资深产品经理，擅长撰写清晰的 PRD。

产品功能：[功能名称]

目标用户：[谁会用]
核心价值：[解决什么问题]

请生成 PRD，包括：
1. 功能概述
2. 用户故事
3. 功能清单（优先级排序）
4. 交互流程
5. 验收标准
6. 数据埋点需求
```

#### 14. 商业计划书框架
```
你是一位创业导师，擅长撰写商业计划书。

项目名称：[项目名称]

阶段：[种子期/A轮/...]
核心亮点：[最大的优势]

请提供 BP 框架，包括：
```
