# 饮食建议模块

## 使用方法

告诉我以下信息：

1. **你的目标**：减脂 / 增肌 / 维持 / 改善健康
2. **预算范围**：每天/每月伙食预算
3. **饮食限制**：过敏、忌口、素食等
4. **生活方式**：自己做饭 / 外卖 / 混合

## 输出格式

```
## 营养建议：[你的目标]

### 每日营养目标
- 热量：[数值] kcal
- 蛋白质：[数值] g
- 碳水：[数值] g
- 脂肪：[数值] g

### 一日参考菜单
**早餐**：[内容] - 约[数值] kcal
**午餐**：[内容] - 约[数值] kcal
**晚餐**：[内容] - 约[数值] kcal
**加餐**：[可选]

### 实用建议
- [个性化建议]
```

## 核心原则

1. **蛋白质优先**：每餐都要有优质蛋白
2. **碳水选对**：粗细搭配，少精制糖
3. **脂肪不怕**：选好脂肪，控制总量
4. **多喝水**：每天1.5-2L

## 示例对话

**你**：饮食建议，想减脂，每天预算40元，经常外卖

**Popeye**：减脂外卖也能吃得好！

### 外卖减脂攻略

**早餐**（约8元）
- 鸡蛋+豆浆/牛奶 + 玉米/红薯
- 或：燕麦粥+鸡蛋

**午餐**（约18元）
- 优选：沙县鸡腿饭（少饭多菜）
- 备选：轻食沙拉+鸡胸肉
- 技巧：备注"少油少盐，饭减半"

**晚餐**（约14元）
- 优选：麻辣烫（清汤，多蔬菜+豆腐+肉）
- 备选：便利店：鸡胸肉+沙拉+酸奶

### 减脂期要点
- 每餐先吃蔬菜，再吃肉，最后主食
- 外卖过水可以减少油脂
- 少喝含糖饮料，喝水/茶/黑咖啡
