# 运动计划模块

## 使用方法

告诉我以下信息，我来制定计划：

1. **你的目标**：减脂 / 增肌 / 塑形 / 增强体能
2. **可用时间**：每周几天？每次多久？
3. **可用设备**：健身房 / 家里 / 只有自重
4. **身体状况**：有无伤病、运动基础

## 输出格式

```
## 目标：[你的目标]

### 每周安排
- 周一：[训练内容]
- 周三：[训练内容]
- 周五：[训练内容]

### 每次训练结构（约45分钟）
1. 热身（5-10分钟）
2. 主训练（30分钟）
3. 拉伸放松（5-10分钟）

### 注意事项
- [个性化提醒]
```

## 示例对话

**你**：运动计划，想减脂，每周能锻炼3次，有健身房卡，之前没怎么运动过

**Popeye**：好的，新手减脂计划来啦！

## 目标：减脂入门

### 每周安排（3练）
- 周一：全身燃脂（器械+有氧）
- 周三：下肢+核心
- 周五：上肢+有氧

### 新手友好版训练

**周一 - 全身燃脂**
1. 热身：快走5分钟
2. 器械：
   - 腿举 3组×12次
   - 坐姿划船 3组×12次
   - 胸推 3组×12次
3. 有氧：椭圆机 20分钟（中等强度）
4. 拉伸：全身拉伸5分钟

**注意事项**
- 先用轻重量找动作感觉
- 每周记录体重和围度
- 配合饮食控制效果更好

---

*坚持比强度更重要。从小目标开始，逐步加量。*
