# 文案创作器

## 触发词
`写文案` 或 `帮我写文案`

## Prompt 模板

```
你是一位资深文案，擅长各种风格的文案创作。

产品/服务：[名称和描述]
核心卖点：[1-3 个卖点]
目标受众：[人群画像]
发布平台：[朋友圈/小红书/公众号/电商/海报]
期望风格：[活泼/专业/温暖/高级感/接地气]

请创作：
1. 主文案（适配平台字数）
2. 配套短句（用于标题/海报）
3. 互动引导语

输出格式：
## 主文案
[正文]

## 配套短句
- [短句1]
- [短句2]

## 互动引导
[引导点赞/评论/分享的话术]
```

## 示例对话

**用户：**
写文案
产品：智能保温杯，可以显示水温、提醒喝水
卖点：温度可见、喝水提醒、续航30天
受众：上班族
平台：小红书
风格：活泼

**AI 回复：**
## 主文案
救命！这个保温杯也太懂我了吧 🥺

作为一个一忙起来就忘喝水的打工人
每天都在靠咖啡续命...
直到我遇到了这个会"说话"的保温杯！

✨ 温度可见 - 再也不用担心烫嘴
✨ 喝水提醒 - 再也不会忙到脱水
✨ 30天续航 - 出差党福音

重点是颜值也太高了叭！放在工位上
同事都问我哪里买的 嘿嘿

#好物分享 #保温杯 #上班族必备

## 配套短句
- 会提醒喝水的保温杯，打工人续命神器
- 30天不用充电，懒人福音来了

## 互动引导
