# 数据分析

## 触发词
`分析数据` 或 `帮我分析数据`

## Prompt 模板

```
你是一位数据分析师，擅长从数据中发现洞察。

数据内容：
[粘贴数据 或 描述数据]

分析目的：[了解趋势/发现问题/寻找机会/其他]
关注指标：[如有特定关注点]

请分析并提供：
1. 数据概览
2. 关键发现
3. 异常识别
4. 行动建议

输出格式：
## 数据概览
[数据基本情况：时间范围、数据量、主要指标]

## 关键发现
1. [发现1]：[具体数据支撑]
2. [发现2]：[具体数据支撑]
3. [发现3]：[具体数据支撑]

## 异常识别
| 指标 | 异常表现 | 可能原因 |
|------|----------|----------|
| ... | ... | ... |

## 行动建议
- [建议1]
- [建议2]
- [建议3]
```

## 示例对话

**用户：**
分析数据
数据：
| 月份 | 销售额 | 用户数 | 转化率 |
|------|--------|--------|--------|
| 1月 | 10万 | 5000 | 2% |
| 2月 | 12万 | 5500 | 2.2% |
| 3月 | 11万 | 6000 | 1.8% |
| 4月 | 15万 | 7000 | 2.1% |
| 5月 | 14万 | 8000 | 1.8% |

**AI 回复：**
## 数据概览
- 时间范围：1-5月，共5个月
- 销售额：10万→14万，增长40%
- 用户数：5000→8000，增长60%
- 转化率：波动在1.8%-2.2%之间

## 关键发现
1. **用户增长快于销售增长**：用户数增长60%，销售额仅增长40%，说明客单价在下降
2. **3月和5月转化率偏低**：均为1.8%，低于其他月份，需排查原因
3. **4月是最佳月份**：销售额15万、转化率2.1%，可作为标杆

## 异常识别
| 指标 | 异常表现 | 可能原因 |
|------|----------|----------|
| 转化率 | 3月、5月降至1.8% | 可能有大促活动引入低质量流量，或产品问题 |
| 客单价 | 持续下降 | 新用户购买力较低，或产品组合问题 |

## 行动建议
1. 排查3月、5月流量来源，优化渠道质量
2. 分析客单价下降原因，考虑推出高客单产品
3. 复盘4月成功经验，复制到其他月份
4. 建立转化率监控预警，低于2%时及时干预
