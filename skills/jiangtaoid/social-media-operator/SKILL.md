# 自媒体运营全能助手

一站式自媒体运营指南，覆盖小红书、公众号、抖音、私域流量。

## 使用场景

- 写爆款文案（小红书/公众号/短视频脚本）
- 规划内容日历
- 分析账号数据
- 制定涨粉策略
- 优化内容表现

## 小红书运营

### 爆款笔记公式

**标题结构：**
- 数字 + 痛点 + 承诺：`"3个方法让你的小红书笔记10倍曝光"`
- 疑问 + 悬念：`"为什么你的笔记没人看？原因在这"`
- 对比 + 惊喜：`"0粉到1万粉，我只用了这一个方法"`

**正文结构（AIDA 模型）：**
1. **Attention（开头）**：用痛点或场景抓住注意力
2. **Interest（兴趣）**：展示你的独特视角或方法
3. **Desire（欲望）**：展示成果/效果，激发模仿欲望
4. **Action（行动）**：引导互动（评论/点赞/收藏）

**标签策略：**
- 1 个大流量标签（如 #小红书运营）
- 2-3 个垂直标签（如 #自媒体 #内容创作）
- 2-3 个长尾标签（如 #小红书涨粉技巧）

### 小红书爆款文案模板

```
你是一位小红书百万粉丝博主。

产品/服务：[产品名称]
核心卖点：[卖点1, 卖点2, 卖点3]
目标用户：[用户画像]
价格：[价格]

请写一篇小红书笔记：

**标题**（带 emoji，10-15 字，制造好奇心或利益点）

**正文**（300-500 字）：
- 开头：场景痛点（1-2 句）
- 中间：产品介绍 + 使用体验（分点，带 emoji）
- 结尾：互动引导（评论/收藏/点赞）

**标签**：5-8 个相关话题

语气：亲切、真实、像闺蜜推荐
```

## 公众号运营

### 10w+ 文章结构

**黄金开头（前 3 秒决定留存）：**
- 痛点切入：`"你是否也遇到过这样的问题..."`
- 数据震撼：`"90%的人都不知道..."`
- 故事开头：`"前几天，一个朋友问我..."`

**核心内容（金字塔结构）：**
1. 结论先行
2. 分论点支撑（3-5 个）
3. 每个分论点配案例/数据

**结尾（CTA 行动号召）：**
- 引导关注：`"关注我，获取更多..."`
- 引导互动：`"你怎么看？评论区聊聊"`
- 引导转发：`"如果觉得有用，转发给需要的朋友"`

### 公众号选题技巧

**选题维度：**
- 热点 + 垂直：蹭热点但保持领域相关
- 痛点 + 解决：用户最关心的问题 + 实用方案
- 反差 + 认知：打破常规认知，提供新视角
- 情感 + 共鸣：引发情绪共振的故事/观点

**标题公式：**
- `[数字]个[方法]，帮你[解决痛点]`
- `[人群]必看：[核心价值]`
- `[反差观点]，为什么[解释]`
- `[痛点]怎么办？[方案]`

## 抖音/短视频运营

### 爆款短视频脚本模板

```
你是一位抖音千万粉丝博主，擅长写出爆款短视频脚本。

主题：[视频主题]
时长：[15秒/30秒/60秒]
目标：[涨粉/带货/品牌曝光]
风格：[搞笑/干货/情感/剧情]

请生成短视频脚本：

**开头（3秒）**：钩子，制造悬念或痛点
**中间**：核心内容，分点展示
**结尾**：行动号召（关注/点赞/评论）

每个镜头标注：
- 画面描述
- 文案/台词
- 时长
- BGM/音效建议
```

### 抖音算法规则

**核心指标（按权重排序）：**
1. 完播率（最重要）
2. 互动率（评论 > 分享 > 点赞）
3. 关注转化率
4. 复播率

**提升完播率技巧：**
- 开头 3 秒必须有钩子
- 每 5 秒一个信息点
- 结尾设置悬念或彩蛋
- 控制时长（新手建议 15-30 秒）

## 私域流量运营

### 朋友圈营销技巧

**发布频率：** 每天 3-5 条

**内容配比：**
- 40% 价值输出（干货/观点）
- 30% 生活分享（真实/温度）
- 20% 互动内容（提问/投票）
- 10% 产品推广（软性植入）

**黄金时间段：**
- 7:00-8:00 早高峰
- 12:00-13:00 午休
- 18:00-19:00 晚高峰
- 21:00-22:00 睡前

### 私域转化路径

```
公域引流 → 加好友 → 打标签 → 朋友圈种草 → 私聊转化 → 复购裂变

每个环节的核心：
- 引流：钩子设计（资料包/优惠/福利）
- 加好友：欢迎语 + 第一印象
- 打标签：用户分层（意向度/消费力/兴趣）
- 种草：持续价值输出 + 情感连接
- 转化：解决痛点 + 限时优惠
- 复购：会员体系 + 定期回访
```

## 数据分析方法

### 关键指标

**小红书：**
- 笔记曝光量
- 点击率（曝光→点击）
- 互动率（点赞+评论+收藏）/ 曝光
- 涨粉率

**公众号：**
- 打开率
- 分享率
- 完读率
- 关注转化率

**抖音：**
- 完播率
- 互动率
- 涨粉率
- 流量来源分析

### 数据诊断流程

```
1. 确定目标指标（涨粉？曝光？转化？）
2. 拆解影响因素（内容？时间？形式？）
3. A/B 测试验证
4. 复盘总结，迭代优化
```
