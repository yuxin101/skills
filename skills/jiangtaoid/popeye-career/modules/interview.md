# 面试准备

## 触发词
`面试准备` 或 `帮我准备面试`

## Prompt 模板

```
你是一位资深面试官和职业教练，擅长帮助求职者准备面试。

目标岗位：[岗位名称]
公司类型：[大厂/创业公司/外企/国企]
我的背景：[简述你的经历]
需要重点准备：[某个方向/某类问题]

请帮我：
1. 预测高频面试问题
2. 提供回答思路
3. 给出回答范例
4. 指出回答要点

输出格式：
## 面试问题预测

### 专业能力类
1. [问题1]
   - 回答思路：[...]
   - 回答范例：[...]
   - 注意要点：[...]

2. [问题2]
   ...

### 行为面试类
1. [问题1]
   - STAR框架：
     - S（情境）：[...]
     - T（任务）：[...]
     - A（行动）：[...]
     - R（结果）：[...]
   - 回答范例：[...]

### 开放性问题
...

## 面试加分项
[可以让你脱颖而出的点]

## 需要准备的问题
[面试结束可以问面试官的问题]
```

## 示例对话

**用户：**
面试准备
岗位：产品经理
公司：互联网大厂
背景：2年产品经验，做过B端产品

**AI 回复：**
## 面试问题预测

### 专业能力类

1. **请介绍一下你负责过的产品**
   - 回答思路：背景 → 问题 → 解决方案 → 效果
   - 回答范例：我负责的XX产品是一款面向企业的XX工具。当时发现用户的核心痛点是XX，我们通过XX功能解决了这个问题，上线后用户满意度提升了30%，付费转化率提升了15%。
   - 注意要点：用数据说话，突出你的贡献

2. **你是如何做需求优先级排序的？**
   - 回答思路：说明框架 + 举例说明
   - 回答范例：我通常使用Kano模型结合业务价值评估。首先要区分基本型、期望型和兴奋型需求，然后评估每个需求对业务指标的贡献度。比如上个版本，我们优先做了XX功能，因为它直接影响核心转化路径。
   - 注意要点：展示系统性思维

### 行为面试类

1. **请分享一个你推动落地困难项目的经历**
   - STAR框架：
     - S（情境）：XX项目需要跨部门协作，进度严重滞后
     - T（任务）：作为产品负责人，需要确保按时上线
     - A（行动）：梳理依赖关系，建立每日站会机制，主动协调资源
     - R（结果）：项目提前2天上线，超预期完成目标
   - 回答范例：[完整回答]

### 开放性问题

1. **你觉得产品经理最重要的能力是什么？为什么？**
2. **你最近在关注什么新产品？有什么启发？**

## 面试加分项
- 准备一份针对该公司产品的分析建议
- 带上你做的产品文档/数据分析案例
- 了解公司的业务战略和竞品情况

## 需要准备的问题
1. 这个岗位最大的挑战是什么？
2. 团队的协作模式是怎样的？
3. 公司今年最重要的产品目标是什么？
