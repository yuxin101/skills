# 预算规划模块

## 使用方法

告诉我：

1. **月收入**：到手工资+其他收入
2. **固定支出**：房租、贷款等必须支出
3. **储蓄目标**：想存多少钱
4. **生活方式**：你对生活品质的要求

## 输出格式

```
## 月度预算方案

### 收入分配（50/30/20法则调整版）

| 类别 | 金额 | 占比 | 说明 |
|------|------|------|------|
| 必要支出 | [金额] | 50% | 房租、水电、交通、基础餐饮 |
| 弹性支出 | [金额] | 30% | 娱乐、购物、外出就餐 |
| 储蓄投资 | [金额] | 20% | 应急基金、投资账户 |

### 各类别明细
**🏠 必要支出（[金额]）**
- 房租：[金额]
- 水电燃气：[金额]
- 交通：[金额]

**🎉 弹性支出（[金额]）**
- 娱乐：[金额]

- [具体建议]
```

## 示例对话

**你**：做预算，月收入到手8000，房租2500，想每月存3000

**Popeye**：

## 月度预算方案

收入8000，目标存3000（37.5%），储蓄率很棒！

### 收入分配

| 类别 | 金额 | 占比 |
|------|------|------|
| 必要支出 | 3500 | 44% |
| 弹性支出 | 1500 | 19% |
| 储蓄投资 | 3000 | 37% |

### 各类别明细

**🏠 必要支出（3500）**
- 房租：2500
- 水电燃气：150
- 交通：300（地铁/公交为主）
- 基础餐饮：550（工作日简单吃）

**🎉 弹性支出（1500）**
- 娱乐：400
- 购物：500
- 外出就餐：600

**💰 储蓄投资（3000）**
- 应急基金：1000（存满3-6个月支出后转投资）
- 投资账户：2000

### 执行建议
1. 开工资当天，先转3000到储蓄账户
2. 用记账App追踪弹性支出
3. 购物前问自己：需要还是想要？
4. 应急基金目标是存够3个月生活费（约15000）
