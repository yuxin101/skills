# 投资入门模块

## 使用方法

告诉我：

1. **投资经验**：小白 / 略懂 / 有经验
2. **可用资金**：打算用多少钱投资
3. **投资期限**：短期（<1年）/ 中期（1-3年）/ 长期（3年以上）
4. **风险承受**：保守 / 稳健 / 激进

## 核心原则

1. **只用闲钱**：不影响生活的钱才能投资
2. **先学习再行动**：不懂不投
3. **分散投资**：不把鸡蛋放一个篮子
4. **长期主义**：时间是最大的复利

## 示例对话

**你**：投资建议，小白，有1万闲置资金，想长期投资，偏稳健

**Popeye**：

## 投资入门指导

### 你的画像
- 经验水平：新手小白
- 资金规模：1万
- 投资期限：长期（3年以上）
- 风险偏好：稳健型

### 适合你的投资路径

**阶段1：打基础（现在开始）**
- 开立证券账户（选择大券商）
- 开通货币基金（如余额宝）作为现金管理
- 开始学习：读书《指数基金投资指南》

**阶段2：起步投资（了解后开始）**
- 建议：宽基指数基金定投
- 方式：每月定投1000-2000
- 标的：沪深300、中证500等指数基金
- 策略：长期持有，不择时

**阶段3：逐步进阶（1-2年后）**
- 学习更多资产类别
- 了解债券基金、REITs等
- 建立自己的资产配置比例

### 1万元的分配建议
- 应急备用：3000（放货币基金）
- 学习储备：2000（买书/课程）
- 启动投资：5000（开始定投体验）

### 风险提醒
⚠️ 投资有风险，本金可能亏损
⚠️ 不要追涨杀跌，不要借钱投资
⚠️ 本建议仅供学习，不构成投资建议

### 推荐学习资源
- 书：《指数基金投资指南》
- 书：《小狗钱钱》（入门读物）
- 网站：且慢、蛋卷基金（学习社区）
