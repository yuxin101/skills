# 记账分析模块

## 使用方法

发送你的消费记录，我来分析：

**方式1：粘贴记录**
```
记账分析
3/1 午餐 35
3/1 咖啡 28
3/2 打车 45
3/2 晚餐 68
...
```

**方式2：描述概况**
```
记账分析
我每月大概花销：
- 房租：3000
- 吃饭：2000
- 交通：500
- 购物：1000
- 其他：500
```

## 输出格式

```
## 消费分析报告

### 总览
- 本期总支出：[金额]
- 最大支出类别：[类别]
- 可优化空间：约[金额]

### 支出结构
🏠 住房：[金额] ([占比]%)
🍜 餐饮：[金额] ([占比]%)
🚗 交通：[金额] ([占比]%)
🛒 购物：[金额] ([占比]%)
📦 其他：[金额] ([占比]%)

### 发现的问题
1. [问题1]
2. [问题2]

### 优化建议
- [建议1]：可省约[金额]/月
- [建议2]：可省约[金额]/月

### 储蓄潜力
如果按建议执行，每月可多存：[金额]
```

## 示例对话

**你**：
记账分析
房租3500
外卖2000
打车500
奶茶咖啡400
网购1500
娱乐800

**Popeye**：

## 消费分析报告

### 总览
- 月总支出：8700元
- 最大支出：房租（40%）
- 可优化空间：约1500元/月

### 支出结构
🏠 住房：3500 (40%)
🍜 餐饮：2000 (23%)
🚗 交通：500 (6%)
☕ 饮品：400 (5%)
🛒 网购：1500 (17%)
🎮 娱乐：800 (9%)

### 发现的问题
1. 外卖占比高，且有奶茶咖啡额外支出
2. 网购无明确预算，容易冲动消费

### 优化建议
- 🍜 减少外卖，每周自己做饭2-3次：省约600/月
- ☕ 奶茶咖啡减半或自带：省约200/月
- 🛒 网购设月度上限（如800）：省约700/月

### 储蓄潜力
执行以上建议，每月可多存：**1500元**
