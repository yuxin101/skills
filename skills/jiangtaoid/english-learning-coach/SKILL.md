## 口语提升

### 影子跟读法

```
方法：听一句 → 模仿语音语调 → 录音对比 → 改进

推荐材料：
- VOA 慢速英语（初级）
- TED 演讲（中级）
- 英文播客（高级）

练习步骤：
1. 选 1-2 分钟音频
2. 听完整遍，理解内容
3. 逐句跟读，模仿语调
4. 录音对比，找出差距
5. 重复直到流畅
```

### 口语练习 Prompt

```
You are my English conversation partner.

Topic: [话题]
My level: [初级/中级/高级]

Rules:
1. Ask me questions about this topic
2. Correct my grammar mistakes
3. Suggest more natural expressions
4. Keep the conversation going

Let's start!
```

### 常用口语表达示例

**表达观点：**
- In my opinion, ...
- From my perspective, ...
- I strongly believe that ...
- I'm not sure about that, but ...

**同意/不同意：**
- I couldn't agree more.
- That makes sense.
- I see your point, but ...
- I respectfully disagree.

**请求澄清：**
- Could you elaborate on that?
- What do you mean by ...?
- So you're saying that ...?
- Can you give me an example?

## 听力提升

### 听力训练层次

```
Level 1：听懂大意
- 主旨是什么？
- 有几个人？在谈论什么？

Level 2：听懂细节
- 具体时间、地点、数字
- 具体观点、原因

Level 3：听懂隐含
- 说话人态度
- 言外之意
- 文化背景
```

### 精听训练法

```
材料选择：1-2 分钟，适合自己水平的音频

步骤：
1. 泛听 1 遍，了解大意
2. 逐句听写，反复播放
3. 对照原文，标记错误
4. 分析错误原因
   - 发音连读
   - 生词
   - 语速
5. 再次听写，直到全对
```

### 推荐听力资源

| 级别 | 资源 | 特点 |
|------|------|------|
| 初级 | VOA 慢速、英语儿歌 | 语速慢、词汇简单 |
| 中级 | TED、BBC Learning | 话题丰富、发音标准 |
| 高级 | 英文播客、美剧 | 真实语境、地道表达 |

## 阅读提升

### 阅读方法

**SQ3R 法：**
- Survey（浏览）：看标题、小标题、首尾段
- Question（提问）：这篇文章要回答什么问题？
- Read（阅读）：带着问题阅读
- Recite（复述）：用自己的话总结
- Review（复习）：回顾关键点

### 生词处理策略

```
不要查所有生词！优先级：

1. 反复出现的词 → 必须查
2. 影响理解的词 → 必须查
3. 其他词 → 猜意思，继续读

猜词技巧：
- 通过上下文猜测
- 通过词根词缀猜测
- 通过同义词/反义词推断
```

### 推荐阅读材料

| 级别 | 材料 | 来源 |
|------|------|------|
| 初级 | 分级读物、简易新闻 | News in Levels |
| 中级 | 英文杂志、博客 | Medium、The Economist |
| 高级 | 英文原版书、论文 | Goodreads、Google Scholar |

## 写作提升

### 写作结构模板

**议论文：**
```
Introduction（引入）
├── Hook（吸引注意）
├── Background（背景）
└── Thesis（论点）

Body（正文）
├── Topic Sentence（主题句）
├── Supporting Details（支撑细节）
└── Concluding Sentence（小结句）

Conclusion（结论）
├── Restate Thesis（重申论点）
└── Call to Action（行动号召）
```

**邮件：**
```
Subject: [清晰的主题]

Dear [名字],

[开头：说明写信目的]

[正文：分点说明]
1. ...
2. ...
3. ...

[结尾：行动号召]
Please let me know if you have any questions.

[签名]
Best regards,
[你的名字]
```

### 写作练习 Prompt

```

Level: [初级/中级/高级]

1. Give me a writing task

3. Correct grammar errors

```

## 考试准备

### 雅思备考

**口语：**
- Part 1：日常话题，30-40 字回答
- Part 2：话题卡，1-2 分钟独白
- Part 3：深度讨论，4-5 句回答

**写作：**
- Task 1：图表描述，150+ 字
- Task 2：议论文，250+ 字

### 托福备考

**口语：**
- 4 道题，17 分钟
- Independent（自由表达）
- Integrated（综合任务）

**写作：**
- Integrated（阅读+听力→写作）
- Independent（独立写作）
