# BettaFish 情感分析指南

本指南说明 BettaFish 舆情分析 Skill 的情感分析框架和标准。

## 情感分类体系

### 基础情感极性

| 极性 | 标签 | 说明 | 典型表达 |
|------|------|------|----------|
| 正面 | `positive` | 积极、满意、推荐 | 喜欢、好用、推荐、惊喜 |
| 负面 | `negative` | 消极、失望、批评 | 失望、垃圾、避雷、踩雷 |
| 中性 | `neutral` | 客观、一般、观望 | 一般、还行、观望、客观 |

### 细分情绪类型

**正面情绪**：
- **喜悦 (joy)** - 开心、高兴、兴奋、愉快
- **信任 (trust)** - 信赖、认可、放心、安心
- **期待 (anticipation)** - 盼望、憧憬、看好、希望
- **赞赏 (admiration)** - 佩服、称赞、推荐

**负面情绪**：
- **愤怒 (anger)** - 生气、气愤、恼火、愤慨
- **失望 (disappointment)** - 失落、遗憾、沮丧
- **担忧 (worry)** - 担心、焦虑、忧虑、害怕
- **厌恶 (disgust)** - 反感、讨厌、恶心
- **惊讶-负面 (surprise-)** - 震惊（负面）

**复杂情绪**：
- **矛盾 (conflicted)** - 既有好评也有差评
- **观望 (wait-and-see)** - 不确定、等待更多信息

## 情感强度评估

| 强度等级 | 置信度范围 | 特征 |
|----------|-----------|------|
| 极强 | 0.90-1.00 | 感叹号密集、极端词汇、情绪词叠加 |
| 强 | 0.70-0.90 | 明确表态、强烈形容词、强调词 |
| 中 | 0.40-0.70 | 一般评价、普通表达 |
| 弱 | 0.20-0.40 | 轻微倾向、委婉表达 |
| 无 | 0.00-0.20 | 客观陈述、无明显倾向 |

| 价格 | 价格感知、性价比 | 价格、贵、便宜、性价比 |
| 服务 | 客服、售后、体验 | 服务、客服、售后、态度 |

## 行业特定情感词库

### 汽车行业

**正面**：
- 操控性好、加速快、续航扎实、内饰精致、性价比高、智能化程度高

**负面**：
- 续航虚标、异响、车机卡顿、售后服务差、降价快、做工粗糙

### 美妆护肤

**正面**：
- 效果好、显白、持久、温和、性价比高、种草

**负面**：
- 过敏、假白、脱妆快、智商税、踩雷、鸡肋

### 餐饮食品

**正面**：
- 好吃、正宗、性价比高、干净卫生、值得排队

**负面**：
- 难吃、不新鲜、服务态度差、踩雷、网红但难吃

### 电子产品

**正面**：
- 流畅、清晰、续航好、设计美、真香、黑科技

**负面**：
- 卡顿、发热、死机、品控差、割韭菜

## 情感分析注意事项

### 1. 语境理解

同一词汇在不同语境下情感可能不同：

| 词汇 | 正面语境 | 负面语境 |
|------|---------|---------|
| 绝了 | 太棒了，绝了！ | 太差了，绝了... |
| 真香 | 从拒绝到认可 | 讽刺用法 |
| yyds | 极高赞美 | - |
| 6 | 表示赞赏 | 表示无语/讽刺 |

### 2. 反讽识别

- "太好了，又涨价了" → 表面正面，实际负面
- "这就是你们说的'优质'服务？" → 反问表否定
- "真是'惊喜'不断呢" → 引号表讽刺

### 3. 情绪混合

用户可能同时表达多种情绪：
- "产品很好，但是价格太贵了" → 产品(正面) + 价格(负面)
- "物流很快，包装却破了" → 物流(正面) + 包装(负面)

### 4. 否定转换

否定词会翻转情感：
- "不好" → 负面（好 → 正面）
- "不是很喜欢" → 中性偏负
- "不得不服" → 正面（双重否定）

## 情感趋势类型

| 趋势类型 | 描述 | 示例 |
|---------|------|------|
| 持续正面 | 口碑稳定，偶有波动 | 长期优质品牌 |
| 持续负面 | 存在系统性问题 | 质量门事件 |
| V型反转 | 危机应对成功，口碑回升 | 成功的危机公关 |
| 倒V型 | 事件发酵，口碑恶化 | 负面事件曝光 |
| 波动型 | 受外部因素影响，情绪反复 | 争议性话题 |

## 实战技巧

### 1. 先抓大放小
先判断整体情感极性，再分析具体情绪类型和方面级情感。

### 2. 关注极端评价
高情绪强度（置信度 > 0.8）的评价往往更有参考价值。

### 3. 对比验证
单一平台的情感可能偏差，多平台对比更准确。

### 4. 结合上下文
脱离语境的情感判断可能失真，需要看前后文。

### 5. 量化与质性结合
- 量化：统计正/负/中性比例
- 质性：分析具体情绪类型和驱动因素
