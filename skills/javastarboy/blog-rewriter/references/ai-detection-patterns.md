# AI 痕迹识别与优化指南

## 常见 AI 写作特征

### 1. 过度使用连接词

**AI 特征：**
- 首先...其次...最后...
- 综上所述...
- 总而言之...
- 值得注意的是...
- 需要强调的是...

**优化方案：**
- 直接用小标题分隔
- 用自然过渡句
- 直接说重点，不要铺垫

### 2. 人称使用问题

**AI 特征：**
- "让我们一起..."
- "我们可以..."
- "笔者认为..."

**优化方案：**
- 用"你"直接对话读者
- 直接陈述，省略主语
- 用第一人称"我"增加真实感

### 3. 句式过于工整

**AI 特征：**
- 排比句过多
- 每段长度相似
- 结构过于对称

**优化方案：**
- 打乱节奏，长短句交错
- 增加口语化表达
- 加入不完整句、感叹句

### 4. 缺乏情绪表达

**AI 特征：**
- 语气平淡
- 没有个人观点
- 过于客观中立

**优化方案：**
- 加入感叹号、问号
- 表达明确立场
- 使用情绪化词汇

### 5. 陈词滥调

**AI 特征：**
- "在当今数字化时代..."
- "随着科技的飞速发展..."
- "这是一个值得探讨的问题..."

**优化方案：**
- 直接切入主题
- 用具体数据/案例开头
- 用悬念或反差吸引注意

## 人类写作特征

| 特征 | 表现 |
|------|------|
| 个人经历 | "我上周遇到..." |
| 具体细节 | 时间、地点、人名 |
| 情绪波动 | 兴奋、疑惑、吐槽 |
| 不完美 | 口语化、省略句 |
| 观点鲜明 | 敢说、有态度 |

## 优化检查清单

- [ ] 删除所有"首先/其次/最后"
- [ ] 删除"综上所述/总之"
- [ ] 检查是否有个人经历/观点
- [ ] 句式是否有变化（长短交错）
- [ ] 是否有情绪表达（感叹/疑问）
- [ ] 开头 3 句话是否足够吸引人
- [ ] 是否像朋友聊天而非报告
