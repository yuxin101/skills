# CPBL Skill - 中華職棒資訊查詢 ⚾🏆

查詢中華職棒 CPBL 比賽結果、賽程、戰績、球員數據。支援所有賽事類型（例行賽/熱身賽/明星賽/季後賽等）。

## 資料來源

| 來源 | 用途 |
|------|------|
| CPBL 官方隱藏 API | 比數、賽程、球員數據 |
| 台灣棒球維基館 | 年度 MVP、獎項、歷史紀錄 |

## 功能

- ⚾ 比賽結果查詢（含勝敗投手、MVP）
- 📅 賽程查詢（日期、球隊過濾）
- 📊 球員數據（打擊/投球排行榜）
- 🏆 年度獎項（MVP、新人王等）
- ⚠️ 戰績排名（API 受限）

## 授權

僅供學習和個人使用。資料來源為 CPBL 官網與台灣棒球維基館，請遵守各官方使用條款。
