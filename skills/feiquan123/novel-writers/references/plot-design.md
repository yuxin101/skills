# 情节设计指南

## 一、番茄都市科幻的情节公式

**核心公式：日常困境 → 科幻介入 → 爽点释放 → 新问题出现**

重复这个循环，每3-5章一个小高潮，每15-20章一个大高潮。

## 二、大纲设计框架

### 2.1 开篇设计（第1-5章）

**第1章：黄金开局**
- 前3000字必出科幻爽点
- 主角现状：普通打工族/学生，但有现实困境（缺钱、单身、职场受挫）
- 触发事件：获得能力/发现道具/遇到异常
- 章末钩子：科幻元素带来第一个疑问或危机

**第2-3章：能力探索**
- 测试能力边界和限制
- 第一次用能力解决小问题
- 引入第一个配角（协助者或观察者）
- 埋下世界观伏笔（这个能力不是唯一的）

**第4-5章：首次冲突**
- 遭到反派势力注意或试探
- 用能力化解小危机
- 确立主线目标（赚钱/复仇/守护/探索）

### 2.2 主线推进（第6-50章）

**每5章一个节点：**
- 第6-10章：能力进阶 + 第一个小反派
- 第11-15章：世界观展开 + 更大势力登场
- 第16-20章：主角卷入大事件 + 第一次高潮
- 第21-30章：战利消化 + 新能力解锁 + 感情线/事业线推进
- 第31-40章：中段高潮 + 主要反派现身
- 第41-50章：第一阶段完结 + 为下一阶段铺路

### 2.3 爽点分布密度

**频率控制：**
- 每1-2章至少1个小爽点（打脸、装逼、赚钱）
- 每5章1个中等爽点（解决危机、反杀、获得新道具）
- 每15-20章1个大爽点（击败大反派、突破能力、世界观揭秘）

**爽点类型矩阵：**

| 爽点类型 | 频率 | 示例 |
|---------|------|------|
| 能力装逼 | 每2-3章 | 主角用超能力解决普通人无法处理的问题 |
| 扮猪吃虎 | 每5章 | 被轻视后展示实力，震惊配角 |
| 财富逆袭 | 每10章 | 从穷小子月入三千到通过能力月入十万 |
| 情感收获 | 每15章 | 收获爱情/友情/家庭温暖 |
| 身份反转 | 每20章 | 从底层打工者变成某领域大佬 |

## 三、科幻设定设计原则

### 3.1 落地化三大场景

**场景1：职场/校园**
```
科幻设定：记忆回溯能力
落地方案：主角在互联网公司当审核员，发现同事偷删他的业绩，
用能力回溯屏幕操作记录，当众揭穿，升职加薪。
```

**场景2：居住社区**
```
科幻设定：空间折叠技术
落地方案：主角在城中村租房，房间只有15平米，用能力把内部
空间折叠成100平米豪华公寓，然后邀请朋友来"参观"，震惊全场。
```

**场景3：城市公共空间**
```
科幻设定：预知未来5分钟
落地方案：坐地铁时预知到有人要抢劫，提前躲开，顺便录视频
报警，成为"正义市民"，上本地新闻。
```

### 3.2 能力设计四要素

每个科幻设定必须回答以下4个问题：

**1. 来源是什么？**
- 合理来源：外星科技/未来道具/基因变异/系统觉醒/远古传承
- 避免无缘无故的能力（"我就是会了"不行）

**2. 能做什么？**
- 具体功能：能看数据流/能操控电子设备/能预知危险
- 避免全能设定（"什么都能干"太假）

**3. 限制是什么？**
- 能量限制：使用次数/体力消耗/冷却时间
- 条件限制：需要接触/需要特定场景/有时间限制
- 后果限制：使用后副作用/会引来关注/有道德风险
- **重要：限制比能力更让读者有代入感**

**4. 代价是什么？**
- 能力不是免费的，每个使用都要付出代价
- 代价类型：社会代价（暴露能力被追杀）、身体代价（头痛、晕厥）、
  道德代价（侵犯隐私）、资源代价（需要钱维护设备）

### 3.3 设定示例：完整的科幻能力

**能力名称：真实之眼（能看到物品的"历史"和"现状"）**

**来源：**
半年前在旧货市场买了一个老花镜，戴上后发现自己能看到物品的"信息标签"，摘下来就没有了。后来发现是眼镜上沾了一种特殊的陨石粉末。

**能做什么：**
- 看食物：显示保质期、真伪、是否有毒
- 看人：显示健康状态、近期情绪波动、是否说谎
- 看物品：显示制造时间、使用痕迹、隐藏功能

**限制是什么：**
- 每天只能使用30次，超过会头痛欲裂
- 只能看到表层信息，不能深入挖掘（比如能看出"这人撒谎"，但看不出"撒谎的具体原因"）
- 对非生命物体有效，对人和动物只能看基础状态

**代价是什么：**
- 使用次数多了后，主角开始分不清现实和信息，偶尔出现幻觉
- 一个神秘组织正在寻找这个眼镜，主角已经被盯上了

## 四、钩子设计技巧

### 4.1 章末钩子类型库

**类型1：新危机出现（最常用）**
> 就在陈默以为可以收工回家时，他的手机突然响了。来电显示："未知号码"。
> 接通后，一个冰冷的电子合成音说道："陈先生，我们注意到了您的能力。希望您能考虑一下我们的提议——当然，您也可以拒绝。但您的家人，可能就不会那么幸运了。"
> 电话挂断。陈默盯着手机屏幕，发现刚才的通话记录消失了。
> 就像没发生过一样。

**类型2：设定反转（颠覆认知）**
> "你真的以为，这个系统是随机选择的宿主？"老男人突然笑了起来，笑得有些诡异，"陈默，有没有人告诉过你，你的左眼瞳孔里有一道细小的裂纹？"
> 陈默下意识摸了摸眼睛。
> "那是五年前的事了吧？"老男人继续说，"你在那次车祸中失去了左眼，现在的，是义眼。但你有没有想过……那个义眼，到底是谁给你的？"

**类型3：角色背叛（信任崩塌）**
> "对不起。"李雪的声音在颤抖，"他们抓住了我弟弟，如果我不配合……"
> 陈默盯着这个曾经帮助过自己的女人，突然意识到，那个所谓的"神秘组织"，离自己比想象中更近。
> "没关系。"陈默笑了笑，"但你有没有想过，万一他们把你弟弟已经……"
> "什么意思？"
> "我刚才查了，你弟弟三年前就死了。"

**类型4：道具失控（意想不到）**
> 林晨把那枚硬币收进口袋，正准备离开。突然，口袋里的硬币开始发烫，烫得他不得不掏出来。
> 硬币悬浮在空中，表面浮现出一行字："警告：检测到高维生物接近，建议立即逃跑。"
> "什么高维生物？"林晨皱眉。
> 硬币没反应。三秒后，补充显示："就是天花板上的那个。"
> 林晨抬头。
> 天花板正在融化，像被某种看不见的火焰灼烧。

### 4.2 钩子强度分级

**弱钩子（能猜到下一步）：**
- 主角决定明天去某地
- 主角准备提升能力

**中钩子（有期待但不急迫）：**
- 主角收到神秘短信
- 有人开始调查主角

**强钩子（必须看下一章）：**
- 主角被反派包围，生死未卜
- 能力突然失控，可能引发灾难
- 重要角色被抓/失踪
- 设定出现重大反转（原来XXX是XXX）

**建议：每5章中至少2个强钩子**

## 五、冲突设计模板

### 5.1 个人层面冲突（最常见）

**冲突公式：主角需求 vs 现实阻碍**
- 需求：赚钱 → 阻碍：被老板压榨/被同行排挤
- 需求：爱情 → 阻碍：配角的误解/反派从中作梗
- 需求：守护家人 → 阻碍：家人被威胁/能力不足

**解决路径：**
1. 主角用能力初步解决（小爽点）
2. 引发更大问题（冲突升级）
3. 主角能力进化或获得新道具
4. 最终解决（中/大爽点）

### 5.2 社会层面冲突

**冲突类型：**
- 主角 vs 职场/学校环境（被排挤、打压）
- 主角 vs 犯罪组织（被追杀、勒索）
- 主角 vs 政府机构（被监视、征召）

**解决原则：**
- 不要一味的对抗，可以合作、利用、反间
- 反派不要太蠢，要有智商和动机
- 主角不是无敌的，需要智慧和盟友

### 5.3 世界观层面冲突

**大冲突设计：**
- 表世界都市 vs 里世界科幻（普通人类不知道超凡者存在）
- 秩序 vs 混乱（主角的能力可能改变社会结构）
- 守护 vs 利用（有人想保护世界，有人想统治世界）

**展示节奏：**
- 前10章：主角只知道自己的能力，不知道大冲突
- 10-30章：逐渐发现更大的世界
- 30章后：卷入大冲突，成为关键人物

## 六、伏笔埋设技巧

### 6.1 伏笔类型

**道具伏笔：**
- 开篇：主角捡到一个奇怪的道具，不知道用处
- 中段：道具在关键时刻派上用场
- 高潮：道具的真实来源和用途揭晓

**角色伏笔：**
- 开篇：某个路人角色说了句奇怪的话
- 中段：主角偶然再遇到这个角色
- 高潮：原来这个角色是反派/重要人物

**设定伏笔：**
- 开篇：主角的能力有一个奇怪的副作用
- 中段：副作用越来越严重
- 高潮：原来副作用是能力进化的前兆

### 6.2 埋设原则

**1. 自然不刻意**
```
❌ 强行埋伏笔：
主角在抽屉里发现了一张纸条，上面写着："记住，关键时刻打开。"
然后主角就把纸条收起来，忘了这回事，直到50章后才想起来。

✅ 自然埋伏笔：
主角帮奶奶整理老屋，翻出一个旧铁盒，里面有个生锈的金属圆盘，
上面刻着看不懂的符号。奶奶说："这是你爷爷留下的，说将来有一天
你会用到它。"主角当玩笑听了，随手放进抽屉。
```

**2. 有回收价值**
不是所有事情都需要伏笔，只有重要设定才需要。
过度使用伏笔会让读者疲惫。

**3. 回收要有惊喜**
```
❌ 无惊喜回收：
主角30章前捡了一把钥匙，50章后打开门，里面果然是宝藏。

✅ 有惊喜回收：
主角30章前捡了一把钥匙，50章后打开门，里面是一个空房间，
墙上贴着他自己的照片——他的一举一动都被监视了。
这把钥匙，是反派故意让他捡到的。
```

## 七、节奏控制技巧

### 7.1 张弛节奏

**5章循环：**
- 第1章：日常 + 科幻介入（张）
- 第2章：能力探索 + 小爽点（张）
- 第3章：都市日常 + 情感戏（弛）
- 第4章：小冲突爆发（张）
- 第5章：解决冲突 + 埋新钩子（弛中带张）

### 7.2 强弱交替

**强度等级：**
- 1级：日常对话、内心戏
- 2级：小矛盾、小打脸
- 3级：中冲突、配角危机
- 4级：大冲突、主角生死危机
- 5级：高潮、世界观冲击

**分布建议：**
- 连续3级以上高强度不超过5章
- 每5章必须穿插至少1章1-2级内容
- 大高潮（5级）前后必须有低强度缓冲

## 八、常见情节陷阱

### 8.1 避免的套路

❌ **机械降神：**
> 主角被打得半死，突然来了个神秘人救场，没有任何铺垫。
✅ **合理解决：**
> 主角被打得半死时，想起之前埋的伏笔（比如某个道具能力），用智慧反杀。

❌ **能力滥用：**
> 主角遇到任何问题都用能力解决，没有代价。
✅ **能力克制：**
> 主角能不用能力就不用，因为每次使用都有代价（头痛、暴露风险、道德负担）。

❌ **降智打击：**
> 反派明明能秒杀主角，但非要废话，给主角时间反杀。
✅ **智商在线：**
> 反派也很聪明，主角能赢是因为利用了信息差、环境优势、或者反派的轻敌（合情合理）。

### 8.2 逻辑检查清单

每次设计完情节后，用以下清单自检：

- [ ] 主角的每个重要决定都有动机？
- [ ] 反派的行动符合其利益和性格？
- [ ] 能力的使用遵守了之前设定的限制？
- [ ] 配角的行为符合其性格，没有工具人感？
- [ ] 世界观设定前后一致，没有吃书？
- [ ] 时间线合理，没有逻辑矛盾？
- [ ] 情感转折有铺垫，不突兀？
- [ ] 钩子有吸引力，但不刻意制造悬念？

## 九、实战大纲示例

### 示例：《时空快递员》前50章大纲

**开篇（1-5章）**
- 第1章：外卖员陈默获得时空快递系统，首次接单（穿越到5分钟前送外卖）
- 第2章：测试能力，发现限制（每次消耗体力，穿越距离不超过10公里）
- 第3章：第一次用能力赚钱（帮人送"不可能送到的订单"），引起注意
- 第4章：遭同行嫉妒，设局陷害，用能力化解（小爽点）
- 第5章：系统发布新任务："前往废弃工厂，有特殊包裹等待"，结尾到达工厂

**前期发展（6-15章）**
- 第6章：在工厂遇到第一个组织成员"观察者"，得知自己不是唯一的异能者
- 第7-8章：观察者试探主角，主角用智慧周旋，暂时脱身
- 第9-10章：能力进阶（可以穿越到10年前），用能力解决童年遗憾（情感爽点）
- 第11-12章：神秘组织正式接触，邀请加入，主角拒绝
- 第13-15章：组织施压（主角家人被威胁），主角反击，第一次小高潮

**中期发展（16-30章）**
- 第16-20章：主角发现组织真实目的（收集异能者做实验），卷入大事件
- 第21-25章：结识盟友（其他异能者），组建小团队
- 第26-30章：中段高潮，团队突袭组织基地，救出被困异能者

**后期发展（31-50章）**
- 第31-35章：战后休整，主角能力再次进化，感情线推进
- 第36-40章：组织幕后BOSS现身，世界观大揭秘（异能来源）
- 第41-45章：最终决战准备，主角团队集结
- 第46-50章：第一阶段完结，击败BOSS，但引出更大的威胁（第二阶段钩子）

## 十、AI辅助创作提示

当让AI辅助设计情节时，使用以下提示模板：

```
请为我设计一个都市科幻小说的大纲，要求：

1. 背景：[具体城市/职业/社会阶层]
2. 科幻设定：[具体能力/道具/规则]
3. 主角设定：[普通身份 + 特殊能力的反差]
4. 核心冲突：[个人/社会/世界观层面]
5. 目标读者：[番茄小说读者，喜欢快节奏、强爽点]
6. 风格要求：幽默风趣、逻辑合理、科幻落地日常

请输出：
- 前50章的章节大纲
- 标注每章的爽点类型和强度
- 标注每章结尾的钩子类型
- 标注关键伏笔的埋设和回收点
```

然后人工审核和改写，确保符合番茄风格。
