# 角色塑造指南

## 一、核心原则

**番茄都市科幻的角色公式：鲜明个性 + 科幻设定 + 现实困境**

每个重要角色都必须同时具备"人性"（普通人的情感和烦恼）和"科幻性"（与异能/科技/超自然力量的关联）。

## 二、主角设计模板

### 2.1 反差公式（最重要）

**公式：普通身份 + 特殊能力 + 现实困境**

#### 示例1：《信息全知者》式主角
```
普通身份：农村少年，轻度智力障碍，视力差
特殊能力：能看到别人看不到的信息（实际上是数据流）
现实困境：被同学嘲笑、跟不上学业、爷爷担心他未来
反差爽点：傻小子其实是天才，只是世界不理解他
```

#### 示例2：都市打工人主角
```
普通身份：互联网公司审核员，月薪4000，每天看垃圾信息
特殊能力：能通过物品看到"历史"和"真实状态"
现实困境：没钱交房租、单身狗、被同事排挤
反差爽点：底层打工仔能用能力解决大佬解决不了的问题
```

#### 示例3：学生主角
```
普通身份：高二学生，成绩中等，体育废柴
特殊能力：能暂停时间5秒（每天3次）
现实困境：被学霸碾压、被女神无视、被体育课欺负
反差爽点：废柴学生用能力逆袭，震惊全校
```

### 2.2 主角个性维度表

每个主角需要在以下5个维度有明确设定：

| 维度 | 问题 | 回答示例 |
|------|------|----------|
| 核心欲望 | 他最想要什么？ | 钱（脱离贫困）/认可（证明自己）/守护（保护家人） |
| 核心恐惧 | 他最怕什么？ | 穷一辈子/被看不起/失去亲人 |
| 道德底线 | 他绝对不会做什么？ | 出卖朋友/伤害无辜/背叛信任 |
| 性格优点 | 他人格中最好的部分？ | 善良/坚韧/幽默/责任感 |
| 性格缺点 | 他的人格缺陷？ | 贪财/冲动/自卑/多疑 |

**重要：性格缺点必须影响剧情，不能只是标签**

❌ **标签化缺点：**
> 主角"比较贪财"（但剧情中没有任何体现）

✅ **真实缺点：**
> 主角贪财，导致他在关键时刻犹豫要不要收反派的钱，差点失去盟友的信任，最后醒悟但已造成损失

### 2.3 主角成长弧线

**三阶段成长：**

**第一阶段（1-20章）：觉醒期**
- 状态：普通、迷茫、被动
- 事件：获得能力，初次尝试，小范围使用
- 成长：从"我不敢用能力"到"我能用能力解决一些问题"

**第二阶段（21-40章）：发展期**
- 状态：自信、主动、但也有困惑
- 事件：卷入大冲突，认识其他异能者，发现能力的代价
- 成长：从"我只想过好日子"到"我有责任保护重要的人"

**第三阶段（41章+）：成熟期**
- 状态：坚定、有担当、但也有痛苦
- 事件：面对最终BOSS，做出重大选择
- 成长：从"被动接受能力"到"主动选择自己的道路"

## 三、配角设计模板

### 3.1 配角分类与功能

**功能性配角（不需要太复杂）：**
- 打工型：主角的上司、同事、同学
- 工具型：提供情报的NPC、卖道具的商人
- 氛围型：搞笑的邻居、唠叨的房东

**重要配角（必须有个性）：**
- 盟友：和主角一起行动的角色
- 反派：和主角对立的角色
- 情感线：恋爱对象/家人/重要朋友

### 3.2 重要配角设计公式

**公式：鲜明个性 + 专属科幻设定 + 和主角的关系**

#### 示例1：盟友配角
```
名字：林小溪
身份：高中生，黑客
科幻设定：能和电子设备"对话"（不是黑客技术，是真的能听到设备的"心声"）
个性：天然呆、社恐、但在网上是大佬、对设备特别温柔
和主角关系：被主角救过，主动帮忙，成为技术支援
缺点：过度依赖设备，现实社交能力极差
```

#### 示例2：反派配角
```
名字：王"医生"
身份：社区诊所医生，表面温和
科幻设定：能通过触摸读取人的记忆（但只能读取表层记忆）
个性：表面儒雅、实际冷酷、认为"牺牲少数人是必要的"
和主角关系：前期是主角的医生，后期发现是反派组织的成员
动机：想用主角的能力治疗自己患绝症的女儿
```

#### 示例3：情感线配角
```
名字：苏晴
身份：同城快递员，骑摩托车送大件物品
科幻设定：不知道主角有超能力，但总觉得主角"不正常"
个性：直爽、话痨、热心肠、好奇心重
和主角关系：同事（主角伪装成普通快递员），逐渐发展成恋人
作用：让主角保持"普通人"的感觉，避免彻底异化
```

### 3.3 配角禁忌

❌ **避免的配角类型：**
- 工具人：只为了推动剧情出现，用完就忘
- 无个性：说话做事都像模板，没有记忆点
- 过度崇拜：无脑吹主角，没有自己的想法
- 无脑反派：为了坏而坏，没有合理动机

✅ **正确的配角塑造：**
- 每个重要配角都有自己的目标（不是围绕主角转）
- 配角也会质疑、反对主角
- 配角有自己的成长线（不能永远是配角，要有高光时刻）
- 配角的个性要"到极致"（比如一个配角可能极度抠门，但对自己的猫很大方）

## 四、对话设计技巧

### 4.1 角色说话方式差异化

**方法：给每个角色设计"说话标签"**

| 角色 | 说话标签 | 示例 |
|------|----------|------|
| 主角 | 直爽、偶尔吐槽、不矫情 | "行吧，你说得对，但我还是觉得这事儿悬。" |
| 天然呆盟友 | 真诚、有点傻、疑问多 | "啊？为什么啊？我不是很明白诶。" |
| 冷酷反派 | 简短、嘲讽、自信 | "你觉得你有选择？" |
| 热心邻居 | 唠叨、八卦、语气词多 | "哎哟喂，小伙子你听说了没？就隔壁楼那事儿……" |

### 4.2 对话推动剧情原则

**每段对话必须满足至少1个功能：**
1. 推进情节（提供新信息、做出决定）
2. 展现角色性格（通过说话方式）
3. 制造冲突（角色之间的分歧）
4. 释放情绪（幽默、紧张、悲伤）

❌ **无效对话：**
> "你好。"
> "你好。"
> "吃饭了吗？"
> "吃了。"
> "哦。"

✅ **有效对话：**
> "陈默，你最近是不是……有点不对劲？"林小溪盯着他的眼睛。
> "什么不对劲？"陈默心虚地移开视线，"我挺好的。"
> "你以前从来不迟到的，这周已经三次了。"林小溪皱眉，"而且你昨天吃饭的时候，盯着一杯水看了五分钟，你在看什么？"
> "没什么，"陈默站起来，"我还有个快递要送，先走了。"
> "陈默！"林小溪叫住他，"如果遇到麻烦，可以告诉我。"
> 陈默背对着她站了一秒，然后挥了挥手，没有回头。

**分析：**
- ✅ 推进情节：林小溪发现主角异常（主角能力暴露风险）
- ✅ 展现性格：林小溪的关心 vs 主角的回避
- ✅ 制造张力：主角想隐瞒，但盟友已经开始怀疑

## 五、角色关系网络

### 5.1 关系类型矩阵

| 关系类型 | 示例 | 作用 |
|---------|------|------|
| 师徒 | 老前辈教主角使用能力 | 传递世界观设定、加快主角成长 |
| 宿敌 | 长期对立的竞争者 | 提供持续冲突、互相促进成长 |
| 误解 | 最初敌对，后发现是盟友 | 制造反转、增加剧情深度 |
| 背叛 | 信任的角色背叛主角 | 情感冲击、推动剧情转折 |
| 救赎 | 反派洗白或主角赎罪 | 展现人性复杂、增加主题深度 |

### 5.2 关系发展节奏

**关系变化不能突兀，需要铺垫：**

**示例：背叛关系发展（10章铺垫）**
- 第1-3章：角色A帮助主角，建立信任
- 第4-6章：暗示A有问题（偶尔说奇怪的话、行为不一致）
- 第7-8章：主角开始怀疑，但不愿相信
- 第9章：A背叛主角
- 第10章：主角接受现实，但发现背叛有隐情（为后续反转铺垫）

## 六、角色出场设计

### 6.1 主角出场（第1章）

**黄金开篇公式：现状展示 + 困境 + 科幻介入**

```
陈默是个外卖员。

在这个拥有两千三百万人口的超级都市里，像他这样的外卖员有十七万。他们穿着统一的黄色制服，骑着改装电动车，像工蚁一样穿梭在早晚高峰的洪流中。

陈默不同。

他有个奇怪的外卖箱。

那是他半年前在二手市场淘的，摊主说这是"智能恒温箱"，但陈默怀疑那老头就是想找个理由把快报废的箱子卖出去……
```

**分析：**
- 第1段：身份设定（外卖员）
- 第2段：现状细节（工蚁比喻，画面感强）
- 第3-5段：引出科幻元素（奇怪的外卖箱）

### 6.2 配角出场

**配角出场必须做到：印象 + 个性 + 功能**

```
"林小溪？名字挺好听的，人呢？"
"在那儿。"
陈默顺着朋友指的方向看过去。

角落里坐着一个瘦小的女生，低着头，手指在手机屏幕上飞快地敲击，旁边放着一杯早就凉透的咖啡。她穿着一身oversize的黑色卫衣，帽衫拉到头顶，像只缩在壳里的寄居蟹。

"她社恐，"朋友压低声音，"但在网上是大神，黑客圈都有名。"
陈默皱眉："你让我找这么个社恐帮忙？"
"她欠我人情，"朋友说，"而且……她好像对你感兴趣。"
"什么意思？"
"她刚才看了你一眼，"朋友说，"然后脸色变了。"
陈默转头，发现那个女生正在盯着自己，眼神很奇怪——不像是在看人，倒像是在看一台待拆解的电脑。
```

**分析：**
- ✅ 印象：瘦小、黑色卫衣、低着头（视觉记忆点）
- ✅ 个性：社恐但在网上是大神（反差）
- ✅ 功能：黑客技术支援 + 对主角有兴趣（剧情作用）
- ✅ 钩子：她为什么对主角感兴趣？（悬念）

## 七、角色弧光设计

### 7.1 主角弧光（核心）

**弧光 = 起点状态 → 突破点 → 终点状态**

**示例弧光：从自我中心到责任担当**
- 起点（1-10章）："我只想用能力赚钱，过好日子"
- 突破点（11-20章）：能力害无辜者受伤，主角内疚
- 终点状态（21-30章）："我必须学会控制能力，不能伤害别人"

### 7.2 配角弧光

**重要配角也应该有成长：**

**示例：天然呆盟友的弧光**
- 起点：过度依赖能力，现实社交无能
- 突破点：主角陷入危机，她必须独自面对现实世界
- 终点状态：学会平衡能力和现实生活，变得更自信

## 八、常见角色陷阱

### 8.1 避免的角色问题

❌ **玛丽苏/杰克苏：**
> 主角完美无缺，所有人都爱他/她，任何问题都能轻松解决。

✅ **真实主角：**
> 主角有优点也有缺点，会犯错会受伤，不是所有人喜欢他/她。

❌ **工具人配角：**
> 配角存在的意义就是为了服务主角，没有自己的生活。

✅ **独立配角：**
> 配角有自己的目标、生活、情感，不是围绕主角转。

❌ **脸谱化反派：**
> 反派就是为了坏而坏，没有合理的动机。

✅ **立体反派：**
> 反派有自己的逻辑和理由，从他的角度看自己是对的。

### 8.2 角色一致性检查清单

每次修改角色相关内容时，用以下清单自检：

- [ ] 主角的每个重要决定符合他的性格？
- [ ] 配角的说话方式有记忆点，不会和其他角色混淆？
- [ ] 角色的能力使用遵守了设定限制？
- [ ] 角色的情感转变有铺垫，不突兀？
- [ ] 反派的动机合理，不是"为了坏而坏"？
- [ ] 配角有自己的生活，不是只围绕主角转？
- [ ] 角色之间的关系发展有逻辑？
- [ ] 重要配角有高光时刻？

## 九、AI辅助角色创建提示

当让AI辅助创建角色时，使用以下提示模板：

```
请为都市科幻小说设计一个[主角/配角/反派]，要求：

1. 身份设定：[具体职业/社会阶层]
2. 科幻设定：[具体能力/与科幻元素的关联]
3. 性格特点：[至少2个优点 + 1个缺点]
4. 核心欲望：[他最想要什么]
5. 核心恐惧：[他最怕什么]
6. 和主角的关系：[盟友/反派/中立]
7. 风格要求：个性鲜明、有记忆点、避免脸谱化

请输出：
- 角色基本信息（姓名、年龄、外貌）
- 性格详细描述
- 背景故事（简短）
- 在剧情中的作用
- 一段体现个性的对话示例
```

## 十、角色档案示例

### 示例：完整的主角档案

**姓名：陈默**
**年龄：24岁**
**职业：外卖员（之前是程序员，因行业寒冬失业）**

**外貌：**
- 中等身高，偏瘦，长期户外工作晒得有点黑
- 眼神很平静（因为能看到太多真相，所以对很多事都淡定）
- 总是背着一个看起来很破旧的外卖箱

**科幻设定：**
- 能力名："真实之眼"（能看到物品的"信息"）
- 来源：半年前在旧货市场买的老花镜，沾了陨石粉末
- 限制：每天只能用30次，超过会头痛；只能看表层信息
- 代价：使用多了会分不清现实和信息的边界

**性格：**
- 优点：务实、善良但不圣母、有责任感
- 缺点：贪财（想赚钱买房）、有点被动（不喜欢主动惹事）、多疑
- 说话方式：简洁直接，偶尔吐槽，不太会说漂亮话

**核心欲望：**
- 表层：赚钱（脱离贫困，买套小房子）
- 深层：被认可（证明自己不是失败者）

**核心恐惧：**
- 表层：永远穷下去
- 深层：失去唯一在乎的人（奶奶）

**背景故事：**
- 父母早逝，奶奶带大
- 读计算机专业，毕业进互联网公司，结果遇到行业寒冬被裁员
- 找不到程序员工作，只能送外卖维生
- 半年前获得能力，开始偷偷用能力赚外快
- 目前最大的目标：攒够首付，买套小房子，让奶奶搬出城中村

**角色弧光：**
- 起点：只想用能力赚钱，过好日子
- 突破点：奶奶因为住在城中村，环境差生病住院，陈默意识到钱不是万能的
- 终点状态：学会用能力帮助更多人，而不只是赚钱

**代表性对话：**
> "我知道你想说什么，"陈默打断了老板的话，"你想说我有能力就该免费帮忙。但老板，我奶奶住院一天要花三千，我送一单外卖才赚五块。你要我怎么选？"
>
> 他顿了顿，又补了一句："当然，如果是我能做到的事，不用你开口，我也会帮。但如果是让我去拼命……对不起，我还要留着命照顾我奶奶。"
