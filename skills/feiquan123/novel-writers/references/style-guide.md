# 写作风格指南

## 一、核心理念

**番茄都市科幻 = 都市烟火气 + 科幻设定 + 强爽点节奏**

三者在内容中占比：都市日常 40% + 科幻元素 40% + 爽点节奏 20%

## 二、《信息全知者》风格解析

### 2.1 语言特点

**朴实接地气，但不粗糙**
```
✅ 好例子：
老爷子上了柱香，愁苦道："求黄帝老爷，保佑我家墟儿，健健康康……咳咳咳……"
"爷爷！吃饭了。"一名少年端着菜盘子进屋，十分利索地把桌子收拾了一下。

❌ 避免：
老人充满悲悯地祈祷道，"哦，伟大的黄帝啊，请赐予我的孙子健康吧……"
"爷爷，"少年轻柔地说道，"该用餐了。"然后少年拿着盘子进来，收拾了桌子。
```

**细节丰富，但克制不矫情**
```
✅ 好例子：
孩子命苦，从小脑子就有问题。而且他天生眼睛的瞳孔就跟别人不一样，瞳仁白斑过多，连成一圈白环，看起来像是大瞳仁套着小瞳仁。

❌ 避免：
他从小就被命运所抛弃，承受着无法言语的痛苦。他那双与众不同的眼睛，瞳仁周围环绕着凄美的白环，仿佛上帝给他开的残酷玩笑……
```

### 2.2 对话风格

**口语化，但不粗俗**
- 使用"墟儿"、"老爷子"、"梁医生"等称谓
- 村民对话带地方感，但不用生僻方言
- 对话推进剧情，不为了说话而说话

**情感真挚，但不过度煽情**
```
✅ 好例子：
老爷子咳嗽着坐下，发愁地看着孙子。
问道："墟儿，你的情况，上午梁医生怎么说？"
少年笑了笑，说道："梁医生说我情绪稳定，已经没事了。"

❌ 避免：
"爷爷！"墟儿眼含热泪，紧紧握住老人的手，"医生说我没事了！真的没事了！"
老人激动得浑身颤抖，老泪纵横："苍天有眼啊！我的乖孙终于……"
```

### 2.3 叙事视角

**第三人称限制视角**
- 读者只跟随主角视角了解信息
- 通过主角的所见所闻展现世界观
- 主角的困惑 = 读者的困惑（同步体验）

**节奏：慢起快走**
- 开篇日常细节多，节奏慢
- 科幻元素出现后，节奏加快
- 每章结尾加速，留钩子

## 三、番茄都市科幻风格适配

### 3.1 科幻落地化

**错误示范：**
> 主角在家启动量子超算，连接银河联邦网络，开始入侵外星母舰的防御系统……

**正确示范：**
> 陈默把外卖箱往楼道角落一放，掏出手机点开那个奇怪的APP。屏幕上的外卖订单突然开始跳动，每一个订单背后都浮现出一条半透明的数据流——"用户真实需求：想喝冰可乐解暑，但怕胖，正在纠结"。
> 这是什么鬼功能？陈默盯着屏幕，突然看见有个订单的数据流变成了刺眼的红色："用户真实需求：用差评威胁商家免单，已操作13次"。

**对比分析：**
- ✅ 外卖场景（都市日常）+ 数据流能力（科幻设定）
- ❌ 量子超算、银河联邦（脱离日常）

### 3.2 语言通俗化

**避免专业术语堆砌**

❌ **错误：**
> 林晨启动了脑机接口的神经突触同步协议，调用了海马体的记忆缓存区，通过量子纠缠态的瞬间传输，将意识上传到了元宇宙的服务器集群中……

✅ **正确：**
> 林晨摸了摸太阳穴上的那个硬币大小的金属贴片。说明书上说这玩意儿能读取记忆，但他一直以为是骗人的。
> 直到刚才，当他闭上眼睛回想昨晚的聚餐时，脑海里突然弹出了一个进度条——"记忆上传中：23%"。
> "卧槽？"林晨猛地睁开眼，"这不是梦？"

### 3.3 画面感营造

**科幻场景描写公式：视觉细节 + 声音/触感 + 主角反应**

示例：
> 突然，地铁车厢的灯光开始闪烁，不是普通的跳闸，而是所有光线都被扭曲成了螺旋状，像被一只无形的手搅动的奶油。耳边传来嗡嗡的低鸣，震得林晨耳膜发麻。他下意识抓紧扶手，却发现手掌穿过了金属杆——整个车厢都在半透明化。
> "这是……"林晨喉咙发干，"地铁时空裂缝？"

**公式拆解：**
- 视觉：灯光扭曲成螺旋状（具体意象）
- 听觉：嗡嗡低鸣（声音）
- 触觉：耳膜发麻、手掌穿过金属杆（触感）
- 反应：喉咙发干、内心独白（主角感受）

## 四、幽默元素注入

### 4.1 内心吐槽（最常用）

**场景：高大上科幻设定 vs 粗糙现实**
```
官方说法："使用时空折叠技术，实现零延迟的跨城物流传输。"
林默内心：翻译成人话就是——我把快递扔进公司那个像微波炉一样的传送门，
然后对面城市的同事从另一个"微波炉"里掏出来，中间省了八小时高铁和五百块路费。
但问题是，这玩意儿经常卡快递，上次那袋薯片传过去只剩半袋了，客服说是
"量子力学层面的正常损耗"，不赔。
```

### 4.2 对话包袱（配角互动）

**天然呆型配角：**
"老大，这个机器人怎么不动了？"
"你没开机。"
"开了啊，你看灯都亮着。"
"那是充电指示灯，你没按开关。"
"哦……那开关在哪儿？"
"就在你手里按着的地方。"

**毒舌型配角：**
"你管这叫超能力？充其量就是个人形WiFi！"
"至少省了流量费。"
"你用超能力连网看猫片，要不要这么励志？"

### 4.3 场景反差萌

**严肃场合+意外反转**
```
特工会议室内，气氛凝重。
"敌方组织已经渗透进了城市的供水系统，"组长指着全息投影，"他们计划
在明晚八点投放神经毒素，预计伤亡人数将达到……"
"等等，"林默举手，"我能不能问个问题？"
"说。"
"明晚八点我有相亲，能不能提前或者推迟投毒？"
全场沉默三秒。
"……林默，你是单身，不是智障。"
```

## 五、语言禁忌

### 5.1 避免的句式

❌ **过度排比：**
> 他感受到了力量，他感受到了希望，他感受到了未来。
✅ **改为：**
> 他感觉浑身涌起一股劲儿，像是喝了三罐红牛。

❌ **滥用比喻：**
> 那目光如同寒夜中的孤星，像是深渊里的明灯，仿佛沙漠中的清泉……
✅ **改为：**
> 那目光冷得吓人，看得他后背发毛。

### 5.2 避免的词汇

- 矫情词：凄美、沧桑、缥缈、绝世、倾城
- 网络梗：yyds、绝绝子、芭比Q（除非是角色特意玩梗）
- 翻译腔：似乎、仿佛、某种（直接说是什么）

## 六、改写自检清单

每次生成内容后，用以下清单自检：

- [ ] 开篇3000字内是否有科幻爽点？
- [ ] 科幻设定是否融入了都市场景？（地铁、写字楼、小区、学校）
- [ ] 主角是否有"普通身份+特殊能力"的反差？
- [ ] 是否有至少1处幽默元素？（内心吐槽/对话包袱/场景反差）
- [ ] 专业术语是否用直白语言解释了？
- [ ] 每段是否控制在6-8行？（便于手机阅读）
- [ ] 对话是否口语化，没有翻译腔？
- [ ] 是否避免了过度煽情和矫揉造作？
- [ ] 配角是否有鲜明个性，不是工具人？
- [ ] 章末是否留了钩子？

## 七、实战示例

### 7.1 开篇设计（前3000字出爽点）

```
第一章 那个奇怪的外卖箱

    陈默是个外卖员。

    在这个拥有两千三百万人口的超级都市里，像他这样的外卖员有十七万。他们穿着统一的黄色制服，骑着改装电动车，像工蚁一样穿梭在早晚高峰的洪流中。

    陈默不同。

    他有个奇怪的外卖箱。

    那是他半年前在二手市场淘的，摊主说这是"智能恒温箱"，但陈默怀疑那老头就是想找个理由把快报废的箱子卖出去。箱子外观普普通通，甚至有点破，表面布满划痕，像是被人从二楼窗户扔下去过三次。

    但陈默还是买下了，因为它便宜——三十块钱。

    半年来，这个箱子确实表现出一些奇怪的地方。比如保温效果出奇好，冬天放进去的热咖啡三小时后还能烫嘴，夏天塞进去的冰激凌不会化。但陈默觉得这应该是密封性好，没多想。

    直到今天。

    下午三点，陈默送完最后一单，准备去快餐店对付午饭。他把外卖箱停在路边，自己进店点餐。

    等他端着餐盘出来时，看见一个老太太正站在他的箱子前，手里拿着一根拐杖，在那儿戳。

    "大娘，您干嘛呢？"陈默放下餐盘走过去。

    老太太转过头，露出满口假牙："小伙子，你这箱子……怎么在抖？"

    陈默愣了一下。

    箱子确实在轻微震动，而且发出很细微的嗡嗡声，像手机震动。但问题是，箱子里是空的。

    "可能是我把手机落里面了，"陈默随口胡扯，走过去掀开箱盖。

    箱子里没有手机。

    但有一个拳头大小的发光球体，正悬浮在箱子中央，缓慢旋转。表面像是流动的液体，不断变换颜色——蓝、紫、金、红，周而复始。

    陈默盯着看了五秒。

    然后他把箱子盖上了。

    "大娘，"陈默转过身，面无表情地说，"您看错了，箱子没抖。"

    "我看见了，"老太太坚持道，"里面发光呢。"

    "那是您的白内障，"陈默骑上电动车，"赶紧去医院看看吧，别耽误了。"

    说完，他一拧油门，窜了出去。

    陈默的心跳得很快。

    他把车停在一条没人的巷子里，再次掀开箱盖。那个发光球体还在，而且似乎比刚才更亮了。

    "这特么是什么东西？"陈默小声嘀咕。

    就在这时，球体突然射出一道蓝光，在他眼前形成一行悬浮文字：

    【已检测到适配宿主】
    【时空快递系统正在激活……】
    【激活进度：1%】

    陈默盯着那行字看了三秒，然后掏出手机百度：
    "我是不是出现幻觉了，求在线等，急。"
```

**分析：**
- ✅ 都市日常：外卖员身份、快餐店、老太太互动
- ✅ 开篇1000字出科幻爽点（发光球体、系统激活）
- ✅ 幽默元素：陈默的胡扯、白内障梗、百度搜索
- ✅ 主角反差：普通外卖员 vs 特殊快递箱
- ✅ 章末钩子：系统激活进度1%，悬念留下一章

## 八、AI改写提醒

当使用AI辅助创作时，必须人工改写以下内容：

1. **句式重复**：AI喜欢用"突然""仿佛""似乎"，需替换
2. **情感堆砌**：AI容易过度煽情，需克制
3. **对话生硬**：AI对话偏书面，需口语化
4. **细节缺失**：AI场景描写空泛，需添加具体细节
5. **逻辑漏洞**：AI可能忽略因果关系，需补全

**改写比例要求：≥30%**
