# Novel Writer Skill - 使用指南

## 简介

这是一个专门用于创作番茄小说风格都市科幻小说的 Claude Code skill。它将科幻元素融入都市日常，构建快节奏、强爽点、逻辑合理的网络小说。

## 文件结构

```
novel-writer/
├── SKILL.md                    # 主配置文件，定义 skill 的基本信息
└── references/
    ├── style-guide.md          # 写作风格指南（文风、幽默、语言）
    ├── plot-design.md          # 情节设计指南（大纲、爽点、钩子）
    ├── character-build.md      # 角色塑造指南（人设、对话、关系）
    └── chapter-pace.md         # 章节节奏指南（字数、节奏、对话）
```

## 核心特色

### 1. 文章风格
- 朴实接地气、细节丰富、情感真挚
- 番茄都市科幻：都市烟火气 + 科幻设定 + 强爽点节奏
- 幽默风趣：内心吐槽、对话包袱、场景反差

### 2. 逻辑合理
- 能力必有来源、限制和代价
- 反派动机合理有层次
- 世界观设定前后一致

### 3. 结构完整
- 开篇3000字必出科幻爽点
- 单章2000-2200字
- 每章结尾留钩子
- 快慢交替节奏

## 使用场景

### 场景1：创建小说大纲
```
你：帮我设计一个都市科幻小说的大纲
Claude：（加载 plot-design.md）
    - 设定背景、主角、科幻能力
    - 规划前50章的章节结构
    - 标注爽点和钩子分布
```

### 场景2：塑造角色
```
你：帮我设计一个外卖员主角，要有反差
Claude：（加载 character-build.md）
    - 创建"普通身份+特殊能力"的反差人设
    - 设计性格优缺点、核心欲望和恐惧
    - 提供对话示例
```

### 场景3：撰写章节
```
你：帮我写第1章，主角获得超能力
Claude：（加载 style-guide.md 和 chapter-pace.md）
    - 按照番茄节奏撰写2000-2200字
    - 融入幽默元素
    - 章末留钩子
```

### 场景4：检查逻辑
```
你：帮我检查这段剧情的逻辑问题
Claude：（加载 plot-design.md）
    - 检查能力使用是否遵守限制
    - 检查角色行为是否符合性格
    - 检查时间线是否一致
```

## 实战示例

### 示例1：创作开篇
```
你：帮我写一个都市科幻小说的开篇，主角是外卖员，获得时空折叠能力

Claude：（加载多个参考文档）
    - 根据 character-build.md 创建主角人设
    - 根据 style-guide.md 确定文风
    - 根据 plot-design.md 设计开篇剧情
    - 根据 chapter-pace.md 控制节奏

输出：
- 主角档案（姓名、年龄、背景、性格）
- 科幻能力设定（来源、功能、限制、代价）
- 第1章完整内容（2000字左右）
- 章末钩子说明
```

### 示例2：设计情节冲突
```
你：我的主角现在被困在反派组织了，帮我设计3个逃脱方案

Claude：（加载 plot-design.md）
    - 根据主角能力设定（时空折叠）
    - 设计3个不同难度的逃脱方案
    - 每个方案包含：使用什么能力、遇到什么阻碍、如何解决、代价是什么

输出：
- 方案1：利用空间折叠能力（高难度）
- 方案2：策反盟友（中等难度）
- 方案3：等待救援+配合反击（低难度）
```

## 注意事项

### AI 辅助创作的改写要求

当使用 AI 生成内容后，必须人工改写 ≥30%，重点修改：

1. **句式重复**：AI 喜欢"突然""仿佛""似乎"，需替换
2. **情感堆砌**：AI 容易过度煽情，需克制
3. **对话生硬**：AI 对话偏书面，需口语化
4. **细节缺失**：AI 场景描写空泛，需添加具体细节
5. **逻辑漏洞**：AI 可能忽略因果关系，需补全

### 逻辑检查清单

每次创作后，用以下清单自检：

- [ ] 开篇3000字内是否有科幻爽点？
- [ ] 科幻设定是否融入了都市场景？
- [ ] 主角是否有"普通身份+特殊能力"的反差？
- [ ] 是否有至少1处幽默元素？
- [ ] 专业术语是否用直白语言解释了？
- [ ] 每段是否控制在6-8行？
- [ ] 对话是否口语化？
- [ ] 是否避免了过度煽情？
- [ ] 配角是否有鲜明个性？
- [ ] 章末是否留了钩子？
