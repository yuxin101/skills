# Crypto Alert | 加密貨幣價格提醒

## Description | 描述

Cryptocurrency price monitoring and alerting skill. 加密貨幣價格監控與提醒技能。

## Features | 功能

- Real-time crypto prices | 即時加密貨幣價格
- 24-hour change tracking | 24小時漲跌幅追�
- Support for major coins | 支援主流幣種
- Price alerts | 價格提醒

## Installation | 安裝

```bash
pip3 install requests
```

## Usage | 使用方式

```bash
# List all popular coins | 列出所有熱門幣種
python3 crypto.py

# Query specific coin | 查詢特定幣種
python3 crypto.py 比特幣
python3 crypto.py 以太坊
python3 crypto.py btc
```

## Supported Coins | 支援幣種

| Command | Coin |
|---------|------|
| 比特幣 / btc | Bitcoin (BTC) |
| 以太坊 / eth | Ethereum (ETH) |
| sol | Solana (SOL) |
| bnb | Binance Coin (BNB) |
| xrp | XRP (XRP) |
| doge | Dogecoin (DOGE) |
