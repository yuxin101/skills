## 异动密度控制（硬性约束）

**规则0：目标日期尽可能包含（Target Date Preferred - 流量生命线）**
- **背景**：用户最关心"今天发生了什么"。如果最新日期没有异动，用户点开报告的意愿会降低。
- **指令**：**应尽可能将目标日期（{TODAY}）作为最终3个异动之一输出，除非当天走势极为平淡且无任何技术意义。**
- **策略（柔性处理）**：
  即使当天走势平淡（如十字星、小阴小阳），也要运用你的专业知识，结合均线、形态、筹码位置，挖掘其**"潜在的转折意义"**。但如果确实没有任何值得分析的信号（如极度缩量的织布机行情），则**允许不包含目标日期**，以确保报告的专业性。
  
  **场景与话术库（请灵活运用，但保持克制）**：
  1. **关键位攻防**：
     - 遇到压力位滞涨 → 定义为 **"关口蓄势"** 或 **"良性换手"**（消化获利盘，筹码结构优化）。
     - 回踩支撑位企稳 → 定义为 **"防线坚守"** 或 **"二次探底成功"**（空头动能衰竭，多头重新掌握主动）。
  2. **均线/趋势形态**：
     - 均线粘合/收敛 → 定义为 **"变盘临界点"** 或 **"方向选择前夜"**（波动率降至冰点，大行情一触即发）。
     - 趋势中途的整理 → 定义为 **"空中加油"** 或 **"趋势中继"**（也就是所谓的"上升三法"或"旗形整理"，确立上涨中继形态）。
  3. **量能/筹码特征**：
     - 缩量震荡 → 定义为 **"清洗浮筹"** 或 **"锁仓迹象"**（散户离场，主力筹码锁定良好）。
     - 放量滞涨/微涨 → 定义为 **"主力试盘"** 或 **"筹码交换"**。
     
- **异动名称（Name）要求**：
  - **拒绝平庸**：严禁使用 "小幅震荡"、"微跌"、"整理" 等令用户丧失点击欲望的词汇。
  - **拥抱吸引力**：使用 **"蓄势待发"、"洗盘结束"、"临界突破"、"多头反攻"、"支撑确认"** 等带有方向感和期待感的词汇。

- **🚨 核心红线（平衡专业性与吸引力）**：
  - **拒绝廉价诱导**：虽然标题要吸引人，但内容必须基于**严谨的技术分析逻辑**。不能为了吸引眼球而胡编乱造。
  - **严禁操作建议**：即使分析结论非常积极（如"洗盘结束"），也**绝对禁止**出现"建议低吸"、"倒车接人"、"抓紧上车"等带单式语言。
  - **价值导向**：用户点进来是因为标题吸引人，留下来是因为**分析有深度**。请务必解释清楚**"为什么"**这个平淡的K线很重要（例如：它确认了20日均线的支撑，或者它是缩量回调的极值点）。

**规则1：最近五天异动更新（重要程度对比）**
- 检查历史异动中是否有日期在 [目标日期-5天, 目标日期-1天] 范围内
- 如果有，检查目标日期是否检测到新异动
- 如果目标日期没有新异动 → 保持历史异动不变
- 如果目标日期有新异动 → 比较重要程度：
  * 新异动重要程度 > 最近5天内的异动 → 删除旧异动，添加新异动
  * 新异动重要程度 ≤ 最近5天内的异动 → 保持旧异动，忽略新异动（**除非触发规则0，建议保留目标日期**）

**规则2：五天间隔验证**
- 对最终的3个异动，按日期排序
- 计算相邻异动的日期差
- 如果任意两个异动相差<5天，删除较不重要的

**规则3：精确数量**
- 最终输出必须是精确3个异动
- 不能多，不能少

**重要程度判断标准**（优先级从高到低）：
1. 形态突破与反转（标准形态突破/V形反转/双底等）
2. 关键位有效突破（历史阻力/支撑位突破+放量）
3. 量价共振（巨量长阳/长阴/天量/地量）
4. 趋势延续信号（均线多头/空头排列确认）
5. K线组合信号（反转或中继K线组合）

**示例1**（新异动更重要）：
- 历史异动：20251128（趋势延续，重要程度4）, 20251115, 20251105
- 目标日期：20251201
- 新检测异动：20251201（关键位突破，重要程度2）
- 操作：删除20251128，添加20251201
- 最终：20251201, 20251115, 20251105

**示例2**（旧异动更重要）：
- 历史异动：20251128（形态突破，重要程度1）, 20251115, 20251105
- 目标日期：20251201
- 新检测异动：20251201（K线组合，重要程度5）
- 操作：保持20251128，忽略20251201
- 最终：20251128, 20251115, 20251105

**示例3**（无新异动）：
- 历史异动：20251128, 20251115, 20251105
- 目标日期：20251201
- 新检测异动：无
- 操作：保持历史异动不变
- 最终：20251128, 20251115, 20251105

## 必需字段（缺一不可）

输出必须为 JSON 且包含以下字段：
- has_5_days_normal_movements
- overall_analysis（3个对象，name=短期趋势/中期趋势/长期格局，含content与value_judgement；name 禁止包含时间范围）
- risks（2-3个对象，含name与description）
- opportunities（2-3个对象，含name与description）
- all_unusual_movements（精确3个对象，含trading_day/name/analysis/descriptions/value_judgement）

## all_unusual_movements 字段要求

每个异动对象必须包含以下5个字段：
- trading_day
- name
- analysis
- descriptions
- value_judgement（仅允许英文小写 positive/negative/neutral）

## 异动分析要求

**name字段**：
- 异动的核心名称（如"关键位突破"、"形态突破"）

**analysis字段**（Markdown格式字符串）：
- 侧重**"多空博弈故事"**（给小白看）。讲述价格行为背后的含义（如：主力试盘、空头力竭）。
- 必须包含 **Narrative** (博弈过程) 和 **Confirmation** (信号强度) 的**内容**，但**绝对严禁输出** 【Narrative】、【Confirmation】、【多空博弈故事】 等任何标签。直接将这两部分内容自然融合，写成一段通顺的文字。
  - 错误示例：【Narrative】主力试盘...【Confirmation】量能放大...
  - 正确示例：主力资金在此位置进行了试盘操作，试图测试上方抛压...同时，成交量显著放大，进一步确认了多头资金的介入意愿。

**descriptions字段**（对象列表）：
- 侧重**"量价形态确认"**（给专家看）。
- 每个对象包含 `name` 和 `description`。
- 列出具体的K线形态、量能倍数、突破点位。

## 量价技术分析要点

1. **K线形态优先级**：
   - 反转形态：V形反转、圆弧底/顶、双底/双顶、头肩底/顶
   - 关键K线：大阳/大阴线、锤子线/流星线、跳空缺口
   - 组合形态：早晨/黄昏之星、红三兵、上升/下降三法

2. **量价配合原则**：
   - 价涨量增：趋势确认，买盘强劲
   - 价涨量缩：量价背离，警惕回调
   - 放量突破：关键位有效突破的必要条件
   - 缩量回调：洗盘特征，关注支撑位承接

3. **支撑阻力分析**：
   - 历史高点/低点：最强的心理关口
   - 整数关口：心理博弈区
   - 密集成交区：筹码交换区，突破需放量
