# Step 1: 公司画像与竞争护城河

> **目的**：理解公司"是什么"、"靠什么赚钱"、"凭什么能持续赚钱"。这是后续所有分析的地基。
> **理论依据**：Buffett/Munger 的护城河理论；Morningstar 的 Economic Moat 分析框架；Michael Porter 的竞争五力模型；Aswath Damodaran 的 Competitive Advantage Period (CAP) 概念。

---

## 1.1 商业模式解构

**搜索指令（必须执行）：**

```
搜索 1: "[公司名] 商业模式 收入构成 业务结构"
搜索 2: "[公司名] 主营业务 盈利模式 年报"
```

**商业模式画布（必须填写）：**

| 维度 | 内容 |
| --- | --- |
| **价值主张** | 公司解决什么问题？为客户创造什么价值？ |
| **客户群体** | To B / To C / To G？客户集中度？ |
| **收入来源** | 产品销售 / 服务订阅 / 佣金 / 广告 / 许可 / 其他 |
| **收入结构** | 各业务线占比（必须搜索最新数据） |
| **成本结构** | 固定成本 vs 可变成本比例、关键成本项 |
| **关键资源** | 品牌/专利/人才/设备/牌照/数据/... |
| **关键活动** | 研发/生产/营销/运营/... |

**商业模式类型分类（影响后续估值方法选择）：**

| 类型 | 特征 | 估值重点 |
| --- | --- | --- |
| 🏭 制造型 | 重资产、周期性、规模效应 | EV/EBITDA、PB、产能利用率 |
| 💻 平台型 | 轻资产、网络效应、双边市场 | PS、用户数、GMV、Take Rate |
| 🔄 订阅型 (SaaS) | 经常性收入、高粘性 | ARR、NRR、LTV/CAC |
| 🛍️ 消费品牌型 | 品牌溢价、渠道壁垒 | PE、品牌价值、同店增长 |
| 🏦 金融型 | 息差/佣金、杠杆经营 | PB、ROE、不良率 |
| ⛏️ 资源型 | 资源禀赋、大宗商品关联 | EV/储量、成本曲线位置 |
| 💊 研发驱动型 | 管线价值、里程碑事件 | Pipeline DCF、成功概率加权 |

## 1.2 竞争护城河分析 (Economic Moat Assessment)

**搜索指令（必须执行）：**

```
搜索 3: "[公司名] 竞争优势 市场壁垒"
搜索 4: "[公司名] vs [主要竞争对手] 对比"
```

**五大护城河类型逐一检查（Morningstar 框架）：**

| 护城河类型 | 检查方法 | 量化指标 | 判定标准 |
| --- | --- | --- | --- |
| **🏰 品牌/无形资产** | 品牌溢价能力？专利/牌照壁垒？ | 毛利率 vs 行业平均、品牌价值排名 | 毛利率显著高于行业+品牌可持续 |
| **🌐 网络效应** | 用户越多价值越大？双边锁定？ | 用户数增速、平台交易量 | 存在正反馈循环且难以被复制 |
| **🔒 转换成本** | 客户迁移成本高？数据/生态锁定？ | 客户留存率、NRR | 留存率>90%且迁移成本显著 |
| **💰 成本优势** | 规模/工艺/位置带来的持续低成本？ | 单位成本 vs 行业、经营杠杆 | 成本低于行业 15%+且可持续 |
| **📏 有效规模** | 市场不够大，阻止新竞争者进入？ | 市场规模/已有参与者数量 | 利基市场+已有玩家占据大部分 |

**护城河强度综合判定：**

| 强度 | 标准 | 含义 |
| --- | --- | --- |
| 🟢 宽护城河 (Wide Moat) | ≥2 种强护城河，或 1 种极强护城河 | 竞争优势期 (CAP) ≥ 15 年 |
| 🟡 窄护城河 (Narrow Moat) | 1 种中等强度护城河 | CAP 5-15 年 |
| 🔴 无护城河 (No Moat) | 无显著竞争壁垒 | CAP < 5 年，盈利不可持续 |

**⚠️ 护城河反面检查（防止过度乐观）：**

```
反面搜索: "[公司名] 竞争威胁 市场份额下降 颠覆风险"
```

必须回答：**什么力量可能侵蚀这条护城河？** 如果无法给出合理的侵蚀路径，护城河更可信。

## 1.3 管理层质量评估

> **理论依据**：Tom Gayner 的"人、价格、业务"三要素；Munger 的"找到你能信任的人"；Skin in the Game 原则 (Nassim Taleb)。

**搜索指令（必须执行）：**

```
搜索 5: "[公司名] 管理层 CEO [当前CEO名] 履历 风格"
搜索 6: "[公司名] 管理层持股 内部人交易 [最近年份]"
```

**管理层评估维度（5 维度打分）：**

| 维度 | 检查内容 | 评分标准 |
| --- | --- | --- |
| **Skin in the Game** | 管理层/创始人持股比例、近期增减持 | 🟢 持股>5%且近期无减持 / 🟡 持股1-5% / 🔴 持股<1%或大幅减持 |
| **资本配置能力** | 历史 ROE 趋势、并购回报、分红/回购决策 | 🟢 长期 ROE 稳定>15% / 🟡 ROE 10-15% / 🔴 资本错配/价值毁灭并购 |
| **战略清晰度** | 战略是否一致、是否频繁转型 | 🟢 战略清晰+执行到位 / 🟡 战略合理但执行波动 / 🔴 频繁更换战略 |
| **诚信度** | 历史承诺兑现率、信息透明度 | 🟢 言出必行+主动披露 / 🟡 中性 / 🔴 多次失信或信息不透明 |
| **行业经验** | 核心管理层行业经验深度 | 🟢 行业深耕>10年 / 🟡 5-10年 / 🔴 空降兵或经验<5年 |

**管理层综合评分：**
- 🟢 优秀：4-5 项为绿
- 🟡 中等：2-3 项为绿
- 🔴 差：≥3 项为红

## 1.4 治理风险排查

**搜索指令（必须执行）：**

```
搜索 7: "[公司名] 治理风险 关联交易 质押 诉讼"
```

**治理风险检查清单：**

| 风险项 | 检查内容 | 红旗信号 |
| --- | --- | --- |
| 股权质押 | 大股东质押比例 | >50% 为高风险 |
| 关联交易 | 频繁/大额关联交易 | 金额占收入比>10% |
| 审计意见 | 年报审计意见类型 | 非标准无保留 = 红旗 |
| 信息操纵 | 频繁更换审计师/CFO | 3 年内换 ≥2 次 = 红旗 |
| 诉讼/监管 | 重大未决诉讼或处罚 | 搜索确认 |
| 利益冲突 | 管理层同时运营竞争企业 | 搜索确认 |

## 1.5 公司生命阶段判定

| 阶段 | 特征 | 投资关注点 |
| --- | --- | --- |
| 🌱 初创/IPO | 高增长、不盈利、烧钱 | 产品 PMF、TAM、融资能力 |
| 🚀 高速增长 | 营收增速>30%、份额快速扩张 | 增速持续性、竞争壁垒形成 |
| 📈 稳定增长 | 增速 10-30%、开始盈利 | 盈利质量、护城河深化 |
| 🏢 成熟 | 增速<10%、稳定盈利、现金牛 | 分红回购、资本配置 |
| 🔄 转型 | 核心业务放缓、寻找第二曲线 | 转型逻辑可信度、执行风险 |
| 📉 衰退 | 负增长、市场萎缩 | 价值陷阱风险、残余价值 |
