# Step 4: 股价行为分析

> **目的**：从市场行为的角度验证基本面结论。股价走势包含了所有已知信息和部分未知预期。
> **设计原则**：技术面不是目的，而是验证工具。核心是理解 **"市场在定价什么"**。
> **理论依据**：有效市场假说 (EMH) 的弱式版本 — 价格反映信息但不完美；Wyckoff 方法 — 成交量确认价格趋势；Flow of Funds 分析 — 资金流向预示趋势持续性；George Soros 的反身性理论 — 价格变化影响基本面。

---

## 4.1 股价趋势判断

**搜索指令（必须执行）：**

```
搜索 1: "[公司名/代码] 股价 走势 [当前年份]"
搜索 2: "[公司名/代码] 股价 近期 涨跌 表现"
搜索 3: "[公司名/代码] K线 均线 技术分析"
```

**趋势判断框架：**

| 时间维度 | 判断方法 | 输出 |
| --- | --- | --- |
| **短期 (1-4 周)** | 最近走势 + 成交量配合 | 📈上涨/➡️横盘/📉下跌 |
| **中期 (1-6 月)** | 中期均线方向 + 趋势结构 | 📈上涨/➡️横盘/📉下跌 |
| **长期 (6-24 月)** | 长期均线 + 大级别形态 | 📈上涨/➡️横盘/📉下跌 |

**趋势综合判断表（必须输出）：**

| 维度 | 方向 | 强度 | 依据 |
| --- | --- | --- | --- |
| 短期 | [方向] | [强/中/弱] | [具体数据] |
| 中期 | [方向] | [强/中/弱] | [具体数据] |
| 长期 | [方向] | [强/中/弱] | [具体数据] |

**趋势一致性检查：**

| 短期+中期+长期 | 含义 | 投资含义 |
| --- | --- | --- |
| 全部同向上 | 强势上升趋势 | 趋势投资者关注，但注意追高风险 |
| 全部同向下 | 强势下降趋势 | 避免抄底，等待企稳信号 |
| 方向不一致 | 趋势转折期/震荡 | 需等待方向明确，不宜重注 |

## 4.1b 股价驱动力归因 ⭐ 必填

> **设计原理**：股价走势本身不是分析目标，**理解"市场在定价什么"才是**。如果股价近期大幅上涨但主业基本面没有改善，必须找到真正的驱动力——可能是跨板块概念炒作、资金轮动、政策预期等。忽略驱动力归因 = 分析脱离现实。

**搜索指令（⛔ 必须执行）：**

```
搜索 DA1: "[公司名/代码] 近期 上涨原因 [近期月份]" 或 "[公司名/代码] 近期 下跌原因"
搜索 DA2: "[公司名/代码] 概念 题材 异动"
```

**⚠️ 必须与 Step 0.2c 的跨板块概念扫描结果交叉验证。**

**归因分析框架（必须输出）：**

| 驱动力类型 | 是否为近期主要驱动 | 事实依据 |
| --- | --- | --- |
| 主业基本面改善/恶化 | [是/否] | [引用具体数据] |
| 跨板块概念/题材炒作 | [是/否] | [引用 Step 0.2c 发现 + 搜索结果] |
| 行业板块轮动 | [是/否] | [板块整体走势] |
| 资金面/流动性 | [是/否] | [资金流向数据] |
| 政策/事件催化 | [是/否] | [具体政策/事件] |

**⛔ 关键规则：**
- 如果股价走势与主业基本面明显背离（如"业绩亏损但股价大涨"），**必须找到非基本面驱动力并在报告中明确说明**
- 如果归因结果为"跨板块概念炒作"，在投资论题（Step 5）和反面审查中必须评估该概念的可持续性

## 4.2 关键支撑与阻力位

**搜索指令（必须执行）：**

```
搜索 4: "[公司名/代码] 支撑位 阻力位 关键价位"
```

**支撑/阻力位识别方法：**

| 方法 | 说明 | 优先级 |
| --- | --- | --- |
| **前期高/低点** | 历史重要极值点 | 最高 |
| **成交密集区** | 大量换手发生的价格区域 | 高 |
| **整数关口** | 心理价位（如 ¥100、¥50） | 中 |
| **均线支撑** | 关键均线（60日/120日/250日） | 中 |
| **缺口位置** | 跳空缺口的上下沿 | 中 |

**必须输出：**
- 当前股价: ¥XX
- 第一支撑位: ¥XX (依据: XX)
- 第二支撑位: ¥XX (依据: XX)
- 第一阻力位: ¥XX (依据: XX)
- 第二阻力位: ¥XX (依据: XX)

**⚠️ 支撑阻力是参考而非预测，禁止将其作为"目标价"。**

## 4.3 成交量与资金面分析

**搜索指令（必须执行）：**

```
搜索 5: "[公司名/代码] 成交量 换手率 资金流向"
搜索 6: "[公司名/代码] 北向资金 机构持仓 [最近季度]"
```

**成交量信号矩阵：**

| 价格方向 | 成交量 | 信号含义 | 可信度 |
| --- | --- | --- | --- |
| 📈 上涨 | 放量 | 多头强势推动 | 🟢 高 |
| 📈 上涨 | 缩量 | 无量上涨，持续性存疑 | 🟡 中 |
| 📉 下跌 | 放量 | 恐慌性抛售/机构出逃 | 🔴 高（负面） |
| 📉 下跌 | 缩量 | 自然回调，卖压有限 | 🟡 中性 |
| ➡️ 横盘 | 极度缩量 | 关注不足或准备变盘 | ⚠️ 需观察 |

**资金面评级：**

| 评级 | 信号 |
| --- | --- |
| 🟢 净流入 | 北向/机构连续净买入、大宗交易溢价成交 |
| 🟡 平衡 | 无明显方向性资金流 |
| 🔴 净流出 | 机构连续减仓、大宗折价、融券余额上升 |

## 4.4 筹码结构分析

**搜索指令（必须执行）：**

```
搜索 7: "[公司名/代码] 股东户数 筹码集中度 [最近季度]"
搜索 8: "[公司名/代码] 十大股东 持仓变化"
```

**筹码分析要素：**

| 要素 | 检查内容 | 信号 |
| --- | --- | --- |
| **股东户数变化** | 减少=筹码集中(利好)，增加=筹码分散(利空) | 关键领先指标 |
| **机构持仓比例** | 基金/保险/社保持仓变化 | 机构加仓=认可度提升 |
| **内部人交易** | 管理层/董事近期买卖 | 增持=内部信心，减持=注意 |
| **融资融券** | 融券余额变化 | 融券大幅增加=做空力量聚集 |
| **大宗交易** | 成交价格溢价/折价 | 溢价=有买家愿意付出更高价 |

**筹码结构综合判断：**
- **集中** (🟢)：股东户数减少+机构加仓+内部人增持
- **分散** (🔴)：股东户数增加+机构减仓+内部人减持
- **中性** (🟡)：信号混合

## 4.5 近期异动排查

> **理论依据**：异常交易行为往往预示未被公开的信息或即将到来的催化事件。

**搜索指令（必须执行）：**

```
搜索 9: "[公司名/代码] 异动 利好 利空 公告 [最近月份]"
```

**异动检查清单：**

| 异动类型 | 信号 | 需要调查 |
| --- | --- | --- |
| 突然放量(成交量>3倍均值) | 可能有未公开信息 | 检查近期公告/传闻 |
| 股价大幅偏离指数 | 个股独立行情 | 公司特有事件驱动？ |
| 涨停/跌停 | 极端市场情绪 | 触发原因、是否可持续 |
| 盘后大宗交易 | 机构换手 | 买方身份、溢价/折价 |
| 龙虎榜上榜 | 游资/机构交易 | 买卖方属性 |

## 4.6 股价与基本面的一致性检验

> **核心问题**：股价走势是否与基本面（Step 2 的财务趋势）一致？如果不一致，谁在说谎？

**一致性矩阵：**

| 基本面 | 股价 | 判断 | 可能原因 |
| --- | --- | --- | --- |
| 🟢 改善 | 📈 上涨 | ✅ 一致 | 市场正确定价 |
| 🟢 改善 | 📉 下跌 | ⚠️ 背离 | 预期差？被错杀？系统风险？ |
| 🔴 恶化 | 📈 上涨 | ⚠️ 背离 | 市场过度乐观？反转预期？ |
| 🔴 恶化 | 📉 下跌 | ✅ 一致 | 市场正确定价 |

**背离处理规则：**
- 基本面好+股价跌 → 深入调查是否存在尚未公开的利空
- 基本面差+股价涨 → 深入调查是否存在尚未计入的利好/反转预期
- 背离必须有解释，不能简单归因于"市场非理性"

## Gate Check (验收门禁)

```
□ 三个时间维度趋势已判断
□ ⭐ 股价驱动力归因已完成（与 Step 0.2c 交叉验证）
□ ⭐ 如股价与主业基本面背离 → 已找到非基本面驱动力并记录
□ 关键支撑/阻力位已标注（至少各 2 个）
□ 成交量/资金面方向已确认
□ 筹码结构已分析
□ 近期异动已排查
□ 股价与基本面一致性已检验
□ 已执行 ≥9 次搜索
□ ⭐ 预结论辩证已执行：关键假设已在结论形成前挑战，裁定结果已记录（见 SKILL.md 三步辩证协议）
□ 🔄 纠错协议检查：股价行为是否与 Step 0-3 的基本面结论矛盾？（如矛盾 → 必须找到原因 + 触发纠错协议）
```

> **预结论辩证提示**：在给出技术面结论之前——暂停。问自己"我认定股价处于 [X 阶段]，这最依赖什么假设？"然后执行反面搜索。常见需挑战的假设：技术指标在A股有效、资金流向数据完整可信、无隐藏的大宗交易/定增解禁。
> 
> **纠错协议提示（⚠️ 高频触发点）**：Step 4 是纠错协议最常被触发的步骤。如果股价走势与基本面严重背离（如基本面差但股价暴涨），**不能简单归因为"市场非理性"**——必须搜索是否有未被前序步骤捕获的概念驱动力（跨板块概念、政策利好、并购预期等），找到后触发纠错协议更新 Step 0。
