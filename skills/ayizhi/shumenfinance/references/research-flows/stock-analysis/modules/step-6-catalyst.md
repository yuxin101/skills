# Step 6: 催化剂、情景矩阵与风险管理

> **目的**：从时间维度规划投资，定义不同场景的结果，给出综合评级和风险信号。
> **核心原则**：好的分析不仅回答"该不该看好"，还回答"什么时候"和"什么情况下失效"。
> **理论依据**：Stanley Druckenmiller — "催化剂的识别比估值更重要"；Howard Marks 的"第二层思维" — 不同场景的概率分布；Nassim Taleb 的肥尾风险 — 必须考虑极端场景。


**催化剂分类与日历表（必须输出）：**

### 近期催化剂（0-3 个月）

| 时间 | 催化剂 | 类型 | 方向 | 影响力 |
| --- | --- | --- | --- | --- |
| [日期] | [事件] | [财报/产品/政策/资本市场] | [🟢正/🔴负/🟡不确定] | [高/中/低] |
| ... | ... | ... | ... | ... |

### 中期催化剂（3-12 个月）

| 时间(预计) | 催化剂 | 类型 | 方向 | 影响力 |
| --- | --- | --- | --- | --- |
| [时间窗口] | [事件] | [财报/产品/政策/行业/宏观] | [🟢正/🔴负/🟡不确定] | [高/中/低] |
| ... | ... | ... | ... | ... |

**催化剂类型说明：**

| 类型 | 示例 | 特点 |
| --- | --- | --- |
| 📊 财报催化剂 | 季度/年度业绩发布 | 时间确定，影响可预判 |
| 🚀 产品催化剂 | 新品发布/重大合同 | 时间较确定，影响看兑现度 |
| 📜 政策催化剂 | 监管政策/补贴 | 时间不确定，影响可能剧烈 |
| 💰 资本市场催化剂 | 回购/增发/纳入指数 | 可量化影响 |
| 🏭 行业催化剂 | 供给侧变化/需求爆发 | 影响全行业 |
| 🌍 宏观催化剂 | 利率变动/汇率/贸易 | 系统性影响 |

**⚠️ 催化剂的"时机陷阱"**：
- 已被市场充分预期的催化剂 = 利好出尽
- 时间不确定的催化剂 = 不宜作为短期投资依据
- 需区分"确定性催化剂"（如已公布的财报日期）和"可能性催化剂"（如传闻中的政策）

## 6.2 情景矩阵 (Scenario Matrix)

> **设计原则**：使用定性描述而非伪精确概率。

**四场景模型（必须输出）：**

| 场景 | 描述 | 关键假设 | 对应估值含义 | 置信度 |
| --- | --- | --- | --- | --- |
| 🐂 **Bull** | [最乐观但合理的情境] | [需要什么条件成立] | [估值会到什么水平] | [高/中/低] |
| 📊 **Base** | [最可能的情境] | [基于当前趋势外推] | [估值基本合理] | **最高** |
| 🐻 **Bear** | [合理的悲观情境] | [什么条件恶化] | [估值下行空间] | [高/中/低] |
| 💀 **Tail** | [极端但不可忽视的风险] | [黑天鹅事件] | [重大损失可能] | 低但不可忽视 |

**情景构建规则：**
1. **每个场景必须有明确的触发条件**，不能只是"如果涨了就是牛市"
2. **Bull/Bear 必须基于可验证的假设变化**，不是简单的±X%
3. **Tail 场景是强制的**，哪怕概率很低（Taleb 原则）
4. **禁止给场景赋精确概率**，如"Bull 30%、Base 50%、Bear 20%" — 用定性描述

## 6.3 综合评级 (Investment Rating)

**基于全流程分析结论，给出综合评级：**

| 评级 | 含义 | 触发条件 |
| --- | --- | --- |
| 🟢 **强烈看好** | 高置信度买入信号 | 三角全对齐 + 估值低估 + 正向预期差 + 红队稳固 + 明确催化剂 |
| 🟢 **看好** | 值得关注 | 三角基本对齐 + 估值合理偏低 + 红队稳固或有轻微弱点 |
| 🟡 **谨慎乐观** | 有机会但需谨慎 | 逻辑成立但估值偏高 / 或催化剂不明确 / 或红队有弱点 |
| ⚪ **观望** | 等待更好时机 | 信号混乱 / 估值无吸引力 / 或红队动摇 |
| 🔴 **回避** | 风险大于收益 | 逻辑有硬伤 / 估值高估 / 财务红旗 / 红队摧毁 |

**评级约束规则（硬性）：**

| 条件 | 评级上限 |
| --- | --- |
| 红队评级 = ❌ 已动摇 | 最高 ⚪ 观望 |
| 财务红旗 ≥ 2 个高严重度 | 最高 🟡 谨慎乐观 |
| 护城河 = 🔴 无 | 最高 🟡 谨慎乐观 |
| 宏观 = 🔴 紧缩 + 行业 = 📉 下行 | 最高 🟡 谨慎乐观 |
| 估值 = 高估 + 无安全边际 | 最高 🟡 谨慎乐观 |
| 反身性风险 = 🔴 高 | 当前评级降一档 |
| 监管 = 🔴 毁灭级 | 最高 🔴 回避 |

**⚠️ 评级必须附加周期位置修饰语：**

> **设计原因**：单纯的"看好"可能让投资者在短期高位追高。方向性评级必须结合当前股价位置和估值给出**时机建议**。

| 评级 | 修饰语规则 | 输出示例 |
| --- | --- | --- |
| 🟢 强烈看好/看好 | **必须**指明当前是"低位积极配置"还是"高位注意回调后关注" | "🟢 看好（当前估值处历史低位，逢低积极关注）" / "🟢 看好（短期涨幅已大，注意回调风险后再介入）" |
| 🟡 谨慎乐观 | 指明等待什么条件转为看好 | "🟡 谨慎乐观（等待 XX 催化剂确认 / 估值回落至 XX 区间）" |
| ⚪ 观望 | 指明观望原因和关注信号 | "⚪ 观望（逻辑未被证伪但缺乏催化剂，关注 XX 信号）" |

**⛔ 评级禁止事项：**
- ❌ 禁止给出"买入"/"卖出"的具体交易指令
- ❌ 禁止给出具体仓位比例建议（如"建议配置 X%"）
- ❌ 禁止给出具体买入/卖出价位（用支撑/阻力区间替代）
- ❌ 禁止裸输出评级——每个评级必须附带修饰语和逻辑依据
- ✅ 只给方向性判断 + 时机提示 + 逻辑依据 + 关注要点

## 6.4 风险管理框架

**风险信号分类：**

| 风险类型 | 描述 | 监控指标 |
| --- | --- | --- |
| **逻辑失效风险** | 核心投资论题被证伪 | 逻辑失效 触发 |
| **估值泡沫风险** | 股价透支未来太多年增长 | DCF 隐含增速 vs 合理增速 |
| **财务恶化风险** | 盈利质量/财务健康度下滑 | 季度跟踪财务红旗 |
| **流动性风险** | 成交量萎缩导致退出困难 | 日均成交额、换手率 |
| **黑天鹅风险** | 不可预见的极端事件 | Tail 场景定义 |
| **时间损耗风险** | 论题正确但催化剂迟迟不来 | 机会成本评估 |

**最终风险提示输出（必须）：**

```
⚠️ 核心风险：
1. [风险描述] — 逻辑失效: [触发条件]
2. [风险描述] — 监控指标: [具体指标]
3. [风险描述] — 时间窗口: [X 个月内关注]
```

## 6.5 分析时间框架声明 (Analysis Time Horizon)

> **设计原理**：所有投资分析都隐含一个时间框架，但大多数框架从不显式声明。不说明时间框架等于默认 6-12 个月——在范式级变革中（石油替代鲸油用了 25 年），这个默认值可能导致系统性误判：方向判断正确但时间窗口错误 = 实际投资亏损。

**时间框架根据范式量级动态调整：**

| 范式量级 | 默认分析视野 | 论题有效期 | 复核频率 |
| --- | --- | --- | --- |
| **S1 增量型 / 不适用** | 6-12 个月 | 下一个财报周期 | 每季度 |
| **S2 显著型** | 12-24 个月 | 2-4 个季度的兑现窗口 | 每季度 |
| **S3 范式型** | 12-36 个月 | 允许更长期论题，但每季度复核进展 | 每季度（强制） |

**⚠️ 长视野 ≠ 忽略短期风险。**

即使分析视野是 36 个月，以下短期风险仍**优先级最高、不受时间框架影响**：
- 流动性危机 / 债务违约风险
- 退市 / 重大合规风险
- 重大诉讼 / 监管处罚
- 管理层诚信危机

> 时间框架扩展只影响"论题方向判断"和"催化剂兑现预期"，**不影响"生存性风险评估"**。

**必须输出：**

```
分析时间框架: [6-12月/12-24月/12-36月]
时间框架依据: [范式量级 S[x] + 成熟度迁移速度预判]
短期生存性风险: [有(列出)/无]
论题复核节点: [下一次复核的具体时间/事件]
```

## 6.7 主预期方向判定 (Primary View Determination) ⭐

> **设计原理**：高盛、摩根士丹利的研报在给出评级后，仅展示**一个方向预判**——这是分析师的核心价值所在。投资者需要的不是"三种可能"，而是"基于你的分析，最大概率的方向是什么"。
>
> 方向判定必须**逻辑+事实驱动**，禁止主观猜测。每个判定要素都来自前序步骤的确定输出。

**判定流程（五要素加权）：**

| 优先级 | 判定要素 | 数据来源 | 评估标准 |
| --- | --- | --- | --- |
| ⭐⭐⭐ | **核心论题有效性** | Step 5 红队评级 | ✅稳固→看好方向 / ⚠️有弱点→中性偏上 / ❌动摇→看空方向 |
| ⭐⭐ | **估值位置** | Step 3 估值综合结论 | 低估→向上弹性大 / 合理→空间有限 / 偏高→需消化 / 高估→向下压力 |
| ⭐⭐ | **近期股价动能** | Step 4 趋势分析 | 上行→顺势 / 横盘→等待 / 下行→逆势需更强逻辑 |
| ⭐ | **最近催化剂方向** | Step 6.1 催化剂日历(0-3月) | 多数催化为正→正面预期 / 不确定→中性 / 多数为负→负面预期 |
| ⭐ | **行业景气方向** | Step 0 行业评估 | 上行→顺风 / 稳定→中性 / 下行→逆风 |

**判定→方向映射矩阵：**

| 论题有效性 | 估值位置 | → 主预期方向 | 趋势描述 |
| --- | --- | --- | --- |
| ✅稳固 | 低估/合理偏低 | **↗ 看好（明确上行）** | 明确上行，弹性较大 |
| ✅稳固 | 合理/偏高 | **↗→ 谨慎乐观（先横后上）** | 先横向，催化事件后转上 |
| ✅稳固 | 高估 | **→↗ 谨慎乐观（先消化后兑现）** | 先微跌/横向消化估值，再缓慢上行 |
| ⚠️有弱点 | 低估/合理 | **↗→ 谨慎乐观（修复中）** | 缓慢向上但幅度有限 |
| ⚠️有弱点 | 偏高/高估 | **→ 观望（逻辑+估值双压）** | 横向震荡 |
| ❌动摇 | 任何 | **↘ 回避** | 下行趋势 |

**⛔ 硬性覆盖规则（以下条件直接决定方向，不受矩阵影响）：**
- 宏观环境 🔴 紧缩 → 主预期方向**最高**→ 横向，禁止画明确上行
- 监管风险 🔴 毁灭级 → 主预期方向强制 ↘ 回避
- 反身性风险 🔴 高 → 主预期方向**降一档**

**必须输出：**

```
主预期方向: [↗看好/↗→谨慎乐观/→观望/↘回避]
方向依据: [论题有效性] + [估值位置] + [近期动能] → [判定逻辑一句话]
方向标注: "[方向图标] [核心逻辑一句话]"
```

## Gate Check (验收门禁)

```
□ 催化剂日历已完成（近期+中期）
□ 情景矩阵已完成（Bull/Base/Bear/Tail）
□ 综合评级已给出
□ 评级约束规则已检查
□ 确认无具体仓位/买卖点建议
□ 风险信号清单已列出
□ 逻辑失效 已关联到风险管理
□ 分析时间框架已显式声明
□ 短期生存性风险已独立于时间框架评估
□ ⭐ 主预期方向判定已完成（五要素评估 + 方向映射 + 方向标注）
□ ⭐ 主预期方向与综合评级强对应（🟢↗ / 🟡↗→ / ⚪→ / 🔴↘）
□ 投资论题一句话总结已输出（含视野+范式定位）
□ 已执行 ≥3 次搜索
□ ⭐ 预结论辩证已执行：综合评级/方向判定的关键假设已挑战（见 SKILL.md 三步辩证协议）
□ 🔄 纠错协议最终检查：全流程是否有未解决的矛盾？共享上下文中的 🔄 修正标记是否已全部反映到评级中？
```

> **预结论辩证提示**：在给出最终综合评级之前——暂停。问自己"催化剂时间线假设是否合理？如果催化剂延迟 6 个月，评级是否还能维持？"
> 
> **纠错协议最终检查提示**：Step 6 是最后一个分析步骤。回顾共享上下文表中所有 🔄 修正标记，确认每一处修正已正确传播到最终评级。如果发现遗漏——现在修正，不要带入报告。
