# Step 6: 风险管理框架 (Risk Management & Direction Assessment)

> **输入**: 全部前序步骤上下文 + 红队评级(Step 4) + 宏观评级(Step 0)
> **输出**: 综合置信度、方向性建议、止损逻辑、情景矩阵、跨赛道风险提醒
> **核心原则**: 本框架提供方向性判断和风险提示，不提供具体仓位和买卖指令

### 9. Direction Assessment & Risk Discipline (方向判断与风险纪律) ⭐⭐ 重构

> **设计原则：** 本 skill 定位为**方向性分析框架**——告诉投资者"这个赛道方向怎么看、需要关注什么风险"，而不是"买多少股、什么价位入场"。具体的仓位管理和交易执行是投资者自己的责任，不在本框架的输出范围内。

**⚠️ 报告输出方向性建议和风险提示，不输出具体仓位比例和买卖点位。**

#### A. 方向判断矩阵（双轴驱动：逻辑强度 × 环境修饰）

> **设计原则：方向判断的本质是回答"这个赛道的逻辑对不对"。宏观环境和周期位置影响的是"现在介入的时机好不好"，而非逻辑本身的方向。因此：核心逻辑是主驱动轴，宏观/监管/世俗力量是修饰轴。**

**第一步：基于核心逻辑确定方向基线**

| 核心逻辑状态（Step 4 红队评级） | 世俗力量方向 | → 方向基线 |
| --- | --- | --- |
| **✅ 稳固**（红队反证未成立或仅边缘风险） | 📈上行 或 ➡️稳态 | 🟢 **看好** |
| **✅ 稳固** | 📉下行 | 🟡 **谨慎乐观**（逻辑对但长期趋势有阻力） |
| **⚠️ 结构性裂缝**（核心前提受质疑） | 任何 | 🟡 **谨慎乐观** |
| **❌ 已动摇**（核心前提被推翻） | 任何 | 🔴 **看空/远离** |

**第二步：环境修饰（在基线上调整）**

| 环境因素 | 对方向基线的影响 |
| --- | --- |
| **宏观🟢 宽松** | 不变（顺风环境，基线已充分） |
| **宏观🟡 中性** | **不变**（中性环境是常态，不应惩罚方向判断） |
| **宏观🔴 紧缩** | **降一档**（🟢→🟡，🟡→⚪） |
| **宏观⚫ 危机** | **降至⚪或🔴**（系统性风险压倒一切） |
| **监管🟢/🟡中低（规范化）** | 不变，仅作 Tips 提示 |
| **监管🟡中** | 最多降半档 |
| **监管🟠中高** | 降一档 |
| **监管🔴毁灭级** | 直接降至🔴 |

> **⚠️ 关键变化说明（vs 旧版）：**
> - **旧版**：🟢看好 需要"✅稳固 + 宏观🟢 + 红队通过 + 监管🟢/🟡中低"——四条件 AND 逻辑，🟡中性宏观即封死🟢
> - **新版**：🟢看好 由"✅稳固 + 世俗力量非📉下行"决定，宏观🟡**不降级**，只有🔴才降级
> - **设计原因**：宏观🟡是绝大多数时间的常态。如果🟡就堵死🟢，则"看好"这个档位形同虚设，所有报告无差别输出"谨慎乐观"——这对投资者的决策帮助为零

**第三步：结合周期位置输出（保护投资者不追高）**

| 方向判断 | 当前周期位置 | 输出表述 |
| --- | --- | --- |
| 🟢 看好 | 底部/回调中 | "🟢 看好 — 逢低值得重点关注" |
| 🟢 看好 | 高位/阶段性顶部 | "🟢 看好 — 方向向好，但短期处于高位，**注意回调风险，不宜追高**" |
| 🟡 谨慎乐观 | 底部/回调中 | "🟡 谨慎乐观 — 逢低关注，但需跟踪验证" |
| 🟡 谨慎乐观 | 高位 | "🟡 谨慎乐观 — 方向偏正，**高位追入风险大，等待回调**" |
| ⚪ 观望 | 任何 | "⚪ 观望 — 等待更多确认信息" |
| 🔴 看空 | 反弹中 | "🔴 看空 — 逻辑已失效，反弹不改方向" |

> **核心原则："看好"≠"现在就追"。看好是方向判断，是否适合介入还要看当前价格位置。**
> **核心原则：方向判断的区分度才是价值所在。如果所有概念都输出同一个结论，等于没有分析。**

#### B. 风险信号与逻辑失效触发条件

**核心原则：列出可能导致方向判断改变的关键风险信号，帮助投资者建立自己的"退出条件"。**

| 风险类型 | 触发条件 | 含义 | 说明 |
| --- | --- | --- | --- |
| **逻辑失效信号** | 核心逻辑被证伪（反面审查❌已动摇） | 方向判断需要根本性修正 | 最重要的信号——逻辑变了，方向就变 |
| **极端异动信号** | 板块指数大幅偏离预期（如跌幅超 -30% 且无已知利空） | 可能存在未识别的负面因素 | 需重新审视分析假设 |
| **时间失效信号** | 核心逻辑超过 6 个月未兑现，且无新催化剂 | 逻辑可能正确但时点可能错误 | 需重新评估时间线 |
| **宏观恶化信号** | 宏观体制从🟢/🟡恶化为🔴/⚫ | 系统性风险优先于赛道逻辑 | 所有赛道都可能受影响 |

#### C. 报告输出要求

**在所有报告中，必须输出：**

```markdown
### 方向判断与风险提示

| 项目 | 内容 |
| --- | --- |
| **综合置信度** | [High/Medium/Low/Negative] |
| **方向判断** | [🟢看好 / 🟡谨慎乐观 / ⚪观望 / 🔴看空远离] + **周期位置修饰语**（如"逢低关注"/"高位注意回调"） |
| **核心关注点** | [一句话描述当前最需要关注的变量] |
| **逻辑失效触发条件** | 如果 [具体事件] 发生 → 方向判断需修正 |
| **时间窗口** | 关键验证/催化事件预计在 [时间] 出现 |
| **宏观风险提醒** | 当前宏观环境对该赛道的影响方向 |
```

> **⚠️ 免责声明：** 本报告仅提供方向性分析和风险提示，不构成具体的交易指令、仓位建议或投资建议。投资者应结合个人风险承受能力、资金规模、投资期限等因素自行决策。

#### D. 跨赛道相关性风险提醒 (Cross-Sector Correlation Warning) ⭐ 新增

> **设计原因：** 如果用户同时对两个赛道使用此框架（如"猪肉"和"饲料"），两个独立分析可能同时给出"买入"建议，但实际风险敞口高度相关甚至对冲。孤立分析会忽视组合层面的风险集中。

**强制提醒（在报告末尾输出）：**

> ⚠️ **跨赛道风险提醒：** 本报告为单一赛道的独立分析。如果您同时持有或关注以下相关赛道，请注意风险相关性：
> - **上下游关联**：[概念]的上游/下游赛道（如猪肉↔饲料、光伏↔多晶硅），持仓方向相同则风险叠加，方向相反则部分对冲
> - **替代品关联**：[概念]的替代品赛道（如燃油车↔新能源车），两者通常负相关
> - **宏观敏感度**：如果多个持仓赛道对同一宏观因子（如利率、汇率）高度敏感，需注意系统性风险集中
>
> 建议：对组合中各赛道的宏观敏感度、产业链位置进行交叉检查，避免"看似分散实则集中"的风险。

### Institutional Verdict (情景矩阵) — v2.0 定性化重构

> **⚠️ 概率政策统一声明：** 本框架禁止输出伪精确概率百分比（见 Step 4.6 设计原则）。此原则适用于所有模块，包括情景矩阵。以下使用定性置信度区间替代百分比数字。
>
> **原因：** LLM 基于搜索结果给出"乐观30%/基准50%/悲观20%"是数字安慰剂——这些百分比没有任何统计基础，但投资者会将其锚定为决策依据。

| 情景 | 置信度 | 逻辑推演 | 方向含义 |
| --- | --- | --- | --- |
| 🐂 **乐观 (Bull)** | [高/中/低] 可能 | 核心逻辑超预期 + 流动性充裕 | 方向进一步看好 |
| ⚖️ **基准 (Base)** | **最可能** | 当前趋势延续 | 维持当前方向判断 |
| 🐻 **悲观 (Bear)** | [高/中/低] 可能 | **逻辑证伪** 或 **流动性枯竭** | **方向需修正，需关注风险信号** |
| 🐱 **死猫反弹 (Dead Cat)** | 仅限夕阳赛道 | 夕阳赛道短期反弹后继续下跌 | **方向向下未变，反弹不改趋势** |
| 🦢 **尾部风险 (Tail)** | 小概率但致命 | 监管🔴毁灭级黑天鹅/致命缺陷/不可预见事件（注：监管🟡中低的"规范化"不属于此类） | 极限风险提示 |

**⛔ 禁止行为：**
- ❌ 禁止在情景矩阵中填写百分比数字（如"30%"、"50%"）
- ✅ 仅允许使用定性置信度：`最可能` / `高可能` / `中可能` / `低可能` / `小概率但致命`
- ✅ 必须标明哪个情景是"基准情景"（最可能发生的）

### 关键变量敏感性 (Key Variable Sensitivity) ⭐ 华尔街标准

> **理论依据**：UBS 风险管理 — 赛道分析的方向判断高度依赖少数关键变量。识别并量化这些变量的影响，让投资者知道"什么变了我的结论就变了"。

**关键变量识别（必须输出，选取 2-3 个对赛道方向影响最大的变量）：**

| 关键变量 | 当前状态 | 来源 | 如果恶化 → 方向判断变化 | 如果改善 → 方向判断变化 |
| --- | --- | --- | --- | --- |
| [变量 1: 如核心需求增速] | [当前值/趋势] | Step 3 数据 | [🟢→🟡 / 🟡→⚪ / etc.] | [🟡→🟢 / etc.] |
| [变量 2: 如政策方向] | [当前状态] | Step 0 宏观 | [方向变化描述] | [方向变化描述] |
| [变量 3: 如供给端产能] | [当前状态] | Step 4 数据 | [方向变化描述] | [方向变化描述] |

**情景路径图（将情景矩阵与关键变量联动）：**

```
当前方向判断: [🟢/🟡/⚪/🔴]
                          │
     ┌────────────────────┼────────────────────┐
     │                    │                    │
  变量1 恶化           变量维持            变量1 改善
  → [新方向]           → [当前方向]         → [新方向]
     │                    │                    │
  触发信号:            维持条件:            催化信号:
  [具体可观测事件]      [XX 继续 YY]         [具体可观测事件]
```

**⚠️ 敏感性核心发现（必须输出）：**

```
最敏感变量: [变量名] — 该变量方向变化将直接改变赛道方向判断
方向判断鲁棒性: [高(需多个变量同时恶化才改变)/中(单变量恶化即影响)/低(高度依赖单一变量)]
投资者最应跟踪: [具体指标/事件] — [建议跟踪频率：月度/季度/事件驱动]
```

> **传导**: 关键变量敏感性直接影响 B. 风险信号的定义 — "逻辑失效信号"应聚焦于最敏感变量的恶化。
