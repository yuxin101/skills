### 9.1 [事实] vs [推演] 标注规则 — ⛔ 致命级

| 标注            | 定义                             | 示例                                               |
| --------------- | -------------------------------- | -------------------------------------------------- |
| **[事实]**      | 来自搜索结果的客观数据、官方统计 | **[事实]** 美联储 YYYY-MM-DD 维持利率 X%-Y%            |
| **[推演]**      | 基于事实的逻辑推理、趋势预判     | **[推演]** 利率维持意味着流动性不会进一步收紧          |
| **[事实+推演]** | 事实数据 + 基于该数据的直接推断  | **[事实+推演]** CAGR 15%（数据）→ 属于朝阳赛道（判断） |

规则：

1. 每段关键分析至少 1 个标注
2. 定量数据必须标 **[事实]** + 数据来源
3. 未来预期必须标 **[推演]** + 推理依据
4. ⛔ 禁止将推演伪装为事实
5. ⛔ 标注必须加粗：输出为 `**[事实]**`、`**[推演]**`、`**[事实+推演]**`，不允许不加粗的裸标注

---

### 9.2 来源归因（内联标注，不设独立章节）

| 数据类型      | 归因要素                   | 示例格式                                   |
| ------------- | -------------------------- | ------------------------------------------ |
| 量化数据      | 来源机构 + 发布日期        | `XX亿（来源: [机构], [YYYY-MM-DD]）`       |
| 机构预测      | 机构名 + 发布日期          | `预测XX —— [机构]（[日期]）`               |
| 资金/持仓数据 | 数据平台 + 日期            | `净流出XX亿（来源: [平台], [YYYY-MM-DD]）` |
| 政策信息      | 政策名称 + 发布机构 + 日期 | `[政策名]（[机构], [日期]）`               |

**⛔ 来源可信度约束：**

- 禁止使用低可信度转载平台（豆丁网、道客巴巴、百度文库等）作为唯一来源
- 如仅从转载平台获取数据，必须标注"（转引自[原始机构]，未能验证原文）"
- 优先使用：官方公告/统计局 > 券商研报 > 权威财经媒体 > 行业机构报告

### 9.3 报告结构模板

### ⛔ 精简输出原则 — 强制执行

> 投资者需要的是**结论+关键证据**，而非分析过程的完整记录。

1. **结论前置**: 每章第一行直接给核心结论
2. **苏格拉底引导**: 每章结论后紧跟 1 行斜体关键问题（引导读者独立判断）。格式：`*关键问题：...？*`。不使用引用块 `>`，仅用斜体。
3. **禁止元信息**: 不输出"生成时间"、"分析模型"等元数据头
4. **禁止规范性标题**: 章节标题简洁，不加"（仅文字呈现）"等说明
5. **表格优先**: 同类数据用表格呈现
6. **删除套话**: 修饰语全部删除
7. **不设独立的来源归因/信息验证章节**: 来源在正文中内联标注
8. **纠错透明度**: 报告末尾（免责声明前）输出 1 行纠错状态声明
9. **字数控制**: 总报告 3000-4500 字
10. **禁止伪精确**: 情景矩阵中预期表现用涨跌幅区间，禁止精确目标价

### 视觉规范 — 必须遵守

- **决策仪表盘**: 每行评估列必须含状态 icon（🟢🟡🔴⚪📈←↗✅⚠️❌等）
- **章节编号**: 使用中文数字序号 `## 一、` `## 二、`，子节用阿拉伯数字 `### 1.` `### 2.`
- **龙头对比表**: 关键公司/指标用 **粗体** 突出
- **反面审查裁决 icon**: ✅辩护成立 / ⚠️部分成立 / 🔴质疑成立
- **催化剂**: 类型用 icon（📊财报/🚀产品/🏭产能/📜政策/🌍国际），方向用 🟢正/🔴负
- **逻辑失效**: 表格化（类型+条件+验证方法）
- **情景矩阵前缀**: 🟢乐观/⚪基准/🔴悲观/⚫极端
