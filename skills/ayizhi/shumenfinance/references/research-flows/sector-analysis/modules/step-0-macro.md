# Step 0: 宏观体制检查 (Macro Regime Check)
> **输入**: 用户输入的概念名称
> **输出**: 宏观体制评级、反身性风险、监管风险
> **阻断条件**: 宏观⚫ → 终止分析

---

### 0. Macro Regime Check (宏观体制检查) ⭐⭐ 新增 — 高盛投委会要求

**⚠️ 在分析任何赛道之前，必须先判断宏观环境（Beta）。在系统性熊市中，95%的赛道逻辑都无法对抗流动性枯竭。**

> 💡 **业内共识原则**: _"Don't fight the Fed. No sector alpha can survive a liquidity drought. Check the macro regime BEFORE you fall in love with any micro story."_
> （不要与央行作对。没有任何赛道 Alpha 能在流动性枯竭中生存。先检查宏观体制，再爱上任何微观故事。）

**强制搜索指令：**

| 维度           | 搜索指令                                                                 | 判断标准                 |
| -------------- | ------------------------------------------------------------------------ | ------------------------ |
| **利率环境**   | `美联储 利率决议 最新`, `中国 LPR 利率 [当前年份]`, `央行 货币政策 方向` | 加息/降息/持平           |
| **市场流动性** | `A股 成交量 近期`, `两融余额 变化`, `M2 增速`, `社融 数据`               | 充裕/中性/枯竭           |
| **风险偏好**   | `VIX 恐慌指数`, `A股 大盘 走势 近3月`, `北向资金 流入 流出`              | Risk-On/Neutral/Risk-Off |
| **系统性风险** | `金融风险 [当前年份]`, `信用违约 事件`, `汇率 压力`, `地缘政治 风险`     | 有/无重大系统性风险      |

**宏观体制判定矩阵：**

| 宏观体制                 | 特征                                                   | 对个股/赛道分析的影响                                                                                        | 信号灯  |
| ------------------------ | ------------------------------------------------------ | ------------------------------------------------------------------------------------------------------------ | ------- |
| **🟢 宽松 (Easing)**     | 降息/降准周期 + 流动性充裕 + 成交量放大 + 风险偏好上升 | **正常执行赛道分析**。宏观顺风，赛道逻辑可以充分兑现                                                         | ✅ 绿灯 |
| **🟡 中性 (Neutral)**    | 利率持平 + 流动性适度 + 市场震荡                       | **正常执行但提高警惕**。赛道逻辑可行，但需关注边际变化                                                       | ⚠️ 黄灯 |
| **🔴 紧缩 (Tightening)** | 加息/缩表周期 + 流动性收紧 + 成交量萎缩 + 风险偏好下降 | **大幅提高买入门槛**。即使赛道基本面良好，也可能因流动性撤退而下跌。所有"买入"建议必须附加"宏观逆风"风险提示 | 🚨 红灯 |
| **⚫ 危机 (Crisis)**     | 系统性金融风险爆发 + 信用冻结 + 恐慌性抛售             | **暂停所有买入建议**。输出"等待宏观企稳后再评估"。仅允许输出"避险/现金为王"建议                              | ⛔ 熔断 |

**输出要求（在报告最前面体现）：**

```markdown
> **宏观体制：[🟢 宽松 / 🟡 中性 / 🔴 紧缩 / ⚫ 危机]** > **宏观依据：[一句话描述，如："美联储降息周期+A 股成交量回暖"]** > **对本次分析的影响：[正常执行 / 提高警惕 / 买入门槛提高 / 暂停买入建议]**
```

**⚠️ 宏观对方向判断的影响（与 Step 6 矩阵联动）：**

- 🟢 **宽松体制**：不影响方向基线（顺风环境）
- 🟡 **中性体制**：**不影响方向基线**（中性是常态，不应系统性惩罚方向判断）
- 🔴 **紧缩体制**：方向判断**降一档**（🟢→🟡，🟡→⚪），报告标题增加 `[⚠️ 宏观逆风]` 标签
- ⚫ **危机体制**：终止赛道分析流程，直接输出"系统性风险警告，建议持有现金/避险资产"

**⚠️ 反身性检查 (Reflexivity Check) — 索罗斯原则：**

> **核心问题：股价下跌本身是否会反过来恶化基本面？**

| 反身性风险   | 触发条件                          | 恶化链条                                        | 搜索验证                                           |
| ------------ | --------------------------------- | ----------------------------------------------- | -------------------------------------------------- |
| **质押爆仓** | 大股东高比例质押 + 股价接近平仓线 | 股价 ↓ → 强制平仓 → 股价继续 ↓                  | `[龙头股] 股权质押 比例`, `[龙头股] 平仓线`        |
| **融资断裂** | 企业依赖再融资 + 市场冻结         | 股价 ↓ → 无法融资 → 现金流断裂 → 基本面恶化     | `[龙头企业] 再融资 计划`, `[龙头企业] 现金流 状况` |
| **客户信心** | 预付款/长约模式 + 股价暴跌        | 股价 ↓ → 客户担忧 → 订单取消 → 收入下降         | `[概念] 客户 流失`, `[概念] 订单 取消`             |
| **人才流失** | 科技/创新型企业 + 股权激励失效    | 股价 ↓ → 期权水下 → 核心人才离职 → 创新能力下降 | `[龙头企业] 人才 流失`, `[龙头企业] 裁员`          |

**反身性风险评级：**

- **低**：企业现金流充裕，无质押风险，不依赖再融资 → 股价下跌不影响基本面
- **中**：存在部分质押或融资依赖，但尚有缓冲 → 需要关注但不构成致命威胁
- **高**：高质押 + 高杠杆 + 依赖再融资 → **股价下跌会自我强化，形成死亡螺旋。必须在报告中红色标注！**

**⚠️ 行业监管风险评估 (Regulatory Risk Check) ⭐ 新增 — 独立模块**

> **设计原因：** 许多赛道最大的风险不是市场风险，而是监管政策风险。2021年教培"双减"政策导致整个行业归零；2020-2021年互联网反垄断导致头部公司市值腰斩。宏观层面的系统性风险检查无法覆盖这种行业级监管风险。

**强制搜索指令：**

| 维度 | 搜索指令 | 检查目的 |
| --- | --- | --- |
| **现行政策环境** | `[概念] 政策 监管 [当前年份]`, `[概念] 行业法规 最新` | 当前监管态度是鼓励、中性还是收紧？ |
| **政策风险信号** | `[概念] 整顿 约谈 处罚`, `[概念] 反垄断 合规` | 是否有收紧信号？ |
| **历史教训** | `[概念] 政策变化 影响 股价`, `[类似行业] 监管风险 案例` | 同类行业是否有过监管黑天鹅？ |
| **政策依赖度** | `[概念] 补贴 政策依赖`, `[概念] 牌照 准入门槛` | 如果补贴取消/政策转向，行业是否能独立生存？ |

**监管风险评级：**

> **⚠️ 核心区分原则：区分"规范型监管"和"毁灭型监管"。**
> 绝大多数行业的监管是"管而不禁"——政府既要规范秩序又要产业发展。只有极少数情况（如教培"双减"）才是彻底否定商业模式的毁灭型监管。
> **不要把"正常的行业规范化"与"行业归零风险"混为一谈。**

| 评级 | 特征 | 典型场景 | 对投资结论的影响 |
| --- | --- | --- | --- |
| 🟢 **低风险** | 政策明确鼓励 + 无收紧信号 + 不依赖补贴 | 新能源(补贴期)、半导体(国产替代) | 正常分析，监管为正面因子 |
| 🟡 **中低风险(规范化)** | **"促进+规范并行"**：政策总体鼓励发展，同时出台行业规范/标准 | 短剧(行动计划+管理提示)、游戏(版号+防沉迷)、直播电商(合规要求) | **作为 Tips 提示风险，不影响趋势判断和置信度。** 在催化剂日历中标注政策节点 |
| 🟡 **中风险** | 政策态度模糊 或 部分依赖补贴 或 有轻微收紧信号但未形成系统性打压 | 光伏(补贴退坡)、医药(集采) | 标注风险，纳入情景矩阵悲观情景 |
| 🟠 **中高风险** | 已有明确收紧信号且力度持续加大 + 行业部分环节已受实质性冲击 | 电子烟(禁售令)、P2P(清退中) | **提高警惕，情景矩阵必须包含监管恶化情景，适度降低置信度** |
| 🔴 **高风险(毁灭级)** | 政策明确否定商业模式 或 行业存在"双减"式归零可能 或 已进入全面清退/取缔状态 | 教培(2021双减)、ICO(全面禁止) | **红色突出标注。情景矩阵必须包含"监管黑天鹅"尾部情景。显著降低置信度** |

**⚠️ 判定要点——防止过度恐慌：**

1. **"出台管理办法"≠"收紧"**：制定行业标准是规范化的标志，通常意味着行业被官方认可并纳入正式管理体系，是行业成熟的信号而非打压信号
2. **"治理整顿部分违规内容"≠"行业级风险"**：清理低质内容有利于头部合规企业，属于竞争优化
3. **中国市场特征**：对内容行业的监管通常是"管而不禁"（管理≠禁止），尤其当行业贡献就业和税收时，彻底打压的概率极低
4. **只有同时满足以下条件才应定为🔴**：(a) 政策明确否定核心商业模式 (b) 有前例对标（如双减、P2P清退） (c) 行业无法通过合规调整生存

**监管评级对下游的影响传导规则：**

| 监管评级 | 对综合置信度的影响 | 对仓位建议的影响 | 在报告中的呈现方式 |
| --- | --- | --- | --- |
| 🟢 | 无影响 | 无影响 | 不需要特别提及 |
| 🟡中低(规范化) | **不影响置信度** | **不影响仓位** | 作为 **Tips 短期风险提示**，在催化剂日历中标注政策节点 |
| 🟡中 | 最多降半档 | 不直接影响 | 纳入悲观情景描述 |
| 🟠中高 | 降一档 | 建议降低仓位上限 | 独立风险段落 + 情景矩阵包含监管恶化情景 |
| 🔴毁灭级 | 降至 Low 或 Negative | 压至试探或零仓 | 红色突出标注 + 尾部情景 |
