# 圆桌论坛技能

**版本**: v1.0  
**创建日期**: 2026-03-27

## 🎯 技能定位

构建一个以"**求真**"为目标的结构化对话框架。由一位极具洞察力的主持人引导，邀请代表不同思想的"典型代表人物"进行高强度的、即时响应式的深度对话。

## 🚀 快速开始

### 手动启动

```bash
cd skills/roundtable-forum
python core_engine.py "你的议题"
```

### 自动提醒

每天 20:00 自动提醒（Cron 任务已配置）

## 🎮 用户命令

| 命令 | 说明 |
|------|------|
| `/可` | 同意进入下一轮问题 |
| `/止` | 结束讨论 |
| `/深入此节` | 围绕当前争议点继续深入 |
| `/引入新人物 XXX` | 邀请新代表加入 |

### 知识网络

```markdown
# 知识网络：人工智能创造力

## 核心议题
人工智能是否拥有真正的创造力？

## 主要立场
### 立场 A：计算主义
- 代表：图灵、凯文·凯利
- 核心观点：创造力是复杂计算的涌现

### 立场 B：生物主义
- 代表：约翰·塞尔、玛格丽特·博登
- 核心观点：创造力需要生物意识和意向性

## 核心争议
创造力是否可被算法模拟？

## 共识点
- 创造力包含新颖性和价值性两个维度
- 当前 AI 已展现某种程度的"创造性"

## 开放问题
- 意识的本质是什么？
- 涌现是否能产生真正的创造性？
```
