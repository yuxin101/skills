# 圆桌论坛技能 · SKILL.md

**技能名称**: roundtable-forum（圆桌论坛）  
**版本**: v1.0  
**创建日期**: 2026-03-27  
**定位**: 结构化深度对话框架，邀请不同思想代表进行高强度思想交锋

---

## 📋 技能概述

### 核心价值

构建一个以"**求真**"为目标的结构化对话框架。由一位极具洞察力的主持人引导，邀请代表不同思想的"典型代表人物"进行高强度的、即时响应式的深度对话。

### 核心特点

- **主持人引导**：理性之锚，冷静客观，驾驭思想交锋
- **代表人物**：每位代表不同思想流派/立场
- **ASCII 图表**：每轮总结生成视觉化思考框架
- **用户控制**：可/止/深入/引入新人物
- **知识网络**：最终生成结构化知识网络

### 使用场景

1. **复杂议题探讨** - 需要多角度分析的复杂问题
2. **决策辅助** - 需要权衡利弊的重要决策
3. **学习探索** - 深度学习某个领域
4. **思维训练** - 锻炼批判性思维

## 🛠️ 核心功能

### 功能 1: 议题启动

**输入**:
```
用户提供一个议题，如：
"人工智能是否拥有真正的创造力？"
```

**输出**:
```
【主持】：感谢您。本次圆桌对话正式开始。
【主持】：核心议题为「人工智能是否拥有真正的创造力？」。
【主持】：为穷尽其理，我已邀请以下几位代表人物：
- 艾伦·图灵 (INTJ) - 计算机科学之父
- 玛格丽特·博登 (ENFP) - 创造力研究专家
- 约翰·塞尔 (ISTJ) - 意识哲学代表
- 凯文·凯利 (ENTP) - 技术演化论者

【主持】：在我们深入探讨之前，为了确保讨论建立在共同的基础之上，
我想先请各位阐述：我们应当如何定义「创造力」？它的核心要素是什么？
```

---

### 功能 2: 多轮对话

**流程**:
```
1. 主持人提出问题
2. 各位代表依次发言
3. 主持人总结争议点
4. 生成 ASCII 思考框架
5. 提出更深层问题
6. 用户选择：可/止/深入/引入新人
```

**用户命令**:
| 命令 | 说明 |
|------|------|
| `/可` | 同意进入下一轮问题 |
| `/止` | 结束讨论 |
| `/深入此节` | 围绕当前争议点继续深入 |
| `/引入新人物 XXX` | 邀请新代表加入 |

### 功能 4: 知识网络生成

**讨论结束时生成**:
```markdown
# 知识网络：人工智能创造力

## 核心议题
人工智能是否拥有真正的创造力？

## 主要立场
### 立场 A：计算主义
- 代表：图灵、凯文·凯利
- 核心观点：创造力是复杂计算的涌现
- 论据：AlphaGo 创造性走棋、GPT 生成新颖内容

### 立场 B：生物主义
- 代表：约翰·塞尔、玛格丽特·博登
- 核心观点：创造力需要生物意识和意向性
- 论据：中文房间论证、现象学体验

## 核心争议
创造力是否可被算法模拟？

## 共识点
- 创造力包含新颖性和价值性两个维度
- 当前 AI 已展现某种程度的"创造性"
- 关于"真正"的定义仍需探讨

## 开放问题
- 意识的本质是什么？
- 涌现是否能产生真正的创造性？
```

## 📝 使用指南

### 启动讨论

```bash
# 命令行启动
python skills/roundtable-forum/main.py "你的议题"

# 或通过 Cron 自动提醒
# 每天 20:00 自动邀请
```

### 讨论过程

```
【主持】：问题：如何定义「创造力」？

【艾伦·图灵】【陈述】：我认为创造力可以定义为...
【玛格丽特·博登】【陈述】：我不同意，创造力应该...
【约翰·塞尔】【质询】：请问图灵先生，您如何解释...
【凯文·凯利】【补充】：我想补充一个视角...

【主持】：本轮核心争议点在于「创造力是否可被算法模拟」。

┌─────────────────────────────────────────┐
│          创造力本质争议                  │
├─────────────────────────────────────────┤
│   计算主义 ←→ 生物主义                  │
│   (图灵/凯利) (塞尔/博登)               │
└─────────────────────────────────────────┘

【主持】：基于以上框架，一个更深层的问题浮现了：
「如果创造力可被模拟，那模拟的创造力是否「真实」？」

(指令：可 / 止 / 深入此节 / 引入新人物)
```

### 结束讨论

```
用户：/止

【主持】：今天的对话已非常深入，暂告一段落。
我们从一个议题开始，通过多轮激烈的思想碰撞，
共同构建了一个关于此议题的思维网络。

# 知识网络已生成：output/knowledge-network-YYYY-MM-DD.md
```
