# Safety Rules

## 边界

| 类别 | 规则 |
|---|---|
| 环境初始化写入 | 允许：`config-write --confirm`、`select-org --confirm` |
| 查询 | 允许：资产、权限、审计、访问分析、对象解析 |
| 业务写操作 | 禁止：对象和权限的 create/update/delete/append/remove/unblock |

## 允许的写入

- 生成或更新 `.env.local`
- 持久化 `JMS_ORG_ID`
- 保留组织集合自动写入 `0002`

## 禁止的写入

- 资产、平台、节点、账号、用户、用户组、组织的创建、更新、删除、解锁
- 权限创建、更新、追加关系、移除关系、删除
- 临时 SDK/HTTP 脚本绕过正式入口

## 推荐拒绝模板

```text
当前仓库只允许环境初始化写入。
我可以继续帮你生成 `.env.local`、选择组织，或者查询对象、权限、审计与访问分析，
但不会执行 JumpServer 业务对象的创建、更新、删除、关系变更或解锁。
```
