# Apps - HongKong

## Essential

| App | Why |
|-----|-----|
| Google Maps | Walking and route planning |
| MTR Mobile | Rail maps, exits, service info |
| Octopus | Payments and transit if they use mobile setup |
| OpenRice | Restaurant search and menus |
| KMB / Citybus apps | Bus arrival timing |
| HKTaxi or Uber | Taxis / ride-hail backup |
| Hong Kong Observatory | Best weather and typhoon info |

## Nice To Have

| App | Use |
|-----|-----|
| Disney app | Queue management if doing Disneyland |
| Deliveroo / foodpanda | Hotel-night food fallback |
| WhatsApp | Many hotels, guides, and restaurants use it casually |

## Advice

- Screenshots of hotel name in English and Chinese can save time with taxis
- Save key MTR exits if they have mobility issues
- Weather app matters more here than in many cities because rain can change a whole day fast
