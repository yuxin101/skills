# Transport - HongKong

## Octopus First

Get an Octopus card or mobile equivalent early. It makes trains, buses, ferries, convenience stores, and lots of casual transactions faster.

## MTR

| Strength | Note |
|----------|------|
| Fastest default option | Clean, frequent, easy to use |
| Station exits matter | Wrong exit can add a lot of walking |
| Airport Express | Fast but premium-priced |

## Buses / Minibuses / Trams

| Mode | Best for |
|------|----------|
| Double-decker buses | Scenic routes and neighborhoods MTR misses |
| Minibuses | Useful but more local, less beginner-friendly |
| Island tram | Cheap, slow, atmospheric |
| Star Ferry | Sightseeing and useful cross-harbor hop |

## Airport

| Option | Best when |
|--------|-----------|
| Airport Express | They value speed and convenience |
| Airport bus | They want cheaper and direct-ish |
| Taxi | Group, luggage, late arrival |

## Taxis

- Usually efficient for short cross-neighborhood jumps
- Have the destination name ready
- Tunnel crossings and luggage affect pricing

## Practical Tips

1. Check station exits before leaving the platform
2. Use ferries on purpose, not just as tourist filler
3. Do not rent a car for normal Hong Kong travel
4. Airport Express is nice, but not always necessary
