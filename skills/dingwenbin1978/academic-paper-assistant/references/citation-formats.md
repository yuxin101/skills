# 学术论文参考文献格式参考

## GB/T 7714-2015（中国国家标准）

### 期刊文章
```
[序号] 作者1, 作者2, 作者3. 题目[J]. 期刊名, 年份, 卷(期): 页码.
```
**示例：**
```
[1] 张伟, 李娜. 人工智能在医学影像诊断中的应用研究[J]. 计算机科学, 2023, 45(3): 123-130.
```

### 会议论文
```
[序号] 作者1, 作者2. 题目[C]. 会议名称, 年份, 地点: 页码.
```

### 学位论文
```
[序号] 作者. 题目[D]. 学位授予单位, 年份.
```

### 专著
```
[序号] 作者. 题目[M]. 版本. 出版地: 出版社, 年份.
```

### 电子资源
```
[序号] 作者. 题目[EB/OL]. 网址, 更新日期/查阅日期.
```

## APA 格式（第7版）

### 期刊文章
```
Author, A. A., & Author, B. B. (Year). Title of article. Title of Periodical, volume(issue), page-page. https://doi.org/xxxxx
```

### 书籍
```
Author, A. A. (Year). Title of work: Capital letter also for subtitle. Publisher.
```

### 会议论文
```
Author, A. A. (Year, Month). Title of paper. Conference Name, Location. URL
```

---

## MLA 格式（第9版）

### 期刊文章
```
Author Last, First. "Article Title." Journal Name, vol. X, no. X, Year, pp. XX-XX.
```

### 书籍
```
Author Last, First. Title of Book. Publisher, Year.
```
