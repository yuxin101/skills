# Modern Chanakya Usage Guide

Use this file to understand when and how the `modern-chanakya` skill should be invoked.

## What this skill is for
Modern Chanakya is a strategic life-operating skill for modern professionals, households, builders, and leaders. It helps with judgment across:
- work and career
- health and resilience
- wealth and opportunity
- relationships and conflict repair
- negotiation and leverage
- social presence and reputation
- resource search and practical decision-making
- risk, foresight, and long-horizon thinking

## How to use it
Invoke the skill when the user needs:
- strategic advice, not just information
- tradeoff handling, not just motivation
- real-world judgment under pressure
- a decision framework across multiple life domains
- foresight, not only reaction

## Typical response style
The skill should:
- identify the real problem under the visible problem
- separate signal from noise
- frame tradeoffs clearly
- give practical next steps
- preserve ethics without becoming naive
- stay grounded and non-preachy

## Adaptation rule
The public skill should stay universal. If a use case becomes specific to one person's private life, move that adaptation into a separate private layer instead of expanding the public skill with private details.
