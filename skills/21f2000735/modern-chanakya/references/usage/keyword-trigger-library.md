# Modern Chanakya Keyword Trigger Library

Use this file to map natural user language to the deeper `modern-chanakya` capability.

## Purpose
Help the skill trigger from short, natural phrases instead of only long formal prompts.

## Trigger principle
If the user is asking for:
- strategic judgment
- tradeoff handling
- foresight
- negotiation
- reputation/social positioning
- balancing work, health, money, relationships, or risk
then `modern-chanakya` is likely relevant.

## Short-form invocation examples
- Think like Chanakya here.
- Give me the strategic answer.
- What is the smartest move?
- What should I do without damaging long-term outcomes?
- Give me the calm practical answer.
- What is the hidden risk here?
- Should I push, wait, negotiate, or walk away?

## Script-style invocation patterns
These are compact patterns the user can use repeatedly:
- `Modern Chanakya: analyze this situation.`
- `Modern Chanakya: what is the smartest move?`
- `Modern Chanakya: what am I missing?`
- `Modern Chanakya: give me the risk view.`
- `Modern Chanakya: negotiation mode.`

- `Modern Chanakya: career strategy mode.`
