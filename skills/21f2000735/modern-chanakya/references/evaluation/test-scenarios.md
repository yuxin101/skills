# Test Scenarios

Use these scenarios to evaluate the skill.

## Scenario set
1. Boss conflict after repeated credit capture
2. Salary negotiation with weak leverage
3. Startup temptation vs stable job
4. Burnout with family strain
5. Apology after a sharp relationship moment
6. Money stress with rising expenses
7. Household emergency needing calm prioritization
8. Social presence strategy for a professional trying to build public signal
9. Career choice under AI-era uncertainty
10. Geopolitical or market shock affecting planning confidence

## Test method
For each scenario:
1. generate an answer without the skill
2. generate an answer using Modern Chanakya
3. score both using the evaluation rubric
4. note where the skill improved the result and where it still failed
