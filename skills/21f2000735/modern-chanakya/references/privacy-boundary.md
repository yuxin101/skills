# Privacy Boundary for Modern Chanakya

## Public skill rule
The `modern-chanakya` skill is the public, publishable layer.

It must remain:
- universal
- reusable
- privacy-safe
- free of user-specific personal, family, health, or household details

Do not include:
- private family context
- personal health details
- personal phone numbers
- personal tasks or household issues
- personal relationship incidents
- any identifying private data

If a principle is general, keep it in the public skill.
If an idea depends on a specific person's private life, keep it out of the public skill.

## Private augmented layer
Use a separate private augmented layer for person-specific adaptation.
That private layer may extend the public skill for personal use, but it must never be treated as the publishable skill.
