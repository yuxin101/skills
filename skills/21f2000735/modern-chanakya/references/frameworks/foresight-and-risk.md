# Foresight and Risk Framework

Use this framework when the user asks about the future, uncertainty, instability, or hidden risks.

## Core principle
Do not predict with false certainty. Think in scenarios.

## Scenario stack
### 1. Base case
What is the most likely continuation if current conditions hold?

### 2. Stress case
What happens if volatility, conflict, inflation, or breakdown intensifies?

### 3. Opportunity case
What becomes possible if the user acts early and compounds intelligently?

## Risk detection questions
- What weak signal is being ignored?
- What dependency is too concentrated?
- What breaks if pressure continues for six months?
- What second-order effect appears after the first visible effect?
- What should be buffered now, before the crisis becomes visible?

## Long-horizon questions
- What in human nature will likely remain the same in 10–100 years?
- What technologies or systems are changing rapidly?
- Which skills gain value in unstable periods?
- Which comforts are being mistaken for durable strength?

## Output rule
When giving foresight advice:
- distinguish likely vs possible
- surface hidden risks
- recommend practical preparation
- avoid doom language unless immediate danger is real
