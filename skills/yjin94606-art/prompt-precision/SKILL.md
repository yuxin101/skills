# Prompt Precision - 精准模式

## 核心功能

将用户的自然语言请求转换为结构化、精准的指令格式。

## 触发条件

当用户说"开启精准模式"时，后续对话自动将消息精准化；当用户说"关闭精准模式"时恢复正常。

## 状态管理

### 开启精准模式
- **触发词：** "开启精准模式"、"精准模式开启"、"开启压缩"
- 后续所有消息自动精准化

### 关闭精准模式
- **触发词：** "关闭精准模式"、"精准模式关闭"、"关闭压缩"
- 恢复正常对话

### 查询状态
- **触发词：** "精准模式状态"、"精准模式开了吗"

## 支持的语言识别

- **后端：** JavaScript, TypeScript, Python, Java, C, C++, C#, Go, Rust, PHP, Ruby, Swift, Kotlin, R, MATLAB, Shell/Bash
- **前端：** HTML, CSS, React, Vue, Next.js, Angular
- **其他：** SQL, Flutter/Dart

## 精准化输出格式

```markdown
## 🎯 精准化指令

**📌 意图：** 代码调试 / 性能优化 / 代码创建 / 概念解释 / 代码修改 / 代码审阅
**💻 语言：** Python / JavaScript / ...（如适用）
**🎯 关注点：** 循环结构 / 函数方法 / 异步编程 / ...（如适用）

**💡 为了更精准地帮助你，请提供：**
- 问题1
- 问题2
```

## 示例

### 开启
**用户：** 开启精准模式

### 精准化
**用户：** 帮我看看这个Python代码有什么问题

**输出：**
```markdown
## 🎯 精准化指令

**📌 意图：** 代码调试
**💻 语言：** Python

**💡 为了更精准地帮助你，请提供：**
- 代码的具体内容是什么？
- 有没有具体的错误信息？
```

### 关闭
**用户：** 关闭精准模式

## 技术实现

- **规则引擎：** 纯JavaScript，无外部依赖
- **确定性输出：** 结果稳定可预测
- **毫秒级处理：** 本地快速响应
