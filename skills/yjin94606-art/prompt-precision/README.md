# 🎯 Prompt Precision - 精准模式

> **中文自然语言 → 结构化精准指令**
> 让AI精准理解你的意图

---

## 功能说明

**Prompt Precision** 将用户冗长的自然语言转换为结构化、精准的指令格式，帮助AI更准确地理解用户意图。

### 核心特性

- ✅ **中文优化** - 专为中文用户设计，精准识别中文意图
- ✅ **英文支持** - 同时支持英文关键词识别
- ✅ **多语言检测** - 自动识别20+种编程语言
- ✅ **智能引导** - 根据意图主动询问缺失信息
- ✅ **零依赖** - 纯JavaScript规则引擎，稳定快速

## 使用场景

### 开启精准模式
```
用户：开启精准模式
```

### 精准化示例

**输入：**
```
你好，我想请教一下，关于Python代码的for循环，为什么它运行这么慢啊，谢谢
```

**输出：**
```markdown
## 🎯 精准化指令

**📌 意图：** 性能优化
**💻 语言：** Python
**🎯 关注点：** 循环结构

**💡 为了更精准地帮助你，请提供：**
- 当前性能指标是什么（如响应时间、执行时间）？
- 目标性能指标是多少？
```

### 支持的意图类型

| 意图 | 说明 |
|------|------|
| 代码调试 | 找出并修复代码问题 |
| 性能优化 | 分析性能瓶颈并优化 |
| 代码创建 | 实现新功能 |
| 概念解释 | 解释概念和原理 |
| 代码修改 | 按要求修改代码 |
| 代码审阅 | 代码审查和建议 |

## 技术细节

### 规则引擎
- 纯JavaScript实现，无外部依赖
- 确定性输出，结果稳定可预测
- 毫秒级处理速度

### 算法
1. **废话去除** - 删除问候语、客套话
2. **意图识别** - 7种意图类型
3. **语言检测** - 23种编程语言
4. **关注点识别** - 代码结构/性能/安全等
5. **智能补全** - 自动生成目标描述
6. **引导询问** - 主动询问缺失信息
