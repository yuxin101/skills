# Clarity Engine
## 精准指令转换引擎

> **Transform verbose input into precise instructions / 将冗长输入转化为精准指令**

---

## Overview | 概述

**Clarity Engine** is a natural language processing tool that converts verbose, conversational input into structured, precise instructions. It helps AI assistants understand user intent more accurately.

**Clarity Engine** 是一款自然语言处理工具，将冗长、口语化的输入转化为结构化、精准的指令，帮助AI更准确地理解用户意图。

## Usage | 使用方法

### Activation | 激活
```
User: 开启精准模式 / Enable precision mode
```

### Example | 示例

**Input | 输入：**
```
你好，我想请教一下，关于Python代码的for循环，为什么它运行这么慢啊，谢谢
```

**Output | 输出：**
```markdown
## 🎯 Precision Instruction | 精准指令

**📌 Intent | 意图:** Performance Optimization | 性能优化
**💻 Language | 语言:** Python
**🎯 Focus | 关注点:** Loop Structure | 循环结构

**💡 To help you better | 为了更精准地帮助你:**
- What are the current performance metrics?
- What is the target performance?
```

### Deactivation | 关闭
```
User: 关闭精准模式 / Disable precision mode
```
