# Xiaohongshu (小红书)

## 概述

本技能提供小红书账号与内容管理的完整能力,包括笔记发布、内容编辑、数据统计、互动管理和账号配置。适用于内容创作者、运营人员或需要管理小红书账号的场景。

## 核心功能

### 1. 笔记发布
- **发布图文笔记**:上传 1-9 张图片,添加标题和正文
- **发布视频笔记**:上传视频文件,添加标题、正文和标签
- **批量发布**:从文件或模板批量创建笔记
- **发布草稿**:先保存为草稿,手动审核后发布

### 2. 笔记管理
- **编辑笔记**:修改已发布或草稿笔记的内容
- **删除笔记**:按 ID 批量清理笔记
- **笔记状态**:查看笔记是发布、草稿还是审核中
- **置顶/加精**:管理高质量内容的优先级

### 3. 数据与分析
- **个人笔记列表**:获取已发布、草稿、删除的笔记
- **单篇阅读数据**:浏览量、点赞量、收藏量、评论量
- **粉丝增长**:查看粉丝增长趋势
- **互动详情**:评论、点赞、收藏的详细数据

### 4. 互动管理
- **评论管理**:查看、回复、删除评论
- **点赞管理**:查看评论用户的点赞情况
- **私信管理**:查看和回复私信
- **黑名单管理**:管理禁言用户列表

### 5. 账号配置
- **个人资料**:修改昵称、头像、简介、背景图
- **账号信息**:获取粉丝数、获赞数等基础统计
- **标签管理**:添加/删除个人标签(如"时尚博主"、"美食达人"等)
- **关联账号**:绑定其他平台的账号(微信、微博等)

## 使用场景

### 何时使用此技能?

1. **发布内容**:当需要发布小红书笔记时
   - 单个笔记发布
   - 批量发布系列内容
   - 发布带标签和分类的笔记

2. **内容管理**:当需要管理已发布内容时
   - 检查某篇笔记的阅读/互动数据
   - 编辑笔记中的文字或图片
   - 清理过时或低质量内容

3. **数据分析**:当需要查看运营效果时
   - 查看单篇或全部笔记的数据
   - 获取粉丝增长趋势
   - 分析用户互动偏好

4. **互动维护**:当需要回复用户时
   - 批量回复评论
   - 处理私信消息
   - 管理黑名单用户

5. **账号准备**:当需要配置账号时
   - 更新头像和简介
   - 添加个人标签
   - 关联其他平台

## 快捷参考

### 发布图片笔记
```
发布包含图片的笔记,包含标题、正文和标签。
- 图片:1-9 张(JPG/PNG,每张图片最大 10MB)
- 标题:最多 20 个字符
- 正文:最多 2000 个字符
- 分类:可选(生活、时尚、美食等)
```

### 获取笔记数据
```
获取指定笔记的阅读、点赞、收藏数据。
参数:笔记 ID(必需)
输出:阅读次数、点赞数、收藏数、评论数
```

### 管理评论
```
查看、回复或删除笔记的评论。
- 查看:列出最近 10 条评论
- 回复:指定评论 ID 进行回复
- 删除:按评论 ID 批量清理
```
