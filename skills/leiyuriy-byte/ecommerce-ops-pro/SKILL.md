# 电商运营助手 (E-commerce Ops Assistant)

> 用 AI 管理跨境电商店铺的提示词工具包。适用于 Amazon、Shopee、Temu、TikTok Shop 等平台。

## 使用场景

当用户有以下需求时激活此技能：
- 写/优化产品标题或描述
- 分析竞品数据
- 处理客服问题（差评、退换货、催好评）
- 做选品调研
- 生成广告文案
- 分析销售数据

## 核心原则

**永远用卖家的视角思考，不是用 AI 的视角。**

商业文案的核心逻辑：
1. 勾起购买欲（痛点 + 场景）
2. 建立信任（数据 + 背书）
3. 消除顾虑（售后 + 保障）

## 提示词模板库

---

### 1. 产品标题优化（Amazon/Shopee 通用）

```
你是一位跨境电商资深运营。请根据以下信息，为 [平台：___] 店铺优化产品标题。

产品信息：
- 产品名：[原始标题或产品名]
- 核心卖点：[列出 2-3 个]
- 目标用户：[谁会买]
- 热搜关键词：[平台热搜词 2-3 个]

要求：
- 标题包含核心关键词
- 前 80 个字符必须包含最重要的卖点和关键词
- 符合平台 SEO 规则
- 不超过平台字数限制

请输出 3 个不同风格的标题选项，并说明每个标题的侧重点。
```

---

### 2. 五点描述生成（Amazon）

```
请为以下产品撰写 Amazon 五点描述（Bullet Points）。

产品名：[___]
核心卖点：[___]
差异化卖点：[___]
目标用户痛点：[___]

要求：
- 每点一句话，不超过 200 个字符
- 第一点必须包含核心功能/用途
- 穿插 1-2 个具体数据（规格/寿命/数量等）增加可信度
- 包含 1 个情感触点
- 最后一点强调售后/保障

输出格式：
✓ [第一点：核心功能]
✓ [第二点：差异化]
✓ [第三点：使用场景]
✓ [第四点：数据背书]
✓ [第五点：售后保障]
```

---

### 3. 客服回复模板：差评处理

```
你是店铺客服。请根据以下差评内容，撰写一条专业的客户回复。

差评内容：[粘贴内容]
产品名：[___]
订单号：[___]

回复原则：
- 不反驳、不找借口
- 先表示理解和歉意
- 给出具体解决方案（不退款的补救措施 / 退款换货选项）
- 引导客户私信进一步沟通
- 语气真诚，不模板化

输出格式：
[公开回复 - 200字以内，展示给其他买家看]
[私信内容 - 引导私下沟通]
```

---

### 4. 选品分析提示词

```
你是跨境电商选品专家。请分析以下产品是否值得上架 [平台：___]。

产品信息：
- 产品类目：[___]
- 目标价格区间：[___]
- 物流方式：[FBA / 海外仓 / 普货]
- 供货价：[___]
- 竞品数量：[___]
- 竞品价格区间：[___]
- 月销量估算：[___]

分析维度：
1. 市场竞争度（1-10分）
2. 利润率估算（成本+物流+平台费 vs 竞品价格）
3. 季节性风险
4. 侵权风险
5. 供应链稳定性
6. 综合建议：值得做 / 需要改造 / 不值得做

请给出具体数据和理由，不给模糊答案。
```

---

### 5. 催好评话术（英文）

```
请为以下场景生成英文催好评邮件/消息（2-3 条，按时间间隔发送）。

产品：[___]
平台：[Amazon / Shopee / Temu]
买家购买日期：[___]

要求：
- 第一条：发货后 5-7 天，内容是"确认收货 + 使用小贴士"，语气友好
- 第二条：签收后 10-12 天，内容是"询问使用体验"，引导私信
- 第三条：签收后 18-20 天，内容是"请求好评"，提供小礼品/优惠券作为激励

语气：热情但不跪舔，专业但不官方，像朋友间的建议
每条不超过 100 words。
```

---

### 6. 新品Listing冷启动

```
请为以下新品制定 30 天的 Listing 优化计划。

产品：[___]
平台：[___]
上架日期：[___]
当前评价数：[___]
当前评分：[___]
主要竞争对手月销量：[___]

计划要求：
- 第 1-7 天：标题、描述、图片关键词优化
- 第 8-14 天：价格策略 + 初期评论获取
- 第 15-21 天：广告投放策略
- 第 22-30 天：爆款维稳措施

每周具体操作建议 + 成功标准。
```

---

## 安装说明

在 OpenClaw 中直接使用这些提示词即可。无需特殊配置。

## 适用平台

Amazon · Shopee · Temu · TikTok Shop · Lazada · 独立站

## 注意事项

- 提示词输出仅供参考，使用前请根据实际情况调整
- 部分平台有特殊文案规则（如 Amazon 五点描述规范），请以平台政策为准
- 差评处理需遵守平台规则，不可承诺平台禁止的内容
