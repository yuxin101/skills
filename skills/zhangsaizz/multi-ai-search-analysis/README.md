## 输出内容

### 标准报告结构

1. **核心概览** - 事件时间线、关键数据汇总
2. **分维度分析** - 按主题划分（政治/经济/军事/外交等）
3. **影响预测** - 基准情景/风险情景/极端情景
4. **数据对比表** - 各家 AI 关键数据横向对比
5. **各方观点** - 机构/专家观点汇总
6. **关键观察点** - 短期/中期风险、缓和路径
7. **AI 特色对比** - 各家分析优势说明

## 使用流程

### 串行模式（v1.0）

```
用户提问
   ↓
依次询问 4 家 AI（DeepSeek → Qwen → 豆包 → Kimi）
   ↓
等待所有回复（含登录验证）
   ↓
记录关键数据和观点
   ↓
综合汇总生成报告
   ↓
输出精炼对比分析

总耗时：12-18 分钟
```

### 并行模式（v1.1+ 推荐）

```
用户提问
   ↓
同时打开 4 个标签页
   ↓
同时输入问题并发送
   ↓
并行等待响应（10-20 秒）
   ↓
依次获取回复内容
   ↓
综合汇总生成报告
   ↓
输出精炼对比分析

总耗时：3-5 分钟（节省 60-75% 时间）
```

## 最佳实践

### ✅ 推荐

- 统一问题表述，确保可比性
- 等待完整回复再继续
- 交叉验证关键数据
- 标注时间戳便于追踪

### ❌ 避免

- 中途修改问题
- 盲目相信单一数据点
- 忽略 AI 免责声明
- 用于紧急决策

## 🚀 快速开始

### 方式 1：一键安装（推荐）

```bash
cd skills/multi-ai-search-analysis
python scripts/install.py
```

### 方式 2：手动安装

```bash
# 1. 安装 Python 依赖
pip install -r requirements.txt

# 2. 安装浏览器
playwright install msedge

# 3. 测试安装
python scripts/test.py
```

### 首次登录

```bash
python scripts/login.py
```

手动完成各平台登录后关闭浏览器。

### 运行分析

```bash
# 并行模式（推荐）
python scripts/run.py -t "你的分析主题"

# 示例：时事分析
python scripts/run.py -t "伊朗局势分析" -d 政治 经济 军事 外交

# 示例：技术对比
python scripts/run.py -t "Python vs Java" -d 性能 易用性 生态 就业

# 示例：产品评测
python scripts/run.py -t "iPhone 16 Pro" -d 性能 拍照 续航 价格
```
