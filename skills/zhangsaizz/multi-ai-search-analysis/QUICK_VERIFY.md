## 🐛 如果验证失败

### 问题 1：某平台提示"未登录"但实际已登录

**可能原因**：
- 登录状态未持久化
- 浏览器配置文件损坏

**解决方案**：
```bash
# 1. 清除浏览器配置，重新登录
rm -rf browser-profile/*

# 2. 重新运行登录工具
python scripts/login.py

# 3. 再次验证
python scripts/test-stability.py
```

### 问题 2：输入框查找失败

**可能原因**：
- 平台 UI 变更
- 选择器不匹配

**解决方案**：
```bash
# 1. 查看调试输出，确认尝试的选择器
# 2. 手动打开平台，检查输入框的实际选择器
# 3. 更新 config/ai-platforms.json 中的选择器配置
```

**示例**（更新 Qwen 选择器）：
```json
{
  "name": "Qwen",
  "selectors": {
    "input": "#chat-input, textarea#chat-input, textarea[aria-label*='Prompt'], ..."
  }
}
```

### 问题 3：响应提取失败

**可能原因**：
- 响应内容选择器不匹配
- 内容提取方式不合适

**解决方案**：
```bash
# 1. 查看调试输出，确认使用的选择器
# 2. 手动检查平台响应内容的实际 HTML 结构
# 3. 更新 config/ai-platforms.json 中的 response 选择器
```
