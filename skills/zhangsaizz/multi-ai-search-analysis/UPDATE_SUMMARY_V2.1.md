### v2.1.0：数据提取与质量评分

#### 新增功能

1. **数据自动提取** (`extractor.py`)
   - ✅ 百分比数据提取（15%、3.2%、0.5% 等）
   - ✅ 日期提取（2026-03-15、2026 年底等）
   - ✅ 预测性陈述提取（"预计上涨 15%"）
   - ✅ 关键洞察提取（包含关键词的句子）
   - ✅ 数据点提取（带上下文的数字）

2. **质量评分系统**
   - ✅ 5 维度评分体系：
     - 字数（20% 权重）
     - 数据点（30% 权重）
     - 结构清晰度（20% 权重）
     - 引用来源（15% 权重）
     - 具体性（15% 权重）
   - ✅ 等级划分：优秀（≥85）、良好（≥70）、一般（≥55）、较差（<55）
   - ✅ 详细评分 breakdown

3. **对比表生成**
   - ✅ 百分比数据对比表
   - ✅ 质量评分对比表
   - ✅ 自动插入报告

4. **报告增强**
   - ✅ 在 `run.py` 中自动调用数据提取
   - ✅ 在 `reporter.py` 中支持传入对比表
   - ✅ 自动在报告中插入质量评分汇总

## 🎯 核心功能演示

### 1. 数据提取

```python
from extractor import DataExtractor

extractor = DataExtractor()
text = "2026 年全球油价预计将上涨 15%，达到每桶 95 美元"

data = extractor.extract_all(text)
# 提取结果：
# - percentages: [15.0%]
# - data_points: [15%, 95 美元]
# - key_insights: [...]
```

## 💡 使用建议

### 最佳实践

1. **明确数据需求**
   ```bash
   # 在问题中指定需要数据
   python run.py -t "油价分析" \
     -d "价格预测" "供需数据" "地缘风险"
   ```

2. **查看质量评分**
   - 优先参考"优秀"和"良好"等级的回复
   - 关注各维度得分，了解优劣势

3. **交叉验证数据**
   - 对比多家 AI 的数据点
   - 共识值通常更可靠
   - 差异大的数据需进一步验证

### 注意事项

1. **评分非绝对** - 质量评分是参考，不是绝对标准
2. **语言限制** - 目前主要针对中文优化
3. **上下文理解** - 基于正则，复杂语义可能遗漏
4. **人工审核** - 关键决策仍需人工判断
