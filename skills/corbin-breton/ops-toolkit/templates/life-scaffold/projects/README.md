# Projects

Time-bound deliverables. Each project has an outcome, deadline, and team.

## Structure

Each project folder contains:
- `project.md` — Project charter (goal, timeline, success criteria)
- `status.md` — Current status and blockers
- `items.json` — Atomic facts about this project (decisions, learnings, metrics)

## Examples

- `product-launch/` — Shipping a new feature
- `marketing-campaign/` — Launch a campaign
- `research-sprint/` — Investigate a new technology

Delete or modify based on your actual projects.
