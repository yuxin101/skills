## 🚀 使用方式

### CLI 命令行

```bash
# 搜索职场技巧，生成 10 条干货
python ~/.openclaw/skills/local-life-hotspot/run.py -k "职场技巧" -n 10

# 搜索学习方法，生成 15 条干货，自动发布
python ~/.openclaw/skills/local-life-hotspot/run.py -k "学习方法" -n 15 -a

# 搜索科技知识，生成 8 条干货
python ~/.openclaw/skills/local-life-hotspot/run.py -k "科技知识" -n 8

# 使用默认参数（职场干货 10 条）
python ~/.openclaw/skills/local-life-hotspot/run.py
```

### 参数说明

| 参数 | 简写 | 说明 | 默认值 |
|------|------|------|--------|
| `--keyword` | `-k` | 搜索关键词 | `职场干货` |
| `--top-n` | `-n` | 干货技巧数量 | `10` |
| `--auto-publish` | `-a` | 自动发布到小红书 | `false` |
| `--content` | `-c` | 直接提供干货内容 | `''` |

## 🔍 工作原理

### 1. 搜索干货内容
```
搜索词："职场技巧 方法 经验"
搜索词："职场实用干货 2026"
搜索词："职场最佳实践 指南"
```

### 2. 读取网页内容
使用 Jina Reader 读取搜索结果网页，提取正文内容。

### 3. 提取实用技巧
识别并提取以下格式的内容：
- 数字列表：`1. xxx` `2. xxx`
- 符号列表：`• xxx` `- xxx`
- 关键词：`技巧` `方法` `建议` `首先` `其次`

### 4. 去重和筛选
- 去除重复内容
- 限制每条技巧长度（<200 字）
- 保留最相关的 Top N 条

### 5. 创作文案
- 生成吸引眼球的标题
- 结构化正文（含表情符号）
- 添加互动引导和标签

### 6. 发布到小红书
- 自动生成封面图
- 调用小红书 MCP 发布

## 🎯 内容质量标准

### ✅ 好的干货内容
- 具体可执行（不是空话）
- 有实际指导意义
- 经过实践验证
- 语言简洁明了

### ❌ 避免的内容
- 空洞的口号（如"努力工作"）
- 模糊的建议（如"提升自己"）
- 热点摘要（如"行业迎来变革"）
- 无实际价值的信息
