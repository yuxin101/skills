# 快速开始

## 使用方法

直接给 openclaw 发消息：

```
把这篇文章转成小红书信息图：

[粘贴文章内容]
```

或者：

```
帮我把这段笔记做成信息图 HTML：

[粘贴笔记内容]
```

## 输出

生成一个自包含的 HTML 文件，保存在当前对话的工作目录下。

- 文件名格式：`infographic-{主题简称}.html`
- 所有样式内嵌，无外部依赖
- 浏览器打开即可预览

## 截图方法

### 方法 1：浏览器截图

1. 用浏览器打开生成的 HTML 文件
2. 使用浏览器开发者工具调整视口为 1080×1440
3. 截图保存

### 方法 2：命令行截图（推荐）

如果配置了 Chrome DevTools MCP：

```
帮我截图这个 HTML
```

openclaw 会自动调用 Chrome 打开并截图。

## 自定义

生成后如果需要微调：

- **改文字**: 直接编辑 HTML 中的文本
- **改颜色**: 搜索色值替换（见设计规范）
- **改布局**: 调整 CSS grid 或 flex 属性
- **改尺寸**: 修改 `.container` 的 width/height
