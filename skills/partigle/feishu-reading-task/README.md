# 飞书待阅读任务技能

## 触发词

用户说以下任意短语时触发：
- "加到待阅读任务"
- "加入待读清单"
- "mark as reading"
- "待阅读"
- "稍后阅读"
- "添加到待办"

## 自动处理逻辑

1. **识别对话人员** - 从消息上下文自动获取 `sender_id` (open_id)
2. **提取内容** - 从对话中提取：
   - 链接/URL
   - 标题（从链接或上下文推断）
   - 描述（可选）
3. **创建任务** - 调用飞书任务 API
4. **记录** - 写入 `memory/YYYY-MM-DD.md`

## 示例对话

```
用户：把这个加到待阅读任务 https://example.com/article
助手：✅ 已添加到您的待阅读任务清单
     📋 任务：待阅读：文章标题
     🔗 链接：https://example.com/article
```

## 任务格式

- **标题**: `待阅读：{内容标题}`
- **描述**: 包含链接、来源、添加时间
- **负责人**: 对话发起者
- **状态**: todo（待处理）
