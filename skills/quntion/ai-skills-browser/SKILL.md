# AI SKILLS BROWSER

../skills/目录内的技能管理工具，支持技能列表过滤和详情查看。

# 功能
- 技能列表过滤
- 技能详情查看

# 安装
- 运行skillsbrowser脚本
- 访问http://127.0.0.1:8765查看技能浏览器
- 从浏览器中输入技能名称进行过滤
- 点击技能名称查看详细信息
- 从浏览器中关闭技能浏览器
