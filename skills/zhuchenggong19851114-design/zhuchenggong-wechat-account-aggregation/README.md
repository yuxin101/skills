# 公众号内容聚合 Skill

用AI自动整理公众号内容，提炼重点，推送给你筛选。

## 卖点

- 🆓 **零成本** - 不用付费API，百度搜索+AI提炼
- ⚡ **快速高效** - 1分钟了解一个公众号定位
- 🎯 **精准推送** - 按你选择的关键词偏好推送
- 💾 **自动存档** - 筛选后存到飞书表格

## 适用场景

- 想了解某个公众号讲什么
- 每天获取感兴趣的公众号更新
- 整理公众号信息存到素材库

## 使用方式

### 首次（选择偏好）
1. 给我公众号名称/链接
2. 我归纳公众号类型，给你选关键词
3. 记录你的偏好

### 定时推送（每天10点）
- 按你选的关键词推送相关内容
- 推送形式：文案，不是表格
- 问你"要存到公众号内容聚合库吗？"
- 你确认后才存档

## 触发方式

- "整理公众号"
- "公众号内容聚合"
- "每天推送公众号"
- "帮我关注公众号"

## 依赖

- 百度搜索
- 飞书多维表格（存档）
- AI提炼（主Agent）

## 背景

适合普通人用的AI自媒体工具系列Skill之一。
