# GitHub可信来源验证

验证GitHub上中国开发者身份是否可信。

## 特点

- 🆓 **零成本** - 不用付费工具，浏览器直接搜
- 🔒 **安全可靠** - 多平台验证，来源可追溯
- 👶 **小白友好** - 自动化搜素，一键验证

## 概述

验证GitHub上中国开发者的真实身份，通过搜索其在B站、抖音、小红书、知乎等平台的存在来确认可信度。

## 适用场景

- "验证这个作者可信度"
- "查一下这个开发者有没有其他平台账号"
- "看看这个项目作者靠不靠谱"

## 验证平台

- B站（视频教程）
- 抖音（短视频）
- 小红书（图文）
- 知乎（问答）
- 掘金（文章）
- CSDN（博客）

## 输出格式

| 作者 | GitHub项目 | 验证平台 | 可信度 |
|------|------------|----------|--------|
| @JoeanAmier | TikTokDownloader | B站 794粉/19视频 | ✅ |
| @xszyou | Fay数字人 | B站 3246粉/77视频 | ✅ |

## 验证标准

- 各平台数据（中上水平）= 可信
- B站：粉丝500+、有视频教程
- 抖音/小红书：有发过相关内容

## 工作流程

1. 获取GitHub作者名
2. 在各平台搜索该作者
3. 统计粉丝/视频/文章数量
4. 给出可信度判断

## 使用示例

```
用户：帮我验证一下这个GitHub项目@xszyou/Fay靠不靠谱

助手：
正在验证@xszyou...
- B站：space.bilibili.com/2111554564 粉丝3246/视频77
- 结论：✅ 可信 - 是真实开发者，有教程视频
```

## 前置要求

- 浏览器（用于搜索各平台）
- 网络访问
