# xiaoxiyouxi-5min-report

## Description
Automated 5-minute WeChat mini games industry dynamics report generator. Creates concise, insight-driven reports focused on revenue performance with comprehensive market analysis.

## Usage Scenarios
- Daily/weekly WeChat mini games industry intelligence
- Investment opportunity identification in casual gaming
- Market trend analysis for game developers and publishers
- Competitive intelligence for mobile gaming sector

## Report Template Structure
### 1. Core Data Metrics (Revenue Focus Only)
- **Selling Chart TOP 5**: Revenue performance ranking with game categories
- *Note: Hot Chart and New Release Chart are excluded from data display but used for insights*

### 2. Business Model Innovation
- Monetization strategy analysis
- Multi-tag fusion gameplay innovation
- Casual game depth trend analysis

### 3. User Behavior Insights  
- Category preference stability and changes
- Social attribute importance trends
- Innovation acceptance patterns

### 4. Investment Opportunity Alerts
- High-potential tracks with specific examples
- Risk warnings with actionable recommendations

## Workflow
1. **Data Source**: Navigate to https://sj.qq.com/wechat-game
2. **Extract Rankings**: Parse Hot, Revenue, and New Release leaderboards
3. **Structure Report**: Apply standardized template focusing on revenue data + comprehensive insights
4. **Deliver**: Create Feishu document with proper formatting and send via direct message

## Key Features
- Real-time scraping from official Tencent application store
- Revenue-focused data presentation with comprehensive market insights
- Actionable investment recommendations based on verified data
- Automated Feishu delivery with professional formatting

## Dependencies
- Browser automation capability
- Feishu document creation and messaging permissions
- Markdown formatting support

## Output Standards
- Data source attribution with timestamp
- Specific game examples for all insights
- Clear distinction between data facts and analytical insights
- Actionable recommendations with risk assessments

## Best Practices
- Run during business hours for most accurate real-time data
- Always verify data source availability before execution
- Maintain consistent formatting for historical comparison
- Focus insights on actionable business intelligence
