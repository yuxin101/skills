# Source Credibility Rubric

## Acceptable Source Tiers
1. Primary: official company pages, filings, conference presentations, reputable publications with direct quotes.
2. Secondary: recognized industry analysis with transparent sourcing.
3. Tertiary: aggregators only for discovery, never as sole evidence.

## Minimum Evidence Standard
- Material claim should be backed by at least one primary or two independent secondary sources.
- Time-sensitive claims should prefer sources published within 12 months unless historical context is required.

## Rejection Criteria
- Anonymous rumor without corroboration.
- AI-generated summaries with no source trace.
- Outdated profiles contradicting newer primary sources.

## Citation Format
Use compact inline citations:
- `[Source Name, YYYY-MM-DD]`
- Include link in final source appendix when available.

## Confidence Labels
- High: directly supported by primary sources.
- Medium: supported by multiple reputable secondary sources.
- Low: partial support or stale evidence; call out uncertainty.
