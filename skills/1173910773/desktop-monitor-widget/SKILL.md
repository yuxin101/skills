# Desktop Monitor Widget

一个优雅的桌面监控悬浮球，实时显示系统资源状态。

## 功能特性

- 🎯 **悬浮球设计** - 透明悬浮窗，不干扰其他操作
- 📊 **实时监控** - CPU、内存、磁盘、温度
- 🎨 **贴边自动缩小** - 靠近屏幕边缘自动折叠
- 🖱️ **可拖动位置** - 随意放置在屏幕任意位置
- 🔄 **自动更新** - 每 2 秒刷新一次数据
- 👆 **双击切换** - 展开/折叠视图

## 使用方法

直接询问 Agent "系统状态"、"资源使用情况" 或 "打开监控悬浮球"。

本技能提供两种模式：
1. Web 浏览器模式（推荐，无需额外依赖）
2. Tkinter 窗口模式（需要 tkinter）
