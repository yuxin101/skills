# Facilitation Guide — End-to-End Brainstorming

**Level:** Intermediate  
**Time:** 15-20 Min Lesen  
**Use:** Als Facilitator Session moderieren

---

## 1. Pre-Session Checklist

### 1.1 Raum & Setup (vorher)

```
□ Stühle im Kreis (kein Konferenztisch)
□ Whiteboard/Flipchart bereit
□ Timer sichtbar (z.B. Time Timer App)
□ Namensschilder
□ Marker (4 Farben)
□ Klebepunkte (falls Dot-Voting)
□ Getränke + Snacks
```

### 1.2 Remote Setup

```
□ Video-Call Link verschickt
□ Miro/Mural Board erstellt + Link geteilt
□ Kamera + Mikrofon getestet
□ Breakout-Room vorbereitet (falls >6 Personen)
□ Timer-App geteilt (z.B. TweetWall, Clockify)
```

### 1.3 Agenda verschickt

```
□ Ziele klar kommuniziert
□ Zeitrahmen (60/90/120 Min)
□ Erwartungen (Vorbereitung?
□ Rolle: Facilitator vs. Participant
```

## 2. Session Phasen

### Phase 1: Start (5-10 Min)

**Ziel:** Energie aufbauen, Ziel klären, Regeln etablieren

**Ablauf:**

```
00:00 - Ankunft, Smalltalk (2 Min)
00:02 - Willkommen + Agenda-Check (1 Min)
00:03 - Ziel vortragen: "Heute brainstormen wir [Topic]"
00:04 - Rollen: Facilitator = Zeit + Struktur, Participants = Ideen
00:05 - Regeln etablieren (1 Min):

  REGELN (sag laut):
  ✅ Quantität vor Qualität
  ✅ Keine Kritik in Divergenter Phase
  ✅ Build on Ideas (nicht "aber...")
  ✅ Alle Stimmen zählen
  ✅ Wild Ideas welcome
  ✅ Active Listening

00:06 - Warm-Up (optional, bei neuen Teams):

  "In 2 Minuten: Was ist das nervigste an Meetings?"
  → Jeder schreibt 3 Antworten auf
  → Kurze Runde: Top-Auswahl teilen
  → Übung: Es geht um Ideen-Generierung, nicht Bewertung
```

**Facilitation-Tipps:**
- Pünktlich starten (auch wenn 50% noch nicht da)
- Telefon auf lautlos
- Keine Laptops/Tablets (nur für Miro/Notizen)

---

### Phase 2: Divergente Phase (20-40 Min)

**Ziel:** Möglichst viele Ideen generieren

**Techniken (wähle nach Situation):**

| Technik | Dauer | Team-Größe | Wann |
|---------|-------|------------|------|
| SCAMPER | 30-40 Min | 3-8 | Bestehendes verbessern |
| 6-3-5 | 30-40 Min | 3-6 | Anonym brainstormen |
| Mind Mapping | 20-30 Min | 1-6 | Visuelles Explorieren |
| Brainwriting Pool | 25-35 Min | 4-10 | Dominante Teilnehmer |
| Role Storming | 40-50 Min | 5-12 | Perspektivwechsel |

**Typischer Ablauf SCAMPER:**

```
10:00 - Technik erklären (2 Min)
       "Wir nutzen SCAMPER für 7 Kategorien"
       
10:02 - Buchstabe S (Substitute) (4 Min)
       "Was kann ersetzt werden?"
       → Jeder schreibt 3-5 Ideen
       → Facilitator notiert auf Board
       
10:06 - Buchstabe C (Combine) (4 Min)
10:10 - Buchstabe A (Adapt) (4 Min)
10:14 - Buchstabe M (Modify) (4 Min)
10:18 - Buchstabe P (Put to other uses) (4 Min)
10:22 - Buchstabe E (Eliminate) (4 Min)
10:26 - Buchstabe R (Reverse) (4 Min)

10:30 - Pause (5 Min)
```

**Regeln während Divergenter Phase:**

```
🚫 KEINE Kritik (auch nicht non-verbal)
🚫 KEINE "Ja, aber..." Sätze
🚫 KEINE Bewertung
✅ JEDE Idee notieren
✅ Build on others' ideas
✅ Wild ideas explizit erwünscht
```

**Facilitation-Tipps:**
- Alle Ideen laut vorlesen (auch die "dummen")
- Bei Kritik: "Note captured! We'll evaluate later."
- Bei Dominanter Person: "Great idea! Let's hear from others first."
- Bei Schweigen: "What would [famous person] hier vorschlagen?"

---

### Phase 3: Pause (5-10 Min)

```
Zwangspause nach 40 Min (kognitive Ermüdung)
Getränke, Toilette, frische Luft
Keine Diskussionen über Ideen
```

---

### Phase 4: Konvergente Phase (15-30 Min)

**Ziel:** Von vielen Ideen zu Top 3-5

**Techniken (wähle nach Stakeholder):**

| Technik | Dauer | Wann |
|---------|-------|------|
| Dot-Voting | 10-15 Min | Schnelle Entscheidung |
| Impact/Effort Matrix | 20-30 Min | Portfolio-Entscheidung |
| MoSCoW | 15-25 Min | Scope-Verhandlung |
| RICE | 20-35 Min | Data-driven Priorisierung |

**Typischer Ablauf Dot-Voting:**

```
40:00 - Technik erklären (2 Min)
       "Jeder hat 3-5 Klebepunkte"
       "Vote on what excites you most"
       
40:02 - Voting (5 Min)
       - Stille (keine Diskussion)
       - Punkte verteilen
       - Max 2 Punkte auf eine Idee
       
40:07 - Top 5 zählen (2 Min)
       → Facilitator sortiert nach Votes
       
40:09 - Top 3 Diskussion (10 Min)
       - Warum diese drei?
       - Any surprises?
       - Consensus?
       
40:19 - Next Steps (5 Min)
       → Wer macht was bis wann?
       → Nächste Session?
```

**Konvergenz-Regeln:**

```
✅ Fokus auf Top 3-5 (nicht 20)
✅ Konkrete Next Steps mit Ownern
✅ Scope explizit: Was ist NICHT dabei
✅ Timebox: 30 Min MAX für Konvergenz
```

---

### Phase 5: Wrap-Up (5 Min)

**Ablauf:**

```
55:00 - Zusammenfassung (2 Min)
       "Wir haben [X] Ideen, Top 3 sind..."
       
55:02 - Action Items (2 Min)
       → Wer macht was bis wann?
       → Facilitator dokumentiert
       
55:04 - Follow-Up (1 Min)
       "Ergebnis kommt per Email bis [Datum]"
       "Nächste Session: [Datum] [Topic]"
       
55:05 - Dankeschön + Closure
       "Danke für eure Ideen!"
```

[Liste falls vorhanden]

### 3.3 Follow-Up Cadence

```
48h nach Session:
□ Action Items gestartet?
□ Email verschickt?
□ Stakeholder informiert?

1 Woche nach Session:
□ Action Items im Sprint/OKRs?
□ Blockaden identifiziert?
□ Erste Ergebnisse messbar?

4 Wochen nach Session:
□ ROI messbar?
□ Entscheidung getroffen?
□ Projekt gestartet/verworfen?
```

## 4. Troubleshooting

### Problem: Dominante Teilnehmer

**Symptom:** Eine Person redet 80% der Zeit

**Lösungen:**
1. **Pre-Session:** Individuell ansprechen, um Ideen bitten
2. **During:** "Great point! Let's hear from others."
3. **Anonymous:** Brainwriting Pool (alle schreiben, nicht reden)
4. **Round-Robin:** Jeder kommt dran, 2 Min pro Person

### Problem: Stille Teilnehmer

**Symptom:** Nur 2-3 Personen generieren Ideen

**Lösungen:**
1. **Warm-Up:** Erst Warm-Up-Exercise (2 Min)
2. **Brainwriting:** Jeder schreibt erst, dann teilen
3. **Subgroups:** 2er-Gruppen für 5 Min, dann zusammen
4. **Parking Lot:** "Was dich gerade blockiert?" → aufschreiben

### Problem: Ideen-Kritik in Divergenter Phase

**Symptom:** Teilnehmer bewerten Ideen direkt ("Das geht nicht...")

**Lösungen:**
1. **Reminder:** "Wir sind in Divergenter Phase - noch keine Bewertung!"
2. **Yellow Card:** Visuelles Signal (gelbe Karte bei Kritik)
3. **Idea Jail:** Kritische Ideen -> "Idea Jail" Zone, später Review
4. **Reframe:** "Was ist gut daran?" statt "Das geht nicht"

### Problem: Scope Creep

**Symptom:** Thema wird immer größer, keine Fokussierung

**Lösungen:**
1. **Pre-Defined Scope:** "Heute NUR [Topic], nicht [Related-Topic]"
2. **Parking Lot:** Off-Topic -> "Parking Lot" für später
3. **Time-Box:** Harte Zeitgrenze pro Phase
4. **Scope-Dokument:** Zu Beginn schriftlich fixieren

### Problem: Remote-Teilnehmer disengaged

**Symptom:** Kamera aus, kein Input

**Lösungen:**
1. **Camera-On Policy:** Zu Beginn etablieren
2. **Breakout-Rooms:** Für 2er-Diskussionen
3. **Chat nutzen:** Ideen in Chat schreiben
4. **Mentimeter/Poll:** Anonym abfragen
5. **Rotating Facilitator:** Remote-Teilnehmer einbeziehen

## 5. Facilitator Skills

### 5.1 Aktives Zuhören

```
DO:
✅ Augenkontakt
✅ Nickend bestätigen
✅ Zusammenfassen ("Also, du sagst...")
✅ Nachfragen bei Unklarheit

DON'T:
❌ Unterbrechen
❌ Gedankenlos nicken
❌ eigenen Kommentar vorbereiten während andere reden
❌ Handy/ Laptop checken
```

### 5.2 Zeitmanagement

```
Tools:
- Time Timer (physisch oder App)
- TweetWall (Twitter-Integration)
- Clockify (Team-Timer)

Regeln:
- Zeit pro Phase vorher festlegen
- 5-Min-Warnung ("Noch 5 Min für Phase 2")
- Bei Überziehung: Entscheidung "Weiter oder kürzen?"
```

### 5.3 Neutralität

```
DO:
✅ "Was denkt ihr darüber?" statt "Ich finde..."
✅ Ideen zusammenfassen ohne Bewertung
✅ Alle Perspektiven repräsentieren

DON'T:
❌ Eigene Meinung durchscheinen lassen
❌ Favoriten haben
❌ Ideen die man gut findet öfter erwähnen
```
