# 电子签章组件 API 文档

## 接口地址及配置

| 环境 | 接口请求地址 |
|------|-------------|
| 线上环境 | `https://api.dataqin.com` |

## 认证方式

同链开放能力，使用 HMAC-SHA256 签名认证。

## 业务流程

### 创建印章流程

```
创建主体（企业/个人）→ 上传印章文件（可选）→ 创建印章
```

### 签署合同流程

```
上传文件 → 创建合同模板 → 配置模板 → 创建签署流程 → 获取签署链接 → 用户签署 → 回调通知
```

## 接口列表

### 文件模块

#### 1. 上传文件（Base64）

- **接口地址**: `POST /1952208013663731714`
- **描述**: 上传文件，文件大小不超过 10M

**请求参数**:

| 名称 | 类型 | 必填 | 说明 |
|------|------|------|------|
| inputSource | string | 是 | Base64 编码的文件内容 |
| fileName | string | 是 | 文件名 |
| fileExt | string | 是 | 文件后缀 |

---

#### 2. Word 转 PDF（Base64）

- **接口地址**: `POST /1952207666752847874`
- **描述**: 只接受 docx/doc 文档，不超过 10M

**响应**: 返回临时下载链接，有效期 1 小时

---

#### 3. 文件下载

- **接口地址**: `POST /1952189609007448065`
- **响应**: 返回文件流

---

#### 4. 上传印章文件

- **接口地址**: `POST /1952189892311711746`
- **描述**: 文件大小不超过 5M

**请求参数**:

| 名称 | 类型 | 必填 | 说明 |
|------|------|------|------|
| inputSource | string | 是 | Base64 编码的文件内容 |
| type | integer | 是 | 0=企业，1=个人 |
| fileName | string | 是 | 文件名 |

## 回调通知

### 签名规则

1. 参数按 key 字典序排序（排除 sign 字段）
2. 拼接 key=value，用 & 连接
3. 末尾追加 &appSecret=xxx
4. MD5 加密，转小写

### 回调类型

| 回调 | 触发时机 |
|------|---------|
| 模板配置完成 | 人工配置完模板后 |
| 创建流程完成 | 签署流程创建完成 |
| 签署完成 | 所有签署方签署完成 |
| 签署方完成 | 单个签署方签署完成 |
