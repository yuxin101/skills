# 保管单组件 API 文档

## 接口地址及配置

| 环境 | 接口请求地址 |
|------|-------------|
| 线上环境 | `https://api.dataqin.com` |

## 认证方式

同链开放能力，使用 HMAC-SHA256 签名认证。

## 接口列表

### 1. 生成保管单

- **接口地址**: `POST /1824399010247397378`
- **描述**: 渲染 HTML + 生成 PDF + 签章

**请求参数**:

主要包含模板信息和保管单数据，具体字段根据模板配置而定。

**请求示例**:
```json
{
  "templateId": "模板ID",
  "certNo": "证书编号（可选，不传则自动生成）",
  "defaultFields": {
    "applicantName": "申请人名称",
    "applicantIdCard": "身份证号",
    "applyTime": "申请时间"
  },
  "customFields": []
}
```

**响应参数**:

| 名称 | 说明 |
|------|------|
| certNo | 存证编号 |

---

### 2. 获取保管单下载地址

- **接口地址**: `POST /1824398465122095105`
- **描述**: 返回 PDF 文件下载地址

**请求参数**:

| 名称 | 类型 | 必填 | 说明 |
|------|------|------|------|
| certNo | string | 是 | 存证编号 |

**响应参数**:

| 名称 | 说明 |
|------|------|
| url | PDF 下载地址 |

---

## 服务响应状态码

| 状态码 | 说明 |
|--------|------|
| 200 | 成功 |
| 500 | 程序异常 |
| 500043 | 文件类型获取失败 |
| 500044 | 不支持的文件类型转换 |
| 500129 | 模板名称已存在 |
| 500130 | 模板名称不能为空 |
| 500131 | 模板描述最大长度为200 |
| 500132 | 模板名称最大长度为30 |
| 500133 | templateId 记录非启用状态 |
| 500134 | 文件不存在 |
| 500135 | 证书编号已存在 |
| 500136 | certNo 字段不能为空 |
| 500137 | certNo 字段长度不能超过32位 |
| 500138 | 未查询到存证信息 |
| 500139 | 未查询到存证文件 |
| 500140 | 字段内容不能为空 |
