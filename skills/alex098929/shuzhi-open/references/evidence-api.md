# 自动化取证 API 文档

## 接口地址及配置

| 环境 | 接口请求地址 |
|------|-------------|
| 线上环境 | `https://api.dataqin.com` |

## 认证方式

同链开放能力，使用 HMAC-SHA256 签名认证。

## 接口列表

### 1. 创建批量任务

- **接口地址**: `POST /1890212993579274242`
- **描述**: 创建批量取证任务

**请求参数**:

| 名称 | 类型 | 必填 | 说明 |
|------|------|------|------|
| batch_id | string | 是 | 批次ID |
| tasks | array | 是 | 任务列表 |

**tasks 数组元素**:

| 名称 | 类型 | 必填 | 说明 |
|------|------|------|------|
| att_id | string | 是 | 保全号（≤100字符） |
| url | string | 是 | 取证地址（≤1000字符） |
| evidence_name | string | 是 | 取证名称（≤128字符） |
| evidence_label | string | 否 | 取证标签 |
| type | integer | 是 | 取证类型：1=拼多多，3=淘宝，4=抖音 |
| business_license_status | boolean | 否 | 是否取证营业执照 |
| video_publisher | string | 否 | 视频发布者 |
| callback_url | string | 否 | 回调地址（≤256字符） |

**请求示例**:
```json
{
  "batch_id": "123456789",
  "tasks": [
    {
      "att_id": "123456789",
      "url": "https://mobile.yangkeduo.com/goods.html?goods_id=xxx",
      "evidence_name": "商品取证",
      "type": 1,
      "business_license_status": true
    }
  ]
}
```

**响应参数**:

| 名称 | 说明 |
|------|------|
| batch_id | 批次号 |
| ws_url | 手机 websocket 连接 |
| url | 取证页面链接 |
| bind_time | 绑定时间 |

## 状态枚举

### 批次状态

| 值 | 说明 |
|----|------|
| NEW | 新建 |
| ASSIGNED | 已分配 |
| LOGIN | 账号登录中 |
| CHECKING | 账号自检中 |
| CHECKED | 账号自检完成 |
| UNDERWAY | 取证中 |
| SUCCESSED | 取证成功 |
| FAILED | 取证失败 |

### 任务状态

| 值 | 说明 |
|----|------|
| NEW | 新建 |
| RUNNING | 执行中 |
| SUCCESSED | 成功 |
| FAILED | 失败 |

### 取证类型

| 值 | 说明 |
|----|------|
| 1 | 拼多多电商取证 |
| 3 | 淘宝电商取证 |
| 4 | 抖音电商取证 |
