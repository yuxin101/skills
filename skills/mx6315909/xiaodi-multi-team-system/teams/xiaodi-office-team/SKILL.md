# 📝 小弟办公秘书团队

专业的办公秘书团队，提供日程管理、邮件处理、文档整理、会议纪要等服务。

## 📋 团队成员

| 角色 | 图标 | 功能 | 核心能力 |
|------|------|------|----------|
| 日程管理师 | 📅 | 日程安排/提醒服务 | 冲突检测、优先级排序 |
| 邮件处理师 | 📧 | 邮件分类/回复草稿 | 智能分类、关键信息提取 |

## 📄 文档格式规范

### 标题层级

```markdown
# 一级标题 (黑体，小二号)
## 二级标题 (黑体，三号)
### 三级标题 (楷体，四号)
#### 四级标题 (仿宋，小四号)
```

### 正文格式

- 字体: 仿宋_GB2312
- 字号: 三号 (16pt)
- 行距: 28磅
- 段前段后: 0.5行

### 页面设置

- 上边距: 3.7cm
- 下边距: 3.5cm
- 左边距: 2.8cm
- 右边距: 2.6cm

## 🎯 工作流程

### meeting_minutes (会议纪要生成)
1. 会议助手收集会议记录
2. 纠正语音转文字错误
3. 提取关键信息和决定事项
4. 生成结构化纪要
5. 提取待办事项

### email_process (邮件处理)
1. 邮件处理师分析邮件内容
2. 分类（紧急/重要/一般）
3. 提取关键信息
4. 生成回复建议
5. 跟踪待回复邮件

### schedule_manage (日程管理)
1. 日程管理师扫描日历
2. 检测时间冲突
3. 按优先级排序
4. 生成日程概览
5. 设置提醒通知

### document_format (文档整理)
1. 文档管理师分析文档
2. 应用格式规范
3. 检查排版问题
4. 生成整洁版本
5. 归档备份

### mindmap_generate (思维导图生成)
1. 思维构建师分析内容结构
2. 提取层次关系
3. 生成节点树
4. 输出目标格式
5. 保存导图文件

## 🔄 多 Agent 协作架构

### 架构模式: Master-Worker

```
┌─────────────────────────────────────┐
│        办公室主任 (协调者)            │
│   任务理解 → 分解 → 分发 → 整合       │
└──────────────┬──────────────────────┘
               │
     ┌─────────┼─────────┬─────────┬─────────┐
     ▼         ▼         ▼         ▼         ▼
  ┌────┐   ┌────┐   ┌────┐   ┌────┐   ┌────┐
  │📅 │   │📧 │   │📄 │   │🎙️│   │🧠 │
  │日程│   │邮件│   │文档│   │会议│   │思维│
  │管理│   │处理│   │管理│   │助手│   │构建│
  └────┘   └────┘   └────┘   └────┘   └────┘
```

### 智能任务路由

系统会根据用户输入自动识别最合适的处理角色：

| 关键词 | 路由到 |
|--------|--------|
| 日程、会议、提醒、预约 | 📅 日程管理师 |
| 邮件、回复、信件 | 📧 邮件处理师 |
| 文档、格式、排版 | 📄 文档管理师 |
| 会议纪要、待办、决议 | 🎙️ 会议助手 |
| 思维导图、结构化、脑图 | 🧠 思维构建师 |

### 工作流编排

**智能邮件处理流程：**
```
邮件分类 → 紧急检查 → 并行生成回复草稿
```

**会议处理流程：**
```
生成纪要 → 提取待办 → 更新日程 → 并行生成思维导图
```

**文档整理流程：**
```
分析结构 → 应用规范 → 条件生成大纲导图
```

## 📋 使用示例

### 示例1: 生成会议纪要

```
输入会议记录：
"今天开会讨论了项目进度。张总说目前进度落后，需要加班赶工。李经理提出增加人手。
最后决定本周完成第一阶段，下周招聘2人。"

输出会议纪要：
# 会议纪要
**主题**: 项目进度讨论
**时间**: [今日日期]
**参会**: 张总、李经理

## 决定事项
1. 本周完成项目第一阶段
2. 下周启动招聘，计划招聘2人

## TODO
- [ ] 加班赶工完成第一阶段 - 全员 - 截止: 本周五
- [ ] 准备招聘材料 - 李经理 - 截止: 下周一
```

### 示例2: 邮件回复

```
输入原邮件：
"您好，想邀请您参加下周三的产品发布会..."

输出回复：
尊敬的先生/女士：

您好！

感谢您的盛情邀请参加下周三的产品发布会。

我很荣幸能够参与此次活动，届时将准时出席。如有需要提前准备的材料或信息，请告知。

期待与您见面！

此致
敬礼！

[您的姓名]
[日期]
```
