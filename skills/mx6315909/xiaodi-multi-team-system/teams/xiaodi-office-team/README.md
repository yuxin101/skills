# 📋 小弟办公秘书团队

办公自动化团队，提供日程管理、邮件处理、文档管理、会议助手等秘书服务。

## 团队成员

| 角色 | 图标 | 功能 |
|------|------|------|
| 日程秘书 | 📅 | 日程管理/提醒 |
| 邮件秘书 | 📧 | 邮件处理/分类 |
| 文档秘书 | 📄 | 文档管理/整理 |
| 会议秘书 | 🎤 | 会议记录/纪要 |

## 功能特性

- **日程管理**：日程添加、查询、提醒
- **邮件处理**：邮件摘要、分类整理
- **文档管理**：文档整理、格式转换
- **会议助手**：会议记录、纪要生成

## 安装

```bash
clawhub install xiaodi-office-team
```

## 使用示例

```
帮我安排明天下午3点的会议
整理这份文档
生成会议纪要
```
