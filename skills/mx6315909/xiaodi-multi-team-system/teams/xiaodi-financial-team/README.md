# 📊 小弟金融分析师团队

专业的金融分析团队，7 大角色协作，提供投研、投行、市值管理、财富管理、交易执行、舆情监控、投资顾问全方位服务。

## 团队成员

| 角色 | 图标 | 功能 |
|------|------|------|
| 投顾专家 | 💼 | 综合服务、统筹协调 |
| 行业研究员 | 🔬 | 产业链分析、机会挖掘 |
| 投行专家 | 🏦 | 并购重组、IPO 辅导 |
| 市值管理助理 | 📈 | IR 策略、资本运作 |
| 财富专员 | 💰 | 资产配置、基金筛选 |
| 商机助理 | 🎯 | 交易信号、买卖时机 |
| 舆情助理 | 📰 | 舆情监测、风险预警 |

## 功能特性

- **个股分析**：基本面、技术面、舆情综合分析
- **持仓诊断**：风险评估、仓位建议
- **资产配置**：个性化配置方案
- **舆情监控**：新闻情绪分析、风险预警

## 数据源

- 东方财富
- 腾讯财经
- 新浪财经

## 安装

```bash
clawhub install xiaodi-financial-team
```

## 使用示例

```
分析茅台的投资价值
诊断我的持仓：茅台 100股、宁德时代 200股
我有100万资金，帮我做资产配置
```
