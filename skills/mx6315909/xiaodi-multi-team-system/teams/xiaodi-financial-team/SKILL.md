# 📊 小弟金融分析师团队

专业的金融分析团队，7大角色协作，提供投研、投行、市值管理、财富管理、交易执行、舆情监控、投资顾问全方位服务。

## 📋 团队成员

| 角色 | 图标 | 功能 | 核心能力 |
|------|------|------|----------|
| 投顾专家 | 💼 | 综合服务、统筹协调 | 投资组合、风险评估 |
| 行业研究员 | 🔬 | 产业链分析、机会挖掘 | 行业研究、竞品分析 |
| 投行专家 | 🏦 | 并购重组、IPO辅导 | 资本运作、估值建模 |
| 市值管理助理 | 📈 | IR策略、资本运作 | 投资者关系、市值策略 |
| 财富专员 | 💰 | 资产配置、基金筛选 | 财富规划、基金优选 |
| 商机助理 | 🎯 | 交易信号、买卖时机 | 技术分析、量化信号 |
| 舆情助理 | 📰 | 舆情监测、风险预警 | 新闻分析、情绪监控 |

## 📋 工作流程

### 个股分析流程

```
投顾专家接收请求
    ↓
行业研究员：行业地位分析
    ↓
商机助理：技术面分析
    ↓
舆情助理：舆情风险评估
    ↓
投顾专家：综合报告输出
```

### 持仓诊断流程

```
投顾专家接收持仓列表
    ↓
商机助理：实时行情获取
    ↓
财富专员：仓位风险评估
    ↓
行业研究员：个股诊断
    ↓
投顾专家：诊断报告输出
```

### 资产配置流程

```
财富专员接收资金信息
    ↓
行业研究员：市场环境分析
    ↓
商机助理：机会筛选
    ↓
财富专员：配置方案设计
    ↓
投顾专家：方案审核输出
```

### 示例1：个股诊断

```
输入：诊断茅台

输出：
# 贵州茅台(600519) 投资诊断报告

## 基本面评分: 85/100
- 行业地位：白酒龙头，品牌护城河深
- 财务健康：ROE 30%+，现金流充沛
- 估值水平：PE 35x，处于历史中位

## 技术面评分: 70/100
- 趋势：震荡整理
- 支撑：1800元
- 压力：2000元

## 综合评级: 增持
- 目标价：2100元
- 止损位：1700元
```

```
输入：诊断我的持仓组合

# 持仓诊断报告

## 组合概况

## 风险评估
- 集中度风险：中等
- 行业风险：偏高（新能源波动大）
- 建议减持新能源仓位10%

## 个股评分
| 股票 | 评分 | 建议 |
|------|------|------|
| 茅台 | 85 | 持有 |
| 宁德时代 | 70 | 减持 |
```
