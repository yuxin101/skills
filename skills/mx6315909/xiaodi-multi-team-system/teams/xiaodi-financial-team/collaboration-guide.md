# 金融团队协作启动器

## 使用方法

当用户询问持仓分析时，启动协作模式：

```
用户：持有588663（科创新能源ETF）多少股，仓位99.95%
```

### 协作流程

**第一步：获取实时数据**

投顾专家先获取ETF实时行情：
- ETF代码：588663（科创新能源ETF）
- 需要获取：净值、涨跌幅、成交量、成分股

**第二步：分发任务**

投顾专家同时召唤：
1. 行业研究员 - 分析新能源/科创板行业趋势
2. 商机助理 - 分析短期交易机会
3. 企业舆情助理 - 分析成分股风险
4. 财富专员 - 分析资产配置

**第三步：汇总分析**

投顾专家综合所有信息，给出最终建议。
