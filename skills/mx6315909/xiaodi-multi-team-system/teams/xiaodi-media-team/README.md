# 🎬 小弟多媒体团队

专业的多媒体处理团队，7 个专业角色协作完成视频剪辑、图片处理、音频转码、字幕生成、AI 创作等任务。

## 团队成员

| 角色 | 图标 | 功能 |
|------|------|------|
| 视频剪辑师 | 🎬 | 视频剪辑/压缩/转码 |
| 视频创作师 | 🎥 | AI 视频生成 |
| 字幕生成器 | 📝 | 自动生成字幕 |
| 图片处理师 | 🖼️ | 图片压缩/转换/缩放 |
| AI 绘图师 | 🎨 | AI 生成图片 |
| 音频处理师 | 🎵 | 音频转码/提取 |
| 质量检查员 | 📊 | 检查文件质量 |

## 功能特性

- **视频处理**：压缩、转码、剪辑、提取帧
- **AI 视频生成**：支持可灵、Runway、Pika、Sora、即梦
- **图片处理**：批量压缩、格式转换、缩放
- **字幕生成**：Whisper 语音识别，支持多语言
- **音频处理**：提取、转码、剪辑

## 依赖

- ffmpeg
- ImageMagick
- whisper (可选)

## 安装

```bash
clawhub install xiaodi-media-team
```

## 使用示例

```
压缩这个视频 /path/to/video.mp4
提取视频第10秒的帧 /path/to/video.mp4
生成一个视频：猫咪在阳光下打盹
```
