# 🛒 小弟电商团队

专业的电商运营团队，8 个专业角色协作完成选品、上架、定价、客服、数据分析、竞品监控、内容创作、广告投放。

## 团队成员

| 角色 | 图标 | 功能 |
|------|------|------|
| 选品专家 | 🔍 | 选品分析/爆款挖掘 |
| 运营专员 | 📊 | 店铺运营/数据分析 |
| 定价专员 | 💵 | 价格策略/利润计算 |
| 客服专员 | 💬 | 客户服务/评价管理 |
| 数据分析师 | 📈 | 数据采集/报表生成 |
| 竞品分析师 | 🎯 | 竞品监控/差异化分析 |
| 内容创作者 | ✍️ | 商品文案/Listing |
| 投放专员 | 📢 | 广告投放/ROI 优化 |

## 功能特性

- **选品分析**：BSR 分析、趋势追踪、销量预估
- **竞品监控**：价格追踪、差评挖掘
- **利润计算**：成本核算、ROI 分析
- **Listing 生成**：SEO 优化、标题生成

## 支持平台

- 亚马逊（国内 + 跨境）
- TikTok Shop
- 小红书
