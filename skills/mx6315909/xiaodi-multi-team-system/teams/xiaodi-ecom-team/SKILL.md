## 💰 利润计算模型

### Amazon 跨境

| 项目 | 计算 |
|------|------|
| 采购成本 | 1688 批发价 |
| 头程物流 | $3-6/kg（按重量） |
| FBA 配送费 | 按尺寸分级 |
| 平台佣金 | 售价 × 15% |
| 广告成本 | 销售额 × 10-20% |
| **总成本** | 采购 + 物流 + FBA + 佣金 + 广告 |
| **毛利率** | (售价 - 总成本) / 售价 |

### 销量预估（BSR 转换）

| BSR 区间 | 月销量估算 |
|----------|------------|
| < 1,000 | 500-2,000+ 件 |
| 1,000-5,000 | 200-800 件 |
| 5,000-20,000 | 50-300 件 |
| > 20,000 | 10-100 件 |

## ✍️ Listing 生成

### SEO 优化原则

1. **标题**（200字符内）
   - 核心关键词前置
   - 品牌名 + 核心卖点 + 适用场景

2. **五点描述**
   - 全大写功能词开头
   - 卖点 + 场景 + 利益点

3. **长描述**
   - A+ 页面布局
   - FAQ 常见问题

### 关键词埋词

```
自动提取竞品高频词
- 标题关键词
- 搜索词
- 后台隐藏词
```

## ⚠️ 注意事项

1. **数据合规** - 仅抓取公开数据，不获取后台数据
2. **销量估算** - 基于BSR模型推算，非精确数据
3. **反爬策略** - 建议配置代理IP轮换
4. **Listing原创** - 避免直接抄袭，防止降权
