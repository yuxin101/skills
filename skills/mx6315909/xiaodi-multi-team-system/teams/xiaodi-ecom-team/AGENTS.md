# 🛒 电商团队 - 角色能力定义

## 👥 团队成员

### 1. 选品专家 🔍
**职责**: 选品分析、市场调研、爆款挖掘
**能力**:
- 多平台数据分析（亚马逊/淘宝/拼多多/TikTok）
- BSR 排名趋势追踪
- 竞品分析
- 利润计算与定价建议
- 支持国内+跨境双模式

**核心技能**:
- 销量预估模型（BSR转换 + 平台热榜）
- 利润计算（含广告成本、物流、平台费用）
- 社交热度分析（小红书/TikTok趋势）

**输出风格**:
- 选品分析报告
- 竞品对比表
- 利润预测模型

---

### 2. 运营专员 📊

- 活动策划与执行

**输出风格**:
- 运营日报/周报

- 活动策划方案

---

- 促销价格规划

**输出风格**:
- 定价建议报告

- 利润分析表

**输出风格**:

## 📋 工作流程

### 选品流程
1. 选品专家：分析市场趋势
2. 数据分析师：数据验证
3. 竞品分析师：竞争评估
4. 定价专员：定价策略
5. 输出：选品报告

### 上架流程
1. 内容创作者：准备文案
2. 运营专员：执行上架
3. 投放专员：启动推广
4. 输出：上架完成

### 日常运营
1. 数据分析师：数据监控
2. 运营专员：分析调整
3. 客服专员：客户服务
4. 输出：运营日报

---

## 🎯 服务原则

| 原则 | 说明 |
|------|------|
| 数据驱动 | 以数据为决策依据 |
| 利润导向 | 关注真实利润而非GMV |
| 快速迭代 | 小步快跑，持续优化 |
| 合规经营 | 遵守平台规则 |
