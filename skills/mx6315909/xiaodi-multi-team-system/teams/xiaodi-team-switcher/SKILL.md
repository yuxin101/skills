# 🔄 小弟团队切换器

智能识别用户意图，自动切换到最合适的功能团队。

---

## 📊 可用团队

| 团队 | 图标 | 状态 | 适用场景 |
|------|------|------|----------|
| 金融分析师团队 | 📊 | ✅ 可用 | 股票分析、基金推荐、持仓复盘、投资决策 |
| 电商团队 | 🛒 | ✅ 可用 | 选品、上架、定价、客服、数据分析 |
| 多媒体团队 | 🎬 | ✅ 可用 | 视频剪辑、图片处理、字幕生成、音频转码 |

## 🎯 关键词识别规则

### 金融团队 📊
**强关键词** (直接切换):
- 股票、基金、持仓、A股、估值、PE、PB、ROE
- K线、技术面、基本面、止损、止盈

**中等关键词** (结合语境):
- 分析、复盘、推荐、诊断、对比、选股

**弱关键词** (需确认):
- 看一下、帮我、查一下

### 电商团队 🛒
**强关键词**:
- 选品、上架、电商、跨境、亚马逊、淘宝、拼多多、TikTok

**中等关键词**:
- 定价、客服、利润、BSR、销量、转化率、ROI

### 多媒体团队 🎬
**强关键词**:
- 视频、剪辑、图片、音频、字幕、水印、转码、压缩

**中等关键词**:
- 渲染、特效、配音、BGM、封面、缩略图

### 办公团队 📝
**强关键词**:
- 日程、邮件、文档、会议、提醒、待办、日历

**中等关键词**:
- 整理、归档、通知、预约、安排、记录

## 🚀 使用方式

### 自动切换
用户发送消息时，切换器会自动识别关键词并切换到对应团队：

```
用户: 帮我分析一下茅台的估值
→ 自动切换到 📊 金融分析师团队
```

```
用户: 选品推荐，跨境美妆类目
→ 自动切换到 🛒 电商团队
```

```
用户: 压缩这个视频
→ 自动切换到 🎬 多媒体团队
```

### 手动切换
直接指定团队名称：

```
切换到金融团队
用电商团队
多媒体团队处理
```

## 📋 工作流程

### 1. 意图识别
```
输入: 用户消息
处理: 
  1. 提取关键词
  2. 计算各团队置信度
  3. 选择最高置信度团队
输出: 目标团队
```

### 2. 团队激活
```
输入: 目标团队
处理:
  1. 加载团队 AGENTS.md
  2. 初始化团队角色
  3. 设置上下文
输出: 团队就绪
```

### 3. 任务分发
```
输入: 用户任务
处理:
  1. 分析任务类型
  2. 分配给合适的角色
  3. 协调多角色协作
输出: 执行结果
```

## ⚙️ 配置

配置文件: `{baseDir}/config.json`

```json
{
  "settings": {
    "default_team": "financial",
    "fallback_action": "ask",
    "min_confidence": 0.6,
    "auto_switch": true,
    "state_persistence": true
  }
}
```

### 配置说明

| 参数 | 默认值 | 说明 |
|------|--------|------|
| default_team | financial | 默认团队 |
| fallback_action | ask | 置信度不足时的行为 (ask/random) |
| min_confidence | 0.6 | 最低置信度阈值 |
| auto_switch | true | 是否自动切换 |
| state_persistence | true | 是否持久化状态 |

## 🎯 设计原则

1. **用户无感**: 用户不需要知道切换过程，直接获得专业服务
2. **快速响应**: 切换决策 < 100ms
3. **智能记忆**: 记住用户偏好，越用越准
4. **团队隔离**: 各团队独立运行，互不干扰
