# 🔄 小弟团队切换器

多团队协作切换系统，根据用户需求自动识别并切换到合适的功能团队。

## 支持的团队

| 团队 | 图标 | 功能 |
|------|------|------|
| 金融团队 | 📊 | 股票分析、投资决策 |
| 电商团队 | 🛒 | 选品、运营、定价 |
| 多媒体团队 | 🎬 | 视频、图片、音频处理 |
| 办公团队 | 📋 | 日程、邮件、文档 |

## 功能特性

- **智能识别**：自动分析用户意图
- **一键切换**：无缝切换到目标团队
- **团队隔离**：各团队独立运行

## 安装

```bash
clawhub install xiaodi-team-switcher
```

## 使用示例

```
切换到金融团队
分析茅台
帮我选品
压缩这个视频
```
