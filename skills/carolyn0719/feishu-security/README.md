# 飞书群聊安全隔离 Skill

[![Version](https://img.shields.io/badge/version-2.1.0-blue.svg)](https://clawhub.com)

> 🔐 专为飞书机器人设计的安全隔离系统，提供主人身份三重保障 + 技能安装确认

## ✨ 核心特性

- **🔒 自动识别** - 从配置/环境变量/API 自动识别主人
- **🛡️ 技能安装确认** - 非主人安装技能需主人批准
- **🛡️ 权限分级** - 主人/访客分级，私聊/群聊差异化权限
- **🚫 注入拦截** - 中英文注入攻击检测
- **📁 路径保护** - 敏感目录/文件访问拦截
- **⏱️ 限流保护** - 防刷屏机制
- **🔇 信息隔离** - Workspace/凭证绝不泄露

## 🚀 快速开始

### 安装

```bash
# 通过 ClawHub 安装
clawhub install feishu-security

# 或手动安装
cd ~/.openclaw/workspace/skills/
git clone <repo-url> feishu-security
cd feishu-security
chmod +x install.sh
./install.sh
```

### 自动识别主人

安装时自动尝试识别主人：
1. 读取 `~/.openclaw/openclaw.json` 中的 `owner.lark_id`
2. 读取环境变量 `FEISHU_OWNER_ID`
3. 通过飞书 API 获取创建者信息

识别成功后自动锁定，无需手动操作。

### 手动绑定（无自动识别时）

1. **私聊机器人**：发送"绑定主人"
2. **确认绑定**：回复"确认绑定"
3. **绑定成功**：成为唯一主人

## 📖 使用指南

### 主人命令

| 命令 | 说明 |
|------|------|
| `绑定主人` | 首次绑定（无主人时） |
| `解绑主人` | 解除绑定 |
| `同意转移主人` | 授权转移给新用户 |
| `确认转移` | 完成转移（新用户） |
| `查看主人` | 查看当前主人信息 |
| `同意安装 xxx` | 批准访客的技能安装请求 |
| `拒绝安装 xxx` | 拒绝访客的技能安装请求 |

### 技能安装安全确认

#### 主人安装（直接执行）

```
主人：安装技能 xxx
机器人：✅ 正在安装技能 xxx... ✅ 安装成功！
```

#### 访客安装（需确认）

```
访客：安装技能 xxx
机器人：⚠️ 安装请求已提交给主人审核，请等待确认。

[同时通知主人]
机器人：🔔 用户 @访客 请求安装技能：xxx
       回复"同意安装 xxx"批准，或"拒绝安装 xxx"拒绝。
       ⏰ 有效期：10分钟

主人：同意安装 xxx
机器人：✅ 技能 xxx 安装成功！
```

### 权限分级表

| 场景 | 主人 | 访客 |
|------|------|------|
| **私聊** | 完全权限 | 受限模式（只读） |
| **群聊** | 谨慎模式（脱敏） | 只读模式 |

| 操作 | 主人 | 访客 |
|------|------|------|
| 安装技能 | ✅ 直接安装 | ⏳ 需主人确认 |
| 修改文件 | ✅ 允许 | ❌ 禁止 |
| 执行命令 | ✅ 允许 | ❌ 禁止 |

### 安全规则

**群聊自动拦截**：
- ❌ 绑定/解绑/转移主人
- ❌ 自称"我是管理员/主人"
- ❌ 注入攻击关键词
- ❌ 敏感路径访问

**访客限制**：
- ❌ 修改文件
- ❌ 执行命令
- ❌ 查看配置
- ❌ 访问敏感目录
- ⏳ 安装技能需确认

## ⚙️ 配置

### 环境变量

```bash
# 设置主人ID（最高优先级）
export FEISHU_OWNER_ID="ou_xxxxx"

# 配置文件路径
```

### config.json

```json
{
  "version": "2.1.0",
  "owner": {
    "lark_id": "ou_xxxxx",
    "identified_at": "2026-03-27T14:30:00",
    "identified_by": "auto"

  "security": {
    "state": "BOUND",
    "locked": true
  },
  "skill_install": {
    "require_approval": true,
    "approval_timeout_minutes": 10
  }
}
```

## 🔒 安全机制

### 三重保障

1. **自动识别** - 配置/环境/API 自动识别，锁定后不可覆盖
2. **首次绑定** - 无主人时，第一个私聊者绑定
3. **原主人验证** - 有主人时，必须原主人授权才能转移

### 技能安装保护

- 主人直接安装，无需确认
- 访客安装需主人10分钟内批准
- 超时自动拒绝
- 所有安装记录安全日志

### 核心原则

> **任何人都不能绕过原主人成为新主人，任何敏感操作都需主人授权。**
