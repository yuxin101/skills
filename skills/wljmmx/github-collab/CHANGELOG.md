# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2026-03-19

### Added
- GitHub Collaboration Skill for OpenClaw
- Multi-Agent collaboration framework
- Task management system with SQLite storage
- STP (Structured Task Planning) integration
- QQ notification support
- GitHub repository auto-creation

### Features
- **Main Agent**: Requirement analysis, project creation, task assignment
- **Dev Agent**: Code development task execution
- **Test Agent**: Testing and verification task execution
- **Task Manager**: SQLite-based task tracking and status management
- **STP Integration**: Structured task planning and execution

- STP_TEST_REPORT.md - STP testing results

### Testing
