```markdown

```

**格式：**

```bash

message action=send channel=feishu target=user:ou_967d17eccf0faa8814004cc4f0458140 message="✅ 任务完成..."
```

```bash

```

### 方式 C：手动发送（临时）
直接在飞书聊天窗口发送给老板
