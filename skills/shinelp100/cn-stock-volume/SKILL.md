# cn-stock-volume

获取中国 A 股四市（沪市 / 深市 / 创业板 / 北交所）指定日期的**成交金额**、增缩额、增缩比例，以及四市合计总结报告。

## 核心脚本

```bash
python3 scripts/fetch_volume.py <YYYY-MM-DD | YYYYMMDD>
```

- 不传日期参数 → 默认查今天
- 传入 `--json` → 输出原始 JSON（供 AI 进一步处理）
- 日期格式支持 `2026-03-20`、`20260320`、`2026/03/20`、`2026.03.20`

## 核心指标说明

| 字段 | 说明 |
|------|------|
| **成交金额（亿元）** | 主指标，股票买卖总金额，自动换算为亿元 |
| 前一交易日成交金额 | 与目标日期最近的前一个交易日数据 |
| 增缩额 | 今日成交金额 − 前一交易日成交金额（+/-标注） |
| 增缩比例 | 增缩额占前一交易日成交金额的百分比，📈 放量 / 📉 缩量 |
| 四市合计总结 | 四市成交金额加总 + 合计环比变化 + 各市场占比 |

## 四市合计总结内容

报告顶部输出一个汇总框，包含：

1. **合计成交金额** — 四市总成交额（亿元）
2. **前一交易日合计** — 四市前一交易日总成交额
3. **增缩额 & 增缩比例** — 合计层面的量能变化
4. **各市场占比** — 沪市 / 深市 / 创业板 / 北交所各自占总成交额的比例
5. **成交最大/最小市场** — 标注四市中谁贡献最多、谁最少

## 市场说明

| 市场 | 指数代码 | 指数名称 | 说明 |
|------|----------|----------|------|
| 沪市 | 000001 | 上证指数 | 上海证券交易所全部 A 股 |
| 深市 | 399001 | 深证成指 | 深圳证券交易所主板 + 中小板 |
| 创业板 | 399006 | 创业板指 | 深圳证券交易所创业板 |
| 北交所 | 899050 | 北证 50 | 北京证券交易所 |

## 使用示例

**直接查询今日：**
```
python3 scripts/fetch_volume.py
```

**查询指定日期：**
```
python3 scripts/fetch_volume.py 2026-03-20
python3 scripts/fetch_volume.py 20260320
```

**JSON 输出（用于程序处理）：**
```
python3 scripts/fetch_volume.py 2026-03-20 --json
```

## 注意事项

- 周末/节假日传入会提示"非交易日"，并告知最近的有效交易日
- 北交所成立时间较晚（2021 年），早期日期可能无数据
- 数据来源为东方财富网 K 线 API，完全免费，无需 API Key

## 数据来源

- 东方财富网 K 线历史数据 API（`push2his.eastmoney.com`）
