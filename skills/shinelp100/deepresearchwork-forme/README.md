# Deep Research Work For Me

个人深度产研报告生成器 - 自动化研究、优化、可视化完整工作流

## 安装

```bash
clawhub install deepresearchwork-forme
```

## 使用方法

```
帮我生成一份关于 [主题] 的深度产研报告
```

## 功能特点

- 📊 **深度研究**：多源数据采集、交叉验证、分析综合
- 📝 **文档优化**：格式规范化、结构优化、可读性提升
- 🎨 **图表可视化**：Mermaid 图表自动转换为 PNG 图片

## 依赖

- deepresearchwork
- markdown-formatter
- @mermaid-js/mermaid-cli
