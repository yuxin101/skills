# Deep Research Work For Me - 使用案例

## 案例：赣锋锂业深度研究报告

### 执行命令

```
帮我生成一份关于 赣锋锂业所在锂电细分领域 的深度产研报告
```

### 执行流程

#### 阶段一：深度研究

1. **研究规划**
   - 研究主题：赣锋锂业所在锂电细分领域
   - 关键问题：产业链定位、竞争格局、核心优势、发展前景

2. **信息收集**
   - 搜索赣锋锂业业务布局、产能数据
   - 收集锂电池产业链分析报告
   - 获取提锂技术路线对比数据
   - 整理竞争对手（天齐锂业等）对比信息
   - 调研固态电池和电池回收业务进展

3. **分析综合**
   - 整理核心数据表格
   - 绘制产业链结构图
   - 分析竞争格局
   - 提炼投资价值观点

4. **报告生成**
   - 输出文件：`赣锋锂业深度研究报告.md`

#### 阶段二：文档优化

| 优化项 | 具体操作 |
|--------|---------|
| 目录 | 添加完整的 10 章节目录 |
| 表格 | 统一对齐方式，数值右对齐 |
| 引用 | 中文引号改为「」格式 |
| 层级 | 使用 🌏🌎🇨🇳🥇🥈🥉 等 emoji |
| 提示 | 添加免责声明和来源说明 |

#### 阶段三：图表可视化

生成的图表：

| 图表 | 类型 | 说明 |
|------|------|------|
| chart1.png | flowchart LR | 锂电池产业链全景 |
| chart2.png | mindmap | 全球资源布局（3640万吨LCE） |
| chart3.png | pie | 2024年营收结构 |
| chart4.png | graph TB | 国内竞争格局梯队 |
| chart5.png | flowchart TB | 固态电池产业链 |
| chart6.png | flowchart TB | 回收业务布局 |
| chart7.png | mindmap | 核心竞争优势 |

### 报告结构

1. **执行摘要** - 核心数据一览
2. **产业链全景** - 结构图 + 定位表
3. **上游资源开发** - 技术路线 + 全球布局
4. **中游锂盐加工** - 产品矩阵 + 营收结构
5. **竞争格局分析** - 市场份额 + 双雄对比
6. **固态电池布局** - 产业链 + 技术指标
7. **电池回收业务** - 布局图 + 战略意义
8. **风险因素分析** - 行业风险 + 公司风险
9. **投资价值评估** - 护城河 + 成长驱动
10. **研究结论** - 核心发现 + 待研究问题
