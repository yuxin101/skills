# Stand-Up Reminder 久坐提醒

> 🧍 工作日定时提醒你站起来活动，告别久坐伤身，一键配置，持续生效。

---

## ✨ 功能特性

- ⏰ **工作日自动提醒** — 每个工作日（周一至周五）定时触发，周末自动休息
- 🌍 **多语言支持** — 中文 / English / 日本語，按你的习惯来
- 🎨 **四种文案风格** — 活泼 / 简洁 / 严肃 / 励志，总有一款适合你
- 🗓️ **节假日智能跳过** — 自动识别中国法定节假日及调休安排，假期不打扰
- 🔧 **高度可定制** — 自由设置开始时间、结束时间、提醒间隔、时区

---

## 🚀 快速开始

直接告诉 AI：

> "提醒我每小时站起来活动一下"

AI 会自动创建提醒任务，默认配置：

| 参数 | 默认值 |
|------|--------|
| 开始时间 | 08:30 |
| 结束时间 | 17:30 |
| 间隔 | 每 60 分钟 |
| 语言 | 中文 |
| 风格 | 活泼 |
| 节假日跳过 | 关闭 |

---

## 💬 示例对话

**基础用法：**
```
用户：提醒我每小时站起来活动
AI：✅ 已创建提醒！每个工作日 8:30–17:30，每小时提醒一次，共 10 次/天。
```

**自定义时间：**
```
用户：从 9 点开始，每 90 分钟提醒我站起来，下班时间 18 点
AI：✅ 已创建提醒！每个工作日 9:00–18:00，每 90 分钟提醒一次。
```

**开启节假日跳过：**
```
用户：帮我设置久坐提醒，节假日不用提醒
AI：✅ 已创建提醒，节假日自动跳过！
```

**英文提醒：**
```
用户：Set up an hourly stand-up reminder in English
AI：✅ Done! You'll get hourly reminders on weekdays from 8:30 to 17:30.
```
