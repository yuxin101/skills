## Report Formatting Standards

### Header
- Centered title with box drawing
- Bilingual (Chinese + English)
- Clear hierarchy

### Sections
- Numbered with clear titles
- Box drawing for boundaries
- Consistent spacing

### Content
- Bullet points (✓ for positive, ✗ for negative)
- Bold key terms
- Emoji for visual aid
- Chinese translation for each English point

### Tables
- ASCII art borders
- Consistent width
- Aligned columns

### Final Output
- Professional, publication-quality
- Print-ready
- Screen-friendly
