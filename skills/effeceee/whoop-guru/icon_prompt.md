## Icon Description

A minimalist square app icon for WHOOP + AI. Dark navy to black gradient background. Center: a sleek WHOOP fitness band (no screen, no display, simple horizontal strap) at the bottom. Above the band: a subtle AI neural network pattern forming a brain silhouette in soft green glow. Floating pulse waves and data streams rising from the band in emerald green. Small particles suggesting data analysis. No numbers, no text, no clutter. Clean negative space. Premium tech aesthetic. Square format, centered composition, edge to edge.
