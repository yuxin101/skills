# 镜头语言与视觉风格词汇表（中英对照）

在 Seedance 2.0 prompt 中使用以下英文术语实现精确的镜头控制。中文说明帮助理解含义。

---

## 景别 Shot Types

| 英文术语 | 中文 | 说明 |
|----------|------|------|
| extreme close-up (ECU) | 特写 | 眼睛、手部细节充满画面 |
| close-up (CU) | 近景 | 面部或单个物体 |
| medium close-up (MCU) | 中近景 | 头肩 |
| medium shot (MS) | 中景 | 腰部以上 |
| medium full shot | 中全景 | 膝盖以上 |
| full shot (FS) | 全景 | 完整人体，头到脚 |
| wide shot (WS) | 远景 | 人物+大量环境 |
| extreme wide shot (EWS) | 大远景 | 广阔场景，人物很小 |
| two-shot | 双人镜头 | 两个人物同框 |
| over-the-shoulder (OTS) | 过肩镜头 | 从一个人肩后看另一个人 |
| POV shot | 主观镜头 | 摄像机充当角色的眼睛 |

## 运镜 Camera Movements

| 英文术语 | 中文 | 说明 |
|----------|------|------|
| push-in | 推进 | 镜头向主体靠近（制造紧张/聚焦） |
| pull-back / pullout | 拉远 | 镜头远离主体（揭示规模） |
| dolly | 推轨 | 沿轨道平滑前后/左右移动 |
| truck left/right | 横移 | 镜头水平滑动 |
| pan left/right | 摇镜（水平） | 镜头原地水平旋转 |
| tilt up/down | 摇镜（垂直） | 镜头原地上下旋转 |
| crane up/down | 升降 | 镜头垂直升起或下降 |
| orbit / arc | 环绕 | 镜头绕主体旋转 |
| follow shot / tracking shot | 跟拍 | 镜头跟随移动的主体 |
| handheld | 手持 | 轻微晃动，纪录片质感 |
| Steadicam | 稳定器 | 平滑浮动穿越空间 |
| whip pan | 甩镜 | 极快水平旋转（转场用） |
| zoom in/out | 变焦 | 镜头焦距变化（非物理移动） |

## 角度 Camera Angles

| 英文术语 | 中文 | 说明 |
|----------|------|------|
| eye-level | 平视 | 中性自然视角 |
| low angle | 仰拍 | 向上看主体（力量感、英雄感） |
| high angle | 俯拍 | 向下看主体（脆弱感） |
| bird's eye / top-down | 鸟瞰 | 正上方垂直向下 |
| dutch angle / tilted | 倾斜镜头 | 画面倾斜（不安、紧张） |
| worm's eye | 蚁视 | 极低角度，地面向上看 |

## 焦距与景深 Focus & Depth

| 英文术语 | 中文 | 用法 |
|----------|------|------|
| shallow depth of field | 浅景深 | 背景虚化，突出主体 |
| deep focus | 深焦 | 前后景都清晰 |
| rack focus | 焦点转移 | 从前景切换到背景（或反向） |
| split diopter | 分屈光镜 | 两个平面同时清晰 |
| bokeh | 散景 | 柔和圆形背景光斑 |

## 转场 Transitions

| 英文术语 | 中文 | 说明 |
|----------|------|------|
| cut | 硬切 | 瞬间切换 |
| dissolve / crossfade | 叠化 | 两个画面渐变融合 |
| wipe | 划变 | 一个画面滑过另一个 |
| match cut | 匹配剪辑 | 相似形状/动作间切换 |
| smash cut | 突切 | 突然切换制造冲击 |
| fade to black / white | 淡入淡出 | 渐变到纯色 |
| whip transition | 甩镜转场 | 快速摇镜连接两个场景 |

## 节奏 Pacing

| 英文术语 | 中文 | 用法 |
|----------|------|------|
| slow motion / slow-mo | 慢动作 | 降速播放增加戏剧感 |
| time-lapse | 延时 | 压缩时间，云/人群快速移动 |
| speed ramp | 变速 | 同一镜头内从慢到快（或反向） |
| freeze frame | 定格 | 单帧静止 |
| quick cuts | 快切 | 快速切换镜头增加能量 |
| long take | 长镜头 | 不剪辑的延续镜头 |

## 灯光 Lighting

| 英文术语 | 中文 | 说明 |
|----------|------|------|
| key light | 主光 | 主要光源，定义氛围 |
| rim light / backlight | 轮廓光/背光 | 边缘发光，分离主体和背景 |
| fill light | 补光 | 柔化主光产生的阴影 |
| hero lighting | 英雄光 | 戏剧性产品广告式强高光 |
| volumetric light | 体积光 | 可见光线穿过大气（丁达尔效应） |
| neon lighting | 霓虹光 | 彩色人造光，赛博朋克美学 |
| golden hour | 黄金时刻 | 温暖低角度自然阳光 |
| blue hour | 蓝调时刻 | 冷色黄昏环境光 |
| tungsten warm | 钨丝暖光 | 室内橙暖实景灯光 |
| chiaroscuro | 明暗对比 | 光暗区域的强烈对比 |

## 色彩 Color Palettes

| 英文术语 | 中文 | 情绪 |
|----------|------|------|
| warm palette | 暖色调 | 红/橙/金 — 能量、热情 |
| cool palette | 冷色调 | 蓝/银/青 — 冷静、科技 |
| desaturated / muted | 低饱和 | 写实或压抑 |
| high saturation / vibrant | 高饱和 | 动画、奇幻、广告 |
| monochromatic | 单色 | 单一色相的色调变化 |
| complementary split | 互补色 | 如青橙对比 |
| pastel | 粉彩 | 柔和浅色，温柔梦幻 |

| pixel art | 像素风 | 复古低分辨率方块美学 |

| anime | 日式动画 | 日本动画风格 |
