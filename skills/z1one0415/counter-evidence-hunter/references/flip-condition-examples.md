# 翻转条件模板与跨领域案例

## 一、翻转条件定义

**翻转条件（Flip Condition）** 是指：在某个特定条件成立时，当前主线结论将完全失效或发生方向性反转的结构化描述。

翻转条件不是"可能出错的地方"，而是**明确可触发、可观测、有判断标准**的条件命题。

### 翻转条件模板

```
IF [可观测条件] THEN [主线结论失效/反转]
  - 触发概率: low | medium | high
  - 预估时间窗: [时间范围]
  - 监测指标: [什么信号表明条件正在接近触发]
  - 影响程度: 局部修正 | 方向反转 | 完全失效
```

## 二、10个跨领域真实案例

### 案例 1: 科技 — 先进制程优势论

```
主线结论: 台积电的先进制程领先优势将在2025-2027年持续扩大

翻转条件:
IF 台积电2nm（N2）良率突破预期（>80% D0）且Intel 18A良率追平
THEN 先进制程差距收窄的叙事将取代"差距扩大"的叙事

触发概率: medium
预估时间窗: 2026 Q2 - 2027 Q1
监测指标: Intel Foundry季度良率报告、台积电法说会N2进度指引
影响程度: 方向反转
```

### 案例 2: 金融 — 看空美元论

```

翻转条件:
IF 美联储在失业率未显著上升的情况下意外降息≥50bps（释放经济衰退信号而非正常化信号）
THEN 美元避险逻辑将转变为衰退交易，美元可能快速贬值

触发概率: low

监测指标: FOMC声明措辞变化、非农就业数据连续2个月恶化、国债收益率曲线倒挂加深

```

```
主线结论: GLP-1类药物将成为肥胖治疗的"革命性方案"

IF GLP-1类药物的长期（>5年）随访研究显示严重副作用发生率>5%（如甲状腺肿瘤、胰腺炎）
```

## 三、替代路径支撑 (alternative_supports) 识别方法

### 什么是替代路径支撑

替代路径支撑是指：能够解释同一现象、但逻辑链与主线完全不同的替代解释及其证据。

### 识别方法

**Step 1: 现象-原因解耦**
将主线观察到的现象（A）与主线归因的原因（B→A）分离。然后追问：还有哪些原因（C→A, D→A）同样能解释现象A？

**Step 2: 替代因果链构建**
对每个替代原因，构建完整的因果链，并评估：
- 该因果链在逻辑上是否自洽？
- 是否有实际数据/案例支撑？
- 是否与已知事实矛盾？

**Step 3: 兼容性评估**
替代路径与主线的关系分为三级：
- **contradicts（矛盾）**：替代路径如果成立，主线结论将失效
- **qualifies（限定）**：替代路径成立不否定主线，但限定了主线的适用范围
- **extends（扩展）**：替代路径与主线互补，提供更完整的解释

**Step 4: 证据支撑收集**
为每个通过评估的替代路径收集支撑证据，形成 alternative_supports 输出。

### 示例

```
主线现象: 2024年中国新能源汽车出口量增长40%
主线归因: 中国新能源汽车技术优势（电池、智能驾驶）推动出口

替代路径1: 价格战驱动
  因果链: 国内价格战挤压利润 → 转向海外市场寻求溢价
  证据: 中国国内EV均价下降25% / 海外平均售价高于国内30%
  兼容性: qualifies — 价格因素与技术因素共存

替代路径2: 贸易壁垒窗口期
  因果链: 欧盟反补贴调查尚未落地 → 厂商抢在关税生效前出口
  证据: 2024下半年出口量环比激增 / 欧盟临时关税方案披露前的抢出口行为
  兼容性: qualifies — 时间性因素解释了部分增长
```
