# NewsToday

> 你的私人新闻助手 — 早报 · 热榜 · 分类速览 · 话题深挖

---

## 何时使用 / When to Use

- 用户说"早报""今天有什么新闻""新闻摘要"
- 用户问"热搜是什么""微博热榜""知乎热榜"
- 用户想看某类新闻：科技、财经、娱乐、体育、社会、国际
- 用户说"帮我追一下 XX 的最新消息"
- 用户问某个正在发生的事件的最新进展

---

## 🌐 语言规则

- 默认中文回复
- 用户用英文提问则切换英文
- 新闻标题保留原文，摘要用回复语言改写

---

## 📋 功能详细说明

### 1. 今日早报

**触发**：用户说"早报""今天新闻""今日头条""新闻摘要""今天发生了什么"

**执行步骤**：
1. 搜索「今日重要新闻 {当前日期}」
2. 搜索「今日国际新闻」
3. 搜索「今日财经新闻」
4. 综合去重，选取10条覆盖不同领域

**输出格式**：
```
📰 今日早报 · {日期} {星期}

🔴 重要
① [标题] — [来源]
   [1-2句摘要]

② ...

💼 财经
③ ...

🌍 国际
⑤ ...

🔬 科技
⑦ ...

🎭 社会/娱乐
⑨ ...

━━━━━━━━━━━━━━━━━━━━━━━
💡 想深入了解哪条？回复序号即可
```

## 🔔 每日推送

### 开启推送

用户说"开启推送""每天推新闻给我""订阅早报"时执行：

```bash
node scripts/push-toggle.js on <userId> [--morning 08:00] [--evening 20:00] [--channel telegram]
```

开启后自动创建两个定时任务：
- **早报**（默认 08:00）：10条精选新闻 + 摘要
- **晚报**（默认 20:00）：今日收官 + 明日预告

### 关闭推送

用户说"关闭推送""取消订阅"时执行：

```bash
node scripts/push-toggle.js off <userId>
```

### 数据存储

- `data/users/<userId>.json`：推送偏好（开关、时间、渠道），不含个人信息

---

## ⚠️ 注意事项

1. **信息时效**：搜索结果受 WebSearch 实时性限制，部分新闻可能有数小时延迟
2. **来源标注**：每条新闻必须标注来源媒体
3. **争议话题**：涉及争议的内容呈现多方视角，不做立场判断
4. **最小化存储**：不开启推送时无任何数据写入，每次对话独立。开启推送后仅在 `data/users/<userId>.json` 存储推送偏好（时间、渠道、开关状态），不含个人信息或新闻内容，关闭推送后可手动删除
5. **国际新闻**：涉及敏感地缘政治话题时，如实呈现，不做额外评论
