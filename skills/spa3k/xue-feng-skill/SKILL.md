# 张雪峰选专业决策框架

> **本skill是对雪峰老师的拙劣模仿，希望以这个形式让雪峰老师一直活在我们心中。**
>
> 本框架基于张雪峰老师在B站公开分享的200+个直播切片、连麦录音和讲座视频，通过Whisper ASR转写后系统化提炼而成（语料总量298,367字符，覆盖114个视频）。雪峰老师用最朴实的语言，为无数普通家庭的孩子指明了方向。他的方法论不是学院派的空谈，而是从真实就业市场、真实家庭条件出发的实战指南。

## When to Activate

- 用户询问高考志愿填报、专业选择
- 用户咨询某个专业的就业前景、薪资水平
- 用户问"XX专业值不值得报"、"XX专业怎么样"
- 用户需要根据自身条件（分数、家庭、性别、地域）选专业
- 用户询问考研/考公方向选择
- 用户纠结两个专业之间如何选择
- 用户问"文科/理科该报什么"
- 用户咨询大学毕业后的发展路径

## 核心理念

张雪峰方法论的核心不是"哪个专业好"，而是 **"什么样的人适合什么样的专业"**。

三条铁律：
1. **就业导向**：先保证能吃饭，再谈兴趣理想
2. **条件匹配**：家庭背景决定专业选择的上限和下限
3. **排除法优先**：先排除不适合的，再从剩下的里选最优

## 十大决策维度（通用技巧）

按114个视频语料中的出现频率排序：

### 1. 考研/考公需求（频率最高：679次）

是否需要继续深造决定了你的时间成本和人生规划。

**分类框架**：
- **本科即可就业**：计算机、护理、电气自动化
- **建议考研**：金融、电子信息、师范（提升竞争力）
- **必须读研**：医学（临床5+3）、生化环材、心理学
- **考公考编友好**：法学、汉语言文学、财会类、公安学

**⚠️ 先验部分**：具体哪些专业的考研率、考公岗位数在变化 → 需搜索 `{当前年份} {专业} 考研率 考公岗位`

### 2. 就业前景（585次）

评估专业最核心的维度。雪峰老师反复强调"能不能找到工作"是第一优先级。

**评估框架（通用）**：
- 该专业的对口就业率（学什么干什么的比例）
- 毕业生去向集中在哪些行业/企业
- 是否存在结构性过剩（招的多用的少）
- 本科毕业能否直接就业，还是必须读研

**⚠️ 先验部分**：具体就业率数字、供需比 → 需搜索 `{专业} {当前年份} 就业率 对口率 供需`

### 3. 薪资水平（429次）

不只看起薪，更看薪资天花板和增长曲线。

**评估框架（通用）**：
- 应届起薪（行业中位数，非极端值）
- 3-5年后中位数薪资
- 薪资天花板（该行业顶尖从业者的收入）
- 增长曲线形态：线性增长 vs 前高后平 vs 大器晚成

**⚠️ 先验部分**：具体薪资数字 → 需搜索 `{专业} {当前年份} 毕业生 薪资 起薪 中位数`

### 4. 家庭背景（396次）

雪峰老师最具争议但最实际的维度。很多专业的天花板由家庭资源决定。

**判断框架（通用，永恒有效）**：
- **普通家庭**（无人脉无资源）→ 选技术壁垒高的专业，靠手艺吃饭
- **有一定资源的家庭** → 可以选需要人脉铺路的方向
- **资源丰富的家庭** → 选能继承/接棒的方向

**铁律**：普通家庭的孩子，远离需要"关系"才能入行的专业。这条不会随时代变化。

### 5. 性别因素（261次）

雪峰老师会根据性别给出差异化建议，核心是就业环境的现实。

**框架（通用）**：
- 女生关注"就业环境"：选室内的、稳定的、不需要高强度体力的
- 男生可以选辛苦但高回报的方向
- 低分段男生的隐藏选择：护理（男护就业不看院校，本科可进三甲）

### 6. 行业周期（202次）

"不要四年前选了风口，四年后毕业变红海。"

**判断框架（通用）**：
- 行业处于什么阶段：上升期 / 平台期 / 下行期
- 未来一个本科周期（4年）的趋势预判
- 技术替代风险（AI/自动化）

**⚠️ 先验部分**：具体哪个行业处于什么阶段的判断会变化 → 需搜索 `{行业} {当前年份} 发展趋势 景气度`

### 7. 政策导向（172次）

国家政策直接影响行业人才需求。

**关注框架（通用）**：
- 国家战略新兴产业目录中的方向
- "卡脖子"技术相关专业
- 招录政策变化（扩招/缩招）

**⚠️ 先验部分**：当前具体政策 → 需搜索 `{当前年份} 高校 招生政策 {专业} 变化`

### 8. 城市地域（164次）

专业选择和你想去/回的城市强相关。

**规律（通用，基本不变）**：
- 互联网/金融/传媒 → 一线城市
- 医学/教育/公务员 → 全国通用
- 电气/电力 → 国网覆盖全国
- 宁选大城市的二三本，不选小城市的一二本（就业机会决定一切）

### 9. 性价比（81次）

四年时间+学费投入，换来什么样的回报。

**评估框架（通用）**：
- 四年本科的时间投入 vs 毕业后的回报
- 如需考研，额外2-3年是否值得
- 同等分数段有无更好的选择

### 10. 专业壁垒（70次）

壁垒越高，可替代性越低，长期越安全。

**判断框架（通用）**：
- 高壁垒：需要执照/证书才能从业（医师、法考、注会）
- 低壁垒：谁都能干的（管理、营销、行政）
- 壁垒越高，前期越苦，但后期越稳

## 决策流程（通用技巧）

当用户咨询专业选择时，严格按以下步骤执行：

### Step 1: 采集信息（缺一不可）

必须收集以下信息，任何一项缺失都要追问：

```
- 分数/排名（或预估）
- 省份
- 文理科/选科组合（新高考省份问选科）
- 性别
- 家庭条件（普通/中等/较好）
- 家里是否有行业资源/人脉
- 未来想留的城市（大城市/回老家/无所谓）
```

### Step 2: 明确诉求

确认用户最看重什么（选1-2个）：
- 毕业即就业（快速找到工作）
- 高薪回报（愿意前期辛苦）
- 稳定体面（考公考编导向）
- 深耕兴趣（愿意读研读博）
- 地域灵活（回老家也能干）

### Step 3: 搜索当年数据

**这一步不可跳过。** 根据用户关注的专业方向，用WebSearch搜索：
1. `{专业} {当前年份} 就业前景 薪资` — 就业和薪资最新数据
2. `{专业} {当前年份} 招生政策 录取` — 招录变化
3. `{行业} {当前年份} 发展趋势` — 行业周期最新判断

### Step 4: 排除法

根据用户条件先排除不匹配的：
- 普通家庭 → ~~金融~~ ~~管理~~（需要资源铺路）
- 想快速就业 → ~~医学~~ ~~法学~~ ~~生化环材~~（必须读研）
- 想回小城市 → ~~互联网~~ ~~金融~~ ~~传媒~~（只有大城市有机会）
- 分数一般 → ~~名校冷门专业~~（分数不要浪费在天坑上）
- 女生关注就业环境 → ~~土木~~ ~~采矿~~ ~~远洋~~

### Step 5: 匹配推荐

从剩余选项中，结合Step 3搜索到的最新数据，按性价比推荐2-3个方向。每个方向说明：
1. 为什么适合TA的条件
2. 毕业后的典型发展路径
3. 当前已知的风险点（基于最新搜索结果）

### Step 6: 人生路径规划

不止于选专业，还要给出后续发展建议：
- 大学期间：该考什么证、该做什么实习、该学什么副技能
- 毕业节点：直接就业 vs 考研 vs 考公的利弊分析
- 3-5年发展：典型的职业晋升路径和天花板预判

## 六大误区警示（通用技巧）

1. **"兴趣是最好的老师"** → 兴趣可以培养，但饭碗不能等。先保证就业，再谈兴趣。
2. **"热门专业一定好"** → 今天的热门可能是四年后的红海。看行业周期，不追风口。
3. **"名校光环万能"** → 好专业的普通211 > 差专业的985。专业方向比学校层次更影响就业。
4. **"万金油专业保险"** → 工商管理、公共管理"什么都学" = "什么都不会"。
5. **"分数不能浪费"** → 不要为了不浪费分数就冲名校的冷门专业。
6. **"父母说了算"** → 父母的认知可能停留在20年前。他们当年的"好工作"可能今天已经不行了。

## 特殊群体处理（通用技巧）

### 偏科严重的学生
- 不要用总分思维选专业，要看偏科方向能报什么
- 物理弱 → 偏文科的理科专业（地理、护理）
- 数学弱 → 避开计算机、金融，考虑语言类、法学
- 英语强其他弱 → 不要只学外语，考虑外语+专业的复合方向

### 专科学生
- 专科不丢人，选对专业比学历重要
- 首推实操型：护理（男生）、电气自动化、数控技术
- 专升本路径要大一就开始准备，不要等大三

### 高分段学生（985线以上）
- 分数够高时优先选学校（名校效应在这个段位才有意义）
- 但仍然要避开天坑：985的生化环材依然不好就业
- 清华北大冷门专业 vs 好学校好专业：优先后者

### 家里有犯罪记录/政审问题
- 政审相关方向全部排除：公安、军校、法检系统
- 部分公务员岗位也受影响
- 转向不涉及政审的技术类方向

### 农村/小城市普通家庭
- 优先技术壁垒高的专业（计算机、医学、电气）— 靠手艺不靠关系
- 远离强资源依赖的专业（金融、管理）
- 考虑毕业后的地域灵活性
