# 家财险产品对比参考指南

## 一、主要险种品类速查

### 1. 房屋主体险
**保障范围（标准）：** 火灾、爆炸、雷击、暴风、暴雨、洪水、冰雹、地面下陷、空中坠落物体
**常见除外：** 战争、核辐射、自然磨损、人为故意破坏、空置超过30天未通知保险公司

### 2. 室内装修险
**保障范围：** 固定装修（地板、墙面、天花板、门窗）因承保灾因损毁
**注意：** 部分产品含水管爆裂导致的装修损失，部分不含，需重点核查

### 3. 室内财产险
**保障范围：** 家用电器、家具、衣物等动产
**子类：**
- 普通家庭财产：按打包方式承保，保额有上限
- 贵重物品单独申报：珠宝、名表、艺术品等，需提供购买凭证

### 4. 家用电器安全险
**保障范围：** 家用电器因意外碰撞、电流异常、操作失误导致的损坏
**与室内财产险区别：** 财产险通常只保承保灾因，电器险可保"意外损坏"

### 5. 居家责任险
**保障范围：** 被保险人因以下原因对第三方造成的人身或财产损失：
- 所有或使用的房屋设施损坏（如管道爆裂）
- 家庭成员的过失行为
- 饲养宠物伤人（部分产品含）
**核心指标：** 每次事故赔偿限额 / 年累计赔偿限额

### 6. 家政人员责任险（雇主责任）
**保障范围：** 家政人员在工作中发生意外伤亡，由雇主（投保人）承担的赔偿责任
**适用人群：** 雇佣保姆、月嫂、钟点工等

### 7. 宠物责任险
**保障范围：** 宠物咬伤他人或损坏他人财物的赔偿责任
**注意：** 通常仅限犬类，部分产品对烈性犬不承保

## 二、产品对比核心维度

### 维度1：保障责任（保哪些）
| 对比项 | 关注重点 |
|--------|---------|
| 主险灾因范围 | 是否含洪水/内涝（部分产品不含）|
| 水暖管爆裂 | 是否作为主险或附加险，是否含修复费用 |
| 盗窃险 | 是否要求有门锁/防盗设施作为前提 |
| 电器损坏 | 是否仅限意外，还是含老化损坏 |

### 维度2：保障额度（赔多少）
| 对比项 | 关注重点 |
|--------|---------|
| 是否分项保额 | 房屋、装修、财产各自独立保额 vs 共享总额 |
| 免赔额 | 每次绝对免赔额（如500元）or 免赔率（损失×10%）|
| 责任险限额 | 单次/年度是否满足实际赔偿风险 |
| 贵重物品限额 | 单件限额是否满足实际价值 |

### 维度3：理赔服务（好不好赔）
| 对比项 | 关注重点 |
|--------|---------|
| 理赔流程 | 线上/线下、单证要求复杂程度 |
| 快赔服务 | 是否有小额快赔通道（如1万以内） |
| 增值服务 | 管道抢修、临时住宿安排、24小时救援 |
| 公司评级 | 偿付能力评级、投诉率排名 |

### 维度4：投保条件（能不能保）
| 对比项 | 关注重点 |
|--------|---------|
| 房龄限制 | 多数产品对30年以上房龄有限制 |
| 建筑结构 | 砖木结构通常为拒保或加费 |
| 地域限制 | 部分地区台风/洪水风险区可能拒保或除外 |
| 使用性质 | 经营性用途需明确告知 |

## 三、常见高风险场景与应对建议

### 场景1：老旧小区（房龄>20年）
- 主要风险：水管/暖气管老化爆裂、电路老化火灾、隔音墙渗水
- 重点配置：**水暖管爆裂险 + 居家责任险**（漏水殃及邻居）
- 注意：部分产品对房龄有限制，需提前核查投保资格

### 场景2：台风/暴雨高发地区（广东/福建/海南/浙江）
- 主要风险：台风直接损毁、暴雨内涝、洪水倒灌
- 重点配置：**主险含洪水/台风 + 确认无等待期**
- 注意：台风季投保有等待期（通常48-72小时），需提前配置

### 场景3：有贵重物品（总价值>5万元）
- 主要风险：盗窃、意外损毁、火灾损毁
- 重点配置：**贵重物品单独申报险 + 确认盗窃险条款**
- 注意：珠宝、名表、收藏品需单独列明，保留购买凭证

### 场景4：出租房（房东视角）
- 主要风险：租客损坏房屋、租客行为导致邻居损失
- 重点配置：**房屋主体险 + 装修险 + 居家责任险**
- 注意：部分产品在出租状态下需特别告知，否则可能拒赔

### 场景5：租房客（租客视角）
- 主要风险：自身财产损毁/盗窃、因过失损坏房东设施
- 重点配置：**室内财产险 + 居家责任险**（含对房东财产的赔偿责任）
- 注意：租客通常无需也无法为房屋主体投保

### 场景6：有家政人员
- 主要风险：家政人员工作中受伤、家政人员造成第三方损失
- 重点配置：**雇主责任险（家政险）**
- 注意：长期固定保姆与临时钟点工的保障需求不同

## 四、常见问题与专业解答

**Q：保额应该怎么定？**
A：房屋主体保额参考"重置成本"（重建同等房屋的费用），而非市场价。装修保额以实际装修成本为参考，室内财产以实际购置价的80%以上为宜。保额过低会导致"不足额保险"，赔付时按比例打折。

**Q：家财险和意外险有什么区别？**
A：家财险保的是"物"（房屋、财产），意外险保的是"人"（被保险人的人身）。家财险中的责任险保障因自家财产问题殃及他人的赔偿责任。

**Q：租客需要买家财险吗？**
A：需要，但侧重不同。租客无需为房屋主体投保，但自己的室内财产（电器、衣物、电脑等）有保障需求，同时也应考虑居家责任险（如因自己失误导致水管破裂损坏房东设施或殃及邻居）。

**Q：有统保（物业统保）还需要自己买吗？**
A：统保通常保额很低（多为象征性保障），保障范围有限，建议核查统保内容后判断是否存在保障缺口，缺口部分自行补充。
