# PDF Processing

PDF文档处理工具集，支持文本提取、表格提取、表单填写、文档合并等操作。

## 功能列表

- 提取 PDF 文本内容
- 提取 PDF 表格数据
- 填写 PDF 表单
- 合并多个 PDF 文件

## 使用方法

当需要对 PDF 文件进行操作时使用此技能。

## 依赖

需要安装 pdfplumber：
```bash
pip install pdfplumber pypdf
```

## 脚本说明

将相关处理脚本放到 `scripts/` 目录。
