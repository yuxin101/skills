# 📖 诡世命簿

**回合制异世界人生模拟游戏引擎**

穿越到随机生成的异世界，从出生开始，每回合做出选择，经历完整一生。人生结束时，自动生成属于你的短篇小说。

---

## ✨ 核心特性

| 特性 | 说明 |
|------|------|
| 🌍 随机世界观 | 7种世界类型，每次都是新冒险 |
| 👤 随机出身 | 种族、天赋、家庭，命运从出生开始 |
| ✏️ 命名环节 | 可自定义角色名，或让命运决定 |
| ⏳ 回合制推进 | 6个人生阶段，每回合一年 |
| 🎲 事件选择 | 你的选择塑造命运走向 |
| 💰 财产系统 | 装备、背包、金币、房产 |
| 📖 故事记录 | 自动记录你的人生故事 |
| 🖼️ 图片生成 | 角色卡、场景图（可选） |
| 📚 终局成书 | 人生结束生成完整短篇小说 |
| 💾 存档系统 | 随时存档读档 |

---

## 🎮 快速开始

### 开始新游戏

说：**"翻开命簿"** 或 **"开始新游戏"**

系统会：
1. 随机生成异世界世界观
2. 随机生成你的出身
3. 让你为角色命名
4. 进入人生第一回合

### 继续游戏

| 命令 | 说明 |
|------|------|
| `继续` / `下一回合` | 推进时间 |
| `状态` / `属性` | 查看角色状态 |
| `故事` | 查看人生故事 |
| `存档` | 保存游戏 |

## 🌏 世界类型

| 类型 | 风格 | 特点 |
|------|------|------|
| 奇幻中世纪 | 剑与魔法 | 骑士、法师、巨龙 |
| 东方玄幻 | 修仙问道 | 宗门、灵根、渡劫 |
| 蒸汽朋克 | 机械齿轮 | 蒸汽、发明、冒险 |

| 废土末世 | 文明废墟 | 生存、变异、重建 |

## 👤 人生阶段

| 阶段 | 年龄 | 特点 |
|------|------|------|
| 婴儿期 | 0-3岁 | 天赋觉醒，被动事件 |
| 童年期 | 4-12岁 | 学习成长，早期社交 |
| 少年期 | 13-17岁 | 教育选择，初恋叛逆 |
| 青年期 | 18-35岁 | 职业婚姻，冒险闯荡 |
| 中年期 | 36-55岁 | 家庭事业，传承培养 |
| 老年期 | 56岁+ | 回顾人生，遗产终局 |

## 📚 终局成书

人生结束时，自动生成完整短篇小说：

**小说结构：**
- 序章：世界观背景、出生
- 第1-6章：各人生阶段
- 尾声：死亡、遗产、后世评价
- 附录：属性记录、成就列表

**可选风格：**
| 风格 | 说明 |
|------|------|
| 详细版 | 所有事件细节 |
| 精简版 | 只保留关键事件 |
| 传奇版 | 突出戏剧性 |
| 编年史版 | 客观历史记录 |

## 📋 完整命令

### 游戏控制

| 命令 | 说明 |
|------|------|
| `新游戏` / `开始` | 开始新的人生 |
| `继续` / `下一回合` | 推进时间 |
| `跳过N回合` | 快进N个回合 |

### 信息查看

| 命令 | 说明 |
|------|------|
| `状态` / `属性` | 显示当前状态 |
| `故事` | 显示故事记录 |
| `世界` / `世界观` | 显示世界信息 |
| `人物 [名字]` | 显示角色卡 |
| `关系` | 显示人际关系 |
| `背包` | 显示物品背包 |
| `财产` | 显示金币房产 |

### 存档管理

| 命令 | 说明 |
|------|------|
| `存档` | 保存游戏 |
| `存档为 [名字]` | 命名存档 |
| `读档 [名字]` | 加载存档 |
| `存档列表` | 显示所有存档 |

### 图片生成

| 命令 | 说明 |
|------|------|
| `生成角色卡` | 生成角色卡图片 |
| `生成场景` | 生成当前场景图片 |

### 小说生成

| 命令 | 说明 |
|------|------|
| `成书` | 人生结束后生成小说 |
| `重新成书` | 重新生成小说 |
| `查看小说` | 显示生成的小说 |
| `导出小说` | 导出为文件 |

## 📁 文件结构

```
诡世命簿/
├── SKILL.md                    # 技能主文件
├── README.md                   # 本文件
├── references/
│   ├── world_gen.md            # 世界观生成规则
│   ├── event_system.md         # 事件系统规则
│   └── time_system.md          # 时间系统规则
├── assets/
│   ├── templates/
│   │   ├── character_template.md   # 角色卡模板
│   │   └── event_template.md       # 事件模板
│   └── prompts/
│       ├── image_prompt.md     # 图片提示词
│       └── story_prompt.md     # 故事生成规则
├── saves/                      # 存档目录
│   ├── template.json           # 存档模板
│   └── current.json            # 当前存档
└── stories/                    # 故事存档
    └── [角色名]/
        ├── story.md            # 故事文本
        └── images/             # 生成的图片
```
