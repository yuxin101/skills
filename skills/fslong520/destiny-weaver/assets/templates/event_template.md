>

## 判定事件格式

### 判定成功

| 判定 | 结果 |
|------|------|
| 属性 | 智力 |
| 需求 | ≥ 15 |
| 你的值 | 15 ✓ |
| **结果** | **成功** |

### 判定失败

| 判定 | 结果 |
|------|------|
| 属性 | 力量 |
| 需求 | ≥ 10 |
| 你的值 | 7 ✗ |
| **结果** | **失败** |
