# 故事生成规则

## 故事结构

### 章节组织

```
# [角色名]的一生

## 序章 · 命运的起点
[世界观简介 + 出生]

## 第一章 · [阶段名]（[年龄范围]）
### 第X年 · [事件标题]
[事件描述]

## 第二章 · [阶段名]（[年龄范围]）
...

## 尾声 · [结局标题]
[人生总结]
```

### 阶段命名建议

| 阶段 | 年龄 | 命名建议 |
|------|------|----------|
| 婴儿期 | 0-3岁 | 晨曦初露、懵懂时光、生命的起点 |
| 童年期 | 4-12岁 | 无忧岁月、纯真年代、成长足迹 |
| 少年期 | 13-17岁 | 青葱岁月、抉择时刻、青春序曲 |
| 青年期 | 18-35岁 | 风华正茂、逐梦年华、人生华章 |
| 中年期 | 36-55岁 | 岁月沉淀、责任与担当、成熟季节 |
| 老年期 | 56岁+ | 夕阳余晖、归途漫漫、岁月如歌 |

## 故事风格

### 叙事视角

- **主要视角**：第三人称全知视角
- **情感描写**：第一人称内心独白（可选）
- **重大事件**：可切换第一人称

### 语言风格

| 世界类型 | 语言风格 | 示例 |
|----------|----------|------|
| 奇幻中世纪 | 古典优雅、诗意 | "那一年，命运的齿轮悄然转动..." |
| 东方玄幻 | 空灵飘逸、古风 | "是年，天降异象，风云变幻..." |
| 蒸汽朋克 | 机械感、工业风 | "齿轮转动，蒸汽升腾，时代在变迁..." |
| 黑暗奇幻 | 压抑、阴郁 | "黑暗中，唯一的光芒也渐渐熄灭..." |
| 异能都市 | 现代、简洁 | "那个夏天，一切都不一样了..." |

### 描写原则

1. **场景有画面感** — 描写颜色、声音、气味
2. **情感有共鸣** — 描写内心波动、选择困境
3. **转折有张力** — 描写冲突、悬念
4. **结局有回响** — 描写结果、影响

## 单回合故事模板

### 事件描述

```markdown
### 第[回合]年 · [事件标题]

那是[角色名][年龄]岁那年的[季节]。

[场景描写：时间、地点、氛围]

[事件发生：发生了什么]

[角色反应：内心感受、外在行动]

[选择记录：玩家做出了什么选择]

[结果描述：选择带来的后果]

[影响说明：这个选择的影响]
```

### 示例

```markdown
### 第16年 · 学院的邀请

那是艾莉丝十六岁那年的初夏。

魔法学院的信鸽在一个阳光明媚的下午飞到了小镇。那封印着学院徽章的信件，
在小镇引起了不小的轰动——这是十年来的第一次。

艾莉丝的手微微颤抖着接过信件。母亲眼中的骄傲与忧虑交织，父亲则是沉默地
站在一旁，手上还沾着商铺的账本墨迹。

"去吧，"父亲最终开口，声音沙哑，"这是你的机会。"

艾莉丝选择了接受邀请。那一刻，她的人生轨迹被改写了。

学费是个沉重的负担，父亲卖掉了半个商铺，母亲的戒指也换了主人。但艾莉丝
知道，这是她改变命运的唯一机会。

**"我会让一切值得。"** 她在心里默默许下承诺。

[智力+3，获得"学院学生"身份，金币-200，父母关系-5，获得母亲的戒指作为遗物]
```

## 阶段总结模板

### 人生阶段转换时

```markdown
### · [阶段名] ·

[阶段总结段落]

---

**[角色名]的[阶段]回顾**

| 年龄 | 大事记 |
|------|--------|
| [年龄] | [事件] |
| [年龄] | [事件] |
| ... | ... |

**属性成长**
- 力量：[起始值] → [结束值]
- 敏捷：[起始值] → [结束值]
- 智力：[起始值] → [结束值]
- 魅力：[起始值] → [结束值]
- 幸运：[起始值] → [结束值]（固定）

**获得技能**
- [技能列表]

**建立关系**
- [关系列表]

**阶段评价**
[根据选择生成评价：这阶段的角色是怎样的？取得了什么？失去了什么？]

---
```

### 示例

```markdown
### · 青葱岁月 ·

十六岁到二十岁，这是人生中最美好的年华。艾莉丝在魔法学院度过了
她的少年时代——学习、成长、遇见爱情、面对分离。

---

**艾莉丝的少年期回顾**

| 年龄 | 大事记 |
|------|--------|
| 16岁 | 进入魔法学院 |
| 17岁 | 遇见一生的挚友艾拉 |
| 18岁 | 在学院比武中受伤 |
| 19岁 | 发现自己的特殊天赋 |
| 20岁 | 学院毕业，面临人生抉择 |

**属性成长**
- 力量：8 → 9（战斗训练）
- 敏捷：10 → 12（魔法练习）
- 智力：16 → 20（学院学习）
- 魅力：12 → 14（社交活动）
- 幸运：10（固定）

**获得技能**
- 火焰魔法（精通）
- 治疗术（熟练）
- 古代文献解读（入门）

**建立关系**
- 艾拉（挚友/恋人）
- 马格努斯教授（导师）
- 学院同学若干

**阶段评价**
这是一个求学的时代。艾莉丝展现出非凡的魔法天赋，但也因为太过专注于
学业而错过了很多与家人相处的时光。父亲的商铺最终倒闭，母亲的健康也
开始走下坡路。学业与亲情，她似乎总是难以两全。

---
```

## 结局模板

### 死亡结局

```markdown
## 尾声 · [结局标题]

[死亡场景描写]

[人生回顾]

[遗产/影响]

---

**[角色名]的一生**

**出生**：[年份]，[地点]
**去世**：[年份]，[地点]，[年龄]岁
**死因**：[原因]

**人生评价**：[一句话总结]

**主要成就**：
- [成就1]
- [成就2]
- ...

**遗产**：
- 子女：[数量]人
- 财富：[数量]
- 声望：[级别]
- 影响：[描述]

**后世评价**：
[根据一生选择生成：史书/人们如何评价这个角色]

**传记评分**：[根据选择丰富度打分]

---

*[角色名]的故事到此结束。但在[世界名]的传说中，[他/她]的名字将被永远铭记。*
```

### 不同死亡类型的结局风格

| 死亡类型 | 结局风格 |
|----------|----------|
| 自然老死 | 平静、回顾、温馨 |
| 战死 | 悲壮、英勇、豪迈 |
| 疾病死 | 遗憾、告别、释然 |
| 意外死亡 | 突然、震惊、悲伤 |
| 被谋杀 | 悬疑、愤怒、不平 |
| 牺牲 | 伟大、无私、敬仰 |

### 结局评价标准

```
评分维度：
1. 人生完整度（是否经历了各阶段重要事件）
2. 选择多样性（是否有各种类型的选择）
3. 关系丰富度（是否建立了多样的人际关系）
4. 成就达成度（是否完成了有意义的事情）
5. 故事独特性（是否有独特的经历）

总分：S/A/B/C/D
```

## 故事生成流程

### 每回合结束时

```
1. 记录事件标题、内容
2. 记录玩家选择
3. 记录结果影响
4. 生成事件描述段落
5. 追加到故事文件
```

### 阶段转换时

```
1. 汇总该阶段所有事件
2. 生成阶段总结
3. 计算属性变化
4. 列出重大事件
5. 生成阶段评价
6. 追加到故事文件
```

### 游戏结束时

```
1. 汇总人生所有事件
2. 生成死亡场景
3. 生成人生总结
4. 计算人生评分
5. 生成后世评价
6. 生成结局图
7. 保存完整故事文件
```

## 故事文件格式

### 存档格式

```

└── [角色名]_[世界名]_[日期]/
    ├── story.md           # 完整故事
    ├── data/
    │   ├── world.json     # 世界数据
    │   ├── player.json    # 角色数据
    │   └── history.json   # 事件历史

        ├── character_*.png
        └── scene_*.png
```
