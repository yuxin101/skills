# 小说生成规则

## 核心机制

**静默生成原则**：
- 小说是后台产物，不干扰游戏体验
- 每回合结束后，AI自动将剧情转化为小说段落
- 玩家可通过「小说」命令查看已生成的内容
- 人生结束时，自动整合为完整小说

## 写入时机

| 时机 | 写入内容 |
|------|----------|
| 世界观生成后 | 序章：世界观背景、时代描述 |
| 选择角色后 | 第一章开头：出生场景 |
| 每回合结束 | 对应章节的剧情段落 |
| 人生阶段结束 | 阶段总结段落 |
| 人生结束 | 尾声：结局、遗产、后世评价 |

## 小说结构

| 章节 | 内容 | 对应阶段 |
|------|------|----------|
| 序章 | 世界观背景 | 世界生成后 |
| 第1章 | 婴儿期 | 0-3岁 |
| 第2章 | 童年期 | 4-12岁 |
| 第3章 | 少年期 | 13-17岁 |
| 第4章 | 青年期 | 18-35岁 |
| 第5章 | 中年期 | 36-55岁 |
| 第6章 | 老年期 | 56岁+ |
| 尾声 | 结局、遗产 | 人生结束 |

## 小说风格

根据世界观自动匹配：

| 世界类型 | 小说风格 |
|----------|----------|
| 东方玄幻 | 古风优雅，意境深远，武侠味 |
| 西幻中世纪 | 史诗感，古典叙事 |
| 诡秘克苏鲁 | 悬疑压抑，氛围描写重 |
| 科幻未来 | 简洁硬朗，技术感 |
| 异能都市 | 现代快节奏，都市感 |
| 废土末世 | 苍凉粗砺，末世风 |
| 蒸汽朋克 | 机械浪漫，工业感 |
| 武侠江湖 | 江湖气，侠义风 |

## 写作要点

### 叙事视角

- 第三人称全知视角
- 聚焦主角，但可描写其他角色
- 适当的环境描写

### 章节开头

每章开头用一句话概括：

```markdown
## 第X章 章节名

> {章节引言/开篇句}

{正文内容}
```

### 事件转化

将游戏事件转化为小说段落：

| 游戏事件 | 小说处理 |
|----------|----------|
| 判定成功 | 描写成功的过程，展现能力 |
| 判定失败 | 描写挫折，增添戏剧性 |
| 获得物品 | 自然融入剧情，不生硬 |
| NPC对话 | 保留关键对话，增加描写 |
| 选择结果 | 展现选择的影响 |

### 示例转化

**游戏输出：**
```
> **📜 奇怪的旅人**
>
> 一个披着黑袍的旅人在你家门口停下...
>
> 「小朋友，这个护身符送给你...」
```

**小说段落：**
```
那年的冬天格外寒冷。一个披着黑袍的旅人在门前停下，
他的面容藏在兜帽的阴影里，只能看见一双深邃的眼睛。

「小朋友，」他蹲下身，从怀中取出一枚古旧的护身符，
「这个...送给你。」

那是我第一次感到命运的重量。
```

## 存储格式

```markdown
# 《{剧本名}》

## 序章

{世界观背景}

## 第1章 {章节名}

{婴儿期内容}

## 第2章 {章节名}

{童年期内容}

...
```

## 质量控制

- 每个事件至少转化为一个自然段
- 对话要自然，不生硬
- 环境描写要与世界观契合
- 人物性格要前后一致
- 重要选择要有体现
