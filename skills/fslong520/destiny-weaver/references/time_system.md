# 时间系统规则

## 回合设计

### 时间单位

| 人生阶段 | 1回合 = | 说明 |
|----------|---------|------|
| 婴儿期 | 1年 | 0-3岁，快速成长 |
| 童年期 | 1年 | 4-12岁，稳定成长 |
| 少年期 | 1年 | 13-17岁，重要选择 |
| 青年期 | 1年 | 18-35岁，核心人生 |
| 中年期 | 1年 | 36-55岁，稳定期 |
| 老年期 | 1年 | 56岁+，衰退期 |

### 特殊时间

- **关键时刻**：单次重要事件可单独成回合（如战斗、重大抉择）
- **时间跳跃**：玩家可请求跳过N回合

## 世界时间

### 年份计算

```
世界年 = 初始年 + 玩家年龄
```

### 季节系统（可选）

| 季节 | 事件影响 |
|------|----------|
| 春 | 生育事件+20%，新开始事件 |
| 夏 | 冒险事件+20%，旅行事件 |
| 秋 | 收获事件+20%，经济事件 |
| 冬 | 疾病事件+20%，休养生息 |

### 节日系统（可选）

根据世界观生成节日：
- 国庆日
- 宗教节日
- 季节节日
- 个人纪念日

## 时间流逝效果

### 属性变化

| 阶段 | 力量 | 敏捷 | 智力 | 魅力 | 幸运 |
|------|------|------|------|------|------|
| 婴儿期 | +0~1 | +0~1 | +0~2 | +0~1 | 固定 |
| 童年期 | +1~2 | +1~2 | +1~3 | +0~2 | 固定 |
| 少年期 | +1~3 | +1~3 | +2~4 | +1~2 | 固定 |
| 青年期 | +0~2 | +0~2 | +1~3 | +0~2 | 固定 |
| 中年期 | -1~+1 | -1~+1 | +0~2 | +0~1 | 固定 |
| 老年期 | -1~0 | -1~0 | -1~+1 | -1~+1 | 固定 |

### 技能衰减

```
未使用技能每年衰减：-5%
正在使用技能每年提升：+2~5%
```

### 关系变化

```
未维护关系每年衰减：亲密度 -5
定期互动关系每年提升：亲密度 +2~5
```

## 世界发展

### 每回合世界更新

1. **推进年份**
2. **更新国家状态**：
   - 人口变化
   - 经济波动
   - 政治局势
3. **触发世界事件**（按概率）
4. **更新NPC状态**：
   - 年龄增长
   - 位置变化
   - 关系变化

### 世界事件触发

```python
def check_world_events(world):
    events = []
    
    # 周期性事件
    if world.year % 100 == 0:
        events.append("世纪节点事件")
    
    # 随机事件
    if random() < 0.05:  # 5%概率
        events.append(generate_random_world_event())
    
    # 条件事件
    if world.tension > 80:
        if random() < 0.3:  # 30%概率
            events.append("战争爆发")
    
    return events
```

## 死亡系统

### 死亡触发

| 触发条件 | 概率 |
|----------|------|
| 年龄 >= 70 | 每年5% |
| 年龄 >= 80 | 每年10% |
| 年龄 >= 90 | 每年20% |
| 年龄 >= 100 | 每年30% |
| 重大事件 | 事件决定 |
| 战斗失败 | 事件决定 |
| 疾病 | 事件决定 |

### 死亡类型

- 自然老死
- 战死
- 疾病死
- 意外死亡
- 被谋杀
- 牺牲

## 快进系统

### 跳过回合

玩家可以说：
- "跳过1回合" → 快进1年
- "跳到成年" → 快进至18岁
- "跳到X岁" → 快进至指定年龄

### 快进处理

```
1. 计算跳过年数
2. 生成期间主要事件摘要
3. 更新属性和关系
4. 更新世界状态
5. 显示快进总结
```
