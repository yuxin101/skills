# 角色生成规则

## 生成规则

新游戏开始时，**必须生成3个不同的角色选项**：

1. 三个角色在世界观内**种族、出身、天赋各不相同**
2. 每个角色都要**完整描述性别、外貌、属性、特质**
3. 玩家选择后，进入该角色的人生

## 角色属性生成

### 属性范围

| 属性 | 最低 | 最高 | 说明 |
|------|------|------|------|
| STR 力量 | 3 | 18 | 物理战斗能力 |
| DEX 敏捷 | 3 | 18 | 闪避和灵巧 |
| INT 智力 | 3 | 18 | 学习和魔法 |
| CHA 魅力 | 3 | 18 | 社交和领导 |
| LUK 幸运 | 1 | 20 | 随机事件概率 |

### 属性评价

| 数值 | 评价 |
|------|------|
| 1-5 | 极弱/愚笨 |
| 6-8 | 较弱/平庸 |
| 9-11 | 普通/一般 |
| 12-14 | 良好/出色 |
| 15-17 | 优秀/天才 |
| 18-20 | 极限/传说 |

### 属性分配方式

- **随机分配**：15+2d6（基础15加2个d6随机）
- **倾向分配**：根据出身调整（如战士家庭力量+3）
- **总属性控制**：总属性值60-75之间

## 特质生成

### 特质类型

| 类型 | 图标 | 说明 |
|------|------|------|
| 正面特质 | ✓ | 有益效果 |
| 负面特质 | ✗ | 有害效果 |
| 中性特质 | ◆ | 双面效果 |

### 特质示例

**正面特质：**

| 特质 | 效果 |
|------|------|
| 灵识觉醒 | 天生可感知超自然存在 |
| 过目不忘 | 学习速度+30% |
| 铁意志 | SAN值上限+20，抵抗精神攻击+30% |
| 猎妖天赋 | 对妖魔伤害+20% |
| 通灵之眼 | 可与灵体交流 |
| 人见人爱 | 社交判定+2 |

**负面特质：**

| 特质 | 效果 |
|------|------|
| 妖血诅咒 | 满月之夜san值-10，被视为异类 |
| 厄运体质 | 随机负面事件概率+20% |
| 创伤后应激 | 特定场景san值-10，闪避-2 |
| 駑钝 | 学习速度-20% |
| 体弱 | 生命值上限-20% |

**中性特质：**

| 特质 | 效果 |
|------|------|
| 灵体缠身 | 常有灵体跟随，有时帮忙有时捣乱 |
| 命运之子 | 幸运事件+30%，但也会吸引危险 |

## 外貌生成

### 必须包含

- 性别
- 发色、发长
- 眼睛颜色、特殊特征
- 肤色
- 身材特征
- 特殊标记（疤痕、胎记等）

### 种族特征

| 种族 | 典型外貌特征 |
|------|-------------|
| 人类 | 多样化 |
| 精灵 | 尖耳、俊美、长发 |
| 矮人 | 矮壮、胡须、粗糙 |
| 兽人 | 兽耳、兽尾、野性 |
| 龙裔 | 鳞片、竖瞳、犄角 |
| 混血 | 双族特征混合 |

## 出身生成

### 出身类型

| 类型 | 说明 |
|------|------|
| 贵族 | 资源丰富，地位高 |
| 平民 | 普通，资源一般 |
| 商人 | 较富裕，社交广 |
| 学者 | 智力倾向，知识多 |
| 战士 | 力量倾向，战斗强 |
| 猎人 | 敏捷倾向，野外强 |
| 贫民 | 资源少，韧性强 |
| 孤儿 | 无人照管，自由度高 |

### 出身影响

- 起始资源
- 起始技能
- 社会关系
- 初始属性倾向

## 角色平衡原则

三个角色应该：

1. **定位不同**：一个战斗型、一个智力型、一个均衡型
2. **各有优劣**：没有绝对强的角色
3. **风格各异**：满足不同玩家偏好
4. **契合世界**：符合世界观设定
