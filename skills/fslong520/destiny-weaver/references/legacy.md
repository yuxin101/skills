# 代代相传规则

## 核心机制

玩家可以「续缘」之前的剧本，在同一个世界继续游玩：
- 时间已过去若干年
- 前代角色成为NPC
- 世界根据前代行为演进

---

## 续缘流程

### 1. 选择剧本

玩家输入「续缘 [剧本名]」，系统加载：
- 世界数据
- 前代角色数据
- 世界状态数据

### 2. 时间推进

根据前代角色结局时的年龄，计算世界过去的时间：

| 前代结局 | 时间跳跃 |
|----------|----------|
| 英年早逝（<30岁） | 20-50年 |
| 正常终老（60-80岁） | 10-30年 |
| 长寿（>80岁） | 5-15年 |

### 3. 世界演进

AI根据前代角色的重大行为，推理世界变化：

**军事行为：**
- 参与战争 → 边境变化，势力消长
- 击杀将领 → 家族复仇，政治动荡
- 建立军队 → 新势力崛起

**经济行为：**
- 发展商业 → 城镇繁荣
- 控制资源 → 价格变化
- 建立商会 → 经济格局变化

**社会行为：**
- 救人济世 → 传说流传，好感传承
- 作恶多端 → 恶名远扬，敌意传承
- 建立组织 → 组织壮大或衰落

**秘密行为：**
- 泄露秘密 → 政治格局变化
- 发现遗迹 → 新的势力介入
- 获得神器 → 后人觊觎

### 4. NPC生成

前代角色转为NPC，具有：

| 属性 | 说明 |
|------|------|
| 年龄 | 时间跳跃后的年龄（可能已故） |
| 状态 | 在世/已故/失踪/传说 |
| 记忆 | 记得前代经历的人和事 |
| 立场 | 对新角色友善/中立/敌对 |
| 影响 | 在世界中的地位和权力 |

**NPC类型：**

| 类型 | 条件 | 新角色关系 |
|------|------|-----------|
| 传说英雄 | 名声>80，已故 | 可能被传颂，有后人 |
| 在世长辈 | 在世，有后代 | 可能是祖父母/父母 |
| 隐世高人 | 在世，低调 | 可能成为导师 |
| 权势人物 | 权力>70 | 可能是贵族/官员 |
| 复仇者 | 被前代伤害 | 可能是敌人 |
| 友善者 | 亲密度>80 | 可能帮助新角色 |

### 5. 新角色生成

新角色可能与前代有联系：

| 关系 | 概率 | 效果 |
|------|------|------|
| 血缘后代 | 30% | 继承家族天赋，继承声望 |
| 弟子/传人 | 20% | 继承部分技能，有师门关系 |
| 仇人后代 | 15% | 继承仇恨，有敌对势力 |
| 无关新人 | 35% | 全新开始 |

## 继承要素

### 家族天赋

前代高属性可能遗传给后代：

| 前代属性 | 遗传效果 |
|----------|----------|
| STR≥18 | 力量天赋+2 |
| INT≥18 | 智力天赋+2 |
| CHA≥18 | 魅力天赋+2 |
| 任意两属性≥16 | 天赋特质 |

### 家族声望

前代行为影响后代起点：

| 前代名声 | 后代影响 |
|----------|----------|
| 英雄（>80） | 社交判定+2，贵人相助概率提高 |
| 恶人（<20） | 社交判定-2，被敌视概率提高 |
| 普通（40-60） | 无特殊影响 |

### 家族关系

前代的关系网影响后代：

| 关系类型 | 影响 |
|----------|------|
| 友人 | 可能帮助后代 |
| 敌人 | 可能报复后代 |
| 组织 | 可能加入或被针对 |
| 家族 | 可能获得资源或仇恨 |

### 物品继承

前代遗留的物品可能被后代获得：

| 物品类型 | 获得方式 |
|----------|----------|
| 传家宝 | 直接继承 |
| 隐藏物品 | 探索发现 |
| 流失物品 | 任务找回 |
| 被夺物品 | 击败持有者 |

## 前代NPC互动

### 见面前代

当新角色遇到前代角色（NPC）时：

**在世NPC：**
```
NPC认出新角色：「你是...[前代名字]的后人？」
NPC回忆：「当年你的[父亲/母亲]...」
NPC给予：帮助/考验/警告
```

**已故NPC（通过传说/后人）：**
```
听闻传说：「听说当年[前代名字]...」
后人讲述：「我的[祖父/祖母]常说...」
遗迹发现：前代留下的痕迹
```

### 特殊事件

| 事件 | 触发条件 | 内容 |
|------|----------|------|
| 前代遗愿 | 继承关系 | 完成前代未竟之事 |
| 家族秘密 | 血缘后代 | 发现前代隐藏的秘密 |
| 复仇之路 | 仇人后代 | 前代仇人找上门 |
| 传承试炼 | 弟子/传人 | 通过考验获得传承 |
| 传说重现 | 声望>80 | 被人认出是英雄后代 |

## 多代传承

支持多代续缘，形成家族史诗：

### 家族树

每代续缘，记录家族树：

```
第一代：艾拉（英雄）
  └─ 第二代：希尔文（学者）
      └─ 第三代：？
```

### 家族加成

多代传承累积加成：

| 代数 | 加成 |
|------|------|
| 第2代 | +1随机属性 |
| 第3代 | +2随机属性，+1天赋 |
| 第4代+ | +2随机属性，+1天赋，特殊特质 |

### 家族诅咒

如果前代有负面结局，可能产生家族诅咒：

| 负面结局 | 诅咒效果 |
|----------|----------|
| 被背叛而死 | 家族信任危机 |
| 未完成的承诺 | 命运纠缠 |
| 隐藏的罪行 | 真相迟早曝光 |
| 得罪大势力 | 家族被追杀 |
