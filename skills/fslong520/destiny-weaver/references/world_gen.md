# 世界观生成规则

## 生成流程

```
1. 基础设定 → 2. 魔法体系 → 3. 种族设定 → 4. 地理国家 → 5. 历史背景
```

## 一、基础设定

### 世界类型（随机选择或用户指定）

| 类型 | 描述 | 典型元素 |
|------|------|----------|
| 奇幻中世纪 | 剑与魔法、骑士、巨龙 | 城堡、骑士团、魔法学院 |
| 东方玄幻 | 修仙、灵气、宗门 | 灵山、秘境、仙人 |
| 蒸汽朋克 | 机械、齿轮、蒸汽 | 飞艇、工厂、机械师 |
| 黑暗奇幻 | 克苏鲁、深渊、绝望 | 古神、诅咒、末日 |
| 异能都市 | 现代背景+超能力 | 都市、组织、隐秘战争 |
| 原始部落 | 远古、自然、图腾 | 部落、神灵、荒野 |
| 废土末世 | 文明崩塌、生存 | 废墟、变异、拾荒 |

### 世界命名

根据类型生成适合的世界名：
- 奇幻：艾尔德兰、奥斯托利亚、瓦伦西亚
- 东方：九洲、灵域、玄天大陆
- 蒸汽：齿轮城、蒸汽纪元
- 黑暗：沉沦之地、深渊边缘

## 二、魔法体系

### 魔法类型（1-3种共存）

| 类型 | 描述 | 代表能力 |
|------|------|----------|
| 元素魔法 | 火/水/风/土/雷/光/暗 | 火球术、冰封、雷电 |
| 灵魂魔法 | 精神、灵魂、意识 | 读心、附体、精神攻击 |
| 符文魔法 | 刻画、阵法、契约 | 法阵、符文武器 |
| 自然魔法 | 生命、植物、动物 | 治疗、变形、召唤 |
| 时空魔法 | 时间、空间、维度 | 瞬移、时间停止 |
| 亡灵魔法 | 死者、诅咒、黑暗 | 召唤亡灵、诅咒 |
| 星辰魔法 | 星象、命运、预言 | 预言、星辰之力 |

### 魔法稀有度

| 稀有度 | 人口占比 | 能力上限 |
|--------|----------|----------|
| 无魔法者 | 60-80% | 无 |
| 微弱天赋 | 15-25% | 初级法术 |
| 普通法师 | 5-10% | 中级法术 |
| 强力法师 | 1-3% | 高级法术 |
| 大魔导师 | 0.1-0.5% | 禁咒级 |
| 传说级 | 0.01% | 神级 |

### 魔法来源

- 天生血脉（贵族血统）
- 契约获得（与精灵/恶魔/神灵）
- 学习获得（学院/宗门）
- 事件获得（奇遇/事故）
- 物品获得（神器/魔器）

## 三、种族设定

### 核心种族（随机3-6种）

| 种族 | 特点 | 寿命 | 分布 |
|------|------|------|------|
| 人类 | 适应性强、创造力 | 80年 | 广泛 |
| 精灵 | 魔法亲和、长寿 | 800年 | 森林 |
| 矮人 | 锻造大师、坚韧 | 300年 | 山脉 |
| 兽人 | 强壮、野性 | 60年 | 荒野 |
| 龙裔 | 龙血、强大 | 500年 | 秘境 |
| 妖精 | 小型、自然魔法 | 无限 | 幻境 |
| 魅魔/恶魔 | 魅惑、黑暗魔法 | 无限 | 深渊 |
| 天使/天族 | 光明、神圣 | 无限 | 天界 |
| 混血 | 两族特征 | 取平均 | 边缘 |

### 种族关系

- 同盟种族
- 敌对种族
- 疏远种族
- 奴役关系

## 四、地理与国家

### 地理生成

```
大陆数量：1-3个主要大陆
岛屿数量：随机
主要地形：森林、沙漠、山脉、海洋、平原
特殊地点：秘境、禁地、圣地
```

### 国家生成

每个大陆生成2-5个国家：

| 政体类型 | 描述 | 特点 |
|----------|------|------|
| 王国 | 国王统治 | 传统、骑士、贵族 |
| 帝国 | 皇帝统治 | 庞大、军事、扩张 |
| 共和国 | 议会/执政官 | 商业、自由、多元 |
| 神权国 | 祭司统治 | 宗教、神殿、信仰 |
| 部落联盟 | 酋长会议 | 分散、传统、自然 |
| 法师议会 | 法师统治 | 魔法、学院、研究 |
| 商业城邦 | 商会统治 | 贸易、富裕、开放 |

```
时代特点：魔法逐渐消失，科技开始兴起
主要矛盾：传统魔法势力 vs 新兴科技势力
```

## 六、世界观示例

### 示例1：传统奇幻

```
世界名：艾尔德兰大陆
类型：奇幻中世纪
魔法体系：元素魔法（火水风土）、符文魔法
种族：人类、精灵、矮人、兽人、龙裔
国家：
  - 艾尔德兰王国（人类，骑士强国）
  - 银月森林（精灵，魔法圣地）
  - 铁山联邦（矮人，锻造之都）
  - 血爪部落（兽人，游牧民族）
历史：
  - 300年前：黑暗魔王降临，五大种族联手击败
  - 100年前：魔法开始衰退，引发恐慌
  - 10年前：艾尔德兰王子失踪
当前局势：表面和平，暗流涌动
```

### 示例2：东方玄幻

```
世界名：九洲灵域
类型：东方玄幻
修炼体系：炼气→筑基→金丹→元婴→化神→大乘
种族：人族、妖族、灵族、魔族
势力：
  - 七大宗门（正道）
  - 魔教三宗（魔道）
  - 皇朝（世俗政权）
  - 妖族联盟（异族）
历史：
  - 万年前：上古仙人大战，天地破碎
  - 千年前：人族统一建立皇朝
  - 百年前：魔教崛起，正魔大战
当前局势：正邪对峙，妖族虎视眈眈
```

## 七、生成权重

### 倾向调整

根据用户偏好调整权重：

| 用户说 | 调整方向 |
|--------|----------|
| "多点魔法" | 魔法稀有度降低，魔法种类增加 |
| "轻松一点" | 减少黑暗元素，增加喜剧元素 |
| "现实向" | 减少魔法，增加政治经济 |
| "冒险向" | 增加未知区域，减少文明程度 |
| "战争向" | 增加国家冲突，减少和平时期 |
