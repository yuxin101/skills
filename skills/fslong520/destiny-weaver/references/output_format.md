# 输出格式规范

## 回合输出模板

```markdown
---

**第 {回合数} 回合 · {年龄}岁 · {季节} · {地点}**

---

> **📜 {事件标题}**
>
> {事件描述，2-4句话}
>
> 「{NPC对话}」

**【判定】**

| 项目 | 内容 |
|------|------|
| 属性 | {属性名} |
| 需求 | ≥ {数值} |
| 你的值 | {当前值} |
| 结果 | ✓ 成功 / ✗ 失败 |

**【选择】**

| 选项 | 行动 | 暗示 |
|------|------|------|
| [1] | {选项1} | {可选暗示} |
| [2] | {选项2} | {可选暗示} |
| [3] | {选项3} | {可选暗示} |

**【状态】**

| 属性 | 数值 | 进度 | 评价 |
|------|------|------|------|
| STR 力量 | 7 | ███████░░░ | 体弱 |
| DEX 敏捷 | 12 | ████████████ | 良好 |
| INT 智力 | 15 | ███████████████ | 天才 |
| CHA 魅力 | 10 | ██████████ | 正常 |
| LUK 幸运 | 8 | ████████░ | 普通 |
```

## 奖励输出格式

```markdown
**【获得】**

| 类型 | 内容 |
|------|------|
| ◆ 属性 | 敏捷 +1（12→13） |
| ★ 技能 | 暗影潜行（入门） |
| ✓ 物品 | 恢复药剂 x2 |
| ◇ 线索 | 发现神秘组织踪迹 |
| ★ 称号 | 梦境行者 |
```

## 阶段总结格式

```markdown
---

**{阶段名} 回顾**

---

**【大事记】**

| 年龄 | 事件 |
|------|------|
| 0岁 | 天赋觉醒，银色眼眸引人注目 |
| 1岁 | 遇见神秘旅人，获得护身符 |

**【属性变化】**

| 属性 | 旧值 | 新值 | 变化 |
|------|------|------|------|
| INT | 15 | 17 | +2 |

---
```

## 图标约定

| 类型 | 图标 | 说明 |
|------|------|------|
| 判定成功 | ✓ | 通过判定 |
| 判定失败 | ✗ | 未通过判定 |
| 大成功 | ★ | 超出预期 |
| 正面特质 | ✓ | 有益特质 |
| 负面特质 | ✗ | 有害特质 |
| 中性特质 | ◆ | 双面特质 |
| 获得 | ★ | 获得物品/技能 |
| 危险 | ⚠️ | 警告信号 |
| 解锁 | 🔓 | 解锁新内容 |
| 成就 | 🏆 | 达成成就 |

## 世界观展示格式

```markdown
---

## 📜 剧本：{剧本名}

---

**【世界观】**

| 项目 | 内容 |
|------|------|
| 世界名称 | {名称} |
| 世界类型 | {类型} |
| 核心力量 | {力量体系} |
| 主要种族 | {种族列表} |
| 当前时代 | {纪元} |
| 核心矛盾 | {主要冲突} |

**【世界背景】**

> {详细描述，3-5句话}

---
```

## 角色选项格式

```markdown
> **📜 命运纺锤转动，三条命运线浮现...**

---

**【角色一】**

| 项目 | 信息 |
|------|------|
| 姓名 | {名字} |
| 性别 | 男/女 |
| 种族 | {种族} |
| 出身 | {出身描述} |

**外貌**

> {详细外貌描述}

**属性**

| 属性 | 数值 | 评价 |
|------|------|------|
| STR | {值} | {评价} |
| DEX | {值} | {评价} |
| INT | {值} | {评价} |
| CHA | {值} | {评价} |
| LUK | {值} | {评价} |

**特质**

| 类型 | 特质 | 效果 |
|------|------|------|
| ✓ | {正面特质} | {效果} |
| ✗ | {负面特质} | {效果} |

---

**【选择】**

| 选项 | 行动 |
|------|------|
| [1] | 选择角色一 |
| [2] | 选择角色二 |
| [3] | 选择角色三 |
| [4] | 都不喜欢，重新生成 |
```
