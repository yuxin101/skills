# 自由行动规则

## 核心机制

每回合玩家有**行动力（AP）**，可以用来做想做的事。

行动力是有限的资源，需要权衡如何使用。

---

## 行动力上限

| 人生阶段 | 行动力 | 说明 |
|----------|--------|------|
| 婴儿期 | 1 | 只能被动接受，无法主动行动 |
| 童年期 | 2 | 初步自主，选择有限 |
| 少年期 | 3 | 青春躁动，更多选择 |
| 青年期 | 4 | 精力充沛，最佳时期 |
| 中年期 | 3 | 责任加重，时间紧张 |
| 老年期 | 2 | 体力下降，精力有限 |

## 行动规则

### 基本规则

1. **行动力不累积**：每回合重置，不用就浪费
2. **主线优先**：主线事件不消耗行动力，必须处理
3. **行动可组合**：可以组合多个行动（如学习+社交）
4. **行动有风险**：某些行动可能触发意外事件

### 行动力分配

每回合开始时，输出行动力状态：

```
【行动力】当前：3/3
```

玩家选择行动后，扣除对应行动力：

```
【行动】学习（消耗1AP）→ 剩余2AP
```

### 行动结果

根据属性进行判定，决定行动效果：

| 结果 | 效果 |
|------|------|
| 大成功 | 额外收益 |
| 成功 | 正常收益 |
| 失败 | 收益减少或负面效果 |
| 大失败 | 负面事件 |

## 行动示例

### 示例1：童年期

```
【行动力】当前：2/2

主线事件：学校的数学竞赛

【自由行动】选择你想做的事：
[1] 学习（1AP）——准备竞赛，提升INT
[2] 社交（1AP）——和同学聊天
[3] 探索（1AP）——放学后去后山看看
[4] 休息（0AP）——回家休息
```

玩家选择：[1]学习 + [2]社交（消耗全部2AP）

### 示例2：青年期

```
【行动力】当前：4/4

主线事件：收到了征兵令

【自由行动】选择你想做的事：
[1] 锻炼（1AP）——提升STR，为战斗做准备
[2] 打听消息（1AP）——了解征兵的详情
[3] 找小雅（1AP）——去见她一面
[4] 深入探索（2AP）——寻找躲避征兵的方法
```

玩家选择：[3]找小雅（1AP）+ [4]深入探索（剩余3AP不够，需选择其他）

## 特殊情况

### 行动力不足

如果玩家想做的行动消耗超过当前行动力：
- 提示行动力不足
- 建议选择其他行动或组合

### 强制事件

某些情况下，主线事件会消耗行动力：
- 紧急情况：如被追杀，必须行动
- 强制社交：如有人找上门，必须应对
- 被动事件：如生病，行动力减半

### 行动力恢复

- 正常回合：自动恢复到上限
- 特殊状态：可能减少上限（如受伤）
- 特殊物品：可能增加上限（如精力药剂）
