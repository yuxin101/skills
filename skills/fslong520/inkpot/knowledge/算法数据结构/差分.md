# 差分

## 一句话解释

差分是前缀和的**逆运算**，用于高效处理**区间修改**问题，将 O(n) 的区间操作降为 O(1)。

## 详细解释

### 定义

给定原数组 `a[1..n]`，定义差分数组 `d[1..n]`：

```
d[i] = a[i] - a[i-1]   (i ≥ 2)
d[1] = a[1]
```

### 核心性质

1. **前缀和还原**：`a[i] = d[1] + d[2] + ... + d[i]`
2. **差分定义**：`d[i] = a[i] - a[i-1]`

### 区间修改原理

对 `[l, r]` 区间所有元素加 `k`，只需：

```cpp
d[l] += k;
d[r+1] -= k;
```

**原理分析**：
- `d[l] += k`：从位置 l 开始，前缀和都多了 k
- `d[r+1] -= k`：从位置 r+1 开始，前缀和抵消掉 k
- 结果：恰好 `[l, r]` 区间各加了 k

## 关键要点

- 差分适合**多次区间修改，最后一次性查询**的场景
- 如果需要实时查询，要配合树状数组或线段树
- 二维差分类似：`d[x1][y1] += k, d[x2+1][y1] -= k, ...`

## 代码示例

```cpp
const int MAXN = 100005;
int n;
int a[MAXN], d[MAXN];

// 区间修改：[l, r] 每个元素加 k
void range_add(int l, int r, int k) {
    d[l] += k;
    d[r + 1] -= k;
}

// 还原原数组
void build() {
    a[0] = 0;
    for (int i = 1; i <= n; i++) {
        a[i] = a[i - 1] + d[i];
    }
}
```

## 示例演示

原数组：`[1, 3, 5, 7, 9]`
差分数组：`[1, 2, 2, 2, 2]`

对 `[2, 4]` 区间加 3：
```
d[2] += 3 → d = [1, 5, 2, 2, 2]
d[5] -= 3 → d = [1, 5, 2, 2, -1]
```

前缀和还原：`[1, 6, 8, 10, 9]` ✓
