# InkPot - 墨池

> 砚田勤耕，墨香留痕。让AI陪你沉淀每一份知识。

[![ClawHub](https://img.shields.io/badge/ClawHub-inkpot-blue)](https://cnb.cool)

## 简介

墨池是一个AI驱动的知识管理技能，它会在你日常使用AI的过程中，自动识别知识点并沉淀下来。

### 核心特性

- **自动学习** - 识别对话中的知识点，自动归档
- **用户画像** - 分析用户行为，建立能力模型
- **知识关联** - 构建个人知识网络
- **智能推荐** - 基于画像推荐相关学习内容

## 安装

```bash
clawhub install inkpot
```

## 使用

安装后无需手动调用，墨池会在对话中自动工作。

### 命令

| 命令 | 说明 |
|------|------|
| `/墨池 画像` | 查看用户画像 |
| `/墨池 复习` | 智能推荐复习 |
| `/墨池 推荐` | 个性化学习推荐 |
| `/墨池 索引` | 查看知识库索引 |
| `/墨池 搜索 <关键词>` | 搜索知识点 |

## 目录结构

```
inkpot/
├── SKILL.md           # 技能说明文档
├── skill.yaml         # 技能元数据
├── knowledge/         # 知识库
│   ├── 编程语言/
│   ├── 算法数据结构/
│   ├── 数学/
│   ├── 计算机基础/
│   ├── 人工智能/
│   └── 其他/
└── profile/           # 用户画像
    ├── index.md       # 知识索引
    └── records.md     # 学习记录
```

## 工作原理

### 自动学习机制

墨池会分析你的对话行为：

| 行为 | 墨池的分析 |
|------|-----------|
| 询问概念 | 记录知识点 + 推断正在学习此领域 |
| 让AI写代码 | 分析代码类型 + 推断技术栈 |
| 纠正AI错误 | 高权重标记 + 优先学习 |

### 用户画像

墨池会根据你的行为建立能力模型：
- 擅长领域及熟练度
- 兴趣领域
- 常用技能
- 学习偏好

## 为什么叫墨池？

古人磨墨，墨汁在砚池中慢慢积蓄。知识也是这样——不是复制粘贴，是一次次思考、一次次使用，慢慢沉淀下来的。

墨池，就是你的AI砚台。
