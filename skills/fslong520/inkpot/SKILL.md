# InkPot - 墨池

> 砚田勤耕，墨香留痕。让AI陪你沉淀每一份知识。

**英文名**: InkPot（墨水瓶，寓意知识的源泉）

---

## 核心理念

墨池**始终在线**，在日常对话中：

1. **自动识别知识点** - 你问什么，墨池就沉淀什么
2. **分析用户行为** - 你让AI做什么，墨池就懂你擅长什么
3. **建立用户画像** - 记录你的擅长领域、兴趣点、常用技能
4. **智能推荐** - 基于画像推荐相关知识

---

## 自动学习机制

### 触发场景（无需手动调用）

| 场景 | 墨池的行为 |
|------|-----------|
| 询问概念 | → 记录知识点 + 推断正在学习此领域 |
| 让AI写代码 | → 分析代码类型 + 推断擅长编程 |
| 让AI搬运题目 | → 分析题目类型 + 推断做信奥竞赛 |
| 讨论某技术 | → 记录技术栈 + 推断技术背景 |
| 纠正AI错误 | → 特别标记 + 优先学习 |

### 用户行为分析

```
用户行为 → 推断用户擅长领域

让AI搬算法题 → 信奥竞赛教练，擅长算法教学
让AI写C++代码 → C++开发者，可能做竞赛/系统开发
让AI解释数学概念 → 数学爱好者或教学者
让AI调试代码 → 有实际开发经验
让AI写文档 → 技术文档撰写者
```

## 智能功能

### 上下文感知
- 根据用户画像调整回答深度
- 关联用户已知的知识点

### 主动推荐
- 基于画像推荐相关知识
- 学习路径建议

### 知识关联
- 相关知识点互相链接
- 构建个人知识网络
