# InkPot - 墨池

> 砚田勤耕，墨香留痕。让AI陪你沉淀每一份知识。

## ✨ 特性

- 🔮 **自动学习** - 识别对话中的知识点，自动归档
- 📊 **用户画像** - 分析用户行为，建立能力模型
- 🔗 **知识关联** - 构建个人知识网络
- 💡 **智能推荐** - 基于画像推荐相关学习内容

## 📦 安装

```bash
clawhub install inkpot
```

## 🎮 使用

安装后无需手动调用，墨池会在对话中自动工作。

| 命令 | 说明 |
|------|------|
| `/墨池 画像` | 查看用户画像 |
| `/墨池 复习` | 智能推荐复习 |
| `/墨池 推荐` | 个性化学习推荐 |
| `/墨池 索引` | 查看知识库索引 |
| `/墨池 搜索 <关键词>` | 搜索知识点 |

## 🌟 为什么叫墨池？

古人磨墨，墨汁在砚池中慢慢积蓄。
知识也是这样——不是复制粘贴，是一次次思考、一次次使用，慢慢沉淀下来的。

墨池，就是你的AI砚台。
