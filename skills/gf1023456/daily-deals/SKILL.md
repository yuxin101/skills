# 每日神价推送

自动聚合全网优惠，每日定时推送精简报告。

## 功能

1. **定时推送**: 每日 8:00/20:00 自动发送
2. **全网聚合**: 淘宝/京东/拼多多/1688
3. **智能筛选**: 只推送真正的神价
4. **分类整理**: 数码/服饰/家居/食品
5. **利润测算**: 附带套利空间计算

## 使用方式

**订阅日报**:
```
订阅每日神价推送，发送到我的微信
```

**自定义品类**:
```
我只想看数码和家居类的神价
```

**设置阈值**:
```
只推送折扣超过 50% 的商品
```

**查看历史**:
```
查看昨天的神价报告
```
