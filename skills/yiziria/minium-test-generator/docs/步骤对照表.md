## 📋 方法封装示例

### 首页方法

```python
class Home(BaseDef):
    """首页页面对象"""
    
    # 元素声明
    element_listItem = 'view.list-item'
    
    def goto_page(self):
        """step_1: 打开首页"""
        self.relaunch_to_open('/pages/home/home')
    
    def click_list_item(self):
        """step_2: 点击列表项"""
        self.tap(self.element_listItem)
```

### 详情页方法

```python
class Detail(BaseDef):
    """详情页页面对象"""
    
    # 元素声明
    input_username = 'input.username'
    input_password = 'input.password'
    btn_submit = 'button.submit'
    
    def fill_form_and_submit(self):
        """
        step_3-5: 填写表单并提交
        
        录制脚本对应步骤:
            step_3: 输入用户名
            step_4: 输入密码
            step_5: 点击提交
        """
        # step_3: 输入用户名
        self.send_key(self.input_username, "test")
        
        # step_4: 输入密码
        self.send_key(self.input_password, "123456")
        
        # step_5: 点击提交
        self.tap(self.btn_submit)
```
