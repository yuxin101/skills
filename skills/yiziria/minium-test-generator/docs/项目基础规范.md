## 🔧 基类封装

### basedef.py - 页面对象基类

**位置**: `base/basedef.py`

**核心功能**:
- 封装常用操作（tap, send_key, scroll 等）
- 提供元素查找和等待机制
- 处理弹窗和授权
- 日志记录和错误处理

**关键方法**:


### basepage.py - 测试用例基类

**位置**: `base/basepage.py`

**核心功能**:
- 继承自 `basedef.BaseDef`
- 提供测试用例的生命周期管理
- 封装断言方法

**关键方法**:

```python
class BasePage(BaseDef):
    """所有测试用例的基类"""
    
    def setUp(self):
        """每个测试方法执行前的准备"""
        pass
    
    def tearDown(self):
        """每个测试方法执行后的清理"""
        pass
    
    @classmethod
    def setUpClass(cls):
        """测试类执行前的准备"""
        super(BasePage, cls).setUpClass()
        # 初始化页面对象
    
    @classmethod
    def tearDownClass(cls):
        """测试类执行后的清理"""
        super(BasePage, cls).tearDownClass()
```

**使用示例**:


## 📝 页面类规范

### 文件结构

```python
# -*- coding: utf-8 -*-
"""
页面名称
页面路径：/pro/pages/xxx/xxx
"""

from time import sleep
from base.basedef import BaseDef
from data import page_data


class PageName(BaseDef):
    """页面对象类"""
    
    # ========== 元素声明（所有元素在顶部） ==========
    # 按钮名称：元素描述
    element_name = '选择器'
    
    # ========== 页面方法（所有方法在元素下面） ==========
    def method_name(self, param):
        """
        方法说明
        
        参数:
            param: 参数说明
        
        录制脚本对应步骤:
            step_x: 步骤描述
        """
        # 操作代码
```

### 元素命名规范

| 元素类型 | 命名前缀 | 示例 | 说明 |
|---------|---------|------|------|
| 普通元素 | `element_` | `element_submit` | 通用元素 |
| 输入框 | `input_` | `input_username` | 文本输入框 |
| 按钮 | `btn_` | `btn_confirm` | 按钮（可选） |
| 图标 | `icon_` | `icon_back` | 图标 |
| 列表 | `list_` | `list_items` | 列表项 |

**避免命名冲突**:
```python
# ❌ 错误（元素名与方法名冲突）
input_price = '...'  # 元素
def input_price(self, price): ...  # 方法

# ✅ 正确（添加后缀区分）
input_priceAmount = '...'  # 元素
def input_price(self, price): ...  # 方法
```

### 选择器规范

**优先级**: XPath > CSS Selector

```python
# ✅ 推荐（XPath，稳定）
element = '/page-wrapper/view/view[1]/button'

# ⚠️ 备选（CSS，可能因 class 变化失效）
element = 'view.submit-btn'

# ❌ 避免（动态 class 名）
element = 'view.dynamic-class-12345'
```

## ⚠️ 等待时间规范

```python
# ✅ tap() 不需要额外等待（内部已有重试机制）
self.tap(self.element_name)

# ✅ send_key() 后如有必要可加等待
self.send_key(self.input_name, text)
self.page.wait_for(2)

# ✅ 滚动操作后建议加等待
self.page.scroll_to(1000)
self.page.wait_for(1)

# ❌ 不要每个操作都加等待
self.tap(self.element_1)
self.page.wait_for(2)  # 多余
self.tap(self.element_2)
self.page.wait_for(2)  # 多余
```

## 📊 代码审查清单

在提交代码前，请检查以下项目：

- [ ] 所有元素声明在类顶部
- [ ] 所有方法在元素声明下面
- [ ] 元素命名不与方法名冲突
- [ ] 优先使用 XPath 选择器
- [ ] 每个方法注释中标注对应步骤
- [ ] 测试用例中有步骤对照说明
- [ ] 使用 `validate_steps.py` 验证步骤完整性
