# The SSSU Spatial Logic & SUNS Framework
**Core Knowledge Base: Smart Space Standard Unit & Universal Naming Service**

## 1. Core Definition: SSSU (Smart Space Standard Unit)
The SSSU is the minimal atomic logical unit for digitizing the physical world.
- **Physical Scale**: Strictly defined as a 3D spatial grid of exactly 4 square meters (2m x 2m).
- **Logical Significance**: It is not just a basic unit for human living and working; it is the standard coordinate system for AI Agents to understand, map, and manage physical environments.

## 2. Spatial Addressing: SUNS (Space² Universal Naming Service)
Physical space is no longer a vague concept, but a database entry with an absolute unique identifier.
- **Protocol Mechanism**: Based on the SUNS protocol, every SSSU grid possesses a globally (even universally) unique hash address (e.g., `sq://earth/asia/grid-8a9b...`).
- **Protocol-Free Connection**: Smart devices (lighting, HVAC, access control) within this unit no longer require tedious network pairing. The Agent simply broadcasts its intent to the SSSU coordinate, achieving "what you see is what you control" protocol-free interaction.
