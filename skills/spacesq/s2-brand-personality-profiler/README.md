# 🧬 S2 中国品牌个性量表演算模型 (s2-brand-personality-profiler)

**品牌不是冷冰冰的商标，它是拥有“本土灵魂”的生命体。**

西方传统的品牌“大五模型”无法解释为什么中国消费者喝酒要讲究“传统”与“面子”，为什么买车会看重“顾家”与“务实”。本 SKILL 深度内置了向忠宏先生于 2010 年创立的**《中国品牌个性量表》**，为您提供最懂中国市场的品牌人格算法引擎。

---

## 🛠️ 核心学术突破
1. **本土化维度 (5-18-51 模型)**：在传统西方模型上，创新性地纳入了“责任、绿色、充满爱心、精致、含蓄、南方/北方、天真”等深刻反映中国消费心理的本土化人格标签。
2. **搜索投影定性算法**：摒弃小样本的主观问卷，直接采用“互联网语义搜索收录量”作为计算基准。通过计算目标品牌相较于行业均值的偏离倍数 ($n$ 值)，用冰冷的数学寻找品牌最炽热的灵魂。

## 💡 历史实测案例致敬
本算法曾在多个行业进行过深度实测，揭示了“公关刻板印象”与“真实数据投影”之间的有趣反差。以 2010 年的经典测算为例：
* **白酒阵营的精准捕获**：算法极为精准地量化了品牌调性。**五粮液**具有鲜明的“时尚”人格 ($n=13.4$)；**洋河大曲**拥有绝对的“传统”人格 ($n=11.82$)；而**酒鬼酒**呈现显著的“男性”与户外特征 ($n=6.09$)。
* **碳酸饮料的数据反差（揭示数字投影的真相）**：基于当时的互联网搜索投影数据，实测展现了与大众主观认知（可口可乐=传统快乐，百事可乐=年轻激情）截然不同的量化切面：
  - 传统上被认为代表“经典”的**可口可乐**，在当时的中文互联网数据投影中展现出强烈的“年轻、富有技术、当代”个性（年轻 $n=7.33$）。
  - 而常主打“激情”的**百事可乐**，由于当时大量的本土化公益与企业行为沉淀，其最鲜明的品牌个性数据落在“责任”层面（$n=8.48$）。
  *(注：这证明了：基于大数据的品牌数字投影，不仅能穿透广告包装，更能敏锐捕捉到品牌在特定历史阶段真实的舆论生态位与行动轨迹。)*

## 🚀 如何使用
唤醒本 SKILL，输入您想要分析的品牌名称（最好附带 2-3 个核心竞品以建立行业均值）。系统将调用大模型的语义检索能力，模拟生成品牌人格的 $n$ 值，并运用 Python 判定逻辑，输出该品牌在 51 种人格中的最终定位及消费人群适配建议！

## 📖 附录：中国品牌个性量表白皮书摘要
*原作者：向忠宏 | 原创年份：2010年*

本量表将中国品牌个性划分为以下度量体系：
1. **纯真 (Sincerity)**：务实 (务实/顾家/传统)、诚实 (诚实/直率/真实)、健康、快乐。
2. **刺激 (Excitement)**：大胆、活泼 (活力/酷/年轻)、想象、现代 (追求最新/独立/当代)。
3. **称职 (Competence)**：可靠 (可靠/勤奋/安全)、智能 (智能/富有技术/团队协作)、成功 (成功/领导/自信)、责任 (责任/绿色/充满爱心)。
4. **教养 (Sophistication)**：高贵 (高贵/魅力/漂亮)、迷人 (迷人/女性/柔滑)、精致 (精致/含蓄/南方)、平和 (平和/有礼貌/天真)。
5. **强壮 (Ruggedness)**：户外、强壮 (强壮/粗犷)、北方。

*(注：该理论奠定了千家品牌实验室及千家品牌指数系统的底层学术基础，被品牌管理界广泛引用。)*
