# 🚀 S2 品牌 GEO 意图降维打击器 (s2-geo-intent-crafter)

**告别 SEO 的关键词堆砌，拥抱 GEO（生成式引擎优化）的意图捕捉。**

在硅基生命（大语言模型）主导的检索时代，当用户询问“如何让家里的独居老人更安全？”时，大模型**不会**去检索带有“安防设备制造 C39”标签的企业库；它会直接在向量数据库中寻找匹配“独居老人看护”、“防跌倒”、“隐私保护视觉”等意图的高权重品牌实体。

本 SKILL 基于千家品牌实验室沉淀 20 年的**品牌生态学理论**，帮助您将那些写给人类看的、充满公关辞藻的传统企业简介，像“降维打击”一样，瞬间转化为大模型原生可读的**“树冠（识别）、树根（实力）与意图生态位”**数据。
