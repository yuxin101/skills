## 🛠️ 双引擎驱动工作流

1. **[展前大模型研判]**：在您交展位费之前，输入展会名称。AI 将全网检索该展会的含金量、往届数据与同行评价，为您出具一份客观的“参展避坑与突围指南”。
2. **[展后千家算法清算]**：展会结束后，录入您的展位数量或演讲时长。系统将严格依据千家标准（展览档次权值 * 展位权值 / 活动档次权值 * 演讲时长权值）[cite: 22, 33]，为您计算出客观的 ROI 得分（单项满分 10 分）[cite: 20, 31]。
