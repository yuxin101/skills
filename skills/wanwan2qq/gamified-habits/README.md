# 🎮 游戏化习惯养成 (Gamified Habits)

> 把人生变成 RPG 游戏，你就是主角！

## 📖 项目简介

一个将习惯养成游戏化的 APP，通过 RPG 角色成长系统让坚持变得有趣。

**核心理念**：不是"另一个打卡 APP"，而是"你的人生游戏"

## 🎯 核心功能

### 1. 角色系统
- 等级、经验值、属性面板（体力/智力/财富/心灵/社交）
- 成就徽章系统

### 2. 习惯任务
- 每日任务（Daily Quests）
- 周常任务（Weekly Quests）
- 成就系统

### 3. 社交玩法
- 组队打卡
- PK 排行榜
- 公会系统

### 4. 可视化反馈
- 人生日历
- 属性成长图
- 数据统计

### 5. 故事模式
- 世界观设定
- 剧情推进
- Boss 挑战
