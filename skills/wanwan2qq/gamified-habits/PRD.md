# 📋 PRD - 游戏化习惯养成 (OpenClaw Skill 版)

## 1. 产品概述

### 1.1 产品定位
OpenClaw Skill，将习惯养成游戏化的对话式助手

### 1.2 目标用户
- OpenClaw 用户
- 想培养好习惯但难以坚持的人
- 喜欢游戏化体验的年轻人

### 1.3 核心价值
- **有趣**：把枯燥的坚持变成游戏
- **对话式**：用自然语言打卡，无需打开 APP
- **可视化**：清晰看到自己的成长

### 1.4 技术架构
- **Phase 1**：纯本地存储（`~/.gamified-habits/user-data.json`）
- **Phase 2**：Mac mini 本地服务器（可选）
- **Phase 3**：完整后端（验证成功后）

## 2. 功能需求

### 2.1 习惯管理

#### 2.1.1 创建习惯
**命令**：`habits create`
**参数**：
- 习惯名称（必填）
- 习惯类型（体力/智力/财富/心灵/社交）
- 频率（每日/每周/自定义）
- 目标值（如 30 分钟、5km）
- XP 奖励（默认根据难度计算）

**示例**：
```
用户：创建一个早起习惯，每天 7 点前起床
技能：✅ 已创建习惯「早起」
      类型：体力 | 频率：每日 | 奖励：50 XP
```

#### 2.1.2 删除习惯
**命令**：`habits delete <习惯名>`

#### 2.1.3 查看习惯列表
**命令**：`habits list`

### 2.2 打卡系统

#### 2.2.1 打卡
**命令**：`habits checkin <习惯名>` 或自然语言

**示例**：
```
用户：我今早 7 点起床了
技能：✅ 早起打卡成功！
      +50 XP | 连击 5 天 🔥
      体力 +2，当前等级 Lv.3 (280/500 XP)
```

#### 2.2.2 自然语言识别
支持多种表达：
- "我起床了" → 匹配"早起"习惯
- "跑了 5 公里" → 匹配"跑步"习惯
- "看了 30 分钟书" → 匹配"阅读"习惯

#### 2.2.3 连击系统
- 连续打卡获得连击加成（1.1x - 2.0x）
- 断签后连击重置
- 连击保护道具（后续）

### 2.3 游戏化系统

#### 2.3.1 等级系统
```
等级公式：XP_total = Σ(习惯打卡 XP × 连击加成)
升级所需 XP: Level_N = N² × 100
```

等级示例：
- Lv.1: 0-100 XP
- Lv.2: 100-400 XP
- Lv.3: 400-900 XP
- Lv.4: 900-1600 XP
- Lv.5: 1600-2500 XP

#### 2.3.2 属性系统
| 属性 | 关联习惯 | 说明 |
|------|---------|------|
| 体力 | 早起、跑步、健身 | 运动健康类 |
| 智力 | 阅读、学习、写作 | 知识成长类 |
| 财富 | 储蓄、记账、理财 | 财务管理类 |
| 心灵 | 冥想、日记、感恩 | 内心平静类 |
| 社交 | 聚会、networking | 人际关系类 |

#### 2.3.3 成就系统
**成就类型**：
- 连击成就（连续 X 天）
- 累计成就（累计 X 次）
- 里程碑成就（等级/属性达到 X）

**示例成就**：
| 成就名称 | 条件 | 奖励 |
|---------|------|------|
| 晨型人 | 连续 7 天早起 | +500 XP + 徽章 |
| 新手上路 | 累计打卡 20 次 | +200 XP |
| 健身达人 | 累计跑步 50km | +1000 XP + 徽章 |
| 学霸 | 一年读完 50 本书 | +2000 XP + 称号 |

### 2.4 查询系统

#### 2.4.1 查看属性面板
**命令**：`habits status` 或 `我的属性`

**输出示例**：
```
🎯 Lv.5 自律勇者 (1680/2500 XP)

💪 体力：72  ████████░░
🧠 智力：85  █████████░
💰 财富：60  ██████░░░░
😌 心灵：55  █████░░░░░
🤝 社交：48  ████░░░░░░

🔥 当前连击：早起 5 天，跑步 3 天
```

#### 2.4.2 查看打卡记录
**命令**：`habits history [习惯名] [--days 7]`

**输出示例**：
```
📊 本周打卡情况 (3/7 天)

✅ 早起：5/7 天 (71%)
✅ 跑步：3/5 次 (60%)
❌ 阅读：1/7 天 (14%)

本周完成率：65%
```

#### 2.4.3 查看成就
**命令**：`habits achievements`

### 2.5 随机事件（Phase 2）

#### 2.5.1 双倍经验日
- 随机触发
- 当天打卡 XP ×2

#### 2.5.2 幸运事件
- "今天运气不错，额外 +20 XP！"

## 3. 技术实现

### 3.1 项目结构
```
~/.openclaw/skills/gamified-habits/
├── SKILL.md              # OpenClaw 技能定义
├── package.json
├── bin/
│   └── habits.js         # CLI 入口
└── src/
    ├── index.js          # 主逻辑
    ├── xp-system.js      # XP 计算
    ├── habits.js         # 习惯管理
    ├── achievements.js   # 成就系统
    └── storage.js        # 数据存储
```
