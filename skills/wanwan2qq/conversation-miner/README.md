# 💬 对话助手 (Conversation Miner) v2.0

> 从 AI 对话中提取结构化知识的智能助手

## 🎯 核心能力

1. **自动获取**当前会话的对话历史
2. **智能分析**对话内容，提取有价值信息
3. **结构化输出**总结、待办、想法、决策等
4. **询问用户**是否保存到 memory

## 📋 使用方式

### 自然语言触发
- "总结对话" → 自动总结当前会话
- "提取待办" → 提取待办事项
- "提取想法" → 提取想法/灵感
- "提取代码" → 提取代码片段
- "导出对话" → 导出为 Markdown
- "搜索对话" → 搜索历史对话

### 指定内容总结
- "总结关于项目进度的讨论"
- "总结一下 API 调试的过程"

### 保存对话
- 总结后回复"保存"或"yes"
- 自动保存到 `memory/{date}.md`
