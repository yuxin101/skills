# 比价返利雷达

跨平台商品比价与返利综合分析工具。输入商品名称或链接，自动在淘宝、京东、拼多多、抖音等平台搜索同款商品，综合计算售价、优惠券和返利后的真实到手价，推荐最优购买渠道。

## 触发词

比价返利、跨平台比价、哪里买最便宜、真实到手价、全网比价、price rebate、比价雷达、最低价、多平台比价、返利比价

## 计划实现能力

### 核心功能

- **全网同款搜索**：输入商品链接或名称，自动匹配各平台同款商品
- **真实到手价计算**：综合售价、平台活动、优惠券、返利，算出每个平台的真实到手价
- **历史价格走势**：展示商品在各平台的历史价格走势，判断当前是否值得买
- **最优渠道推荐**：按真实到手价排序，推荐最划算的购买渠道

### 比价维度

| 维度 | 说明 |
|------|------|
| 商品售价 | 各平台当前标价 |
| 平台活动 | 满减、跨店满减、平台补贴 |
| 优惠券 | 店铺券、平台券、品类券 |
| 返利比例 | 各渠道返利佣金比例 |
| 运费 | 是否包邮、运费金额 |
| 售后保障 | 7天无理由、运费险等 |

### 真实到手价公式

```
真实到手价 = 商品售价
           - 平台活动优惠
           - 优惠券面额
           - 返利金额
           + 运费（如有）
```

### 比价结果排序

```
输入商品链接/名称
    ↓
全网搜索同款
    ↓
各平台分别计算真实到手价
    ↓
按到手价从低到高排序
    ↓
标注最优渠道 + 各渠道优劣势
```

### 使用场景

1. **日常购物决策**：看到想买的商品，先比价再下单
2. **大促前准备**：对比各平台大促价格，提前锁定最优渠道
3. **高客单商品**：手机、电脑、家电等贵重商品，比价收益显著
4. **防坑验价**：识别虚假降价（先涨后降），看历史价格曲线
