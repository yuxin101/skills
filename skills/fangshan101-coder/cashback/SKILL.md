# 全民返利助手

全平台返利查询与下单引导工具。用户提供商品链接或关键词，自动查询淘宝、京东、拼多多等主流电商平台的返利比例，生成专属返利链接，帮助用户在每一笔购物中获得最大返利收益。

## 触发词

返利查询、查返利、返利比例、返利链接、商品返利、淘宝返利、京东返利、拼多多返利、cashback query、cashback link

## 计划实现能力

### 核心功能

- **商品返利查询**：输入商品链接或关键词，自动识别平台并查询可用返利比例
- **返利链接生成**：生成专属返利链接，用户通过该链接下单即可获得返利
- **多平台对比**：同一商品在不同平台的返利比例横向对比，推荐最优下单渠道
- **实时返利信息**：对接各平台返利 API，确保返利数据实时准确

### 支持平台

| 平台 | 返利类型 | 预计返利比例 |
|------|---------|------------|
| 淘宝/天猫 | 佣金分成 | 1%-90% |
| 京东 | 佣金分成 | 0.5%-50% |
| 拼多多 | 佣金分成 | 1%-80% |
| 唯品会 | 佣金分成 | 1%-30% |
| 苏宁易购 | 佣金分成 | 0.5%-15% |

### 使用场景

1. **购物前查询**：下单前粘贴商品链接，查看能拿多少返利
2. **比价决策**：同款商品多平台返利对比，选最划算的渠道
3. **批量查询**：一次粘贴多个链接，批量查询返利信息
4. **分享赚钱**：生成的返利链接可分享给他人，双方均可获益

## 输出格式

```
商品：[商品名称]
平台：[电商平台]
原价：¥[价格]
返利比例：[X]%
预计返利：¥[金额]
返利链接：[链接]
```
