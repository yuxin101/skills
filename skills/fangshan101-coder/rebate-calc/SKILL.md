# 返利计算器

返利精算工具，帮助用户在下单前精确计算商品经过各种优惠叠加后的真实到手价。支持复杂的多层优惠叠加计算（满减、折扣、优惠券、返利、红包等），告别"算不清到底省了多少"的困扰。

## 触发词

返利计算、算返利、到手价多少、算一下到手价、省了多少钱、rebate calc、返利计算器、优惠计算、真实价格

## 计划实现能力

### 核心功能

- **多层优惠计算**：精确处理满减、折扣、优惠券、返利、红包等多层叠加
- **凑单计算器**：输入满减门槛和购物车商品，自动推荐最优凑单方案
- **分摊计算**：多件商品参与满减时，精确分摊每件商品的实际到手价
- **对比计算**：多套优惠方案并行计算，对比哪套最划算

### 支持的优惠类型

| 优惠类型 | 计算方式 | 叠加顺序 |
|---------|---------|---------|
| 商品折扣 | 原价 × 折扣率 | 第 1 层 |
| 店铺满减 | 满 X 减 Y | 第 2 层 |
| 跨店满减 | 满 X 减 Y（跨店） | 第 3 层 |
| 优惠券 | 直减 / 满减 | 第 4 层 |
| 平台补贴 | 直减 | 第 5 层 |
| 红包 | 直减 | 第 6 层 |
| 返利 | 实付 × 返利比例 | 最后计算 |

### 计算引擎

```
输入：
  商品原价、数量
  可用优惠（满减、券、折扣、红包...）
  返利比例

计算流程：
  原价 × 数量 = 商品总价
  → 应用折扣
  → 应用满减（检查是否达到门槛）
  → 应用优惠券（检查使用条件）
  → 应用平台补贴
  → 应用红包
  = 实付金额
  → 计算返利（实付 × 返利比例）
  = 真实到手价

输出：每一步的扣减明细 + 最终到手价
```

### 使用场景

1. **单品计算**：算一件商品叠加所有优惠后的到手价
2. **凑单优化**：购物车多件商品，怎么凑满减最划算
3. **方案对比**：两个平台都有活动，哪边更便宜
4. **大促复杂场景**：满减+券+红包+返利多层叠加，精确计算
