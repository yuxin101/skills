# 返利全能助手

返利购物全流程助手，覆盖从选品比价、领券下单到返利追踪、提现指导的完整链路。无论是返利新手还是资深玩家，都能通过本工具实现购物全程省钱最大化。

## 触发词

返利助手、返利全流程、选品返利、返利提现、返利攻略、购物省钱、全能返利、cashback assist、full cashback、返利指导

## 计划实现能力

### 核心功能

- **智能选品推荐**：基于用户偏好和购物历史，筛选高返利商品并推荐
- **比价引擎**：跨平台比价，综合商品价格、优惠券和返利比例计算真实到手价
- **下单引导**：一步步引导用户完成返利下单流程，确保返利链路不断裂
- **返利追踪**：订单提交后自动追踪返利状态，异常时及时提醒
- **提现指导**：返利到账后指导用户完成提现操作，支持多种提现方式

### 全流程覆盖

```
选品阶段          下单阶段          售后阶段
   │                │                │
   ├─ 热门推荐      ├─ 领券提醒      ├─ 订单追踪
   ├─ 高返筛选      ├─ 链路检查      ├─ 到账提醒
   ├─ 跨平台比价    ├─ 下单确认      ├─ 异常预警
   └─ 历史低价      └─ 返利绑定      └─ 提现指导
```

### 各阶段详细能力

| 阶段 | 功能 | 说明 |
|------|------|------|
| 选品 | 热门高返推荐 | 每日更新高返利商品榜单 |
| 选品 | 跨平台比价 | 同商品多平台真实到手价对比 |
| 选品 | 历史价格 | 查看商品历史价格走势，判断是否真低价 |
| 下单 | 优惠券查找 | 自动查找并匹配可用优惠券 |
| 下单 | 链路完整性检查 | 确保返利跟踪链路未断裂 |
| 下单 | 下单流程引导 | 分步引导操作，避免新手操作失误 |
| 售后 | 订单状态同步 | 实时追踪返利订单各阶段状态 |
| 售后 | 提现方式对比 | 对比支付宝、微信、银行卡等提现方式的手续费和到账速度 |
| 售后 | 异常处理 | 退款、失效订单的返利处理指导 |

### 使用场景

1. **新手入门**：第一次接触返利，从注册到提现全程引导
2. **日常购物**：每次购物前查返利、比价、领券，养成省钱习惯
3. **大促备战**：提前选品加购，大促当天快速下单锁定返利
4. **返利管理**：追踪所有待到账返利，确保每笔都不遗漏
5. **提现操作**：返利余额达到门槛后，指导完成提现
