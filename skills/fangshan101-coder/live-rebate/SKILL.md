# 直播返利助手

直播间购物场景的返利优化工具。追踪淘宝直播、抖音直播、快手直播等平台热门直播间的商品信息，查询直播专属价叠加返利后的真实到手价，避免冲动消费的同时最大化省钱效果。

## 触发词

直播返利、直播间省钱、直播购物返利、淘宝直播返利、抖音直播返利、快手直播返利、live rebate、直播带货返利、主播返利

## 计划实现能力

### 核心功能

- **直播商品返利查询**：输入直播间链接或商品链接，查询该商品是否有返利
- **直播价 vs 日常价对比**：对比直播专属价与日常价，判断是否真优惠
- **多平台直播追踪**：追踪淘宝直播、抖音、快手等平台的高返利直播间
- **冲动消费防护**：购买前展示商品历史价格，避免伪低价陷阱

### 支持的直播平台

| 平台 | 返利方式 | 特点 |
|------|---------|------|
| 淘宝直播 | 淘宝联盟佣金 | 返利比例稳定，品类齐全 |
| 抖音直播 | 精选联盟佣金 | 爆款多，限时秒杀 |
| 快手直播 | 好省等渠道 | 下沉市场，性价比高 |
| 京东直播 | 京东联盟佣金 | 3C数码返利较高 |
| 视频号直播 | 微信生态返利 | 新兴渠道，红利期 |

### 直播购物决策流程

```
发现直播间商品
    ↓
查询返利比例
    ↓
对比日常价格（是否真低价？）
    ↓
查看历史价格走势
    ↓
计算直播价+返利的真实到手价
    ↓
值得买 → 生成返利链接下单
不值得 → 提示日常购买更划算
```

### 使用场景

1. **直播间蹲守**：主播上链接时快速查询返利，决定是否下单
2. **直播预告查询**：提前查看即将开播的高返利直播间
3. **防坑指南**：对比直播价和历史价，识别虚假低价
4. **复盘购物**：直播结束后核算已下单商品的返利到账情况
