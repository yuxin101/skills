# 返利提醒官

返利信息实时提醒工具。主动追踪高返利商品的限时活动、关注商品的返利变动、返利订单到账状态以及优惠券过期预警，让用户不错过任何省钱机会，也不遗漏任何一笔待到账返利。

## 触发词

返利提醒、返利通知、返利到账提醒、高返提醒、优惠券过期提醒、rebate alert、返利预警、省钱提醒、返利监控

## 计划实现能力

### 核心功能

- **高返商品提醒**：用户关注的商品类目出现高返利活动时主动提醒
- **价格降价+返利提醒**：关注商品降价且有返利时双重提醒
- **返利到账通知**：返利订单状态变更（确认/结算/到账）时逐笔通知
- **优惠券过期预警**：已领取的优惠券即将过期时提醒使用
- **异常订单预警**：返利订单失效、退款扣除等异常情况及时提醒

### 提醒类型

| 提醒类型 | 触发条件 | 紧急程度 |
|---------|---------|---------|
| 高返活动 | 关注品类出现高返商品 | 普通 |
| 限时秒杀 | 高返商品即将开抢 | 紧急 |
| 降价+返利 | 关注商品降价且有返利 | 较高 |
| 返利到账 | 订单返利状态变更 | 普通 |
| 券即将过期 | 优惠券剩余 24 小时内 | 较高 |
| 订单异常 | 返利订单失效/退款 | 紧急 |
| 提现可用 | 返利余额达到提现门槛 | 普通 |

### 提醒渠道

```
提醒触发
    ├── 推送通知（APP 内）
    ├── 微信服务号消息
    ├── 短信提醒（紧急事项）
    └── 邮件日报（每日汇总）
```

### 使用场景

1. **蹲守降价**：关注某商品，降价+高返利时自动提醒下单
2. **大促抢购**：双11/618 限时秒杀商品开抢前提醒
3. **返利追踪**：下单后不用手动查，到账自动通知
4. **防止遗忘**：已领的券快过期时提醒，已到账的返利提醒提现
