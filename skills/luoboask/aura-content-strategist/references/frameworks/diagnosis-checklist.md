# Diagnosis Checklist

Common content failure patterns. Use in Step 2 to identify fatal weaknesses.

## Hook Failures (封面/标题)

| Symptom | Diagnosis | Fix |
|---------|-----------|-----|
| No number in title | Missing数字钩子 | Add specific number: "5个" "3分钟" |
| No pain point | No emotional entry point | Lead with frustration: "后悔没早知道" |
| Too generic | Zero differentiation | Add specificity: "干皮秋冬" not "护肤" |
| Too long (>20 chars) | Won't display fully on XHS | Ruthlessly cut to core message |
| No curiosity gap | No reason to click | Add incompleteness: "最后一个绝了" |
| Cover text too small | Invisible in feed scroll | ≤8 chars, large and bold |

## Visual Failures (视觉)

| Symptom | Diagnosis | Fix |
|---------|-----------|-----|
| White background + product float | Stock photo vibe | Add real-life context/texture |
| Perfect symmetry | AI smell | Break symmetry, add human elements |
| Too many fonts/colors | Visual noise | ≤2 fonts, ≤3 colors |
| Low contrast cover text | Unreadable in feed | High contrast text + shadow/backdrop |
| Horizontal image | Wrong aspect ratio | 3:4 for XHS, 2:3 for Pinterest |

## Content Failures (内容)

| Symptom | Diagnosis | Fix |
|---------|-----------|-----|
| Lists everything about product | No focus, no hook | Pick ONE angle, go deep |
| All positive, no contrast | Unbelievable | Add minor flaw or caveat |
| No CTA or question | Dead-end engagement | End with question or "save for later" |
| Same format every post | Predictable = boring | Rotate: 清单/对比/教程/故事 |
| No personal voice | Could be anyone's post | Add specific personal detail |

## Platform Mismatch

| Symptom | Diagnosis | Fix |
|---------|-----------|-----|
| XHS post in formal Chinese | Wrong register | 口语化, add 语气词 |
| Pinterest with Chinese text | Wrong language | English only for Pinterest |
| Same content both platforms | Lazy cross-post | Adapt angle, format, length per platform |
| Tutorial on Pinterest as text-only | Wrong format | Make step-by-step long pin |

## Audience Mismatch

| Symptom | Diagnosis | Fix |
|---------|-----------|-----|
| Premium product → student audience | Price mismatch | Reframe as "投资" or find dupes |
| Tech tutorial → 精致女孩 audience | Interest mismatch | Reframe benefit: "效率=更多自由时间" |
| Everyone is target | No target = no one | Pick ONE primary persona |

## Severity Scale

- 🔴 **Fatal**: Will definitely flop (wrong platform specs, no hook, AI smell)
- 🟡 **Serious**: Underperform significantly (weak hook, audience mismatch)
- 🟢 **Fixable**: Quick adjustments needed (tag strategy, timing, minor copy)
