# Xiaohongshu Platform Reference

## Algorithm (2026)

- **Ranking signals**: 完播率 > 互动率 (点赞+收藏+评论) > 分享率
- **Search traffic**: Rising. Title SEO weight increasing. 搜索流量占比持续提升
- **Content format**: 图文笔记回暖, 视频笔记 ≤60s 表现最佳
- **Anti-AI**: Platform flags "疑似AI内容", triggers shadow-ban / limited distribution

## Hard Specs

| Element | Constraint |
|---------|-----------|
| Image ratio | 3:4 (1080×1440px) preferred, 1:1 acceptable |
| Cover text | ≤8 characters, 占画面上1/3或居中 |
| Title | ≤20 characters |
| Body | ≤1000 chars limit, ≤800 optimal for engagement |
| Tags | 5-10 per post |
| Carousel | 1-9 images per post |

## Title Formula

```
[数字/反差/痛点] + [解决方案/情绪钩子]
```

Examples:
- ✅ "月薪3k住出3w的感觉" (反差+利益)
- ✅ "后悔没早知道的5个神器" (痛点+数字)
- ✅ "救命！这面霜也太绝了" (情绪+好奇)
- ❌ "好用的面霜推荐" (无钩子, 无情绪)

## Tag Strategy

Mix three tiers:
- **大词** (百万级搜索): #护肤 #穿搭 → 曝光但竞争大
- **中词** (十万级): #平价面霜 #秋冬穿搭 → 精准流量
- **长尾词** (万级以下): #干皮救星面霜 #小个子冬季穿搭 → 搜索排名容易

## Post Timing

| Time Slot | Audience | Best For |
|-----------|----------|----------|
| 07:00-09:00 | 通勤刷手机 | 轻内容、日常分享 |
| 12:00-13:30 | 午休时间 | 种草、美食 |
| 18:00-20:00 | 下班通勤 | 干货、教程 |
| 21:00-23:00 | 睡前刷手机 | 情感、好物、护肤 |

Weekend peaks: 10:00-12:00, 15:00-17:00

## Comment Strategy

- **首条自评**: 补充细节, 用"打字中途改口"的口吻 ("对了忘说了，这个...")
- **置顶评论**: 引导互动 ("姐妹们你们用的什么？")
- **回复模板**: 热情但不刻板，每条回复略有不同
