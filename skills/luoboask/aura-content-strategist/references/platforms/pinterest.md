# Pinterest Platform Reference

## Algorithm (2026)

- **Ranking signals**: Pin freshness > relevance > engagement > domain authority
- **Search-first**: Pinterest is a visual search engine. SEO > virality.
- **Content format**: Idea Pins for reach, Static Pins for evergreen search traffic
- **Anti-spam**: Duplicate/scraped content deprioritized. Original pins rewarded.

## Hard Specs

| Element | Constraint |
|---------|-----------|
| Image ratio | 2:3 (1000×1500px) standard, up to 1:3 for long pins |
| Title | ≤100 characters |
| Description | ≤500 characters, first 40 chars = search snippet |
| Board name | Keyword-rich, ≤50 chars |
| Alt text | ≤500 chars, descriptive for accessibility + SEO |

## SEO Strategy

### Title
- Natural language, 1-2 long-tail keywords
- ✅ "10 Cozy Fall Outfit Ideas for Petite Women"
- ❌ "Fall Outfits" (too generic, no long-tail)

### Description
- First 2 sentences = most important (shown in search snippets)
- 3-5 keywords woven naturally, NOT keyword-stuffed
- End with soft CTA: "Save for later" / "Click for full guide"

### Board Strategy
- 10-15 boards per account, keyword-rich names
- Each pin → 1 primary board + 2-3 relevant secondary boards
- Board descriptions: 2-3 sentences with keywords

## Pin Design Best Practices

| Element | Guideline |
|---------|-----------|
| Text overlay | Bold headline + smaller subtitle, ≤2 font families |
| Readable on mobile | Text ≥16pt equivalent at mobile viewport |
| Brand consistency | Same 2-3 colors across all pins |
| Image quality | Sharp, well-lit, no pixelation |
| Whitespace | Generous — cluttered pins underperform |

## Post Timing (UTC)

| Time Slot | Audience | Best For |
|-----------|----------|----------|
| 14:00-16:00 UTC | US afternoon | Shopping, lifestyle |
| 20:00-23:00 UTC | US evening | Recipes, DIY, planning |
| Saturday 08:00-11:00 UTC | Weekend browsers | Home decor, travel |

Best days: Saturday > Friday > Sunday

## Link Strategy

- Every pin should link to a destination (blog, shop, landing page)
- Rich Pins (article/product/recipe) get priority distribution
- UTM parameters for tracking: `?utm_source=pinterest&utm_medium=pin&utm_campaign={board}`
