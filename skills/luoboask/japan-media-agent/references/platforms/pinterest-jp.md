# Pinterest 日本版 (Pinterest JP)

日本市場向け Pinterest 戦略ガイド。

## ユーザー統計

| 指標 | 数値 |
|------|------|
| 月間アクティブユーザー | 400 万 + |
| 性別 | 女性 80%、男性 20% |
| 年齢層 | 20-40 代中心（70%） |
| 利用目的 | アイデア収集、保存、ショッピング |

## 人気カテゴリ

1. **料理・レシピ** — 簡単レシピ、お弁当、ダイエット
2. **インテリア** — 北欧風、ミニマル、収納
3. **ファッション** — コーディネート、プチプラ
4. **美容** — スキンケア、メイク、ヘアスタイル
5. **占い・スピリチュアル** — 星占い、タロット、運気アップ
6. **旅行** — 国内旅行、海外旅行、ホテル
7. **ハンドメイド** — DIY、編み物、刺繍

## 投稿仕様

| 要素 | 制限 | 推奨 |
|------|------|------|
| 画像比率 | 2:3, 3:4, 1:1 | 2:3 (1000×1500px) |
| タイトル | 100 字 | 30-50 字 |
| 説明 | 500 字 | 150-300 字 |
| ハッシュタグ | 20 個 | 5-10 個 |
| リンク | 任意 | ブログ/サイトがある場合は必須 |

## 日本語コピーライティング

### タイトル例

```
✅ 水逆期間を乗り切る方法 2026 年版｜運気を守る 7 つのヒント
✅ 【保存版】水星逆行中にやるべきことまとめ
✅ 占星術師が教える！水逆前の準備リスト

❌ 水星逆行について
❌ 運気を上げる方法
```

**ポイント**：
- 数字を入れる（7 つ、2026 年版）
- ベネフィット明確に（運気を守る）
- 信頼性（占星術師が教える）
- 緊急性（保存版）

### 説明例

```
水星逆行中に気をつけたいことまとめ。3 年間占星術を研究してわかった、
通信トラブルや機械の故障から身を守る簡単ルーティン。
次の水逆前にぜひ保存して！メールの返信を一旦保留するテンプレート付き。
占星術初心者さんも参考にしてください。

#水星逆行 #占星術 #スピリチュアル #運気アップ #2026 年運勢
```

**構造**：
1. 共感（気をつけたいこと）
2. 信頼性（3 年間研究）
3. 具体的ベネフィット（簡単ルーティン）
4. CTA（保存して）
5. おまけ（テンプレート付き）
6. ハッシュタグ

## デザインガイド

### 日本向けビジュアル

- **色使い**: 柔和なパステル、和風カラー
- **フォント**: 丸ゴシック、手書き風
- **絵文字**: 適切に使用（✨🌸🎋🍁❄️）
- **余白**: 多め（「間」の美学）
- **画像**: 高品質、明るめ

### 避けるデザイン

- 暗すぎる色（黒背景など）
- 文字が多すぎる
- 英語のみ（日本語必須）
- 低解像度
- 過度に派手

## 成功パターン

### パターン 1: まとめ系

```
タイトル：【保存版】水逆前にやるべき 10 のこと
画像：チェックリスト形式
説明：全部で 10 個、簡単なので今日からできます
CTA: 「保存して水逆前に確認してね」
```

### パターン 2: 比較系

```
タイトル：水逆前 vs 水逆中｜この違いを知っておいて
画像：左右比較
説明：意外と知らない水逆の本当の意味
CTA: 「友達にも教えてあげて」
```

### パターン 3: 体験談系

```
タイトル：水逆でスマホが 3 回壊れた私が学んだこと
画像：実写 + テキスト
説明：実体験だからこそのリアルな対策
CTA: 「同じ経験した人いる？」
```
