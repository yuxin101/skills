# 日本語コピーライティングガイド

Native-level Japanese copywriting for social media.

## 基本原則

### 1. 読み手を意識

```
❌ あなたは水星逆行中に注意が必要です
✅ 水逆中っていろいろありますよね…
✅ みなさん、水逆大丈夫ですか？
```

**ポイント**：
- 「あなた」→「みなさん」「さん」
- 一方的→対話的
- 上から目線→共感

### 2. 具体的なベネフィット

```
❌ 運気が上がります
✅ 明日から使える簡単ルーティン 3 つ
❌ 注意してください
✅ これをやらないと後悔します
```

### 3. 信頼性の提示

```
❌ 私は占いができます
✅ 3 年間毎日星占いをつけてわかったこと
✅ 占星術師 10 年目の私が本気でおすすめ
```

## 文体バリエーション

### 丁寧語（デフォルト）

**用途**: 大多数のユーザー、信頼性重視

```
水星逆行の期間中は、コミュニケーションに注意が必要です。
大切な契約は期間をずらすことをおすすめします。
よかったら試してみてくださいね。
```

**特徴**：
- です/ます調
- 丁寧な言葉遣い
- 信頼感がある
- 年代を問わない

### ため口

**用途**: 10-20 代向け、親近感

```
水逆ヤバい…🌀 通信トラブルマジで多い
契約事とかは絶対避けたほうがいいよ〜
でもこれやっとけば大丈夫！
```

**特徴**：
- だ/である調
- 友達感覚
- 若年層にウケる
- 親近感

### 関西弁

**用途**: 親しみやすさ、個性

```
水逆始まったで〜🌀
携帯とか PC 壊れやすいから気ぃつけてな
契約事は延期した方がええで
```

**特徴**：
- 関西圏に親近感
- 全国でも受け入れられやすい
- 個性が出る
- 注意：やりすぎない

### 専門的

**用途**: 上級者向け、信頼性

```
水星逆行期間中（3/15-4/7）は、
水星が支配するコミュニケーション・移動・機械に注意が必要です。
占星術的には、重要な決定を避け、
内省の時間とすることが推奨されます。
```

**特徴**：
- 専門用語使用
- 正確性重視
- 上級者向け
- 信頼性が高い

## NG 表現（AI 臭い）

### 不自然な接続詞

```
❌ しかし、水星逆行中は注意が必要です
✅ でも、水逆中は気をつけて
✅ ただ、一つだけ注意点があって

❌ さらに、以下のことも重要です
✅ あと、これも大事で
✅ それから、これも忘れずに
```

### 硬い表現

```
❌ 水星逆行はコミュニケーションに影響を与えます
✅ 水逆中って、コミュニケーションうまくいかないことが多いんだ
✅ 水逆のせいで、連絡ミスが増えがち…

❌ 推奨されます
✅ おすすめです
✅ やったほうがいいよ

❌ 私たちはあなたを支援します
✅ よかったら参考にしてください
✅ 応援してます
```

### 機械的な列挙

```
❌ 第一に、第二に、第三に
✅ まず、次に、そして
✅ 1 つ目は、2 つ目は、3 つ目は
✅ とりあえず、あと、最後に
```

## 自然な日本語パターン

### 共感から入る

```
「水逆始まりましたね…🌀」
「また水逆かよって思った人いる？」
「水逆中でなんかうまくいかない…ってことない？」
```

### 体験談を交える

```
「私が実際に水逆中にやらかしたこと」
「3 年間毎日つけてわかったこと」
「占星術を勉強して初めて知った真相」
```

### 具体的なシチュエーション

```
「メール送信前に 2 回確認するだけで OK」
「契約書は水逆明けに送る」
「バックアップだけはやっといて」
```

### 柔らかい CTA

```
「よかったら試してみて」
「保存して水逆前に確認してね」
「友達にも教えてあげて」
「コメントで教えてくれたら嬉しいです」
```

- [ ] 絵文字は適切か（多すぎ/少なすぎ）
- [ ] 専門用語は解説があるか
- [ ] 読み手への呼びかけはあるか

- [ ] 季節感はあるか

## 改善例

### 改善前（AI 臭い）

```
水星逆行はコミュニケーションに影響を与えます。
以下のことに注意してください：
第一に、メールの確認をしましょう。
第二に、契約事は避けましょう。
第三に、バックアップを取りましょう。
```

### 改善後（自然）

```
水逆中って、コミュニケーションうまくいかないことが多いんだ🌀

私が実際にやってる対策はこれ：
・メールは送信前に 2 回確認
・契約事は水逆明けに延期
・データはこまめにバックアップ

これだけでだいぶ楽になるよ！
```

## 季節の挨拶

```
1-2 月：新春の候、厳寒の候
3-4 月：春暖の候、桜花の候
5-6 月：新緑の候、梅雨の候
7-8 月：盛夏の候、暑中の候
9-10 月：新秋の候、紅葉の候
11-12 月：初冬の候、師走の候
```

### カジュアルな季節感

```
1-2 月：「寒い日が続きますが」「お正月はいかがでしたか」
3-4 月：「桜の季節になりました」「新生活スタート」
5-6 月：「ゴールデンウィーク」「梅雨入りしましたね」
7-8 月：「暑い日が続きます」「夏休み」
9-10 月：「過ごしやすい季節」「秋ですね」
11-12 月：「寒くなってきました」「年末」
```
