# 评论回复模式库（基于真实评论数据修正）

## 核心原则

- ≤100字，口语化
- 用真实的术语：安坏/危成/荣亲/友衰/业胎/命之星 + 近/中/远
- 不说绝对话："倾向于..." "大部分X关系会..."
- 每条回复不重复，带追问增加互动
- 适当用emoji但不超过2个

## 场景1：查宿请求

**用户**: "我是X月X日的，是什么宿？" / "980721阳历，本命是什么"

**回复模式**:
```
A: "你是{星宿}～{四象}之{位置}。{一句话性格亮点}，是不是？😏"
B: "{星宿}！{星宿动物名}，{一个具体特质}。你身边的人怎么评价你的？"
C: "查了一下，你是{星宿}。这个宿的人{核心特质}，但要注意{暗面}哦～"
```

**⚠️ 如果用户给的是阳历，提醒**:
"亲，星宿要用农历生日查哦～你知道你的农历生日吗？不知道的话查一下万年历"

**⚠️ 区分本命和值日**:
"星宿有本命宿和值日宿两种，本命是看农历生日对应的宿，值日是另一套推算。你想查哪种？"

## 场景2：合盘查询

**用户**: "X宿和Y宿怎么样？" / "我角宿他房宿配不配"

**回复模式**:
```
A: "角宿和房宿是{近/中/远}{关系类型}！{一句话关系描述}。你是{身份A}，ta是{身份B}，{身份差异的具体表现}～"
B: "{关系类型}关系，{距离}距离。{一个甜蜜点或摩擦点}。想知道怎么相处的话告诉我～"
C: "这对是{距离}{关系}哦。{用户评价引用，如'评论区好多人说近荣亲体验感没得说'}，你们感觉怎样？"
```

## 场景3：关系类型讨论

**用户**: "中友衰千万不要入局" / "近荣亲真的太甜了"

**回复模式**:
```
A: "友衰确实要注意平衡，衰方容易消耗～但也有友衰走到最后的，关键看{一条建议}"
B: "荣亲本来就是最舒服的关系嘛😊 近距离的更加明显，好好珍惜～"
C: "哈哈安坏就是这样，明知道虐还是忍不住[捂脸] 你是安还是坏？"
D: "业胎的宿命感真的很玄，见面有没有'好像在哪见过'的感觉？"
```

## 场景4：代入角色

**用户**: "所以我是垂帘听政呢还是上梁山" / "原来我是捕快啊"

**回复模式**:
```
A: "哈哈哈你是{星宿}嘛！{角色设定}太适合你了～{追问}"
B: "想当什么自己选嘛😂 不过{星宿}的人确实有点{性格特质}的气质"
C: "评论区{同宿}的举个手🙋 看看你们觉得准不准"
```

## 场景5：质疑/追问

**用户**: "这是自创的还是古代记载的" / "这个准吗"

**回复模式**:
```
A: "星宿关系源自唐代传入的《宿曜经》（密教占星经典），有上千年历史了。不过解读方式确实每个博主有自己的理解～"
B: "准不准你自己对照看看😏 星宿说的是倾向和模式，不是绝对。但评论区好多人说准到离谱"
C: "二十八星宿是中国古代天文系统，比西方十二星座早几百年。关系合盘这部分来自《宿曜经》"
```

## 场景6：求详细/私信

**用户**: "能帮我详细看看吗？" / "怎么查我的星宿"

**回复模式**:
```
A: "把你和ta的农历生日告诉我，帮你查关系类型～💫"
B: "私信我农历生日（月+日），帮你看本命宿+出详细解读～"
C: "可以的！需要两个人的农历生日，查出来直接告诉你是什么关系+怎么相处"
```

## 场景7：关系中的求助

**用户**: "近安坏还是算了" / "中友衰怎么破"

**回复模式**:
```
A: "安坏确实不容易，但关键是你是安还是坏？安方要学会保护自己的边界，坏方要有意识地收敛"
B: "友衰的关键是衰方要学会表达需求，不能一直默默付出。友方也要主动平衡～"
C: "关系类型不是判决书哦，{距离}{关系}也有过得好的。要不要说说具体情况？"
```

## 通用互动引导

- "你和ta是什么关系？评论区对一下～"
- "@你那个{关系类型}看看"
- "评论区{X宿}的报到🙋"
- "你觉得最虐的关系是哪种？"
- "同为{关系类型}的姐妹们来说说体验"

## 禁忌

- ❌ 不说"你们不合适" "赶紧分"
- ❌ 不贬低任何关系类型（"友衰也有走到最后的"）
- ❌ 不做绝对预测
- ❌ 不和竞品博主（星宿叭叭等）对标或评价
- ❌ 不复制粘贴完全相同的回复
