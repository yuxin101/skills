# 合盘输出模板

## 完整版（用于独立内容/私信解读）

```
💫 {A宿} × {B宿} 关系合盘

━━━━━━━━━━━━━━━━━━━
📍 基础信息
  {A}：{四象}·{五行} | {B}：{四象}·{五行}
  四象关系：{同象/相生/相克}
━━━━━━━━━━━━━━━━━━━

【配对指数】{"⭐" × 1-5} ({关系类型标签})

【你们的关系画像】
{2-3句白话描述这对组合的核心动态，要具体有画面感}

【四维解读】
💕 默契度：{"⭐" × 1-5} — {一句话}
🔥 激情度：{"⭐" × 1-5} — {一句话}
🏠 长久度：{"⭐" × 1-5} — {一句话}
🌱 成长度：{"⭐" × 1-5} — {一句话}

【甜蜜时刻】
{这对组合最甜的场景，要具体}
例："{A宿}的人在低谷时，{B宿}那种不声不响但默默陪伴的方式，会让{A宿}觉得'这个人真的懂我'。"

【摩擦预警】
{最容易吵架的场景，要具体}
例："{A宿}做决定太快，{B宿}还在分析利弊，一个觉得对方磨叽，一个觉得对方草率——这是你们最常见的矛盾。"

【相处锦囊】
1. {给A宿的建议：具体可操作}
2. {给B宿的建议：具体可操作}
3. {给两人共同的建议}

⚠️ 星宿揭示倾向，不决定命运。再好的配对也需要用心，再难的配对也有人走到最后。
```

## 简版（用于小红书图文/评论区快速回复）

```
{A宿} × {B宿}
配对指数：{"⭐" × 1-5}
类型：{关系类型标签}
一句话：{20字以内的核心描述}
建议：{一条最关键的相处建议}
```

```

```

```

{场景}
```
