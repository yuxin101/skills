# 农历生日→星宿查询表

## 查询方法

横轴 = 农历月份（正月~腊月）
纵轴 = 农历日期（初一~三十）

## 使用说明

1. 确认用户的**农历**生日，不是公历
2. 如果用户只知道公历，需要先转换为农历（提醒用户查万年历）
3. 二月只有29天（闰年）或28天，廿九/三十标"—"
4. 此表为传统流传版本，不同门派可能有细微差异

## 快速换算提示

- 每月初一的星宿固定循环：室→奎→胃→毕→参→鬼→柳→张→角→氐→心→箕
- 每日递推一宿，按二十八宿顺序循环
- 顺序：角亢氐房心尾箕 → 斗牛女虚危室壁 → 奎娄胃昴毕觜参 → 井鬼柳星张翼轸
