# 二十八星宿关系推算规则（基于宿曜经三九秘宿法）

## 推算原理

宿曜经原文使用**二十七宿**（不含牛宿），但中国传统使用**二十八宿**。
本表采用中国二十八宿序列。

### 二十八宿顺序（环形）

```
角→亢→氐→房→心→尾→箕→斗→牛→女→虚→危→室→壁→奎→娄→胃→昴→毕→觜→参→井→鬼→柳→星→张→翼→轸→(回到角)
```

### 三九法规则

以本命宿为起点（第0位），按顺序数：

**第一轮（命/荣/衰/安/危/成/坏/友/亲）**：

| 位置 | 关系 | 身份说明 |
|------|------|---------|
| 0 | **命** | 命之星（同宿） |
| +1 | **荣** | 对方是你的荣（滋养你） |
| +2 | **衰** | 对方是你的衰（消耗你） |
| +3 | **安** | 对方是你的安（你安于对方） |
| +4 | **危** | 对方是你的危（你有危于对方） |
| +5 | **成** | 对方是你的成（成就你） |
| +6 | **坏** | 对方是你的坏（破坏你） |
| +7 | **友** | 对方是你的友（友善于你） |
| +8 | **亲** | 对方是你的亲（亲近你） |

**第二轮（业/荣/衰/安/危/成/坏/友/亲）**：

| 位置 | 关系 |
|------|------|
| +9 | **业** |

| +12 | 安 |
| +13 | 危 |
| +14 | 成 |
| +15 | 坏 |
| +16 | 友 |
| +17 | 亲 |

**第三轮（胎/荣/衰/安/危/成/坏/友/亲）**：

| 位置 | 关系 |
|------|------|
| +18 | **胎** |
| +19 | 荣 |
| +20 | 衰 |
| +21 | 安 |
| +22 | 危 |
| +23 | 成 |

## 六种关系的映射

宿曜经的九种关系对应小红书流行的六种关系：

| 宿曜经术语 | 小红书流行术语 | 配对方式 |
|-----------|--------------|---------|
| 安 + 坏 | **安坏** | 你的安是对方的坏，反之亦然 |
| 危 + 成 | **危成** | 你的危是对方的成，反之亦然 |
| 荣 + 亲 | **荣亲** | 你的荣是对方的亲，反之亦然 |
| 友 + 衰 | **友衰** | 你的友是对方的衰，反之亦然 |
| 业 + 胎 | **业胎** | 互为业胎 |
| 命 | **命之星** | 同宿 |

## 距离判断

根据宿在三轮中的位置判断距离：

| 轮次 | 位置范围 | 距离 |
|------|---------|------|
| 第一轮 | +1 ~ +8 | **近** |
| 第二轮 | +9 ~ +17 | **中** |
| 第三轮 | +18 ~ +26 | **远** |

**业胎特殊**：业(+9)和胎(+18)各只出现一次，**没有远近之分**。

## 双向关系

**关键**：A对B的关系 ≠ B对A的关系。

如果A是角宿，B是亢宿：
- A→B：亢是角的+1位 → B是A的**荣**
- B→A：角是亢的+27位(即-1位) → A是B的**亲**
- 因此角×亢 = **近荣亲**关系，角是亲方，亢是荣方

## 合盘查询方法

给定两人星宿A和B：

1. 在28宿环上，从A数到B的步数 = N
2. N mod 9 的余数决定关系类型：
   - 0 = 命(同宿) 或 业(+9) 或 胎(+18)
   - 1 = 荣
   - 2 = 衰
   - 3 = 安
   - 4 = 危
   - 5 = 成
   - 6 = 坏
   - 7 = 友
   - 8 = 亲
3. N 的大小决定距离：
   - 1~8 = 近
   - 9 = 业（无距离）
   - 10~17 = 中
   - 18 = 胎（无距离）
   - 19~26 = 远
4. 同时计算 B→A 的关系（从B数到A）
5. 合并得出关系对：安×坏=安坏，荣×亲=荣亲，友×衰=友衰，危×成=危成
