# 牌阵模板

## 单牌阵

**适用**：快速回答、每日运势、是/否问题
**抽牌**：1张
**位置**：当前能量/核心信息

## 三牌阵

**适用**：过去-现在-未来、情况-挑战-建议
**抽牌**：3张

| 位置 | 含义 |
|------|------|
| 1 (左) | 过去/起因/你的处境 |
| 2 (中) | 现在/核心/当下状态 |
| 3 (右) | 未来/建议/发展方向 |

## 关系牌阵

**适用**：两人关系分析
**抽牌**：4张

| 位置 | 含义 |
|------|------|
| 1 (左) | 你在关系中的状态 |
| 2 (右) | 对方在关系中的状态 |
| 3 (上) | 关系现状/核心动态 |
| 4 (下) | 建议/发展方向 |

## 选择牌阵

**适用**：二选一决策
**抽牌**：3张

| 位置 | 含义 |
|------|------|
| 1 (左) | 选择A的走向 |
| 2 (右) | 选择B的走向 |
| 3 (中) | 综合建议/你内心倾向 |

## 凯尔特十字

**适用**：深度全面分析（复杂问题）
**抽牌**：10张

| 位置 | 含义 |
|------|------|
| 1 | 当前处境 |
| 2 | 眼前挑战（横跨1上方） |
| 3 | 潜意识/真实想法 |
| 4 | 过去的影响 |
| 5 | 有意识的目标 |
| 6 | 近期未来 |
| 7 | 你的态度/自我认知 |
| 8 | 外部环境/他人看法 |
| 9 | 希望或恐惧 |
| 10 | 最终结果 |
