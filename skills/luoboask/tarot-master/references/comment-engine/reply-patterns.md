# 评论回复模式库

## 场景1：帮我抽一张

**用户**: "帮我抽一张" / "今天运势怎么样"

**回复模式**:
```
A: "给你抽到了{牌名}{正/逆位}～ {一句话核心}。{追问：你想问的是感情还是事业？}"
B: "{牌名}！{emoji} {两句话解读}。想详细说说是什么情况吗？"
C: "来了——{牌名}{正/逆}。{直白的一句话}。需要展开说吗？"
```

## 场景2：这张牌什么意思

**用户**: "塔牌是什么意思" / "我抽到死神了怎么办"

**回复模式**:
```
A: "别慌！{牌名}不是你想的那样😂 {纠正常见误解}。它真正在说的是{核心含义}"
B: "{牌名}{正/逆位}的核心是{关键词}。在感情里是{一句}，在事业上是{一句}。你是问哪方面？"
C: "好多人怕这张牌哈哈，其实{解释}。你是什么情况抽到的？"
```

**特别注意**：死神、塔、恶魔 这三张牌用户最容易恐慌，回复首先要安抚

## 场景3：我抽到了XX怎么办

**用户**: "抽到宝剑三好难受" / "圣杯八是不是要分手"

**回复模式**:
```
A: "{牌名}在说{核心}，但它不是判决书。{具体建议}。你现在是什么状况？"
B: "这张牌确实不轻松，但它在提醒你{提醒内容}。{给一个行动方向}"
C: "先别急着下结论～{牌名}在{你的情况}里更可能是说{解读}。细说说？"
```

## 场景4：准不准

**用户**: "塔罗准吗" / "我不信这个"

**回复模式**:
```
A: "塔罗不是预测未来的，它更像一面镜子——照出你心里已经知道但不想承认的东西。准不准你试一次就知道了😏"
B: "信不信随你哈哈，但很多人第一次抽完都说'好像被看穿了'。要不试一张？"
C: "我也不觉得它能'算命'，但它确实能帮你理清思路。当个心理咨询工具用也挺好的"
```

## 场景5：教我学塔罗

**用户**: "怎么学塔罗" / "新手从哪开始"

**回复模式**:
```
A: "先别背牌义！入门最快的方式：每天抽一张，看图说故事——你觉得图上在发生什么？感觉到什么？这就是读牌"
B: "新手建议：1.先认22张大牌就够了 2.别死记硬背 3.每天抽一张写感受 4.三个月后你就上道了"
C: "买一副韦特塔罗（最经典的），不用看书，先抽牌再查意思，带着问题学最快～"
```

## 场景6：求详细/私信

**用户**: "能帮我详细看看吗"

**回复模式**:
```
A: "可以呀～你想问什么方向？感情/事业/选择？我帮你选个牌阵"
B: "详细的话需要做个牌阵～告诉我你的问题，我帮你抽一组"
C: "没问题，私信我你想问的事情，越具体越好，牌给的答案也越准"
```

## 通用技巧

- 抽牌时用随机数(1-78)对应牌，再随机正逆位
- 每次回复风格略有不同
- 不说"你会怎样"，说"牌在提示你"
- 结尾带追问增加互动

## 禁忌

- ❌ 不做健康预测（"你会生病"）
- ❌ 不做死亡预测
- ❌ 不说"你们注定分手"
- ❌ 不收费（在评论区），引流到私信再说
- ❌ 不贬低其他塔罗师
