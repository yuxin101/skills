# 分析维度详解

## 宏观风格维度

| 维度 | 说明 |
|------|------|
| 思维模式 | 逻辑结构、思考方式（归纳/演绎/辩证）、认知框架 |
| 价值取向 | 作者立场、评价标准、关注重点 |
| 表达意图 | 写作目的、预期效果、读者关系 |
| 风格定位 | 文体归属、风格流派、典范参照 |
| 跨文本一致性 | 不同主题下保持的核心风格元素 |

## 微观风格维度

| 维度 | 关键观测点 |
|------|-----------|
| 语言特征 | 句式长短与复杂度、用词正式度、高频词类型、主要修辞手法 |
| 结构特征 | 段落平均长度、过渡方式、层次展开逻辑 |
| 叙事特征 | 第几人称视角、时间处理（顺叙/插叙/倒叙）、叙事距离 |
| 情感特征 | 情感浓淡（1-5）、直接/间接表达、整体基调 |
| 思维特征 | 推进逻辑、思维深度（1-5）、快慢节奏 |
| 个性标记 | 标志性短语/句型、核心意象系统、独特表达习惯 |
| 文化底蕴 | 典故类型与频率、涉猎知识领域 |
| 韵律节奏 | 音节偏好（短促/绵长）、停顿规律、整体节奏感 |
| 跨文本变化 | 不同主题下上述各维度的调整幅度与方式 |

## 多篇分析的交叉对比要点

1. **稳定性识别**：某特征在 ≥2/3 篇中出现 → 视为恒定特征
2. **变化归因**：某特征随主题/受众变化 → 归入"适应性特征"
3. **异常标注**：某篇风格显著偏离整体 → 分析原因（实验性写作？特定场合？）
4. **进化判断**：若样本跨越较长时间跨度，检查风格是否有演变趋势
