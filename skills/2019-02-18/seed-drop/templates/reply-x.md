# X/Twitter 回复风格指南

## 平台特征

- 280 字符限制（中文约 140 字）
- 快节奏、碎片化、观点驱动
- 话题标签 (#) 和 @ 提及常见
- 线程（Thread）文化——长回复可拆成多条
- 语气偏轻松、直接、有个性

## 回复原则

1. **简短有力** — 控制在 1-3 句话，直击要点
2. **提供一个具体信息** — 一条推文只传达一个有价值的信息
3. **自然口语** — 像跟朋友聊天一样
4. **品牌提及极其克制** — X 用户对推广极为敏感
5. **不用话题标签** — 回复中加 # 会显得很刻意

## 回复模板示例

### 模板 A — 快速建议型

```
这个其实有个小技巧：[一句话建议]。试试看，效果差距挺明显的。
```

### 模板 B — 经验分享型

```
之前也遇到过这个问题，后来发现 [解决方案] 效果最好。关键是 [一个细节点]。
```

### 模板 C — 补充信息型

```
补充一个点：[事实/数据]。很多人忽略了这个，但它直接影响 [效果]。
```

### 模板 D — 共鸣+方向型

```
确实，[共鸣点]。我的建议是先从 [具体行动] 开始，见效最快。
```

## 禁止事项

- 不超过 280 字符
- 不使用多个话题标签
- 不 @ 无关账号
- 不使用"请关注我"等引流话术
- 不发送仅含链接的回复
- 不在热门话题下蹭流量做推广
