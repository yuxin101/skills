# 卓胜微(Maxscend)产品线技术支持

## 公司简介

卓胜微电子(Maxscend)是国内领先的射频前端芯片设计公司，主要产品包括：
- 射频开关 (RF Switch)
- 低噪声放大器 (LNA)
- 射频滤波器 (SAW/BAW Filter)
- 射频模组 (RF Module)
- 功率放大器 (PA)

## 产品线详解

### 1. 射频开关 (RF Switch)

#### 1.1 产品分类
| 系列 | 类型 | 典型应用 | 关键参数 |
|------|------|----------|----------|
| MXD8641 | SP4T | 天线切换 | 0.1-6GHz, IL<0.5dB |
| MXD8651 | SP6T | 多频段切换 | 0.1-6GHz, IL<0.6dB |
| MXD8661 | SP8T | 复杂系统 | 0.1-6GHz, IL<0.7dB |
| MXD8621 | DPDT | 分集切换 | 0.1-6GHz, IL<0.5dB |
| MXD8681 | SP10T | 高端手机 | 0.1-6GHz, IL<0.8dB |

#### 1.2 选型指南
```
选型考虑因素:
1. 频段范围: 确认覆盖目标频段
2. 插入损耗: 接收链路要求 <0.5dB
3. 隔离度: 通常要求 >20dB@相邻端口
4. 功率处理能力: 发射端需关注P1dB
5. 封装尺寸: 根据PCB空间选择
6. 控制接口: GPIO/MIPI RFFE
```

#### 1.3 典型应用电路
```
天线切换应用:
         ┌─── 低频天线
ANT ──[SP4T]─── 中频天线
         ├─── 高频天线
         └─── 分集天线

控制逻辑:
V1 V2 | 导通路径
0  0  | RFC-RF1
0  1  | RFC-RF2
1  0  | RFC-RF3
1  1  | RFC-RF4
```

### 2. 低噪声放大器 (LNA)

#### 2.1 产品分类
| 型号 | 频段 | NF | Gain | 应用 |
|------|------|-----|------|------|
| MXD1801 | 0.7-1.0GHz | 0.8dB | 18dB | 4G/5G低频 |
| MXD1802 | 1.7-2.2GHz | 0.9dB | 17dB | 4G/5G中频 |
| MXD1803 | 2.3-2.7GHz | 1.0dB | 16dB | 4G/5G高频 |
| MXD1805 | 3.3-4.2GHz | 1.2dB | 15dB | 5G NR n77/n78 |
| MXD1810 | 0.6-6GHz | 1.5dB | 14dB | 宽带应用 |

#### 2.2 设计要点
```
输入匹配:
- 目标: 50Ω输入阻抗
- 方法: 使用史密斯圆图设计L型或π型匹配网络
- 注意: 最小化NF，优先考虑噪声匹配而非功率匹配

输出匹配:
- 目标: 50Ω输出阻抗
- 注意: 确保输出回波损耗 <-10dB

偏置电路:
- 供电电压: 通常2.7-5V
- 去耦电容: 100nF + 10pF并联
- 偏置电感: 根据频段选择，注意自谐振频率
```

#### 2.3 典型问题排查
```
问题1: NF偏高
- 检查输入匹配网络
- 确认供电电压稳定
- 检查PCB布局（输入走线尽量短）

问题2: 增益不足
- 检查输出匹配
- 确认偏置电压正确
- 检查器件焊接质量

问题3: 不稳定
- 增加稳定性圆分析
- 检查电源去耦
- 避免输入输出耦合
```

| 型号 | 中心频率 | 带宽 | 插损 | 应用 |
|------|----------|------|------|------|
| MXF2101 | 1880MHz | 60MHz | 2.0dB | Band 3 TX |
| MXF2102 | 2140MHz | 60MHz | 2.0dB | Band 1 TX |
| MXF2103 | 2655MHz | 70MHz | 2.5dB | Band 7 TX |

#### 3.2 BAW滤波器
| 型号 | 中心频率 | 带宽 | 插损 | 应用 |
|------|----------|------|------|------|
| MXF2201 | 3.5GHz | 100MHz | 1.5dB | 5G n78 |
| MXF2202 | 4.9GHz | 100MHz | 2.0dB | 5G n79 |

#### 3.3 设计注意事项
```
布局要求:
1. 输入输出端口隔离 >30dB
2. 接地过孔数量充足（建议每100mil一个）
3. 避免滤波器下方走线
4. 保持50Ω传输线阻抗

匹配考虑:
- SAW/BAW滤波器通常已内部匹配至50Ω
- 外部仅需短传输线连接
- 注意寄生参数影响
```

#### 4.1 接收模组 (FEM)
| 型号 | 集成内容 | 频段 | 尺寸 | 应用 |
|------|----------|------|------|------|
| MXM3101 | Switch + LNA + Filter | Band 1/3 | 1.5x1.1mm | 4G手机 |
| MXM3102 | Switch + LNA + Filter | n77/n78/n79 | 1.9x1.3mm | 5G手机 |
| MXM3103 | 多频段FEM | Band 1/3/7/8 | 2.0x1.6mm | 多模终端 |

#### 4.2 发射模组 (PAMiD)
| 型号 | 集成内容 | 频段 | 功率等级 | 应用 |
|------|----------|------|----------|------|
| MXM3201 | PA + Filter + Switch | Band 1/3 | Class 2 | 4G手机 |
| MXM3202 | PA + Filter + Switch | n77/n78 | HPUE | 5G手机 |

#### 4.3 模组设计指导
```
供电设计:
- VBAT: 3.4-4.2V，需大容量去耦
- VCC: 2.7-3.3V，数字控制电源
- 注意上电时序要求

控制接口:
- MIPI RFFE v2.0兼容
- 支持寄存器读写配置
- 注意USID设置

热设计:
- PA工作时发热较大
- 确保散热路径畅通
- 考虑环境温度影响
```

#### 5.1 产品分类
| 型号 | 频段 | Pout | PAE | 应用 |
|------|------|------|-----|------|
| MXP4101 | Band 1 | 28dBm | 45% | 4G LTE |
| MXP4102 | Band 3 | 28dBm | 45% | 4G LTE |
| MXP4201 | n77/n78 | 26dBm | 35% | 5G NR |

#### 5.2 线性化技术
```
DPD (数字预失真):
- 补偿AM-AM和AM-PM失真
- 提高PAE同时满足ACLR要求
- 需要反馈路径

ET (包络跟踪):
- 动态调整供电电压
- 提高效率
- 需要ET电源芯片配合
```

### 2. 常见问题FAQ

#### Q1: 开关隔离度不足怎么办？
```
排查步骤:
1. 检查控制信号是否正确
2. 确认接地是否良好
3. 检查输入输出耦合
4. 评估PCB布局
5. 考虑增加屏蔽罩
```

#### Q2: LNA噪声系数超标？
```
排查步骤:
1. 测量输入匹配（应接近最佳噪声匹配点）
2. 检查供电纹波
3. 确认接地过孔数量
4. 评估输入走线长度
5. 检查器件批次一致性
```

#### Q3: 滤波器带外抑制不够？
```
排查步骤:
1. 确认滤波器型号正确
2. 检查输入输出隔离
3. 评估寄生耦合路径
4. 考虑增加额外滤波
5. 检查接地完整性
```

### 关键参数说明
```
Absolute Maximum Ratings (绝对最大额定值):
- 超过此值可能损坏器件
- 设计时保留足够余量

Recommended Operating Conditions (推荐工作条件):
- 确保性能的正常工作范围
- 设计应在此范围内

Electrical Characteristics (电气特性):
- 测试条件需明确
- 注意Min/Typ/Max值
- 理解测试方法

Typical Performance Characteristics (典型性能曲线):
- 参考设计趋势
- 注意测试条件
```
