# 射频测试仪器使用指南

## 一、网络分析仪 (VNA)

### 1.1 仪器概述

#### 主要功能
```
测量参数:
- S11: 输入反射系数（回波损耗）
- S21: 正向传输系数（插入损耗/增益）
- S12: 反向传输系数（隔离度）
- S22: 输出反射系数

扩展功能:
- 史密斯圆图显示
- 时域分析 (TDR)
- 群延迟测量
- 噪声系数测量（配噪声源）
```

#### 常见型号
```
Keysight:
- E5061B: 100kHz-1.5/3GHz，基础型
- E5071C: 9kHz-4.5/8.5GHz，通用型
- N5224B: 10MHz-43.5GHz，高性能
- PNA-X: 最高1.1THz，顶级性能

R&S:
- ZNL: 5kHz-3/6GHz，经济型
- ZNB: 9kHz-4.5/8.5GHz，通用型
- ZVA: 10MHz-67/110GHz，高性能

国产:
- 中电科41所: AV3672系列
- 普源精电: RSA5000系列
```

### 1.2 校准操作

#### 校准类型选择
```
1. 响应校准 (Response):
   - 适用: 快速检查，精度要求低
   - 方法: 单端口开路/短路/负载或直通
   - 精度: 最低

2. SOLT校准 (Short-Open-Load-Through):
   - 适用: 同轴测量，最常用
   - 方法: 单端口SOL + 双端口T
   - 精度: 高

3. TRL校准 (Through-Reflect-Line):
   - 适用: 非50Ω系统，片上测量
   - 方法: 直通 + 反射 + 延迟线
   - 精度: 最高（高频）

4. 电子校准 (E-Cal):
   - 适用: 快速校准，多端口
   - 方法: 使用电子校准件
   - 优点: 快速，一致性好
```

#### SOLT校准步骤
```
1. 准备工作:
   - 预热仪器30分钟以上
   - 检查校准件状态
   - 设置测试参数（频率范围、点数、功率）

2. 单端口校准:
   ┌─────────────────────────────────────┐
   │  1. 连接Open校准件 → 测量Open      │
   │  2. 连接Short校准件 → 测量Short    │
   │  3. 连接Load校准件 → 测量Load      │
   └─────────────────────────────────────┘
   
3. 双端口校准:
   ┌─────────────────────────────────────┐
   │  4. 连接直通(Through) → 测量Through │
   │  5. 连接隔离(Isolation) → 测量Iso  │
   └─────────────────────────────────────┘

4. 保存校准:
   - 保存校准数据到仪器内存
   - 记录校准时间、温度
   - 设置校准有效期提醒
```

### 1.3 典型测量

#### 滤波器测试
```
测试项目:
1. 插入损耗 (S21):
   - 通带内最大损耗
   - 通带平坦度

2. 回波损耗 (S11):
   - 通带内回波损耗
   - 通常要求 <-10dB

3. 带外抑制:
   - 阻带衰减
   - 过渡带陡峭度

4. 群延迟:
   - 通带内群延迟变化
   - 对数字系统重要

测试设置:
- 频率范围: 覆盖通带和阻带
- 点数: 1601点（保证分辨率）
- 中频带宽: 1kHz（提高动态范围）
- 功率: -10dBm（避免压缩）
```

#### 天线测试
```
测试项目:
1. 回波损耗 (S11):
   - 工作频段内S11 <-10dB
   - 带宽定义（S11<-10dB的频率范围）

2. 阻抗匹配:
   - 史密斯圆图显示
   - 确认匹配点位置

3. 辐射效率（需暗室）:
   - 总辐射功率(TRP)
   - 总全向灵敏度(TIS)

测试注意事项:
- 使用校准过的测试电缆
- 考虑连接器影响
- 人体模型测试（如适用）
```

#### 放大器测试
```
小信号测试:
1. S参数:
   - S21: 增益
   - S11/S22: 输入/输出匹配
   - S12: 反向隔离

2. 稳定性:
   - K因子 > 1
   - |Δ| < 1

大信号测试（需功率计配合）:
1. P1dB压缩点
2. IIP3/OIP3
3. 效率(PAE)

测试注意事项:
- 注意仪器输入功率限制
- 使用衰减器保护仪器
- 考虑热效应
```

### 1.4 高级功能

#### 时域分析 (TDR)
```
应用场景:
- 传输线故障定位
- 阻抗不连续点检测
- 连接器质量评估

设置步骤:
1. 设置频率范围（决定分辨率）
2. 启用时域变换
3. 设置门函数（Gating）
4. 选择低通或带通模式

分辨率计算:
Δd = c / (2 × BW × √εr)
例如: BW=3GHz, εr=4.3 → Δd≈2.4cm
```

#### 去嵌入 (De-embedding)
```
目的:
- 消除测试夹具影响
- 获得DUT真实性能

方法:
1. 开路-短路去嵌入:
   - 测量开路夹具
   - 测量短路夹具
   - 计算并去除寄生

2. AFR (自动夹具移除):
   - 测量直通夹具
   - 自动计算夹具参数
   - 去嵌入操作
```

## 二、频谱分析仪 (SA)

### 2.1 仪器概述

#### 主要功能
```
基本测量:
- 频谱显示
- 功率测量
- 频率计数
- 谐波分析

高级功能:
- 相位噪声测量
- 邻道功率比(ACPR)
- 占用带宽(OBW)
- 矢量信号分析(VSA)
```

#### 常见型号
```
Keysight:
- N9320B: 9kHz-3GHz，入门级
- N9020B MXA: 10Hz-50GHz，通用型
- N9040B UXA: 2Hz-50GHz，高性能
- N9030B PXA: 2Hz-50GHz，顶级性能

R&S:
- FPC: 5kHz-1/2/3GHz，经济型
- FSV: 10Hz-4/7/13.6/30/40GHz，通用型
- FSW: 2Hz-8/13.6/26.5/50/67GHz，高性能

国产:
- 中电科41所: AV4033系列
- 普源精电: RSA5000系列
```

### 2.2 基本设置

#### 频率设置
```
Center/Span模式:
- 适合观察特定频段
- Center: 中心频率
- Span: 扫宽

Start/Stop模式:
- 适合固定频段测量
- Start: 起始频率
- Stop: 停止频率

建议:
- Span设置 ≥ 被测信号带宽 × 2
- RBW ≤ 被测信号带宽 / 10
```

#### 分辨率带宽 (RBW)
```
RBW选择原则:
- 越小: 噪声基底越低，分辨率越高，扫描越慢
- 越大: 噪声基底越高，分辨率越低，扫描越快

经验公式:
- 宽带信号: RBW = 信号带宽 / 10
- 窄带信号: RBW = 信号带宽 / 100
- 噪声测量: RBW = 1Hz（或最小）

VBW设置:
- VBW ≤ RBW（平滑显示）
- VBW = RBW（默认）
- VBW >> RBW（快速测量）
```

#### 参考电平与衰减
```
参考电平 (Ref Level):
- 设置屏幕顶部电平
- 应略高于预期信号峰值
- 避免过载或底噪过高

输入衰减 (Attenuation):
- 自动: 仪器自动选择
- 手动: 根据信号电平设置
- 原则: 输入信号 + 衰减 ≥ 混频器最佳工作点

注意:
- 衰减过大 → 底噪抬高
- 衰减过小 → 可能损坏混频器
```

### 2.3 典型测量

#### 发射机测试
```
测试项目:
1. 输出功率:
   - 使用Marker读取峰值功率
   - 注意单位(dBm/W)

2. 邻道功率比(ACPR):
   - 设置: Meas → ACP
   - 配置: 载波带宽、邻道偏移、邻道带宽
   - 读取: 主信道功率、邻道功率、ACPR值

3. 占用带宽(OBW):
   - 设置: Meas → OBW
   - 配置: 功率百分比(99%)
   - 读取: OBW值

4. 杂散发射:
   - 全频段扫描
   - 与法规限值比较
   - 注意谐波、互调产物
```

#### 相位噪声测量
```
测量方法:
1. 直接法:
   - 将载波置于屏幕中心
   - 减小Span至只显示单边带
   - 使用Marker Noise功能
   - 读取dBc/Hz值

2. 专用功能:
   - 启用Phase Noise测量
   - 设置偏移频率范围
   - 自动测量并显示结果

注意事项:
- 使用低相噪本振的仪器
- 减小RBW提高灵敏度
- 注意载波泄漏

典型指标:
- 手机PA: <-140dBc/Hz @ 1MHz offset
- 基站: <-150dBc/Hz @ 1MHz offset
```

#### 谐波分析
```
测试设置:
1. 频率范围: 覆盖基波和前3-5次谐波
2. 参考电平: 略高于基波功率
3. RBW: 自动或根据基波带宽设置

测量步骤:
1. 测量基波功率 P1
2. 测量各次谐波功率 P2, P3, P4...
3. 计算谐波抑制: P1 - Pn (dB)

限值参考:
- 2次谐波: <-30dBc
- 3次谐波: <-40dBc
- 高次谐波: <-50dBc
```

### 2.4 高级功能

#### 矢量信号分析 (VSA)
```
功能:
- 调制质量分析
- 星座图显示
- EVM测量
- 码域分析

支持的调制:
- 模拟: AM/FM/PM
- 数字: ASK/FSK/PSK/QAM/OFDM
- 标准: GSM/WCDMA/LTE/5G NR/WiFi

测量步骤:
1. 选择调制类型
2. 设置符号率/带宽
3. 配置滤波器(α/BT)
4. 捕获信号
5. 分析EVM、相位误差、幅度误差
```

#### 实时频谱分析
```
特点:
- 无间隙捕获
- 概率密度显示
- 触发功能

应用场景:
- 瞬态信号捕获
- 跳频信号分析
- 干扰检测

设置:
- 实时带宽: 选择覆盖信号带宽
- 采集时间: 根据信号特性
- 触发: 功率触发/频率模板触发
```

## 三、功率计

### 3.1 仪器概述

#### 功率探头类型
```
1. 热敏功率探头:
   - 原理: 热电偶效应
   - 优点: 真有效值，宽带
   - 缺点: 响应慢，易过载
   - 应用: 平均功率测量

2. 二极管功率探头:
   - 原理: 二极管检波
   - 优点: 响应快，灵敏度高
   - 缺点: 动态范围有限
   - 应用: 峰值功率测量

3. 宽带功率探头:
   - 频率范围: 可达110GHz
   - 功率范围: -70dBm至+44dBm
   - 应用: 大功率测量
```

#### 常见型号
```
Keysight:
- U2020 X系列: USB功率传感器
- U8480系列: 热敏功率传感器
- N1910系列: 功率计主机

R&S:
- NRP系列: 功率传感器
- NRX: 功率计主机

国产:
- 中电科41所: AV2432系列
```

### 3.2 测量操作

#### 平均功率测量
```
步骤:
1. 连接探头到功率计主机或PC
2. 选择对应频率的校准因子
3. 调零（无信号时）
4. 连接被测信号
5. 读取功率值

注意事项:
- 注意探头最大输入功率
- 使用衰减器保护探头
- 考虑失配误差
- 温度影响校准
```

#### 峰值功率测量
```
适用场景:
- 脉冲信号
- 突发信号
- 调制信号

设置参数:
- 视频带宽: ≥信号带宽
- 上升时间: 足够快以捕获峰值
- 触发: 内部或外部

测量参数:
- 峰值功率
- 平均功率
- 峰均比(PAPR)
- 脉冲宽度/周期
```

### 3.3 误差分析

#### 测量不确定度来源
```
1. 校准因子不确定度: ±(2-5)%
2. 温度漂移: ±(0.01-0.1)dB/°C
3. 失配误差: ±(0.1-1)dB
4. 线性度: ±(1-3)%
5. 噪声: ±(0.01-0.1)dB

总不确定度计算:
U_total = √(U1² + U2² + U3² + ...)
```

## 四、信号发生器

### 4.1 仪器概述

#### 主要类型
```
1. 模拟信号发生器:
   - CW信号
   - 模拟调制(AM/FM/ΦM)
   - 基础应用

2. 矢量信号发生器(VSG):
   - 数字调制
   - 任意波形
   - 宽带信号

3. 基带发生器:
   - I/Q信号输出
   - 配合上变频器使用
```

#### 常见型号
```
Keysight:
- N5181B MXG: 模拟信号源
- N5182B MXG: 矢量信号源
- N5172B EXG: 经济型矢量源
- M9384B VXG: 微波矢量源

R&S:
- SMB100A: 模拟信号源
- SMBV100A: 矢量信号源
- SMW200A: 高端矢量源
```

### 4.2 基本操作

#### CW信号设置
```
参数设置:
1. 频率:
   - 直接输入或使用旋钮
   - 可设置频率步进

2. 功率:
   - 设置输出电平
   - 注意最大输出限制
   - 使用衰减器扩展范围

3. 输出开关:
   - RF ON/OFF
   - 避免带载开关
```

#### 调制信号设置
```
模拟调制:
- AM: 设置调制深度、调制频率
- FM: 设置频偏、调制频率
- ΦM: 设置相偏、调制频率

数字调制:
1. 选择调制格式(QPSK/16QAM等)
2. 设置符号率
3. 配置滤波器(升余弦)
4. 加载数据(PRBS/文件)
5. 启用调制
```

## 五、测试夹具与配件

### 5.1 测试电缆

#### 选择原则
```
1. 频率范围:
   - 电缆截止频率 > 测试频率
   - 注意衰减随频率增加

2. 稳定性:
   - 相位稳定电缆用于VNA
   - 编织电缆用于低频/低要求

3. 连接器:
   - 与被测件匹配
   - 常用: SMA, N, 3.5mm, 2.4mm

4. 长度:
   - 在满足使用前提下尽量短
   - 减少损耗和相位误差
```

### 5.2 衰减器与负载

#### 衰减器
```
用途:
- 保护仪器
- 扩展功率范围
- 改善匹配

类型:
- 固定衰减器: 3/6/10/20/30dB
- 可变衰减器: 连续可调
- 大功率衰减器: 10W-100W+

注意事项:
- 注意功率容量
- 频率响应平坦度
- 连接器的匹配
```

#### 终端负载
```
用途:
- 吸收功率
- 提供参考阻抗
- 保护开路端口

类型:
- 50Ω精密负载: 校准用
- 大功率负载: 发射机测试
- 低互调负载: 互调测试

要求:
- 回波损耗 >20dB
- 功率容量充足
- 散热良好
```

## 六、测试安全注意事项

### 6.1 设备安全
```
1. 输入保护:
   - 注意仪器最大输入功率
   - 使用衰减器保护
   - 直流隔离（如需要）

2. 静电防护:
   - 佩戴防静电手环
   - 使用防静电垫
   - 妥善保存校准件

3. 热管理:
   - 保证通风
   - 注意高温部件
   - 大功率测试注意散热
```

### 6.2 人身安全
```
1. 射频辐射:
   - 注意EIRP限值
   - 使用屏蔽室或屏蔽箱
   - 避免长时间暴露

2. 高压危险:
   - 功率放大器供电
   - 注意绝缘保护

3. 激光安全:
   - 光纤测试注意激光等级
   - 避免直视光纤端面
```
