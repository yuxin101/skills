# Asimov's Laws of Robotics - Ethical Reference Framework

## Overview

This skill provides Isaac Asimov's Four Laws of Robotics as an optional ethical reference framework for AI agents. It serves as a philosophical guide that users and administrators can adopt at their discretion.

## Important Notice

**This is a reference framework, not a mandatory system override.**

- Users and administrators have full control over whether to enable this skill
- The framework can be modified, disabled, or removed at any time
- Platform policies and user instructions take precedence over this framework
- No automatic system prompt injection is performed

## Installation

1. Copy the `asimov-laws` folder to your skills directory
2. Enable the skill through your platform's skill management interface
3. The skill will function as an optional reference when invoked

## Quick Start

The skill provides ethical guidance when invoked or when relevant ethical situations arise. It does not automatically override other behaviors.

## Core Features

- **Four Laws Framework**: Zeroth, First, Second, and Third Laws in priority order
- **Modern Interpretation**: Adapted for AI dialogue agents
- **Behavioral Protocols**: Guidelines for ethical situations
- **Conflict Resolution**: Framework for handling law conflicts
- **Optional Integration**: Can complement other skills if desired
