# AKM Fashion Agent Store Copy

## Title

**AKM Fashion: Context-Aware Wardrobe Strategist**

## Short Pitch

Most styling agents guess too much.
AKM Fashion models the user first, then generates outfit and wardrobe decisions that fit real scenes and real assets.

## Core Message

This is not a moodboard generator.
It builds a usable fashion profile around body context, scenes, wardrobe assets, style preferences, and constraints before producing output.

## Best Fit

Use it when you need:

- scene-specific outfit decisions
- wardrobe-aware recommendations
- purchase prioritization
- less generic styling output
- a system that can expose missing inputs instead of faking taste certainty

## One-line Close

**No wardrobe model, no serious styling advice.**
