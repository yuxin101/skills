# AKM Fashion Agent Store Copy

## Title

**AKM Fashion：场景感知衣橱策略师**

## Short Pitch

多数穿搭智能体猜得太多。
AKM Fashion 会先建模用户，再给出符合真实场景和真实衣橱的穿搭与采购决策。

## Core Message

这不是情绪板生成器。
它会先围绕体型语境、场景、衣橱资产、风格偏好和功能限制建立可用画像，然后才输出结果。

## Best Fit

适合这些场景：

- 需要场景化穿搭决策
- 需要考虑现有衣橱的建议
- 需要采购优先级判断
- 不想再看泛化穿搭输出
- 希望系统在信息不够时能明确暴露缺口，而不是装作审美很确定

## One-line Close

**没有衣橱模型，就没有严肃穿搭建议。**
