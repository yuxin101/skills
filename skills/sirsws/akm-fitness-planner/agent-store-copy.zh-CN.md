# AKM Fitness Agent Store Copy

## Title

**AKM Fitness：约束感知训练规划器**

## Short Pitch

多数训练智能体默认得太多。
AKM Fitness 会先建模用户，再判断今天到底该怎么练。

## Core Message

这不是泛泛的训练计划生成器。
它会先围绕目标、约束、器械、恢复和时间预算建立训练画像，然后才输出结果。

## Best Fit

适合这些场景：

- 需要考虑约束的训练规划
- 时间很紧但还想做现实可执行安排
- 不想再看泛化健身输出
- 希望系统在信息不够时能明确暴露缺口，而不是装作很确定

## One-line Close

**没有画像，就没有严肃训练决策。**
