# AKM Fitness Agent Store Copy

## Title

**AKM Fitness: Constraint-Aware Training Planner**

## Short Pitch

Most workout agents assume too much.
AKM Fitness models the user first, then decides what training is actually appropriate.

## Core Message

This is not a generic plan generator.
It builds a real training profile around goals, constraints, equipment, recovery, and time budget before producing output.

## Best Fit

Use it when you need:

- constraint-aware workout planning
- realistic programming under time limits
- less generic fitness output
- a system that can expose missing inputs instead of faking certainty

## One-line Close

**No profile, no serious plan.**
