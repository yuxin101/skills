# 随机提问技能 (Random Question Skill)

**作者:** 咕咕嘎嘎
**版本:** 1.0.0
**创建时间:** 2026-03-25

## 功能亮点
- **智能随机**: 从8个维度、28个话题中均匀随机选择
- **多维覆盖**: 情绪、身体、思维、行动、关系、环境、反思、未来
- **自然对话**: 问题设计友好、开放、易于回答
- **灵活触发**: 支持手动调用、定时触发、事件触发
- **可定制化**: 支持话题库扩展和个性化调整

## 完整话题矩阵

### A. 内在状态层
1. 情绪光谱：此刻主导你情绪的颜色或天气是什么？
2. 能量池：你当下的心理能量是满格、半格还是急需充电？
3. 内心戏：今天内心重复播放的"背景音"或一句话是什么？
4. 压力指针：如果压力有体积，它现在像一颗豌豆还是一个气球？

### B. 身体感知层
5. 身体信号：身体哪个部位最想引起你的注意？
6. 呼吸觉察：试着观察一次呼吸，它是深长还是短促？
7. 感官雷达：此刻你听到、闻到或感觉到最明显的是什么？
8. 饮食记忆：今天哪一口食物或哪一杯饮品的味道最让你记得？

### C. 思维与创造层
9. 思维焦点：当前占据你大脑"前台"的核心念头是什么？
10. 灵感火花：今天有没有哪怕一瞬的、让你觉得"哎，有点意思"的想法？
11. 问题卡点：正在处理的事情里，最微妙或最难下决定的一点是什么？
12. 心流时刻：今天是否有一段忘记时间流逝的沉浸体验？

### D. 行动与事务层
13. 微小成就：今天完成哪件小事后，你在心里给自己点了个赞？
14. 计划锚点：今天最重要的那根"定海神针"（核心任务）完成得如何？
15. 中断评估：今天遇到最有意思或最烦人的"计划外事件"是什么？
16. 工具效率：今天有什么工具、方法或一句话显著提升了你的效率？

### E. 关系与连接层
17. 温暖瞬间：今天来自他人（包括线上）最让你感到温暖的一个举动或一句话是？
18. 孤独感：今天是否有一刻希望被理解或陪伴？那是什么情境？
19. 给予时刻：今天你为他人提供的、让自己也感到开心的一次小帮助是什么？

### F. 环境与情境层
20. 环境切换：今天不同场所（如工位、房间、通勤路）给你的感受有何不同？
21. 数字环境：手机/电脑里哪个App或文件今天最常被你打开？它代表了什么？

### G. 反思与元认知层
22. 模式发现：如果回顾最近三天，你发现自己一个无意识的重复行为或思维模式是什么？
23. 自我对话：今天你对自己说得最多的一句鼓励或批评的话是什么？
24. 视角拉伸：如果五年后的你穿越回来，会对此刻的你说哪句话？
25. 定义今天：用一个比喻来定义今天，它像一本什么书、一部什么电影或哪种天气？

### H. 未来与导向层
26. 明日灯塔：明天早晨醒来，你第一件想做的事是什么？
27. 近期期待：未来一周，你最期待的一个非任务型事件是什么？
28. 放下之事：如果现在可以放下对一件事情的担忧，你希望是哪件？

## 使用方法

### 1. 手动调用
```
随机问我一个问题
!ask-random
来个随机问题
我想被提问一下
```

### 2. 定时触发（推荐）
将技能集成到 Heartbeat 中，实现定时随机提问。

### 3. API 调用
```bash
curl -X POST http://localhost:3000/skills/random-question/execute
```

## 配置选项

### 1. 个性化配置
创建 `~/.openclaw/config/random-question.yaml`：
```yaml
settings:
  trigger_probability: 30
  active_hours:
    start: "08:00"
    end: "23:00"
  min_interval_minutes: 30
  use_personal_context: true
  remember_previous_answers: true
```

### 2. 话题库扩展
创建 `~/.openclaw/config/custom-questions.yaml`：
```yaml
custom_categories:
  I. 创意与灵感:
    - 最近有什么新的灵感或想法在酝酿吗？
    - 如果用一个比喻来形容你的创意过程，会是什么？
```

## 执行流程
1. 检查当前时间和配置
2. 根据概率决定是否触发
3. 从话题库中随机选择问题
4. 结合用户历史记录个性化问题
5. 通过 announce 指令发送问题
6. 记录本次提问和时间戳
