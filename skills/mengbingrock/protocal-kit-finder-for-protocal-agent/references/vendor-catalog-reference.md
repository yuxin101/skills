# Vendor Catalog Reference

## Common Vendor Domains and Product URL Patterns

### Thermo Fisher Scientific / Life Technologies / Invitrogen / Applied Biosystems / Gibco
- **Domain:** thermofisher.com
- **Product URL pattern:** `https://www.thermofisher.com/order/catalog/product/[CATALOG#]`
- **Catalog number format:** Typically 7-8 digits (e.g., `4472908`, `15596026`) or alphanumeric (e.g., `AM1906`)
- **Notes:** Many legacy brands (Invitrogen, Applied Biosystems, Ambion, Gibco) now consolidated under Thermo Fisher

### QIAGEN
- **Domain:** qiagen.com
- **Product URL pattern:** `https://www.qiagen.com/us/products/discovery-and-translational-research/[category]/[product-name]` or search by catalog number
- **Catalog number format:** Typically 5-6 digits (e.g., `74104`, `79254`, `205311`)
- **Notes:** RNeasy, QIAamp, miRNeasy product families

### New England Biolabs (NEB)
- **Domain:** neb.com
- **Product URL pattern:** `https://www.neb.com/en-us/products/[CATALOG#]`
- **Catalog number format:** Letter + 4 digits (e.g., `M0368`, `E7760`, `M0253`)
- **Notes:** Known for enzymes, cloning, and RT-PCR kits

### Bio-Rad
- **Domain:** bio-rad.com
- **Product URL pattern:** `https://www.bio-rad.com/en-us/sku/[CATALOG#]`
- **Catalog number format:** Typically `###-####` format (e.g., `172-5271`, `170-8880`)
- **Notes:** CFX qPCR instruments, SsoAdvanced and iTaq master mixes

### Sigma-Aldrich / MilliporeSigma / EMD Millipore
- **Domain:** sigmaaldrich.com
- **Product URL pattern:** `https://www.sigmaaldrich.com/US/en/product/sigma/[CATALOG#]`
- **Catalog number format:** Letter(s) + digits (e.g., `C2432`, `I9516`, `T9424`)
- **Notes:** Commodity chemicals, molecular biology grade solvents

### Takara Bio / Clontech
- **Domain:** takarabio.com
- **Product URL pattern:** `https://www.takarabio.com/products/[category]/[product-name]`
- **Catalog number format:** Typically `###.#` or alphanumeric (e.g., `639505`, `RR420A`)
- **Notes:** PrimeScript RT kits, TB Green (SYBR equivalent), SMART-Seq

### Promega
- **Domain:** promega.com
- **Product URL pattern:** `https://www.promega.com/products/[category]/[product-name]`
- **Catalog number format:** Letter + 4 digits (e.g., `A1120`, `M3001`, `A6001`)
- **Notes:** GoScript RT, GoTaq qPCR, Maxwell extraction systems

### IDT (Integrated DNA Technologies)
- **Domain:** idtdna.com
- **Product URL pattern:** `https://www.idtdna.com/pages/products/[category]`
- **Catalog number format:** Custom oligos — no standard catalog number; pre-designed assays have numeric IDs
- **Notes:** Custom primers/oligos, PrimeTime qPCR assays
