# Style Guide — Common Journal Formats

## Epidemiology / GBD-style Papers (e.g., Lancet, Lancet Neurology, Lancet Global Health)

- **Abstract**: Structured, 4 labels (Background / Methods / Results / Interpretation or Conclusions), ≤300 words
- **Introduction**: 3–4 tight paragraphs; final paragraph = explicit study aims list as (i)/(ii)/(iii)
- **Methods**: Sub-headings without numbers; Data Source → Disease Definition → Outcomes → Statistical Analysis (sub-sub-headings for each method)
- **Results**: Sub-headings match methods order; no inline interpretation
- **Discussion**: Opens with 1-sentence "core finding" paragraph; 4–5 paragraphs; limitations in penultimate paragraph; conclusion in final paragraph (no separate section)
- **References**: Vancouver numbered; journal abbreviations per NLM
- **Register**: Passive voice for Methods; active allowed in Discussion; no contractions; British English for Lancet family

## Clinical Neurology Papers (e.g., Brain, Annals of Neurology, JNNP)

- **Abstract**: Structured (Objective / Methods / Results / Conclusions), ≤250 words
- **Introduction**: 2–3 paragraphs; ends with "We therefore …" aim statement
- **Discussion**: Usually 5–6 paragraphs; opens with principal finding; subtypes/mechanisms discussed explicitly
- **Register**: More active voice; shorter sentences

## GBD Epidemiology Paper Checklist

Key items reviewers look for:

- [ ] ICD codes stated explicitly (ICD-9 and ICD-10)
- [ ] GBD year version stated (e.g., GBD 2023)
- [ ] Age-standardization method stated (GBD global standard population)
- [ ] Joinpoint software version and permutation test stated
- [ ] ARIMA: state auto.arima + AIC/BIC selection criteria
- [ ] Decomposition: cite at least one GBD decomposition methods paper
- [ ] Ethics: GBD IRB exemption (University of Washington) stated
- [ ] All rate metrics stated as "per 100,000 population"
- [ ] 95% CI/PI reported for all key estimates
- [ ] P-value threshold stated (two-sided P < 0.05)
