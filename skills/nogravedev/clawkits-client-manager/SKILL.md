# Client Manager — Automated Client Communication

Monitor client channels, auto-triage by urgency, draft responses using conversation history, and schedule follow-ups. Built for freelancers and agencies managing multiple clients.

## What This Skill Does

### 1. Request Triage
Automatically categorizes incoming messages:
- 🔴 **Urgent** — Service down, deadline today, billing issue
- 🟡 **Normal** — Feature request, question, update needed
- 🟢 **Low** — FYI, thank you, casual check-in

### 2. Smart Response Drafting
- Pulls context from previous conversations
- Matches client's communication style (formal/casual)
- Includes relevant project details and timeline references
- Suggests next steps and action items

### 3. Follow-Up Scheduling
- Tracks promises made ("I'll get back to you by Friday")
- Sets reminders for pending responses
- Alerts when a client hasn't heard back in X days
- Escalation rules for overdue items

### 4. Client Context Tracking
Maintains per-client memory:
- Project status and milestones
- Communication preferences
- Past issues and resolutions
- Key contacts and roles
- Contract/payment status

## Setup

1. Configure your communication channels (email, Slack, Discord, WhatsApp)
2. Add client profiles with basic context
3. Set urgency rules and response time targets
4. Agent starts monitoring and triaging automatically

## Usage

- "Check for urgent client messages"
- "Draft a response to [client]'s last message"
- "What's pending with [client]?"
- "Schedule a follow-up with [client] for Friday"

- "Summarize my client interactions this week"
