# On-Page SEO Checklist

Apply this before publishing any article.

## Title Tag
- [ ] Primary keyword near the start (within 50 chars)
- [ ] Under 60 characters total
- [ ] Unique — never duplicate titles across pages
- [ ] Numbers or year ("2026") improve CTR

## Meta Description
- [ ] Exactly 150-155 characters
- [ ] Primary keyword included
- [ ] Compelling CTA or curiosity hook
- [ ] Matches search intent

## Heading Structure
- [ ] Single H1 per page (article title)
- [ ] Logical H2/H3 hierarchy
- [ ] Keywords in H2s where natural
- [ ] No orphan headings (every H2 has content)

## Content Quality
- [ ] Word count: 1500-2500 words for competitive topics, 800-1200 for low competition
- [ ] First 100 words: keyword + clear value proposition
- [ ] Keyword density: 0.5-2% (1-2 instances per 100 words)
- [ ] Related keywords throughout (not just the primary)
- [ ] Data/stats cite sources

## Technical
- [ ] Canonical URL set correctly
- [ ] Internal links: 2-5 to other site pages
- [ ] External links: 2-3 to authoritative sources
- [ ] Images have alt text
- [ ] Page loads fast (no heavy images above fold)
- [ ] Mobile readable

## Engagement Signals
- [ ] Table of contents for long articles (1500+ words)
- [ ] FAQ section at bottom
- [ ] Clear conclusion with takeaway
- [ ] Related articles section

## Common Mistakes
- Keyword stuffing — don't do it
- Thin content (<800 words) for competitive topics
- No internal links — siloed content doesn't rank
- Ignoring search intent — don't write tutorials for comparison queries
