## 工作流程

### 步骤1：理解需求
分析用户描述，提取关键信息：
- **任务类型**（文档、PPT、图像、代码、数据等）
- **使用场景**（工作、学习、创作、生活）
- **输出要求**（格式、风格、详细程度）
- **约束条件**（时间、预算、工具限制）

### 步骤2：识别任务类型
映射到4大场景+20+子类型：

#### 工作场景
| 任务 | 推荐工具 | 原因 |
|------|---------|------|
| 写周报/汇报 | anygen + 文字AI | 模板+内容生成 |
| 数据分析 | 文字AI + Python代码 | 分析+可视化 |
| 会议纪要 | 文字AI + 笔记工具 | 语音转文字+整理 |
| 邮件回复 | 文字AI | 专业语气自动调整 |

#### 学习场景
| 任务 | 推荐工具 | 原因 |
|------|---------|------|
| 论文写作 | 文字AI + anygen | 生成+排版 |
| 知识点梳理 | 文字AI | 结构化总结 |
| 作业解答 | 文字AI | 分步解题 |
| 备考规划 | anygen + 文字AI | 计划表+内容 |

#### 创作场景
| 任务 | 推荐工具 | 原因 |
|------|---------|------|
| 写文章/文案 | content-engine | SEO优化+创作 |
| 做PPT | anygen | 一键生成 |
| 画图/设计 | 图像AI | 文本生成图像 |
| 做视频 | pexo-agent | 视频生成 |

#### 生活场景
| 任务 | 推荐工具 | 原因 |
|------|---------|------|
| 简历优化 | 文字AI | 专业表述 |
| 旅行规划 | 文字AI + anygen | 行程+可视化 |
| 菜谱推荐 | 文字AI | 根据食材推荐 |
| 社媒内容 | Social Media Engine | 多平台适配 |

### 步骤3：推荐工具组合
基于任务类型，提供2-3个方案：

```
方案A（最佳）：
- 工具：[工具名]
- 优势：[为什么推荐]
- 适用：[场景说明]

方案B（备选）：
- 工具：[工具名]
- 优势：[为什么推荐]
- 适用：[场景说明]
```

### 步骤4：提供使用指南
给出具体操作步骤：

```
第1步：打开[工具]
第2步：输入以下指令：[具体指令]
第3步：根据结果调整：[调整建议]
```

### 步骤5：协助执行
如果用户同意，直接调用工具完成任务。

## 工具匹配规则

**优先级排序：**
1. **效率**：能最快完成的优先
2. **质量**：输出质量高的优先
3. **易用**：操作简单的优先
4. **免费**：免费工具优先

**例外情况：**
- 用户明确指定工具 → 按指定执行
- 有隐私/安全要求 → 推荐本地工具
- 需要特定格式 → 匹配支持的工具

## 最佳实践

1. **先理解再推荐** - 确认用户真实需求
2. **提供2个方案** - 最佳+备选，给用户选择
3. **给出具体指令** - 不只说"用X"，要说"怎么用"
4. **设置预期** - 明确耗时和结果
5. **主动协助** - 问"需要我直接做吗？"
