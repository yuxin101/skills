# Failures - 失败的教训和避免的错误

本文件记录失败的教训，防止重复犯错。

## 密码存储

- **【2026-03-16】德明：** ❌ 费曼最初使用了明文存储密码，被审查发现
  - **教训：** 密码必须哈希后存储，使用bcrypt
  - **已修复：** 是

## JWT验证

- **【2026-03-16】德明：** ❌ 费曼忘记验证JWT签名，导致安全漏洞
  - **教训：** 使用jwt.verify()而不是jwt.decode()
  - **已修复：** 是

## 边界case

- **【2026-03-16】德明：** ❌ 费曼没有处理空字符串输入，导致崩溃
  - **教训：** 所有输入都需要验证，包括空值、null、undefined
  - **已修复：** 是
