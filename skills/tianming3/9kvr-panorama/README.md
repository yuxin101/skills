# 9kvr-panorama

`9kvr-panorama` 是芊云全景面向 AI 的全景业务技能说明，帮助 AI 助手准确理解用户意图，并完成全景内容管理相关任务。

## 产品定位

面向 AI 场景的智能全景系统。

## 关于芊云全景

芊云全景（9kvr.cn）是一个面向 AI 的全景系统，帮助个人与企业快速完成全景作品的生产、编辑、交付与传播。  
无论是展馆展示、门店导览、文旅项目还是企业介绍，芊云全景都致力于把复杂的全景制作流程变成更清晰、更高效的业务动作。

## 这是什么

这是一个“全景作品运营助手”技能集，核心目标是让用户用自然语言就能完成全景项目中的常见工作，例如：

- 管理作品与场景
- 管理素材与热点
- 配置背景音乐与语音讲解
- 查看作品评分与优化方向
- 获取接入指引与代码建议
- 配置账号登录信息（uid / token）

## 我们的目的

- 让新手也能快速上手全景项目管理
- 让 AI 更懂全景业务语境，减少沟通损耗
- 让高频任务标准化，提升团队协作效率
- 让内容交付从“会用工具”升级到“会做结果”

## 能力总览

### 1. 账号配置

- 配置登录信息（uid、token）
- 校验当前账号状态
- 支持“先配置再执行后续任务”的工作流

### 2. 作品管理

- 查看作品列表
- 查询作品详情
- 更新作品基础信息
- 基于素材创建作品

### 3. 素材管理

- 上传与整理素材
- 查看素材信息
- 更新素材描述
- 删除或下载素材

### 4. 场景与热点管理

- 查看作品场景
- 添加场景跳转热点
- 添加文本热点
- 删除热点并回查结果

### 5. 音频能力

- 搜索或智能匹配背景音乐
- 挂载背景音乐到作品
- 生成语音讲解并挂载

### 6. 评分与诊断

- 查看作品评分
- 结合评分结果做优化动作

### 7. 接入指导

- 小程序接入建议
- Web 接入建议
- 现有系统接入建议
- 常见接入场景的示例指引

## 如何触发（提示词示例）

以下是推荐给 AI 助手的自然语言提示词写法。  
不需要刻意说出技能名，直接描述你的目标即可。

### 账号配置类

- 帮我配置登录信息，uid 是 `<UID>`，token 是 `<TOKEN>`
- 帮我登录全景账号，uid=`<UID>` token=`<TOKEN>`

### 作品管理类

- 帮我查一下我的全景作品列表
- 把作品 `<WORK_ID>` 的名称改成“春季展厅”

### 素材与场景类

- 帮我上传这张全景图并创建作品
- 给作品 `<WORK_ID>` 增加一个跳转热点

### 音频类

- 给作品 `<WORK_ID>` 匹配一段背景音乐
- 把这段讲解词生成语音并挂到作品 `<WORK_ID>`

### 评分与优化类

- 查看作品 `<WORK_ID>` 的评分，并告诉我优先优化什么

### 接入指导类

- 给我一份 React 项目接入全景的建议
- 小程序接入全景需要准备什么

## 这套技能有什么用

- 降低沟通成本：用户只需描述目标，不必记复杂操作流程
- 提升任务成功率：AI 能更准确匹配业务场景，减少误操作
- 提高执行效率：同类任务可复用同一套提示词模板
- 适合新手用户：将复杂流程转换为清晰、可理解的业务动作

## 适用人群

- 全景内容运营人员
- 项目交付与实施人员
- 需要快速配置全景内容的产品/运营同学
- 希望通过 AI 完成日常全景管理任务的新手用户

## 使用建议

- 先说明目标，再给关键标识（如作品 ID）
- 涉及删除、覆盖时，先让 AI 复述目标再执行
- 多步骤任务可拆分为“先查询、再修改、后验证”三段描述
