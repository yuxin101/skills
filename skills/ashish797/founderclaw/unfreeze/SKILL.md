# Unfreeze — Clear Freeze Boundary

Remove the edit restriction set by `freeze`.

## How to Use

1. Clear the locked directory from context.
2. Announce: "Freeze removed. Edits unrestricted across all directories."

That's it. Simple command, simple effect.
