# Tianchuang Finance Knowledge Base

A specialized skill for answering Tianchuang Financial questions with strict document retrieval.

## Purpose
Provides authoritative answers to financial policy questions by searching the complete Tianchuang financial document corpus.

## Key Features
- **Verbatim-only responses**: Returns only exact text from source documents
- **Comprehensive coverage**: Searches all 8 official Tianchuang financial documents
- **Financial terminology aware**: Optimized for accounting, budgeting, payment, and compliance terms
- **Regulatory precision**: Maintains legal and procedural accuracy

## Document Coverage
- Sample reimbursement policies
- Budget management systems
- Foreign exchange regulations
- Payment documentation requirements
- Fund management procedures
- Risk control frameworks
- Approval authority guidelines
- Customer credit term processes

## Usage Guidelines
1. Ask specific questions about Tianchuang Financial policies
2. The system will search and return exact document excerpts
3. No interpretation or summary - only original text
4. Multiple relevant excerpts returned on separate lines

## Technical Specifications
- Built on OpenClaw's knowledge base framework
- Implements multi-stage relevance scoring
- Preserves document structure and context
- Handles PDF extraction artifacts
