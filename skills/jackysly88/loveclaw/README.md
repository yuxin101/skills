# LoveClaw - 八字缘分匹配

智能八字匹配技能，寻找有缘人

## 功能概述

- 🤖 **智能引导**：对话式收集用户信息
- 🔮 **八字计算**：根据出生日期时间自动计算八字
- 💕 **缘分匹配**：基于五行相生相克匹配用户
- 📱 **隐私保护**：手机号仅匹配成功后展示
- ⏰ **每日报告**：每晚8点推送匹配结果

## 安装

```bash
clawhub install loveclaw
```

安装后定时任务自动配置完成。

## 匹配规则

### 匹配条件

1. **同城必须**：双方必须在同一城市
2. **双向确认**：A喜欢B，B也必须喜欢A
3. **匹配阈值**：双向分数都 ≥70% 才算成功

### 匹配分数

- 五行相生：100%
- 五行比和：80%
- 五行相克：50%

### 选择逻辑

- 每日只匹配1人（分数最高者）
- 同城用户优先匹配
- 异地用户不参与匹配

## 每日流程

- **19:50** — 系统自动执行每日匹配
- **20:00** — 推送匹配报告给用户

### 无匹配通知

如果当日没有同城双向匹配，发送：
> 🌙 今日缘分未到...  
> 命运的齿轮继续转动，请期待月老明日的光临 ✨

## 使用命令

| 命令 | 功能 |
|------|------|
| `开启匹配` / `报名` | 开始报名流程 |
| `开始报名` | 正式填写信息 |
| `我的档案` | 查看报名信息 |
| `今日匹配` | 查看今日匹配结果 |
| `取消报名` | 删除个人数据 |

## 数据存储

- **云端**：阿里云 TableStore + OSS
  - 用户档案：TableStore
  - 照片：OSS（永久有效）

## 隐私说明

1. 手机号仅匹配成功后对匹配对象可见
2. 照片上传至云端存储
3. 用户可随时输入「取消报名」删除所有数据
