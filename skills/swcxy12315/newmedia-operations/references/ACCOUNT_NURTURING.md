# 账号养号操作手册

## 账号资料完善清单

### 通用资料设置

| 项目 | 要求 | 示例 |
|------|------|------|
| 昵称 | 品牌名+品类+核心关键词，简洁易记，≤20字 | 「XX护肤·祛痘专家」 |
| 头像 | 高清PNG，≥300×300px，辨识度高 | 品牌Logo / 人设IP |
| 简介 | 价值定位+核心服务+合规引导，≤150字 | 见下方模板 |
| 背景图 | 品牌视觉一致，可包含二维码（平台允许情况下） | 品牌主色调海报 |

### 简介写作模板
```
[核心价值定位，1句话]
✅ [服务/产品特点1]
✅ [服务/产品特点2]
✅ [服务/产品特点3]
📩 [合规引导，如：私信「资料」领取XX]
```

**禁止在简介中出现**：
- 微信号/二维码（抖音/小红书）
- 最高级词汇（最、第一、国家级）
- 承诺性表述（保证、100%效果）

### 平台差异化设置

#### 抖音
- 实名认证：个人号用身份证，企业号用营业执照
- 行业标签：选择最接近的二级品类
- 主页链接：可设置商品橱窗或联系方式

#### 小红书
- 认证：可申请专业号或品牌账号
- 标签：选择1-3个行业标签
- 置顶笔记：放转化率最高的笔记

#### 视频号
- 与微信公众号绑定（同主体）
- 开通视频号小店（如有电商需求）
- 关联公众号增加流量入口

## 每日养号计划

### 时间表（建议三个时段）

| 时段 | 时间 | 行为 | 数量 |
|------|------|------|------|
| 早间 | 09:00-09:30 | 搜索核心关键词+浏览内容 | 5-8条 |
| 午间 | 12:00-12:20 | 点赞同行爆款视频 | 10-15个 |
| 晚间 | 19:00-21:00 | 评论互动+收藏+转发 | 评论3-5条，收藏5-8个，转发2-3个 |

### 关注账号策略
```
每天关注：5-10个账号
账号类型分配：
- 同行业竞品：40%（了解对手）
- 目标用户喜欢的账号：40%（了解用户）
- 行业权威/媒体账号：20%（提升账号权重）
```

### 内容互动质量要求
```
评论质量要求：
❌ 不要：纯表情、"好的"、复制粘贴
✅ 要做：结合视频内容的真实感受，15-30字
✅ 示例："这个方法我试过，效果确实不错，特别是第2步很关键"

完播率要求：
- 目标：完整观看80%以上的视频
- 每天深度浏览10+条视频
```

### 直播互动（每天1-2场）
```
选择：同行业直播间
停留：10-15分钟
互动：发1-2条评论，内容要与直播相关
禁止：刷屏、广告词
```

## 冷启动期（0-1000粉）建议

### 第1-7天：建立账号标签
- 每天搜索3-5个核心关键词
- 完整观看同类型内容，不中途退出
- 完善账号所有基础信息
- 不急于发布内容（先养7天）

### 第8-30天：内容测试期
- 发布第一批3-5条内容（不同风格）
- 观察哪类内容完播率高
- 确定内容方向后稳定更新

### 第31-90天：稳定成长期
- 每日固定发布（1-2条）
- 固定发布时间（与用户活跃高峰对齐）
- 积极回复每一条评论
- 开始测试互动钩子

## 账号权重提升技巧

### 信号优先级（从高到低）
1. 完播率（最重要）
2. 互动率（点赞+评论+收藏/播放量）
3. 分享率（转发到好友/私信）
4. 关注转化率（看了视频后关注账号）
5. 搜索进入率（通过搜索找到账号）

### 提升完播率方法
- 前3秒必须有强钩子
- 视频节奏快（避免超过1分钟）
- 字幕完整（提升无声观看体验）
- 结尾预告下一条内容

### 提升互动率方法
- 视频末尾明确引导点赞/评论
- 在评论区发布引导性内容
- 发布后30分钟内自我互动（回复假设性问题）
