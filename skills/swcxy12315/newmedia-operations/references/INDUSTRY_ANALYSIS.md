# 行业分析完整指南

## 第一步：读取客户资料

接收客户提供的 PPT/资料后，识别以下核心信息：

### 客户诉求确认
| 诉求类型 | 判断标准 | 内容策略方向 |
|----------|----------|--------------|
| 涨粉 | 要求粉丝增长数 | 泛内容、干货分享、跟热点 |
| 引流 | 要求私域转化 | 钩子内容、价值感建立、粉丝运营 |
| 转化 | 要求销售线索/成交 | 产品种草、痛点解决、信任背书 |
| 品牌曝光 | 要求曝光量/认知度 | 品牌故事、权威内容、跨平台传播 |

### 现有账号现状盘点
- 抖音：粉丝数、近30天发布数、近30天总播放量、最高播放单条
- 视频号：粉丝数、历史内容方向
- 小红书：粉丝数、笔记数、近30天互动量
- 历史内容方向（是否与新策略一致）

## 第三步：用户画像分析

### 人群画像维度
| 维度 | 分析内容 |
|------|----------|
| 基础属性 | 年龄/性别/地域/收入 |
| 消费习惯 | 购买频率/决策周期/价格敏感度 |
| 内容偏好 | 喜欢看什么类型内容/活跃时段 |
| 痛点需求 | 核心痛点/潜在需求/情绪触发点 |
| 关注账号 | 关注哪些类型的内容创作者 |

### 获取用户画像方法
1. 分析竞品账号的评论用户（职业、地域、语言风格）
2. 利用 china-demand-mining 的需求挖掘结果
3. 参考艾媒咨询行业用户画像报告
4. 分析品牌现有粉丝账号主页

## 第四步：关键词体系构建

### 5118 关键词挖掘
```
WebFetch: https://www.5118.com/
WebSearch: "site:5118.com [行业] 关键词"

挖掘维度：
- 商机热词：搜索量大、竞争度适中
- 需求词：含"怎么"/"如何"/"哪款"/"推荐"
- 痛点词：含"差评"/"问题"/"吐槽"/"坑"
- 长尾词：具体场景词、人群词
```

### 关键词四层体系
```
第一层（主关键词）：[品牌名] / [品类名]
  示例：护肤品 / 祛痘面霜

第二层（核心关键词）：[品类] + [功效/场景]
  示例：祛痘面霜推荐 / 敏感肌护肤

第三层（产品关键词）：[成分/功能/卖点]
  示例：水杨酸祛痘 / 无激素祛痘

第四层（用户痛点关键词）：[问题/症状/需求]
  示例：反复长痘怎么办 / 青春痘闭口怎么解决
```
