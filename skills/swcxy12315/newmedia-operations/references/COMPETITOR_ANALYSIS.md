# 竞品分析完整指南

## 分析对象选择标准

### 筛选竞品账号（优先级排序）
1. **直接竞品**：同品类、同价位的品牌账号
2. **间接竞品**：不同品类但目标用户高度重合的账号
3. **标杆账号**：同类型但运营更成熟的高参考账号

### 筛选维度
- 近30天爆款多（单条播放>10万）
- 评论活跃（评论数/播放数>0.5%）
- 粉丝规模：1万-100万（避免头部失真）

## 平台竞品搜索方法

### 抖音竞品搜索
```bash
# 搜索竞品账号
python scripts/fetch_douyin.py \
  --keyword "[竞品品牌名/行业词]" \
  --type account \
  --limit 50

# 抓取竞品爆款视频评论
python scripts/fetch_douyin.py \
  --account_id [账号ID] \
  --type comments \
  --sort hot \
  --limit 500
```

### 小红书竞品搜索
```bash
# 搜索竞品笔记
python scripts/fetch_xiaohongshu.py \
  --keyword "[竞品词]" \
  --sort hot \
  --limit 200

# 抓取竞品笔记评论
python scripts/fetch_xiaohongshu.py \
  --note_id [笔记ID] \
  --type comments \
  --limit 300
```

### 视频号竞品搜索
```bash
python scripts/fetch_wechat_video.py \
  --keyword "[行业关键词]" \
  --limit 100
```

### 电商竞品分析
```bash
# 淘宝/天猫差评抓取
python scripts/fetch_ecommerce_reviews.py \
  --platform taobao \
  --keyword "[竞品名]" \
```

## 账号画像分析维度

### 基础信息层
| 维度 | 分析要点 | 记录内容 |
|------|----------|----------|
| 昵称 | 关键词是否清晰 | 品牌名+品类+卖点 |
| 头像 | 辨识度/专业度 | 真人/Logo/IP形象 |
| 简介 | 价值主张清晰度 | 核心卖点/钩子词 |
| 置顶视频 | 引导逻辑 | 是否有明确引流动作 |
| 背景图 | 品牌感/信息量 | 是否包含联系方式 |

### 内容策略层
```
分析近30天发布内容：
1. 内容品类分布（干货/种草/娱乐/产品展示占比）
2. 爆款内容特征（标题公式/开头钩子/结尾设计）
3. 发布时间规律（早/中/晚，工作日/周末）
4. 视频时长分布（<30s / 30s-1min / >1min）
5. 封面风格（真人出镜/产品展示/文字封面）
```

### 评论分析层
使用 `clean_data.py` 处理后进行分析：
```bash
python scripts/clean_data.py \
  --input comments_raw.json \
  --output comments_clean.json

python scripts/extract_demands.py \
  --input comments_clean.json \
  --output demands.json
```

**重点提取**：
- 用户关注重点（出现频率最高的词）
- 高频问题（"怎么"/"如何"/"多少钱"类）
- 差评内容（负面反馈）
- 品牌方回复高频话术

## 爆款内容拆解框架

### 标题拆解
```
标题公式识别：
□ 痛点型：[痛点描述] + [解决方案]
□ 数字型：[N个方法] / [N天结果]
□ 对比型：[普通做法] vs [正确做法]
□ 场景型：[人群] + [场景] + [结果]
□ 疑问型：[问题] + [惊喜答案]
```

### 内容结构拆解
```
前3秒：钩子设计（什么让用户不划走）
黄金30秒：核心价值传递
结尾：行动引导（点赞/评论/关注/私信）
```

## 实时监测配置

使用 `scripts/monitor_competitors.py` 进行每日监测：

```bash
# 先配置 templates/accounts_config.json 中的竞品账号列表
python scripts/monitor_competitors.py \
  --accounts templates/accounts_config.json \
  --check_interval 24h \
  --alert_threshold 100000  # 播放量超过10万的新内容提醒

# 监测内容包括：
# - 新发布内容
# - 播放量突增（可能是爆款）
# - 评论区新增需求/痛点
# - 账号数据变化（粉丝增减）
```
