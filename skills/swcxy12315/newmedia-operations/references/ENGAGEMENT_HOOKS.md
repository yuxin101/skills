# 互动钩子设计与话术库

## 视频内互动钩子模板

### 提问类（触发评论）
```
通用模板：
「你是不是也[行业痛点]？」
「遇到[问题场景]的举个手」
「[A做法] 还是 [B做法]？评论区告诉我」

示例（护肤行业）：
「你是不是也搞不清楚精华和乳液该先用哪个？」
「用了三瓶祛痘产品还没效果的举个手」
「你选护肤品看成分还是看测评？评论区告诉我」
```

### 引导收藏类（触发收藏）
```
「点赞收藏，不然刷着刷着就没了」
「这条内容很重要，先收藏再看，以后肯定用得上」
「干货有点多，建议先收藏慢慢看」
```

### 资料引导类（触发私信）
```
「想要[完整方案/资料/话术]的，找我发你」
「完整版[XX清单]太长放不下，需要的在评论区说[关键词]」
「更详细的[行业干货]整理好了，看我主页简介」
```

### 预告引导类（触发关注）
```
「这只是第一步，关注我，下期讲[更重要的内容]」
「想知道[具体技巧]还是[避坑攻略]？关注后评论区告诉我，优先更你们想看的」
```

## 评论区互动钩子

### 选择题型（降低互动门槛）
```
「给大家个选择：[选项A] 还是 [选项B]？
点赞选A，评论1选B，看看大家的想法」

场景：引发用户碰撞，植入品牌/行业痛点
```

### 资源分发型（收集需求）
```
「有用就收藏一下～需要完整版[方案/话术/清单]的，我发你参考。」
「大家的疑问我都看到啦，统一整理后会发给有需要的朋友。」
```

### 需求收集型（引导内容方向）
```
「想知道[XX技巧] 还是 [XX避坑]？评论区告诉我，优先更你们想看的！」
「下期想看[话题A]还是[话题B]？投票决定」
```

### 情感共鸣型（收集痛点）
```
「说说你用过最坑的[行业]产品/服务，一起吐槽」
「[行业]里有什么让你特别崩溃的事？来聊聊」
「你[入行/开始用XX]多久？遇到过[痛点]吗？」
```

### 挑战测试型（知识互动）
```
「测试题：[行业知识判断]，对的评论"对"，错的评论"错"，答案明天公布」
「[情景题]，你会怎么选？说说你的想法」
```

### 催更型（提升活跃度）
```
「想看[具体内容]的扣1，够[N]个就出」
「上条视频有[N]个人说想看[续集]，今天来了！」
```

## 账号私域引导话术

### 评论区引导
```
「想要完整[资料/报价/方案]的朋友，可以看我主页简介，我发你完整版。」
「更多干货和细节我都整理好了，需要的朋友可以到主页找我领取哦。」
「想了解具体[价格/效果/使用方法]的，我单独发你详细说明，看我简介即可。」
「每天只回复少量咨询，有需要的朋友可以通过主页联系我。」
「为了避免错过，详细[教程/清单]我统一发，看我简介就能拿到。」
```

### 私信自动回复（欢迎语）
```
模板1（资料引导型）：
「你好～感谢关注！我会发你【[行业]干货 + 产品资料 + 常见问题解答】，方便你快速了解。」

模板2（需求了解型）：
「你好呀～为了更好帮你，你可以简单说下你的需求，我给你针对性方案。」

模板3（菜单型）：
「我已为你准备好干货资料，直接回复对应数字即可领取：
1️⃣ [资料A]
2️⃣ [资料B]  
3️⃣ [资料C]
4️⃣ 了解产品/合作」

模板4（信任建立型）：
「感谢信任！后续新品、活动、干货我都会第一时间发你，不打扰、不刷屏。」
```

## 用户唤醒策略

### 触发条件
- 7天未观看任何视频
- 7天未产生任何互动（点赞/评论/收藏）
- 14天未私信互动

### 唤醒内容生成逻辑
```
结合用户历史行为生成个性化唤醒内容：

行为数据输入：
- 用户上次互动的内容类型
- 用户曾问过的问题
- 用户曾收藏的内容类型
- 当前热点事件

AI 生成唤醒消息：
「[个性化开场] + [与TA相关的新内容预告] + [低门槛行动引导]」

示例：
「好久没见你了！上次你问的[XX问题]，我最新整理了[相关内容]，发你参考看看」
「这周出了一个[热点]，刚好和你之前关注的[话题]相关，来看看？」
```

## 人工客服转接标准

### 触发转人工关键词
```
强意向词（立即转人工）：
- 「怎么买」「在哪买」「多少钱」「如何合作」
- 「我想要」「帮我预订」「有现货吗」
- 「我想了解合作」「加盟怎么操作」

转人工话术：
「感谢您的咨询！您的问题我已收到，我们专属顾问会在[N分钟]内联系您，请稍候～」
```

### 明确拒绝 AI 处理
```
触发词：
- 「转人工」「找真人」「要人工客服」

话术：
「好的！稍等一下，为您转接专属顾问～」
+ 立即通知人工跟进
```

## 互动数据监测指标

| 指标 | 健康阈值 | 优化方向 |

| 评论率 | >0.5%（评论数/播放量） | 加强提问钩子 |
| 收藏率 | >1%（收藏数/播放量） | 加强干货价值 |

| 私信转化率 | >0.1%（私信数/播放量） | 优化资料钩子 |
| 粉丝转化率 | >1%（新增粉丝/播放量） | 优化关注引导 |
