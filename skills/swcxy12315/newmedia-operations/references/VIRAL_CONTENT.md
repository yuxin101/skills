## 爆款标题公式

### 通用公式
```
痛点型：你还在用[错误方法]？难怪[负面结果]！
数字型：[N]个[行业]人都不知道的[秘密/技巧]
对比型：同样[品类]，[普通选择]VS[好选择]，差距一目了然
场景型：[人群特征]必看！[场景]下[问题]的正确解法
反转型：以为[常识]，其实[真相]，很多人搞错了
结果型：坚持[做法]N天，[惊喜结果]（附完整方案）
```

### 小红书专用公式
```
种草型：[价位]的[品类]，我找到了！[效果]真的太好了
避坑型：[品类]避坑指南！这[N]款踩雷了，别再买了
测评型：买了[N]款[品类]测评，这款性价比最高（附对比）
教程型：[N]步搞定[效果]，手残党也能学会
打卡型：坚持[做法][N]天，变化肉眼可见
```

## 二次创作工作流

### 标准流程
```
1. 找到爆款原文（易撰/微信搜一搜/平台爬取）
2. 分析爆款要素（标题公式/核心痛点/内容结构）
3. 替换策略：
   ├── 保留逻辑框架
   ├── 替换具体案例（改为客户品牌案例）
   ├── 保留核心痛点
   ├── 替换技术/方案（结合客户产品）
   └── 加入品牌专属：卖点、品牌符号、记忆点
4. 违禁词检测（重要！）
5. 发布
```

### 内容类型与创作要求

#### 品牌宣传文章
```
结构：
- 品牌故事/理念（建立情感连接）
- 核心产品/服务介绍（突出差异化）
- 用户信任背书（案例/数据/认证）
- 行动号召（引导关注/咨询）
发布：大号（品牌主账号）
```

#### 产品种草文章  
```
结构：
- 场景引入（目标用户痛点）
- 产品出场（解决方案）
- 使用效果（具体数据/对比图）
- 使用方法（降低购买门槛）
- 价格锚点 + 购买引导
发布：小号1（种草账号）
```

#### 科普知识内容
```
结构：
- 普遍认知错误（引发共鸣）
- 正确知识点（建立权威感）
- 延伸价值（让用户有收藏欲）
- 轻度品牌引导（不强推）
发布：小号2（引导到大号和小号1）
```

#### 招商文章
```
结构：
- 市场机会（数据支撑）
- 品牌优势（差异化竞争力）
- 合作模式（清晰可操作）
- 成功案例（已有合作商）
- 门槛说明 + 联系方式
发布：大号
```

## 节日热点融合日历

### 每日检测路径
```
1. 检查抖音官方活动：
   WebFetch: https://www.douyin.com/activity
   
2. 检查小红书官方活动：
   WebSearch: "小红书 官方活动 [当月]"
   
3. 检查视频号活动：
   WebSearch: "微信视频号 官方活动 [当月]"
   
4. 节气/节日检查：
   每天12:00 播报相关热点节点
```

### 内容融合方法
```
节日/热点 + 品牌行业 → 融合内容

示例：
- 春节 + 护肤品 → "过年走亲戚，皮肤状态决定第一印象"
- 开学季 + 教育 → "新学期，这些学习方法让孩子弯道超车"
- 情人节 + 餐饮 → "情人节不知道送什么？这份攻略帮你解决"
```

## 违禁词处理规范

### 检测工具
```bash
# 本地检测
python scripts/detect_forbidden_words.py \
  --text "[内容文本]" \
  --platform [douyin/xiaohongshu/wechat]

# 在线检测
WebFetch: https://www.check51.com/
将内容粘贴检测，红色标出词进行替换
```

### 违禁词类型及替换策略

| 类型 | 示例 | 替换方法 |
|------|------|----------|
| 最高级 | 最好/第一/顶级/绝对 | 非常/极致/深受好评 |
| 医疗功效 | 治疗/治愈/消炎/杀菌 | 改善/调节/舒缓/呵护 |
| 平台引流词 | 微信/二维码/私聊 | 主页/简介/私信 |
| 夸大效果 | 100%有效/立竿见影 | 多数用户反馈/持续使用后 |
| 绝对化 | 保证/承诺/必须 | 通常/一般/建议 |

## 图文与视频制作要点

### 小红书图文
```
素材来源：当天日期标注的图片素材文件夹
AI 扩图工具：可爱/创可贴/美图秀秀/豆包
图片风格：匹配最近节日节气和热点
图片数量：1张封面 + 3-9张内容图
文字风格：活泼/治愈/干货感（根据内容类型）
```

### 短视频制作
```
素材来源：当天日期标注的视频素材文件夹
剪辑工具：剪映
必要元素：
  - 字幕（提升无声观看体验）
  - BGM（匹配视频情绪）
  - 封面帧（最吸引人的画面）
视频时长：30秒-1分钟
分镜数量：建议15-20个分镜（每3-4秒一个切换）
```
