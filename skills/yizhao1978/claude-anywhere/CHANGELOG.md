# Changelog

All notable changes to Claude Anywhere are documented here.

---

## [1.6.1] — 2026-03-28

### Changed
- 付款后 Pro 自动开通，无需填写 License Key

---

## [1.6.0] — 2026-03-27

### Added
- **微信扫码支付**：支持微信支付，付款后 Pro 自动激活
- **升级提醒**：免费用户到达每日上限或试用到期时，Bot 自动推送付款链接

---

## [1.5.1] — 2026-03-18

### Fixed
- QQ Bot 定时任务结果推送异常修复

---

## [1.5.0] — 2026-03-15

### Added
- **QQ Bot 图片/文件分析**：支持发送图片或文件附件给 Claude 分析
- **QQ Bot WebSocket 模式**：无需公网 IP
- **三平台一键启动**：`npm start` 同时启动全部已配置的平台

---

## [1.4.0] — 2026-02-28

### Added
- **定时任务** (`/cron`)：用自然语言设置定时 Claude 任务，结果自动推送到对话

---

## [1.3.0] — 2026-02-10

### Added
- **会话管理** (`/sessions`, `/resume`)：跨设备、跨平台恢复历史对话

---

## [1.2.0] — 2026-01-20

### Added
- **文件分析**（Pro）：发送 PDF、Excel、CSV、代码文件给 Claude 分析
- **图片分析**（Pro）：发送截图或照片进行视觉分析

---

## [1.1.0] — 2026-01-05

### Added
- **企业微信支持**：完整的企业微信 AI Bot 接入
- Pro / 免费分级，每日配额管理

---

## [1.0.0] — 2025-12-15

### Added
- 初始版本：Telegram 接入 Claude Code CLI
- 读写文件、执行命令、图片分析
- 免费版：5条/天，7天试用
