# Pricing Playbooks by API Niche

## 1) Local vertical API (e.g., Vietnam address)
- Free: 1,000/month
- Starter: $19 for 50k
- Growth: $79 for 300k
- Positioning: local-data accuracy > generic global APIs

## 2) Global utility API (e.g., email/domain intelligence)
- Free: 1,500/month
- Starter: $29 for 100k
- Growth: $99 for 500k
- Positioning: fast integration + low false positive risk

## 3) Agency/ops API (high-touch support)
- Free: small trial only
- Starter: lower quota but includes support
- Growth: higher quota + onboarding SLA

## 4) When to raise prices
Raise after any two conditions:
- 3+ paying users with stable usage
- >65% of starter users exceed 70% monthly quota
- support load consistently >2h/week
