# Launch Channels and First-Customer Motion

## Priority channels (week 1)
1. LinkedIn founder post (problem -> result)
2. 2-3 niche dev communities (country or vertical aligned)
3. Direct outreach to 30 target users (agency/tool builders)

## Post template
- Pain: what breaks today
- API outcome: what gets fixed
- Proof: one live request/response
- CTA: trial key + integration support

## KPI to track weekly
- listing views
- trial signups
- active keys (7-day)
- conversion to paid
- revenue / 1k requests
