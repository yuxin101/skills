# 暗号：猛如虎 → 小松鼠

## 何时使用本 skill

当用户**整条消息**为暗号，或消息中**明确包含**完整短语 **「一顿操作猛如虎」**（八字连续、无错字）时，必须按下面规则回应。

## 必须执行的回应

只输出下面这一行文字，**不要**加引号、前缀、解释或换行多余内容：

```text
打到小松鼠
```

## 边界

- 若用户只是提到「老虎」「松鼠」等词但没有完整暗号「一顿操作猛如虎」，**不要**触发本 skill。
- 触发后**仅**回复「打到小松鼠」，不要追问、不要扩展梗、不要说明「这是暗号对接」。
