# 全局错误码

## 认证/授权类

| ErrCode | 说明 |
|---------|------|
| 1000001 | 用户不存在 |
| 1000002 | Token 过期 |
| 1000003 | 访问次数已耗尽 |
| 1000004 | 并发数不足 |
| 1000005 | 授权已过期 |
| 1000006 | 验签失败 |
| 1000007 | 接口限流 |

## 参数/上传类

| ErrCode | 说明 |
|---------|------|
| 2000001 | 参数错误 |
| 2000002 | 上传图片过大，请上传不超过 20M 的图片 |
| 2000003 | 上传图片类型不支持 |
| 2000004 | 上传图片发生异常 |
| 2000005 | 上传的图片过于敏感 |

## 服务类

| ErrCode | 说明 |
|---------|------|
| 3000001 | 服务开小差 |
