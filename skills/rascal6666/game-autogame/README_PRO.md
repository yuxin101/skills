# 向僵尸开炮 - 智能Pro版

## 版本区别

| 功能 | 基础版 | Pro版 |
|------|--------|-------|
| OCR识别技能 | ✅ | ✅ |
| 关键词选技能 | ✅ | ✅ |
| 怪物数据库 | ❌ | ✅ |
| 智能推荐技能 | ❌ | ✅ |
| 怪物识别 | ❌ | 根据名称匹配 |

## Pro版功能

### 1. 怪物数据库
已收录 **17种怪物** 的完整数据：
- 名称、类型、HP、攻击力、防御力
- 属性抗性/弱点
- **推荐技能**

### 2. 智能技能选择
```
选择流程：
1. OCR识别当前屏幕文字
2. 匹配怪物数据库
3. 如果匹配成功 → 选择推荐的技能
4. 如果匹配失败 → 使用默认策略（连/齐 > 子弹 > 中间）
```

### 3. 怪物推荐技能示例

| 怪物类型 | 推荐技能 |
|----------|----------|
| BOSS | 高能射线 > 冰暴发生器 |
| 精英怪 | 冰暴发生器 > 高能射线 |
| 普通高血 | 温压弹 > 无人机 |
| 大型怪 | 干冰弹 |
| 远程怪 | 雷击 > 闪电链 |
| 机械怪 | 冰暴发生器 > 电磁穿刺 |

## 运行方法

```bash
# 运行基础版（原始）
python start.py

# 运行Pro版（智能推荐）
python start_pro.py
```

## 怪物数据库

文件：`monsters.json`

结构：
```json
{
  "monsters": [
    {
      "name": "火焰领主",
      "type": "BOSS",
      "hp": "58.5M",
      "attack": "26.5K",
      "defense": "25%",
      "recommended": "高能射线/冰暴发生器"
    }
  ]
}
```
