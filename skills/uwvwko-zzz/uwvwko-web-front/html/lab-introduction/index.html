研究方向

探索机器人技术的边界，推动智能系统的革新

🤖

机器人感知与控制

研究机器人的环境感知、运动规划与精确控制，实现复杂环境下的自主操作能力。

👁️

计算机视觉

开发先进的视觉算法，包括目标检测、语义分割、3D重建等，赋予机器人类人的视觉能力。

🗺️

SLAM与导航

研究同步定位与建图技术，实现机器人在未知环境中的自主导航与探索。

🧠

深度学习

将深度学习技术应用于机器人系统，提升感知、决策与学习的智能化水平。

🤝

人机协作

研究人与机器人的自然交互与协作机制，构建安全高效的人机共融系统。

🚗

自动驾驶

开发自动驾驶核心技术，包括环境感知、路径规划、决策控制等关键模块。
