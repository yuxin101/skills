# 优秀案例收集

## 设计亮点分析

### Stripe Dashboard
- 链接：https://stripe.com
- 设计亮点：
  - 极简的配色，大面积留白
  - 精致的阴影和圆角
  - 流畅的微交互
  - 清晰的层次结构
- 可复用思路：
  - 卡片组件设计
  - 数据可视化展示
  - 侧边栏导航

### Apple
- 链接：https://apple.com
- 设计亮点：
  - 全屏 Hero 视频/图片
  - 超大标题文字
  - 优雅的滚动动画
  - 极致的细节打磨
- 可复用思路：
  - Hero 区域设计
  - 滚动视差效果
  - 产品展示动效

### Vercel
- 链接：https://vercel.com
- 设计亮点：
  - 深色科技风
  - 渐变色运用
  - 网格背景
  - 代码风格展示
- 可复用思路：
  - 深色模式配色
  - Landing page 布局
  - 特性展示卡片

### Tailwind UI
- 链接：https://tailwindui.com
- 设计亮点：
  - 组件库设计规范
  - 大量可复用的组件模板
  - 一致的间距系统
  - 专业的配色方案
- 可复用思路：
  - 按钮、卡片、表单组件
  - 栅格布局
  - 响应式设计模式

### Linear
- 链接：https://linear.app
- 设计亮点：
  - 极简深色界面
  - 精致的图标
  - 流畅的动画
  - 键盘快捷键设计
- 可复用思路：
  - 深色主题设计
  - 列表/表格设计
  - 空状态设计

## 收集灵感的方法

1. **日常浏览**：遇到好看的网站截图保存
2. **平台发现**：Dribbble、Awwwards 每日精选
3. **竞品分析**：同类型产品的设计对比
4. **设计趋势**：关注当下的设计流行趋势

## 设计 Checklist

- [ ] 首屏是否吸引眼球
- [ ] 导航是否清晰易懂
- [ ] 关键信息是否突出
- [ ] CTA 是否明显
- [ ] 配色是否和谐
- [ ] 排版是否有层次
- [ ] 动画是否流畅
- [ ] 是否适配移动端
