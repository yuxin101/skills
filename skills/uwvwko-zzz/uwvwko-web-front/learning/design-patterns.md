# 设计模式与布局技巧

## 常见布局模式

### Z-Pattern
适合线性阅读的内容，如落地页顶部到中部。
视线从左上→右上→左下→右下的 Z 字形流动。
- 首屏大标题/CTA 放在左上
- 副标题/产品图放右上
- 底部 CTA 放左下或右下

### F-Pattern
适合文字内容为主的页面，如博客文章。
用户习惯性从左到右阅读，形成 F 形状的热区。
- 标题放最上方
- 重要内容靠左
- 侧边栏放右侧

### Golden Pattern
视觉焦点在页面的黄金分割点（约 61.8% 处）。
适合展示单一核心内容。

## 响应式 Breakpoints

| 设备 | 宽度 | CSS |
|------|------|-----|
| 手机 | < 640px | `@media (max-width: 639px)` |
| 平板 | 640-1024px | `@media (min-width: 640px) and (max-width: 1024px)` |
| 桌面 | > 1024px | `@media (min-width: 1025px)` |
| 大屏 | > 1440px | `@media (min-width: 1440px)` |

## 卡片设计

- 卡片圆角：8-16px
- 卡片阴影：`0 2px 8px rgba(0,0,0,0.1)`
- 卡片间距：16-24px
- 卡片内边距：16-24px

## 间距系统

使用 4px 或 8px 的倍数：
- xs: 4px
- sm: 8px
- md: 16px
- lg: 24px
- xl: 32px
- 2xl: 48px
- 3xl: 64px

## 栅格系统

常用 12 栅格系统：
- 列宽 = (100% - (间隙 * 11)) / 12
- 常用：1/3 (4列), 1/2 (6列), 2/3 (8列)
