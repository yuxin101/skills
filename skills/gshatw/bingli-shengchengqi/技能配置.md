# 病历生成器技能配置

## 技能信息
- 名称：病历生成器
- 版本：1.0.0
- 触发词：写病历、生成病历、病历助手、入院记录、medical record、病历生成
- 类别：医疗健康

## 工作流程说明

### 触发阶段
当用户说出触发词时，技能自动启动。

### 交互流程
1. **第一步：变量收集**
   - 主动询问：主诉及相关情况
   - 主动询问：性别
   - 主动询问：年龄
   - 可选询问：其他信息（婚姻状况、职业等）

2. **第二步：信息确认**
   - 展示已收集的信息
   - 询问是否需要补充或修改
   - 等待用户确认

3. **第三步：病历生成**
   - 重新整理所有变量
   - 询问是否开始生成
   - 用户确认后生成规范的入院记录

### 输出格式
严格按照以下模板生成：
```
**入院记录**
科室：全科医疗科
病房：01
病案号：2026010001
姓名：张无及
性别：[患者性别]
年龄：[患者年龄]
...
```

### 默认值规则
- 体温未提及：36.5℃
- 脉搏未提及：72次/分
- 呼吸未提及：18次/分
- 血压未提及：120/80mmHg
- 其他未提及信息：填写"正常"或临床默认值
