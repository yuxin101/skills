# Accepted Attribute Keys

Complete list of accepted `--attr` keys for `face:create` and `face:update`. Keys not on this list are silently ignored.

## Age & Birth

| Key | Example value |
|---|---|
| `age` | `34` |
| `age_range` | `30-39` |
| `birth_date` | `1991-06-15` |
| `birth_year` | `1991` |
| `birth_month` | `June` |
| `birth_decade` | `1990s` |
| `date_of_birth` | `1991-06-15` |
