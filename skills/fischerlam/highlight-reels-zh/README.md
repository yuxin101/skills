> **自动提炼最精彩的片段，做成节奏更强、信息更集中的高光集锦。**

## 适合什么场景
适合比赛、活动、演讲、赛事 recap 和任何只想保留“最佳时刻”的视频。

## 一条命令直接开始

```bash
sparki run --file ./video.mp4 --mode prompt-driven --prompt "提炼最精彩的部分，做成高光集锦" --aspect-ratio 9:16
```

## 推荐提示词
- 提炼最精彩的部分，做成高光集锦。
- 把比赛里最强的片段剪出来。
- 把活动视频做成一个更有冲击力的 recap。

## 相关技能
- `长视频转短视频`
- `TikTok 爆款剪辑`
- `视频尺寸调整`
