# ⏰ 定时任务配置指南

## 🚀 快速开始（3 步搞定）

### 第 1 步：配置环境变量

**方式 A：使用配置向导（推荐）**

1. 双击 `配置环境变量.bat`
2. 按提示输入企业微信 Token 和接收人账号
3. 配置完成后可以选择立即测试

**方式 B：手动设置系统环境变量**

1. `Win + R` → 输入 `sysdm.cpl` → 回车
2. "高级" → "环境变量"
3. "系统变量" → "新建"：
   - 变量名：`WEWORK_X_TOKEN`，变量值：`你的完整 token`
   - 变量名：`WEWORK_TO_USER`，变量值：`你的企业微信账号`

---

### 第 2 步：手动测试运行

双击 `一键运行.bat`，确认：
- ✅ 依赖包安装成功
- ✅ HTML 报告生成到桌面
- ✅ 企业微信推送成功

---

### 第 3 步：配置定时任务

**方式 A：PowerShell 脚本（推荐）**

1. 右键点击 `SetupTask.ps1`
2. 选择"以管理员身份运行"
3. 脚本会自动创建每天 9:00 运行的定时任务

**方式 B：手动创建**

1. `Win + R` → 输入 `taskschd.msc` → 回车
2. 右侧"创建基本任务"
3. 名称：`每日金融课件推送`
4. 触发器：每天 `09:00`
5. 操作：启动程序
   - 程序：`python.exe`（自动检测）
   - 参数：`C:\Users\wwwir\.openclaw\workspace\skills\wework-financial-daily\scripts\generate_and_send.py`
6. 勾选"当单击完成时打开属性对话框"
7. 在属性中勾选：
   - ✅ "不管用户是否登录都要运行"
   - ✅ "使用最高权限运行"

## 🧪 测试定时任务

### 立即运行测试

```powershell
schtasks /Run /TN "每日金融课件推送"
```

### 查看任务状态

```powershell
schtasks /Query /TN "每日金融课件推送"
```

### 查看下次运行时间

```powershell
schtasks /Query /TN "每日金融课件推送" /FO LIST | findstr "Next Run"
```

---

## 🔧 管理任务

### 删除任务

```powershell
schtasks /Delete /TN "每日金融课件推送" /F
```

### 修改运行时间

1. 任务计划程序 → 找到任务
2. 右键 → "属性"
3. "触发器"选项卡 → 编辑
4. 修改时间为所需时间

### 查看运行历史

1. 任务计划程序 → 任务计划程序库
2. 找到"每日金融课件推送"
3. 点击"历史记录"选项卡

## ❓ 常见问题

### Q: 任务显示"就绪"但不执行
**A:** 检查是否勾选了"不管用户是否登录都要运行"，并确保以管理员身份创建任务。

### Q: 推送失败，提示 Token 无效
**A:** 
1. 检查 `WEWORK_X_TOKEN` 是否正确
2. 确认 Token 未过期
3. 手动运行 `一键运行.bat` 测试

### Q: 图表中文乱码
**A:** 确保系统已安装中文字体（SimHei/Microsoft YaHei），Windows 通常默认已安装。

### Q: 如何更改推送时间？
**A:** 在任务计划程序中修改任务触发器时间，或重新运行 `SetupTask.ps1` 修改脚本中的 `$taskTime`。

### Q: 环境变量不生效？
**A:** 
1. 系统环境变量需要重新打开命令行窗口才能生效
2. 或者直接在 `scripts/generate_and_send.py` 中硬编码配置
