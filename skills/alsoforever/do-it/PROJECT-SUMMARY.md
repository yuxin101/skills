## 🎯 核心功能

### 用户流程
```
打开网页 → 填写问题 → 提交分析 → 等待加载
    ↓
查看结果 → 提交反馈 → 保存历史 → 后续跟进
```

### 技术架构
```
┌─────────────┐     ┌──────────────┐     ┌─────────────┐
│  Web 前端    │────▶│  API 服务     │────▶│  数据文件   │
│ (HTML/JS)   │     │  (Python)    │     │  (JSON)     │
└─────────────┘     └──────────────┘     └─────────────┘
      ↓                     ↓                     ↓
  用户交互              分析判断              数据支撑
```

## 📈 数据流程

```
用户提问
    ↓
API 接收问题
    ↓
加载薪资/城市/案例数据
    ↓
分析问题关键词
    ↓
匹配数据和案例
    ↓
生成判断建议
    ↓
返回 HTML 结果
    ↓
前端展示
    ↓
保存到历史记录
```
