## 🎯 技能描述

**do-it** 是一个帮助人类做人生选择的 AI 判断技能。

**核心理念：**
> "用户负责输入信息，AI 负责判断，用户负责 do it"

**适用场景：**
- 职业选择（去哪个城市、换不换工作）
- 感情问题（要不要继续、要不要放下）
- 投资决策（要不要投、投多少）
- 生活选择（买房还是租房、结婚还是单身）
- 任何难以抉择的事情

## 🔧 技术实现

### API 接口

```javascript
POST /api/skills/do-it/analyze

Request:
{
  "problem": "我该留在长沙还是去广州工作？",
  "situation": {
    "current": "在长沙工作，年薪 15 万",
    "options": ["留长沙", "去广州（年薪 40 万）"],
    "concerns": "家在长沙，但广州发展好",
    "preference": "有点想去广州"
  },
  "additional_info": "35 岁，单身，父母在长沙"
}

Response:
{
  "success": true,
  "data": {
    "analysis": "...",
    "recommendation": "去广州",
    "reasons": [...],
    "action_plan": {...},
    "risks": {...}
  }
}
```

### 调用方式

```javascript
// 前端调用示例
async function analyzeWithDoIt(problem, situation) {
  const response = await fetch('/api/skills/do-it/analyze', {
    method: 'POST',
    headers: { 'Content-Type': 'application/json' },
    body: JSON.stringify({ problem, situation })
  });
  
  const result = await response.json();
  return result.data;
}
```
