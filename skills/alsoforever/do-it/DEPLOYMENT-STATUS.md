## 🎯 产品功能

### 用户流程

```
打开网页 → 点击开始 → 填写问题 → 提交分析
    ↓
等待加载 → 查看结果 → 提交反馈 → 保存历史
```

### 核心页面

1. **首页** - 展示 Slogan + 最近案例
2. **输入页** - 问题详情表单
3. **加载页** - 分析进度动画
4. **结果页** - 滚滚的判断 + 反馈


