## 💡 数据来源优先级

1. **官方数据** (统计局/教育部) - 最权威
2. **招聘平台** (拉勾/Boss/猎聘) - 最实时
3. **研究报告** (艾瑞/易观) - 最深度
4. **用户反馈** (案例复盘) - 最真实

## 🚀 贡献指南

### 如何贡献数据？

1. **手动录入** - 使用 `--manual` 模式
2. **提供数据源** - 提交网址/报告
3. **分享案例** - 填写案例模板
4. **反馈错误** - 提交 issue

### 数据格式

```json
{
  "city": "城市",
  "role": "职位",
  "experience_years": "经验年限",
  "salary_min": 最低薪资,

  "salary_max": 最高薪资,
  "sample_size": 样本数量,
  "data_source": "数据来源",
  "updated_at": "更新日期"
}
```
