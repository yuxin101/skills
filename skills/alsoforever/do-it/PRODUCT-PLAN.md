## 🛠️ 技术栈

### 前端
```
- Web: Vue 3 + Tailwind CSS
- 小程序：uni-app (一套代码多端)
- 状态管理：Pinia
- HTTP 客户端：Axios
```

### 后端
```
- 框架：Node.js (Express/NestJS) 或 Python (FastAPI)
- 数据库：PostgreSQL + Redis
- ORM: Prisma 或 SQLAlchemy
- 认证：JWT
```

### AI 集成
```
- 调用：OpenClaw API
- 模型：qwen3.5-plus
- 提示词：do-it SKILL.md
- 缓存：Redis (相同问题复用)
```

### 部署
```
- 服务器：阿里云/腾讯云
- 容器：Docker
- CI/CD: GitHub Actions
- 监控：Sentry + Prometheus
```
