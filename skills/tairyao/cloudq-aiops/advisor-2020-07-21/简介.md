## 概述
欢迎使用腾讯云智能顾问服务。该服务基于腾讯云海量运维专家经验，帮助客户打造卓越架构，实现便捷、灵活的一站式云上治理。
### 服务主要使用场景
- **持续架构优化**：对用户云架构进行卓越指数评级，并为用户的系统优化给出可操作的治理改进建议。
- **定期巡检扫描**：通过对用户云上资源进行定期风险巡检，为用户提供云资源健康状况的全局图景，帮助用户了解当前云资源设置与使用现状。
- **业务连续性评估**：检查超过服务限制 80% 的服务使用情况。并及时周知客户，提工单做扩容。
- **云上治理**：通过对用户云上资源进行容量监测、混沌演练等方式，帮助客户发现及验证云资源问题并推进问题治理，提升系统可用性。

本章节介绍的智能顾问 API 接口均为 API 3.0接口。您可以调用 API 对智能顾问进行操作，例如风险巡检、创建混沌演练任务等。
- 智能顾问-云巡检等功能支持的所有接口信息，请参见 [API 概览](https://cloud.tencent.com/document/product/1264/63122)。
- 智能顾问-混沌演练支持的所有接口信息，请参见 [API 概览](https://cloud.tencent.com/document/api/1500/71793)。

智能顾问 API 接口的常见术语请参见下表：


| 术语 | 描述 | 
|---------|---------|
| 评估项 | 评估项是智能顾问提供的一系列风险检测策略，可帮助您发现用云过程中的各类隐患风险，并进行评估及进一步治理。智能顾问的风险评估项包含安全、可靠、性能、成本、服务限制五个类别。您可以通过控制台的 [评估设置](https://console.cloud.tencent.com/advisor/switch) 界面查看各评估项详情。 |

## API 快速入门
您可以使用 API Explorer 工具在线调用 API。
本文以查询评估项信息为例，通过 API Explorer 工具调用 API 接口的步骤如下：
1. 进入 [API Explorer](https://console.cloud.tencent.com/api/explorer) 工具页面。更多 API Explorer 工具使用信息，请参见 [使用 API Explorer](https://cloud.tencent.com/document/product/1278/46697)。
2. 调用 [DescribeStrategies](https://cloud.tencent.com/document/product/1264/63110) 接口，查询评估项信息。
3. 在返回结果中查看具体评估项详情。
