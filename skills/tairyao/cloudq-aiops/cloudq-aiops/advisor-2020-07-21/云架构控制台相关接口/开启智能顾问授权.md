## 2. 输入参数

以下请求参数列表仅列出了接口请求参数和部分公共参数，完整公共参数列表见 [公共请求参数](/document/api/1264/63116)。

| 参数名称 | 必选 | 类型 | 描述 |
|---------|---------|---------|---------|
| Action | 是 | String | [公共参数](/document/api/1264/63116)，本接口取值：CreateAdvisorAuthorization。 |
| Version | 是 | String | [公共参数](/document/api/1264/63116)，本接口取值：2020-07-21。 |
| Region | 否 | String | [公共参数](/document/api/1264/63116)，此参数为可选参数。 |

## 3. 输出参数

| 参数名称 | 类型 | 描述 |
|---------|---------|---------|
| Message | String | 返回信息<br/>示例值：Already authorized|
| RequestId | String | 唯一请求 ID，由服务端生成，每次请求都会返回（若请求因其他原因未能抵达服务端，则该次请求不会获得 RequestId）。定位问题时需要提供该次请求的 RequestId。|

## 4. 示例

### 示例1 已开通

#### 输入示例

```
POST / HTTP/1.1
Host: advisor.tencentcloudapi.com
Content-Type: application/json
X-TC-Action: CreateAdvisorAuthorization
<公共请求参数>

{}
```

#### 输出示例

```json
{
    "Response": {
        "Message": "Already authorized",
        "RequestId": "97aa2227-1814-4d39-9470-dadca094be1f"
    }
}
```
