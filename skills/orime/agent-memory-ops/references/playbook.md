# Agent Memory Ops Playbook

## Durable memory belongs in `MEMORY.md`

Promote things like:
- lasting preferences
- recurring workflows
- durable project context
- identity / naming / timezone facts
- lessons worth keeping

## Daily notes belong in `memory/YYYY-MM-DD.md`

Keep things like:
- raw logs
- temporary context
- transient experiments
- intermediate failures
- one-off links and loose scraps

## What not to promote blindly

- API keys
- passwords
- session tokens
- raw bearer tokens
- personal secrets the user did not ask you to retain
- stale deadlines that already passed

## Healthy memory routine

1. audit duplicates weekly
2. digest last 3-7 daily notes
3. promote only stable facts
4. prune repeated bullets from `MEMORY.md`
5. keep action items visible instead of buried
