# Z-Library Skill

Search and download books from Z-Library using Claude Code.

**Install:** https://clawhub.ai/imnoahcook/zlibrary

```bash
clawhub install zlibrary
```

## Setup

Set your Z-Library credentials:

```bash
export ZLIBRARY_EMAIL="your@email.com"
export ZLIBRARY_PASSWORD="yourpassword"
```

## What it does

- Search books by title, author, language, format, or year
- View available download formats (EPUB, PDF, MOBI, etc.)
- Download books directly to `~/Downloads/`
- Session caching so you only log in once
