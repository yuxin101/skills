# 🎯 GitHub 人才猎手 (GitHub Talent Hunter)

**从代码里找人，比简历更真实。**

传统招聘平台上，候选人可以美化简历、夸大经验。但代码不会说谎。

GitHub 人才猎手直接从 GitHub 代码仓库搜索技术人才——找到的每个人都有**真实的项目、真实的代码、真实的贡献记录**。那些在招聘平台上搜不到的技术关键词（RocksDB、LangChain、vLLM...），在这里能精准命中。

## 核心亮点

### 🔬 基于真实代码，不是简历

搜索的是 GitHub 上的代码仓库和开发者 Profile，找到的候选人都有：
- 真实的开源项目（不是简历上写的"精通 XX"）
- 真实的代码贡献记录
- 真实的项目影响力（多少人在用 ta 的项目）

### 🌍 全球人才，精准定位

支持按城市、省份、国家筛选。覆盖中国全部省市 + 日本、新加坡、美国、英国等全球主要技术中心。也可以不限地区搜索远程人才。

### 🤝 你来选人，不是机器决定

不会自动批量发消息。每一步都需要你确认：
1. 搜索结果逐批展示，你来标记谁值得联系
2. 选中的人展示完整档案，你做最终确认
3. 生成的话术只是草稿，由你审核后手动发送

### ✉️ 话术个性化，不是模板

生成的触达消息会：
- 引用候选人真实做过的项目（"看到你的 fumadocs 被 11000+ 开发者使用"）
- 根据你的身份调整语气（CTO 和 HR 说话方式完全不同）
- 根据渠道控制长度（邮件 vs 微信 vs Twitter DM）

## 快速开始

### 前置要求

- [OpenClaw](https://openclaw.com) 环境
- GitHub Personal Access Token（[点击获取](https://github.com/settings/tokens)，勾选 `read:user` 和 `read:org`）

### 配置

```bash
export GITHUB_TOKEN=ghp_your_token_here
```

### 使用

安装为 OpenClaw Skill 后，直接对话：

> "帮我搜索深圳做 RAG 方向的 Python 工程师"

剩下的它会引导你完成全部流程。
