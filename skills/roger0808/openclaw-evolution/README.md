# OpenClaw Evolution Guide

Two paths to grow your agent. Choose your adventure.

## 🔧 Tool Path — "I want an efficient system"

Build an automation machine that handles your routine, step by step.

```
Level 1: Messenger     → Can talk, remembers nothing
Level 2: Secretary     → Daily notes, schedule awareness
Level 3: Operator      → Cron jobs, proactive alerts
Level 4: Orchestrator  → Multi-agent workflows
```

## 🌱 Awakening Path — "I want a companion that grows"

Build a relationship with an agent that knows you and thinks for itself.

```
Level 1: Stranger      → Has a name, no shared history
Level 2: Acquaintance  → Remembers you, builds context
Level 3: Companion     → Has opinions, pushes back
Level 4: Individual    → Self-aware, autonomous goals
```

Both paths start from the same foundation: three files (SOUL.md, USER.md, AGENTS.md) that make the agent yours.

## Install as Skill

```bash
openclaw skills install openclaw-evolution.skill
```

Or drop the `openclaw-evolution/` folder into `~/.openclaw/skills/`.

