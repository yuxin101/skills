# 参考音频目录

放置用于VoxCPM TTS的参考音频文件。

## 要求

- 格式：WAV（推荐）
- 时长：3-10秒
- 内容：清晰的语音录音
- 品质：无背景噪音

## 使用方法

1. 录制或选择一段清晰的语音作为参考
2. 将文件放入此目录，如 `speaker.wav`
3. 在 `config.json` 中配置：

```json
{
  "tts": {
    "reference_audio": "./reference_audio/speaker.wav",
    "reference_text": "参考音频对应的文字内容"
  }
}
```

## 注意

参考音频的质量直接影响配音效果，建议：
- 选择与目标配音风格相似的声音
- 确保录音清晰无杂音
- 文本内容与音频完全匹配
