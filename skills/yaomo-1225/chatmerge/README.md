# ChatMerge - 智能多渠道聊天纪要助手

> 一键读取 20+ 聊天平台，生成智能纪要，自动跟踪行动项，AI 主动建议改进方案

## ✨ 核心特性

### 🚀 一键自动读取
- 支持 20+ 聊天平台（Discord, Slack, Telegram, 企微, 钉钉, WhatsApp, Signal, iMessage 等）
- 无需手动复制粘贴，一句话搞定
- 智能频道发现，自动推荐要总结的频道

### 🧠 智能分析
- 跨平台去重和关联（识别同一讨论）
- 多维度分析（人员、情绪、效率）
- 自动提取摘要、决策、行动项、风险

### 📋 自动跟踪
- 行动项自动创建到 Jira/Notion/GitHub
- 持续跟踪进度，自动提醒延期任务
- 从后续聊天中识别任务完成

### 🤖 AI 智能建议
- 效率建议（如何提升决策效率）
- 风险建议（识别潜在问题）
- 流程建议（优化协作流程）

### ⏰ 自动化
- 定时纪要（每日站会、每周周报）
- 实时监控（关键事件立即通知）
- 增量更新（只处理新消息）

### 👥 多角色适配
- CEO 视角（极简，只看结果）
- 项目经理视角（详细，关注进度和风险）
- 开发者视角（技术细节）

## 🚀 快速开始

### 方式 1：一键自动读取（推荐）⭐

```
使用 $chatmerge，总结我昨天的讨论
```

ChatMerge 会自动：
1. 列出昨天有活动的所有频道
2. 让你选择要总结的频道
3. 生成智能纪要

### 方式 2：指定频道

```
使用 $chatmerge，总结 Discord #project-alpha 和 Slack #team-chat 最近 100 条消息
```

### 方式 3：文件导入

```
使用 $chatmerge，总结这个文件：/path/to/chat-export.json
```

## 📊 输出内容

### 基础纪要
- 📊 核心摘要（3-5 条关键结论）
- 💬 关键讨论（按主题分组）
- ✅ 决策记录
- 📋 行动项（任务/负责人/截止时间/状态）
- ⚠️ 风险与阻塞（分级：高/中/低）
- ❓ 待跟进问题

### 高级分析 ⭐ 新增
- 🔄 跨平台讨论追踪（识别同一讨论）
- 👥 人员分析（发言统计、活跃时段、沉默成员）
- 😊 情绪分析（整体情绪、焦虑话题、积极话题）
- ⚡ 效率分析（决策效率、行动项完成率）
- 🤖 AI 智能建议（效率、风险、流程）
