# 快速开始

## 最简单的使用方式

直接粘贴聊天记录，无需任何配置：

```
使用 $multi-channel-chat-minutes

[粘贴你的聊天记录]
```

## 常见场景

### 1. 每日站会纪要
```
使用 $multi-channel-chat-minutes，整理昨天的项目群聊天，生成站会纪要

[Telegram 群聊记录]
[Slack 频道记录]
```

### 2. 客户反馈汇总
```
使用 $multi-channel-chat-minutes，把这些客服群消息整理成客户反馈摘要

[客服群聊天记录]
```

### 3. 管理层周报
```
使用 $multi-channel-chat-minutes，汇总本周所有项目群讨论，输出管理层周报

[多个群的聊天记录]
```

## 支持的输入格式

✅ 直接粘贴的文本
✅ 平台导出的聊天记录（TXT、JSON、CSV）
✅ OCR 识别后的截图文本
✅ 已经整理好的结构化数据

## 输出内容

默认包含：
- 📊 核心摘要（3-5 条关键结论）
- 💬 关键讨论（按主题分组）
- ✅ 决策记录
- 📋 行动项（任务/负责人/截止时间/状态）
- ⚠️ 风险与阻塞
- ❓ 待跟进问题

## 高级用法

需要 JSON 格式输出？
```
使用 $multi-channel-chat-minutes，输出 JSON 格式

[聊天记录]
```

需要自定义配置？参考 [config-schema.md](references/config-schema.md)

## 注意事项

⚠️ 本 Skill 不会自动连接或登录任何聊天平台
⚠️ 请确保你有权限处理这些聊天内容
⚠️ 敏感信息会被自动过滤（token、密码等）
