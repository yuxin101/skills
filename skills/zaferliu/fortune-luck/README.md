# fortune_luck - 每日运势计算

基于中国农历黄历和西方星座的每日运势计算工具。

## 功能

- 🎂 根据生日计算每日运势
- 🏮 命理评分（生肖+五行+值神+宜忌）
- 🌟 星座评分（星座+星期能量）
- 🍀 幸运元素（颜色/数字/方位）
- 📋 宜忌提醒

## 安装

```bash
pip install lunar-python
```

## 使用方法

```python
from fortune_luck import FortuneCalculator, save_birthday

# 首次使用：保存生日
save_birthday("2004-03-08", "solar")  # 阳历
# save_birthday("2004-01-15", "lunar")  # 阴历

# 计算运势
calc = FortuneCalculator()
result = calc.calculate()

# 输出格式
print(result.to_text("share"))  # 分享版
print(result.to_text("detailed"))  # 详细版
print(result.to_json())  # JSON
```

## 评分逻辑

### 命理评分
- 值神吉凶（黄道日+6 / 黑道日-6）
- 生肖与日支关系（六合/冲/三合）
- 宜忌关键词

### 星座评分
- 星座先天运势
- 星期能量加成

### 幸运元素
- 基于日干五行：
  - 甲、乙（木）→ 绿色/青色, 3/8, 正东/东南
  - 丙、丁（火）→ 红色/紫色, 2/7, 正南
  - 戊、己（土）→ 黄色/棕色, 5/10, 中宫/西南/东北
  - 庚、辛（金）→ 白色/金色, 4/9, 正西/西北
  - 壬、癸（水）→ 黑色/蓝色, 1/6, 正北
