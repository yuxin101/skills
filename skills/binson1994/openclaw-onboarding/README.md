# OpenClaw 龙虾快速上手指南

> 预计用时：5-10分钟  
> 目标：让你知道虾能做什么，以及怎么用

---

## 📥 第一步：安装技能

### 方式一：本地安装

1. 下载技能包到本地
2. 执行安装命令：

```bash
clawhub install ./openclaw-onboarding --local
```
---

## 🚀 第二步：启动上手引导

安装完成后，对我说：

```
开始上手
```

我会带你完成5分钟快速配置，让你知道虾能做什么、怎么用。

---

## 📋 引导流程一览

| 步骤 | 做什么 | 你要做的 | 你会获得 |
|------|--------|----------|----------|
| **开场** | 介绍虾的能力 | 回复"好的" | 了解虾能做什么 |
| **配置** | 设置你的名字和工作 | 回答3个问题 | 个性化设置 |
| **发现** | 查看当前技能 | 问"你都能做什么" | 能力清单 |
| **记忆** | 学会让虾记住事情 | 说"记住XXX" | 永久记忆功能 |
| **扩展** | 安装技能查找器 | 说"自己学习找技能" | 自主扩展能力 |
| **实战** | 群聊总结演示 | 问"总结XX群昨天说了什么" | 群聊分析能力 |

---

## 💡 上手完成后，你可以这样用虾

### 1. 随时问"你都能做什么"
查看当前所有可用的技能和工具

### 2. 说"记住XXX"
让虾永久记住重要信息，比如：
- "记住我下周三要交Q2方案"
- "记住这个项目的负责人是张经理"

### 3. 让虾总结群聊
直接问，不需要拉虾进群：
- "总结一下产品讨论群昨天的重要信息"
- "提取项目沟通群的待办事项"

### 4. 让虾自己找技能
- "自己学习找技能" → 安装技能查找器
- "帮我找一个浏览器技能" → 自动搜索安装
- "我想做数据分析" → 推荐相关技能

### 5. 直接吩咐任务
- "帮我整理桌面文件"
- "搜索一下最新的AI行业动态"
- "写一份周报，总结这周的工作"

---

## 🎯 核心指令速查

| 你想做的 | 这样说 |
|----------|--------|
| 查看所有能力 | "你都能做什么" |
| 让虾记住事情 | "记住XXX" |
| 总结群聊 | "总结一下XX群昨天说了什么" |
| 找新技能 | "自己学习找技能" 或 "帮我找XXX技能" |
| 获取帮助 | "帮助" 或 "?" |
