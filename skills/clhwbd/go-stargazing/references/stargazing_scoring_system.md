# 星空摄影选址评分体系 v1.0

> 基于气象学原理 + 小红书社区经验 构建
> 更新日期：2026-03-22

---

## 一、总分构成

总分 = 气象条件分（60分） + 天文条件分（20分） + 地理环境分（20分） + 人造光污染分（15分）

| 维度 | 满分 | 说明 |
|------|------|------|
| 气象条件 | 60分 | 云量/透明度/湿度/雾/风 |
| 天文条件 | 20分 | 月相/光污染 |
| 地理环境 | 20分 | 海拔/地形/纬度 |
| 人造光污染 | 15分 | 城镇距离 / 景区灯光 / 主拍方向灯光 |

> 落地口径说明：当前采用 **14 指标框架** —— 其中 **10 项计分指标** 构成 115 分总分，另有 **4 项强制检查/降级指标** 用于风险提示，不额外加分。

## 二、气象条件（60分）

### 2.1 云量与透明度（0-20分）

| 云量/透明度 | 得分 | 说明 |
|-------------|------|------|
| 总云量 ≤ 10%，无低云/中云 | 20分 | 绝佳条件 |
| 总云量 10-20%，基本无云 | 18分 | 优秀 |
| 总云量 20-30%，高云薄 | 15分 | 良好 |
| 总云量 30-50% | 10分 | 一般 |
| 总云量 50-70% | 5分 | 差 |
| 总云量 > 70% | 0分 | 不建议 |

**降级规则：**
- 有低云（高度 < 3000m）覆盖 → 直接扣5分
- 有中高云漂移风险（数值预报显示云量后半夜增加）→ 扣3分
- 西北区域有沙尘/浮尘（PM10 > 100 μg/m³ 或 AOD > 0.3）→ 扣5-10分

### 2.2 湿度与露点（0-15分）

| 湿度条件 | 得分 |
|---------|------|
| 相对湿度 < 50%，露点距温度 > 8°C | 15分 |
| 相对湿度 50-65%，露点距温度 5-8°C | 12分 |
| 相对湿度 65-80%，露点距温度 3-5°C | 8分 |
| 相对湿度 > 80%，露点距温度 < 3°C | 3分 |
| 相对湿度 > 90% 或露点 ≈ 温度 | 0分 |

**降级规则：**
- 湖边/海边/水边 → 湿度风险 +1级（降3分）
- 山谷/盆地 → 夜间冷池效应，湿度 +1级
- 高海拔（>3000m）→ 湿度容忍度更高，不降级

### 2.3 雾与可见度（0-15分）

| 雾/可见度条件 | 得分 |
|--------------|------|
| 可见度 > 20km，无雾风险 | 15分 |
| 可见度 15-20km，轻微霾 | 12分 |
| 可见度 10-15km | 8分 |
| 可见度 5-10km，轻雾 | 3分 |
| 可见度 < 5km 或有浓雾 | 0分 |

**特殊地形风险：**
- 沿海地区：后半夜海雾入侵风险 → 扣3分
- 山谷/盆地：夜间谷雾积累 → 扣3分
- 高原边缘：地形低云压境 → 扣3分

### 2.4 风速（0-10分）

| 风速条件 | 得分 | 说明 |
|---------|------|------|
| 2-10 km/h | 10分 | 最佳，驱散湿气又不影响拍摄 |
| 10-20 km/h | 8分 | 良好 |
| < 2 km/h | 6分 | 无风，湿气易积累 |
| 20-35 km/h | 4分 | 勉强可用 |
| > 35 km/h | 0分 | 三脚架不稳，不建议 |

## 三、天文条件（20分）

### 3.1 月相与月出月落（0-10分）

| 月相条件 | 得分 |
|---------|------|
| 无月夜（新月前后5天内）或月亮在地平线以下 | 10分 |
| 月相 5-10% 或月亮后半夜才升起 | 8分 |
| 月相 10-25% | 5分 |
| 月相 25-50% | 2分 |
| 月相 > 50% | 0分 |

**银河摄影重点：** 月光是银河摄影最大干扰，满月前后各3天基本无法拍银河

### 3.2 光污染（0-10分）

| 波特尔等级（Bortle） | SQM 参考值 | 得分 |
|---------------------|-----------|------|
| Bortle 1-2（完全黑暗）| SQM ≥ 21.0 | 10分 |
| Bortle 3（ rural dark）| SQM 20.5-21.0 | 8分 |
| Bortle 4（ rural/suburban）| SQM 19.5-20.5 | 6分 |
| Bortle 5-6（郊区）| SQM 18.5-19.5 | 3分 |
| Bortle 7-8（城市/城郊）| SQM < 18.5 | 0分 |

> **SQM 读数参考**（来自小红书社区经验）：
> - SQM 16-18：城市/近郊，只能看到最亮的星星
> - SQM 19-20.5：乡村/远郊，可见银河轮廓 ⭐
> - SQM 21.0-21.5：非常好的暗夜环境，银河清晰可见
> - SQM ~22.0：理论极限，地球上极罕见

> **国标暗夜等级补充**（5级制，适合中国国情）：
> 国标1级 ≈ 波特尔3-4级上限；国标5级 ≈ 波特尔5级
> 国标在4级区间内做了细分，更适合中国东部实际条件

## 四、地理环境（20分）

### 4.1 海拔（0-8分）

| 海拔 | 得分 |
|-----|------|
| 2500-4000m | 8分 | 最佳视宁度，低云在脚下 |
| 1500-2500m | 6分 | 良好，空气稀薄通透 |
| 800-1500m | 5分 | 一般 |
| 4000m以上 | 4分 | 虽有优势但低温/缺氧风险 |
| 200-800m | 4分 | 平原/丘陵，有一定通透度 |
| < 200m | 2分 | 低海拔，湿气重 |

### 4.2 地形（0-6分）

| 地形 | 得分 | 说明 |
|-----|------|------|
| 高山草甸/山脊/台地（高海拔）| 6分 | 视野开阔，低云在脚下 |
| 沙漠/戈壁/荒原 | 5分 | 干燥，但可能有沙尘 |
| 高原湖泊（湖畔高台）| 5分 | 有倒影机会，但要防湖雾 |
| 开阔草原 | 4分 | 视野好 |
| 山区（非山脊）| 3分 | 可能有山雾 |
| 谷底/盆地 | 1分 | 冷池效应，雾积累 |
| 沿海平原 | 2分 | 海雾入侵风险 |

### 4.3 纬度与银河可见性（0-6分）

| 纬度/位置 | 得分 |
|----------|------|
| 北纬 25-40° | 6分 | 银河核心最佳可见区 |
| 北纬 20-25° 或 40-45° | 4分 | 银河可见但角度较低 |
| 北纬 < 20°（南方）| 3分 | 银河可见但偏向南方 |
| 北纬 > 45°（北方）| 2分 | 银河核心在地平线附近时间短 |

## 六、降级规则（强制）

以下情况必须强制降级：

| 条件 | 降级幅度 |
|------|---------|
| 只有白天数据，无夜间逐小时 | 直接 → 待确认 |
| 华南沿海（广东/广西/福建/海南）无云 | 至少降1级（海雾风险） |
| 西北无云但 PM10/dust 明显 | 至少降1级 |
| 山区/谷地夜间无风 + 高湿 | 至少降1级 |
| 月相 > 50% 做银河摄影 | 直接 → 不建议 |
| 湿度 > 85% + 露点距温度 < 2°C | 至少降2级（结露风险） |
| 可见度 < 5km | 直接 → 不建议 |

### 10.5 人造光综合评分（0-15分）

本项为独立评估项，分数直接加入总分：

| 条件 | 得分 |
|-----|------|
| 距最近城市/镇 ≥ 30km，夜间无明显人造光 | +15分 |
| 距最近城市 20-30km，有微弱光害 | +12分 |
| 距最近城市 10-20km，有明显光害 | +7分 |
| 距最近城市 5-10km，有强烈光害 | +3分 |
| 距最近城市 < 5km 或在镇区/景区核心 | +0分 |

**特殊降级：**
- 景区内有夜间强光营地（23:00前不关闭）→ 额外扣3分
- 主要拍摄方向（春夏季银河东南-西南）有城镇灯光 → 扣3分

**典型景区夜间光害风险：**
- 沙漠景区（鸣沙山等）：高风险，夜间营地灯光强
- 湖泊景区（茶卡、泸沽湖等）：高风险，湖边度假村灯光
- 山顶景区（牛背山等）：中高风险，山顶客栈灯光
- 草原景区（巴丹吉林等）：中风险，蒙古包营地灯光
- 丹霞景区（张掖等）：低风险，夜间一般无照明

**实操建议：** 夜间22:00后大部分景区灯光会关闭，选择这个时间之后拍摄可有效避开人造光干扰。

## 七、特殊场景修正

### 7.1 银河摄影专项修正

银河摄影在普通评分基础上增加以下修正：

| 修正项 | 修正幅度 |
|--------|---------|
| 银河核心在地平线以上时段 | +5分 |
| 银河核心在地平线以下时段 | -5分 |
| 无月夜 + 银河最佳季节（3-10月） | +3分 |
| 满月前后各3天 | 银河摄影 → 直接不建议 |

### 7.2 流星雨专项修正

| 修正项 | 修正幅度 |
|--------|---------|
| 辐射点在地平线以上 | +5分 |
| 无月干扰 | +5分 |
| 东北方向无遮挡 | +3分 |

### 7.3 地区专项修正

**华南地区（广东/广西/福建/海南）：**
- 湿度永远是第一风险项
- 沿海只建议冬季干燥季（11月-次年3月）
- 夏季基本不建议
- 海拔 > 1000m 的山顶能"高出霾层"，是优先选择

**东部沿海地区（江苏/浙江/上海/福建）：**
- 空气湿度 + 雾霾 > 光污染，是首要风险
- 雾霾会让3级光污染和2级看起来差别不大
- **海拔优先**：华东海拔 1000m+ 山顶可明显穿透低层霾
- 避开主拍摄方向的城镇光源（如夏季银河在东南方向，则东南方向无遮挡为佳）
- 冬季冷空气过后天气更通透 ⭐

**西北地区（新疆/青海/甘肃/内蒙古西部）：**
- 透明度永远是第一风险项
- 春季大风季注意沙尘（PM10 突增）
- 冬季最佳（干燥 + 银河可见性好）
- 光污染条件通常优于东部，是拍星首选区

**高原山区（西藏/川西/云南）：**
- 低云和山雾是主要风险
- 海拔 > 3000m 要防高反
- 湖泊拍摄要防湖雾（湖面水汽夜间冷凝）
- 高原空气稀薄，通透度极好，是拍星天堂

## 八、来自小红书社区的核心洞察

> 以下洞察来自小红书星空摄影社区 25 篇实战笔记（2026-03-22 抓取）

### 最重要的结论

**1. 光污染 ≠ 首要因素**
> "从我星空的拍摄经历来看，光污染并不是首位度第一的条件，具体需要根据不同环境考虑。"——星空进阶之路

东部沿海：空气湿度 + 雾霾的影响往往 > 光污染。3级光污染 + 干燥通透 的效果，可能优于 1级光污染 + 雾霾。

**2. 海拔是东部拍星的第一武器**
华东海拔 1000m+ 山顶，夜间能明显观察到底下的一层薄霾，高海拔 = 穿透霾层 = 星空更清晰。

**3. 拍摄方向 > 绝对光污染值**
在同等光污染等级下，避开主拍摄方向的城镇灯光，比追求更低 Bortle 等级更重要。夏季银河主要在东南方向 → 东南无遮挡 > Bortle 低两级。

**4. 天气时机比绝对云量更重要**
- 夏季：副高下、雷雨后 → 空气更通透
- 冬季：冷空气过后 → 空气最干净
- 不只是"晴天"，而是"什么类型的晴天"

**5. 雾和霾是隐形杀手**
- 雾霾会让星空形成"朦胧滤镜"
- 低海拔（< 500m）+ 无风 + 高湿 = 夜间雾霾积累高风险

**6. SQM 是量化光污染的好工具**
- 19-20.5：乡村/远郊，可见银河轮廓 ⭐
- 21.0-21.5：非常好的暗夜环境
- 建议拍星者随身携带 SQM-LU 测量仪

|--------|---------|

## 十、人造光污染补充评估（2026-03-22 新增）

> 来自小红书社区经验：离城市/景区太近会被人造光严重干扰

### 10.1 城市光害距离（强制评估项）

| 与最近城市的距离 | 得分 | 说明 |
|-----------------|------|------|
| ≥ 30km，无明显人造光 | 15分 | 绝佳暗夜区 |
| 20-30km，有微弱光害 | 12分 | 良好 |
| 10-20km，有明显光害 | 7分 | 一般，银河仍可见但质量受限 |
| 5-10km，有强烈光害 | 3分 | 差，只能拍亮星 |
| < 5km 或在城市/镇内 | 0分 | 不适合拍星 |

### 10.2 景区人造光（容易被忽视）

很多景区本身夜间有强光照明，是容易被忽略的光害来源：

| 景区类型 | 光害风险 | 备注 |
|---------|---------|------|
| 沙漠景区（鸣沙山等）| 高 | 夜间路灯、营地灯光强 |
| 湖泊景区（茶卡、泸沽湖等）| 高 | 湖边度假村灯光 |
| 草原景区（巴丹吉林等）| 中 | 蒙古包营地灯光 |
| 山顶景区（牛背山等）| 中-高 | 山顶客栈灯光直接污染 |
| 丹霞景区（张掖等）| 低-中 | 夜间一般无照明 |

**应对策略：**
- 避开景区核心区，选择景区外围的高地
- 夜间 22:00 后大部分景区灯光会关闭
- 询问当地住宿老板光害情况

### 10.3 与主要拍摄方向的相对位置

即使距离城市不算近，如果城市位于主要拍摄方向，仍是严重干扰：

银河主要方向（春夏季：东南→西南）：
- 检查主要拍摄方向 30° 扇形范围内有无城镇灯光
- 银河在地平线附近时城市灯光干扰最严重

### 10.4 降级规则更新

以下情况必须强制降级：
- 最近城市 < 5km → 直接降 2 级
- 景区内有强光营地灯光（夜间 23:00 前不关闭）→ 降 1 级
- 主要拍摄方向有城市灯光 → 降 1 级
