# 中考短文填空格式规范参考

## 题目结构识别

中考短文填空题目通常包含：
1. **短文正文**：若干段落，包含编号填空（36.～45.）
2. **备选词框**：通常在短文下方或右侧，10个英语单词用空格分隔
3. **答案图片**：单独的答案页，标注每题所填单词

## 答案提取规则

### 答案图片识别 prompt

```
请准确识别这张答案图片，提取每个编号对应的答案单词。
返回格式：编号.答案，如 36.find, 37.camera
如果某题未填写或图片模糊，标注 36.未提供
```

### 答案展示位置

| 输出渠道 | 答案位置 | 格式 |
|---------|---------|------|
| Word 文档 | 题目正文下方 | `36.答案  37.答案  ...`  横向排列 |
| 飞书多维表格 | 右侧列（独立字段） | 同上，纵向或横向均可 |

## 格式还原检查清单

- [ ] 原文段落结构（换行位置）完全保留
- [ ] 填空编号 36.～45. 全部存在
- [ ] 下划线数量与原图一致
- [ ] 备选词已转为顿号分隔
- [ ] 大小写与原文一致
- [ ] 括号注释完整保留
- [ ] 答案图片已识别并正确填写答案

## Word 文档结构模板

```markdown
# 中考短文填空-标准化题目

## 备选词
by、camera、clearly、delicious、find、future、great、offer、rush、way

## 短文正文
[还原后正文，保留段落]

## 答案
36.________  37.________  38.________  39.________  40.________
41.________  42.________  43.________  44.________  45.________
```

## 推送格式模板

```markdown
# 📝 中考短文填空-标准化题目

## 短文正文
[还原后正文，保留段落]

## 备选词（10个）
by、camera、clearly、delicious、find、future、great、offer、rush、way

## 答案（如有）
36.future  37.clearly  38.offer  39.rush  40.way
41.great  42.camera  43.delicious  44.find  45.by
```

## 常见错误警示

⚠️ **过度合并**：不要把多段正文合并成一段
⚠️ **遗漏备选词**：10个词一个都不能少
⚠️ **答案错位**：答案编号必须与题目编号严格对应
⚠️ **大小写错误**：如 camera 不能写成 Camera
⚠️ **下划线不足**：____数量必须与原题一致
