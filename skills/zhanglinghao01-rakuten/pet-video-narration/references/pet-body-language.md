# 宠物肢体语言参考手册

按物种整理常见肢体动作与情绪/意图解读。

---

## 🐰 兔子（Rabbit）

### 行为库

| 动作 | 行为名称 | 含义 |
|------|---------|------|
| 四肢完全伸展侧趴（Sploot） | Full Flop / Sploot | **极度放松**，对环境完全信任，安全感爆棚 |
| 突然侧翻倒下 | Flop | 放松到极致，准备睡觉，有时看起来像抽搐 |
| 母鸡蹲（脚收在身下） | Loaf | 休息但保持警觉，两种状态之间的过渡 |
| 耳朵向前竖立 | Attentive | 好奇或警觉，专注于某事 |
| 耳朵向后倾斜 | Relaxed | 放松、满足 |
| 耳朵贴近背部 | Very Relaxed | 极度放松或准备入睡 |
| 鼻子快速抖动 | 正常生理 | 探测气味，也可能是好奇/紧张 |
| 原地跳跃旋转（Binky） | Binky | **极度开心**，兴奋到失控 |
| 后腿快速奔跑（无方向） | Zoomies | 释放能量，通常在放松后 |
| 用下巴蹭物/蹭人 | Chinning | 宣示主权，该物品/人是我的 |
| 绕脚转圈 | Circling | 发情或求关注，或表示领地巡逻 |
| 后腿用力蹬地 | Thumping | 警告：感到危险或不满 |
| 低沉咕咕叫 | Teeth grinding | **身体不适或极度紧张**，需注意 |
| 轻声磨牙 | Soft teeth grinding | **满足、舒服**（如被抚摸时） |
| 突然全身绷直僵硬 | 冻结反应 | 感到威胁或高度警觉，分析环境中 |

### 常见场景解读

- **听到响指 → 母鸡蹲 → Sploot**：期待食物/关注落空，但判断环境安全 → 赌气式放松
- **绕圈 + 蹭脚**：领地巡逻，或想引起主人注意
- **Binky + Zoomies**：太开心了，需要释放

## 🐱 猫（Cat）

### 行为库

| 动作 | 含义 |
|------|------|
| 缓慢眨眼 | 信任、喜欢你（猫的微笑） |
| 耳朵向前 | 好奇、感兴趣 |
| 耳朵向后/平贴 | 紧张、害怕或愤怒 |
| 尾巴竖直 + 末端弯曲 | 开心、友好 |
| 尾巴蓬松炸毛 | 受到惊吓或领地受到威胁 |
| 露出肚子 | **极度信任**（但不一定想被摸） |
| 前爪交替揉捏 | 幼猫记忆，**极度放松和满足** |
| 身体压低蹭地 | 标记领地，或准备捕猎 |
| 眯眼 | 放松、满足、昏昏欲睡 |
| 直视不动 | 挑战或威压，等级展示 |
| 舔毛 | 整理自己，也是在平复情绪 |
| 后腿踢蹬 | 捕猎本能，或玩耍过于兴奋 |

## 🐶 狗（Dog）

### 行为库

| 动作 | 含义 |
|------|------|
| 尾巴大力摇摆 + 屁股跟着动 | **极度开心**，见到主人 |
| 尾巴竖高僵硬 | 警觉或兴奋，处于警戒状态 |
| 尾巴夹在两腿中间 | **恐惧、投降** |
| 露出牙齿 + 低吼 | 警告，守护领地 |
| 前爪趴下 + 屁股翘高 | 邀请玩耍（Play Bow） |
| 仰面露出肚子 | **完全信任**，请求抚摸 |
| 耳朵竖起前倾 | 好奇，专注于某事 |
| 耳朵贴后脑勺 | 紧张或顺从 |
| 用鼻子轻推 | 引起注意，或表示亲近 |
| 舔人脸/手 | 讨好、顺从，或饿了 |
| 打哈欠 | 压力大/紧张，也可能是困了 |
| 四处嗅闻 | **探索环境**，信息收集 |
| 前腿刨地 | 捕猎本能延续，或储存食物的习性 |

## 🐹 仓鼠/荷兰猪（Hamster / Guinea Pig）

### 仓鼠

| 动作 | 含义 |
|------|------|
| 整理毛发 | 放松，环境感到安全 |
| 跑滚筒不停 | 正常行为，但过度可能焦虑 |
| 塞满两腮 | 正常觅食，储备食物 |
| 突然僵住 | 警觉，感到威胁 |
| 夜间活跃白天睡觉 | 正常作息 |

### 荷兰猪

| 动作 | 含义 |
|------|------|
| 尖叫 | 饿了或不满 |
| 轻轻哼叫 | 满足、舒服 |
| 头部轻推 | 求关注 |
| 跳跳（Popcorning） | **极度开心**，青少年尤为明显 |
| 舔手 | 表示信任 |

## 通用意图判断框架

### 第一问：宠物在视频里**想要什么**？

| 目标 | 典型动作 | 常见场景 |
|------|---------|---------|
| 食物/零食 | 看向主人、鼻子抽动、前爪抬起 | 饭点、主人吃东西时 |
| 关注/陪伴 | 蹭腿、绕圈、轻声叫 | 主人长时间不理会 |
| 探索/好奇 | 耳朵前倾、嗅闻、不停张望 | 新环境、新气味 |
| 释放精力 | Binky、Zoomies、疯跑 | 情绪极度兴奋 |
| 休息/睡觉 | Sploot、Loaf、侧躺 | 感到安全、满足 |
| 领地宣示 | 蹭下巴、绕圈、喷尿 | 多宠环境 |

### 第二问：**期待是否被满足**？

| 状态 | 心理状态 | 文案语气 |
|------|---------|---------|
| 期待满足 | 开心、满足 | 愉快、撒娇、得意 |
| 期待落空 | 委屈、赌气、失落 | 小脾气、傲娇、略带不满 |
| 期待未知 | 好奇、警觉 | 探索感、紧张、神秘 |
| 无期待，处于安全状态 | 极度放松 | 得意、傲娇、或者懒散 |

## 物种差异与文案语气建议

| 物种 | 推荐语气 | 避免 |
|------|---------|------|
| 兔子 | 傲娇、小心眼、假装不在意 | 太热情、太讨好 |
| 猫 | 高冷、爱搭不理、若即若离 | 过于讨好、过度兴奋 |
| 狗 | 热情、直接、忠诚但有点傻 | 过于高冷 |
| 仓鼠/荷兰猪 | 呆萌、贪吃、小心翼翼 | 太长、太成熟 |
| 鸟类 | 好奇、活泼、有点话痨 | 太严肃 |
