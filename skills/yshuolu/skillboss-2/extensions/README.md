## Included Extensions

### Remotion Skills
- **Source**: https://github.com/remotion-dev/skills
- **License**: Check original repository
- **Maintainer**: Remotion team
- **Description**: Best practices for developing video applications with Remotion (React-based video framework)

## Usage

Each extension has its own SKILL.md. When a user needs an extension's
capability, read the corresponding SKILL.md for instructions.

| Extension | Path | Use When |
|-----------|------|----------|
| Remotion | `./extensions/remotion/SKILL.md` | Develop video apps with React (Remotion framework) |
