# stealth-break - 健康摸鱼技能 🎯

> 健康第一，适度休息，隐蔽放松，长久工作

## 功能特性

- 📋 **摸鱼策略建议**: 4 大类 12 种隐蔽放松方法
- ⏰ **定时健康提醒**: 自动提醒眼睛休息、腰椎保护、深呼吸、提肛运动
- 🔔 **macOS 系统通知**: 使用系统通知推送，无需额外配置
- 🎯 **隐蔽放松技巧**: 开会时也能做，没人知道

## 使用场景

- 长时间工作后需要休息
- 想摸鱼但怕被发现
- 需要缓解工作压力
- 预防颈椎病、腰间盘突出、猝死

## 触发方式

直接对话触发：
- "我累了，想摸鱼"
- "有什么隐蔽的休息方法"
- "帮我设置健康提醒"
- "工作太紧张，怕猝死"

## 提醒类型

| 类型 | 频率 | 内容 |
|------|------|------|
| 👁️ 眼睛休息 | 每 60 分钟 | 远眺 + 闭眼 + 眨眼 |
| 🪑 腰椎保护 | 每 90 分钟 | 起身走动 + 腰部伸展 |
| 🧘 深呼吸 | 每 120 分钟 | 4-4-6 呼吸法 |
| 💪 提肛挑战 | 每 120 分钟 | 收缩放松练习 |

## 安装

```bash
npx clawhub install stealth-break
```

## 配置

技能会自动创建 cron 定时任务，通过 macOS 系统通知推送。
