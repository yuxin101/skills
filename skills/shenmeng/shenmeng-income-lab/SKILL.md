# Income Lab 收入实验室

> 💰 **本 Skill 已接入 SkillPay 付费系统**
> - 每次调用费用：**0.01 USDT**
> - 支付方式：BNB Chain USDT

系统性尝试、记录、优化赚钱方式的实验框架。

> "赚钱不是运气，是可迭代的实验。"

## 核心理念

1. **小步快跑**：快速尝试多种小额赚钱方式
2. **数据驱动**：记录投入和产出，用数据决策
3. **持续迭代**：不断调整策略，放大有效的方式
4. **复利思维**：从即时收入逐步建立被动收入

## 适用场景

| 场景 | 说明 |
|------|------|
| **从零开始** | 没技能、没本金，不知道怎么赚钱 |
| **多点尝试** | 同时做几种赚钱方式，需要跟踪效果 |
| **优化收入** | 已有收入来源，想要提升效率 |
| **建立系统** | 从主动收入过渡到被动收入 |

## 核心功能

### 1. 赚钱方式数据库
- 分类整理各种合法赚钱方式
- 标注门槛、投入、预期收益、风险
- 根据个人情况推荐适合的方式

### 2. 实验追踪系统
- 记录每次赚钱尝试的详细数据
- 计算投入产出比（ROI）
- 可视化收入趋势

### 3. 执行提醒与复盘
- 定时提醒执行任务
- 每周/每月自动复盘
- 生成优化建议

### 4. 策略优化引擎
- 分析哪些方式有效
- 建议放大或放弃的策略
- 提供下一步行动计划

## 赚钱方式分类

### Tier 1：立即变现（当天见钱）
- 众包平台（阿里众包、蜂鸟众包）
- 跑腿配送
- 问卷调查
- 临时任务

**特点**：门槛低，收入低，不稳定

### Tier 2：技能变现（1-4周见效）
- 写作/翻译
- 设计/PPT制作
- 视频剪辑
- 数据录入

**特点**：需要学习，收入中等，较稳定

### Tier 3：小生意（1-3个月见效）
- 闲鱼无货源
- 信息中介
- 社群运营
- 内容创作

**特点**：需要运营，收入较高，有复利

### Tier 4：资产性收入（3-6个月见效）
- 自媒体账号
- 加密货币空投
- 联盟营销
- 知识付费

**特点**：前期投入大，后期被动收入

## 工作流程

### 第一步：现状评估
了解用户当前情况：
- 有多少时间？
- 有什么技能？
- 有多少本金？
- 风险承受能力？

### 第二步：方式推荐
根据评估推荐3-5个适合的赚钱方式

### 第三步：实验设计
为每个方式设计实验：
- 实验周期（1周/1月）
- 预期投入（时间+资金）
- 成功标准（收入目标）

### 第四步：执行追踪
- 每日/每周记录执行情况
- 记录收入和遇到的问题
- 计算实际ROI

### 第五步：复盘优化
- 分析哪些实验成功
- 总结成功/失败原因
- 制定下一轮实验计划

## 快速开始

### 1. 初始化你的收入实验室
```bash
python scripts/experiment_tracker.py init --name "我的收入实验"
```

### 2. 添加第一个实验
```bash
python scripts/experiment_tracker.py add \
  --name "蜂鸟众包测试" \
  --type "tier1" \
  --duration 7 \
  --time-budget 2 \
  --income-target 100
```

### 3. 每日记录
```bash
python scripts/experiment_tracker.py log \
  --date 2025-03-26 \
  --hours 2 \
  --income 35 \
  --notes "接了3单，平均每单11元"
```

### 4. 生成周报
```bash
python scripts/weekly_reporter.py generate --week 2025-12
```

## 使用示例

### 示例1：从零开始规划

**用户**："我什么都没有，每天4小时，怎么开始？"

**系统响应**：
```
基于你的情况，推荐3个实验：

【实验1】蜂鸟众包测试（Tier 1）
- 周期：1周
- 投入：每天2小时
- 目标：赚100元
- 目的：测试即时变现能力

【实验2】剪映技能学习（Tier 2）
- 周期：2周
- 投入：每天1小时学习
- 目标：能独立完成一个视频
- 目的：建立可变现技能

【实验3】闲鱼无货源调研（Tier 3）
- 周期：1周
- 投入：碎片时间
- 目标：找到3个可操作的商品
- 目的：为未来小生意做准备

建议先启动实验1，同时开始实验2的学习。
```

### 示例2：实验复盘

**用户**："我已经做了一周蜂鸟众包"

**系统响应**：
```
📊 蜂鸟众包实验复盘（第1周）

投入：
- 时间：14小时（每天2小时）
- 收入：126元
- 时薪：9元/小时

分析：
✓ 成功：确实能赚到钱，门槛低
✗ 问题：时薪太低，体力消耗大

建议：
- 继续执行，但减少时间到每天1小时
- 把节省的1小时用于学习剪辑技能
- 目标：3周后转型技能变现

下一步行动：
1. 今天开始在B站学剪映（实验2）
2. 蜂鸟众包减至每天1小时
3. 周末总结本周学习成果
```
