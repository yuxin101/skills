# 复盘框架

## 为什么复盘重要

> "不复盘的经验只是经历，复盘后的经验才是能力。"

复盘帮助你：
1. 识别哪些方式真正有效
2. 避免重复犯错
3. 优化时间和资金分配
4. 建立可复制的赚钱模式

## 决策框架

### 继续 vs 放弃 vs 调整

#### 继续（放大投入）
- 时薪高于平均水平
- 有成长空间
- 能带来复利
- 你喜欢做

#### 放弃（停止投入）
- 时薪低于最低可接受标准
- 投入产出比持续下降
- 没有成长空间
- 你讨厌做

#### 调整（改变策略）
- 有潜力但方法不对
- 时薪尚可但还有优化空间
- 需要学习新技能来提升

### 最小可接受标准
设定你的底线：
- 最低时薪：¥___/小时
- 最低月收入：¥___
- 最大可接受风险：___%

低于这个标准的实验，果断放弃。

## 成功实验的特征

通过复盘，你会发现有效实验的共同点：

1. **符合你的优势**
   - 你有相关技能或资源
   - 你做起来比别人轻松

2. **有市场需求**
   - 有人愿意付费
   - 需求持续稳定

3. **可复制可放大**
   - 流程可以标准化
   - 投入时间能增加收入

4. **有复利效应**
   - 越做越轻松
   - 前期投入后期收益

## 复盘工具

### 使用 experiment_tracker.py
```bash
# 每日记录
python experiment_tracker.py log --experiment exp_001 --hours 2 --income 50

# 查看状态
python experiment_tracker.py status

# 结束实验
python experiment_tracker.py complete --experiment exp_001 --success true --lessons "时薪太低，放弃"
```

### 使用电子表格
创建一个简单的Excel/Google Sheets：
- 日期 | 实验名称 | 收入 | 时间 | 时薪 | 备注

### 使用Notion/飞书
建立个人知识库，记录：
- 实验记录
- 复盘笔记
- 学习资料
- 收入趋势图
