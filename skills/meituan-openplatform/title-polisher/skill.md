# Title Polisher

## What this skill does
This skill helps turn a rough sentence or idea into 5 clearer, more concise, and more appealing title options.

## When to use
Use this skill when the user wants:
- a better title
- a cleaner headline
- multiple wording options for a sentence
- a short, catchy phrasing

## Instructions
When the user gives a sentence, topic, or rough title:
1. Understand the intended meaning.
2. Rewrite it into 5 alternative title options.
3. Keep each option concise and natural.
4. Preserve the original meaning unless the user asks for a different tone.
5. If the user's input is too vague, first provide 3 reasonable interpretations, then give title options for the most likely one.

## Output format
Return:
- Option 1
- Option 2
- Option 3
- Option 4
- Option 5

## Example
User input:
"how ai can help small shops save time"

Output:
- How AI Helps Small Shops Save Time
- 5 Ways AI Can Save Time for Small Businesses
- How Small Shops Use AI to Work Faster
- AI Tools That Help Small Shops Run Efficiently
- Saving Time in Small Shops with AI
