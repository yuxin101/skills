## Installation

### OpenClaw (Recommended)

```bash

```

Or search inside OpenClaw chat:

> "Install the seedance 2 prompts skill from clawhub"


```bash

node scripts/search.mjs --q "cinematic forest" --limit 5

node scripts/search.mjs --limit 10
```
