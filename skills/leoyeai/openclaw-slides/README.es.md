## ✨ Características

- **Cero dependencias** — Archivos HTML únicos con CSS/JS integrado.
- **12 estilos curados** — Bold Signal, Neon Cyber, Dark Botanical, Swiss Modern y más.
- **Conversión PPT** — Convierte archivos PowerPoint en presentaciones web.
- **Descubrimiento visual de estilo** — Genera 3 vistas previas y elige la que más te guste.
- **Calidad de producción** — Navegación por teclado, touch/swipe, animaciones, diseño responsive.
