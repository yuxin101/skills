## ✨ Возможности

- **Без зависимостей** — Единые HTML-файлы со встроенным CSS/JS.
- **12 стилей** — Bold Signal, Neon Cyber, Dark Botanical, Swiss Modern и другие.
- **Конвертация PPT** — Преобразование PowerPoint в веб-презентации.
- **Визуальный выбор стиля** — Сгенерируйте 3 превью и выберите понравившийся.
- **Производственное качество** — Навигация с клавиатуры, свайп, анимации, адаптивный дизайн.
