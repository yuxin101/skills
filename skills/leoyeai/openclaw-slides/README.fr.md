## ✨ Fonctionnalités

- **Zéro dépendance** — Fichiers HTML autonomes avec CSS/JS intégré.
- **12 styles personnalisés** — Bold Signal, Neon Cyber, Dark Botanical, Swiss Modern, et plus encore.
- **Conversion PPT** — Convertissez vos fichiers PowerPoint en présentations web.
- **Découverte visuelle** — Générez 3 aperçus et choisissez ce que vous aimez.
- **Qualité production** — Navigation clavier, swipe tactile, animations, design responsive.
