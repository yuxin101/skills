## ✨ Funzionalità

- **Zero dipendenze** — File HTML singoli con CSS/JS integrato.
- **12 stili curati** — Bold Signal, Neon Cyber, Dark Botanical, Swiss Modern e altri.
- **Conversione PPT** — Converti file PowerPoint in presentazioni web.
- **Scoperta visiva dello stile** — Genera 3 anteprime e scegli quella che preferisci.
- **Qualità produzione** — Navigazione da tastiera, touch/swipe, animazioni, design responsive.
