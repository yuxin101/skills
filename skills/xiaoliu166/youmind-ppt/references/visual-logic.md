# 视觉逻辑映射指南

本文档定义了内容逻辑关系与视觉结构之间的映射规则。阶段2中，为每页内容选择视觉结构时参考此文件。

---

## 核心映射表

### 1. 递进 / 成长

**识别信号**：时间线、阶段、从 A 到 B 的变化、"第一步...第二步..."、进化

| 视觉结构 | 适用情况 | 构图建议 |
|---------|---------|---------|
| 成长路线图 | 3-5 个阶段的演进 | IP 角色在路径上行走，路径从左下到右上 |
| 台阶/阶梯 | 明确的层级提升 | IP 角色站在不同高度的台阶上 |
| 登山路径 | 长期目标达成 | IP 角色在蜿蜒山路上，标记关键里程碑 |
| 时间轴 | 按时间排列的事件 | 水平时间线，IP 角色在关键节点上互动 |

**UI 装饰**：进度条、里程碑标记、日期标签、✓ 完成勾选

### 2. 并列 / 多项

**识别信号**：列举、"包括..."、多个同级选项、特性列表

| 视觉结构 | 适用情况 | 构图建议 |
|---------|---------|---------|
| 悬浮信息卡片 | 3-4 个并列项目 | 卡片环绕 IP 角色悬浮 |
| 气泡矩阵 | 5+ 个关键词/标签 | 大小不一的气泡围绕中心 |
| 图标网格 | 功能/特性展示 | 2x2 或 3x2 网格，每格配图标 |

**UI 装饰**：标签气泡、图标徽章、彩色标记点

### 3. 对比 / 冲突

**识别信号**：before/after、vs、"相比..."、优缺点、传统方式 vs 新方式

| 视觉结构 | 适用情况 | 构图建议 |
|---------|---------|---------|
| 左右分屏 | 两个方案对比 | 左右各一个场景，IP 角色在中间 |
| 对比窗口 | 数据对比 | 两个悬浮窗口，用颜色区分好坏 |
| 天平/跷跷板 | 权重比较 | IP 角色站在天平中央 |

**UI 装饰**：✗/✓ 标记、红绿色对比条、对比箭头

### 4. 因果 / 流程

**识别信号**："因为...所以..."、步骤、流水线、输入→输出

| 视觉结构 | 适用情况 | 构图建议 |
|---------|---------|---------|
| 箭头流 | 简单的 A→B→C | IP 角色站在流程起点，箭头贯穿 |
| 管道图 | 数据/信息流转 | 管道连接不同模块，IP 角色操作阀门 |
| 漏斗 | 筛选/转化过程 | 从宽到窄的漏斗，IP 角色在旁观察 |

**UI 装饰**：流程箭头、代码框 `input → process → output`、齿轮图标

### 5. 总分 / 层级

**识别信号**：核心+分支、"主要包含..."、组织架构、分类

| 视觉结构 | 适用情况 | 构图建议 |
|---------|---------|---------|
| 中心辐射 | 1 个核心 + 3-5 个分支 | IP 角色在中心，分支向四周延伸 |
| 树形展开 | 层级分类 | IP 角色在树根，内容向上生长 |
| 嵌套圆环 | 包含关系 | 同心圆，IP 角色在最外层指向核心 |

**UI 装饰**：连接线、层级标签、分支节点

### 6. 数据 / 成就

**识别信号**：数字、百分比、KPI、里程碑数据、"增长了..."

| 视觉结构 | 适用情况 | 构图建议 |
|---------|---------|---------|
| 大数字高亮 | 1-2 个关键数据 | 巨大数字占据画面中心，IP 角色在旁惊叹 |
| 仪表盘 | 多维度数据 | 仪表盘界面，IP 角色在前方操作 |
| 柱状/趋势图 | 增长趋势 | 简化的图表，IP 角色站在最高柱上 |

**UI 装饰**：悬浮数字气泡 "1,157"、百分比标签、趋势箭头 ↑

## 视觉重心（Focus Point）规划

每页必须有且只有一个视觉重心——用户目光首先落到的地方。

**规则**：
- 封面页：重心在 IP 角色（居中偏上）
- 内容页：重心在核心信息（Golden Sentence 或关键数据）
- 过渡页：重心在 IP 角色的动作

**Copy Space 规划**：
- 图片主体偏左 → 文案空间在右侧
- 图片主体偏右 → 文案空间在左侧
- 图片主体居中偏上 → 文案空间在下方
