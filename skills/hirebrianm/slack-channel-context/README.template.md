# Slack Channel Context Directory

Welcome to the Slack channel context directory!

## What's Here?

- `<CHANNEL_ID>.md` - Context files for specific channels (highest priority)
- `<CHANNEL_NAME>.md` - Context files named by channel name
- `README.md` - This file

## How to Use

**For AI assistants:** Context files are loaded automatically when you're in a Slack channel. No action needed!

**For humans:** To create context for a new channel:
1. Read the [skill documentation](../../skills/slack-channel-context/SKILL.md)
2. Follow the "Creating Channel Context Files" section
3. Save as `<CHANNEL_ID>.md` or `<CHANNEL_NAME>.md` in this directory
