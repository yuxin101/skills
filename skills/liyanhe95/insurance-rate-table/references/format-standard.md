# 保险费率二维表格式标准

## 1. 产品参数维度

| 维度 | 枚举值 | 说明 |
|------|--------|------|
| 产品名称 | 用户输入 | 产品全称 |
| 产品类型 | 年金险/重疾险/寿险/医疗险等 | 产品分类 |
| 领取时间 | 自55周岁、自60周岁 | 可配置 |
| 保障计划 | 计划一、计划二 | 可配置 |
| 交费期间 | 趸交、10年、20年 | 可配置 |
| 交费方式 | 一次交清、月交、年交、半年交、季交、不定期交 | 可配置 |
| 被保人年龄 | 0-70岁 | 可配置 |
| 被保人性别 | 男、女 | 可配置 |

## 2. 表头结构

### 多级表头（5行）
```
保障计划 | 计划一 | 计划一 | 计划一 | 计划一 | 计划二 | ...
交费期间 | 趸交   | 趸交   | 10年   | 10年   | 趸交   | ...
交费方式 | 一次交清 | 月交 | 月交 | 年交 | 一次交清 | ...
领取时间 | 自55周岁 | 自55周岁 | 自55周岁 | 自60周岁 | ...
被保人年龄/被保人性别 | 女 | 男 | 女 | 男 | 女 | ...
```

### 列顺序规则
按照以下顺序排列列组合：
1. 保障计划 → 2. 交费期间 → 3. 交费方式 → 4. 领取时间 → 5. 性别

## 3. 交费方式与交费期间约束

### 有效组合
| 交费方式 | 有效交费期间 |
|----------|--------------|
| 一次交清 | 趸交 |
| 月交 | 10年、20年 |
| 年交 | 10年、20年 |
| 半年交 | 10年、20年 |
| 季交 | 10年、20年 |
| 不定期交 | 10年、20年 |

### 无效组合（不生成）
- 一次交清 + 10年/20年
- 月交/年交/半年交/季交/不定期交 + 趸交

## 4. 数据结构

### 数据行
- A列：被保人年龄（整数，0-70）
- B列及以后：对应方案的女性保费、男性保费交替排列

### 示例数据行
```
年龄 | 女费率 | 男费率 | 女费率 | 男费率 | ...
0   | 34324  | 34325  | 34326  | 34327  | ...
1   | 34325  | 34326  | 34327  | 34328  | ...
```

## 5. Excel 输出规范

- 文件格式：`.xlsx`
- Sheet 名称：`费率表`
- 列宽：A列18字符，其他列10字符
- 编码：UTF-8
