# Style Packs — Drawing

Pick one pack and keep it consistent for the whole request.

## 1. Soft Storybook Watercolor

Best for: bedtime scenes, gentle animals, keepsake illustrations

Prompt fragment:
```text
soft watercolor children's book illustration, rounded shapes, calm palette, gentle lighting, clean focal point
```

## 2. Bold Cartoon

Best for: recognizability, posters, classroom walls, younger kids

Prompt fragment:
```text
bold cartoon illustration, simple shapes, strong silhouette, bright colors, uncluttered background
```

## 3. Kawaii Flat

Best for: cute icons, reward charts, sticker-like visuals

Prompt fragment:
```text
kawaii flat illustration, soft pastel palette, clean vector-like edges, friendly expression, minimal scene
```

## 4. Colored Pencil Classroom

Best for: educational visuals that should feel warm rather than glossy

Prompt fragment:
```text
colored pencil children's illustration, readable forms, light texture, school-friendly tone, simple composition
```

## 5. Paper-Cut Collage

Best for: playful learning visuals and shape-based storytelling

Prompt fragment:
```text
paper-cut collage style, layered shapes, bold separation, simple geometry, high readability
```

## 6. Thick Marker Coloring Page

Best for: preschool coloring sheets

Prompt fragment:
```text
printable coloring page, thick marker-style black outlines, large closed shapes, white background, no shading
```

## 7. Clean Line Worksheet

Best for: science, vocabulary, or themed school sheets

Prompt fragment:
```text
simple educational line illustration, clear subject labeling space, uncluttered layout, print-friendly composition
```

## 8. Friendly Fantasy

Best for: dragons, castles, and imaginative scenes that must stay non-scary

Prompt fragment:
```text
friendly fantasy children's illustration, whimsical not dark, soft shapes, expressive but calm faces, clear scene hierarchy
```

## Selection Rule

If the user only says "for kids", default to either:
- `Bold Cartoon` for color
- `Thick Marker Coloring Page` for black-and-white
