# 高考志愿填报数据来源指南

本文档汇总了进行高考志愿分析时所有真实权威的数据来源，供 AI 引导用户获取最新数据。

> **年份基准（重要）**：使用系统上下文中的 `current_time` 确定当前年份。
> - `[最近一年] = 当前年份 - 1`（已公开，必须优先使用）
> - `[前年] = 当前年份 - 2`（对比参考）
> - `[大前年] = 当前年份 - 3`（趋势参考）
> - 当前年份的高考数据尚未举行，**不得使用**。

## 一、录取分数线数据来源

### 1. 官方权威来源

#### 国家政务服务平台（最权威）
- **网址**: https://gjzwfw.www.gov.cn/col/col1567/index.html
- **数据内容**: 历年高考成绩查询、录取控制分数线、各省普通高等学校本科专业查询
- **特点**: 教育部官方，数据最权威，支持各省分线查询

#### 阳光高考（教育部指定平台）
- **网址**: https://gaokao.chsi.com.cn/
- **数据内容**: 
  - 各高校历年招生计划和录取分数线
  - 院校专业组信息（新高考省份）
  - 特殊类型招生信息（强基计划、保送生等）
- **特点**: 教育部高校招生阳光工程指定平台，所有正规高校均在此发布招生信息

#### 中国教育在线
- **网址**: https://www.eol.cn/
- **数据内容**: 
  - 各省高考一分一段表：https://www.eol.cn/e_html/gk/gkfsd/index.shtml
  - 历年分数线汇总
  - 各省高考政策解读
- **特点**: 专业教育媒体，汇总整理各省数据，便于查阅

#### 掌上高考
- **网址**: https://www.gaokao.cn/
- **数据内容**: 
  - 全国各省一分一段表：https://www.gaokao.cn/colleges/bypart
  - 历年录取分数查询
  - 志愿推荐系统
- **特点**: 官方合作平台，数据较全面

## 三、专业就业与薪资数据

### 1. 麦可思研究（MyCOS）—— 专业就业数据权威来源
- **官网**: https://www.mycos.com.cn/
- **核心报告**: 《中国本科生就业报告》（每年6月发布）
- **数据内容**:
  - 各专业就业率
  - 各专业平均薪资（毕业半年后、三年后）
  - 专业对口率
  - 就业满意度
  - 深造比例
- **访问方式**: 官网购买完整报告，摘要数据可在网上新闻稿获取

### 2. 教育部高等教育司
- **网址**: http://www.moe.gov.cn/
- **核心报告**: 《中国高等教育质量报告》（年度发布）
- **数据内容**: 全国本科毕业生就业数据、各专业布点情况

### 3. 各高校就业质量报告
- **获取方式**: 在各高校官网搜索"就业质量报告"
- **发布时间**: 每年年初（公布上一年就业情况）
- **数据内容**: 就业率、薪资中位数、就业城市分布、就业单位性质

## 七、查询数据的操作步骤

### 查询某高校在某省录取分数线
1. 访问阳光高考 https://gaokao.chsi.com.cn/
2. 点击"院校库" → 搜索目标高校
3. 点击"招生计划" → 选择省份和年份
4. 查看历年录取分数和计划人数

### 查询本省一分一段表（了解自身位次）
1. 访问本省考试院官网（见上表）
2. 或访问中国教育在线 https://www.eol.cn/e_html/gk/gkfsd/index.shtml
3. 选择省份和年份，下载一分一段表
4. 找到自己分数对应的位次（排名）

### 用位次估算可报高校
1. 获取自身位次后，在掌上高考/优志愿中筛选
2. 目标区间：找去年录取位次与自己相近（±3000名以内）的高校
3. "冲、稳、保"配比建议：冲30%、稳40%、保30%
