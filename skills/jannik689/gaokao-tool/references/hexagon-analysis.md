# 高考志愿六边形分析方法论

本文档详细说明如何对考生进行六边形能力雷达分析，用于专业和学校的匹配推荐。

---

## 一、六边形分析概述

六边形分析（Hexagon Analysis）是在霍兰德职业兴趣理论（RIASEC）基础上，结合高考成绩结构、专业需求特征，形成的六维能力-兴趣匹配模型。

### 六个维度

```
           [学科偏好]
              ↑
  [职业兴趣] ←  → [成绩分布]
      ↓              ↓
  [个性特质]      [学习能力]
              ↓
           [发展期望]
```

| 维度 | 英文 | 含义 | 评估内容 |
|------|------|------|---------|
| **学科偏好** | Subject Preference | 对各科目的偏好和擅长程度 | 各科成绩对比、偏科情况、最喜欢/最讨厌的科目 |
| **职业兴趣** | Career Interest | 霍兰德RIASEC类型 | 对不同职业领域的喜好程度 |
| **成绩分布** | Score Distribution | 绝对分数和相对位次 | 总分、各科目分数、省内位次、距线距离 |
| **个性特质** | Personality Traits | MBTI/个性倾向 | 内向/外向、感性/理性、结构化/灵活等 |
| **学习能力** | Learning Ability | 学习风格和能力结构 | 记忆型/理解型、动手能力、创造力 |
| **发展期望** | Development Goals | 对未来的规划和期望 | 就业还是深造、地域偏好、薪资期望、家庭因素 |

---

## 二、各维度详细评估

### 维度1：学科偏好（Subject Preference）

**数据收集问题：**
- 你的各科高考（或模拟考）成绩分别是多少？
- 你最喜欢哪门课？为什么？
- 你最讨厌哪门课？遇到困难时如何应对？
- 选科情况（新高考）：选了哪三门选考科目？为什么这样选？

**偏向分析矩阵：**

| 强势科目 | 擅长领域 | 匹配专业方向 |
|---------|---------|------------|
| 数学+物理 | 理工逻辑 | 计算机、电子、机械、物理、数学 |
| 数学+化学 | 化学工程 | 化工、材料、药学、食品、环境 |
| 数学+生物 | 生命科学 | 生物、医学、农学、生态 |
| 语文+历史 | 人文社科 | 历史、中文、哲学、新闻 |
| 英语+历史 | 国际人文 | 外语、国际关系、翻译、外贸 |
| 政治+历史 | 法律行政 | 法学、行政管理、政治学 |
| 地理+生物 | 地球生态 | 地理、测绘、城乡规划、环境 |
| 艺术类 | 创意设计 | 美术、设计、音乐、影视传媒 |

---

### 维度2：职业兴趣（Career Interest）

基于霍兰德 RIASEC 模型，六种兴趣类型：

| 类型 | 代码 | 特征 | 适合职业/专业 |
|------|------|------|-------------|
| **现实型** | R（Realistic） | 喜欢动手、机械操作，务实 | 工程、机械、建筑、农林 |
| **研究型** | I（Investigative） | 喜欢探索、思考、研究 | 自然科学、医学研究、计算机科学 |
| **艺术型** | A（Artistic） | 创造力强，喜欢表达、审美 | 设计、文学、音乐、影视 |
| **社会型** | S（Social） | 喜欢助人、沟通、教学 | 教育、社工、医护、HR |
| **企业型** | E（Enterprising） | 有领导欲，善于说服、经营 | 工商管理、市场、法律 |
| **常规型** | C（Conventional） | 喜欢有序、精确、规范操作 | 会计、统计、行政、档案 |

**简易测评问卷（可对考生提问）：**

以下各项让学生打1-5分（1=完全不喜欢，5=非常喜欢）：
1. 动手组装、修理设备 → R
2. 分析问题原因、做实验 → I
3. 画画、写作、创作音乐 → A
4. 帮助他人解决困难 → S
5. 组织活动、管理他人 → E
6. 整理数据、记录分类 → C

---

### 维度3：成绩分布（Score Distribution）

**关键指标：**
- **总分**：与当省当年批次线的距离
- **位次**：在全省的排名（最关键！）
- **各科分布**：是否偏科、强弱科差值

**位次区间参考（以单省约30万理科考生为例）：**

| 位次区间 | 可冲目标 | 可稳目标 | 可保目标 |
|---------|---------|---------|---------|
| 前500名 | 清北 | 其他C9 | 知名985 |
| 500-3000名 | C9非热门 | 985主流 | 强势211 |
| 3000-10000名 | 知名985 | 强985/顶级211 | 一般211 |
| 10000-30000名 | 211热门 | 211主流 | 双非强校 |
| 30000-60000名 | 211末尾 | 双非强校 | 普通本科 |
| 60000名以后 | 双非强校 | 普通本科 | 专科 |

*以上为示意，不同省份总人数不同，需按比例调整*

---

### 维度4：个性特质（Personality Traits）

**简化版 MBTI 快速判断：**

| 维度 | 选项A | 选项B |
|------|------|------|
| E/I（精力来源） | 喜欢社交，在集体中活跃 | 喜欢独处，需要安静思考 |
| S/N（感知方式） | 注重现实，关注细节 | 喜欢想象，关注可能性 |
| T/F（决策方式） | 理性分析，看重逻辑 | 感性，看重人的感受 |
| J/P（生活方式） | 喜欢计划，有条理 | 随性灵活，保持开放 |

**个性与专业匹配建议：**

| 个性类型 | 代表MBTI | 推荐专业方向 | 不建议方向 |
|---------|---------|------------|---------|
| 外向+理性 | ENTJ/ESTJ | 工商管理、法律、金融 | 独立研究型 |
| 内向+理性 | INTJ/ISTJ | 计算机、数学、物理、哲学 | 大量社交销售 |
| 外向+感性 | ENFJ/ESFJ | 教育、心理、医护、公关 | 独立技术研发 |
| 内向+感性 | INFP/ISFP | 文学、艺术、心理、设计 | 高压管理类 |
| 探索型 | INTP/ENTP | 科研、计算机、哲学、金融 | 高度规范化岗位 |
| 实践型 | ISTP/ESTP | 机械、建筑、运动医学 | 纯文职行政 |

---

### 维度5：学习能力（Learning Ability）

**评估问题：**
- 你更偏向背记（记忆型）还是推导理解（理解型）？
- 你喜欢动手实验还是理论推导？
- 你自主学习能力强吗（自律程度）？
- 你的英语水平如何（影响部分专业深造）？

**学习能力与专业适配：**

| 学习特征 | 推荐方向 |
|---------|---------|
| 强记忆+系统整理 | 医学、法学、历史、行政 |
| 强逻辑+喜推导 | 数学、物理、计算机、哲学 |
| 动手实践型 | 机械、电子、临床医学、建筑 |
| 创意发散型 | 设计、广告、新闻、创业相关 |
| 语言敏感型 | 英语、翻译、中文、传播 |
| 自律+自主 | 线上/远程相关、科研型专业 |

---

### 维度6：发展期望（Development Goals）

**评估问题：**
- 毕业后想就业还是继续深造（考研/出国）？
- 想留在哪个城市或地区？
- 对薪资的期望和家庭的经济需求是什么？
- 家人对专业选择有哪些意见或要求？
- 是否有特定的职业梦想？

**期望类型与策略匹配：**

| 期望类型 | 选校策略 |
|---------|---------|
| 尽快就业、高薪 | 优先选好就业的专业（CS、电子、金融），学校排名其次 |
| 读研深造 | 优先选学校排名，专业可适当灵活 |
| 出国深造 | 优先一线城市好学校，英语能力重要 |
| 回家乡就业 | 优先当地或近省知名高校 |
| 考公务员/教师编 | 优先本省内高校（部分岗位限户籍院校） |
| 创业 | 优先一线城市高校，校友资源丰富 |

## 三、六边形雷达图生成方法

对考生六个维度分别打分（1-10分）：

```
维度 → 得分方式
学科偏好: 根据各科成绩偏差计算（最高分科目占优10分，均衡6-7分）
职业兴趣: 根据RIASEC测评结果的最高类型得分换算（最高10分）
成绩分布: 按位次区间换算（前500=10，前5000=8，前20000=6，以此类推）
个性特质: 根据MBTI结果与目标专业匹配度打分（高度匹配10，低度3）
学习能力: 综合自评（强=9-10，中=6-8，弱=3-5）
发展期望: 期望清晰度和可行性（清晰且可行=9-10，模糊=5-6）
```

用文字描述雷达图时，按如下格式：
```
【六边形分析雷达图】
📚 学科偏好:  ████████░░ 8/10 - 理科偏强，数物优势明显
🎯 职业兴趣:  ███████░░░ 7/10 - 研究型(I)为主，兴趣在科学探索
📊 成绩分布:  ████████░░ 8/10 - 省内位次约5000名，竞争力较强
🧠 个性特质:  ██████░░░░ 6/10 - 内向理性(INTJ)，适合技术研究
💪 学习能力:  █████████░ 9/10 - 理解能力强，自主学习意识好
🌟 发展期望:  ███████░░░ 7/10 - 期望考研，目标一线城市
```

## 四、专业匹配综合评分

对每个候选专业，按以下标准进行匹配评分：

| 评分项 | 权重 | 说明 |
|-------|------|------|
| 学科基础匹配 | 30% | 该专业所需学科与考生强科的匹配度 |
| 职业兴趣匹配 | 25% | RIASEC类型与专业对应职业的匹配度 |
| 个性特质匹配 | 15% | 人格特征与专业学习风格的匹配度 |
| 就业发展匹配 | 20% | 就业率、薪资与考生期望的匹配度 |
| 录取可能性 | 10% | 基于成绩位次的录取概率评估 |

**综合分 = 各项加权之和（满分100）**

- ≥85分：强烈推荐（与考生高度契合）
- 70-84分：推荐（基本契合，有发展空间）
- 55-69分：可以考虑（部分匹配，需有心理准备）
- <55分：不建议（明显不匹配）

## 五、学校选择策略

### "冲稳保"三档策略

| 类型 | 录取概率 | 分配比例 | 位次参考 |
|------|---------|---------|---------|
| **冲** | 30-40% | 30% | 比自身位次领先10-30% |
| **稳** | 60-70% | 40% | 比自身位次相差±5% |
| **保** | 85%以上 | 30% | 比自身位次落后15-30% |

### 平行志愿填报注意事项

1. **志愿顺序很重要**：从高到低排列，第一个满足条件即录取
2. **专业服从调剂**：建议在可接受范围内服从，避免落档
3. **查看近3年录取数据**：不能只看一年，需看趋势
4. **关注大小年现象**：某校上年爆热可能今年降温
5. **避开"陷阱分数线"**：少数学校报名突然增多导致线上涨

### 学校地域建议

| 地域 | 优势 | 适合人群 |
|------|------|---------|
| 北京 | 资源集中、机会多、校友资源强 | 想进大企业、读研出国 |
| 上海 | 金融经济中心、国际化程度高 | 金融、外贸、创业 |
| 深圳/广州 | 互联网、科技、制造业发达 | 计算机、工程类 |
| 成都/武汉/西安 | 性价比高、就业也不差 | 预算有限、不排斥中西部城市 |
| 本省 | 就近就业资源、家庭负担小 | 倾向回乡就业、家庭经济一般 |
