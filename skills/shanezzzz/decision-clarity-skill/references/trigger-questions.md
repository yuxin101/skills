# Trigger Questions

Use this file to translate vague user prompts into sharper analysis frames.

## Surface question
"Should I build an app?"

## Sharper frame
- What user outcome must be delivered?
- Is an app necessary, or just one candidate delivery mechanism?
- What is the smallest format that could prove adoption?

---

## Surface question
"How do I make this more professional?"

## Sharper frame
- Professional in what sense: trust, clarity, conversion, authority, or price justification?
- Which visible or structural elements actually create that effect?
- Which polish steps are inherited aesthetics rather than outcome drivers?

---

## Surface question
"Why is this so complicated?"

## Sharper frame
- Which part is actually complex?
- Which complexity is real?
- Which complexity is inherited from process, tool, or expectation?
- What would a simpler but still adequate version look like?

---

## Surface question
"What am I missing?"

## Sharper frame
- What assumption is most likely hidden?
- What term is still vague?
- What evidence is missing?
- What simpler interpretation already fits the facts?

---

## Surface question
"Which option makes the most sense?"

## Sharper frame
- What decision are we actually making?
- What criteria matter most?
- Which option requires fewer assumptions while still achieving the goal?
- What is the best current move, not the perfect move?

---

## Surface question
"How do I make better content?"

## Sharper frame
- Better in what dimension?
- What user outcome matters most?
- Which production steps improve that outcome?
- What complexity can be removed without losing value?
