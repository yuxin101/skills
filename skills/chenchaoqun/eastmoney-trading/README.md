# 东方财富证券交易技能

📈 完整的证券交易解决方案 — 支持登录、持仓、买入、卖出、撤单、委托查询、资金查询

## 快速开始

### 1. 安装依赖

```bash
pip install -r scripts/requirements.txt
```

### 2. 配置环境变量

```bash
export EASTMONEY_ACCOUNT=your_account
export EASTMONEY_PASSWORD=your_password
```

### 3. 运行命令

```bash
# 登录 + 持仓查询
python3 scripts/eastmoney_trading.py login

# 买入
python3 scripts/eastmoney_trading.py buy --stock-code 600519 --price 1850 --quantity 100

# 卖出
python3 scripts/eastmoney_trading.py sell --stock-code 600519 --price 1900 --quantity 100

# 撤单
python3 scripts/eastmoney_trading.py cancel --order-id 12345678

# 委托查询
python3 scripts/eastmoney_trading.py orders

# 资金查询
python3 scripts/eastmoney_trading.py balance
```

## 功能特性

✅ **CDP 连接模式** — 复用浏览器会话和 Cookie
✅ **验证码识别** — Tesseract OCR + 通义千问 LLM 双引擎
✅ **持仓查询** — 获取持仓明细、成本、盈亏
✅ **买入/卖出** — 支持限价/市价委托
✅ **撤单操作** — 按委托编号/股票/全部撤单
✅ **委托查询** — 当日/历史委托记录
✅ **资金查询** — 总资产、可用资金、盈亏、仓位
✅ **安全措施** — 密码通过环境变量传递，交易二次确认
✅ **日志截图** — 按天切分日志，关键步骤自动截图

## 注意事项

⚠️ **安全警告**：
- 密码通过环境变量传递，不存储在脚本中
- 自动化登录和交易可能触发风控
- 交易操作涉及真实资金，务必谨慎
- **默认启用二次确认**

⚠️ **交易时间**：
- A 股交易时间：9:30-11:30, 13:00-15:00
- 周末和法定节假日休市

⚠️ **最佳实践**：
- 本地运行，不要部署到服务器
- 定期更换密码
- 监控账号异常
- 定期清理日志文件夹
