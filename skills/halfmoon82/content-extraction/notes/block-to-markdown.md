# Block → Markdown Mapping

## 目标

把 Feishu / 飞书知识库里的结构化 blocks 转成尽可能干净、可读的 Markdown。

## 映射规则

### 标题
- H1 → `#`
- H2 → `##`
- H3 → `###`
- H4+ → 继续降级成更多 `#`

### 段落
- 普通文本保持原句
- 多段内容之间空一行

### 列表
- 无序列表 → `- item`
- 有序列表 → `1. item`
- 嵌套列表 → 缩进 2 空格

### 引用
- 引用块 → `> text`
- 多段引用保持连续引用格式

### 代码块
- 代码块 → fenced code block
- 如果有语言信息，写入 ```lang
- 没有语言就只写 ```

### 待办
- 待办项 → `- [ ] item`
- 完成项 → `- [x] item`

### 表格
- 统一转 Markdown 表格
- 如果表格过复杂，保留原始行列顺序，尽量别丢字段

### 图片
- 图片 → `![caption](url)`
- 没有 caption 时只保留链接

### 公式
- 尽量保持原公式文本
- 如果渲染失败，保留原始表达式

## 失败策略

- 不能映射的 block 类型，保留原始文本并标注类型
- 不要静默丢弃结构信息
- 不要把未知 block 强行编造成普通段落
