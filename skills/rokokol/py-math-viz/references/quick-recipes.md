# Quick recipes (index)

This folder intentionally splits recipes by style.

- `recipes-spec-json.md` — JSON specs for `plot_from_spec.py` (CSV → hist/scatter/line/bar/heatmap)
- `recipes-seaborn-snippets.md` — one-off richer stats plots (KDE/violin/corr/pairplot)
- `recipes-images.md` — tiling plots into one image

See also:
- `spec.md` — the JSON schema reference
