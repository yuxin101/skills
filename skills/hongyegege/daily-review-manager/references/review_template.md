# 复盘模板参考

## 标准复盘结构

```markdown
## YYYY-MM-DD

### 记录时间
HH:MM

### 记录内容
[用户口述原文]

### 主题
[一句话主题概括]

### 内容亮点
1. [亮点1]
2. [亮点2]
3. [亮点3]

### 心情
[心情描述 + emoji]
```

## 简化版复盘

```markdown
## YYYY-MM-DD

- **感悟**：[一句话感悟]
- **亮点**：[1-2个亮点]
- **心情**：[emoji]
```

## 周报模板

```markdown
# 周报汇总 (Week X)

## 本周概览
- 复盘次数：X 次
- 平均时长：X 分钟

## 本周亮点 TOP 3
1. [亮点1]
2. [亮点2]
3. [亮点3]

## 主题分布
| 主题 | 次数 |
|------|------|
| 工作 | X |
| 学习 | X |
| 生活 | X |

## 每日摘要
- 周一：...
- 周二：...

## 下周计划
- [计划1]
- [计划2]
```

## 月报模板

```markdown
# YYYY年MM月 月报

## 本月概览
- 复盘次数：X 次
- 参与天数：X 天
- 最常见主题：XXX

## 本月成就
- [成就1]
- [成就2]
- [成就3]

## 成长分析
- 进步领域：...
- 待改进：...

## 本月金句
> [本月最触动的句子]

## 下月目标
- [目标1]
- [目标2]
```
