# Home Fix-It
**Snap a photo. Get the fix. Save the $150 service call.**

Home Fix-It is your AI-powered home repair assistant. Take a picture of that leaky faucet, cracked drywall, or mysterious appliance error code. We’ll diagnose the problem, tell you exactly what parts to buy, and walk you through the fix—safely and simply.

### 🛡️ Security & Privacy
- **Codex Security Verified** badge
- **Privacy-First:** Your photos and home data stay local. We don't phone home.
- **LLM Agnostic:** Bring your own AI. No hidden API subscriptions.

### Requirements
- NormieClaw environment
- Vision-capable LLM (OpenAI, Anthropic, or Gemini)

### Installation
1. Purchase and download the Home Fix-It package ($4.99).
2. Install via NormieClaw skill manager.
3. Run the setup via `SETUP-PROMPT.md`.

### ⚠️ Safety Disclaimer
Safety is paramount. Home Fix-It provides guidance, but you are responsible for your own safety. The system is programmed to explicitly advise calling licensed professionals for high-risk jobs including gas lines, main electrical panels, and structural modifications. Always turn off power at the breaker and water at the main valve before beginning repairs.
