# Time Awareness

A lightweight skill that adds time and date awareness to your agent. Knows the current time, date, timezone, and can reason about temporal concepts.

## Installation

Via ClawMart:
```bash
clawmart install time-awareness
```

Or manual:
```bash
git clone https://github.com/proceedinghumbly/time-awareness.git
```
