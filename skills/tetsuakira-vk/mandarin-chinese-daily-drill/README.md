# Mandarin Daily Drill — OpenClaw Skill

**A complete, personalised Mandarin practice session every time you ask.**

Level-appropriate, fresh content every session. No repetition, no subscriptions, no app to open.

---

## What it does

Tell it your level. Get back a full practice session:

- **Vocabulary** — 10 words with examples, translations, and memory tips
- **Grammar pattern** — one pattern explained clearly with examples
- **Character/script focus** — language-specific writing system practice
- **Reading passage** — level-appropriate text with full translation
- **Speaking prompts** — realistic dialogues and practice scenarios
- **Quick quiz** — 5 questions with answers to test retention
- **Cultural note** — Chinese customs, character etymology, regional notes

Aligned to **HSK (Level 1–6)**.

---

## Who it's for

- Mandarin learners at any level from complete beginner to advanced
- People studying for HSK (Level 1–6) exams
- Anyone who wants structured daily practice without a fixed app or schedule
- Language enthusiasts who want fresh content on demand

---

## Installation

```bash
npx clawhub@latest install tetsuakira-vk/mandarin-daily-drill
```

---

## Usage

```
Use mandarin-daily-drill
```

It will ask your level, then generate a full session. Or specify directly:

```
Use mandarin-daily-drill at [level]
```

---

## Features

- Fresh vocabulary, grammar, and content every session — no repeats
- Dedicated tone drill section every session
- Memory tips and mnemonics built in
- Ask for another session and get completely new content
- If you keep forgetting something, tell it — it generates a custom memory device

---

## Requirements

- OpenClaw installed and running
- Any supported LLM (Claude recommended for best language accuracy)
- No API keys or external services required

---

## Tips

- Set your level in OpenClaw memory so it never asks: "Remember: my mandarin level is [level]"
- Run a session daily for 15–20 minutes for best results
- Use the speaking prompts section out loud — not just reading
- Save the quiz answers and review them the next day before starting a new session
