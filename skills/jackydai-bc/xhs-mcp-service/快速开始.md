# 小红书 MCP 服务 - 快速参考卡片

## 🚀 快速启动

```bash
# 在你的项目目录下
npm start
```

**服务地址**: http://localhost:18060

## 📋 13个工具速查

### 登录管理
```javascript
check_login_status()          // 检查登录状态
get_login_qrcode()            // 获取登录二维码
delete_cookies()              // 删除登录状态
```

### 内容获取
```javascript
list_feeds()                  // 获取首页推荐
search_feeds(keyword, filters) // 搜索笔记
get_feed_detail(id, token)    // 获取笔记详情
user_profile(user_id, token)  // 获取用户主页
```

### 互动操作
```javascript
like_feed(id, token, unlike?)         // 点赞/取消
favorite_feed(id, token, unfav?)      // 收藏/取消
post_comment_to_feed(id, token, text) // 发表评论
reply_comment_in_feed(id, token, text, cid, uid) // 回复评论
```

### 内容发布
```javascript
publish_content({title, content, images, tags?})     // 发布图文
publish_with_video({title, content, video, tags?})   // 发布视频
```

## 💡 常用示例

### 搜索并点赞
```javascript
// 在项目目录下创建脚本
import { searchFeeds, likeFeed } from './src/xhs-tools.js';

const results = await searchFeeds('美食');
await likeFeed(results.feeds[0].id, results.feeds[0].xsec_token);
```

### 发布图文
```javascript
import { publishContent } from './src/xhs-tools.js';

await publishContent({
  title: '标题',
  content: '正文',
  images: ['./photo.jpg'],  // 相对路径
  tags: ['美食']
});
```

## ⚠️ 注意事项

- ⏱️ 操作间隔 > 2秒
- 📝 标题 ≤ 20字
- 📄 正文 ≤ 1000字
- 📊 发布 ≤ 50篇/天
- 🔄 每周重新登录

## 🔗 默认端点

- 健康检查: `http://localhost:18060/health`
- MCP服务: `http://localhost:18060/mcp`

**注意**: 端口可通过环境变量 `XHS_PORT` 修改

## 📊 性能参考

- 检查登录: 14秒
- 获取推荐: 10.5秒
- 搜索笔记: 5.6秒
- 点赞操作: 4.5-6秒

## 🔧 环境变量

```bash
# Linux/Mac
export XHS_PORT=18060
export XHS_HOST=0.0.0.0
export XHS_PROXY=http://proxy:port

# Windows PowerShell
$env:XHS_PORT=18060
$env:XPS_HOST="0.0.0.0"
$env:XHS_PROXY="http://proxy:port"

# Windows CMD
set XHS_PORT=18060
set XHS_HOST=0.0.0.0
set XHS_PROXY=http://proxy:port
```
