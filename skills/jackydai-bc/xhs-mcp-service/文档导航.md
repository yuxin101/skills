# 小红书 MCP 服务

基于 Model Context Protocol (MCP) 的小红书自动化工具，提供完整的操作能力。

## 🚀 快速开始

```bash
# 1. 克隆项目（使用你的实际地址）
git clone <your-repo-url> xhs-mcp-service
cd xhs-mcp-service

# 2. 安装依赖
npm install

# 3. 登录（首次使用）
npm run login

# 4. 启动服务
npm start
```

服务地址: http://localhost:18060

## 🛠️ 功能列表

✅ **13个完整工具**

- 登录管理: 检查状态、二维码登录、删除cookies
- 内容获取: 首页推荐、搜索笔记、笔记详情、用户主页
- 互动操作: 点赞、收藏、评论、回复
- 内容发布: 发布图文、发布视频

详细列表见 [README.md](./README.md)

## ⚙️ 配置

### 环境变量

```bash
# Linux/Mac
export XHS_PORT=18060
export XHS_HOST=0.0.0.0

# Windows PowerShell
$env:XHS_PORT=18060
$env:XHS_HOST="0.0.0.0"

# Windows CMD
set XHS_PORT=18060
set XHS_HOST=0.0.0.0
```

- 操作间隔 > 2秒
