# 小红书 MCP 服务 - 快速参考卡片

```bash
cd D:\work\xiaohongshu\xhs-mcp-service

```

## 📋 13个工具速查

### 登录管理
```javascript
check_login_status()          // 检查登录状态
get_login_qrcode()            // 获取登录二维码
delete_cookies()              // 删除登录状态
```

### 内容获取
```javascript
list_feeds()                  // 获取首页推荐
search_feeds(keyword, filters) // 搜索笔记
get_feed_detail(id, token)    // 获取笔记详情
user_profile(user_id, token)  // 获取用户主页
```

### 互动操作
```javascript
like_feed(id, token, unlike?)         // 点赞/取消
favorite_feed(id, token, unfav?)      // 收藏/取消
post_comment_to_feed(id, token, text) // 发表评论
reply_comment_in_feed(id, token, text, cid, uid) // 回复评论
```

### 内容发布
```javascript
publish_content({title, content, images, tags?})     // 发布图文
publish_with_video({title, content, video, tags?})   // 发布视频
```

## 💡 常用示例

### 搜索并点赞
```javascript
import { searchFeeds, likeFeed } from './src/xhs-tools.js';

const results = await searchFeeds('美食');
await likeFeed(results.feeds[0].id, results.feeds[0].xsec_token);
```

### 发布图文
```javascript
import { publishContent } from './src/xhs-tools.js';

await publishContent({
  title: '标题',
  content: '正文',
  images: ['./photo.jpg'],
  tags: ['美食']
});
```

## ⚠️ 注意事项

- ⏱️ 操作间隔 > 2秒
- 📝 标题 ≤ 20字
- 📄 正文 ≤ 1000字
- 📊 发布 ≤ 50篇/天
- 🔄 每周重新登录
