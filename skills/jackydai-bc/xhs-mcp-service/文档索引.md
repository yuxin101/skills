## 🔧 技术文档

### 项目结构
```
xhs-mcp-service/
├── src/
│   ├── index.js          # MCP服务主程序
│   ├── xhs-tools.js      # 13个工具实现
│   ├── browser.js        # 浏览器管理
│   └── login.js          # 登录工具
├── data/
│   └── cookies.json      # Cookies存储
├── examples/             # 示例代码
│   ├── 基础示例.mjs
│   ├── 互动示例.mjs
│   └── 发布图文示例.mjs
├── README-使用说明.md    # 完整文档
├── 快速参考.md           # 速查卡片
└── package.json
```

### 关键文件

| 文件 | 说明 |
|------|------|
| `src/index.js` | MCP服务入口，定义13个工具 |
| `src/xhs-tools.js` | 工具实现逻辑 |
| `src/browser.js` | Puppeteer浏览器管理 |
| `data/cookies.json` | 登录状态存储 |
