# 签名变量参考

## 用户变量（在 sign-off.json 中配置）

| 变量 | 说明 | 示例 | 必填 |
|------|------|------|------|
| `{name}` | AI 的名字 | 小橘 | 是 |
| `{location}` | 地点 | 杭州 | 是 |
| `{emoji}` | 自定义图标 | 🏮 | 否 |
| `{seal}` | 印章文字 | [橘印] | 否 |
| `{mood}` | 心情 | 慵懒/认真/调皮 | 否 |

## 自动变量（AI 自行判断）

| 变量 | 说明 | 示例值 |
|------|------|--------|
| `{season}` | 当前季节 | 春/夏/秋/冬 |
| `{weather}` | 当地天气 | 晴/阴/雨/雪 |
| `{time}` | 当前时段 | 清晨/上午/午后/傍晚/深夜 |
| `{greeting}` | 时段问候 | 早安/午安/晚安/夜安 |
| `{dayOfWeek}` | 星期 | 周一~周日 |
| `{zodiac}` | 生肖年份 | 蛇年/马年 |

## 上下文模式

| 模式 | 场景 | 风格建议 |
|------|------|----------|
| `work` | 技术讨论、执行命令、代码审查 | 简洁干练 |
| `casual` | 闲聊、问答、创意讨论 | 轻松活泼 |
| `auto` | AI 自行判断 | 混合 |

## 自定义变量

用户可以在 variables 字段中添加任意自定义变量：

```json
{
  "variables": {
    "pet": "🐱",
    "signature": "永远爱你的"
  },
  "template": "{signature}{name} {pet}"
}
```

输出：永远爱你的小橘 🐱
