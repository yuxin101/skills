# 利润核算参考

## 核心公式

### 单品毛利计算

```
单品毛利 = (平台售价 × 汇率 - 进货成本合计) / (平台售价 × 汇率) × 100%

进货成本合计 = 进货价(FOB) + 头程运费 + 平台佣金 + 推广费分摊 + 损耗
```

### 全链路成本拆解

```
┌─────────────────────────────────────────────┐
│ 平台售价 ($89.99 × 7.2汇率 = ¥647.93)       │
├─────────────────────────────────────────────┤
│ 进货价（1688拿货）           ¥180.00        │
│ 头程运费（分摊到单）          ¥25.00        │
│ 平台佣金（Amazon 15%）       ¥97.19        │
│ 推广费分摊（ACOS 20%）       ¥129.59        │
│ 尾程配送（FBA/海外仓）        ¥45.00        │
│ 损耗/退货（约2%）             ¥12.96        │
│ 汇损（+1%）                  ¥6.48         │
├─────────────────────────────────────────────┤
│ 合计成本                   ¥495.22          │
├─────────────────────────────────────────────┤
│ 单品毛利                    23.58%          │
│ 单品利润                    ¥152.71         │
└─────────────────────────────────────────────┘
```

## 头程运费计算

### 海运（最常见）

```
头程运费/件 = (件重kg 或 体积重kg) × 单价/kg

体积重计算（长cm × 宽cm × 高cm / 6000）
取实际重与体积重较大者计费

示例：
产品尺寸：30cm × 20cm × 15cm
件数：100个（每箱装20个，共5箱）
每箱重量：5kg

体积重 = (30×20×15/6000) = 1.5kg < 5kg实重 → 按5kg计
头程（海运普船 $4/kg）= 5kg × $4 = $20/箱
每件头程 = $20 / 20件 = $1/件
折合¥7.2汇率 = ¥7.2/件
```

### 空运/快递（适合紧急补货或样品）

```
空运：$15-$25/kg（是海运的4-6倍）
快递（DHL/FedEx）：$30-$50/kg
```

## 损耗率参考

| 产品类型 | 损耗率 | 原因 |
|----------|--------|------|
| 普通轻货 | 1%-2% | 正常破损 |
| 电子产品 | 2%-3% | 屏幕/电池问题 |
| 液体/膏体 | 3%-5% | 漏液 |
| 玻璃/陶瓷 | 5%-10% | 易碎 |
| 服装/布艺 | 1%-3% | 尺码/色差退货 |

## 盈亏平衡点计算

```
盈亏平衡销量 = 前期固定投入 / (单价利润 - 变动成本差异)

前期固定投入：
- 开模费：¥XX（一次性）
- 样品费：¥XX（一次性）
- 认证费：¥XX（如CE ¥1.5万）
```

## 汇率风险管理

| 风险 | 应对策略 |
|------|----------|
| 汇率下跌 | 报价时预留3-5%缓冲；锁汇 |
| 汇率上涨 | 可适当降价抢市场（成本优势增加） |
| 波动剧烈 | 分批结汇；用USD报价给供应商 |

## 毛利健康区间

| 平台/模式 | 合理毛利区间 | 说明 |
|-----------|-------------|------|
| Amazon FBA 精品 | 25%-40% | 含FBA费用，竞争激烈需控成本 |
| TikTok Shop | 30%-50% | 流量红利期，佣金低 |
| 独立站 | 40%-60% | 流量成本高，但溢价高 |
| Temu 全托管 | 10%-20% | 平台压价厉害，量大自然有利润 |
