# 选品分析参考

## 核心选品逻辑

### 避开红海产品
- 月销量 > 5000 且品牌集中度高 → 红海，慎入
- 季节性过于集中 → 库存风险高
- 专利壁垒严重（如大牌同款）→ 法律风险

### 潜力新品特征
- 具有"改良点"：功能优化 / 材质升级 / 外观迭代
- 切入细分场景：如"宠物喂食器" → "带摄像头+语音互动款"
- 1688 上同款销量增长趋势向上，但竞争尚未白热化

## 产品基因解析要素

```
材质：ABS / 铝合金 / 不锈钢 / 硅胶 / 尼龙 ...
克重：单件重量（影响运费）
尺寸：长×宽×高（影响装箱率）
工艺：注塑 / 压铸 / CNC / 阳极氧化 / 喷涂 / 丝印 ...
认证：CE / FCC / FDA / RoHS / BSCI / REACH / CPSIA ...
电压/电池：110V/220V / 锂电池类型（影响跨境合规）
```

## 改良点切入方向

| 方向 | 示例 | 1688 切入价值 |
|------|------|---------------|
| 功能升级 | 普通款 → 带APP控制款 | 溢价空间 ¥20-50 |
| 材质升级 | 塑料 → 铝合金/不锈钢 | 溢价空间 ¥15-30 |
| 外观迭代 | 单色 → 马卡龙/莫兰迪色系 | 差异化竞争力 |
| 配件包升级 | 标配配件 → 豪华配件套装 | 提升转化率 |
| 包装升级 | 简包 → 彩盒+说明书 | 提升开箱体验 |
| 定制化 | 支持Logo丝印 / 品牌定制 | 建立品牌壁垒 |

## 典型选品场景

### 场景1：用户给竞品链接
→ 分析竞品热销变体 → 提炼产品基因 → 在1688搜索相似款 → 筛选工厂

### 场景2：用户给图片
→ 识别产品品类/材质/工艺 → 反向搜索1688同款相似款 → 寻找具备类似工艺的工厂

### 场景3：用户给关键词+市场描述
→ 搜索对应品类 → 叠加跨境标签过滤 → 分析价格带分布 → 推荐切入价格带

## 选品结论输出要素

每次选品分析必须包含：
1. 当前热点描述（附数据支撑）
2. 产品基因解析
3. 1688 价格区间参考
4. 竞品平台售价对比
5. 推荐切入角度（差异化卖点）
6. ⚠️ 风险提示（如有专利/认证风险）
