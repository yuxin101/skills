# 工厂筛选标准参考

## 筛选硬性门槛

| 指标 | 门槛 | 说明 |
|------|------|------|
| 诚信通年限 | ≥ 3年 | 低于3年视为新店，风险较高 |
| 诚信通认证 | ✅ 已认证 | 未认证一律排除 |
| 工厂标签 | 实力工厂 / 超级工厂 | 无标签需额外审核 |
| 旺旺响应率 | ≥ 80% | 低于此值沟通成本高 |
| 发货保障 | 48小时内发货 | 库存深度关键指标 |

## 工厂类型识别

### 源头工厂特征
- 营业执照显示：制造商/生产商
- 有独立生产线（看工厂图）
- 有研发/开模能力
- 产品线垂直，不做泛货分销
- 联系方式中有"厂"字

### 中间商特征（需排除）
- 产品种类极多（泛货）
- 无工厂图或工厂图模糊
- 诚信通年限短
- 主要做分销代理
- 无法提供产品实拍视频

## 核心评估维度

### 1. 经营稳定性（必看）
```
诚信通年限：≥ 3年 ✅
近90天成交额：越高越好（>50万更稳）
复购率：≥ 15% 表示客户认可
```
> 1688显示的"复购率" = 回头客订单金额 / 总成交金额

### 2. 供货能力（核心）
```
近30天发货率：需 ≥ 95%（看48小时发货承诺）
库存深度：是否有现货（可一件代发）
产线数量：多产线 = 产能充足
月产能：是否满足你的订单量
```

### 3. 跨境资质（出口必看）
```
CE（欧盟安全认证）
FCC（美国联邦通信委员会）
FDA（美国食品药品认证）
BSCI（商业社会标准认证）
RoHS（欧盟环保指令）
REACH（欧盟化学品注册）
CPSIA（美国儿童产品安全）
UL（美国安全认证）
```
> 根据目标市场选择对应认证，缺失则需要求补充或排除

### 4. 服务能力
```
拿样服务：是否支持
样品费用：是否可抵扣货款
定制能力：Logo丝印/彩盒定制/模具开发
一件代发：是否支持
图数据包：是否提供白底图/视频
```

### 5. 价格结构
```
起订量（MOQ）：是否与你的需求匹配
阶梯报价：数量区间 vs 单价对应关系
含税/不含税：务必确认
报价有效期：原材料波动时需约定

常见阶梯结构：
- 100-499个：¥XX/个
- 500-999个：¥XX/个（降幅约5-10%）
- 1000+个：¥XX/个（降幅约10-15%）
```

## ⚠️ 风险预警清单

### 必排除情况
1. 诚信通年限 < 2年
2. 无工厂标签 + 无工厂图
3. 旺旺不回复或响应极慢（>24h）
4. 报价明显低于市场价（警惕：以次充好/收款不发货）
5. 无法提供基本认证（目标市场所需）

### 需警惕情况
1. 同一个产品多个店铺在卖（可能为分销商）
2. 交货期承诺异常短（虚报产能）
3. 不接受任何形式担保（支付宝/平台交易）
4. 产品图片盗用知名品牌（侵权风险）
