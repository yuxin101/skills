# 询盘话术模板库

## 核心原则

**专业且强势**：模拟大客户姿态，展现长期订单能力

**话术结构**：
1. 破冰：简短自我介绍 + 了解渠道
2. 展示实力：说明自己的业务规模和目标市场
3. 具体需求：明确产品/定制/数量需求
4. 压价试探：用量换价，表达合作意向
5. 行动号召：给出明确的下单时间节点

**禁忌**：
- ❌ "请问这个多少钱？"（暴露零散买家身份）
- ❌ "可以便宜点吗？"（无筹码压价）
- ❌ 长篇大论套近乎（工厂老板没时间看）

## 场景化话术模板

### 模板A：拿样测试

> 适用场景：首次接触，想拿样品测试质量

```
老板，您好！
我们是[公司名/团队名]，专注[目标市场，如：北美]跨境电商，
目前在[TikTok Shop / Amazon / 独立站]运营，月销[XX万美金]。
看到贵厂的[产品名]，觉得[卖点，如：轻量化设计、性价比]很符合我们的需求。
想先拿几个样品测试，请问：
1. 样品费多少？能否在后续订单里抵扣？
2. 样品发货周期是多久？
3. 如果质量OK，我们计划首批下[XX]个，请问能按[阶梯量]的价格走吗？
期待您的回复，谢谢！
```

### 模板B：价格谈判（量换价）

> 适用场景：有明确订单量，想谈更低价

```
老板，您好！
我们是做[平台，如：亚马逊FBA]的，主做[品类，如：户外储能电源]类目，
目前月出[XX]单，正在扩充产品线。
看到贵厂的[产品名]，参数如下：[克重/材质/认证要求]。
我们的计划：
- 首批试单：[XX]个（测款）
- 30天内返单：[XX]个（如样品OK）
- 长期稳定单：[XX]个/月
目前对比了几家工厂，贵厂[优势，如：认证齐全、发货快]，比较有竞争力。
想确认一下：
1. 如果首批[XX]个，能按[XX]个的价格走吗？
2. [XX]个/月的稳定单，月结或[账期]可以吗？
3. 报价含不含税？开票几个点？
麻烦您报个最优价，我们尽快决定，谢谢！
```

### 模板C：OEM定制

> 适用场景：想做自有品牌，需要Logo丝印/彩盒定制

```
老板，您好！
我们是[公司名]，做[目标市场]跨境自有品牌的，
目前想推一款[产品品类]，想找有OEM能力的工厂合作。
具体需求如下：
【产品定制】
- 基础款参考：贵厂的[产品名]
- 定制要求：
  ① 品牌Logo丝印（位置：[侧面/底部]）
  ② 彩盒定制（我们提供设计稿）
  ③ [其他：如线材颜色定制/接口改装]
- 材质/参数要求：[补充细节]
【数量计划】
- 首批：[XX]个（品牌定制需开模，先做这么多测市场）
- 3个月：[XX]个
- 6个月：[XX]个（视市场反馈调整）
【认证需求】
- 目标市场：[美国/欧盟]，需[CE/FCC/FDA等]
请问：
1. 贵厂是否支持上述定制？开模周期和费用多少？
2. 首批[XX]个，定制+产品单价多少？
3. 大货多久能交货？
期待合作，我们有稳定的订单量，希望能长期合作！
```

### 模板D：一件代发测试

> 适用场景：想测试多个款式，不想压库存

```
老板，您好！
我们是做[TikTok直播/抖音小店]的，目前想找支持一件代发的供应商合作。
产品需求：[品类，如：宠物用品]
我们的情况：
- 日均出单：[XX]单（测款期）
- 主要通过直播带货，对发货时效要求较高（最好[24h]内发）
- 希望先测试[5-10]款，看数据再决定主力款
请问：
1. 支持一件代发吗？代发费用怎么算？
2. 是否有产品实拍视频和带佣金链接（图包支持）？
3. 库存深度如何？爆单能来得及补货吗？
4. 能否提供样品我们先拍照/测款？
谢谢，期待合作！
```

## 特殊场景话术

### 砍价试探话术

```
[先用诚恳语气开场]
老板，我跟您说实话，我们对比了[X]家工厂，
贵厂在[认证/品质/交期]上确实有优势，我们很想合作。
但目前[X]家报的价比您低[X]%，
如果能在这个基础上再降[X]%，我们可以[X天内]就下单。
不知道老板这边能不能再让一让？
```

### 催样话术

```
老板您好，请问样品发出了吗？单号方便给一下吗？
我们这边拿到样品会尽快安排测试，有结果第一时间回复您。
确认没问题的话，下周就可以下正式订单了。
谢谢！
```

### 复购谈判话术

```
老板，样品我们测完了，质量OK！
想跟您谈一下长期合作的条款：
1. 每月稳定单量：[XX]个，有没有月结的可能？
2. 价格方面，跑了两个月，我们希望能在现有基础上降[X]%，
   毕竟量在涨，长期合作对双方都好，您看呢？
3. 质量问题如何处理？退换货流程怎么走？
期待长期合作！
```

## 工厂回复应对策略

| 工厂回复 | 应对策略 |
|----------|----------|
| "价格已经是最低了" | 追问阶梯报价，用量换价；或提出长期框架换优惠 |
| "MOQ太高" | 谈分批交货；或提出先付MOQ部分定金锁定价格 |
| "不支持样品" | 警惕，说明正规工厂应支持拿样；换一家 |
| "交期要X天" | 确认是否可加急，加急费用多少 |
| "不含税" | 确认含税含运价格，综合比较总成本 |
| 不回复 | 24小时后跟进一次，仍不回复换下一家 |

## 询盘效率提升建议

1. **同一产品联系 3-5 家工厂**，不吊死一家
2. **收集好对比数据**（别家报价/认证/交期），作为谈判筹码
3. **工厂问你是谁时**，直接说做跨境电商/亚马逊/TikTok，不要遮遮掩掩
4. **时间节点要明确**：给工厂紧迫感，自己也更有谈判筹码
5. **样品费不要省**：正规工厂样品费可抵货款，说明你是认真做生意的
