# X Algorithm Optimization - Quick Reference

## TL;DR
The X For You algorithm uses a Grok transformer to predict engagement probabilities. Optimize for: **replies**, **quotes**, **retweets**, **video views**, **dwell time**. Avoid: **negative signals** (blocks, mutes, reports). Space posts 2-4 hours apart.

## Engagement Value (Inferred Priority)
1. **High Value**: Reply, Quote, Retweet, Share, Video View, Dwell Time, Follow
2. **Medium**: Favorite, Click, Profile Click, Photo Expand
3. **Negative**: Not Interested, Block, Mute, Report

## Content Format Optimization

| Format | Best For | Key Requirements |
|--------|----------|------------------|
| Video | Viral reach, dwell | Native upload, vertical, first 3s hook, captions, duration >2s |
| Carousel | Photo expands | 3+ images, "swipe" hooks |
| Threads | Dwell time, replies | Strong first tweet, medium-length tweets, clear flow |
| Quote bait | Quotes, replies | Bold statements, data points, hot takes |
| Polls | Replies, engagement | Built-in X feature, relevant options |

## Posting Cadence
- **Minimum spacing**: 2-4 hours between posts
- **Daily limit**: 3-5 posts maximum
- **Reason**: Author diversity decay attenuates consecutive posts from same author

## Hook Formula
```
[Emotion or Curiosity] + [Value Proposition] + [Action Trigger]
Examples:
- "This changed how I think about X. Here's why 👇"
- "You're spending too much on Y. Cut costs by 80% with this trick."
- "Hot take: Z is overrated. Change my mind."
```

## Red Flags (Negative Signals)
- Clickbait without delivery
- Misleading claims
- Spammy @mentions
- NSFW without context
- Repetitive content
- Low-effort posts
- Public arguments

## Metrics to Track
Per post:
- Impressions (if available)
- Engagement rate
- Breakdown: replies, retweets, quotes, video views, photo expands, profile clicks, follows
- Dwell time (if available)
- Negative signals: blocks, mutes, reports

## Testing Framework
1. Hypothesis: "X format will increase replies by Y%"
2. Create A/B variants (change one variable)
3. Post at similar times
4. Compare metrics (n=10+ for significance)
5. Iterate

## Common Questions Answered

**Q: Are hashtags useful?**
A: Probably minimal direct impact on ML scoring. Use for discoverability/grouping.

**Q: Should I ask for engagement?**
A: "Like/retweet if you agree" may trigger negative signals if overused. Subtle CTAs better.

**Q: Does tagging help?**
A: Yes, can increase replies and profile clicks. Tag relevant accounts (not spam).

**Q: What's the ideal tweet length?**
A: Medium (200-500 chars) likely optimal for dwell + content density.

**Q: When to post?**
A: When your audience is most active. Test different time windows.

**Q: How important are replies vs likes?**
A: Replies likely weighted higher (conversation indicator).

**Q: Can I game the algorithm?**
A: Short-term tricks often backfire via negative signals. Focus on genuine value.

**Q: Does quoting vs retweeting matter?**
A: Quote likely higher value (adds commentary + amplification).

**Q: Should I reply to my own tweets?**
A: Might boost dwell/conversation but don't spam. Genuine Q&A helpful.

**Q: Do videos get more reach?**
A: If optimized (native, vertical, hook), yes. Short low-effort videos may not.
