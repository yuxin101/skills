# Merchant-facing documentation

Day-to-day **store administration** (how to click through the Shopify admin, billing, domains, etc.) is covered for merchants on **[help.shopify.com](https://help.shopify.com/)**.

This skill’s **`DOC_*.md`** files target **developers** (apps, APIs, themes, extensions). When the user asks purely **merchant** questions, prefer Help Center links over low-level API detail.
