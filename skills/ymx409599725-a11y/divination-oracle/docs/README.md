# Divination Oracle Skill

一个基于梅花易数算法与 AI 解读能力的自动化周易占卜助手。

## 核心功能
- **算法引擎**: 基于梅花易数的时间起卦法，自动计算本卦与变爻。
- **知识库**: 内置 64 卦数据库，实现精准卦辞匹配。
- **智能解读**: 结合用户问题，自动调用大模型进行专业语义分析。

## 安装与使用
1. 安装技能：`clawhub install divination-oracle`
2. 使用方式：Jarvis 中直接对话：`"帮我算个卦：[您的具体问题]"`
