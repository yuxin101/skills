# beddel (OpenClaw Skill)

OpenClaw skill for the [Beddel](https://github.com/botanarede/beddel) declarative YAML workflow engine.

## Install

```bash
openclaw skills install --from ./skill
openclaw plugins install @botanarede/beddel
```

## What's included

- `SKILL.md` — Skill definition (loaded by OpenClaw agents)
- `examples/setup-beddel.yaml` — Automated plugin setup workflow
- `references/` — Detailed docs on workflow format, primitives, strategies, and variables
