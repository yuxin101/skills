## What This Skill Does

Short Video Script generates complete video scripts optimized for short-form platforms. Each script includes an attention hook (first 1-3 seconds), structured body with benefit-driven beats, and a conversion-focused CTA — all calibrated to platform-specific audience behavior and algorithm preferences.

### Core Capabilities

1. **Hook Generation** — Proven first-3-second hooks: controversy, statistic, question, transformation, "wait for it"
2. **Script Structure** — Beat-by-beat timing with visual/audio cues and on-screen text directions
3. **Platform Adaptation** — Same concept, different execution per platform's trending format
4. **Shot List** — Camera angles, transitions, and B-roll suggestions for each beat
5. **Copy Variants** — Multiple script versions for A/B testing
6. **Trending Format Templates** — Pre-built templates based on current viral formats
7. **CTA Optimization** — End-screen CTAs tuned for each platform (follow, save, comment, link-in-bio)

### Script Frameworks

- **Problem → Solution → CTA** — Classic direct response
- **Story Arc** — Mini narrative with emotional payoff
- **Listicle** — "3 things you didn't know about..."
- **Before/After** — Transformation reveal
- **POV/Day-in-Life** — Relatable scenario
- **Reaction/Duet** — Response to trending content
