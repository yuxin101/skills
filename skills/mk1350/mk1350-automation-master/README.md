# 自动化办公大师

## 简介
一站式办公自动化套件，集成文件处理、数据提取、财税智能同步等核心功能。一次安装，拥有全套办公效率工具。

## 核心功能

### 文件处理类
| 功能 | 说明 |
|------|------|
| **文件转换** | PDF/Word/Excel/图片互转，支持批量 |
| **批量重命名** | 按规则批量重命名文件，保留原始时间戳 |
| **数据合并** | 合并多个Excel工作表（pd.concat）——对应技能里的“数据堆叠” |
| **数据拼接** | 拼接多个Excel/CSV文件（pd.merge）——对应技能里的“数据拼接” |
| **模板生成** | 根据Excel数据批量生成Word/Excel文档 |

### 财税处理类
| 功能 | 说明 |
|------|------|
| **发票信息提取** | 从PDF/DOCX提取发票信息（增值税、火车票、机票） |
| **财税智能对账** | 税局数据 vs 财务数据自动对账（支持多对多匹配） |
