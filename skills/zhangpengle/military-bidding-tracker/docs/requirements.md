# 产品需求文档 (PRD)：基于 OpenClaw 的招投标商机跟踪 Skillset

## 1. 产品概述
### 1.1 背景与目标
为了解决公司在招投标过程中信息零散、跟进不及时、依赖人工记忆带来的遗漏和效率低下等问题，本项目依托 OpenClaw 框架，开发一套专门用于“商机跟踪与招投标全生命周期管理”的智能技能组（Skillset）。

### 1.2 核心架构原则
* **多技能协同 (Multi-Skill Architecture)：** 摒弃单体臃肿的 Agent，采用“主线调度 + 专业 Skill”的架构，确保意图识别的精准度。
* **混合存储策略 (Hybrid Storage)：** * **关系型数据库 (SQLite)：** 用于管理结构化元数据（项目状态、负责人、金额、关键时间节点、统计数据等），支持复杂的 SQL 维表查询与权限隔离。
    * **本地文件系统：** 用于按层级（如 `年度/项目名/`）安全归档招标公告、招标文件、投标文件等原件。

## 2. 核心技能 (Skills) 详细设计

该 Skillset 由以下五个核心 Skill 组成，通过 Channel（如企微、飞书、钉钉等）与用户进行自然语言及多模态交互。

### 2.1 智能立项技能 (Information_Extraction_Skill)
* **触发条件：** 用户在 Channel 中上传招标公告（PDF 或图片截图）。
* **核心功能：** 调用视觉/文档解析模型，自动提取非结构化文件中的关键结构化字段。
* **需提取的 JSON Schema：**
    * 项目基本信息：项目名称、预算/价格、招标人、代理公司。
    * 关键时间：报名截止时间、购买标书时间、开标时间。
    * 其他：标书购买地点、源文件路径。
* **执行动作：** 提取信息后，在 Channel 中与用户进行二次确认（“是否确认立项并指派负责人？”）。确认无误后，将数据写入 SQLite 并在本地建立项目文件夹，项目状态变更为 `[待获取标书]`。

### 2.2 标书购买督办技能 (Requirement_Parser_Skill)
* **触发条件：** 项目状态变更为 `[待获取标书]`。
* **核心功能：** 针对购买标书环节的繁琐要求，精读公告中的“获取招标文件”段落。
* **执行动作：** 1.  提取购买标书所需的**携带材料清单**（如：营业执照复印件加盖公章、法人授权委托书、身份证复印件等）。
    2.  生成待办清单推送到 Channel 给项目直接负责人。
    3.  待负责人买回标书后，接收负责人上传的招标文件原件并归档。
    4.  更新 SQLite 数据库状态为 `[标书制作中]`。

### 2.3 智能排期技能 (Timeline_Advisor_Skill)
* **触发条件：** 项目状态变更为 `[标书制作中]`。
* **核心功能：** 根据开标时间和地点，倒推并建议安全的封标与出发时间。
* **执行动作：** 综合考虑物理距离（如异地开标需要的路程时间）和公司内部流程（如财务盖章、装订耗时），向负责人输出倒排期建议。例如：“开标时间为下周五上午9点，建议最迟下周三完成技术标定稿，下周四下午15:00前完成封标”。

### 2.4 角色化通知技能 (Role_Based_Notification_Skill)
* **触发条件：** 设定的定时任务（Cron Job）或临近关键时间节点。
* **核心功能：** 实现“事找人”的自动化跟进，并严格执行数据权限隔离。
* **执行动作：**
    * **项目直接负责人视图：** 仅接收个人负责项目的临近节点提醒（如：“您的【XXX项目】距离报名截止仅剩 1 天，请尽快处理”）。
    * **项目总监视图：** 接收全局项目的大盘播报与风险提示（如：“本周共有 3 个项目开标，其中【YYY项目】暂未完成封标，存在延期风险”）。

### 2.5 效能统计与复盘技能 (Analytics_Skill)
* **触发条件：** 开标结束后的结果录入，或管理层的统计查询指令。
* **核心功能：** 闭环记录招投标结果，沉淀业务数据，支持多维度的数据透视。
* **执行动作：**
    1.  **结果录入：** 引导负责人录入开标现场情况（第一候选人、中标金额、我方排名及失分项）。
    2.  **数据统计：** 响应管理层在 Channel 中的自然语言查询，将指令转化为 SQL 语句，输出如“第一季度整体中标率”、“各负责人业绩对比”、“常见竞争对手出场频次”等关键指标。

## 3. 交互体验设计 (Channel 端)

**场景示例：立项与跟进**
> **用户：** [发送一张招投标公告图片] “建个项目，让王经理跟进。”
> **OpenClaw：** “好的，已为您提取关键信息：
> - 项目名称：XXX系统采购项目
> - 预算金额：150万
> - 开标时间：2026-04-10 14:00
> 是否确认由 @王经理 负责并在数据库立项？”
> **用户：** “确认”
> **OpenClaw：** “立项成功。已提醒 @王经理：该项目需携带【营业执照复印件】及【授权书】前往【XXX地址】购买标书，请在 4月2日 前完成。”
