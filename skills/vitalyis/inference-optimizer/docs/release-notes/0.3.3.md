# Release Notes: v0.3.3

## Summary

This release hardens `inference-optimizer` against real production failures seen in WhatsApp/OpenClaw deployments:

- avoid helper shell commands before the main audit
- treat `allowlist miss` as a hard deny unless an approval ID exists
- reduce allowlist fragility by preferring direct script execution
- make allowlist diagnostics report the actual executable path being checked
- align VPS ownership guidance with the intended topology for this deployment
