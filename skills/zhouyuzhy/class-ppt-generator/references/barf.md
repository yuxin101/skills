# BARF Prompts 框架

## 什么是BARF

BARF 是一个结构化提示词框架，用于生成高质量的PPT大纲：

- **B**ackground (背景) - 课程背景、学生情况
- **A**ction (行动) - 教学活动、互动设计
- **R**esult (结果) - 预期效果、情感目标
- **F**ormat (格式) - PPT页面规范、视觉要求

## 班会课 BARF 示例

```
Background: 初二学生，手机依赖严重，每天平均使用3小时以上
Action: 
  - 撕纸游戏（时间可视化）
  - 小组讨论手机使用公约
  - 案例分析（正面/反面）
Result: 学生制定自律承诺书，增强时间管理意识
Format: 6页PPT，封面+现状+游戏+案例+公约+结语
```

## 英语课 BARF 示例

```
Background: 八年级学生，已掌握基础英语表达
Action:
  - Michael Jordan失败案例视频导入
  - yet句型操练（I can't...yet）
  - 小组海报设计（Growth Mindset）
Result: 学生能用yet句型表达成长型思维
Format: 7页PPT，Title+Lead-in+Vocab+Practice+Project+Show+Summary
```

## 百度AI PPT 规范

- 封面：标题+副标题+作者+日期
- 内容页：文字≤6行，关键词加粗
- 互动页：标注【活动】+时间提示
- 备注栏：Speaker Notes 给教师参考
