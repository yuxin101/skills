# 班会课主题库

## 热门主题推荐

### 成长与心态
1. **The Power of Yet** - 成长型思维，用"yet"转化固定型思维
2. **抗挫力培养** - 面对挫折的正确态度
3. **目标与计划** - 学期/学年目标设定

### 健康与生活
4. **手机依赖** - 合理使用电子产品
5. **睡眠与健康** - 青少年睡眠重要性
6. **食品安全** - 健康饮食习惯

### 人际关系
7. **校园欺凌** - 预防与应对
8. **友谊与孤独** - 正确看待社交
9. **感恩父母** - 亲子沟通

### 考试与压力
10. **考试焦虑** - 缓解考前压力
11. **学习方法** - 高效学习技巧
12. **考试诚信** - 诚实守信

## 典型活动设计

| 主题 | 互动活动 | 时长 |
|------|----------|------|
| 手机依赖 | 撕纸人生（时间可视化） | 10分钟 |
| 成长型思维 | 案例讨论+海报设计 | 15分钟 |
| 考试焦虑 | 放松呼吸练习+分享 | 8分钟 |
| 校园欺凌 | 角色扮演+小组讨论 | 12分钟 |

## 百度AI PPT模板建议

- 配色：蓝绿色（成长）、橙黄色（活力）
- 风格：卡通插画、简洁清新
- 字体：微软雅黑、黑体
