# BARF框架快速参考

## 框架概述

**BARF** = Background + Analysis + Reasoning + Framework

---

## B - Background（背景）

### 多级别缠论结构分析

**分析链条**：
```
日线/4小时 → 定战略方向
    ↓
30分钟/4小时 → 定操作框架与关键中枢
    ↓
5分钟/1分钟 → 定精确买卖点
```

### 中枢界定
- 精确描述区间（上轨、下轨）
- 标注在趋势中的位置（中继、转折）

### 买卖点与背驰
- 明确一、二、三类买卖点
- 综合判断：价格幅度 + 斜率 + MACD面积 + DIF高度
- 趋势背驰：同趋势方向前后两段的力度对比

---

## A - Analysis（分析）

### 均线系统
- MA5、MA13、MA34、MA89、MA144
- 判断支撑/压力位

### 斐波那契回撤
- 23.6%、38.2%、50%、61.8%、78.6%
- 与缠论中枢、阻力/支撑位整合

### 关键位置整合
- 列出所有关键支撑/压力位
- 整合均线、斐波那契、中枢位置

---

## R - Reasoning（推演）

### 走势完全分类
- 至少2-3种概率不同的路径
- 每种分类包含：
  1. 路径描述
  2. 缠论定位
  3. 操作意义

### 核心推演逻辑
- 能否重回关键位置
- 能则趋势结束；不能则趋势延续
- 形成第三类买卖点的可能性

## F - Framework（框架）

### 动态知识库
- 已达成共识的关键结论

---

## 输出格式

### 1. 核心结论
一段话概括市场核心状态、关键结构、战略方向

### 2. 多级别缠论结构分析
分级别阐述：走势结构、中枢、买卖点、背驰

### 3. 关键位置与完全分类
- 关键支撑/压力位
- 2-3种走势分类
- 各路径的缠论定义与操作含义

### 4. 终极操作策略
- 总体方向（主多/主空/震荡）
- 具体操作区域
- 入场信号、止损位、目标位
- 持仓者vs空仓者策略

### 5. 知识库更新（如有）
新确认的结论
