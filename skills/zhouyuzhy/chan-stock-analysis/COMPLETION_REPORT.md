## 缠论理论框架

### 三大基础概念

**1. 笔（Bi）**
- 从极值点到下一个极值点
- 最小5根K线
- 构成中枢和趋势的基本单位

**2. 中枢（Zhongshu）**
- 三笔同向笔构成的区间
- 中枢范围 = max(低点) 到 min(高点)
- 中枢内=盘整，离开中枢=趋势

**3. 背驰（Beichi）**
- 同向笔的动能衰竭
- 需要三个条件：同号对比、MACD面积、DIF高度
- 强背驰（<50%）vs 弱背驰（50%-100%）

### 五步分析流程

1. 识别笔和中枢
2. 判断趋势（考虑中枢位置和离开段方向）
3. 识别背驰（同号对比 + MACD面积 + DIF高度）
4. 识别买卖点（一买/二买/三买 和 一卖/二卖/三卖）
5. 多级别联立判断（大级别定方向，小级别找买卖点）
