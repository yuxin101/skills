# 电商运营助手

## 核心功能

1. **标题优化**：关键词布局 + 点击率优化，符合各平台搜索规则
2. **详情页文案**：卖点提炼、FAB法则、用户痛点攻克
3. **客服话术**：售前咨询、议价应对、差评处理、催好评
4. **活动文案**：双11、618、日常大促文案模板
5. **竞品分析框架**：结构化对比，找到差异化卖点
6. **主播带货脚本**：产品介绍、逼单话术、互动话术

## 工作流程

### Step 1：明确需求类型

判断用户需要哪类帮助：
- A. 标题优化 → 询问：产品名称、核心卖点、目标平台
- B. 详情页文案 → 询问：产品名称、主要卖点、目标人群
- C. 客服话术 → 判断场景类型（售前/议价/差评/催评）
- D. 活动文案 → 询问：活动名称、折扣力度、产品
- E. 主播脚本 → 询问：产品名称、售价、核心卖点

### Step 2：生成内容

参考 references/ 中的框架和规则：
- 标题规则见 references/platform-rules.md
- 文案框架见 references/copywriting-frameworks.md
- 客服话术见 references/customer-service-scripts.md

### Step 3：优化建议

输出内容后，给出1-2条针对性优化建议。

## 商品标题优化规范

### 标题结构公式
```
[品牌词] + [核心品类词] + [核心卖点/属性] + [目标人群/使用场景] + [长尾词]
```

示例：
- 原始标题：女士运动裤
- 优化后：【XX品牌】女士高腰显瘦运动裤 速干透气健身跑步瑜伽裤 秋冬大码宽松卫裤女

### 关键词布局原则
1. **核心词前置**：最重要的关键词放在标题前20个字
2. **属性词堆砌**：材质+颜色+风格+季节+使用场景
3. **人群词精准**：直接指向目标用户（大码/男款/孕妇/老年等）
4. **场景词补充**：使用场景让用户更容易联想到需求
5. **避免重复**：同一个词不要出现2次以上

## 详情页文案结构

### 黄金结构（FAB法则）

```
F（Feature 特征）→ A（Advantage 优势）→ B（Benefit 利益）

例：
F：采用双层隔热设计
A：相比普通保温杯，保温时间延长3倍
B：早上装的热咖啡，下午喝还是温热的，省去了反复加热的麻烦
```

### 详情页内容结构
```
1. 核心卖点Banner（视觉冲击，3-5个卖点）
2. 使用场景图（让用户代入自己）
3. 产品细节展示（材质/工艺/尺寸）
4. 核心优势对比（与竞品的差异化）
5. 用户好评截图（社会认同）
6. 常见问题解答（消除顾虑）
7. 品牌故事/资质（建立信任）
```

### 卖点提炼模板
```
原材料/成分 → 解释这个原材料/成分的好处 → 转化为用户获益

示例：
"本品采用[进口304食品级不锈钢] → 表面无毒无味，通过FDA安全认证 → 
 让您和家人每天喝水更安心，再也不担心塑料微粒问题"
```

## 活动文案写法

### 大促文案公式
```
【超强力感】活动名 + 【价格诱惑】折扣/立减 + 【紧迫感】限时/限量 + 【行动指令】
```

#### 双11文案示例
```
标题：双11狂欢！全场满300减100，爆款直降50%！
副标题：今晚0点开抢，前100名额外赠精美礼盒
紧迫感：距离结束仅剩XX小时
CTA：立即收藏，0点准时开抢！
```

#### 618文案示例
```
标题：618年中盛典 | 今年最大力度，买就是赚到！
卖点：比双11更低价 · 加量不加价 · 老客专属福利
紧迫感：到手价 = 历史最低价，错过等一年
```

#### 日常促销文案
```
轻文案（小折扣）："今天下单立减XX元，明天恢复原价"
中文案（中折扣）："周年庆大促，N折优惠仅剩最后XX件"
重文案（大折扣）："清仓最后一批，售完不补！史上最低价X折"
```

## 主播带货脚本要点

### 单品介绍流程（3-5分钟）

1. **抛钩子**（0-30秒）
   "家人们，接下来这款产品绝对是今天的爆款，我来之前专门用了一个月……"

2. **产品介绍**（30秒-2分钟）
   - 外观第一印象
   - 核心卖点（最多3个，每个都要演示/展示）
   - 使用场景植入

3. **价格铺垫**（2-3分钟）
   "这款产品官方售价是{原价}，今天在我直播间……"
   "我帮大家砍了很久的价格，才争取到了……"

4. **价格公布+赠品**（3-4分钟）
   "到手价{到手价}！还送{赠品}！"
   "算一下，赠品价值{赠品总价}，相当于免费送！"

5. **逼单收尾**（4-5分钟）
   "库存有限，已经下了XX单！"
   "今天这个价格，下播就没有了！"
   "点击下方购物袋，现在就下单！"

### 高频逼单话术

| 场景 | 话术 |
|------|------|
| 库存紧迫 | "只剩最后XX件了，抢到就是赚到！" |
| 价格稀缺 | "这个价格直播间专属，平台搜不到！" |
| 时间紧迫 | "还有5分钟，这个价格就下架了！" |
| 赠品诱惑 | "现在下单还送价值XX元的赠品！" |
| 回头客 | "老粉看一下，今天的力度是最大的一次！" |
| 犹豫者 | "不用想了，七天无理由退换，买错了我帮你退！" |

## 差评处理核心原则

1. **24小时内响应**，不要拖延
2. **先道歉，后解释**，绝不与用户对抗
3. **给出解决方案**，不要只说对不起
4. **私聊解决，不要在评论区争论**
5. 解决后，礼貌请求用户修改评价
