# 客服话术模板库

> 使用原则：话术仅供参考，需根据实际情况灵活调整，避免机械照搬显得生硬。

---

## 一、售前咨询话术

### 1.1 产品功能咨询

**场景：用户询问产品功能是否满足自己的需求**

```
标准回复：
"您好！感谢您的咨询😊 关于您问的{具体问题}——

[产品功能说明（1-3点，简洁明了）]

如果您还有其他疑问，欢迎继续问我，我会第一时间为您解答！
祝您购物愉快！🌟"
```

### 1.2 尺码/规格咨询

**场景：用户不确定选哪个尺码/规格**

```
标准回复（服装类）：
"您好！帮您选尺码，可以提供您的[身高/体重/三围]，我为您精准推荐！

您也可以参考我们的尺码表（详情页有详细说明）。
如果您在两个尺码之间拿不定主意，建议选[大/小]一码，
因为我们的衣服[偏大/偏小/属于正常码]。

放心，如果收到觉得尺寸不对，我们支持免费换货哦！😊"
```

### 1.3 发货时间咨询

```
标准回复：
"您好！关于发货时间——

[现货] 当天14:00前付款，当天发货；14:00后付款，次日发货。
[预售] 预计X月X日开始发货，请您耐心等待～

发货后会第一时间给您发送快递单号，您可以随时追踪物流。
如果有急需，可以告诉我，我帮您看看能否加急处理！😊"
```

## 二、议价应对话术

### 2.1 小幅让步（有优惠空间）

**场景：用户希望便宜一点**

```
话术A（给小优惠）：
"亲，我理解您的心情！这款的利润真的很薄，我帮您申请了一个
小小的优惠：[包邮/赠品/优惠券X元]，这已经是我能申请到的最大力度了！
这个价格真的超值，要不要直接拿下？😊"

话术B（强调价值不降价）：
"亲爱的，我们这款之所以定这个价格，是因为[核心卖点/品质保障]，
品质有保证。同类产品里，我们的性价比已经非常高了！
您可以货比三家，相信您会认可的。如果您下单，我帮您备注
赠送[小赠品]表示心意，怎么样？"
```

### 2.2 无优惠空间时

```
话术（坚守价格）：
"亲，非常感谢您的喜欢！这款我们真的已经是全网最低价了，
利润薄到都快倒贴了😅 

但是我可以确保的是：我们的品质绝对对得起这个价格，
而且我们有[售后保障说明]，买错了也没有后顾之忧！
您觉得怎么样呢？"
```

### 2.3 用户价格比较（说其他店更便宜）

```
话术：
"您好！感谢您的反馈，我了解一下您看到的价格情况——

可能的情况有：
1. 不同活动节点价格有所差异
2. 不同型号/规格的价格区别
3. 部分平台商品为翻新/拆封产品

如果真的有完全一样的商品价格更低，您可以把链接发给我，
我帮您核实一下，如果确认无误，我来帮您争取！

我们最重视的是客户的购物体验，一定让您买得放心！😊"
```

## 三、差评处理话术

### 3.1 产品质量问题差评

**处理步骤：先道歉 → 了解问题 → 提出解决方案 → 私聊解决**

```
公开回复（差评下面的回复，简短得体）：
"您好，非常抱歉给您带来了不好的体验！我们已经看到您的反馈，
客服将在[X小时]内主动联系您，一定为您妥善处理！
感谢您的宝贵意见，我们会持续改进！"

私聊话术：
"您好，我是[店铺名]的客服，看到您对我们产品的反馈，
我们感到非常抱歉！

请问您遇到的具体问题是[确认问题]？我来帮您看看怎么解决：

[方案A] 如果是产品质量问题，我们支持免费补发/退换；
[方案B] 如果是使用方法问题，我来为您详细说明；
[方案C] 如果您完全不满意，我们支持全额退款。

请问您倾向于哪种解决方式？无论如何，我们一定让您满意！😊"
```

### 3.2 物流问题差评

```
公开回复：
"您好，非常抱歉快递配送给您带来了不便！物流问题我们也很无奈，
但我们会督促物流公司改进。请私信我们，我们来核实赔付方案！"

私聊话术：
"您好，关于物流延误/破损的问题，我们非常抱歉！
虽然物流是第三方负责，但作为卖家我们有责任为您跟进。

我已经帮您[联系快递公司查询/申请赔付/安排补发]，
预计[时间]内处理完毕。

给您带来的不便，我们送您一张[X元]优惠券，
下次购买可以直接使用，表示我们的歉意！😊"
```

### 3.3 描述不符差评

```
私聊话术：
"您好！非常感谢您的反馈，也很抱歉产品没能达到您的预期。

关于您提到的[具体问题]，我想先了解一下：
[具体询问问题]

如果确实是我们的描述存在误导，我们支持退货退款，
运费我们来承担，请您放心！

同时，您的反馈对我们改善描述非常有帮助，
非常感谢您的诚实评价！"
```

## 四、催好评话术

> ⚠️ 注意：不得以礼品、现金等方式强制要求用户修改差评，以下为合规的关怀式催评话术。

### 4.1 发货后催评（包裹内附纸条）

```
纸条文案：
"亲爱的朋友，
感谢您选择[品牌名]！

您收到的这份礼物是我们精心挑选/制作的，
希望它能给您带来惊喜 ✨

如果您满意，一句好评就是对我们最大的鼓励，
让我们看到用心的价值💪

如有任何不满意，请第一时间联系我们，
我们承诺 24小时内处理！

[客服联系方式]
[品牌名称/LOGO]"
```

### 4.2 收货后旺旺/微信催评

```
话术：
"亲，您好！我是[店铺名]的客服小X，
您的订单已经签收了，不知道产品是否令您满意？

如果您觉得产品还不错，欢迎给我们留下您的使用感受，
您的好评是对我们最大的支持，也会帮助更多有需要的朋友做参考😊

如果有任何问题，随时联系我，第一时间为您处理！"
```

### 4.3 处理售后后的催评

```
话术（售后解决满意后）：
"亲，很高兴这次的问题顺利解决了！

我们一直致力于提供最好的产品和服务，
如果您对我们这次的处理方式感到满意，
是否可以给我们一个机会修改/补充您的评价？

您的支持对我们小店非常重要，谢谢您！😊
如果以后有任何需要，我随时在！"
```

## 五、特殊场景话术

### 5.1 用户无理取闹

```
原则：保持冷静，不卑不亢，记录沟通记录，必要时走平台申诉。

话术：
"您好，我理解您的心情，也很抱歉您有不愉快的体验。
我们非常愿意帮助您解决问题，但需要您的配合提供[相关证明/信息]。

我们会根据平台规则，为您提供合理的解决方案。
如果您对我们的处理方式有异议，也可以申请平台介入，
我们会配合平台的调查和处理，谢谢您的理解！"
```

### 5.2 询价后消失的客户（召回）

```
话术：
"亲，您好！之前您咨询了我们的[产品名]，不知道后来有没有找到合适的？

今天我们正好有[优惠活动/新款上市/库存紧张]，
想到您之前有兴趣，特意来通知您一下😊

如果您还有需要，欢迎随时联系我！"
```
