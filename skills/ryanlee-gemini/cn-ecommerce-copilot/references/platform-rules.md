# 各平台标题规则与规范

## 淘宝/天猫

### 标题规则
- **字数限制**：30个字（中文字符）
- **关键词权重**：前10个字权重最高
- **搜索逻辑**：平台按关键词分词匹配搜索

### 标题结构
```
品牌词（可选）+ 核心品类词 + 属性词×3-5个 + 人群词/场景词
```

### 必须包含的词类
| 词类 | 说明 | 示例 |
|------|------|------|
| 核心品类词 | 用户搜索的主要词 | 连衣裙、保温杯、无线耳机 |
| 属性词 | 颜色、材质、尺码、风格 | 纯棉、白色、长款、韩版 |
| 人群词 | 明确目标用户 | 女童、孕妇、大码、老年人 |
| 场景词 | 使用场景 | 跑步、通勤、居家、户外 |

### 禁止词汇（敏感词，会被屏蔽）
- 极限词：最、第一、唯一、顶级、国家级
- 违禁词：特效、根治、永久、100%效果
- 夸大词：神奇、奇效、无敌（部分情况）
- 注意：各类目规范不同，建议在发布前用天猫规则工具检测

### 标题质量评估（自查）
- [ ] 核心词是否在前10字？
- [ ] 是否包含3-5个属性词？
- [ ] 是否有人群/场景词？
- [ ] 无重复关键词？
- [ ] 无违禁词？

## 京东

### 标题规则
- **字数限制**：60字节（约30个中文字符）
- **搜索逻辑**：关键词分词匹配，品牌词权重高
- **强调品牌**：京东用户对品牌信任度高，品牌词要靠前

### 标题特点
- 比淘宝更强调品牌和型号
- 适合堆砌型号参数（数码、家电类目）
- 示例："小米（MI）无线蓝牙耳机 降噪 游戏低延迟 Type-C充电 运动跑步 苹果华为安卓通用"

### 各类目建议
| 类目 | 标题侧重 |
|------|----------|
| 数码3C | 品牌+型号+核心参数+功能 |
| 家电 | 品牌+容量/功率+功能特点 |
| 食品 | 产地+品种+规格+口感/用途 |
| 服装 | 品牌+款式+材质+季节 |
| 美妆 | 品牌+系列+产品功效+规格 |

## 拼多多

### 标题规则
- **字数限制**：60字节
- **搜索逻辑**：价格敏感型用户，强调性价比
- **关键词策略**：多堆砌同义词、场景词，覆盖更多搜索词

### 标题特点
- 强调价格竞争力（"工厂直销"、"批发价"）
- 适合堆砌大量同义词（覆盖更多搜索）
- 示例："女式夏季连衣裙法式初恋裙气质小个子裙子过膝长裙仙女飘逸碎花裙2024新款"

### 推荐词库（以服装为例）
```
颜色：黑色 白色 藏青 卡其 杏色 奶茶色
款式：复古 韩版 法式 国风 简约 小清新
人群：小个子 高挑 大码 显瘦 妈妈装
场景：日常 通勤 出游 婚礼 约会
季节：春款 夏款 秋款 冬款 新款 2024年
```

## 抖音小店

### 标题规则
- **字数限制**：60字节
- **搜索逻辑**：结合内容推荐和搜索双引擎
- **特点**：与短视频/直播联动，要考虑话题性

### 标题特点
- 可以更有趣、有故事感
- 与热门话题结合（如 #穿搭）
- 示例："【博主同款】显瘦小香风外套女秋冬百搭气质小个子短款大衣"

### 标题亮点词
```
博主同款 / 明星款 / 素人改造 / 工厂直发
爆款 / 热销 / 新款 / 限时 / 网红
```

## 常见违禁词提醒

> 注意：各平台违禁词可能随政策更新变化，以下为常见违规风险词，使用前建议在对应平台验证。

### 绝对禁用（所有平台）
- 国家级荣誉夸大：最好、第一、唯一、全国最
- 医疗功效夸大：治疗、根治、特效、立即见效
- 虚假承诺：永久有效、100%成功

### 高风险词（谨慎使用）
- 极限词：极致、顶级、超强、无敌
- 效果夸大：立竿见影、彻底改变、奇迹
- 对比词：超越XX品牌、比XX更好

### 食品类特别注意
- 不可声称疾病预防/治疗功效
- 保健食品功能须在批准范围内
- 禁用"减肥、降血糖、防癌"等医疗功效词
