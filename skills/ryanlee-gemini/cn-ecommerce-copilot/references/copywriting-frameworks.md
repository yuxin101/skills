# 电商文案写作框架

## 一、FAB法则

### 什么是FAB
FAB是电商文案的核心框架，从产品特性出发，层层转化为用户利益。

```
F（Feature 特征/功能）：产品本身有什么？
↓
A（Advantage 优势/优点）：这个特征比竞品好在哪？
↓
B（Benefit 利益/好处）：用户因此得到了什么实际好处？
```

### FAB转化示例

| 产品特征 | 优势 | 用户利益 |
|----------|------|----------|
| 采用800蓬鹅绒填充 | 比普通填充物轻20%，保暖性高3倍 | 穿着轻便不臃肿，再冷的天也暖和，上班通勤更自信 |
| 304食品级不锈钢 | 无BPA，通过SGS安全认证 | 每天喝水更安心，家人健康有保障 |
| 四层复合滤芯 | 过滤精度达0.01微米 | 直饮水更纯净，不用再费钱买桶装水 |
| 采用人体工学设计 | 腰部支撑角度精准 | 久坐8小时不累腰，告别下班腰酸背痛 |

### FAB文案模板
```
【{产品名称}】{核心功能特征}（F），相比{对比对象}，{具体优势}（A），
让您{使用场景下的实际获益}，再也不用担心{解决的痛点}（B）。
```

## 二、AIDA模型

### 什么是AIDA
AIDA是注意力→兴趣→欲望→行动的完整说服链路，适合详情页和产品描述。

```
A（Attention 注意）：抓住眼球
↓
I（Interest 兴趣）：激发兴趣
↓
D（Desire 欲望）：制造渴望
↓
A（Action 行动）：促成购买
```

### AIDA在详情页中的应用

**第一屏（Attention 注意）**
- 目标：让用户停下来，愿意继续往下滑
- 内容：震撼的产品大图 + 最大卖点 + 一句话利益点
- 示例："你试过穿上就不想脱的羽绒服吗？"

**第二屏（Interest 兴趣）**
- 目标：让用户觉得"这个跟我有关系"
- 内容：使用场景图 + 用户痛点描述
- 示例："每天通勤2小时，你有没有因为衣服太重而肩颈酸痛？"

**第三四屏（Desire 欲望）**
- 目标：让用户感觉"我要拥有这个"
- 内容：产品细节 + FAB卖点详解 + 与竞品对比
- 示例：逐一展示面料、做工、功能优势

**最后屏（Action 行动）**
- 目标：消除顾虑，促进下单
- 内容：好评截图 + 售后保障 + 限时优惠
- 示例："已有10000+用户好评，七天无理由退换，现在下单立减50元！"

## 三、痛点-方案-价值框架（PAS）

### 框架结构
```
P（Pain 痛点）：描述用户面临的问题/困境
↓
A（Agitate 放大）：扩大痛点的严重性，让用户感受到紧迫
↓
S（Solution 方案）：提出解决方案（你的产品）
↓
V（Value 价值）：强调产品能带来的额外价值
```

### 示例应用（空气净化器）

**痛点（P）：**
"室内空气中的PM2.5浓度，有时是室外的5倍以上……"

**放大痛点（A）：**
"长期在污染空气中生活，会导致呼吸道敏感、睡眠质量下降，
甲醛等有害气体更是儿童哮喘和过敏的重要诱因。
孩子每天在这样的环境里学习、玩耍……"

**方案（S）：**
"XX空气净化器，采用HEPA H13级滤芯+活性炭双重过滤，
30分钟净化25㎡房间，PM2.5净化率达99.97%"

**价值（V）：**
"不只是净化，更是给家人筑起一道健康防护墙。
搭配APP实时监测，空气质量一目了然，安心工作，孩子健康成长。"

## 四、差异化价值主张（UVP）

### 什么是UVP
UVP（Unique Value Proposition）是你的产品与竞品的核心差异，是用户选择你而非竞品的理由。

### UVP提炼方法
1. 列出你的3-5个产品特点
2. 对比竞品，找到差异点
3. 思考这个差异点对用户意味着什么
4. 用一句话总结

### UVP句式模板
```
"对于[目标用户]而言，[产品名]是[品类]中，唯一能[差异化价值]的选择。"

"我们做到了[具体差异化功能]，这意味着[用户获益]，
这是其他[品类]做不到的。"
```

### 常见差异化维度
| 维度 | 示例 |
|------|------|
| 成分/原料 | 唯一采用XX成分/进口原材料 |
| 工艺/技术 | 独有XX专利技术 |
| 性能指标 | 行业唯一XX参数达标 |
| 用户体验 | 唯一XX设计（解决某个体验问题）|
| 服务 | 提供竞品没有的服务（如终身保修）|
| 价格 | 同等质量，价格更低（需要有数据支撑）|

## 五、场景化写作

### 核心原则
把抽象的产品功能转化为具体的生活场景，让用户自动代入。

### 场景化模板
```
【时间】+【人物】+【状态/问题】→ 使用产品 → 【结果/改变】

示例：
"早上7点，赶着上班的你，来不及坐下来好好吃早餐……
[产品名]3分钟准备好营养早餐，让你元气满满迎接新的一天。"
```

### 常用场景词汇库

**时间场景：**
早起匆忙、通勤路上、午休时间、深夜加班、节假日、旅行途中

**情绪场景：**
焦虑、疲惫、想犒劳自己、想送给重要的人、需要放松

**用户角色：**
职场白领、新手妈妈、学生党、健身达人、宅家一族、商务人士
