# 各平台短视频风格指南

## 抖音（TikTok）

### 平台特点
- 用户以18-35岁为主，移动端竖屏观看
- 算法强推新人，冷启动期很重要
- 完播率、互动率（点赞/评论/分享）是核心指标
- 前3秒决定留存，前5秒必须抓住眼球

### 内容风格
- **节奏感强**：快剪辑，每3-5秒切一个镜头
- **直接入题**：不要废话，0秒浪费观众时间
- **强钩子**：开场必须制造悬念、痛点或震惊感
- **情绪驱动**：让用户产生情绪波动（共鸣/惊讶/感动/愤怒）
- **结尾CTA**：明确引导点赞、关注、评论、购买

### 时长建议
| 类型 | 建议时长 | 说明 |
|------|----------|------|
| 爆款短视频 | 15-60秒 | 完播率最高，推流最强 |
| 知识干货 | 60-180秒 | 可以更长，但要保持节奏 |
| 系列内容 | 3-5分钟 | 建立账号矩阵 |
| 直播切片 | 30-90秒 | 精彩片段二次传播 |

### 标题要求
- 字数：最多30字（实际显示约15字）
- 风格：悬念式、数字冲击、痛点直击
- 必须包含1-2个核心关键词
- 示例："99%的人不知道这个手机功能，会的都火了"

### 话题标签
- 3-5个话题，大中小话题搭配
- 例：#干货分享 #生活技巧 #新手必看

## B站（哔哩哔哩）

### 平台特点
- 用户以18-30岁为主，知识/娱乐型社区
- PC端和移动端均有大量用户
- 用户粘性高，喜欢长内容和深度内容
- 弹幕文化浓厚，互动形式独特

### 内容风格
- **内容深度**：B站用户愿意看长视频，信息密度可以更高
- **知识型**：教程、测评、解说、科普都适合
- **个人IP**：强调个人风格和个性化表达
- **弹幕互动**：预设弹幕互动点，如"打XX的来击个掌"
- **UP主感**：有亲和力，像朋友分享而非广告宣传

### 时长建议
| 类型 | 建议时长 |
|------|----------|
| 短片/搞笑 | 1-3分钟 |
| 知识干货 | 5-15分钟 |
| 深度评测 | 10-20分钟 |
| 系列/剧情 | 5-30分钟 |

### 标题要求
- 字数：建议20-40字
- 风格：口语化、有个性、可以有点"毒舌"
- 可以更有创意，不必所有都走"套路标题"
- 示例："我买了XX最贵的产品，结果……（泪目）"
- 数字测评类："用了30天后，我有话说"

### 封面要求
- 高清竖版或横版均可
- 封面文字大且清晰（移动端缩略图也能看清）
- 人物出镜效果好，表情夸张能增加点击率
- 颜色对比强烈，视觉冲击力强

## 小红书

### 平台特点
- 用户以18-35岁女性为主，生活方式平台
- 图文笔记 + 短视频双格式
- 种草属性强，消费决策影响大
- SEO搜索很重要，关键词布局关键

### 内容风格
- **种草感**：真实体验，像朋友推荐而非广告
- **精致美观**：视觉质感高，拒绝粗糙
- **干货清单**："XX个方法"、"必备清单"格式很受欢迎
- **场景化**：在具体生活场景中展示产品/方法
- **亲切平等**：用第一人称，真诚分享

### 标题要求（笔记标题）
- 字数：20字以内，包含核心关键词
- 风格：生活化、种草感、带emoji
- 示例："被朋友问烂了！我的平价护肤秘诀终于来了🧴"
- 关键词在标题前段体现

### 正文格式
- 分段清晰，每段3-5行
- 使用emoji增加可读性和视觉层次
- 关键信息加粗或用符号标注
- 结尾引导关注/收藏/评论

### 话题标签
- 5-10个，包含品牌词、场景词、功能词
- 关键词也要自然融入正文（SEO）

## YouTube

### 平台特点
- 全球最大视频平台，国际用户
- 搜索驱动流量，SEO非常重要
- 长视频友好，广告收益依赖时长
- 订阅+通知是核心转化

### 内容风格
- **搜索优化**：标题、描述、标签都要包含关键词
- **有价值密度**：每分钟信息量要足够
- **开场直接**：前30秒说清楚视频价值，否则跳出
- **章节感**：长视频添加时间章节（chapter markers）
- **专业感**：画质、声音、剪辑质量要高

### 标题要求（英文）
- 60字以内（SERP显示限制）
- 核心关键词在前
- 使用大写增加视觉冲击
- 示例："10 ChatGPT Prompts That Will Change How You Work"

### 描述写法
- 前2-3行最重要（折叠前可见）
- 包含关键词，自然融入
- 添加时间轴（Timestamps）
- 相关链接和社交媒体

## 视频号

### 平台特点
- 微信生态，社交分发为主
- 用户偏中年（30-50岁），覆盖面广
- 朋友圈转发是重要流量来源
- 私域流量变现路径短（直接跳转微信）

### 内容风格
- **信任感**：更正式、更真诚，不必像抖音那么"炸"
- **实用性**：干货实用内容分享欲强
- **温情向**：家庭、亲情、正能量内容表现好
- **中老年友好**：内容不必过于年轻化

### 时长建议
- 最佳：1-3分钟
- 可接受：3-10分钟
- 超过10分钟：需要非常强的内容支撑

### 标题要求
- 简洁明了，20字以内
- 不必过于"标题党"，自然表达即可
- 情感化标题效果好
