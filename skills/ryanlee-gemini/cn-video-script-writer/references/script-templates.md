# 短视频脚本模板库

## 模板1：知识科普类（60秒版）

```
【标题示例】：XX个方法，让你{目标效果}

【开场钩子 0-5秒】
台词："{痛点问题}？今天教你一招，{承诺效果}。"
画面：问题场景展示 或 主播直接出镜

【知识点展开 5-50秒】
第1点（5-15s）：
  台词："第一，{知识点1}。其实很简单——[具体操作/解释]"
  画面：演示/图表/案例

第2点（15-30s）：
  台词："第二，{知识点2}。很多人不知道——[具体内容]"
  画面：演示/对比

第3点（30-50s）：
  台词："第三，{知识点3}。这一步最关键——[核心要点]"
  画面：重点强调

【结尾CTA 50-60秒】
台词："记住这{N}点，{总结效果}。觉得有用的话点个赞，我会持续分享这类干货！"
画面：主播出镜，微笑收尾
```

## 模板2：情感故事类（90秒版）

```
【标题示例】：{反转情节} | 这件事改变了我的{XX}

【情感共鸣开场 0-10秒】
台词："那一年，我{经历了什么}，我以为{错误认知}，结果……"
画面：情感场景，慢镜头，温柔音乐

【故事展开 10-70秒】
第一幕（10-30s）：
  背景交代：[人物/时间/地点/起因]
  台词：自然口语化叙述，第一人称

第二幕（30-55s）：
  矛盾冲突：[发生了什么转折？]
  台词：情绪递进，带入感增强

第三幕（55-70s）：
  解决/顿悟：[如何解决？学到了什么？]
  台词：情感升华

【价值升华 70-90秒】
台词："后来我才明白，{人生感悟}。如果你也{类似经历}，希望这个故事对你有帮助。"
画面：温暖画面收尾
引导互动："你有没有类似的经历？评论区聊聊～"
```

## 模板3：产品测评类（120秒版）

```
【标题示例】：买了XX款{产品}，哪款最值？真实测评来了

【痛点开场 0-10秒】
台词："你是不是也在纠结{产品选择}？我帮你测了{N}款，今天给大家一个最终答案。"
画面：产品铺排展示，制造丰富感

【测评维度介绍 10-20秒】
台词："我会从{维度1}、{维度2}、{维度3}三个维度来评测。"
画面：字幕卡展示评测维度

【逐个测评 20-90秒】
产品A（20-45s）：
  外观/包装展示 → 使用测试 → 数据/对比
  台词："第一款，{产品A}，价格{XX}元。[优点]，但[缺点]……"

产品B（45-70s）：
  同上结构

产品C（70-90s）：
  同上结构

【综合对比 90-110秒】
台词："综合来看，如果你{场景A}，推荐{产品X}；如果你{场景B}，推荐{产品Y}。"
画面：评分表/对比图

【总结推荐 110-120秒】
台词："我的最终推荐是{产品}，性价比最高。链接放评论区了，有问题评论区问我！"
```

## 模板4：带货直播脚本（单品15分钟版）

```
【暖场引流 0-2分钟】
话术："欢迎进入直播间的家人们，今天我们有一个超级福利要送给大家！"
"先把直播间分享给你最好的朋友，待会儿抢购更有优先权！"
互动：让用户打出"来了"、"在看"

【产品介绍 2-6分钟】
品牌故事（1分钟）：
"今天给大家带来的是{品牌}的{产品}，这个品牌……[品牌背书]"

卖点提炼（3分钟）：
"这款产品最大的亮点是{卖点1}——[具体演示/展示]"
"第二个让我心动的是{卖点2}——[对比展示]"
"第三点很多人没注意到，但真的很重要——{卖点3}"

场景植入（1分钟）：
"想想看，你{使用场景}，用了这个之后……[效果描述]"

【价格公布 6-9分钟】
铺垫："这款产品在官方旗舰店的价格是{原价}，我今天给大家争取到了……"
停顿制造期待："直播间专属价——{到手价}！"
赠品说明："而且今天下单还送{赠品1}+{赠品2}，总价值{赠品价值}元！"
限时强调："这个价格只有今天直播间有，下播就恢复原价！"

【逼单话术 9-12分钟】
紧迫感："库存已经不多了，刚才已经有{数量}单了！"
社会认同："你们看，评论区已经有家人收到货了，说{好评内容}"
异议处理：
- 觉得贵："一天只要{日均价格}元，{使用年限}年都用，便宜到哭！"
- 怕质量差："我们提供{天}无理由退换，没有后顾之忧"
- 还在考虑："下单的已经在路上了，现在还没下单的快冲！"

【互动收尾 12-15分钟】
"今天的{产品}就到这里，下一个产品更好！请把直播间扩散出去……"
"新进来的家人刷一下屏，主播再给大家介绍一遍……"
```

## 模板5：小红书种草笔记（图文）

```
【标题】（20字内，含核心关键词+emoji）
示例：被安利了N次的XX，终于入手了！真实体验➡️

【开头（钩子段）】
用亲身经历或提问开头，制造共鸣：
"姐妹们！我真的找到宝藏了😭"
"反复被推荐之后，我终于入手了……"
"结论先说：这个真的值！"

【正文（2-4个核心段落）】
📌 外观/第一印象：
[具体描述，带真实感受]

✨ 使用体验：
[使用过程+效果，要具体，数据/时间/变化]

💰 性价比：
[价格+平台+购买渠道]

⚠️ 小缺点（可选，增加真实感）：
[如实说一个小不足，反而更可信]

【结尾引导】
"总结一下：{一句话种草}！如果你也在选择XX，这个真的可以考虑！"
"有问题评论区问我，看到都会回～❤️"

【标签】
#关键词1 #关键词2 #品类词 #场景词 #平台活动词
```
