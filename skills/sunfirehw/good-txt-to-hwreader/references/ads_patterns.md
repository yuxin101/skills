# 常见广告模式

## 网站水印类

```
更多精彩小说请访问 xxx.com
本文来自 xxx
首发 xxx
xxx 独家首发
转载请注明出处
```

## 推广信息类

```
本书由 xxx 整理发布
下载本书请到 xxx
温馨提示：xxx
xxx 推荐阅读
```

## 章节内广告

```
www.xxx.com
http://xxx.com
=== xxx ===
--- xxx ---
```

## 正则表达式模式

```python
# 网站水印
r'更多精彩小说.*?请访问.*?[\r\n]*'
r'本文来自.*?[\r\n]*'
r'首发.*?[\r\n]*'
r'==+.*?==+[\r\n]*'

# 推广信息
r'本书由.*?整理发布[\r\n]*'
r'下载.*?请到.*?[\r\n]*'
r'温馨提示.*?[\r\n]*'

# URL
r'www\.[a-z0-9]+\.com[^\r\n]*[\r\n]*'
r'http[s]?://[^\s<>"]+[\r\n]*'
```

## 注意事项

- 部分广告可能伪装成正文内容
- 注意不要误删作者信息
- 保留必要的版权声明
