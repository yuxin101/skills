# 常见乱码映射表

## 编码错误类型

### GBK → UTF-8 错误

| 乱码 | 正确 |
|------|------|
| 鏈枃 | 本文 |
| 銆€ | 空格 |
| 銆€銆€ | 空格 |
| 鈥斅 | —— |
| 鈥 | " |
| 鈥 | " |

## 特殊字符

| 乱码 | 正确 |
|------|------|
| 锘 | (BOM标记，删除) |
| 脗 | (编码错误，删除) |
| 涓 | 常见于UTF-8误读GBK |

## 处理策略

1. **优先检测编码** - 使用 chardet 自动检测
2. **批量替换** - 按映射表批量替换
3. **残留处理** - 使用 errors='replace' 处理无法识别的字符
4. **人工确认** - 复杂情况需要人工检查

## Python 实现

```python
MOJIBAKE_MAP = {
    '鏈枃': '本文',
    '銆€': ' ',
    '銆€銆€': ' ',
    '鈥斅': '——',
    # ... 更多映射
}

def fix_mojibake(text):
    for wrong, correct in MOJIBAKE_MAP.items():
        text = text.replace(wrong, correct)
    return text
```
