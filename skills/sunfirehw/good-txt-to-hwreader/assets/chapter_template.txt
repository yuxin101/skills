# 标准章节格式模板

## 章节标题格式

### 中文格式
```
第一章 章节名称

正文内容...


第二章 章节名称

正文内容...
```

### 数字格式
```
1. 章节名称

正文内容...


2. 章节名称

正文内容...
```

## 排版规范

1. **章节标题**
   - 独占一行
   - 前后各空一行
   - 不加特殊符号

2. **段落格式**
   - 段首缩进两个全角空格（　　）
   - 段落之间不空行
   - 每段长度适中

3. **对话格式**
   - 独立成段
   - 使用中文引号「」或""
   - 对话前后可适当空行

4. **标点符号**
   - 使用中文标点
   - 省略号使用……（六个点）
   - 破折号使用——（两个全角破折号）

## 示例

```
　　「这位少侠，可是来参加武林大会的？」身后传来一个清脆的声音。

　　李云飞转身，只见一位白衣少女正含笑看着他。那笑容如春风拂面，让他一时竟有些失神。

　　「正是。」他拱手行礼，「在下李云飞，敢问姑娘如何称呼？」

　　「我叫苏婉儿。」少女眨了眨眼，「看来我们是同路人呢。」

```
