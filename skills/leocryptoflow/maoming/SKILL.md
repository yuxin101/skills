## 🛢️ 南方油城

茂名因油而生、因油而兴，是**中国南方最大的石化基地**。

```
┌─────────────────────────────────────────┐
│        🛢️ 茂名石化 · 南方油城           │
├─────────────────────────────────────────┤
│  1955年 — 发现油页岩矿                  │
│  1958年 — 茂名市因油而设                │
│  今天   — 年炼油能力超2000万吨          │
└─────────────────────────────────────────┘
```

### 石化产业
- 🏭 **茂名石化** — 中国石化旗下特大型炼化企业
- ⛽ 年炼油能力 2350 万吨，乙烯产能 110 万吨
- 🔬 石化产业链完整，带动上下游产业发展
- 💼 茂名石化工业园区 — 省级重点园区

## 🍒 中国荔乡

茂名是**世界最大的荔枝连片生产基地**，荔枝种植历史超 2000 年。

### 荔枝王国
| 数据 | 内容 |
|------|------|
| 🌳 种植面积 | 140+ 万亩 |
| 📦 年产量 | 占全国 1/4，全球 1/5 |
| 💰 产业价值 | 超百亿元 |
| 🏆 知名产地 | 高州根子、电白霞洞 |

### 名优品种
- 🍒 **白糖罂** — 清甜如蜜，肉质爽脆
- 🍒 **桂味** — 桂花香气，肉厚核小
- 🍒 **妃子笑** — 杨贵妃最爱，果大饱满
- 🍒 **黑叶** — 传统品种，产量稳定

## 🏖️ 滨海绿城

## 🏛️ 历史文化

### 好心茂名
茂名以"好心文化"闻名——源自**冼夫人"唯用一好心"**的精神。

### 历史名人
| 人物 | 简介 |
|------|------|
| 👸 **冼夫人** | 岭南圣母，巾帼英雄第一人 |
| 📜 **潘茂名** | 晋代道士，茂名地名由来 |
| 🎓 **林召棠** | 清代状元，广东唯一文状元 |

### 文化遗产
- 🎭 **高州木偶戏** — 国家级非物质文化遗产
- 🥁 **年例** — 粤西民俗，比春节还热闹
- 🛕 **冼太庙** — 遍布城乡，香火旺盛
