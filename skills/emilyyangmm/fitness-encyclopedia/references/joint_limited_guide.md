## 关节受限类型

### 按原因分类

**1. 创伤性受限**
- 韧带损伤（如前交叉韧带损伤）
- 半月板损伤
- 骨折康复期
- 软组织损伤

**2. 退行性受限**
- 骨关节炎
- 类风湿性关节炎
- 颈椎病、腰椎病
- 肩周炎

**3. 结构性受限**
- 髋关节发育不良
- 膝关节外翻/内翻
- 脊柱侧弯
- 先天性关节问题

**4. 功能性受限**
- 肌肉紧张导致的活动受限
- 软组织粘连
- 慢性疼痛导致的保护性受限

### 按关节分类

**肩关节**
- 肩袖损伤（最常见的肩部问题）
- 肩峰撞击综合征
- 冻结肩（肩周炎）
- 脱位后不稳定

**肘关节**
- 网球肘（肱骨外上髁炎）
- 高尔夫球肘（肱骨内上髁炎）
- 尺神经卡压

**腕关节**
- 腕管综合征
- 腱鞘炎
- 三角纤维软骨损伤

**髋关节**
- 髋关节炎
- 髋臼撞击综合征
- 腹股沟综合征

**膝关节**
- 髌骨疼痛综合征
- 髌腱炎
- 半月板损伤
- 前交叉韧带损伤

**踝关节**
- 踝关节扭伤后遗症
- 跟腱炎
- 足底筋膜炎

**脊柱**
- 颈椎病
- 腰椎间盘突出
- 腰肌劳损

## 常见关节受限评估

### 评估流程

**第一步：询问病史**
1. 关节受限的持续时间
2. 受限原因（损伤/疾病/手术史）
3. 当前疼痛程度（0-10分）
4. 是否已就医诊断
5. 医生的治疗建议

**第二步：功能评估**
1. 关节活动范围（主动vs被动）
2. 疼痛触发动作
3. 力量测试
4. 稳定性测试

**第三步：分级判定**
- 轻度受限：活动范围减少<20%，疼痛轻微
- 中度受限：活动范围减少20-50%，疼痛中等
- 重度受限：活动范围减少>50%，疼痛严重

### 各关节具体评估

#### 肩关节评估

**关键问题**：
1. 能否双手举过头顶？
2. 能否在背后双手交叉？
3. 举手时是否有疼痛？在哪个角度？
4. 是否有过肩关节脱位？
5. 是否被诊断过肩袖损伤？

**测试动作**：
1. 肩关节外展：0°-180°
   - 正常：可达180°
   - 轻度受限：140°-180°
   - 中度受限：90°-140°
   - 重度受限：<90°

2. 肩关节外旋：
   - 正常：可达90°
   - 轻度受限：60°-90°
   - 中度受限：30°-60°
   - 重度受限：<30°

**受限判断**：
- 肩袖损伤：外展60-120°疼痛
- 肩峰撞击：外展90-120°疼痛
- 冻结肩：活动范围全面受限

#### 膝关节评估

**关键问题**：
1. 能否完全蹲下？
2. 上下楼梯是否有疼痛？
3. 膝盖是否有弹响？
4. 是否有过膝盖扭伤？
5. 膝盖是否容易打软腿？

**测试动作**：
1. 膝关节屈曲：
   - 正常：可达135°（脚跟贴臀部）
   - 轻度受限：110°-135°
   - 中度受限：90°-110°
   - 重度受限：<90°

2. 膝关节伸展：
   - 正常：0°（完全伸直）
   - 轻度受限：轻微屈曲0-10°
   - 中度受限：屈曲10-20°
   - 重度受限：屈曲>20°

**受限判断**：
- 髌骨疼痛综合征：下蹲、上下楼梯疼痛
- 髌腱炎：跳跃时膝盖下方疼痛
- 半月板损伤：特定角度疼痛

#### 腰椎评估

**关键问题**：
1. 能否弯腰触地？
2. 是否有腰痛？弯腰时加重还是减轻？
3. 是否有腿麻、腿痛？
4. 是否久坐后腰痛？
5. 是否被诊断过腰椎间盘突出？

**测试动作**：
1. 前屈测试：
   - 正常：手指可触地
   - 轻度受限：手指距地面10-20cm
   - 中度受限：手指距膝盖10-20cm
   - 重度受限：无法弯腰至膝盖

2. 直腿抬高测试（阳性提示坐骨神经问题）：
   - 正常：70°以上无痛
   - 轻度异常：40°-70°出现疼痛
   - 明显异常：<40°出现疼痛

**受限判断**：
- 腰椎间盘突出：弯腰疼痛、直腿抬高阳性
- 腰肌劳损：后仰时加重

## 受限情况下的训练调整

### 通用原则

**1. 避免疼痛动作**
- 不做引发疼痛的动作
- 疼痛是重要警示信号
- 微酸可接受，疼痛需停止

**2. 改变动作角度**
- 降低动作幅度
- 避开疼痛角度
- 在无痛范围内训练

**3. 降低训练强度**
- 降低配重至无痛范围
- 增加组数，降低单组强度
- 循序渐进提高

**4. 加强稳定肌群**
- 强化关节周围稳定肌群
- 提高关节稳定性
- 改善动作控制

### 各关节训练调整

#### 肩关节受限训练调整

**避免动作**：
- 大重量过顶推举
- 宽握下拉（肩关节外旋过大）
- 颈后推举

**推荐调整**：
1. 卧推：改为哑铃卧推，手掌内收15-30°
2. 推举：改为阿诺德推举或肩前推
3. 下拉：改为对握或窄握
4. 侧平举：降低至60-90°

**肩袖康复训练**：
- 外旋训练（弹力带或轻哑铃）
- 内旋训练
- 肩胛骨稳定训练

#### 膝关节受限训练调整

**避免动作**：
- 深蹲（改为部分幅度）
- 跳跃类动作
- 大重量腿举

**推荐调整**：
1. 深蹲：改为部分深蹲（只屈曲到无痛范围）
2. 腿举：降低配重，减少屈曲角度
3. 箭步蹲：缩短步幅
4. 跑步：改为快走或骑车

**膝关节康复训练**：
- 股四头肌等长收缩
- 直腿抬高
- 踝泵训练

#### 腰椎受限训练调整

**避免动作**：
- 大重量硬拉
- 负重弯腰
- 仰卧起坐（改为卷腹）
- 仰卧举腿

**推荐调整**：
1. 硬拉：改为罗马尼亚硬拉（腰椎保持中立）
2. 卷腹：改为死虫子、鸟狗式
3. 腿举：注意腰椎稳定

**腰椎康复训练**：
- 平板支撑（注意腰不下沉）
- 鸟狗式
- 死虫子
- 麦吉尔核心训练

## 动作替代方案

### 上肢替代方案

**受限动作 → 替代动作**

- 卧推 → 地板卧推（肩关节受限）
- 推举 → 前平举 + 侧平举（肩关节受限）
- 引体向上 → 高位下拉（肩关节受限）
- 弯举 → 锤式弯举（手腕受限）
- 臂屈伸 → 绳索下压（肘关节受限）

### 下肢替代方案

**受限动作 → 替代动作**

- 深蹲 → 部分深蹲/臀桥（膝关节受限）
- 硬拉 → 罗马尼亚硬拉（腰椎受限）
- 腿举 → 腿屈伸（膝关节受限）
- 跑步 → 快走/骑车（膝关节受限）
- 跳绳 → 踏步机/椭圆机（膝关节受限）

### 核心替代方案

**受限动作 → 替代动作**

- 卷腹 → 死虫子（腰椎受限）
- 仰卧举腿 → 腿举（腰椎受限）
- 俄罗斯转体 → 健腹轮滚动（腰椎受限）

## 康复建议

### 康复原则

1. **休息为主，训练为辅**：急性期休息，恢复期逐渐增加训练
2. **循序渐进**：从小强度、小幅度开始
3. **持续性**：康复训练需要长期坚持
4. **专业指导**：严重受限建议就医，接受专业康复指导

### 康复训练时间

**急性期（0-3天）**：
- 完全休息
- 冰敷（15-20分钟/次，3-4次/天）
- 轻柔活动

**亚急性期（3-14天）**：
- 开始无痛范围内的活动
- 等长收缩训练
- 关节活动度训练

**恢复期（14天-3个月）**：
- 逐渐增加训练强度
- 强化稳定肌群
- 恢复功能训练

### 就医建议

**以下情况建议就医**：
1. 关节受限严重（活动范围减少>50%）
2. 疼痛持续不缓解（>2周）
3. 关节肿胀、发红、发热
4. 关节不稳定、打软腿
5. 神经症状（麻木、无力）
6. 受伤后关节功能明显下降

## 数据来源与误差说明

**数据来源**：
- 运动医学教材
- 物理治疗指南
- 美国运动医学会（ACSM）指南
- 中国康复医学会指南

**误差说明**：
- 关节活动度评估存在个体差异（±10-15%）
- 自我评估可能与专业评估有偏差
- 本指南仅供参考，不能替代专业医疗诊断
- 严重受限请务必就医

**使用建议**：
- 本指南用于初步评估和训练参考
- 如有严重不适，立即停止训练
- 定期评估训练效果，动态调整计划
