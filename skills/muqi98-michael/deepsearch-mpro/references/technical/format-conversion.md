# 输出格式转换指引

## 概述

本技能默认生成 Markdown 格式报告。如需其他格式，AI 会自动识别并调用相应技能。

---

## Word 文档（.docx）

### 触发条件

- 用户明确要求"生成 Word 文档"
- 用户要求".docx 格式"
- 用户要求"可编辑文档"

### 调用方式

AI 自动识别用户需求，调用 **docx** 技能。

### 使用流程

1. AI 生成 Markdown 报告
2. AI 自动调用 docx 技能
3. docx 技能将 Markdown 转换为 Word 文档

### 用户无需操作

用户只需在请求中说明需要 Word 格式，AI 会自动完成转换。

### 示例请求

```
用户：帮我生成一份关于"Z世代护肤市场分析"的研究报告，需要Word格式

系统执行：
1. 生成分析框架
2. 执行数据收集
3. 生成 Markdown 报告
4. 自动调用 docx 技能生成 Word 文档

输出：z-gen-skincare-market-analysis.docx
```

### 注意事项

- Word 文档会保留 Markdown 的格式（标题、表格、列表等）
- 图片会以占位符形式呈现
- 参考文献链接会转换为可点击的超链接

---

## PowerPoint 演示文稿（.pptx）

### 触发条件

- 用户明确要求"生成 PPT"
- 用户要求"演示文稿"
- 用户要求".pptx 格式"
- 用户要求"汇报材料"

### 调用方式

AI 自动识别用户需求，调用 **pptx** 技能。

### 使用流程

1. AI 生成 Markdown 报告
2. AI 自动调用 pptx 技能
3. pptx 技能将报告内容转换为幻灯片

### 幻灯片结构建议

| 章节 | 建议幻灯片数 |
|------|-------------|
| 标题页 | 1 张 |
| 摘要 | 1 张 |
| 每个正文章节 | 3-5 张 |
| 结论 | 1-2 张 |
| 附录（可选） | 1-2 张 |

### 用户无需操作

用户只需在请求中说明需要 PPT 格式，AI 会自动完成转换。

### 示例请求

```
用户：生成一份新能源汽车行业研究报告，需要PPT格式

系统执行：
1. 生成分析框架
2. 执行数据收集
3. 生成 Markdown 报告
4. 自动调用 pptx 技能生成 PPT 演示文稿

输出：ev-industry-report.pptx
```

### 注意事项

- PPT 会自动提取报告中的关键数据和图表
- 每个 subsection 会生成 1-2 张幻灯片
- 表格会转换为幻灯片中的表格
- 标题和要点会自动提取为幻灯片标题和内容

## HTML 网页

### 触发条件

- 用户明确要求"生成网页版本"
- 用户要求"HTML 格式"
- 用户要求"在线查看"
- 用户要求"交互式报告"

### 调用方式

AI 自动识别用户需求，调用 **frontend-design** 或相关前端技能。

### 使用流程

1. AI 生成 Markdown 报告
2. AI 自动调用前端技能
3. 前端技能将 Markdown 转换为 HTML 网页

### 用户无需操作

用户只需在请求中说明需要 HTML 格式，AI 会自动完成转换。

### 示例请求

```
用户：生成一份市场分析报告，需要可以在网页上查看

系统执行：
1. 生成分析框架
2. 执行数据收集
3. 生成 Markdown 报告
4. 自动调用前端技能生成 HTML 网页

输出：market-analysis.html
```

### 注意事项

- HTML 网页支持响应式设计，可在不同设备上查看
- 图表会转换为可交互的 JavaScript 图表
- 表格会支持排序和筛选
- 支持目录导航

## 多格式输出

### 触发条件

用户同时要求多种格式，例如：
- "生成报告，需要 Word 和 PPT 格式"
- "生成 Markdown、Word 和 HTML 三种格式"

### 执行流程

1. AI 生成 Markdown 报告（基础格式）
2. AI 自动调用 docx 技能生成 Word 文档
3. AI 自动调用 pptx 技能生成 PPT 演示文稿
4. （可选）AI 自动调用前端技能生成 HTML 网页

### 示例请求

```
用户：生成一份Z世代护肤市场分析报告，需要Word和PPT格式

系统执行：
1. 生成分析框架
2. 执行数据收集
3. 生成 Markdown 报告
4. 自动调用 docx 技能生成 Word 文档
5. 自动调用 pptx 技能生成 PPT 演示文稿

输出：
- z-gen-skincare-market-analysis.md
- z-gen-skincare-market-analysis.docx
- z-gen-skincare-market-analysis.pptx
```

## 格式选择建议

| 使用场景 | 推荐格式 | 说明 |
|---------|---------|------|
| 个人阅读、标注 | Markdown | 轻量级，易于编辑 |
| 正式报告、提交客户 | Word | 专业格式，易于编辑 |
| 演示汇报 | PPT | 视觉化呈现，易于讲解 |
| 在线分享、协作 | HTML | 可交互，支持多设备 |

---

## 注意事项

### AI 自动识别

用户无需了解具体技能名称，只需在请求中说明格式需求，AI 会自动识别并调用相应技能。

### 格式转换质量

- **Word**：保留完整格式，支持编辑
- **PPT**：提取关键内容，视觉化呈现
- **HTML**：支持交互，适合在线查看

### 原始 Markdown 保留

无论生成哪种格式，原始 Markdown 报告都会保留，方便后续编辑和转换。

## 常见问题

### Q: 如何知道是否需要安装其他技能？

A: 用户只需安装本技能（consulting-analysis）。如果需要生成 Word/PPT/HTML 格式，AI 会自动调用相应的系统技能，用户无需额外安装。

### Q: 生成的 Word/PPT 能编辑吗？

A: 可以。生成的 Word 和 PPT 都是可编辑格式，用户可以根据需要修改内容。

### Q: 如何指定输出格式？

A: 只需在请求中说明，例如：
- "生成 Word 文档"
- "生成 PPT 演示文稿"
- "生成 HTML 网页"
- "生成 Markdown、Word 和 PPT 三种格式"

### Q: 图表会如何处理？

A:
- **Markdown**：图表以代码块形式呈现（Mermaid/Chart.js）
- **Word**：图表以图片形式插入
- **PPT**：图表以图片或可编辑对象形式插入
- **HTML**：图表转换为可交互的 JavaScript 图表
