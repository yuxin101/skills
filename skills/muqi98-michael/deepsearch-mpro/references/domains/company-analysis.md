# 企业竞争分析框架

## 适用场景

当研究主题涉及**单个企业的竞争分析**（如"腾讯竞争分析"、"用友公司分析"、"华为竞争力研究"、"金蝶分析"）时使用本框架。

**典型输入示例**：
- "分析腾讯的竞争优势"
- "用友网络公司深度研究"
- "华为企业业务竞争力分析"
- "金蝶 SaaS 厂商分析"

**与产品分析的区别**：
- 企业分析：分析整个公司的战略、业务组合、财务、组织
- 产品分析：聚焦单个产品的功能、定位、用户、竞争

---

## 核心分析模型

### 推荐框架组合

```
产品概览 → PESTEL → 商业模式画布 → 竞品矩阵 → SWOT → 产品发布与技术演进 → 客户案例 → 生态活动 → 近期动态
```

| 顺序 | 模型 | 分析目的 |
|------|------|---------|
| 1 | 产品概览 | 公司基本情况、业务范围、市场地位 |
| 2 | PESTEL | 宏观环境影响 |
| 3 | 商业模式画布 | 商业模式九大模块分析 |
| 4 | 竞品矩阵 | 与同业竞争对手多维度对比 |
| 5 | SWOT | 内部优劣势 + 外部机会威胁 |
| 6 | 产品发布与技术演进 | 版本更新、技术路线、功能创新 |
| 7 | 客户案例与行业覆盖 | 标杆客户、行业解决方案、客户成功 |
| 8 | 生态活动与合作伙伴 | 战略合作、ISV 生态、开发者社区 |
| 9 | 近期动态 | 近一周产品发布、市场活动、生态合作 |

---

## 分析维度

### 1. 公司概览

| 维度 | 关键信息 | 数据来源 |
|------|---------|---------|
| **基本信息** | 公司名称、成立时间、总部、员工规模 | 官网、天眼查、企查查 |
| **业务范围** | 主营业务、产品线、业务结构 | 官网、财报、业务介绍 |
| **市场地位** | 行业排名、市场份额、品牌价值 | 行业报告、第三方排名 |
| **财务概况** | 营收、利润、增长率、市值 | 财报、证券平台 |

---

### 2. 宏观环境影响（PESTEL）

| 维度 | 分析要点 |
|------|---------|
| **政策（P）** | 行业政策、监管环境、政府关系 |
| **经济（E）** | 宏观经济影响、汇率、通胀 |
| **社会（S）** | 消费习惯、人口结构、社会舆论 |
| **技术（T）** | 技术变革、研发投入、技术壁垒 |
| **环境（E）** | ESG要求、可持续发展 |
| **法律（L）** | 合规风险、知识产权、反垄断 |

---

### 3. 商业模式画布

| 模块 | 分析内容 | 关键问题 |
|------|---------|---------|
| **关键合作** | 战略伙伴、供应商、生态伙伴 | 核心合作伙伴是谁？依赖程度？ |
| **关键活动** | 核心业务、研发、运营 | 主要业务活动是什么？ |
| **关键资源** | 核心资产、技术、人才、品牌 | 核心竞争力来源？ |
| **价值主张** | 产品/服务的核心价值 | 为客户创造什么价值？ |
| **客户关系** | 服务模式、客户维系 | 如何与客户互动？ |
| **客户细分** | 目标客户、细分市场 | 服务哪些客户群体？ |
| **渠道通路** | 销售渠道、分销网络 | 如何触达客户？ |
| **成本结构** | 主要成本项、成本驱动 | 成本主要花在哪里？ |
| **收入来源** | 收入模式、定价策略 | 如何赚钱？收入结构？ |

---

### 4. 竞品对比矩阵

| 对比维度 | 分析内容 |
|---------|---------|
| **业务规模** | 营收、市值、员工规模、市场份额 |
| **产品能力** | 产品矩阵、技术能力、创新能力 |
| **市场地位** | 行业排名、品牌价值、客户口碑 |
| **财务表现** | 增长率、盈利能力、现金流 |
| **竞争优势** | 核心竞争力、护城河、差异化 |
| **战略动向** | 近期战略、投资并购、组织调整 |

**竞品选择原则**：
- 直接竞争对手（同业务、同市场）
- 间接竞争对手（替代产品、跨界玩家）
- 国际对标企业（行业标杆）

---

### 5. SWOT 分析

| 维度 | 分析内容 |
|------|---------|
| **优势（S）** | 核心竞争力、技术壁垒、品牌优势、渠道优势、规模优势 |
| **劣势（W）** | 资源短板、能力不足、业务依赖、组织问题 |
| **机会（O）** | 市场增长、政策利好、技术变革、消费升级 |
| **威胁（T）** | 竞争加剧、政策风险、技术替代、市场萎缩 |

---

### 6. 产品发布与技术演进

| 分析点 | 关键问题 | 数据来源 |
|--------|---------|---------|
| **版本更新** | 最近 1 年内的重大版本发布？功能迭代方向？ | 官网更新日志、发布会、技术博客 |
| **技术路线** | 技术架构演进？云原生/AI 化进展？ | 技术白皮书、CTO 分享、技术大会 |
| **功能创新** | 核心功能创新？差异化功能？ | 产品对比、用户评价、产品演示 |
| **产品定位** | 目标客户？市场定位？竞争策略？ | 官网定位、市场材料、销售话术 |

---

### 7. 客户案例与行业覆盖

| 分析点 | 关键问题 | 数据来源 |
|--------|---------|---------|
| **标杆客户** | 重点客户案例？客户规模？行业分布？ | 官网案例、客户故事、销售材料 |
| **行业覆盖** | 覆盖哪些行业？行业解决方案？ | 官网行业页面、解决方案手册 |
| **客户成功** | 客户成功案例？ROI 数据？复购率？ | 客户证言、案例研究、第三方评测 |
| **客户结构** | 大客户 vs 中小客户比例？客户流失率？ | 财报、分析师报告、销售数据 |

---

### 8. 生态活动与合作伙伴

| 分析点 | 关键问题 | 数据来源 |
|--------|---------|---------|
| **合作伙伴** | 战略合作伙伴？ISV 生态？实施伙伴？ | 官网合作伙伴页面、生态大会 |
| **生态活动** | 生态大会、开发者大会、伙伴大会？ | 官网活动页面、活动报道 |
| **平台开放** | API 开放度？开发者支持？应用市场？ | 开发者文档、API 平台、应用市场 |
| **生态收入** | 生态伙伴贡献收入？分成模式？ | 财报、生态大会演讲 |

---

### 9. 公司近期动态（近一周）

| 序号 | 动态类型 | 动态内容 | 来源 |
|------|---------|---------|------|
| 1 | 产品发布/市场活动/生态合作/投资并购 | 具体事件描述 | 新闻链接 |
| 2 | ... | ... | ... |
| ... | ... | ... | ... |
| 10 | ... | ... | ... |

**动态类型说明**：
- **产品发布**：新品发布、功能更新、版本迭代
- **市场活动**：发布会、展会、营销活动、客户活动
- **生态合作**：战略合作、渠道合作、技术合作、投资并购
- **投资并购**：战略投资、并购、融资

**数据收集方法**：
```markdown
# 近期动态搜索
web_search(query="{公司名} 产品发布 2025", max_results=5)
web_search(query="{公司名} 战略合作 签约 2025", max_results=5)
web_search(query="{公司名} 发布会 活动 2025", max_results=5)
web_search(query="{公司名} 投资 并购 2025", max_results=5)
```
