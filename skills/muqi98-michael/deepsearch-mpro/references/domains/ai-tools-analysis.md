## 核心分析维度

### 通用分析维度

| 维度 | 关键指标 | 数据来源 |
|------|---------|---------|
| **产品功能** | 核心功能、差异化功能、创新点、用户价值 | 官网、产品演示、用户评测 |
| **用户体验** | 易用性、学习曲线、界面设计、响应速度 | 用户评价、Product Hunt、社交媒体 |
| **技术能力** | 生成质量、生成速度、多模态能力、定制化 | 技术评测、第三方对比、官方发布 |
| **商业模式** | 定价策略、免费额度、订阅制、企业版 | 官网定价、用户反馈、销售材料 |
| **市场表现** | 用户数量、增长率、市场份额、收入规模 | 财报、行业报告、第三方数据 |
| **生态整合** | 与其他工具集成、平台生态、开放 API | 开发者文档、合作伙伴、应用市场 |

### 文生图专项维度

- **生成质量**：分辨率、细节表现、风格多样性、提示词理解能力
- **生成速度**：生成时间、并发能力、排队时间
- **易用性**：界面设计、提示词建议、编辑功能、批量生成

### 文生 PPT 专项维度

- **生成能力**：内容生成质量、设计美学、模板多样性、自定义能力
- **整合能力**：与办公软件集成、素材库、品牌模板、团队协作

### 文生视频专项维度

- **生成能力**：视频时长、分辨率、帧率、动作连贯性、多模态能力
- **应用场景**：营销视频、教育内容、社交媒体、企业宣传
