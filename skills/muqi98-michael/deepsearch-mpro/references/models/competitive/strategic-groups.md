# 战略群组图

## 模型概述

战略群组图用于可视化竞争格局，识别市场空白和定位机会。

---

## 分析维度

| 维度 | 说明 | 应用场景 |
|------|------|---------|
| **价格维度** | 高/中/低价定位 | 价格定位分析 |
| **质量维度** | 高/中/低质量 | 品质定位分析 |
| **定位维度** | 奢华/大众/小众 | 市场定位策略 |
| **覆盖维度** | 全球/区域/本地 | 市场覆盖分析 |

---

## 分析框架

```
                        价格高
                          │
         ┌────────────────┼────────────────┐
         │    奢侈群组    │    高端群组    │
         │  ────────────  │  ────────────  │
         │  · 品牌A       │  · 品牌D       │
    覆   │  · 品牌B       │  · 品牌E       │   覆
    盖   │               │               │   盖
    广   ├────────────────┼────────────────┤   窄
         │    大众群组    │    小众群组    │
         │  ────────────  │  ────────────  │
         │  · 品牌C       │  · 品牌F       │
         │  · 品牌G       │  · 品牌H       │
         └────────────────┼────────────────┘
                          │
                        价格低
```

---

## 分析要点

| 分析方向 | 关键问题 |
|---------|---------|
| 群组识别 | 哪些企业属于同一战略群组？ |
| 移动壁垒 | 群组之间移动的壁垒是什么？ |
| 市场空白 | 图中是否有空白区域？ |
| 定位机会 | 企业应该定位在哪个群组？ |

---

## 数据收集要点

| 维度 | 关键指标 | 数据来源 |
|------|---------|---------|
| 价格定位 | 价格区间、定价策略 | 定价页、销售材料 |
| 品质定位 | 质量评级、用户评价 | 用户评价、第三方评测 |
| 市场覆盖 | 区域覆盖、渠道覆盖 | 官网、渠道数据 |
