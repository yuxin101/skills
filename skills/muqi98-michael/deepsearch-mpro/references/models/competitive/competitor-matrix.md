# 竞品对比矩阵（Competitive Matrix）

## 模型概述

竞品对比矩阵用于多维度对比分析竞争对手，直观展示各产品的竞争力差异。

---

## 分析维度

| 维度 | 说明 | 分析要点 |
|------|------|---------|
| **产品功能** | 核心功能对比、差异化功能 | 功能覆盖度、功能深度、差异化功能 |
| **技术能力** | 技术优势、技术指标对比 | 技术架构、性能指标、技术创新 |
| **市场定位** | 目标客户、价格定位、品牌定位 | 目标客户群体、价格区间、品牌形象 |
| **商业模式** | 收入模式、定价策略、渠道策略 | 收入来源、定价模式、渠道结构 |
| **生态整合** | 平台生态、合作伙伴、开放能力 | 生态伙伴、API开放度、平台整合 |
| **市场表现** | 市场份额、用户规模、增长率 | 市场份额、用户数量、增长趋势 |

---

## 最适用场景

- 竞争分析
- 产品定位
- 战略规划
- 差异化策略
- 市场进入决策

---

## 强度表示

| 星级 | 说明 |
|------|------|
| ★★★★★ | 极强/领先 |
| ★★★★☆ | 强/优秀 |
| ★★★☆☆ | 中等/一般 |
| ★★☆☆☆ | 弱/不足 |
| ★☆☆☆☆ | 极弱/缺失 |

---

## 数据收集要点

| 维度 | 关键指标 | 数据来源 |
|------|---------|---------|
| **产品功能** | 功能列表、功能对比 | 官网、产品文档、产品演示 |
| **技术能力** | 技术架构、性能指标 | 技术白皮书、技术评测、技术博客 |
| **市场定位** | 目标客户、价格区间 | 官网定位、定价页、销售材料 |
| **商业模式** | 收入模式、定价策略 | 官网定价、财报、销售材料 |
| **生态整合** | 合作伙伴、API开放 | 开发者文档、合作伙伴页面 |
| **市场表现** | 市场份额、用户数量 | 行业报告、财报、第三方数据 |
