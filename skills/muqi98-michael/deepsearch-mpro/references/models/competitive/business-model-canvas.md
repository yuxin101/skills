# 商业模式画布（Business Model Canvas）

## 模型概述

商业模式画布是描述、分析和设计商业模式的可视化工具，包含九大核心模块。

---

## 分析维度

| 模块 | 说明 | 分析要点 |
|------|------|---------|
| **关键合作** | 战略合作伙伴、供应商、生态伙伴 | 合作伙伴是谁？依赖程度？合作模式？ |
| **关键活动** | 核心业务活动、研发、运营 | 主要业务活动是什么？核心流程？ |
| **关键资源** | 核心资产、技术、人才、品牌 | 核心竞争力来源？关键资源是什么？ |
| **价值主张** | 产品/服务的核心价值 | 为客户创造什么价值？差异化优势？ |
| **客户关系** | 服务模式、客户维系 | 如何与客户互动？服务模式？ |
| **客户细分** | 目标客户群体、细分市场 | 目标客户是谁？细分市场？ |
| **渠道通路** | 销售渠道、分销网络 | 如何触达客户？渠道结构？ |
| **成本结构** | 主要成本项、成本驱动因素 | 成本主要花在哪里？成本结构？ |
| **收入来源** | 收入模式、定价策略 | 如何赚钱？收入结构？定价策略？ |

---

## 最适用场景

- 商业模式设计
- 竞争分析
- 企业诊断
- 产品规划
- 战略转型

## 数据收集要点

| 模块 | 关键问题 | 数据来源 |
|------|---------|---------|
| **关键合作** | 核心合作伙伴？依赖程度？ | 官网、财报、新闻、生态大会 |
| **关键活动** | 核心业务？关键流程？ | 产品文档、官网、财报 |
| **关键资源** | 核心资源？技术？人才？ | 技术白皮书、团队介绍、专利 |
| **价值主张** | 核心价值？差异化？ | 官网、产品介绍、客户案例 |
| **客户关系** | 服务模式？客户成功？ | 客户服务、客户成功团队 |
| **客户细分** | 目标客户？细分市场？ | 官网定位、客户案例、市场调研 |
| **渠道通路** | 销售渠道？分销网络？ | 官网、销售材料、渠道伙伴 |
| **成本结构** | 主要成本？成本结构？ | 财报、成本分析、行业对比 |
| **收入来源** | 收入模式？定价策略？ | 官网定价、财报、销售材料 |
