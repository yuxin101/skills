# Benchmarking（基准对标）

## 模型概述

基准对标是通过与行业最佳实践对比，识别差距和改进机会的分析方法。

---

## 分析维度

| 类型 | 说明 | 应用场景 |
|------|------|---------|
| **内部对标** | 企业内部不同部门对比 | 最佳实践识别、内部优化 |
| **竞争对标** | 与直接竞争对手对比 | 竞品差距分析、竞争策略 |
| **功能对标** | 与行业最佳实践对比 | 流程优化、能力提升 |
| **通用对标** | 与跨行业最佳实践对比 | 创新启发、标杆学习 |

---

```
┌─────────────────────────────────────────────────────────────┐

│  ─────────  │  ─────────  │  ─────────  │   ─────────      │

│  · 对标指标  │  · 数据收集  │  · 优先级    │   · 实施跟踪     │
└─────────────┴─────────────┴─────────────┴──────────────────┘
```

---

## 数据收集要点

| 维度 | 关键问题 | 数据来源 |
|------|---------|---------|
| 内部对比 | 内部最佳实践是什么？ | 内部数据、部门报告 |
| 竞争对比 | 竞争对手优势在哪？ | 竞品分析、行业报告 |
| 行业对比 | 行业标杆是谁？差距？ | 行业报告、研究机构 |
| 跨行业对比 | 跨行业创新启发？ | 案例研究、标杆企业 |
