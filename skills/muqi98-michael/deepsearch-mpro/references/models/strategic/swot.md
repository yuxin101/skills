# SWOT 分析

## 模型概述

SWOT 分析是评估企业内外部环境的经典工具，用于识别优势、劣势、机会和威胁。

---

## 分析维度

| 维度 | 说明 | 分析要点 |
|------|------|---------|
| **Strengths（优势）** | 内部优势资源与能力 | 品牌优势、技术壁垒、渠道优势、规模优势、人才优势、资金优势 |
| **Weaknesses（劣势）** | 内部劣势与不足 | 资源短板、能力不足、业务依赖、组织问题、成本劣势 |
| **Opportunities（机会）** | 外部市场机会 | 市场增长、政策利好、技术变革、消费升级、竞争格局变化 |
| **Threats（威胁）** | 外部环境威胁 | 竞争加剧、政策风险、技术替代、市场萎缩、成本上升 |

---

## 最适用场景

- 品牌评估
- 竞争定位
- 战略规划
- 资源配置
- 风险评估

---

## 分析框架

```
                内部因素                    外部因素
              ┌─────────────┐              ┌─────────────┐
    积极因素   │  优势 (S)    │              │  机会 (O)    │
              │ · 核心竞争力  │              │ · 市场机会   │
              │ · 资源优势    │              │ · 政策利好   │
              │ · 品牌优势    │              │ · 技术红利   │
              └─────────────┘              └─────────────┘
              
              ┌─────────────┐              ┌─────────────┐
    消极因素   │  劣势 (W)    │              │  威胁 (T)    │
              │ · 资源短板    │              │ · 竞争压力   │
              │ · 能力不足    │              │ · 政策风险   │
              │ · 成本劣势    │              │ · 市场萎缩   │
              └─────────────┘              └─────────────┘
```

---

## 战略组合

| 组合策略 | 说明 |
|---------|------|
| **SO 战略**（优势-机会）| 利用优势抓住机会 |
| **WO 战略**（劣势-机会）| 通过改进抓住机会 |
| **ST 战略**（优势-威胁）| 利用优势规避威胁 |
| **WT 战略**（劣势-威胁）| 减少劣势规避威胁 |

---

## 数据收集要点

| 维度 | 关键问题 | 数据来源 |
|------|---------|---------|
| **S** | 核心竞争力是什么？护城河在哪？ | 公司官网、财报、分析师报告 |
| **W** | 短板在哪？客户抱怨什么？ | 用户评价、财报、行业报告 |
| **O** | 市场有哪些增长机会？政策利好？ | 行业报告、政策文件、市场数据 |
| **T** | 竞争对手动态？政策风险？ | 新闻、竞争对手财报、行业分析 |
