# EVA（经济增加值）

## 模型概述

EVA 用于评估企业创造的真实价值，考虑资本成本后的实际利润。

---

## 分析维度

| 公式 | 说明 |
|------|------|
| **EVA = 税后营业利润 - 资本成本** | 价值创造能力评估 |
| **资本成本 = 资本总额 × WACC** | 资本使用成本 |

---

## 分析框架

```
┌─────────────────────────────────────────────────────────────┐
│                      EVA 计算                               │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│   EVA = NOPAT - 资本成本                                    │
│        = NOPAT - (资本总额 × WACC)                          │
│                                                             │
│   其中：                                                    │
│   · NOPAT = 税后营业利润                                    │
│   · WACC = 加权平均资本成本                                 │
│   · 资本总额 = 债务 + 股权                                  │
│                                                             │
├─────────────────────────────────────────────────────────────┤
│   EVA > 0  → 创造价值                                       │
│   EVA = 0  → 刚好覆盖资本成本                               │
│   EVA < 0  → 损毁价值                                       │
└─────────────────────────────────────────────────────────────┘
```

---

## EVA 指标解读

| EVA 值 | 说明 | 管理建议 |
|--------|------|---------|
| EVA > 0 | 创造价值 | 继续投资、扩大规模 |
| EVA = 0 | 刚好覆盖成本 | 优化效率、提升利润率 |
| EVA < 0 | 损毁价值 | 剥离资产、改善经营 |

---

## 数据收集要点

| 维度 | 关键指标 | 数据来源 |
|------|---------|---------|
| NOPAT | 税后营业利润 | 利润表、财务调整 |
| 资本总额 | 债务 + 股权 | 资产负债表 |
| WACC | 加权平均资本成本 | 市场数据、行业研究 |
