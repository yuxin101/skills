# 杜邦分析

## 模型概述

杜邦分析是财务分析的核心工具，将 ROE 拆解为净利率、资产周转率和权益乘数三个维度。

---

## 分析维度

| 维度 | 公式 | 说明 | 分析要点 |
|------|------|------|---------|
| **净利率** | 净利润/营收 | 盈利能力 | 成本控制、定价能力、毛利率 |
| **资产周转率** | 营收/总资产 | 运营效率 | 资产利用效率、库存管理、应收账款 |
| **权益乘数** | 总资产/股东权益 | 财务杠杆 | 负债水平、资本结构、财务风险 |

---

## 最适用场景

- 盈利能力拆解
- 财务健康诊断
- 企业对比分析
- 投资决策

---

## 分析框架

```
┌─────────────────────────────────────────────────────────────────────────────┐
│                              ROE（净资产收益率）                             │
│                           = 净利率 × 资产周转率 × 权益乘数                    │
├─────────────────────────┬─────────────────────────┬─────────────────────────┤
│        净利率           │       资产周转率         │        权益乘数         │
│    ───────────          │      ───────────        │      ───────────        │
│    净利润/营收          │     营收/总资产          │    总资产/股东权益       │
│         │               │          │              │          │             │
│    ┌────┴────┐          │     ┌────┴────┐         │     ┌────┴────┐        │
│    │         │          │     │         │         │     │         │        │
│  毛利率   费用率         │   流动资产周转  │       │   负债率   权益率      │
│                         │   固定资产周转  │       │                         │
└─────────────────────────┴─────────────────────────┴─────────────────────────┘
```

---

## 杜邦公式

```
ROE = 净利率 × 资产周转率 × 权益乘数

其中：
· 净利率 = 净利润 ÷ 营业收入
· 资产周转率 = 营业收入 ÷ 总资产
· 权益乘数 = 总资产 ÷ 股东权益
```

---

## 各维度分析要点

| 维度 | 高ROE来源 | 低ROE原因 |
|------|----------|----------|
| **净利率高** | 成本控制好、定价能力强、高毛利业务 | 成本高、价格战、低毛利业务 |
| **周转率高** | 资产利用效率高、轻资产模式 | 资产闲置、库存积压、应收账款多 |
| **权益乘数高** | 高杠杆、负债经营 | 低杠杆、保守财务 |

---

## 数据收集要点

| 维度 | 关键指标 | 数据来源 |
|------|---------|---------|
| **净利率** | 毛利率、净利率、费用率 | 利润表、财报 |
| **周转率** | 存货周转率、应收账款周转率 | 资产负债表、财报 |
| **权益乘数** | 资产负债率、权益乘数 | 资产负债表、财报 |
