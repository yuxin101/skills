# DCF 估值（折现现金流）

## 模型概述

DCF 估值通过预测未来现金流并折现至现值来评估企业或项目的价值。

---

## 分析维度

| 步骤 | 说明 | 分析要点 |
|------|------|---------|
| **预测自由现金流** | 预测未来现金流 | 收入增长、利润率、资本支出 |
| **确定折现率** | 计算加权平均资本成本(WACC) | 股权成本、债务成本、资本结构 |
| **计算终值** | 预测期后的价值 | 永续增长率、退出倍数 |
| **折现求和** | 将未来现金流折现至现值 | 现值计算、企业价值 |

---

## 计算公式

```
企业价值 = Σ(FCFt / (1+WACC)^t) + TV / (1+WACC)^n

其中：
· FCFt = 第t期自由现金流
· WACC = 加权平均资本成本
· TV = 终值（Terminal Value）
· n = 预测期数
```

---

## 关键假设

| 假设项 | 说明 | 敏感性 |
|--------|------|--------|
| 收入增长率 | 未来收入增长假设 | 高 |
| 利润率 | 未来利润率假设 | 高 |
| 折现率(WACC) | 资本成本 | 极高 |
| 永续增长率 | 终值计算假设 | 中 |

---

## 数据收集要点

| 维度 | 关键指标 | 数据来源 |
|------|---------|---------|
| 现金流预测 | 收入、利润、资本支出 | 财报、预测模型 |
| 折现率 | 无风险利率、β值、风险溢价 | 市场数据、行业研究 |
| 终值 | 永续增长率、退出倍数 | 行业数据、可比交易 |
