# Jobs-to-be-Done (JTBD)

## 模型概述

JTBD 模型用于洞察用户真实需求，理解用户"雇佣"产品要完成的任务。

---

## 分析维度

| 维度 | 说明 | 分析要点 |
|------|------|---------|
| **功能任务** | 用户要解决的具体问题 | 用户想完成什么？核心功能需求？ |
| **情感任务** | 用户的情感需求 | 用户想获得什么感受？情感价值？ |
| **社交任务** | 用户在社交中的需求 | 用户想在社交中获得什么？社交价值？ |

---

## 分析框架

```
┌─────────────────────────────────────────────────────────────┐
│                    Jobs-to-be-Done                         │
├─────────────────┬─────────────────┬─────────────────────────┤
│    功能任务      │    情感任务      │       社交任务          │
│    ─────────    │    ─────────    │       ─────────        │
│  · 解决什么问题  │  · 获得什么感受  │     · 社交价值          │
│  · 完成什么目标  │  · 情感满足      │     · 社交认同          │
│  · 功能需求     │  · 心理需求      │     · 社交展示          │
└─────────────────┴─────────────────┴─────────────────────────┘
```

---

## 核心问题

| 维度 | 关键问题 |
|------|---------|
| **场景** | 用户在什么情境下使用？ |
| **动机** | 用户为什么需要这个？ |
| **期望** | 用户期望获得什么结果？ |
| **阻碍** | 阻碍用户完成任务的障碍？ |
| **替代** | 用户目前如何解决这个问题？ |

---

## 数据收集要点

| 维度 | 关键问题 | 数据来源 |
|------|---------|---------|
| 功能任务 | 用户想完成什么？ | 用户访谈、行为分析 |
| 情感任务 | 用户情感诉求？ | 用户调研、情感分析 |
| 社交任务 | 社交需求？ | 社交分析、用户反馈 |
