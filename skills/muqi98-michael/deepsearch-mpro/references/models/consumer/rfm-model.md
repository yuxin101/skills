# RFM 模型

## 模型概述

RFM 模型用于客户价值分层，根据最近消费时间、消费频率、消费金额三个维度细分客户。

---

## 分析维度

| 维度 | 英文 | 说明 | 分析要点 |
|------|------|------|---------|
| **R** | Recency | 最近一次消费时间 | 距上次消费时间越近，越可能复购 |
| **F** | Frequency | 消费频率 | 消费频率越高，忠诚度越高 |
| **M** | Monetary | 消费金额 | 消费金额越高，价值越大 |

---

## 客户分层

| 客户类型 | RFM 特征 | 价值等级 | 营销策略 |
|---------|---------|---------|---------|
| **重要价值客户** | R高 F高 M高 | ★★★★★ | VIP服务、专属优惠 |
| **重要发展客户** | R高 F低 M高 | ★★★★☆ | 提升消费频率 |
| **重要保持客户** | R低 F高 M高 | ★★★☆☆ | 唤醒激活、防止流失 |
| **重要挽留客户** | R低 F低 M高 | ★★☆☆☆ | 重点挽留、个性化服务 |
| **一般价值客户** | R高 F高 M低 | ★★★☆☆ | 提升客单价 |
| **一般发展客户** | R高 F低 M低 | ★★☆☆☆ | 培养习惯、引导消费 |
| **一般保持客户** | R低 F高 M低 | ★☆☆☆☆ | 维持关系、适度激活 |
| **低价值客户** | R低 F低 M低 | ☆☆☆☆☆ | 低成本维护 |

---

## 分析框架

```
                    F 频率高
                       │
         ┌─────────────┼─────────────┐
         │  重要保持    │  重要价值    │
    M    │  R低 F高 M高 │  R高 F高 M高 │
    金   ├─────────────┼─────────────┤    M
    额   │  一般保持    │  一般价值    │    金
    高   │  R低 F高 M低 │  R高 F高 M低 │    额
         └─────────────┼─────────────┘    低
                       │
                    F 频率低
```

---

## 数据收集要点

| 维度 | 关键指标 | 数据来源 |
|------|---------|---------|
| R | 最近消费日期 | 交易数据、CRM系统 |
| F | 消费次数（时间段内） | 交易数据、CRM系统 |
| M | 消费金额（时间段内） | 交易数据、财务数据 |
