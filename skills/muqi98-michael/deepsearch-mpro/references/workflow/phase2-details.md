# 阶段 2：报告生成 - 详细流程

## 目的

接收分析框架和数据包，将其整合为最终的咨询级 Markdown 报告。

---

## 输入

| 输入项 | 说明 | 是否必需 |
|-------|------|---------|
| **Analysis Framework** | 阶段 1 生成的框架文档 | 是 |
| **Data Package** | 数据收集阶段整理的数据包 | 是 |
| **Chart Files** | 已生成图表的本地文件路径（可选） | 否 |

---

## 详细工作流程

### Step 2.1：接收并校验输入

验证所有必需输入是否齐全：

1. **Analysis Framework** —— 确认其包含章节骨架、数据需求和可视化方案
2. **Data Package** —— 确认其按章节组织，并可与 P0 需求交叉核对

**校验清单**：

```markdown
- [ ] 分析框架文档存在且格式正确
- [ ] 数据包包含所有 P0 数据
- [ ] 数据来源已记录（URL、标题、日期）
- [ ] 缺失数据已标记
```

**缺失数据处理**：

如果缺少任何 P0 数据：
1. 尝试补充搜索（使用备用关键词和来源）
2. 如无法获取，在报告中明确指出"数据暂不可得"
3. 标注数据缺失对分析结论的影响

---

### Step 2.2：映射报告结构

根据分析框架映射最终报告结构：

1. **Abstract（摘要）** —— 高层结论与关键发现
2. **Introduction（引言）** —— 背景、目标、方法论
3. **Main Body Chapters（正文）** —— 映射自框架中的章节骨架
4. **Conclusion（结论）** —— 纯客观综合判断
5. **References（参考文献）** —— 按 **GB/T 7714-2015** 格式列出

**报告结构模板**：

```markdown
# [报告标题]

## Abstract
[高层摘要与关键结论，3-5 句话，200-300 字]

## 1. Introduction
### 1.1 研究背景
### 1.2 研究目标
### 1.3 研究方法

## 2. [正文章节1标题]
### 2.1 [小节标题]
### 2.2 [小节标题]
...

## 3. [正文章节2标题]
...

## N. Conclusion
[纯客观综合判断，不使用 bullet points]

## N+1. References
[1] Author. Title[EB/OL]. URL, Date.
[2] ...
```

### Step 2.3：撰写报告

对每个小节，遵循 **"视觉锚点 → 数据对比 → 综合分析"** 的流程：

#### 2.3.1 视觉锚点（数据展示）

**数据对比表**：为关键指标创建 Markdown 对比表

```markdown
| Metric | Brand A | Brand B | Brand C |
|--------|---------|---------|---------|
| 市场份额 | 25% | 20% | 15% |
| 增长率 | 18% | 12% | 22% |
| 价格指数 | 高 | 中 | 低 |
```

**图表引用**（如果有）：

```markdown
![图表标题](./charts/chart-1.png)
```

#### 2.3.2 数据对比（多维度对比）

**对比维度**：

1. **时间对比**：同比、环比变化
2. **空间对比**：不同地区、市场对比
3. **竞争对比**：与竞争对手对比
4. **基准对比**：与行业基准或目标对比

**对比分析模板**：

```markdown
**时间维度**：
- 2024 年市场规模达到 **3,450 亿元**，同比增长 **12%**
- 过去 5 年复合增长率（CAGR）为 **15%**，显示市场持续扩张

**竞争维度**：
- 品牌A市场份额 **25%**，领先第二名 5 个百分点
- 但增长率落后于品牌B（12% vs 18%），显示增长动能不足

**基准对比**：
- 行业平均增长率为 **10%**，品牌B增长率高于行业平均 8 个百分点
```

#### 2.3.3 综合叙述分析

按 **"What → Why → So What"** 写出分析文字：

**What（现象描述）**：
- 描述数据反映的现象
- 使用具体数字和百分比
- 突出关键发现

**Why（原因分析）**：
- 分析现象背后的原因
- 引用专家观点或行业分析
- 考虑多因素影响

**So What（战略启示）**：
- 揭示对业务的启示
- 提供决策建议
- 指出潜在风险或机会

**示例**：

```markdown
中国护肤品市场在 2024 年达到 **3,450 亿元**，同比增长 **12%**，远超全球平均水平（5%）。这一强劲增长主要得益于三个因素：（1）Z 世代成为核心消费群体，其护肤意识更强、消费频次更高；（2）功效护肤理念普及，消费者愿意为专业成分买单；（3）直播电商和社交平台推动，降低品牌触达成本。

然而，市场增长的结构性分化明显。高端市场（单价>500 元）增长 **18%**，而大众市场增长仅 **8%**。这表明消费升级趋势持续，品牌需重新审视产品定位和价格策略。建议国货品牌加大高端产品线布局，通过成分创新和专业背书提升溢价能力，避免陷入低价竞争泥潭。

> **战略真相**：护肤品市场进入"品质驱动"阶段，成分创新和专业背书成为品牌突围关键。
```

**每个小节必须**：
- 以一个充分的分析段落结尾（至少 200 字）
- 包含具体数据和分析
- 提供明确的洞察或建议

---

### Step 2.4：撰写摘要和结论

#### 2.4.1 摘要（Abstract）

**撰写原则**：
- 3-5 句话，200-300 字
- 包含关键发现和核心结论
- 避免细节和数据堆砌
- 突出报告价值

**摘要模板**：

```markdown
## Abstract

本报告深入分析 [研究主题]，基于 [数据来源和研究方法]，揭示了 [核心发现1]、[核心发现2] 和 [核心发现3]。研究发现，[关键洞察]。报告建议 [核心建议]。本报告为 [目标受众] 提供 [决策支持/战略参考]。
```

**示例**：

```markdown
## Abstract

本报告深入分析中国护肤品市场 2024 年发展态势，基于行业报告、企业财报、市场调研数据，揭示了市场规模、竞争格局、消费趋势三大维度。研究发现，中国护肤品市场规模达到 3,450 亿元，同比增长 12%，功效护肤和成分透明度成为核心消费驱动力。国际品牌主导高端市场，但国货品牌在大众市场快速崛起。建议品牌聚焦成分创新、强化专业背书、布局高端产品线，以抓住消费升级红利。
```

#### 2.4.2 结论（Conclusion）

**撰写原则**：
- 纯客观综合判断，不使用 bullet points
- 整合各章节的核心发现
- 强调战略意义和未来展望
- 避免引入新数据或新观点

**结论模板**：

```markdown
## Conclusion

综合以上分析，[研究主题] 呈现出 [核心特征1]、[核心特征2] 和 [核心特征3] 的态势。[关键判断]。未来，[发展趋势]。对于 [目标受众] 而言，[战略建议]。
```

**示例**：

```markdown
## Conclusion

综合以上分析，中国护肤品市场在 2024 年呈现出稳健增长、竞争加剧、消费升级三大特征。市场规模突破 3,450 亿元，功效护肤成为核心增长引擎，国际品牌与国货品牌的竞争格局正在重塑。未来，随着 Z 世代成为主流消费群体、成分透明度要求提升、监管政策趋严，市场将进入"品质驱动"的新阶段。对于护肤品品牌而言，成分创新、专业背书、渠道多元化将成为核心竞争力，而单纯依靠营销驱动的增长模式将难以为继。
```

### Step 2.5：整理参考文献

按 **GB/T 7714-2015** 格式列出所有参考文献：

**格式规范**：

```markdown
## References

[1] 作者. 标题[文献类型/文献载体]. URL, 日期.
[2] 作者. 标题[文献类型/文献载体]. 出版地: 出版者, 出版年.
```

**文献类型标识**：

| 文献类型 | 标识 |
|---------|------|
| 电子文献 | EB/OL |
| 图书 | M |
| 期刊文章 | J |
| 学位论文 | D |
| 报告 | R |

**示例**：

```markdown
## References

[1] 艾瑞咨询. 2024年中国护肤品行业研究报告[R]. https://www.iresearch.com.cn/report/2024-skincare, 2024-03-15.
[2] 欧莱雅集团. 2024年年度财务报告[EB/OL]. https://www.loreal.com/investors, 2024-02-28.
[3] 京东消费及产业发展研究院. Z世代消费趋势报告2024[R]. 北京: 京东集团, 2024.
```

## 格式与语气标准

### 语气

**麦肯锡 / BCG 风格** —— 权威、客观、专业

- **权威**：使用数据支撑观点，避免主观臆断
- **客观**：呈现事实，平衡观点，不偏不倚
- **专业**：使用行业术语，逻辑清晰，结构严谨

### 数字格式

- **千位分隔**：必须使用英文逗号（`1,000`）
- **百分比**：使用百分号（`12%`），保留一位小数（`12.5%`）
- **货币单位**：明确标注（`3,450 亿元`、`$500 million`）
- **时间格式**：使用标准格式（`2024年3月`、`2024-03-15`）

### 数据强调

- **重要观点和关键数字需加粗**：`**3,450 亿元**`
- **核心结论使用引用块**：
  ```markdown
  > **战略真相**：护肤品市场进入"品质驱动"阶段。
  ```

### 标题编号

- **使用标准编号**：`1.`、`1.1`、`1.1.1`
- **禁用前缀**：不要使用 "Chapter"、"Part"、"Section"

### 参考文献

- **必须严格遵循 GB/T 7714-2015**

## 常见问题

### Q1：数据缺失如何处理？

**A**：
1. 尝试补充搜索（使用备用关键词和来源）
2. 如无法获取，明确标注"数据暂不可得"
3. 说明数据缺失对分析结论的影响
4. 不编造或估算数据

### Q2：如何避免数据幻觉？

**A**：
1. 所有数据必须来自 Data Package
2. 每个 P0 数据至少从 2 个来源验证
3. 标注数据来源（URL、标题、日期）
4. 定期检查数据一致性

### Q3：如何处理数据冲突？

**A**：
1. 标注所有来源及其数值
2. 分析差异原因（时间范围、统计口径、样本差异）
3. 选择最可信来源或使用范围值
4. 在报告中说明差异

### Q4：如何提升报告专业性？

**A**：
1. 使用行业术语和专业表达
2. 数据对比多维度、多角度
3. 分析深入，避免表面化
4. 提供明确的洞察和建议
5. 参考麦肯锡、BCG 报告风格
