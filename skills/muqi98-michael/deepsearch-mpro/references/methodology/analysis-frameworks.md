# 分析框架与模型库

## 概述

本文档提供各领域的专业分析框架，用于指导章节骨架中的分析逻辑。

---

## 分析领域与维度

| 领域 | 常见分析维度 |
|------|--------------|
| 市场分析 | 市场规模、增长趋势、市场细分、增长驱动因素、竞争格局、消费者画像 |
| 行业分析 | 价值链分析、市场规模、竞争格局、政策环境、技术趋势、进入壁垒 |
| 品牌分析 | 品牌定位、市场份额、消费者感知、营销策略、竞品对比 |
| 消费者洞察 | 人口特征画像、购买行为、决策路径、痛点分析、场景分析 |
| 财务分析 | 宏观环境、行业趋势、公司基本面、财务指标、估值、风险评估 |
| 投资尽调 | 商业模式、财务健康度、管理层评估、市场机会、风险因素、退出路径 |
| 竞争情报 | 竞争对手识别、战略动向、市场动态、情报收集 |
| **竞争分析** ⭐ | 商业模式画布、产品矩阵、SWOT 分析、差异化定位、竞品对比、波特五力 |
| **AI 技术分析** ⭐ | 技术架构、产品功能、应用场景、开发者生态、社区活跃度、技术趋势、竞品对比 |
| **SaaS 厂商分析** ⭐ | 产品矩阵、客户结构、收入模型、市场策略、生态建设、战略动向、竞争优势 |

## 战略与环境分析

### SWOT 分析

| 维度 | 说明 | 应用场景 |
|------|------|----------|
| **Strengths（优势）** | 内部优势资源与能力 | 品牌评估、竞争定位 |
| **Weaknesses（劣势）** | 内部劣势与不足 | 战略规划、资源配置 |
| **Opportunities（机会）** | 外部市场机会 | 增长战略、市场进入 |
| **Threats（威胁）** | 外部环境威胁 | 风险评估、防御策略 |

**最适用场景**：品牌评估、竞争定位、战略规划

### PEST / PESTEL 分析

| 维度 | 说明 | 应用场景 |
|------|------|----------|
| **Political（政治）** | 政策法规、政治稳定性 | 宏观环境扫描 |

## 框架选择原则

1. **领域优先**：基于研究主题的领域，选择 2-4 个最相关框架
2. **互补性**：优先选择互补而非重叠的框架（宏观 + 微观）
3. **重深度，轻堆砌**：深入应用 2 个框架，优于表面堆叠 6 个框架
4. **数据可得性**：所选框架必须能被后续数据收集支持
5. **明确映射**：在章节骨架中明确标注每章使用了哪个框架
