# 报告撰写指南

## 概述

本文档提供咨询级报告的撰写标准和最佳实践。

---

## 格式与语气标准

### 咨询风格

- **语气**：麦肯锡 / BCG 风格 —— 权威、客观、专业
- **语言**：简洁、精准、无冗余
- **人称**：第三人称，避免"我们"、"我认为"

### 数字格式

- **千位分隔**：使用英文逗号（`1,000`、`10,000`）
- **百分比**：保留一位小数（`8.5%`）
- **金额**：标注单位（`3,450亿元`、`$500M`）
- **增长率**：标注时间范围（`CAGR 8.5%（2020-2024）`）

### 强调与标注

- **重要观点**：加粗（`**关键发现**`）
- **关键数字**：加粗（`**3,450亿元**`）
- **引用来源**：Markdown 链接（`[来源标题](URL)`）

### 标题规范

- **编号**：使用标准编号（`1.`、`1.1`、`1.1.1`）
- **禁用前缀**：不要使用 "Chapter""Part""Section"
- **标题层级**：最多 4 级（`#` `##` `###` `####`）

## 报告结构

### 标准结构

```
# [报告标题]

## Abstract
[高层摘要与关键结论]

## 1. Introduction
[背景、目标、方法]

## 2. [正文章节]
### 2.1 [小节标题]
[内容]

## N. Conclusion
[纯客观综合判断]

## N+1. References
[参考文献]
```

### Abstract（摘要）

**目的**：高层摘要与关键结论，让读者快速了解报告核心内容。

**内容**：
- 研究背景（1-2 句）
- 核心发现（3-5 点）
- 关键建议（1-2 句）

**示例**：

```markdown
## Abstract

中国护肤市场规模持续增长，2024年达到**3,450亿元**，CAGR为**8.5%**。Z世代（18-28岁）成为消费主力，贡献了**42%**的市场份额。

核心发现：
1. **市场规模稳健增长**：护肤品市场保持8.5%的年复合增长率，高于整体消费品市场
2. **Z世代驱动增长**：18-28岁消费者成为主力，人均年消费达2,300元
3. **功能性护肤崛起**：敏感肌、抗衰老类产品增速超过20%
4. **线上渠道主导**：电商平台占比达65%，直播带货贡献显著

战略建议：品牌应聚焦Z世代需求，加强功能性产品研发，深化线上渠道布局。
```

### Introduction（引言）

**目的**：说明研究背景、目标、方法，建立报告的可信度。

**内容**：
- 研究背景
- 研究目标
- 研究方法
- 报告结构

**示例**：

```markdown
## 1. Introduction

### 1.1 研究背景

中国护肤品市场经历了快速发展，消费者需求日益多元化和个性化。随着Z世代成为消费主力，市场格局正在发生深刻变化。

### 1.2 研究目标

本研究旨在：
1. 量化中国护肤品市场规模与增长趋势
2. 分析Z世代消费者的购买行为与偏好
3. 识别市场竞争格局与头部品牌策略
4. 提供战略建议

### 1.3 研究方法

本研究采用多源数据收集方法：
- **二手研究**：艾瑞咨询、欧睿国际等行业报告
- **公开数据**：国家统计局、上市公司财报
- **新闻资讯**：主流媒体报道、行业新闻

### 1.4 报告结构

本报告分为五个部分：市场规模与增长趋势、消费者画像、竞争格局、品牌策略、战略建议。
```

### Body Chapters（正文）

**目的**：呈现研究发现，提供数据驱动的洞察。

**结构**：每章遵循"数据呈现 → 分析 → 洞察"的模式。

**示例**：

```markdown
## 2. 市场规模与增长趋势

### 2.1 市场规模

中国护肤品市场规模持续增长，2024年达到**3,450亿元**，同比增长**8.5%**。

| 年份 | 市场规模（亿元） | 同比增长 |
|------|------------------|----------|
| 2020 | 2,680 | - |
| 2021 | 2,920 | 9.0% |
| 2022 | 3,120 | 6.8% |
| 2023 | 3,180 | 1.9% |
| 2024 | 3,450 | 8.5% |

**数据来源**：艾瑞咨询《2024年中国护肤品行业研究报告》

**分析**：

市场规模保持稳健增长，2020-2024年CAGR为**6.5%**。2023年增速放缓，主要受宏观经济影响。2024年市场回暖，增速回升至8.5%。

**增长驱动因素**：
1. **消费升级**：人均可支配收入增长，护肤品消费占比提升
2. **人群扩容**：Z世代和男性护肤市场快速增长
3. **品类扩展**：功能性护肤、医美级护肤等新品类涌现

> **战略启示**：市场仍处于增长期，品牌应把握Z世代和男性护肤两大增量市场。
```

### Conclusion（结论）

**目的**：综合所有发现，提供客观的战略判断。

**要求**：
- 不使用 bullet points
- 纯客观综合判断
- 连接数据 → 用户心理 → 战略启示

**示例**：

```markdown
## 5. Conclusion

中国护肤品市场正处于结构性变革期。市场规模稳健增长，但增长动力已从传统的女性护肤转向Z世代和男性护肤。功能性护肤的崛起反映了消费者从"品牌崇拜"转向"功效导向"的理性消费趋势。线上渠道的主导地位要求品牌具备全渠道运营能力，直播带货的兴起更是对品牌的实时响应能力提出了新要求。

品牌需要在三个维度建立竞争优势：一是产品力，通过功能性配方和功效验证赢得消费者信任；二是渠道力，构建"电商+直播+社交"的全渠道矩阵；三是品牌力，在Z世代价值观基础上建立情感连接。未来三年，市场将加速整合，头部品牌的市场集中度将进一步提升，中小品牌需要在细分赛道找到差异化定位才能生存。
```

### References（参考文献）

**目的**：提供数据来源，增强报告可信度。

**格式**：遵循 **GB/T 7714-2015** 标准。

**示例**：

```markdown
## 6. References

[1] 艾瑞咨询. 2024年中国护肤品行业研究报告[R/OL]. https://www.iresearch.com.cn/report, 2024-01-15.

[2] 欧睿国际. 中国护肤品市场分析2024[R/OL]. https://www.euromonitor.com, 2024-02-20.

[3] 国家统计局. 社会消费品零售总额统计[EB/OL]. https://www.stats.gov.cn, 2024-03-10.

[4] 凯度消费者指数. Z世代消费趋势洞察[R/OL]. https://www.kantar.com, 2024-01-20.
```

## 洞察撰写标准

### 洞察三要素

每个洞察必须包含：

1. **What（现象）**：数据呈现了什么
2. **Why（原因）**：为什么会这样
3. **So What（启示）**：对品牌/企业有什么启示

**示例**：

```markdown
**What**：Z世代贡献了42%的市场份额，人均年消费达2,300元。

**Why**：Z世代成长于物质丰富时代，护肤意识启蒙早，且更愿意为品质和功效付费。社交媒体的普及加速了护肤知识的传播，使Z世代成为最懂护肤的消费群体。

**So What**：品牌需要重新审视目标消费者定位，针对Z世代开发功效明确、包装时尚、社交属性强的产品，并通过KOL种草和社群营销建立品牌影响力。
```

### 避免的陷阱

| 陷阱 | 错误示例 | 正确示例 |
|------|---------|----------|
| **空洞陈述** | "市场很大" | "市场规模达3,450亿元，CAGR 8.5%" |
| **缺乏洞察** | "增长率为8.5%" | "增长率8.5%反映了消费升级和人群扩容的双重驱动" |
| **主观判断** | "我认为品牌应该..." | "数据显示品牌应聚焦Z世代..." |
| **脱离数据** | "消费者喜欢..." | "调研显示68%的消费者偏好..." |

## 表格设计标准

### 对比表

**用途**：对比不同品牌/产品/渠道的关键指标。

**示例**：

```markdown
| 品牌 | 市场份额 | 主打产品 | 目标人群 | 价格带 |
|------|---------|----------|----------|--------|
| 兰蔻 | 12.5% | 小黑瓶 | 25-40岁 | 高端 |
| 雅诗兰黛 | 11.8% | ANR | 30-45岁 | 高端 |
| 薇诺娜 | 5.2% | 敏感肌系列 | 18-35岁 | 中端 |
```

### 数据表

**用途**：呈现时间序列或结构化数据。

**示例**：

```markdown
| 年份 | 市场规模（亿元） | 同比增长 | 人均消费（元） |
|------|------------------|----------|----------------|
| 2020 | 2,680 | - | 189 |
| 2021 | 2,920 | 9.0% | 204 |
| 2022 | 3,120 | 6.8% | 215 |
| 2023 | 3,180 | 1.9% | 217 |
| 2024 | 3,450 | 8.5% | 233 |
```

## 引用规范

### 行内引用

使用 Markdown 链接格式：

```markdown
市场规模达3,450亿元[艾瑞咨询]。

根据凯度消费者指数调研，Z世代贡献了42%的市场份额[凯度消费者指数]。
```

### 参考文献格式

遵循 **GB/T 7714-2015**：

| 文献类型 | 格式 |
|---------|------|
| **报告** | `[序号] 作者. 报告名[R/OL]. URL, 日期.` |
| **网页** | `[序号] 作者. 文章名[EB/OL]. URL, 日期.` |
| **期刊** | `[序号] 作者. 文章名[J]. 期刊名, 年, 卷(期): 页码.` |
| **图书** | `[序号] 作者. 书名[M]. 出版地: 出版社, 年.` |

## 质量检查清单

### 内容质量

- [ ] 每个结论都有数据支持
- [ ] 每个数据都有来源标注
- [ ] 每个洞察都包含 What/Why/So What
- [ ] 无幻觉数据
- [ ] 无主观判断

### 格式规范

- [ ] 标题编号正确
- [ ] 数字格式统一
- [ ] 表格格式正确
- [ ] 参考文献格式符合 GB/T 7714-2015

### 语言风格

- [ ] 语气专业、客观
- [ ] 无冗余表达
- [ ] 无口语化表达
- [ ] 关键信息加粗
