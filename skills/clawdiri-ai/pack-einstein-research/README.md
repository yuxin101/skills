# Einstein Research Suite 🔬

**11 quantitative research tools for serious traders and investors.**

Stop making gut-feel trades. The Einstein Research Suite gives your AI agent the analytical toolkit of a professional quant desk — from market breadth to macro regime to options Greeks — all built for OpenClaw.

---

## What's Inside

| Skill | What It Does |
|-------|-------------|
| Backtest Expert | Stress-test strategies with proven methodology |
| Backtest Engine | Run full backtests with Sharpe, Sortino, Calmar, drawdown |
| Market Breadth Analyzer | 0-100 score: is the rally broad-based or hollow? |
| Market Bubble Detector | Quantitative Minsky/Kindleberger bubble risk framework |
| Edge Candidate Generator | Convert anomalies → structured research tickets |
| Follow-Through Day Detector | William O'Neil FTD bottom-confirmation signals |
| Options Strategy Advisor | Black-Scholes pricing, Greeks, earnings plays |
| Portfolio Risk Analyzer | VaR, stress tests, correlation matrix, circuit breakers |
| Macro Regime Detector | Detect Concentration → Broadening → Contraction shifts |
| Headline Scenario Analyzer | 18-month investment implications from news headlines |
| Market Theme Detector | Identify trending themes with lifecycle maturity scoring |

---

## Who This Is For

- **Active investors** who want data-driven signals, not gut feels
- **Quant-curious traders** building systematic strategies
- **Professionals** who need fast, structured research output
- **AI power users** extending OpenClaw with financial intelligence

---

## Requirements

- OpenClaw installed and configured
- Python 3.8+ with `yfinance`, `pandas`, `numpy`, `scipy`
- Some skills optionally use external CSV data sources (no API key required)
