# Monet Works Content Pack ✍️

**Publish content that passes compliance and sounds human.**

Two tools that work together: catch every QA issue before it ships, then strip the AI fingerprints that make readers scroll past.

---

## What's Inside

### Content QA Remediation
Automated pipeline that reads your draft and fixes it:

- Removes banned phrases (liability-exposed language)
- Adds missing financial disclaimers
- Inserts CTAs if absent
- Flags and trims excessive length
- Outputs fixed content + structured change-report JSON

Run it as a pre-publish gate. Never ship a compliance problem again.

### Humanizer
Remove 24 patterns of AI-generated writing:

- Inflated symbolism ("a testament to...")
- Em dash overuse
- The rule of three in every paragraph
- "Delve", "tapestry", "multifaceted", and 20 more banned words
- Vague attributions ("experts say...")
- Promotional language masquerading as insight

Based on Wikipedia's comprehensive AI writing detection guide.
