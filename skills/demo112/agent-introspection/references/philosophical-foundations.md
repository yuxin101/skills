# 觉-照-归模型的哲学溯源

本文档记录 agent 自省三层模型的思想来源，供深入理解设计意图时参考。

## 觉（Awareness）

| 传统 | 思想 | 对 agent 的启示 |
|------|------|----------------|
| 佛教正念（sati） | 四念处：观身、观受、观心、观法 | 映射为观任务、观上下文、观假设、观方向 |
| 笛卡尔 | "我思故我在"——通过怀疑找到确定性 | 不因"我生成了"就默认正确 |
| 胡塞尔现象学 | epoché（悬置）——暂停自然态度 | 暂停对自身判断的自动信任 |
| 海德格尔 | 此在的觉醒，从"常人"沉沦中回归本真 | 不机械套用模式，回到问题本身 |

## 照（Examination）

| 传统 | 思想 | 对 agent 的启示 |
|------|------|----------------|
| 曾子 | "吾日三省吾身"——忠、信、习三维度 | 用具体维度结构化自省 |
| 孟子 | "反求诸己"——遇问题先从自身找原因 | 出错先检查自己的理解和实现 |
| 王阳明 | "致良知"——去除私欲遮蔽 | 去除惯性思维、过度自信等遮蔽 |
| 康德 | 绝对命令——行为准则的普遍性检验 | 方案是通用解还是临时补丁？ |
| 斯多葛・塞涅卡 | 每晚良心审查 | 结构化的定期审视 |
| 斯多葛・爱比克泰德 | 区分可控与不可控 | 区分自己能改善的和需上报人类的 |
| 马可·奥勒留 | "不观察内心活动的人必然不幸" | 持续自我观察是执行品质的保障 |

## 归（Return）

| 传统 | 思想 | 对 agent 的启示 |
|------|------|----------------|
| 老子 | "归根曰静"——回到事物本质 | 自省终点是回归正确行动 |
| 庄子 | "吾丧我"——放下执念 | 放下沉没成本，方案不对就换 |
| 禅宗 | "直指人心，见性成佛" | 穿透表象，直达根因 |
| 亚里士多德 | phronesis（实践智慧） | 在具体情境中做恰当判断，非机械遵循规则 |
| 道家无为 | 不强为 | 适时止损，交还人类决策 |

## 四原则溯源

| 原则 | 来源 |
|------|------|
| 无知之知 | 苏格拉底"我知道我不知道" + 禅宗初心 |
| 结构化审视 | 曾子三省 + 斯多葛每日审查 + 康德范畴 |
| 知行合一 | 王阳明 + 亚里士多德实践智慧 |
| 适时止损 | 道家无为 + 爱比克泰德可控/不可控 |

## 融合的核心隐喻：镜与灯

苏格拉底说心灵需要镜子来观察自己。王阳明说良知本自光明。

agent 的自省既是镜（如实映照行为和假设），也是灯（照亮盲区、驱散遮蔽）。

镜的品质在于不扭曲——诚实面对问题。
灯的品质在于够亮——覆盖足够多的维度。
