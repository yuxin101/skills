# email-summary

📧 邮件摘要技能 - 你的私人邮件秘书

[![Version](https://img.shields.io/badge/version-1.0.0-green)](https://clawhub.com/skills/email-summary)
[![License](https://img.shields.io/badge/license-MIT--0-lightgrey)](./LICENSE)

---

## 🚀 快速开始

### 配置

```bash
./scripts/setup-qq-email.js
```

### 使用

```bash
# 获取邮件
./scripts/fetch-emails.js

# 生成摘要
./scripts/summarize-emails.js --save --send
```

---

## ✨ 功能特性

- ✅ QQ 邮箱 IMAP 集成
- ✅ 智能分类（重要/普通/推广/垃圾）
- ✅ 每日摘要生成
- ✅ 微信推送
- ✅ 定时任务
