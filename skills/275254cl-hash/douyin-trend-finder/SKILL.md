# 抖音热点发现技能

追踪抖音实时热点，发现爆款 BGM，分析热门话题，拆解爆款视频公式。

## 功能

1. **热点榜单**: 实时热榜、上升榜、新人榜
2. **BGM 发现**: 热门音乐、飙升 BGM、原创音乐
3. **话题分析**: 热门挑战、话题参与度、趋势预测
4. **爆款拆解**: 分析爆款视频的结构、节奏、元素
5. **达人监控**: 追踪对标账号的爆款内容
6. **创意灵感**: 基于热点生成创意建议

## 使用方式

```
查看今日抖音热榜 TOP10
```

```
帮我找最近飙升最快的 BGM
```

```
分析这个爆款视频为什么火：[视频链接]
```

```
美妆类最近有什么热门话题
```

```
生成 5 个蹭热点的创意方向
```
