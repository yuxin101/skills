# 什么值得买优惠聚合

实时监控什么值得买，发现神价、BUG 价、历史低价。

## 功能

1. **神价监控**: 实时推送超值优惠
2. **历史低价**: 对比历史价格
3. **BUG 价发现**: 价格错误快速提醒
4. **分类筛选**: 按品类过滤
5. **优惠汇总**: 自动计算最优方案

## 使用方式

**查看今日神价**:
```
今天有什么值得买的神价？
```

**监控特定品类**:
```
监控数码类目的神价，超过 50% 折扣时通知我
```

**查历史低价**:
```
查一下 iPhone 15 的历史低价
```

**BUG 价提醒**:
```
发现 BUG 价时立刻通知我
```
