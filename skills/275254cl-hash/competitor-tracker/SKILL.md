# 竞品监控技能

全方位监控竞品的价格、销量、评价、营销活动，生成竞争分析报告。

## 功能

1. **价格监控**: 实时追踪竞品价格变化，发现调价策略
2. **销量分析**: 估算竞品销量，分析销售趋势
3. **评价监控**: 追踪新品评价，分析用户反馈
4. **活动追踪**: 监控竞品促销活动、优惠券发放
5. **关键词排名**: 追踪竞品在搜索结果的排名
6. **库存监控**: 监控竞品库存变化，判断销售速度
7. **对比报告**: 生成竞品对比分析报告

## 使用方式

**添加竞品**:
```
添加竞品监控：https://item.taobao.com/item.htm?id=xxx
```

**查看价格历史**:
```
查看这个商品的价格历史走势
```

**销量分析**:
```
分析这个竞品的月销量和趋势
```

**评价分析**:
```
分析竞品最近 100 条评价，找出用户最关心的点
```

**生成报告**:
```
生成竞品分析报告，包含价格、销量、评价对比
```

**设置预警**:
```
当竞品价格低于¥100 时提醒我
```
