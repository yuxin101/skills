# 视频脚本生成技能

AI 驱动的短视频脚本创作工具，生成抖音、快手、视频号、B 站等平台的爆款脚本。

## 功能

1. **脚本生成**: 根据主题生成完整脚本（含分镜）
2. **口播稿撰写**: 生成自然流畅的口播文案
3. **分镜设计**: 详细的画面描述和拍摄指导
4. **爆款套用**: 基于热门视频公式生成内容
5. **平台适配**: 适配不同平台的风格和时长
6. **优化建议**: 分析脚本并提供改进方案

## 使用方式

```
帮我写一个 30 秒的抖音口播稿，主题是护肤技巧
```

```
生成一个带货视频脚本，卖点是：天然成分、敏感肌可用
```

```
把这个文案改写成视频脚本：[粘贴文案]
```

```
帮我设计分镜，主题是：办公室零食推荐
```
