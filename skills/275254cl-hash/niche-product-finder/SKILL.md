# 蓝海商品发现技能

发现低竞争高利润的细分市场商品机会。

## 功能

1. **蓝海发现**: 低竞争高需求商品
2. **趋势分析**: 上升期品类预测
3. **利润测算**: 估算毛利空间
4. **竞品分析**: 现有卖家数量
5. **供应链推荐**: 1688 货源对接

## 使用方式

**找蓝海商品**:
```
帮我找 3 个蓝海商品，要求月利润 1 万+
```

**分析品类**:
```
分析一下宠物用品市场的竞争情况
```

**利润测算**:
```
在淘宝卖手机壳能赚多少钱？
```

**找货源**:
```
找一下瑜伽垫的 1688 货源，要求利润 50%+
```

```

├─ 市场需求：月搜索 5 万+

├─ 平均售价：¥89-159
```
