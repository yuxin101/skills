# B 站数据分析技能

分析 B 站 UP 主数据、视频表现、粉丝画像，拆解爆款内容公式。

## 功能

1. **UP 主分析**: 粉丝数、播放量、更新频率、成长趋势
2. **视频数据**: 播放、点赞、投币、收藏、评论分析
3. **粉丝画像**: 年龄、性别、地域、兴趣分布
4. **爆款拆解**: 分析高播放视频的内容结构
5. **竞品监控**: 追踪同领域 UP 主的表现
6. **选题建议**: 基于数据生成选题方向

## 使用方式

```
分析这个 UP 主的数据：https://space.bilibili.com/xxx
```

```
查看这个视频的详细数据：[视频链接]
```

```
帮我找科技区上升最快的 UP 主
```

```
分析这个爆款视频为什么火
```
