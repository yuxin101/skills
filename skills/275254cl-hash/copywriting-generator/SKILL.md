# 文案生成技能

AI 驱动的营销文案创作工具，生成高转化率的标题、正文和 CTA。

## 功能

1. **标题生成**: 一键生成 10+ 个吸睛标题，支持多种风格
2. **正文创作**: 根据产品卖点生成完整文案
3. **CTA 设计**: 设计高转化率的行动号召
4. **A/B 测试**: 生成多版本文案用于测试
5. **风格适配**: 适配小红书、抖音、淘宝、公众号等平台
6. **优化建议**: 分析现有文案并提供改进方案

## 使用方式

**生成标题**:
```
帮我想 10 个小红书风格的标题，产品是抗老精华
```

**创作完整文案**:
```
帮我写一篇淘宝商品详情页文案，卖点是：天然成分、敏感肌可用、30 天见效
```

**设计 CTA**:
```
帮我设计 5 个不同风格的 CTA，用于引导用户下单
```

**优化现有文案**:
```
帮我优化这段文案，让它更有吸引力：[粘贴原文案]
```

**平台适配**:
```
把这篇文案改写成抖音口播稿风格
```
