# 淘宝评价分析技能

分析淘宝/天猫商品评价，提取用户真实反馈，发现质量问题和产品卖点。

## 功能

1. **评价抓取**: 获取商品评价数据（好评、中评、差评）
2. **情感分析**: 判断评价情感倾向（正面/负面/中性）
3. **关键词提取**: 发现用户最关心的产品特性
4. **问题发现**: 识别常见质量问题和投诉点
5. **卖点挖掘**: 提取用户认可的产品优势
6. **竞品对比**: 对比同类商品的评价差异

## 使用方式

```
分析这个商品的评价 https://item.taobao.com/item.htm?id=xxx
```

```
提取这款产品的用户反馈和常见问题
```

```
对比两款商品的口碑差异
```
