# 直播脚本生成技能

生成直播带货脚本，设计话术、把控节奏、互动环节、促单技巧。

## 功能

1. **脚本生成**: 根据产品生成完整直播脚本
2. **话术设计**: 设计引入、介绍、促单话术
3. **节奏把控**: 规划直播节奏和时间分配
4. **互动环节**: 设计抽奖、问答、福利环节
5. **促单技巧**: 设计限时、限量、限价促单策略
6. **问题应对**: 准备常见问题应答话术

## 使用方式

```
帮我生成一个 2 小时的直播带货脚本，卖护肤品
```

```
设计一个促单话术，产品是¥299 的精华液
```

```
直播冷场了，给我一些互动话术
```

```
准备 10 个常见问题的应答话术
```
