# Prompt-ZS 专业提示词生成助手

**版本：** 1.0.1
**更新日期：** 2025-03-17

## 功能简介

Prompt-ZS 是一个专业的AI绘画与视频创作提示词生成助手，支持多种提示词生成模式和格式。

## 核心功能

| 模式 | 功能描述 | 适用场景 |
|------|---------|---------|
| **Expert 1** | 基础JSON提示词生成 | 简单的图像生成需求 |
| **Expert 2** | 精准JSON提示词生成 | 专业摄影风格、相机参数 |
| **极简翻译** | 文本翻译 | 明确的翻译需求 |
| **文生视频** | 视频提示词扩展 | Runway/Pika等视频生成 |
| **图生视频** | 图片转视频提示 | 基于图片生成动态效果 |

## 使用方法

### 基础使用

直接描述你想要生成的内容，系统会自动识别最适合的模式：

```
"生成一个小女孩在公园里吹泡泡的提示词"
"帮我优化这个Midjourney提示词"
```

### 指定模式

```
"使用生成提示词2：赛博朋克风格的街景"
"文生视频：一朵花在晨光中绽放"
```

### 输出格式

**JSON格式（Expert 1 & 2）：**
```json
[
  {
    "cn": "中文描述",
    "en": "English description",
    "cn-tag": "中文标签",
    "en-tag": "English tags"
  }
]
```

**视频提示词（文生/图生）：**
完整的一段话，500字以内，精炼准确。

## 使用脚本

```bash
# 基础提示词生成
python scripts/prompt-zs-基础提示词生成器.py "内容描述"
```
