# 538 IELTS Keywords Reference Guide

## About This Resource

The 538 Keywords list comes from Liu Hongbo's "Cambridge IELTS Reading Keywords" (刘洪波《剑桥雅思阅读考点词真经》). It categorizes high-frequency words that appear repeatedly in IELTS Academic Reading questions.

## How to Use for Vocabulary Rating

When building the vocabulary table in a review note, look up each word against the 538 list:

### Rating Scale

| Rating | Category | Count | Question Appearance Rate |
|--------|----------|-------|------------------------|
| ⭐⭐⭐ | Category 1 | 54 words | ~90% of passages contain these |
| ⭐⭐ | Category 2 | 171 words | ~60% of passages contain these |
| ⭐ | Category 3 | 300+ words | Common but less predictable |
| — | Not listed | N/A | Check COCA 5000 for general frequency |

### Lookup Procedure

1. Search the full 538 list (provided as `雅思阅读538考点词-刘洪波真经.txt` in the workspace) for the target word
2. If found, note its category number and rank
3. Record in the vocabulary table as: `⭐⭐⭐ 538考点词 #XXX` (where XXX is the rank)
4. If not found in the 538 list, check COCA 5000 for general English frequency
5. Words below COCA 3000 that are not in the 538 list can be marked as low-priority

### COCA 5000 Supplementary Use

The COCA (Corpus of Contemporary American English) 5000 frequency list (`COCA_5000词频表.csv`) provides general English word frequency rankings. Use it to:

- Determine if an unknown word is commonly used in English (not just IELTS)
- Prioritize learning: COCA top 2000 = essential everyday English; 2000-5000 = academic/professional English
- A word in COCA top 3000 but not in the 538 list is still worth learning for general proficiency

### Cambridge Test Appearance Column

The "真题出现" (Cambridge Appearance) column tracks where a word has actually appeared in Cambridge IELTS tests. Format:

- Single appearance: `剑4T3P2`
- Multiple appearances: `剑4T3P2, 剑7T1P1, 剑9T2P3`
- This column is **cumulative** — add new appearances as more passages are reviewed
- High-frequency words appearing across multiple books indicate they are particularly important
