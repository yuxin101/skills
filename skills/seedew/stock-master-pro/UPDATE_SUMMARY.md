## 🚀 使用说明

### 访问 Dashboard

**方式 1：本地服务器**
```bash
# 服务器已在运行
http://localhost:3000
```

**方式 2：直接打开**
```
file:///home/yaoha/.openclaw/workspace/skills/stock-master-pro/web/index.html
```

### 查看各模块

1. **大盘概览** - 页面顶部，自动显示
2. **持仓监控** - 向下滚动可见
3. **复盘报告** - 点击"午盘复盘"等按钮
4. **公告监控** - 滚动到公告模块
5. **龙虎榜** - 滚动到龙虎榜模块
6. **财报日历** - 滚动到财报模块
7. **选股结果** - 滚动到选股模块

### 刷新数据

- **自动刷新**：每 10 秒自动更新（交易时段）
- **手动刷新**：点击各模块的"刷新"按钮
- **全部刷新**：点击顶部导航栏的刷新图标

## 📝 数据更新策略

### 交易时段（9:30-11:30, 13:00-15:00）
- ✅ 每 10 分钟调用 QVeris API
- ✅ 每 10 秒读取本地 JSON
- ✅ Dashboard 自动刷新

### 收盘后（15:00-次日 9:30）
- ❌ 停止调用 QVeris API（节省额度）
- ✅ 只读本地缓存
- ✅ 公告监控继续

### 周末/节假日
- ❌ 停止调用 API
- ✅ 只读模式

---

## ⚠️ 注意事项

### 1. 数据时效性
- 大盘/持仓数据：每 10 秒刷新
- 公告/龙虎榜/财报/选股：手动刷新或页面加载时自动加载

### 2. 浏览器缓存
- 如果数据不更新，按 Ctrl+F5 强制刷新
- 或清除浏览器缓存

### 3. 服务器状态
- 检查本地服务器是否运行：`ps aux | grep python`
- 重启服务器：`pkill -f "http.server 3000"` 然后重新运行 `start-web.sh`
