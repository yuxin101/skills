## 🎯 插件概述

### 核心功能
Turret插件为Minecraft服务器添加了自动防御炮塔系统，玩家可以：
- 放置和配置自动攻击炮塔
- 选择攻击目标类型（玩家、动物、敌对生物）
- 使用两种武器系统（弓和激光武器）
- 管理朋友列表避免误伤
- 修复和维护炮塔

## 🎮 快速开始指南

### 第一步：获取炮塔
#### 方法A：管理员给予（推荐新手）
```minecraft
# 管理员执行
/turret give
```

#### 方法B：游戏内合成
1. 打开炮塔GUI：右键点击已放置的炮塔
2. 点击"Find out how it all works"
3. 点击"All Crafts"
4. 查看炮塔合成配方

### 第二步：放置炮塔
1. 手持炮塔方块
2. 右键点击地面放置
3. 放置后自动显示："The turret was installed."

### 第三步：配置炮塔
```
1. 右键炮塔打开菜单
2. 选择"Select a target to attack"
3. 选择攻击模式：
   - 攻击玩家 / 不攻击玩家
   - 攻击动物 / 不攻击动物  
   - 攻击敌对生物 / 不攻击敌对生物
```

### 第四步：添加弹药
#### 弓武器（使用箭）
```minecraft
# 获取箭
/give @p minecraft:arrow 64
```

#### 激光武器（需要合成）
1. 查看合成配方（在"All Crafts"中）
2. 合成激光弹药
3. 放入炮塔弹药槽

**重要**：添加弹药前必须关闭炮塔！

### 第五步：启用炮塔
1. 返回炮塔主菜单
2. 点击"Status: disable"切换到"Status: enable"
3. 炮塔开始自动攻击选定目标

## ⚙️ 武器系统详解

### 1. 弓武器系统
- **弹药**: 普通箭 (minecraft:arrow)
- **伤害**: 4点伤害
- **攻击速度**: 0.7秒/次
- **优点**: 弹药易获取，成本低
- **缺点**: 射速较慢

### 2. 激光武器系统  
- **弹药**: 激光弹药（特殊合成物品）
- **伤害**: 2点伤害
- **攻击速度**: 0.2秒/次（高速）
- **优点**: 射速极快，压制力强
- **缺点**: 弹药需要合成

### 武器对比表
| 特性 | 弓 | 激光武器 |
|------|----|----------|
| 伤害 | 4 | 2 |
| 射速 | 0.7秒/次 | 0.2秒/次 |
| 弹药 | 箭 | 激光弹药 |
| 获取难度 | 简单 | 中等 |
| 推荐场景 | 早期防御、经济型 | 后期防御、高速压制 |

## 🛡️ 炮塔管理

### 炮塔状态管理
- **启用/禁用**: 在菜单中选择"Status: enable/disable"
- **修复炮塔**: 使用铁锭右键点击损坏的炮塔
- **移除炮塔**: 只有所有者可以在菜单中选择"Remove turret"

### 朋友系统
炮塔不会攻击朋友列表中的玩家：
```minecraft
# 添加朋友
/turret friend add <玩家名>

# 移除朋友  
/turret friend remove <玩家名>

# 查看朋友列表
/turret friend list
```

### 炮塔限制
1. **所有权**: 只有放置者可以管理自己的炮塔
2. **损坏系统**: 炮塔有耐久度，损坏后需要用铁锭修复
3. **弹药要求**: 必须装载对应弹药才能攻击
4. **启用状态**: 必须启用才能攻击，关闭状态可以添加弹药

## ⚠️ 常见问题解决

### 问题1：无法放入弹药
```
错误信息: "Turn off the turret to put the supplies in"
解决方案: 确保炮塔处于关闭状态（Status: disable）
```

### 问题2：炮塔不攻击
```
可能原因:
1. 炮塔未启用
2. 没有有效目标
3. 目标在射程外
4. 炮塔损坏需要修复
5. 没有弹药
```

### 问题3：无法移除炮塔
```
错误信息: "Only an owner can remove a turret"
解决方案: 只有炮塔放置者可以移除
```

### 问题4：炮塔损坏
```
修复方法:
1. 手持铁锭
2. 右键点击损坏的炮塔
3. 显示修复信息: "You repaired the turret."
```

- **理由**: 多层次防御
- **配置**: 外层用弓，内层用激光武器

3. **防御规划**:
   - 关键位置优先部署

   - 设置多层防御体系
