# Formatting

Default flomo content format:

```text
<entry content>

#02_读书 #02_读书/「<book title>」
```

Suggestions:
- Keep the note body first.
- Keep tags on the last line.
- Avoid adding too much metadata unless the user asks.
- If needed, extend to include chapter name, source, or created date.
- For users with existing manual flomo archives, prefer minimal formatting during initial rollout to reduce mismatch noise.
