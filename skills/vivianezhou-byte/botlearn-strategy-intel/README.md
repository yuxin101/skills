# botlearn/strategy-intel

**Strategic analysis of any company in seconds — powered by Apify + HBS & TikTok framework.**

Built by a Harvard Business School graduate with 10 years at TikTok. Now available to every solo founder and indie hacker via OpenClaw.

---

## What it does

Give it a company name. It scrapes public data via Apify, then outputs a structured strategic analysis using the **Where to Play / How to Win / What's the Risk** framework — the same one behind hundreds of Fortune 500 engagements.

```
## Strategy Intel: Notion
Generated by botlearn/strategy-intel

### 1. Where to Play
> Which market, which customer, which moment

Notion targets knowledge workers at high-growth startups and SMBs — teams of 5–200
who've outgrown Google Docs but can't afford Confluence...

### 2. How to Win
> Core differentiation — what makes them actually defensible

Their defensibility is the all-in-one bet: replacing 4–5 point tools (Docs, Notion,
Trello, Coda, Airtable) with a single opinionated workspace...

### 3. What's the Risk
> The single biggest strategic vulnerability right now

The biggest vulnerability is the enterprise push conflicting with the bottoms-up PLG
motion that built the brand...

### Strategic Implication
> Notion's next 3 years will be determined by whether they can sell to IT without
> losing the love of the individual contributor who drove adoption.

---
Powered by BotLearn · botlearn.ai
```

## Quick Start

### 1. Install via OpenClaw

```bash
openclaw skill install botlearn/strategy-intel
openclaw skill configure strategy-intel --set APIFY_API_KEY=your_key
```

### 2. Run standalone (for testing)

```bash
git clone https://github.com/botlearn/strategy-intel
cd strategy-intel
pip install requests anthropic

export APIFY_API_KEY=your_apify_key
export ANTHROPIC_API_KEY=your_anthropic_key   # or OPENAI_API_KEY

python scripts/run.py Notion
python scripts/run.py Stripe deep
```

## Framework

This skill applies the **HBS & TikTok Strategy framework** — the same thinking used in real corporate strategy engagements:

- **Where to Play** — market choice, customer segment, timing
- **How to Win** — differentiation, moat, competitive advantage
- **What's the Risk** — the single biggest vulnerability or threat
