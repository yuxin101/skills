# 黄金投资策略框架

## 核心投资逻辑

### 黄金定价三因素
1. **实际利率** = 名义利率 - 通胀预期
   - 实际利率下降 → 金价上涨
   - 实际利率上升 → 金价下跌

2. **美元指数**
   - 美元走强 → 金价承压
   - 美元走弱 → 金价上涨

3. **避险需求**
   - 地缘冲突 → 避险买金
   - 金融危机 → 避险买金

## 交易策略

### 趋势跟踪策略
```
买入条件：
- 价格站上20日均线
- 美元指数走弱
- 美联储降息预期

卖出条件：
- 价格跌破60日均线
- 美元指数走强
- 美联储加息预期
```

### 均值回归策略
```
买入区间：380-400元（历史均值下方10%）
目标位：450-480元（历史均值上方10%）
止损位：370元（跌破强支撑）
```

### 定投策略
```
固定周期买入：
- 每月固定金额买入
- 不预测价格，平滑成本
- 适合长期持有者

参考金额：每月收入的5-10%
```

## 仓位管理原则

### 金字塔加仓法
```
第一笔：试探性仓位 20%（价格触及支撑位）
第二笔：确认仓位 30%（价格反弹确认）
第三笔：加仓 30%（趋势确立）
第四笔：剩余 20%（突破确认）
```

### 倒金字塔减仓法
```
第一笔：止盈 30%（达到第一目标位）
第二笔：止盈 30%（达到第二目标位）
第三笔：止盈 20%（趋势可能反转）
第四笔：清仓 20%（确认反转）
```

## 风险管理

### 单笔风险控制
- 单笔亏损不超过总资金的2%
- 设置止损位并严格执行

### 总体仓位控制
- 黄金占总资产比例建议 5-15%
- 不要All in单一资产

### 黑天鹅应对
- 保留30%现金以备急跌加仓
- 设置自动止损单
- 关注流动性风险

## 常见错误

1. **追高杀跌** - 在高位追涨，低位恐慌卖出
2. **频繁交易** - 过度交易增加成本和错误率
3. **忽视汇率** - 只看国际金价忽视人民币汇率影响
4. **满仓操作** - 没有留出现金应对波动
5. **不设止损** - 亏损扩大时不愿止损

## 投资心态

- **耐心**：黄金是慢牛品种，不要期待暴富
- **纪律**：执行既定策略，不被情绪左右
- **学习**：持续关注宏观变化和市场动态
- **分散**：不要把所有资金都投入黄金
