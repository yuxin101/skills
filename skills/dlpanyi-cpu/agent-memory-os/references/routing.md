# Routing

## Core principle

Decide **scope first**, then choose retrieval path.

Do not default to searching everything.
Do not default to semantic search.

## Practical routing

### Search global memory first when
- the user asks about durable rules
- the question is about preferences, SOPs, or reusable methods
- the issue is cross-project by nature

### Search project memory first when
- the question is about current status
- the issue is clearly local to one project
- the user asks why a project-specific decision was made
- the user needs assets, files, or local milestones

### Use both when
- the question mixes stable rules with current project constraints
- a global method may need project-local adjustment
- a project pattern may need global validation

## Search order
1. structured/navigation-based lookup
2. keyword/full-text lookup
3. semantic recall as a supplement
4. evidence and status check before final answer
