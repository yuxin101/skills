# Chinese Social Media Content Forge

Create platform-native content for Chinese social media ecosystems.

## Platform Profiles

### Xiaohongshu (小红书) — Lifestyle & Discovery
- **Tone**: Personal, enthusiastic, emoji-rich, confessional
- **Structure**: Hook headline → relatable story → product/experience → tips → tags
- **Key patterns**: Title with number + emotion ("5件让我后悔没早买的神器😭"), paragraph emojis (✨🔥💡), 3-5 line paragraphs, closing tags
- **Hashtags**: 5-10 tags, mix broad + niche (#好物分享 #平价好物 #学生党必入)

### WeChat Official Accounts (微信公众号) — Long-form
- **Tone**: Authoritative yet conversational, structured argumentation
- **Structure**: Engaging intro → problem framing → analysis/solution → actionable takeaway
- **Key patterns**: Short paragraphs (2-4 sentences), bold key phrases, inline images every 300-500 words, pull quotes
- **Length**: 1500-3000 characters optimal

### Douyin (抖音) — Short Video Scripts
- **Tone**: Punchy, immediate, curiosity-driven
- **Structure**: 3-second hook → 15-second content → CTA
- **Key patterns**: Question/exclamation opening, fast-paced narration, on-screen text, music cues
- **Length**: 15-60 seconds spoken (~50-200 characters)

### Bilibili (B站) — Creator Content
- **Tone**: Knowledgeable, humorous, community-aware
- **Structure**: Problem/hook → deep dive → meme reference → summary
- **Key patterns**: Bullet-point descriptions, timestamp chapters, reference citations
- **Special**: "一键三连" (like+coin+favorite) CTA, community interaction prompts

## Workflow

### 1. Analyze Request
- Target platform(s)?
- Topic/theme?
- Source material (product review, experience, tutorial)?
- Desired tone (casual/professional/humorous)?
- Any brand guidelines or constraints?

### 2. Generate Platform-Specific Draft
Apply the platform profile above. Always generate at least 2 variations with different hooks.

### 3. Optimize
- **Hook test**: Would you stop scrolling for this?
- **Emoji check**: Xiaohongshu needs 1-2 per paragraph; WeChat needs none; Douyin needs emphasis only
- **Tag strategy**: Mix trending tags with niche tags for discoverability
- **Readability**: Read aloud — does it sound natural when spoken?


## Style Transfer Rules

When adapting English content to Chinese social media:

1. **Drop corporate language** — "leverage synergies" → "用好这几招"
2. **Add cultural context** — References to Chinese holidays, trends, lifestyle
3. **Adjust quantification** — "$50" → "一杯咖啡钱" or "学生党也能入手"
4. **Local examples** — Replace Western brand names with Chinese equivalents when possible
5. **Platform-native formatting** — Don't just translate; restructure for the target platform

## Hashtag Strategy

### Research Before Writing
- Check Xiaohongshu trending tags for the topic
- Use platform search to find high-engagement posts and their tags
- Mix 3 types: broad category, specific niche, trending event

### Tag Formula
```
[2 broad tags] + [3 niche tags] + [1-2 trending/event tags] + [1 personal/brand tag]
```

Example: `#好物推荐 #生活方式 + #平价好物 #宿舍神器 #学生党 + #618必买 + #我的私藏清单`

## Edge Cases

- **Sensitive topics**: Avoid political content, false advertising claims, unverified health claims
- **Platform rules**: Xiaohongshu restricts external links; WeChat limits redirects; Douyin has content review
- **Localization**: Mainland China Simplified Chinese by default. Adjust for Taiwan/HK if requested.
- **Image text**: Xiaohongshu cover images need text overlay — suggest 2-4 headline options for the cover
