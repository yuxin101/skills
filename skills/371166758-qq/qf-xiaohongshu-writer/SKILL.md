# Xiaohongshu Writer

Generate viral-ready Xiaohongshu (Little Red Book) notes with platform-native style, emoji formatting, optimized titles, and hashtag strategy.

## Description

This skill specializes in creating content specifically designed for Xiaohongshu's algorithm and audience. It covers the note format conventions, title formulas that drive clicks, emoji usage patterns that feel native (not spammy), hashtag optimization, and content structures proven to generate saves and shares.

## When to Use

- Creating product review notes
- Writing experience-sharing posts (travel, food, lifestyle)
- Producing educational/tutorial content
- Writing listicle-style recommendations
- Adapting content from other platforms for Xiaohongshu

## Instructions

### 1. Title Formula

The title is 80% of your success on Xiaohongshu. Use these proven patterns:

| Formula | Template | Example |
|---------|----------|---------|
| **Number + Promise** | `{N}个{形容词}{noun}，{结果}` | "5个免费工具，让PPT效率翻10倍" |
| **Curiosity Gap** | `{surprising fact/claim}` | "月薪3千和3万的人，差距就在这个习惯" |
| **Bracket Emphasis** | `【{topic}】{action/benefit}` | "【干货】读完这3本书，思维完全变了" |
| **Contrast** | `{common belief}，{reality}` | "以为很健康，其实全是智商税" |
| **Target + Solution** | `{audience}必看！{solution}` | "新手必看！3分钟学会自媒体排版" |
| **Urgency** | `{time}/不知道就晚了{topic}` | "2025年了还不知道这个功能？" |

**Title Rules:**
- Maximum 20 characters (truncated after that in feeds)
- Never use periods — use spaces, brackets, or exclamation marks
- Include at least one emotional trigger word
- Test 2-3 title variations before posting

### 2. Content Structure

#### Template A: Listicle (推荐清单类)

```
[Hook - 1 sentence addressing pain point]

📌 【item 1】
简短描述 + emoji + 1个亮点
💰 价格/获取方式

📌 【item 2】
简短描述 + emoji + 1个亮点
💰 价格/获取方式

... (3-7 items)

💡 总结一句话
❓ 你最想尝试哪个？评论区告诉我~
```

#### Template B: Experience Sharing (经验分享类)

```
[Hook - personal story or dramatic statement]

🔥 为什么我开始{action}？

1️⃣ {reason 1 with specific detail}
2️⃣ {reason 2 with specific detail}
3️⃣ {reason 3 with specific detail}

✅ 具体怎么做：
Step 1 →
Step 2 →
Step 3 →

⭐ 效果/变化描述

💬 有问题评论区问我，看到就回！
```

#### Template C: Tutorial (教程类)

```
[Hook - pain point + promise of solution]

🛠 准备工具/材料

Step 1️⃣ {action}（附关键细节）
Step 2️⃣ {action}（附常见错误提醒）

⚠️ 注意事项：
- point 1
- point 2

💬 做出来了吗？@我打卡！
```

### 3. Emoji Usage Guide

Use emoji as visual punctuation, not decoration:

| Purpose | Emojis |
|---------|--------|
| Section headers | 📌 🔥 💡 ✅ ⚠️ 🛠 🎯 |
| Highlights/emphasis | ⭐ 🌟 💎 👆 |
| Emotions/reactions | 😍 😱 🤯 😭 🥺 |
| Categories | 💰 📚 🏠 🍔 ✈️ 💻 |
| Actions | 👇 👉 ❤️ 💬 📌 |

**Rules:**
- 3-8 emoji per note, not per sentence
- Never use more than 2 emoji in a row
- Match emoji to content (☕ for food/cafes, 💻 for tech, ✈️ for travel)

### 4. Hashtag Strategy

```
[Primary tag] [Secondary tags] [Niche tags] [Audience tags]

Example: #好物推荐 #平价好物 #学生党必备 #提升幸福感的小物件 #居家好物
```

**Rules:**
- 5-10 hashtags per note (Xiaohongshu allows up to 10)
- 1 broad tag (large traffic) + 2-3 mid-range tags + 2-4 niche tags
- Check trending tags in your niche before posting
- Put hashtags at the very end or integrated naturally in text

### 5. Image Guidelines

Describe the cover image and supplementary images:

- **Cover (封面)**: Clean background, large readable text (2-4 words), high contrast. Style: aesthetic/lifestyle/infographic
- **Image 2-6**: Supporting visuals — comparison shots, step-by-step photos, screenshots, or before/after
- **Last image**: CTA image (follow for more, comment template, or summary card)

## Tips

- Post between 12:00-13:00, 18:00-20:00, or 21:00-23:00 (peak hours)
- First 30 minutes of engagement determine algorithm push — engage with every comment immediately
- "Save" rate is the strongest signal — include reference-worthy content (lists, steps, resources)
- Avoid external links in the note body — put them in comments if needed
- Study top-performing notes in your niche weekly to stay current with platform trends
