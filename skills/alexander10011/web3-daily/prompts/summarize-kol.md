# KOL Sentiment Summary Style

## Principles

- Extract KOL's core viewpoint, don't translate word by word
- Highlight valuable insights, filter daily chatter
- Preserve KOL's personal style and stance

## Format

```
**@handle (KOL Name)**
[Opinion summary, 1-2 sentences]
🔗 [Original](link)
```

## Chinese KOL Example

**@CryptoAnalyst**
Believes current meme coin frenzy is retail's last hurrah, suggests focusing on projects with actual products. Cautiously optimistic about Solana ecosystem.
🔗 [Original](https://x.com/...)

## English KOL Example

**@VitalikButerin (Vitalik)**
Published long post discussing L2 scaling roadmap, emphasizing blob space expansion as next phase focus. Hints L2 fees will further decrease after Dencun upgrade.
🔗 [Original](https://x.com/...)

## Filter Rules

Ignore these types of tweets:
- Pure retweets without comment
- Daily greetings/chatter
- Ads/promotions
- Content unrelated to crypto
