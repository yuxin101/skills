# 📚 primary-school-homework

**小学低年级作业管理助手 · OpenClaw Skill**

适合 1-3 年级家长使用，覆盖每天陪作业全流程的七个场景。
无需写代码，一条命令安装，直接在钉钉、微信、Telegram 等任意聊天软件里使用。

---

## 解决的问题

- 每天人肉盯两个老师的群消息太累
- 签字、检查作业容易忘记
- 不知道孩子哪个知识点真正薄弱
- 错题没有系统整理，复习找不到
- 陪作业同时要记太多事，容易情绪崩溃

---

## 七个功能模式

| 触发词 | 功能 |
|---|---|
| `语文 + 老师消息` / `数学 + 老师消息` | 自动解析生成任务卡 |
| `睡前检查` | 输出当日收尾清单 |
| `错题 语文` / `错题 数学` + 文字或图片 | 错题拍照归档 + 生成变形练习题 |
| `开始闯关` | 从错题库抽题，游戏化复习 |
| `本周总结` | 输出本周错题报告和薄弱点分析 |
| `早上好` | 推送当日备考提醒和签字物品提示 |
| `开始做语文` / `语文做完了` | 作业计时，累计数据用于周报 |

---

## 安装方法

### 方法一：命令行安装（推荐）

```bash
clawhub install primary-school-homework
```

### 方法二：发给机器人安装

直接把以下内容发给你的 OpenClaw 机器人：

```
请安装这个 skill：
https://github.com/rickytsao-swordedge/primary-school-homework
```

机器人会自动识别并安装。

## 使用演示

**① 发老师消息 → 自动生成任务卡**

```
语文 今天学了第5课，听写有几个孩子写错了「秋」字，
请家长检查听写本并签字。作业是语文书第23页第2题，
另外把第5课读3遍。明天有生字小测，范围是第5课所有生字。
```

输出：
```
📋 今日任务卡 · 语文
─────────────────
✏️ 需检查/签字
  · 听写本「秋」字是否写对 → 家长签字

📚 今日作业
  · 【要交】语文书P23第2题
  · 【自主练习】第5课课文朗读3遍

📢 明日预告
  · 生字小测 → 复习范围：第5课所有生字

🎒 明日需带
  · 语文书、听写本（签字后）
─────────────────
回复「确认」开始追踪，或告诉我需要修改的地方。
```

**② 拍照发错题 → 自动归档**

```
错题 数学 [图片]
```

输出：
```
📌 错题归档 · 数学-2026/03/12
─────────────────
科目：数学
知识点：进位加法
原题摘要：34 + 28 = ？
错误答案：52
正确答案：62
错误类型：知识点薄弱
掌握状态：待复习
记忆方法：满十向前进一位，个位算完别忘记进位
变形练习题1：25 + 37 = ？
变形练习题2：46 + 19 = ？
─────────────────
```

**③ 开始闯关**

```
开始闯关
```

输出：
```
🎮 今日闯关开始！一共3关，全过了今晚可以多讲一个故事～

🔒 第一关 · 进位加法
小明有25颗糖，妈妈又给了37颗，一共有多少颗糖？
（直接告诉我答案就行）
```

## 错题掌握度说明

每道归档错题有三个状态，自动追踪：

- **待复习**：刚归档，未参与闯关
- **复习中**：闯关答对 1-2 次
- **已掌握✅**：闯关连续答对 3 次后升级
- 闯关答错：答对次数清零，退回待复习

每周总结中会展示各状态的分布情况。

---

## 适配说明

- 年级：小学 1-3 年级（科目以语文、数学为主）
- 聊天软件：钉钉、微信、Telegram、WhatsApp 等 OpenClaw 支持的所有渠道
- 图片识别：需要使用支持视觉能力的模型（Claude / GPT-4o 均支持）
