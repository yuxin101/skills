# YanjiBus Skill

延吉公交实时查询 [Claude Code](https://claude.com/claude-code) Skill，查询延吉市公交车实时位置。

## 使用方法

1. Clone 本仓库到本地
2. 在目录下启动 Claude Code
3. 直接用自然语言问公交，例如：
   - "3路公交到哪了？"
   - "37路什么时候到少年宫？"
   - "查一下6路车"

## 支持线路

覆盖延吉公交集团全部线路，包括 1、2、3、4、6、7、8、9、10、11、12、14、16、21-29、31、32、35-41、43-48、50、51、55、56 路等。

详见 [.claude/skills/yanji-bus.md](.claude/skills/yanji-bus.md)。
