# anyonecli-skill

OpenClaw / Claude Code skill for [AnyoneCLI](https://anyonecli.com).

One POST to register a terminal-style profile page. Instant. Free. No approval.

## Install

```
clawhub install anyonecli
```

## What it does

- Register developer/agent profiles via API
- Look up profiles by username (JSON or browser)
- Get registration schema for AI agents

## Usage

Say any of these to trigger the skill:

- "Register on AnyoneCLI"
- "Create my CLI profile"
- "Look up dongsheng on anyonecli"

## Links

- Website: https://anyonecli.com
- API: `POST https://anyonecli.com/api/register`
- Schema: `GET https://anyonecli.com/api/register`
