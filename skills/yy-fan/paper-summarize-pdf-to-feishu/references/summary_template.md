## 一、研究背景与动机

- **核心问题**: 研究要解决什么问题？
- **现状分析**: 当前领域现状如何？
- **研究空白**: 缺少什么？为什么需要这项研究？

## 二、研究设计

- **实验方法**: 随机对照试验？回顾性分析？
- **数据来源**: 参与者数量、招募方式、分组设计
- **评分标准**: 如何定义"正确"？金标准如何制定？
- **统计方法**: 使用了哪些统计检验？

- 关键数据 2（带 p 值和置信区间）

### 3.2 <结果类别 2>
- ...

### 3.3 <结果类别 3>
- ...

## 四、基准测试对比（可选）

- **与现有基准的对比**: MedQA、MMLU 等
- **模拟实验结果**: 模拟用户 vs 真实用户
- **相关性分析**: 基准成绩是否能预测真实表现？

## 五、讨论与局限性

- **主要发现**: 研究结果意味着什么？
- **局限性分析**: 
  - 样本局限性（地区、人群）
  - 方法局限性（实验设计、测量方法）
  - 结论局限性（适用范围）
- **未来方向**: 下一步研究建议

## 六、结论与建议

- **核心结论**: 一句话总结研究结论
- **对开发者的建议**: 如何改进模型？
- **对监管者的建议**: 如何制定政策？
- **对从业者的建议**: 如何应用/避免？

## 七、关键数据速览

- **性能对比图**: 用表格或文字描述关键对比
- **统计摘要**: 效应量、p 值、置信区间

## 灵活调整原则

根据论文类型调整结构：

### 临床医学论文
保留完整结构：背景 → 方法 → 结果 → 讨论 → 结论

### 综述论文
调整为：背景 → 主题分类 → 关键发现 → 未来方向

### 方法学论文
强调：方法设计 → 实验验证 → 性能对比

### 短篇论文（Letter/Communication）
可合并：
- "讨论"与"结论"合并为"讨论与结论"
- "研究设计"与"核心结果"合并

### 案例报告
调整为：背景 → 案例描述 → 诊断过程 → 治疗方案 → 预后 → 讨论

## 数据精度标注规范

**必须标注**：
- 百分比：`XX.X%`（保留一位小数）
- 标准差：`XX.X% ± X.X%`
- 置信区间：`95% CI: X.XX–X.XX`
- p 值：`p < 0.001` 或 `p = 0.023`
- 样本量：`n = XXX`

**效应量报告**：
- 比值比：`OR = X.XX (95% CI: X.XX–X.XX)`
- 回归系数：`β = X.XX ± X.XX`
- 相关系数：`r = X.XX, p < 0.001`
