# Video Input

**Node type ID:** `9451e6ff-ac7f-4317-9055-a2f35f641b29`

The input node that accepts video into an agent graph. This is typically the first node in a workflow. Videos are provided via `video_urls`, `video_ids` (from the Uploads API), or `node_render_ids` (from prior run outputs) in the run payload.

To get the full list of configurable parameters, call `GET /node_type/9451e6ff-ac7f-4317-9055-a2f35f641b29`.
