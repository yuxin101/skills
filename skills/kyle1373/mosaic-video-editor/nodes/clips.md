# Clips

**Node type ID:** `bcc5cb04-76b7-48e9-a344-0e8a29c19be0`

Automatically extracts short, standalone clips from longer video. These clips are shareable moments for social platforms. Ideal for repurposing podcasts, interviews, webinars, or speaking content into short-form highlights.

To get the full list of configurable parameters, call `GET /node_type/bcc5cb04-76b7-48e9-a344-0e8a29c19be0`.

Place before style nodes for tighter edits.
