# Reframe

**Node type ID:** `0e3a90e4-173f-42fc-b7ef-0df6a062e26c`

Automatically converts your video into a different aspect ratio (9:16 for TikTok, 1:1 for Instagram, 16:9 for YouTube, etc.) while keeping the subject in view using intelligent face and object tracking. Essential for repurposing videos across platforms without manual cropping.

To get the full list of configurable parameters, call `GET /node_type/0e3a90e4-173f-42fc-b7ef-0df6a062e26c`.

Place BEFORE layout-sensitive nodes (Captions, Cinematic Captions, Motion Graphics, Watermark) so downstream composition respects the final framing.

[Full docs](https://docs.mosaic.so/tiles/reframe)
