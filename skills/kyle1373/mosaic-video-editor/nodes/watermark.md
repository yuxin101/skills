# Watermark

**Node type ID:** `6808af58-2d9a-4e3b-aacb-2d9c143b7729`

Overlays a watermark image on your video. Supports positioning, sizing, and opacity control for brand logos or attribution marks.

To get the full list of configurable parameters, call `GET /node_type/6808af58-2d9a-4e3b-aacb-2d9c143b7729`.

Place after Reframe if aspect ratio changes are needed.
