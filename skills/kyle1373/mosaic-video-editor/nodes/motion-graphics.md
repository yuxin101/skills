# Motion Graphics

**Node type ID:** `19144346-8714-4a02-b19c-32c18bd1ad13`

Generates animated motion graphics overlays for your video. Useful for adding visual flair, data callouts, lower thirds, and animated text. Use `style_video_url` to provide a reference style and `only_generate_full_screen_graphics` to control layout. Do not use the deprecated `avoid_prompt` parameter.

To get the full list of configurable parameters, call `GET /node_type/19144346-8714-4a02-b19c-32c18bd1ad13`.

Place after Reframe if aspect ratio changes are needed.

[Full docs](https://docs.mosaic.so/tiles/motion-graphics)
