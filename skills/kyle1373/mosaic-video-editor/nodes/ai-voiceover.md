# AI Voiceover

**Node type ID:** `9570c960-b32b-4e4d-8871-5775d886f7cb`

Generates narrated voiceover for your video using AI. Takes a written script (or auto-generates one), synthesizes a natural-sounding voice, and aligns it with your video. Useful for explainers, product demos, documentary-style narration, and storytelling.

To get the full list of configurable parameters, call `GET /node_type/9570c960-b32b-4e4d-8871-5775d886f7cb`.
