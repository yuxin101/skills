# 面试邀请邮件模板

## 模板一：正式版（推荐）

```
Subject: 【面试邀请】{公司名称} — {岗位名称} — {候选人姓名}

{候选人姓名} 先生/女士，您好！

感谢您对 {公司名称} {岗位名称} 岗位的关注与投递。

我们很高兴地邀请您参加面试，具体安排如下：

━━━━━━━━━━━━━━━━━━━━
📋 面试信息

⏰ 时间：{日期} {开始时间} — {结束时间}（北京时间）
🏢 形式：线上视频面试（飞书会议）
🔗 会议链接：{meeting_url}
👤 面试官：{面试官姓名}
📌 岗位：{岗位名称}
━━━━━━━━━━━━━━━━━━━━

💡 温馨提示：
1. 请提前 5 分钟点击会议链接，测试音视频设备；
2. 建议使用电脑端参加面试，网络环境稳定；
3. 如需更改面试时间，请尽早回复此邮件说明；
4. 如有技术问题无法加入会议，请拨打：{联系电话}。

期待与您的交流！

此致
{公司名称} 招聘团队
{HR联系方式/邮箱}
```

## 模板二：简洁版

```
Subject: 面试邀请 — {公司名称} — {岗位名称}

{候选人姓名} 您好，

诚邀您参加 {公司名称} {岗位名称} 的线上面试：

📅 {日期} {开始时间}—{结束时间}
🔗 飞书会议：{meeting_url}
👤 面试官：{面试官姓名}

请提前 5 分钟进入会议室。如有时间冲突，请回复本邮件。

祝好，
{公司名称} 招聘团队
```

## 模板三：英文版

```
Subject: Interview Invitation — {Company} — {Position}

Dear {Candidate Name},

Thank you for your interest in the {Position} role at {Company}.

We would like to invite you to an online interview:

📅 Date: {Date}
⏰ Time: {Start Time} — {End Time} (Beijing Time, UTC+8)
🔗 Feishu Meeting: {meeting_url}
👤 Interviewer(s): {Interviewer Names}
📌 Position: {Position}

Tips:
1. Please join 5 minutes early to test your audio/video.
2. A stable internet connection is recommended.
3. If you need to reschedule, please reply to this email.

Looking forward to speaking with you!

Best regards,
{Company} Recruiting Team
```

## 占位符说明

| 占位符 | 说明 | 来源 |
|--------|------|------|
| `{公司名称}` | 公司名 | 配置或用户指定 |
| `{岗位名称}` | 应聘岗位 | 表格"应聘岗位"字段 |
| `{候选人姓名}` | 候选人姓名 | 表格"候选人姓名"字段 |
| `{日期}` | 面试日期 | 推荐时段确认后的日期 |
| `{开始时间}` / `{结束时间}` | 面试起止时间 | 推荐时段确认后的时间 |
| `{meeting_url}` | 飞书视频会议链接 | create-event 响应中的 meeting_url |
| `{面试官姓名}` | 面试官 | 表格"面试官"字段 |
| `{联系电话}` | HR 联系电话 | 配置或用户指定 |
| `{HR联系方式/邮箱}` | HR 邮箱 | 配置或用户指定 |
