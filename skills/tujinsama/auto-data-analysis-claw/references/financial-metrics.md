# 财务分析指标体系

## 盈利能力指标

| 指标 | 公式 | 说明 |
| --- | --- | --- |
| 毛利率 | (收入 - 成本) / 收入 × 100% | 衡量产品/服务的基础盈利水平 |
| 净利率 | 净利润 / 收入 × 100% | 扣除所有费用后的盈利水平 |
| ROE | 净利润 / 股东权益 × 100% | 股东投资回报率 |
| ROA | 净利润 / 总资产 × 100% | 资产使用效率 |
| EBITDA 利润率 | EBITDA / 收入 × 100% | 经营现金流盈利能力 |

## 运营效率指标

| 指标 | 公式 | 说明 |
| --- | --- | --- |
| 应收账款周转天数 | (平均应收账款 / 收入) × 365 | 收款速度 |
| 存货周转天数 | (平均存货 / 成本) × 365 | 存货流转速度 |
| 应付账款周转天数 | (平均应付账款 / 成本) × 365 | 付款周期 |
| 现金转换周期 | 应收天数 + 存货天数 - 应付天数 | 营运资金效率 |
| 总资产周转率 | 收入 / 总资产 | 资产产出效率 |

## 偿债能力指标

| 指标 | 公式 | 说明 |
| --- | --- | --- |
| 流动比率 | 流动资产 / 流动负债 | 短期偿债能力 |
| 速动比率 | (流动资产 - 存货) / 流动负债 | 严格短期偿债能力 |
| 资产负债率 | 总负债 / 总资产 × 100% | 债务杠杆水平 |
| 利息保障倍数 | EBIT / 利息费用 | 利息支付能力 |
| D/E 比率 | 总负债 / 股东权益 | 财务杠杆 |

## 成长性指标

| 指标 | 公式 | 说明 |
| --- | --- | --- |
| 收入增长率 | (本期收入 - 上期收入) / 上期收入 × 100% | 收入增长速度 |
| 利润增长率 | (本期净利润 - 上期净利润) / 上期净利润 × 100% | 盈利增长速度 |
| 同比增长率 | (本期 - 去年同期) / 去年同期 × 100% | 消除季节性影响 |
| 环比增长率 | (本期 - 上期) / 上期 × 100% | 短期趋势变化 |
| CAGR | (终值/初值)^(1/年数) - 1 | 复合年增长率 |

## 分析注意事项

1. **数据质量**：确保财务数据来源一致、口径统一，跨期对比时注意会计政策变化
2. **行业基准**：单独看绝对值意义有限，应与行业均值/标杆对比
3. **趋势优先**：连续多期的趋势比单点数据更有分析价值
4. **交叉验证**：异常指标需交叉核对原始数据，排除数据错误
5. **定性结合**：定量分析结果需结合业务背景（政策变化、市场事件等）解释
