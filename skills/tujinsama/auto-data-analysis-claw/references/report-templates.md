# 报表模板与输出规范

## 标准分析报告结构

```
1. 执行摘要（Executive Summary）
   - 一段话概括核心发现
   - 关键数字高亮（3-5个）

2. 数据概览（Data Overview）
   - 数据来源、时间范围、记录数
   - 字段说明
   - 数据质量评估（完整率、异常值）

3. 核心指标（Key Metrics）
   - KPI 卡片/表格
   - 与上期/预算/行业基准对比
   - 变化率标注（↑↓）

4. 多维分析（Deep Dive）
   - 按维度拆解（产品/地区/渠道/客户等）
   - 结构占比分析
   - Top N / Bottom N 排名

5. 趋势分析（Trend Analysis）
   - 时间序列走势
   - 移动平均线
   - 周期性/季节性识别

6. 异常与洞察（Anomalies & Insights）
   - 显著偏离点标注
   - 根因推测
   - 风险提示

7. 结论与建议（Conclusions & Recommendations）
   - 分条列出（3-5条）
   - 可操作的下一步行动
```

## Markdown 报表格式规范

### KPI 呈现
```markdown
| KPI | 本期 | 上期 | 变化 | 趋势 |
| --- | --- | --- | --- | --- |
| 总收入 | ¥1,234万 | ¥1,100万 | +12.2% | ↑ |
| 毛利率 | 42.5% | 40.1% | +2.4pp | ↑ |
```

### 洞察标注
```markdown
> 💡 **洞察**: 华东区Q3收入环比增长28%，主要由新产品线贡献（占比从12%提升至31%）
```

### 风险提示
```markdown
> ⚠️ **风险**: 应收账款周转天数从45天延长至62天，建议关注信用政策
```

## 数值格式规则

- **金额**：万元为单位，保留2位小数，¥前缀（如 ¥1,234.56万）
- **百分比**：保留1位小数（如 42.5%）
- **百分点差异**：用 pp 表示（如 +2.4pp）
- **增长率**：带符号和箭头（如 +12.2% ↑）
- **大数字**：超万用"万"，超亿用"亿"
