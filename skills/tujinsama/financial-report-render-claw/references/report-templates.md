# Report Templates

各类报告的标准结构模板。根据报告类型选择对应模板。

## 财务报告模板

适用：月度/季度/年度财务报告

```
1. 封面
   - 报告标题、公司名称、报告期间、机密等级

2. 目录（自动生成）

3. 执行摘要（summary）
   - 3-5 句话概括本期核心发现
   - 关键指标高亮卡片（收入、利润、现金流等）

4. 财务概览（kpi-cards）
   - 营业收入、毛利率、净利润、ROE、资产负债率
   - 每项标注目标值和达成状态

5. 收入分析
   - 收入趋势图（chart, line）
   - 收入构成图（chart, pie/bar）
   - 分产品线/区域收入明细表（table）
   - 同比环比分析文字

6. 成本分析
   - 成本结构图（chart, pie）
   - 主要成本项趋势（chart, line）
   - 成本率变化分析

7. 利润分析
   - 利润趋势图（chart, combo - 柱状+折线）
   - 利润构成明细表（table）

8. 现金流分析
   - 现金流趋势（chart, bar）
   - 经营/投资/筹资活动现金流明细

9. 风险与建议（advisory）
   - 财务风险提示
   - 改进建议

10. 附录
    - 数据来源说明
    - 计算方法与假设
    - 术语表
```

## 经营分析模板

适用：业务经营分析、季度经营复盘

```
1. 封面

2. 执行摘要（summary）
   - 本期经营核心结论
   - 关键经营指标

3. 业务概览（kpi-cards）
   - GMV、订单量、客单价、转化率、复购率

4. 核心指标达成（kpi-cards + table）
   - 各 KPI 目标 vs 实际 vs 去年同期

5. 趋势分析
   - 核心指标趋势（chart, line/combo）
   - 环比变动分析

6. 专题分析（text + chart，按需）
   - 用户增长分析
   - 渠道效率分析
   - 产品表现分析

7. 问题诊断（text）
   - 未达标项归因
   - 异常波动说明

8. 行动建议（advisory）
   - 下阶段重点工作
   - 资源分配建议
```

## 月报/季报精简模板

适用：快速汇报、管理层简报

```
1. 标题头（期次 + 日期）

2. 本期亮点（summary, 3-5 条）

3. 核心指标（kpi-cards, 4-6 个）

4. 关键图表（chart × 2-3）

5. 问题与计划（advisory, 精简版）
```

## 项目汇报模板

适用：项目进展报告、阶段性总结

```
1. 封面

2. 项目概况（summary）
   - 项目目标、当前阶段、整体进度

3. 关键成果（kpi-cards + table）
   - 里程碑完成情况
   - 交付物清单

4. 进度分析（chart + text）
   - 甘特图或进度图
   - 延期项说明

5. 风险与问题（advisory）
   - 当前风险
   - 已解决问题

6. 下一步计划（advisory）
   - 下一阶段目标
   - 需要的资源/支持
```

## 通用建议

- **执行摘要**始终放在最前面，控制在半页以内
- **图表数量**：每类报告 5-10 张为宜，避免信息过载
- **表格**：数据行超过 15 行时考虑拆分或使用附录
- **页数控制**：管理层报告 ≤ 10 页，详细报告 ≤ 20 页
- **配色一致性**：同一报告内使用统一色板
