# 数据分析方法论参考

## 变动性质判断标准

### 变动幅度分级

| 等级 | 幅度范围 | 标记 | 含义 |
|------|---------|------|------|
| 正常波动 | ±5% 以内 | 🟢 | 统计噪音范围内，无需特别关注 |
| 温和变动 | ±5%~15% | 🟡 | 有变化但可能属正常区间，持续观察 |
| 显著变动 | ±15%~30% | 🟠 | 需要解释原因，检查是否为趋势拐点 |
| 重大变动 | ±30% 以上 | 🔴 | 必须立即排查原因并评估影响 |

> 注：以上阈值可根据具体业务调整（如零售业波动本身就大，可适当放宽）。

## 趋势判断方法

### 移动平均法
- **3期移动平均**：消除单期噪音，适合月度数据
- **6期移动平均**：观察半年趋势，适合周/月度数据
- **12期移动平均**：观察年度趋势，适合月度数据

### 拐点识别
- 连续2期移动平均方向改变 → 初步拐点信号
- 连续3期确认 → 趋势反转确认
- 配合同比数据验证：环比和同比同向变动更可信

## 归因分析框架

### 差异拆解公式
```
总差异 = Σ(各子维度差异贡献)
单维度贡献 = (本期值 - 上期值) / 总差异 × 100%
```

### 常用拆解维度
- **按产品/品类**：哪个品类拖了后腿/贡献了增长
- **按区域**：哪个区域表现异常
- **按渠道**：线上vs线下、直营vs分销
- **按客户类型**：新客vs老客、大客户vs中小客户
- **按价格段**：高端vs中端vs低端

### 归因追问清单
发现显著变动时，依次追问：
1. 变动是全局性还是局部性？（整体跌还是某个品类跌）
2. 是量差还是价差？（数量变了还是单价变了）
3. 是新业务影响还是老业务变化？（结构性问题还是周期性问题）
4. 是否有一次性因素？（促销、节日、政策、会计调整）
5. 竞争对手是否也有类似变动？（行业趋势还是个体问题）

## 陷阱提醒

- **基数效应**：低基数上的高增长率可能不代表真正改善（如从 100 涨到 200 是 +100%，但从 10000 涨到 11000 只是 +10%）
- **季节性干扰**：零售、旅游等行业季节性强，同比优于环比
- **工作日差异**：月度数据需注意当月工作日天数不同
- **口径一致性**：对比前确认两期数据统计口径一致
