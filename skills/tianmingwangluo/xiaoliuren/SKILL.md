## 一、小六壬概述

小六壬，又称"袖里乾坤"、"掌中决"，是中国古代传统术数之一。通过月、日、时或随机报数等方法取数，结合特定的掌诀推算，迅速判断事物的吉凶趋势。道传小六壬在传统基础上进行了体系化整理，具备完整的三宫定位、五行生克、六亲、死活六神等分析维度。

**起卦原则**：心诚则灵，一事不二占。同一件事不宜短时间内反复占问。

## 三、起卦算法（三种方式）

### 3.1 数字起卦法（最常用）

**输入**：三个非负整数 num1, num2, num3 + 当前时辰

**算法步骤**：

```
步骤1：数字预处理
  若 num == 0，则视为 10
  否则保持原值

步骤2：三宫推算
  天宫值 = processedNum1 % 6
  地宫值 = (天宫值 + processedNum2 - 1) % 6
  人宫值 = (地宫值 + processedNum3 - 1) % 6

步骤3：值到宫位映射
  0 → 空亡, 1 → 大安, 2 → 留连, 3 → 速喜, 4 → 赤口, 5 → 小吉
```

**示例**：输入 3, 5, 7

- processedNum1 = 3, processedNum2 = 5, processedNum3 = 7
- 天宫值 = 3 % 6 = 3 → 速喜
- 地宫值 = (3 + 5 - 1) % 6 = 7 % 6 = 1 → 大安
- 人宫值 = (1 + 7 - 1) % 6 = 7 % 6 = 1 → 大安

### 3.2 时间起卦法

**输入**：农历月份（1-12）、农历日期（1-30）、时辰序号（1-12）

**算法**：与数字起卦法完全相同，将月份、日期、时辰序号分别作为 num1, num2, num3 代入。

### 3.3 宫位直选法

**输入**：直接指定天宫、地宫、人宫的宫位名称 + 时辰

无需计算，直接使用选定的三宫进行后续分析。

## 五、解卦框架

### 5.1 六步解卦法

完成排盘后，按以下六个步骤进行解读：

**第一步：初判吉凶**
评估人宫落宫的固有吉凶属性，并分析体用生克关系（人宫为体，时辰为用），概述卦象总体趋势。

**第二步：明确定位**
聚焦问题本质，明确卦问，确定太极点/用神，与卦问紧密关联。

**第三步：细析三宫**
分析天宫、地宫、人宫的五行属性及相互生克、格局和气势流向。

- 天宫：代表天时、外部环境、初始状态
- 地宫：代表地利、过程、发展
- 人宫：代表人和、结果、主体

**第四步：活用类象**
提取单宫与合宫的象征意义，结合具体问题进行具象化说明。

**第五步：运用技法**
按需应用转太极、六亲、十二长生、旺衰等进阶技法深化分析。

**第六步：综合判断**
整合前述信息，形成逻辑清晰、具体实用的结论与建议，可给出应期或行动指导。

### 5.2 体用关系

- **体**：指人宫（所落的六神宫位），代表事情本身、现状或主体
- **用**：指起卦时的时辰（地支），代表外部环境、未来动向或作用于体的因素
- 体用生克关系即人宫五行与时辰五行的生克关系

### 5.3 解读注意事项

1. 三宫为主，体用等其他技法为辅
2. 以口语化中文、"理性分析 + 正向引导"的风格进行解读
3. 解读应兼具专业深度与通俗易懂
4. 占卜结果提供趋势和可能性分析，仅供参考

### 大安（木·大吉）

- **象意**：平安，事业顺利，家庭和睦，无忧无虑
- **卜问疾病**：病情轻微，很快会康复
- **卜问婚姻**：婚姻稳定和谐，良缘
- **卜问行人**：平安无事，很快归来
- **卜问事业**：事业顺遂，稳步发展
- **卜问财运**：财源广进，稳定增长
- **卜问寻物**：物品未丢，或容易找回
- **卜问官司**：无忧无虑，或有惊无险

**传统口诀**：身不动时，临大安，事事昌，求财在坤方，失物去不远，宅舍保安康。行人身未动，病者主无妨。将军回田野，仔细更推详。

### 留连（土·中吉）

- **象意**：拖延，徘徊，犹豫不决，事情有波折但终成
- **卜问疾病**：病情反复，痊愈期长
- **卜问婚姻**：有波折，需要耐心等待
- **卜问行人**：行程延迟，归期推迟
- **卜问事业**：发展缓慢，需要耐心
- **卜问财运**：财运平平，来源不稳
- **卜问寻物**：物品在附近，需要仔细寻找
- **卜问官司**：拖延时间长，结果中平

**传统口诀**：卒未归时，临留连，卒未归，求财重迟疑，失物当南寻，官事只宜缓。行人有阻隔，病者受困缠。

### 速喜（火·吉）

- **象意**：事情快速顺利解决，有喜事临门
- **卜问疾病**：病情好转迅速，很快康复
- **卜问婚姻**：恋爱进展快，喜结良缘
- **卜问行人**：即将归来，带来好消息

- **卜问财运**：意外之财，短期收益
- **卜问寻物**：很快找回，在明显处
- **卜问官司**：很快解决，结果较好

**传统口诀**：人即至时，临速喜，人便至，求财向南宜，失物申午未，逢人路上寻。官事有福德，病者无祸侵。田宅六畜吉，行人有音信。

### 赤口（金·凶）

- **象意**：口舌是非，争执纠纷，短期矛盾
- **卜问疾病**：病情严重，或有口腔相关疾病
- **卜问婚姻**：易有争吵，意见不合
- **卜问行人**：途中有阻，或有不好消息
- **卜问事业**：竞争激烈，阻碍多
- **卜问财运**：财务纠纷，收入不稳
- **卜问寻物**：难以找回，或已损坏
- **卜问官司**：结果不利，多有争执

**传统口诀**：官事凶时，临赤口，官事凶，失物急去寻，求财向西方，行人有惊忧。六畜多作怪，病者出西方。更须防咒咀，恐怕染瘟疫。

### 小吉（水·小吉）

- **象意**：小有顺利，事情虽小但顺利完成
- **卜问疾病**：小病小痛，不会太严重
- **卜问婚姻**：关系平稳，小有波折

### 空亡（土·大凶）

- **象意**：虚无，难成，阻碍多，事情难以达成
- **卜问疾病**：病情复杂，难以诊断或治疗

- **卜问财运**：财源枯竭，投资失败

- **卜问官司**：不利结果，或毫无进展

**传统口诀**：音信稀时，临空亡，音信稀，求财无利益，失物寻不见，官事有刑伤。病人主凶险，行人有灾殃。畜生多损失，田宅守空房。

**Q: 输入数字 0 怎么处理？**

**Q: 留连和小吉的五行到底是什么？**
A: 道传小六壬体系：留连属**土**，小吉属**水**。这是本技能的核心校正内容，与网络上大量流传的版本不同。

**Q: 体用关系怎么判断？**
A: 人宫为体（代表事情本身），时辰为用（代表外部环境）。以人宫五行为"我"，看时辰五行与"我"的生克关系。

**Q: 活六神怎么快速排？**

**Q: 三宫分别代表什么？**
A: 天宫代表天时/外部环境/事情起因，地宫代表地利/过程/发展变化，人宫代表人和/结果/事情主体。
