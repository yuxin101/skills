# 小六壬起卦验证示例

以下示例可用于验证 AI 代理的起卦算法是否正确。

---

## 示例1：数字起卦 (3, 5, 7)

**输入**：num1=3, num2=5, num3=7

**计算过程**：
- processedNum1 = 3（非0，保持）
- processedNum2 = 5（非0，保持）
- processedNum3 = 7（非0，保持）
- 天宫值 = 3 % 6 = **3** → **速喜**
- 地宫值 = (3 + 5 - 1) % 6 = 7 % 6 = **1** → **大安**
- 人宫值 = (1 + 7 - 1) % 6 = 7 % 6 = **1** → **大安**

**排盘结果**：
- 天宫：速喜（火）
- 地宫：大安（木）
- 人宫：大安（木）

**六亲**（人宫 木 为"我"）：
- 天宫·速喜（火）：我生 → **子孙**
- 地宫·大安（木）：同五行 → **兄弟**
- 人宫·大安（木）：**我**

## 示例5：活六神验证（午时）

**时辰**：午时 → 活六神起于**大安**

六宫顺序：大安(0)→留连(1)→速喜(2)→赤口(3)→小吉(4)→空亡(5)
活六神序：青龙(0)→朱雀(1)→勾陈(2)→白虎(3)→玄武(4)→腾蛇(5)

起始宫 = 大安(index=0)，映射：
- 大安：(0-0+6)%6 = 0 → **青龙**
- 留连：(1-0+6)%6 = 1 → **朱雀**
- 速喜：(2-0+6)%6 = 2 → **勾陈**
- 赤口：(3-0+6)%6 = 3 → **白虎**
- 小吉：(4-0+6)%6 = 4 → **玄武**
- 空亡：(5-0+6)%6 = 5 → **腾蛇**

## 示例7：时间起卦（农历正月初五 卯时）

**输入**：月=1, 日=5, 时辰序号=4（卯时）

**计算过程**：
- processedNum1 = 1
- processedNum2 = 5
- processedNum3 = 4
- 天宫值 = 1 % 6 = **1** → **大安**
- 地宫值 = (1 + 5 - 1) % 6 = 5 % 6 = **5** → **小吉**
- 人宫值 = (5 + 4 - 1) % 6 = 8 % 6 = **2** → **留连**

**排盘结果**：
- 天宫：大安（木）
- 地宫：小吉（水）
- 人宫：留连（土）
- 时辰：卯时（木）

**六亲**（人宫 土 为"我"）：
- 天宫·大安（木）：木克土，他克我 → **官鬼**
- 地宫·小吉（水）：土克水，我克他 → **妻财**
- 人宫·留连（土）：**我**
- 时辰·卯时（木）：木克土，他克我 → **官鬼**
