# Content Idea Generator - 内容创意生成器

帮助内容创作者系统化地生成高质量选题，永远不缺内容灵感。

## 核心功能

- 8 种选题生成方法（痛点、热点、对比、清单、故事、争议、问答、趋势）
- 5 维度选题验证（需求度、差异化、可执行性、传播力、长期价值）
- 选题模板库（教程、评测、故事、观点、盘点）
- 热点追踪方法
- 选题储备策略

## 适用场景

- 不知道写什么内容
- 需要批量生成选题
- 想要找到差异化角度
- 建立选题储备库
- 结合热点创作内容
