## 二、CMYK色彩规范（核心重点）

### 安全色值范围

| 颜色类型 | C少M少Y少K少 | 说明 |
|----------|-------------|------|
| 鲜红 | C0 M100 Y100 K0 | Safe，正红 |
| 深红 | C0 M100 Y100 K20 | Safe |
| 鲜黄 | C0 M0 Y100 K0 | Safe，正黄 |
| 橙黄 | C0 M30 Y100 K0 | Safe |
| 鲜绿 | C100 M0 Y100 K0 | Safe |
| 深绿 | C100 M0 Y100 K30 | Safe |
| 鲜蓝 | C100 M50 Y0 K0 | Safe |
| 深蓝 | C100 M100 Y0 K30 | Safe |
| 紫色 | C50 M100 Y0 K0 | Safe |
| 橙色 | C0 M70 Y100 K0 | Safe |

### 危险色值（印刷中易出问题的颜色）

| 颜色 | 问题 | 建议 |
|------|------|------|
| C100 M100 Y100 K100 | 四色黑，干燥慢、粘脏 | 用 C60 M40 Y40 K100 替代 |
| C0 M0 Y0 K10 以下 | 10%以下网点不结实 | 最低不低于 K15 |
| 纯C色块 C100 | 色差大，易偏蓝 | 加入 M5-10 Y5-10 调整 |
| 纯Y色块 C0 M0 Y100 | 容易粘脏 | 减少Y墨量或加防粘脏粉 |
| 大面积深色底 | 干燥慢，容易粘脏 | 使用快干油墨，减小墨层 |
| 大面积渐变 | 容易出现杠子 | 渐变控制在 30% 以内 |

## 三、各类印刷品设计规范

### 名片（标准尺寸：90×54mm，裁切线外扩3mm）

```
成品尺寸：90×54mm
裁切线外：93×57mm（各边扩3mm）
出血尺寸：93×57mm @ 300dpi
颜色：4+4C（双面四色）
字体：正文不低于6pt，英文不低于5pt
内容安全区：86×50mm（内缩2mm）
```

**名片的AI设计要点：**
- 标志(LOGO)：CMYK模式，300dpi，AI/EPS格式最佳
- 文字内容：姓名/职位/公司/电话/邮箱/地址
- 公司色彩：确认品牌色CMYK值，不确定时用标准色
- 背面设计：通常放公司地址/产品/二维码

### 画册/宣传册（常见尺寸）

| 类型 | 成品尺寸 | 全开出血尺寸 | 说明 |
|------|----------|-------------|------|
| 大16开 | 210×285mm | 216×291mm | 书籍/手册 |
| 大32开 | 148×210mm | 154×216mm | 口袋书 |
| A4横版 | 297×210mm | 303×216mm | 画册/型录 |
| A5竖版 | 148×210mm | 154×216mm | 手册/便签 |

**画册AI设计要点：**
- 封面/封底：出血多扩3mm
- 内页：留装订边（靠装订线内侧15mm内不放文字）
- 跨页图：左右页面各出血3mm，中间留2mm接版余量
- 书脊：封面宽度+内页厚度+封底宽度

### 包装彩盒（常见刀版尺寸）

| 盒型 | 展开尺寸（参考） | 说明 |
|------|----------------|------|
| 天地盖（首饰盒） | (宽+高×2+30) × (深+高×2+30) | 纸盒常用 |
| 封套盒 | (宽+高+30) × (高+30) | CD盒/卡片盒 |
| 展示盒（吊卡） | 宽×高+舌头，展开宽=宽+高×2 | 挂架展示 |
| 手提袋 | (宽+侧宽×2+40) × (高+底宽+提手高+30) | 手提袋 |

**包装盒AI设计要点：**
- 刀版线：设计文件必须提交AI格式，线条加粗0.25pt
- 色块：色块边缘离开刀版线≥5mm
- 粘口位：不干胶标签需标注粘口方向
- 条形码：必须按实际尺寸生成，颜色对比度≥75%

### DM单/宣传单

```
标准尺寸：A4(210×297mm) / A5(148×210mm) / DL(220×110mm)
出血尺寸：各边扩3mm
颜色：4+0C或4+4C
折法需标注（如果有）
```

### 不干胶标签

| 类型 | 常用尺寸 | 说明 |
|------|----------|------|
| 圆形标签 | φ30-100mm | 直径 |
| 方形标签 | 20×20mm起 | 最小 |
| 矩形标签 | 任意mm | 最常用 |
| 透明标签 | 任意mm | 留出血 |

## 四、字体使用规范

### 印刷字体安全使用

| 字体类型 | 风险 | 建议 |
|----------|------|------|
| 宋体/黑体/楷体 | 低，Windows自带 | 正文可用 |
| 微软雅黑 | 中，等宽问题 | 仅限屏幕显示，打印建议改用方正系列 |
| 书法字体 | 高，笔画细处断裂 | 必须转曲线（outline） |
| 外文字体 | 高，缺字风险 | 提交前转曲线 |
| 特殊装饰字体 | 高，版权+兼容问题 | 必须转曲线 |

### 字体转曲线的操作

**在AI中：**
文字 → 文字属性 → 创建轮廓（Type → Create Outlines）
快捷键：Ctrl+Shift+O（Windows）

**在CorelDRAW中：**
安排 → 转换为曲线（Arrange → Convert to Curves）

### 字体字号安全规范

| 用途 | 最低字号 | 说明 |
|------|----------|------|
| 标题 | 14pt+ | 黑体/雅黑 |
| 正文 | 8pt+ | 宋体 |
| 说明文字 | 7pt+ | 宋体/黑体 |
| 英文正文 | 7pt+ | Arial/Helvetica |
| 5号字 ≈ 10.5pt | 3.67mm高度 | 文字对角线 |

## 五、图像分辨率规范

| 印刷类型 | 分辨率要求 | 模式 |
|----------|----------|------|
| 报纸/大幅海报 | 150-200dpi | CMYK |
| 普通印刷品（画册/DM） | 300dpi | CMYK |
| 精细印刷品（名片/包装） | 300-350dpi | CMYK |
| 丝网印刷 | 200-300dpi | 根据网版线数调整 |
| 数码印刷 | 300dpi | RGB或CMYK均可 |

> ⚠️ **图片不能通过提高分辨率来修复模糊**——72dpi的图片拉大到300dpi输出仍然模糊，必须用原始大图！

## 六、文件输出格式规范

### 推荐格式优先级

| 格式 | 适用场景 | 说明 |
|------|----------|------|
| **PDF/X-1a** | 首选！交给印刷厂 | 印刷行业标准格式，字体/图片全部内嵌 |
| **AI** | 源文件格式 | 需打包所有链接，文字转曲线 |
| **EPS** | 矢量图输出 | 图片和矢量混排时用 |
| **TIFF** | 位图输出 | 300dpi，CMYK，不压缩 |
| **JPG** | ⚠️ 不推荐 | 有损压缩，色彩损失，只在无选择时用 |

### PDF/X-1a导出检查清单（InDesign）

```
□ 色彩模式：全部转为CMYK（或PDF/X-1a标准）
□ 字体：全部嵌入（Embed Fonts）
□ 图片：300dpi，CMYK，无RGB链接
□ 出血：各边3mm设置正确
□ 标签/裁切线：添加裁切标记（Crop Marks）
□ 压印黑色：黑色文字/色块叠印设置正确
□ PDF兼容性：Acrobat 4.0（PDF/X-1a:2001）或更高
□ 压缩设置：无损压缩（Flate Compression）
□ ICC配置文件：是否指定了输出配置文件
```

## 七、印前检查清单（交稿前必查）

### 视觉检查

```
□ 所有文字是否已转曲线（outline）？
□ 字体是否嵌入？有无缺字体提示？
□ 图片是否300dpi，CMYK模式？
□ 颜色是否有超出安全色域的颜色？
□ 文字/重要内容是否在安全区内（离开裁切线≥5mm）？
□ 出血是否各边≥3mm？
□ 是否有大于10%面积的纯Y色块？（容易粘脏）
□ 渐变是否有明显的杠子或断层？
□ 黑色文字/色块是否设置为叠印（overprint）？
□ 角线/裁切线是否正确？
□ 套准标记/色彩控制条是否添加？
□ 文档尺寸是否与成品尺寸一致？
□ 有多页的话，页码顺序是否正确？
□ 是否有不应该出现的白页或缺页？
```

### 数值检查

```
□ 出血数值：各边 ≥ 3mm
□ 安全区：内容离裁切线 ≥ 5mm
□ 分辨率：图像 ≥ 300dpi（输入图）
□ 字体大小：正文 ≥ 8pt，英文 ≥ 7pt
□ 线宽：最细线条 ≥ 0.25pt（0.09mm），否则无法印刷
□ 色值：C/M/Y/K 均在 0-100% 范围内
□ 黑色叠印：K100文字需叠印预览确认
□ 四色黑检查：C/M/Y/K均为100的区域，干燥极慢！
```

## 十、客户设计需求快速分析流程

### 第一步：识别印刷品类型

```
名片？→ 检查尺寸90×54mm，是否双面，颜色套数
画册？→ 确定P数/装订方式/是否跨页图
包装盒？→ 确认盒型/材质/后加工工艺
DM单？→ 确定尺寸/折法/发放方式
```

### 第二步：提取关键设计参数

```
颜色数量 → 影响报价和制版费
尺寸大小 → 影响纸张利用率
材质选择 → 影响视觉效果和成本
后加工 → 增加附加值，区分度
```

### 第三步：给出设计方案建议

```
主色调：建议2-3种颜色搭配
字体：建议无衬线/衬线字体组合
版面：建议网格系统/黄金比例/自由排版
重点：突出什么信息？
```

### 第四步：确认可实现性

```
图片分辨率是否够用？
特殊工艺是否有档期？
交稿时间是否充足？
```
