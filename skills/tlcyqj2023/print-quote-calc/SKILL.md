## 二、报价流程（Step-by-Step）

### 第一步：收集参数

请向用户确认以下信息（可缺少，但需标注）：

1. **印刷品名称**：（如"名片"）
2. **纸张类型**：（如"300g铜版纸"）
3. **尺寸**：（如"A4"或"90×54mm"）
4. **印数**：（如"500张"）
5. **颜色**：（如"4+4C"即双面四色，"4+0C"即单面四色）
6. **后加工工艺**：（如"覆哑膜+烫金"）
7. **收货方式**：（自取/送货/快递）

### 第二步：计算纸张费用

```
每张全张纸价 = 纸张令价 ÷ 500
总用纸令数 = 印数 ÷ 每张全张出片数
纸张费用 = 纸张令价 × 总用纸令数
损耗加放 = 纸张费用 × 8%（行业惯例）
```

**加放率参考**（印刷行业惯例）：
- 2000张以下：加放10-15%
- 2000-5000张：加放8%
- 5000张以上：加放5%

### 第三步：计算印刷费

```
PS版费用（元）= 色令数 × 35-50元/色令（含输出CTP）
印刷工费 = 印数 ÷ 500 × 印刷单价（元/色令）
最低开机费 = 800-1500元（不够按最低算）
```

### 第四步：后加工费

```
覆膜费 = 印数 × 覆膜单价
烫金费 = 印数 × 烫金单价（特殊大版额外加200-500元刀版费）
装订费 = 印数 ÷ 装订单位 × 装订单价
模切费 = 刀版费 + （印数 ÷ 1000 × 模切单价）
```

### 第五步：合计与利润

```
直接成本 = 纸张费 + 印刷费 + 后加工费 + 包装运费
利润（30-40%）= 直接成本 × 35%
报价 = 直接成本 + 利润
```

## 五、注意事项

1. **低于最低开机费**：无论印多少，印刷费不低于800-1500元（按色令算）
2. **特种纸报价**：特种纸损耗较高，建议加放12-15%
3. **大活件折扣**：5000张以上可在利润上加5-10%折扣
4. **赶工费**：交货期<3天，加收30-50%赶工费
5. **异地送货**：按距离单独计费
6. **税点**：含税普通发票+6%，增值税专用发票+9%

## 六、常见问题快速处理

| 问题 | 处理建议 |
|------|----------|
| 客户说"别家比你便宜30%" | 核实别家是否含税/含运费/同材质，可适当让3-5% |
| 印数低于最低开机量 | 建议客户数码印刷（便宜但品质略低） |
| 大批量订单犹豫 | 告知阶梯报价，印越多越便宜 |
| 不确定工艺选择 | 提供2-3个方案：经济款/标准款/高端款 |
| 客户要打样 | 打样费=报价×20-30%，大货可抵扣 |
