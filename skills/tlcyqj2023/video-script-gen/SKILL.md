## 四、话术技巧（让你的脚本更有感染力）

### 语气词使用
- 开头加："说实话"、"不瞒你说"、"作为一个干了XX年的老兵"
- 转折加："但是"、"没想到的是"、"最关键的是"
- 强调加："记住了"、"这个很重要"、"敲黑板"
- 互动加："对不对？"、"你有没有遇到过？"、"评论区告诉我"

### 停顿标记
在以下地方标注"[停顿]"，让画面有呼吸感：
- 抛出问题后
- 揭晓答案前
- 强调关键词时

```
"所以印刷成本里，最大的一块不是纸张，而是[停顿]开机费。"
```

### 数字具象化
```
"一张A4纸，纸钱只要1毛钱——但开机费至少800块。"
"这个差价，相当于[停顿]一张名片和一台iPhone的距离。"
```
