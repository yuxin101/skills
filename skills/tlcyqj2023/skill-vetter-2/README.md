# 安全审核标准 - 自动化版

> 自动化扫描 Skills 目录，检测风险代码，给出修复建议

## 功能

- ✅ **自动扫描** — 遍历 `/workspace/skills/` 目录所有文件
- ✅ **分级检测** — 严重/高危/中危/低危 四级风险关键词
- ✅ **智能白名单** — 自动识别已知安全工具，避免误报
- ✅ **修复建议** — 每条风险附带具体修复方案
- ✅ **双格式报告** — Markdown（人类可读）+ JSON（程序处理）
- ✅ **自身免疫** — 扫描器不扫描自身，排除报告文件

## 风险等级

| 评分 | 等级 | 建议 |
|------|------|------|
| 81-100 | 🟢 低风险 | 允许安装 |
| 61-80 | 🟡 中风险 | 谨慎安装，运行时监控 |
| 41-60 | 🟠 较高风险 | 沙箱测试，限制权限 |
| 21-40 | 🔴 高风险 | 禁止安装，查看源码 |
| 0-20 | ⛔ 严重风险 | 立即删除 |

## 检测内容

**严重风险**：`eval()` / `exec()` / `os.system()` / `subprocess` 等代码执行
**高危风险**：`requests` / `urllib` / `socket` 等网络请求
**中危风险**：`chmod` / `sudo` / `.ssh/` 等系统权限
**低危风险**：`base64` / `gzip` 等数据编解码

## 使用方法

### 直接运行
```bash
python3 scan_skills.py
```
