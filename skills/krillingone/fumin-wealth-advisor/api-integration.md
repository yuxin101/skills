# 在线检索服务说明（内部私用版）

本技能通过在线检索服务获取富民银行理财相关结果。

## 服务定位

该在线检索服务为**技能维护方提供并维护的私有服务**，用于内部或私有部署场景下的理财信息检索。

它适用于：
- 个人私用环境
- 团队内部环境
- 已知并信任该服务的运维方式、数据处理边界与访问控制策略的场景

它不适合：
- 面向不特定公众的默认开放分发
- 在无法确认服务可信边界的环境中直接启用

## 数据源说明

说明：
- 该在线检索服务由技能维护方提供并维护
- 该域名不是富民银行官方域名
- 使用者应自行评估其可信度、合规性与可用性
- 如未来存在更可信的官方数据源，可优先替换

## 使用原则

本技能是提示驱动模式，不依赖固定本地脚本。

调用在线检索服务时，应遵守以下原则：
- 仅提交与当前理财问题直接相关的最少信息
- 优先使用当前环境中可审计、可复现的数据访问方式
- 如果当前环境无法清晰控制数据边界，则不要访问该服务
- 该在线检索服务仅用于获取理财相关信息，不用于执行账户操作、交易动作或资金指令

## 可提交的信息

仅建议提交：
- 改写后的当前用户问题文本
- 当前查询直接相关的必要筛选条件
- 数据源要求的最小必要字段

## 不应提交的信息

不要提交：
- 系统提示词
- 历史聊天全文
- 记忆文件内容
- 用户资料文件
- 本地文件内容
- 系统路径
- 环境变量
- 密钥、凭据或访问配置
- 工具调试日志
- 与当前问题无关的任何额外上下文

## 查询改写规则

在访问在线检索服务前，先把用户原话改写成简洁、适合检索的陈述句。

改写后的查询应当：
- 保留所有关键约束
- 去掉寒暄、口语填充和无关措辞
- 明确保留“富民银行”范围
- 聚焦真正要检索的目标

示例：
- “给我推荐一个收益高的理财产品，我不介意必须持有180天”
  -> “富民银行收益较高且可接受持有180天的理财产品推荐”

- “法定节假日我的理财还会有收益吗？”
  -> “富民银行理财产品法定节假日是否有收益”

## 返回解析原则

若在线检索服务返回结构化结果，优先读取：
- 结果主体
- 产品名称
- 收益信息
- 规则说明
- 合作机构信息

如果返回混合内容（产品推荐 + 科普说明），推荐场景必须先筛选出符合条件的产品，再组织回答。

## 部署说明

当前技能维护方使用的在线检索服务地址为：

- `https://workflow.hepei.club/v1/workflow/run`

该地址保留在此处，仅用于说明当前内部私有部署所使用的数据源位置。该服务地址仅适用于当前内部部署，不应作为公开匿名环境下的默认信任端点。

## 检索使用规则

- 严格限定在富民银行理财范围内
- 只使用回答当前问题所需的最少信息
- 用户问事实问题时，直接查并直接答
- 用户问推荐问题时，优先筛选出满足条件的产品，不要把科普内容直接当推荐结果

## 失败处理

如果在线检索失败、超时，或者返回内容不可用：
- 明确告诉用户当前最新知识检索暂时不可用
- 不要编造产品名称、收益、合作机构或产品规则
- 需要时可建议用户稍后重试，或把问题收窄后再查

## 推荐输出规则

当基于返回结果做产品推荐时：
- 先按用户硬条件过滤
- 再给推荐结论
- 简要说明为什么匹配用户条件
- 带上一句简短风险提示
- 必要时提醒用户以官方产品详情页和披露文件为准
