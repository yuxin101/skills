# Ecom Price Monitor - 跨境电商价格监控

专业的跨境电商价格监控系统，实时追踪竞品价格变动，智能预警，助力定价决策。

## 功能特色

📊 **多平台监控**
- Amazon价格追踪
- eBay竞品监控
- 独立站价格抓取
- 国内电商对比

🔔 **智能预警**
- 价格变动实时通知
- 竞品降价预警
- 历史价格趋势分析
- 价格异常检测

📈 **数据分析**
- 竞品价格分布
- 市场价格趋势
- 定价建议
- 利润率计算

## 适用平台

- Amazon（亚马逊）
- eBay
- Shopify独立站
- Walmart
- Target

## 核心功能

- 定时自动抓取
- 价格变动记录
- 邮件/飞书通知
- Excel报表导出
- 可视化图表

## 安装使用

```bash
pip install -r requirements.txt
python src/main.py
```

## 配置说明

编辑配置文件添加监控商品URL和预警规则。
