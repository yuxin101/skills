# taste-impressionism

Aesthetic skill for AI agents — Impressionism movement's visual language.

## Installation

```bash
npx clawhub@latest install taste-impressionism
```

## What is a Taste Skill?

Taste Skills are structured aesthetic knowledge packages designed for AI agents. They encode an art movement's visual language into machine-readable tokens, enabling AI to generate, evaluate, and refine creative content with authentic stylistic direction.

## What's Included (Preview)

- **10 style tokens** — Broken color, atmospheric perspective, plein air, natural light, and more
- **6 anti-tokens** — What to avoid when channeling Impressionism
- **Creative direction** — How to apply these tokens in prompts

## Full Version

The complete Impressionism Taste Skill (available at [InspiredHub](https://inspiredhub.ai/taste/impressionism-movement)) includes color palette, application rules for 5 creative domains, evaluation criteria, and key artists analysis.
