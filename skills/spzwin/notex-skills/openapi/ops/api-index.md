# OpenAPI：OPS 模块索引

本模块接口一览：

1. `POST /openapi/ops/ai-chat`
   - **作用**：OPS 运营数据问答，基于底层运营数据（用户行为、模块统计、告警等）以自然语言回答运营洞察问题。
   - **关键入参**：`message`（用户问题，字符串）
   - **关键返回**：`reply`（自然语言回答）、`historyCount`（历史问答轮数）
   - **鉴权**：Header `access-token`
   - **超时上限**：300000ms（单次请求）
   - **适用场景**：查用户活跃、模块使用率、告警分析、组织/部门维度分析等运营问题
   - **详细文档**：`./ai-chat.md`

脚本映射：
- `../../scripts/creator/skills_run.py`（同脚本支持 `ops-chat`，可独立执行）
- 执行前请先阅读上方接口文档获取完整入参说明
