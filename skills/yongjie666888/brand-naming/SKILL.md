# 品牌命名助手

## 好名字的标准

| 标准 | 说明 |
|------|------|
| 好记 | 简短，2-4个字 |
| 好听 | 朗朗上口，无不良谐音 |
| 好写 | 笔画简单，易于书写 |
| 独特 | 有辨识度，不易混淆 |
| 合法 | 无商标冲突，可注册 |

## 命名方向

### 1. 寓意型
| 方向 | 示例 | 说明 |
|------|------|------|
| 愿景 | 腾达、致远 | 寓意公司发展 |
| 价值 | 品质、诚信 | 体现核心价值 |
| 品质 | 卓越、非凡 | 追求卓越 |
| 自然 | 青山、绿水 | 环保自然 |

### 2. 音译型
| 原文 | 中文 | 特点 |
|------|------|------|
| Tesla | 特斯拉 | 直接音译 |
| Google | 谷歌 | 音译+中国化 |
| Alibaba | 阿里巴巴 | 音译+文化内涵 |

### 3. 组合型
| 组合方式 | 示例 |
|----------|------|
| 英文+中文 | XX科技 |
| 数字+文字 | 360安全 |
| 两个汉字 | 华为、大疆 |

### 4. 创新型
| 类型 | 示例 |
|------|------|
| 自造词 | 谷歌、抖音 |
| 叠词 | 陌陌、脉脉 |
| 动植物 | 蚂蚁、小米 |

## 电机相关命名参考

| 方向 | 示例 |
|------|------|
| 电机属性 | 劲、恒、速、稳 |
| 技术词汇 | 磁、电、驱、控 |

| 可靠感 | 安、恒、信、诺 |

```markdown
- 缺点：XXX

| 标准 | 方案1 | 方案2 | 方案3 |

## 推荐方案
推荐使用：**XXX**

```

## 命名禁区

| 禁区 | 示例 | 原因 |
|------|------|------|
| 与知名品牌相似 | 特拉斯 | 商标风险 |
| 不良谐音 | 长安（长胖） | 负面联想 |
| 生僻字 | 覔、龘 | 难认难记 |
| 地域歧视 | XX人 | 政治风险 |
| 行业忌讳 | 翻、裂 | 负面联想 |

## 域名检查

| 检查项 | 说明 |
|--------|------|
| .com | 首选 |
| .cn | 国内备选 |
| .com.cn | 备选 |
| 长度 | 越短越好 |
