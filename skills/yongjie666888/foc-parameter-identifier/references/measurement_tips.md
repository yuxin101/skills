# FOC参数辨识实测技巧

## 电阻测量技巧

### 四线制测量法（推荐）

普通万用表两线制测量会包含线缆电阻，大电流电机误差显著。

```
四线制接线：
  电流端：Source+ Source- （注入电流）
  电压端：Sense+ Sense- （仅测量电压，无电流流过）

如果只有普通万用表：
  1. 测量线间电阻 R_AB
  2. 测量线间电阻 R_BC
  3. 测量线间电阻 R_CA
  4. 相电阻 R_phase = (R_AB + R_BC + R_CA) / 6
     （数学推导：R_AB = R_A + R_B, etc.）
```

### 温度影响

```
铜电阻温度系数：+0.393%/℃（25℃基准）
铝电阻温度系数：+0.4%/℃

实测温度 → 20℃换算：
R_20 = R_T / [1 + 0.004 × (T - 20)]

高温工作区（如 75℃）：
75℃电阻 = 20℃电阻 × 1.22（约 +22%）
```

## 电感测量技巧

### 阶跃响应法实测波形

```
示波器设置：
  CH1：电流采样电阻电压（I = V_sense / R_sense）
  CH2：注入电压波形
  时基：1~10ms/div（根据电感大小调整）

读取 dI/dt：
  - 在电流上升的线性段（0~63%稳态值前）
  - 线性段斜率 = dI/dt
  - L = V / (dI/dt)

电感小的电机（<1mH）：用更窄脉宽（<1ms），更高电压
电感大的电机（>10mH）：用更宽脉宽（>10ms），更低电压
```

### Ld/Lq 区分技巧

```
对于内嵌式 PMSM（Ld < Lq）：
  - Ld 测量：转子 d轴对准定子磁场轴线时测量
  - Lq 测量：转子 q轴对准定子磁场轴线时测量
  - 机械操作：手动微调转子位置，观察电流响应大小变化

简化方法（表贴式电机）：
  - Ld ≈ Lq，直接用 L_phase 即可
  - 误差 < 5% 时 FOC 性能影响不大
```

## 反电动势测量技巧

### 手动旋转测速

```
方法1：用角磨机/电钻粗略旋转，数波形周期
方法2：用Dr. BLDC或小型伺服，让电机空转测输出电压
方法3：用另一台电机驱动被测电机（推荐，转速稳定）

转速稳定是关键：
  - 转速波动 → E_rms 测量误差大
  - 建议用稳速驱动或大型电机拖动
```

### 转速测量

```
光电转速表：对着转轴反光贴测量
示波器频率计数：
  - 反电动势周期 T（s）
  - 转速 n = 60 / (T × np) rpm
  - 例如：8极电机（np=4），T=5ms
  - n = 60 / (0.005 × 4) = 3000 rpm
```

## 极对数判断技巧

### Hall信号计数法

```
1. 示波器接任意一相 Hall 输出
2. 用手慢慢旋转一圈（约5秒）
3. 数 Hall 方波的上升沿数量
4. Hall 计数 = 2 × np（每极对产生2个边沿）
5. 例如：Hall 上升沿 = 16 → np = 8

注意：Hall 传感器有 A/B/C 三路，但任意一路都能判断
```

### 反电动势波形计数法

```
1. 示波器测任意两相间电压
2. 用手慢慢旋转一圈
3. 数电压正弦波的完整周期数
4. 周期数 = np

例如：旋转一圈看到4个完整正弦波 → np = 4
```

## Ke 和 Kt 换算关系

```
电气关系：
  Ψm = Φ × Nph × Kw（总磁链）
  E_phase = ω × Ψm（相电动势幅值）
  E_line = √3 × ω × Ψm（线间电动势幅值）

转矩关系：
  T_e = (3/2) × np × Ψm × I_q
  Kt = (3/2) × np × Ψm （Nm/A）

Ke 和 Kt 数值关系（有效值）：
  Ke_line_rms = Kt × 2π × np / (3 × √2) 
              ≈ Kt × 2.95 × np × 0.001  (V/Hz 对应关系)
  简化：Ke[V/(rad/s)] = Kt[Nm/A] / np

工程速算：
  np=4, Kt=0.1Nm/A → Ke = 0.025 V/(rad/s)
  或用转速反推：E_rms = Ke × ω_e × np / √3
```
