# 专利文案撰写助手 v1.3

针对电机/机械领域，提供专利申请全套文档的撰写指导，包括技术交底书、权利要求书、说明书。

## v1.3 更新
- 新增完整权利要求书模板（含独立权利要求+从属权利要求）
- 新增电机领域专利实例（无刷力矩电机磁路结构）
- 新增专利新颖性/创造性自我评估方法

---

## 专利类型选择

| 类型 | 保护客体 | 审查周期 | 费用 | 适用场景 |
|------|---------|---------|------|---------|
| 发明专利 | 技术方案（方法/结构） | 2~3年 | 高 | 核心创新、结构性改进 |
| 实用新型 | 形状/结构新方案 | 6~8月 | 低 | 改进型结构、组件创新 |
| 外观设计 | 产品外观造型 | 4~6月 | 中 | 外观创新、差异化竞争 |

> **电机领域建议**：核心磁路创新申请发明专利，结构改进申请实用新型，两者可以同日申请（先申请实用新型，再申请发明）。

## 文件类型说明

| 文件 | 说明 | 核心要点 |
|------|------|---------|
| 技术交底书 | 发明人→代理人的技术说明 | 技术问题、解决方案、有益效果 |
| 权利要求书 | 界定专利保护范围 | 层次清晰、概括适当 |
| 说明书 | 完整技术说明 | 充分公开、支持权利要求 |
| 摘要 | 简要说明（≤300字） | 独立阅读、理解发明 |

## 完整权利要求书模板

### 示例：无刷力矩电机磁路结构

```
权利要求书

1. 一种无刷力矩电机磁路结构，包括：
   定子铁芯（1），其上设有多个槽口；
   永磁体（2），安装在所述定子铁芯的槽口内；
   其特征在于：所述永磁体（2）沿圆周方向呈Halbach阵列排布，
   使得第一侧面的磁密高于第二侧面磁密，
   第一侧面为电机气隙侧，第二侧面为定子铁芯侧。

2. 根据权利要求1所述的无刷力矩电机磁路结构，
   其特征在于：相邻两块永磁体的充磁方向夹角为30°~60°。

3. 根据权利要求1或2所述的无刷力矩电机磁路结构，
   其特征在于：所述永磁体（2）的第一侧面设有弧形倒角（3），
   弧形倒角的曲率半径为永磁体厚度的0.1~0.3倍。

4. 根据权利要求3所述的无刷力矩电机磁路结构，
   其特征在于：所述弧形倒角（3）的曲率半径为永磁体厚度的0.2倍。

5. 一种无刷力矩电机，包括权利要求1~4任一项所述的磁路结构。

6. 根据权利要求5所述的无刷力矩电机，
   其特征在于：所述定子铁芯（1）的槽口数为36，
   极对数为6，槽极比为6:1。
```

### 权利要求书撰写要点

```
独立权利要求（权1）：
- 解决一个技术问题
- 覆盖所有实现方式（概括适当）
- 包含必要技术特征（解决技术问题所不可或缺的特征）
- 格式："一种XXX，包括A、B、C，其特征在于：D"

从属权利要求（权2~N）：
- 引用在前权利要求
- 增加附加技术特征
- 进一步限定保护范围
- 格式："根据权利要求X所述的XXX，其特征在于：Y"
```

## 技术交底书完整模板

### 第一部分：基本信息

```
【发明名称】
一种XXX电机/结构/方法

【申请人】
姓名/单位名称：_______________
联系方式：_______________

【发明人】
姓名：_______________

【申请类型】
□ 发明专利    □ 实用新型    □ 两者同时申请

【优先级声明】
□ 要求优先权（申请号：_______）
□ 首次申请
```

### 第二部分：技术领域

```
【技术领域】
本发明涉及_____电机/_____机械/_____控制领域，
具体涉及一种_____（结构/方法/系统）。

示例：
本发明涉及无刷力矩电机技术领域，具体涉及一种
采用Halbach阵列磁路结构的无刷力矩电机。
```

### 第三部分：背景技术（现有技术及不足）

```
【现有技术描述】
目前，_____（描述现有技术的方案）

检索到的最接近现有技术：
- 文献1：CNXXXXXX《XXXX》，公开了_____（描述其技术方案）
- 文献2：USXXXX《XXXX》，公开了_____（描述其技术方案）

【现有技术存在的技术问题】
1. 技术问题1：现有技术中_____，导致_____（效果差/效率低/成本高）
2. 技术问题2：_____不足，具体表现为_____
3. 技术问题3：现有方案存在_____的缺点

示例：
现有技术中，无刷力矩电机普遍采用传统的径向磁化永磁体
布置方式，气隙磁密分布呈正弦波状，谐波含量高，导致
转矩脉动大、噪音高的缺点。
```

### 第四部分：发明内容（核心）

```
【要解决的技术问题】
本发明要解决的技术问题是：_____

【技术方案】
针对上述问题，本发明提供一种_____方案，包括以下技术特征：

1. 主要结构/方法特征：
   - _____
   - _____
   - _____

2. 进一步改进：
   - _____
   - _____

【关键技术特征详细描述】
特征A（主要创新点）：
- 结构：_____
- 原理：_____
- 作用：_____

特征B：
- _____

【有益效果】
与现有技术相比，本发明具有以下有益效果：

1. 技术效果1：_____（具体可量化）
   - 本发明通过_____，实现了_____，
     相比现有技术，_____提升/降低_____%

2. 技术效果2：_____

3. 技术效果3：_____

示例：
与现有技术相比，本发明通过采用Halbach阵列永磁体排布，
使气隙磁密波形更接近正弦分布，3次谐波含量从15%降低至
3%以下，转矩脉动系数从8%降低至2%以下，电机噪音降低5dB。
```

```
【附图说明】

图2是图1中A-A方向的剖面图；
图3是本发明永磁体排布的放大图；

附图标记说明：

```

### 第六部分：具体实施方式

```
【实施方式一】（基本实施例）
如图1~3所示，本发明包括定子铁芯（1）和永磁体（2），
定子铁芯（1）上设有36个槽口，槽口内安装永磁体（2），
永磁体（2）沿圆周方向呈Halbach阵列排布。

优选地，相邻永磁体的充磁方向夹角为45°，
永磁体（2）的第一侧面设有弧形倒角（3），
弧形倒角曲率半径为永磁体厚度的0.2倍。

【实施方式二】（进一步优化）
在实施方式一的基础上，定子铁芯（1）采用0.35mm厚的
50WW270硅钢片叠压而成，叠压系数≥0.98。
永磁体（2）选用N42SH牌号，烧结钕铁硼材料，
剩磁Br≥1.25T，矫顽力Hcb≥920kA/m。

【实施方式三】（另一种应用场景）
在实施方式一的基础上，本发明的磁路结构应用于
内转子结构时，永磁体（2）安装于转子铁芯的表面，
构成表贴式永磁同步电机，适用于无人机旋翼驱动。

【参数范围实施例】
为说明权利要求中参数范围的合理性：
- 充磁方向夹角30°时：气隙磁密波形正弦度85%
- 充磁方向夹角45°时：气隙磁密波形正弦度96%（最优）
- 充磁方向夹角60°时：气隙磁密波形正弦度90%
- 充磁方向夹角90°时：气隙磁密波形正弦度78%（不优选）
故参数范围30°~60°是合理的。
```

## 专利新颖性/创造性自我评估

### 新颖性判断步骤

```
第一步：确定本发明的技术特征
技术特征清单：
- 特征1：Halbach阵列排布（与对比文件1不同）
- 特征2：弧形倒角结构（与所有对比文件不同）
- 特征3：槽极比6:1（与对比文件2不同）

第二步：逐项比对
- 特征1：对比文件1未公开Halbach阵列 → 有区别
- 特征2：对比文件1、2均未公开弧形倒角 → 有区别
- 特征3：对比文件1槽极比为3:1，对比文件2为4:1 → 有区别

第三步：判断是否影响新颖性
→ 存在区别技术特征 → 具备新颖性
```

### 创造性判断步骤

```
第一步：确认区别技术特征
本发明与最接近现有技术的区别特征：
- 区别特征A：Halbach阵列排布
- 区别特征B：弧形倒角结构

第二步：判断区别特征是否被其他对比文件公开
- Halbach阵列排布：未见公开（检索结果）
- 弧形倒角结构：未见公开（检索结果）

第三步：判断是否具有预料不到的技术效果
- 谐波含量降低：从15%降至3%（预料不到）
- 转矩脉动降低：从8%降至2%（预料不到）

第四步：判断是否得到本领域技术人员的启示
- Halbach阵列：属于永磁电机领域已知技术，
  但将其应用于力矩电机并优化倒角结构，
  需要创造性劳动

结论：具备创造性
```

## 专利法条款速查

| 条款 | 内容 | 适用场景 |
|------|------|---------|
| 第22条 | 新颖性、创造性、实用性 | 驳回/无效理由 |
| 第26条 | 说明书应当充分公开发明 | 公开不充分 |
| 第26条3款 | 说明书对权利要求书的支持 | 保护范围不一致 |
| 第26条4款 | 权利要求书应当以说明书为依据 | 概括过宽 |
| 第33条 | 修改不得超出原说明书范围 | 修改超范围 |

## 注意事项

- 专利申请前不要公开技术方案（丧失新颖性）
- 同一发明点同时申请发明+实用新型可获得双重保护
- 权利要求书层次：独立权利要求保护范围最大，从属权利要求逐层缩小
- 说明书必须支持权利要求书，权利要求概括不得超出说明书范围
- 电机领域专利检索常用数据库：CNPAT、SooPAT、Patentics、Google Patents
