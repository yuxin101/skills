# 电机图纸 GD&T（形位公差）参考规范

## 常用形位公差项目（GB/T 1182）

| 符号 | 名称 | 含义 | 电机应用场景 |
|------|------|------|-------------|
| ⬜ | 圆度 | 圆要素对理想圆的允许变动量 | 轴承位、转子外圆 |
| ◎ | 圆柱度 | 圆柱要素对理想圆柱的允许变动量 | 轴、轴承位 |
| — | 直线度 | 直线要素对理想直线的允许变动量 | 轴线 |
| ⌒ | 平面度 | 平面要素对理想平面的允许变动量 | 端盖止口、端面 |
| ⬡ | 圆跳动 | 圆要素对基准轴线的允许跳动量 | 轴承位、转子外圆 |
| ⟱ | 全跳动 | 圆柱/端面对基准轴线的允许跳动量 | 整体跳动检测 |

## 电机典型形位公差要求

| 部位 | 形位公差 | 等级/数值 | 依据 |
|------|---------|----------|------|
| 定子内圆 | 圆跳动 | ≤0.03mm | 气隙均匀性 |
| 转子外圆 | 圆跳动 | ≤0.03mm | 气隙均匀性 |
| 轴承位 | 圆柱度 | IT6（0.006~0.016mm）| 配合精度 |
| 轴承位 | 圆跳动 | ≤0.01mm | 旋转精度 |
| 轴伸 | 圆跳动 | ≤0.03mm | 联轴器对中 |
| 键槽 | 对称度 | ≤0.03mm | 键受力均匀 |
| 止口端面 | 平面度 | ≤0.02mm | 密封 |
| 端盖配合面 | 圆跳动 | ≤0.03mm | 同轴度 |

## 基准体系

```
典型电机基准体系：
基准A：定子内圆中心轴线（φA）
基准B：机座止口面（面B）
基准C：键槽中心平面（线C）

标注示例：
⬡0.03 A  → 转子外圆对基准A的跳动≤0.03mm
⌒0.01 A  → 轴承位圆柱度IT6
—0.02 A  → 轴伸直线度≤0.02mm
```

## 公差配合速查（GB/T 1801）

### 轴承位配合（轴）

| 轴承类型 | 配合 | 公差等级 | 适用场景 |
|---------|------|---------|---------|
| 深沟球轴承 | k6 | IT6 | 一般用途 |
| 深沟球轴承 | m6 | IT6 | 承受重载 |
| 角接触轴承 | j6/k6 | IT6 | 双向轴向力 |
| 圆锥滚子轴承 | — | — | 按样本选择 |

### 轴承室配合（孔）

| 轴承类型 | 配合 | 公差等级 | 说明 |
|---------|------|---------|------|
| 内圈旋转 | H7 | IT7 | 固定内圈，外圈旋转 |
| 外圈固定 | H7 | IT7 | 固定外圈，内圈旋转 |
| 整体式轴承室 | H7 | IT7 | 通常选H7 |

### 键槽配合（GB/T 1095）

| 连接类型 | 轴槽 | 毂槽 |
|---------|------|------|
| 正常连接 | N9 | P9 |
| 紧密连接 | JS9 | P9 |
| 松连接 | H8 | D10 |

### 未注公差等级（GB/T 1804）

| 公差等级 | 线性尺寸 | 角度尺寸 | 适用场景 |
|---------|---------|---------|---------|
| f（精密级）| ±0.05mm | ±0.5° | 精密电机 |
| m（中等级）| ±0.1mm | ±1° | 一般电机 |
| c（粗级）| ±0.2mm | ±1.5° | 粗加工 |
| v（最粗）| ±0.5mm | ±3° | 无特殊要求 |

> **注**：军用电机通常选用 m（中等级）或 f（精密级）

## 粗糙度 Ra 参考值

| 表面 | Ra (μm) | 加工方法 |
|------|---------|---------|
| 轴承位 | 0.8~1.6 | 磨削 |
| 转子外圆 | 0.8~1.6 | 磨削 |
| 止口配合面 | 1.6~3.2 | 车削 |
| 端盖止口 | 1.6~3.2 | 车削 |
| 键槽侧面 | 1.6~3.2 | 插/铣 |
| 键槽底面 | 3.2~6.3 | 铣 |
| 焊缝 | 12.5~25 | 焊接态 |
| 铸件表面 | 6.3~25 | 铸造态 |

## 电机图纸常见错误速查

| 问题 | 正确做法 |
|------|---------|
| 气隙标注无公差 | 气隙应标公差（如0.5±0.05mm） |
| 键槽宽度无公差 | 标注键槽宽公差（N9/P9等） |
| 形位公差无基准 | 必须标注基准（如对基准A） |
| 粗糙度全图统一 | 不同表面应有不同 Ra 值 |
| 未注公差不说明 | 应注明"未注公差按GB/T 1804-m" |
| 倒角不标注 | 端面倒角应标注（如C1、C2） |
| 材料不标注 | 所有材料均应标注牌号 |
| 热处理不标注 | 轴、齿轮等应标注热处理要求 |
