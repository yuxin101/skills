# F01 利润表模板

适用于：月度/季度财务报表、利润分析。

---

## 模板结构

```
┌─────────────────────────────────────────────────────┐
│  BANNER: 利润表 + 期间（如：2026年第一季度）          │
├─────────────────────────────────────────────────────┤
│  核心指标卡片（4格）                                 │
│  ┌──────────┐ ┌──────────┐ ┌──────────┐ ┌──────────┐│
│  │ 营业收入  │ │ 营业利润  │ │ 净利润   │ │ 净利率   ││
│  │ 同比变化  │ │ 同比变化  │ │ 同比变化  │ │ 行业对比  ││
│  └──────────┘ └──────────┘ └──────────┘ └──────────┘│
├─────────────────────────────────────────────────────┤
│  利润表结构图（瀑布图）                              │
│  ┌─────────────────────────────────────────────┐   │
│  │  瀑布图：收入 → 成本 → 费用 → 利润           │   │
│  └─────────────────────────────────────────────┘   │
├────────────────────────────┬────────────────────────┤
│  收入构成（环形图）        │  成本费用构成（环形图）│
├────────────────────────────┼────────────────────────┤
│  利润率趋势（折线图）      │  明细表格              │
├────────────────────────────┴────────────────────────┤
│  财务健康度评估 & 改进建议                           │
└─────────────────────────────────────────────────────┘
```

---

## 标准利润表科目

```python
income_statement = {
    '营业收入': 580000,
    '减：营业成本': 320000,
    '毛利润': 260000,
    '减：销售费用': 85000,
    '减：管理费用': 62000,
    '减：财务费用': 18000,
    '营业利润': 95000,
    '加：营业外收入': 5000,
    '减：营业外支出': 2000,
    '利润总额': 98000,
    '减：所得税费用': 24500,
    '净利润': 73500,
}
```

---

## 核心指标

| 指标 | 公式 | 健康值 |
|------|------|--------|
| 毛利率 | (收入-成本)/收入 | 30%-50% |
| 净利率 | 净利润/收入 | 10%-20% |
| 费用率 | 三费/收入 | <30% |
| 营业利润率 | 营业利润/收入 | 15%-25% |

---

## 洞察生成规则

### 盈利能力
```
毛利率 {gross_margin:.1f}%{'（优秀）' if gross_margin > 40 else '（良好）' if gross_margin > 30 else '（偏低）'}，
净利率 {net_margin:.1f}%，盈利能力{'强' if net_margin > 15 else '一般' if net_margin > 10 else '较弱'}。
```

### 费用分析
```
费用率 {expense_ratio:.1f}%{'，控制良好' if expense_ratio < 25 else '，偏高需关注'}。
销售费用占比 {sales_ratio:.1f}%{'，高于行业平均（10%）' if sales_ratio > 10 else ''}。
```

### 改进建议
```python
if expense_ratio > 30:
    return "建议：优化费用结构，重点控制销售费用，目标降至25%以下"
elif gross_margin < 30:
    return "建议优化成本结构，提升毛利率至35%以上"
elif net_margin < 10:
    return "建议增加高毛利产品占比，提升整体盈利能力"
```
