# B02 月报模板

适用于：月度业务汇报、月度经营分析。

---

## 模板结构

```
┌─────────────────────────────────────────────────────┐
│  BANNER: 业务月报 + 月份（如：2026年2月）            │
├─────────────────────────────────────────────────────┤
│  KPI 卡片行（4格）                                   │
│  ┌──────────┐ ┌──────────┐ ┌──────────┐ ┌──────────┐│
│  │ 核心指标1 │ │ 核心指标2 │ │ 核心指标3 │ │ 核心指标4 ││
│  │ 环比 同比 │ │ 环比 同比 │ │ 环比 同比 │ │ 环比 同比 ││
│  └──────────┘ └──────────┘ └──────────┘ └──────────┘│
├─────────────────────────────────────────────────────┤
│  月度趋势（全宽）                                    │
│  ┌─────────────────────────────────────────────┐   │
│  │  日趋势折线图 + 月度目标线                    │   │
│  │  标注关键事件点                              │   │
│  └─────────────────────────────────────────────┘   │
│  洞察文字框                                         │
├────────────────────────────┬────────────────────────┤
│  周度对比                  │  品类/渠道分析         │
│  ┌────────────────────┐   │  ┌────────────────────┐│
│  │  各周对比条形图    │   │  │  品类排名+占比     ││
│  │  含目标达成线      │   │  │  横向条形+环形图   ││
│  └────────────────────┘   │  └────────────────────┘│
├────────────────────────────┼────────────────────────┤
│  同比对比                  │  明细表格              │
│  ┌────────────────────┐   │  ┌────────────────────┐│
│  │  本月 vs 去年同月  │   │  │  核心指标明细      ││

│  └────────────────────┘   │  └────────────────────┘│

│  月度总结 & 下月计划                                 │
└─────────────────────────────────────────────────────┘
```

---

## 数据需求

### 必需数据
| 数据 | 字段 | 说明 |
|------|------|------|
| 日趋势 | date, value | 本月每日数据 |
| 上月数据 | 同上 | 计算环比 |
| 去年同月 | 同上 | 计算同比 |
| 周度汇总 | week, value | 各周对比 |

### 派生指标
```python
# 环比
mom = (this_month - last_month) / last_month * 100

# 同比
yoy = (this_month - last_year_same_month) / last_year_same_month * 100

# 月度目标达成
achievement = this_month / target * 100
```

---

## 洞察生成规则

### 核心结论
```
本月{指标} {total:,.0f}，环比{'增长' if mom > 0 else '下降'} {abs(mom):.1f}%，
同比{'增长' if yoy > 0 else '下降'} {abs(yoy):.1f}%。
目标达成率 {achievement:.1f}%{'，超额完成' if achievement >= 100 else ''}。
```

### 周度分析
```
本月第{n}周表现最佳，达 {week_max:,.0f}；
第{m}周表现较弱，仅 {week_min:,.0f}。
周度波动系数 {cv:.1f}%{'，稳定性良好' if cv < 20 else '，波动较大'}。
```

### 预测下月
```
按当前趋势，预计下月达 {predict:,.0f}，
{'有望突破历史新高' if predict > historical_max else '需加大投入'}。
```

## 特殊元素

### 月度目标线
```python
ax.axhline(y=target, color='#ef4444', linestyle='--', linewidth=1.5, label='月度目标')

# 达成区域填充
ax.fill_between(df['date'], 0, df['value'], 
                where=df['value'] >= target,
                alpha=0.2, color='#10b981')
```

### 关键事件标注
```python
events = [
    {'date': '2026-02-14', 'label': '情人节活动'},
    {'date': '2026-02-28', 'label': '月末冲刺'},
]
for event in events:
    ax.axvline(x=pd.to_datetime(event['date']), color='gray', linestyle=':', alpha=0.7)
    ax.annotate(event['label'], xy=(event['date'], max_val*0.9), fontsize=8, rotation=90)
```
