# H03 绩效报告模板

适用于：绩效考核、绩效分析。

## 模板结构

```
┌─────────────────────────────────────────────────────┐
│  BANNER: 绩效报告 + 周期                            │
├─────────────────────────────────────────────────────┤
│  KPI：参评人数 | 平均分 | 优秀率 | 待改进率         │
├─────────────────────────────────────────────────────┤
│  绩效分布直方图                                     │
├────────────────────────────┬────────────────────────┤
│  部门绩效对比              │  等级分布              │
├────────────────────────────┼────────────────────────┤
│  绩效趋势                  │  改进建议              │
└────────────────────────────┴────────────────────────┘
```

## 绩效等级

- S：卓越（前5%）
- A：优秀（前20%）
- B：良好（前50%）
- C：待改进（后30%）
