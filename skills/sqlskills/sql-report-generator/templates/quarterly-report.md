# B03 季报模板

适用于：季度业务汇报。

## 模板结构

```
┌─────────────────────────────────────────────────────┐
│  BANNER: 季度报告 + 季度                            │
├─────────────────────────────────────────────────────┤
│  KPI：季度收入 | 利润 | 用户 | 增长率               │
├─────────────────────────────────────────────────────┤
│  季度趋势图（含季度对比）                           │
├────────────────────────────┬────────────────────────┤
│  月度对比                  │  核心业务              │
├────────────────────────────┼────────────────────────┤
│  季度总结                  │  下季规划              │
└────────────────────────────┴────────────────────────┘
```

## 核心内容

- 季度核心指标
- 季度环比/同比分析
- 季度亮点与挑战
- 下季度目标
