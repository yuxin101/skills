# P02 迭代报告模板

适用于：敏捷开发迭代汇报。

## 模板结构

```
┌─────────────────────────────────────────────────────┐
│  BANNER: Sprint 报告 + 迭代号                       │
├─────────────────────────────────────────────────────┤
│  KPI：故事点 | 完成率 | 缺陷数 | 速率               │
├─────────────────────────────────────────────────────┤
│  燃尽图                                             │
├────────────────────────────┬────────────────────────┤
│  完成情况                  │  缺陷分析              │
├────────────────────────────┼────────────────────────┤
│  团队速率趋势              │  回顾总结              │
└────────────────────────────┴────────────────────────┘
```

## Sprint 指标

- 计划故事点 vs 完成故事点
- 燃尽曲线
- 缺陷数、缺陷率
