# 酒店经营&收益管理看板

适用于：酒店集团经营分析、单体酒店运营管理、收益管理
对标：华住集团数据中台、携程酒店数据、万豪收益管理系统

## 核心指标

- **OCC（入住率）** — 已出租房数/可出租房数 — 目标：>75%（经济型>85%，高端>65%）
- **ADR（平均房价）** — 房间收入/已售房数 — 目标：环比 +2%
- **RevPAR** — 每间可售房收入 = OCC × ADR — 目标：同比 +5%
- **GOPPAR** — 每间可售房毛利 — 目标：> RevPAR × 40%
- **OTA评分** — 携程/美团等平台评分 — 目标：>4.5分
- **直接预订占比** — 非OTA渠道预订占比 — 目标：>30%（降低渠道佣金）

## 图表详细定义

### 1. 入住率趋势
- **图表类型**: line（折线图）
- **数据格式**:
```python
{"categories": ["周一","周二","周三","周四","周五","周六","周日"],
 "series": [{"name": "本周OCC%", "data": [72, 68, 65, 70, 82, 95, 88]},
             {"name": "上周OCC%", "data": [70, 65, 63, 68, 80, 93, 85]},
             {"name": "目标OCC%", "data": [75, 75, 75, 75, 75, 75, 75]}]}
```
- **简洁解读**: 追踪入住率周内波动和周趋势，识别淡旺时段
- **洞察规则**: 工作日 OCC <60% 需推出商务优惠；周末 >95% 可动态调价

### 2. RevPAR 对比（组合图）
- **图表类型**: combo（组合图：柱形=ADR，折线=RevPAR）
- **数据格式**:
```python
{"categories": ["1月","2月","3月","4月","5月","6月"],
 "columns": {"name": "ADR(元)", "data": [380, 350, 400, 420, 450, 480]},
 "lines":   {"name": "RevPAR", "data": [274, 242, 300, 324, 360, 384]}}
```
- **简洁解读**: 同时观察房价和收益效率的关系，判断提价是否带来收益增长
- **洞察规则**: ADR 上涨但 RevPAR 下降说明提价导致入住率下降过快

### 3. 客源渠道分布
- **图表类型**: donut（环形图）
- **数据格式**:
```python
{"labels": ["携程","美团","飞猪","直销APP","协议客户","散客","其他"],
 "values": [30, 20, 8, 15, 12, 10, 5]}
```
- **简洁解读**: 分析渠道依赖度，OTA占比过高需加强直销渠道建设
- **洞察规则**: OTA 总占比 >55% 说明渠道成本压力大；直销 <15% 需重点提升

### 4. 房型收益
- **图表类型**: clustered_column（簇状柱形图）
- **数据格式**:
```python
{"categories": ["大床房","双床房","套房","亲子房","商务房"],
 "series": [{"name": "RevPAR", "data": [350, 320, 800, 420, 380]},
             {"name": "出租率%", "data": [82, 78, 45, 88, 80]}]}
```
- **简洁解读**: 对比各房型的收益效率，优化房型配比和定价
- **洞察规则**: 套房 OCC <50% 需考虑降价或升级套餐；亲子房 RevPAR 最高可加大配比

### 5. 竞对酒店对比
- **图表类型**: table（表格）
- **数据格式**:
```python
{"columns": ["酒店","星级","OCC","ADR","RevPAR","评分","环比"],
 "rows": [["本酒店","四星","78%","¥420","¥328","4.6","+3%"],
          ["竞对A","四星","75%","¥380","¥285","4.5","+1%"],
          ["竞对B","五星","68%","¥680","¥462","4.7","+5%"]]}
```
- **简洁解读**: 多维度对标竞对，识别定价和运营差距
- **洞察规则**: RevPAR 低于竞对 >10% 需调整策略

### 6. 预订提前期
- **图表类型**: bar（条形图）
- **数据格式**:
```python
{"categories": ["当天","1天前","2-3天","4-7天","1-2周","2-4周","1月+"],
 "series": [{"name": "预订占比%", "data": [15, 20, 25, 20, 12, 5, 3]}]}
```
- **简洁解读**: 了解客户预订习惯，优化不同提前期的定价策略
- **洞察规则**: 当天+1天前占比 >40% 说明临时预订多，可设阶梯早鸟价

## 深度洞察

1. **周末满房但周中空置**：周末 OCC 95% 但工作日仅 65-70%，收益波动大
2. **OTA 依赖度高**：携程+美团占比 50%，渠道佣金约 15%，影响毛利
3. **亲子房表现突出**：RevPAR 420 元最高且 OCC 88%，可加大亲子房型比例
4. **竞品A价格侵蚀**：ADR 低 40 元但评分接近，可能分流价格敏感客户

## 运营建议

1. 工作日推出商务+餐饮套餐，提升周中入住率至 75% — 高优先级
2. 加强直销 APP 推广，目标将直销占比提升至 20% — 高优先级
3. 加大亲子房配比至 20%，搭配亲子主题活动 — 中优先级
4. 建立阶梯定价：提前 7 天预订享 9 折 — 中优先级
5. 推出长住优惠（3天以上折扣），提高停留天数 — 中优先级
6. 针对竞品A推出差异化服务（如免费升级、延迟退房） — 低优先级
