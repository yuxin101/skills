## 核心指标

- **OEE（设备综合效率）** — 时间稼动率×性能稼动率×良品率，衡量设备整体效率，世界级水平为85%
- **故障停机时长（Downtime）** — 设备故障导致的停机时间，单位为小时/天或分钟/班次
- **MTBF（平均故障间隔时间）** — 设备两次故障之间的平均运行时间，单位为小时
- **MTTR（平均修复时间）** — 设备故障后从报修到恢复的平均时间，单位为分钟
- **能耗（Energy Consumption）** — 设备单位时间的能源消耗，通常以kWh/件或kWh/小时计
- **设备稼动率（Equipment Availability）** — 实际运行时间/计划生产时间×100%
- **性能稼动率（Performance Efficiency）** — 实际产出/理论产出×100%
- **良品率（Quality Rate）** — 合格品数量/总产出数量×100%

## 图表详细定义

### 1. OEE趋势（组合图）

- **图表类型**: combo（簇状柱形图+折线图）
- **数据格式**:
```python
{
    "xAxis": ["1月", "2月", "3月", "4月", "5月", "6月", "7月", "8月", "9月", "10月", "11月", "12月"],
    "bars": [
        {"name": "时间稼动率(%)", "data": [88, 89, 90, 91, 92, 93, 94, 95, 94, 93, 94, 95]},
        {"name": "性能稼动率(%)", "data": [82, 83, 84, 85, 86, 87, 88, 89, 88, 87, 88, 89]},
        {"name": "良品率(%)", "data": [95, 96, 96, 97, 97, 98, 98, 98, 97, 97, 98, 98]}
    ],
    "lines": [
        {"name": "OEE(%)=时间×性能×良品", "data": [68.5, 70.9, 73.0, 75.1, 77.1, 79.5, 81.3, 82.8, 80.7, 79.0, 80.8, 82.8]},
        {"name": "目标OEE(85%)", "data": [85, 85, 85, 85, 85, 85, 85, 85, 85, 85, 85, 85]}
    ]
}
```
- **简洁解读**: 展示OEE三大构成要素的变化趋势，定位制约OEE提升的关键因子
- **洞察规则**: OEE持续上升但低于目标→需持续优化；时间稼动率低→关注设备故障和换线；性能稼动率低→关注速度损失；良品率低→关注质量

---

### 2. 故障分布（饼图）

- **图表类型**: donut（环形饼图）
- **数据格式**:
```python
{
    "data": [
        {"name": "机械故障", "value": 45, "color": "#e74c3c"},
        {"name": "电气故障", "value": 28, "color": "#e67e22"},
        {"name": "传感器故障", "value": 15, "color": "#f39c12"},
        {"name": "软件/程序故障", "value": 8, "color": "#3498db"},
        {"name": "人为操作失误", "value": 4, "color": "#9b59b6"}
    ],
    "centerText": "总故障: 100次"
}
```
- **简洁解读**: 设备故障类型占比分布，识别主要故障来源，指导预防性维护策略
- **洞察规则**: 机械故障占比>40%→需加强机械部件的润滑、紧固和更换；电气故障占比高→需检查线路老化和接触不良

---

### 3. 设备稼动率热力图

- **图表类型**: heatmap（热力图）
- **数据格式**:
```python
{
    "xAxis": ["早班(8-16)", "中班(16-24)", "夜班(0-8)", "周末"],
    "yAxis": ["A生产线", "B生产线", "C生产线", "D生产线", "E生产线"],
    "data": [
        [92, 88, 75, 65],
        [90, 85, 72, 60],
        [88, 82, 70, 58],
        [85, 78, 68, 55],
        [82, 75, 65, 50]
    ],
    "colorScale": "Red-Yellow-Green"
}
```
- **简洁解读**: 以稼动率着色，红色区域为设备利用率低的风险点，一目了然定位低效时段和产线
- **洞察规则**: 夜班稼动率普遍低于早班>15%→需分析夜班人员配置和交接班问题；某产线全时段红色→该产线设备老化或工艺问题

---

        ["WO-20231201-003", "EQ-C002", "喷涂线#2", "传感器故障", "2023-12-01 14:45", "2023-12-01 15:10", "25分钟", "王工", "温度传感器失灵"],

## 深度洞察

1. **OEE提升路径分析**：通过OEE三大要素分解，定位制约因素。如时间稼动率仅85%而性能稼动率92%，则应优先解决设备故障和换线问题，而非速度优化。

2. **故障预测性维护**：通过维修工单统计表分析故障类型和频次，对高频故障部件建立备件预警机制，在故障发生前主动更换，降低非计划停机。

3. **能耗异常诊断**：能耗趋势图中某月能耗突然上升，结合维修工单发现该月某设备频繁故障导致空转，可定位能耗异常根因。

4. **班次效率差异分析**：通过设备稼动率热力图发现夜班效率普遍低于白班，可分析夜班人员配置、交接班流程、照明条件等影响因素。

5. **设备生命周期管理**：结合设备健康度和设备投用年限，建立设备更新决策模型，避免设备过度老化导致的效率损失。

## 运营建议

1. **P0 - 建立OEE日监控机制**：每日晨会通报前日OEE数据，对OEE低于目标值80%的产线，当日完成根因分析和改进计划。优先级来源：OEE趋势图低于目标线即触发。

2. **P0 - 实施预防性维护计划**：针对机械故障占比高的特点，制定关键部件的定期润滑、紧固、更换计划，目标MTBF提升20%，故障停机时长降低30%。

3. **P1 - 优化夜班生产效率**：针对夜班稼动率偏低问题，优化交接班流程，加强夜班人员配置，开展夜班专项技能培训，目标夜班稼动率提升至白班的90%。

4. **P1 - 建立设备健康度预警**：对健康度处于黄色预警区的设备，每周输出健康度报告，制定维护计划，目标健康度提升至80%以上。

5. **P2 - 推进能耗精细化管理**：建立单台设备能耗台账，识别高能耗设备，开展节能改造，目标单位产品能耗降低10%。

6. **P2 - 缩短MTTR维修响应时间**：建立维修人员快速响应机制，关键设备配备常用备件，目标MTTR从当前60分钟降低至40分钟。

7. **P3 - 推进设备数字化改造**：对老旧设备加装传感器，实现运行状态实时监控，建立设备数字孪生模型，支撑预测性维护决策。
