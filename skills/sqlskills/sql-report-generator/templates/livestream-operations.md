# 直播运营分析看板

适用于：电商直播运营、主播绩效管理、直播团队管理
对标：抖音电商罗盘、快手直播数据中心、淘宝直播数据

## 模板布局

```
┌─────────────────────────────────────────────────────────────┐
│  BANNER: 直播运营分析看板 + 日期 + 主播筛选 + 平台筛选       │
├─────────────────────────────────────────────────────────────┤
│  KPI: GMV | 观看人数 | 转化率 | UV价值 | 客单价 | 新增粉丝  │
├─────────────────────────────────────────────────────────────┤
│  GMV趋势（面积图）             │  直播间GMV排行（条形图）   │
├────────────────────────────────────┼────────────────────────┤
│  实时在线人数（折线图）         │  商品转化漏斗（漏斗图）   │
├────────────────────────────────────┼────────────────────────┤
│  主播效能气泡图                │  时段GMV热力图            │
├────────────────────────────────────┴────────────────────────┤
│  深度洞察 + 运营建议                                      │
└─────────────────────────────────────────────────────────────┘
```

## 核心指标

- **GMV** — 直播成交金额 — 目标：月环比 +15%
- **场均观看人数** — 每场直播平均在线人数 — 目标：> 5000
- **转化率** — 访客→下单比例 — 目标：> 5%
- **UV价值** — 每UV带来的GMV — 目标：> 3元/人
- **客单价** — 每单平均金额 — 目标：> 80元
- **新增粉丝** — 直播带来新增关注 — 目标：场均 +500
- **平均停留时长** — 观众平均停留分钟数 — 目标：> 3分钟

## 图表详细定义

### 1. GMV 趋势
- **图表类型**: area（面积图）
- **数据格式**:
```python
{"categories": ["第1周","第2周","第3周","第4周"],
 "series": [{"name": "GMV(万元)", "data": [120, 135, 155, 180]},
             {"name": "目标", "data": [130, 140, 150, 170]}]}
```
- **简洁解读**: 追踪 GMV 周趋势，识别增长拐点和节庆效应
- **洞察规则**: 节庆前一周 GMV 应环比 +30%；连续 2 周下降需排查内容问题

### 2. 直播间 GMV 排行
- **图表类型**: bar（条形图）
- **数据格式**:
```python
{"categories": ["主播A","主播B","主播C","主播D","主播E","主播F"],
 "series": [{"name": "GMV(万元)", "data": [850, 620, 480, 350, 220, 150]},
             {"name": "观看人数", "data": [25000, 18000, 15000, 10000, 8000, 5000]}]}
```
- **简洁解读**: 综合排行所有直播间，识别头部和尾部主播
- **洞察规则**: TOP3 贡献 60%+ GMV 属正常；尾部主播转化率 <3% 需优化话术

### 3. 实时在线人数
- **图表类型**: line（折线图）
- **数据格式**:
```python
{"categories": ["0h","1h","2h","3h","4h","5h","6h"],
 "series": [{"name": "在线人数", "data": [0, 3000, 8000, 12000, 15000, 8000, 2000]},
             {"name": "上架商品数", "data": [0, 5, 15, 25, 35, 40, 42]}]}
```
- **简洁解读**: 追踪直播节奏和在线峰值，预判最佳推品时机
- **洞察规则**: 开播 30 分钟内人数未达 5000 需调整预热策略；峰值后下滑 >40% 需上引流款

### 4. 商品转化漏斗
- **图表类型**: funnel（漏斗图）
- **数据格式**:
```python
{"stages": ["曝光", "点击", "加购", "下单", "支付"],
 "values": [500000, 80000, 30000, 15000, 12000]}
```
- **简洁解读**: 从曝光到支付追踪全链路转化，定位流失最大环节
- **洞察规则**: 点击率 <15% 需优化封面；加购率 <40% 需优化话术；支付率 <75% 需简化购买流程

### 5. 主播效能气泡图
- **图表类型**: scatter（气泡图）
- **数据格式**:
```python
{"x": [8000, 15000, 12000, 6000, 5000],
 "y": [5.2, 6.8, 4.5, 3.2, 2.8],
 "size": [85, 62, 48, 35, 22],
 "labels": ["主播A","主播B","主播C","主播D","主播E"]}
```
- **简洁解读**: X轴=场均观看人数，Y轴=转化率，气泡大小=GMV，快速识别高效主播
- **洞察规则**: 右上角（高观看+高转化）为主播标杆；左下角需重点培训

### 6. 时段 GMV 热力图
- **图表类型**: heatmap（热力图）
- **数据格式**:
```python
{"x_labels": ["周一","周二","周三","周四","周五","周六","周日"],
 "y_labels": ["9-12时","12-15时","15-18时","18-21时","21-24时"],
 "z_values": [[5,8,10,15,12],
               [6,9,12,18,14],
               [8,10,15,20,16],
               [10,12,18,25,20],
               [15,18,22,30,25]]}
```
- **简洁解读**: 识别最佳直播时段，节省试错成本
- **洞察规则**: 18-21时为黄金时段贡献 40%+ GMV；周末全天均可播

## 深度洞察

1. **UV价值稳步提升**：从 2.5 提升至 3.8，说明流量质量和转化率同步改善
2. **主播B效能最优**：高观看+高转化，气泡位于右上象限，是团队标杆
3. **22时后转化率骤降**：深夜时段观看人数尚可但转化率 <2%，应缩短直播时长
4. **加购→下单流失严重**：漏斗显示加购到下单仅 50%，存在价格顾虑或支付障碍

## 运营建议

1. 重点培养主播B带队主播C，复制高效直播话术 — 高优先级
2. 优化加购→下单链路：提供限时优惠券和免息分期 — 高优先级
3. 聚焦 18-22 时黄金时段，减少低效凌晨场次 — 高优先级
4. 针对新粉丝推出专属福利，提升新粉下单率至 20% — 中优先级
5. 每场直播复盘：记录峰值时段推品，复用成功节奏 — 中优先级
6. 建立主播分级体系，分层匹配品牌/引流/长尾商品 — 低优先级
