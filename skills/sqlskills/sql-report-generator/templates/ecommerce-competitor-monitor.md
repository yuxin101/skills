# 电商行业 - 竞品监控看板模板

适用于：电商市场竞争分析、竞品策略追踪、市场份额监控
对标：阿里数据银行竞品分析、京东商智竞品监控、SimilarWeb

## 模板布局

```
┌─────────────────────────────────────────────────────────────────┐
│  BANNER: 竞品监控中心 + 行业选择 + 时间范围                    │
├─────────────────────────────────────────────────────────────────┤
│  KPI 卡片行：自身GMV | 竞品GMV | 市场份额 | 价格指数           │
├────────────────────────────┬────────────────────────────────────┤
│  市场份额趋势(面积图)      │  价格指数对比(组合图)            │
├────────────────────────────┼────────────────────────────────────┤
│  搜索热度对比(折线图)      │  竞品对比矩阵(表格)              │
├────────────────────────────┼────────────────────────────────────┤
│  品类竞争格局(堆叠柱形图)  │  用户好评率对比(条形图)          │
├────────────────────────────┴────────────────────────────────────┤
│  深度洞察 + 运营建议                                           │
└─────────────────────────────────────────────────────────────────┘
```

## 核心指标

- **自身GMV** — 平台商品交易总额 — 目标值：环比增长≥行业平均
- **竞品GMV** — 主要竞争对手的GMV(第三方数据估算) — 目标值：缩小差距
- **市场份额** — 自身在行业GMV中的占比 — 目标值：持续提升
- **价格指数** — 商品均价相对行业均值的比率 — 目标值：根据定位设定
- **搜索热度指数** — 品牌相关搜索量相对竞品的比率 — 目标值：≥竞品均值
- **用户好评率** — 商品评价中好评占比 — 目标值：≥95%
- **流量份额** — 平台流量占行业流量比例 — 目标值：与市场份额匹配
- **促销力度指数** — 促销频率和折扣深度综合指标 — 目标值：平衡利润和竞争力

## 图表详细定义

### 1. 市场份额趋势
- **图表类型**: stacked_area
- **数据格式**:
```python
{
    "months": ["2024-01", "2024-02", "2024-03", "2024-04", "2024-05", "2024-06"],
    "players": {
        "自身平台": [15.2, 15.8, 16.2, 16.5, 17.0, 17.3],
        "竞品A": [32.5, 32.2, 31.8, 31.5, 31.2, 30.8],
        "竞品B": [18.8, 18.5, 18.2, 18.0, 17.8, 17.5],
        "竞品C": [12.5, 12.8, 13.2, 13.5, 13.8, 14.0],
        "其他": [21.0, 20.7, 20.6, 20.5, 20.2, 20.4]
    }
}
```
- **简洁解读**: 展示各平台市场份额变化趋势,识别竞争格局
- **洞察规则**:
  - 自身份额持续上升为良性竞争
  - 竞品份额下降说明策略有效
  - 新进入者份额增长需关注

### 2. 价格指数对比
- **图表类型**: combo (柱形+折线)
- **数据格式**:
```python
{
    "categories": ["3C数码", "服饰鞋包", "美妆个护", "食品生鲜", "家居日用", "母婴用品"],
    "self_price_index": [98, 105, 92, 108, 102, 95],  # 相对行业均值(100为基准)
    "competitor_a_index": [102, 98, 105, 95, 97, 108],
    "competitor_b_index": [95, 102, 98, 102, 105, 92],
    "self_gmv_growth": [12.5, 8.2, 15.8, 6.5, 9.8, 18.2]  # 自身GMV增速%
}
```
- **简洁解读**: 对比各品类价格指数,分析定价策略与GMV增长关系
- **洞察规则**:
  - 价格指数<100说明有价格优势
  - 价格指数低但增长慢需优化营销
  - 价格指数高且增长快说明品牌溢价

## 深度洞察

1. **市场份额追赶态势**: 自身市场份额从15.2%提升至17.3%,增速高于竞品A(从32.5%降至30.8%),说明竞争力提升但差距仍大,需持续追赶
2. **定价策略平衡**: 3C数码和母婴品类价格指数低于行业(98、95),GMV增速领先(12.5%、18.2%),说明价格策略有效;美妆个护价格低但增长快,说明性价比路线成功
3. **品牌热度差距**: 搜索热度指数约为竞品A的60%,说明品牌知名度和用户心智仍有差距,需加大品牌营销和内容投放
4. **服务优势明显**: 物流时效、好评率、复购率均优于竞品,说明用户体验和口碑是核心竞争力,可加强"服务好"的品牌宣传
5. **品类机会识别**: 家居日用品类市场份额仅12.8%,低于其他品类,但市场空间大,可作为重点突破方向

## 运营建议

1. **差异化定位策略** — 优先级：高
   避免与竞品正面价格战,突出"物流快、服务好、口碑优"差异化优势,建立品质电商品牌形象

2. **重点品类突破** — 优先级：高
   选择3C数码、母婴用品等价格优势+增长快的品类,加大营销投入,提升品类市场份额

3. **品牌营销加强** — 优先级：高
   增加品牌广告投放,提升品牌搜索热度和知名度,缩小与头部竞品的品牌差距

4. **定价策略优化** — 优先级：中
   对竞争激烈品类保持价格优势,对有品牌溢价品类适当提价,平衡市场份额和利润

5. **用户体验持续优化** — 优先级：中
   巩固物流时效、客服服务等优势维度,改进"描述相符"和"性价比"相对较弱的维度

6. **竞品动态监控** — 优先级：中
   建立竞品监控体系,追踪竞品促销活动、新品上架、价格变动等,及时调整应对策略

7. **行业趋势研究** — 优先级：低
   定期研究行业发展趋势和新兴品类,提前布局增长赛道,避免错失机会
