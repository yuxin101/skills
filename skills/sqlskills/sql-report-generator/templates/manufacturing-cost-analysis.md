## 核心指标

- **单位成本（Unit Cost）** — 单件产品的总成本，包括直接材料、直接人工和制造费用
- **材料成本占比（Material Cost Ratio）** — 直接材料成本占总成本的比例，通常制造业为50-70%
- **人工成本（Labor Cost）** — 直接人工成本，包括计件工资、加班费等
- **制造费用（Manufacturing Overhead）** — 间接生产成本，包括折旧、能源、维护、间接人工等
- **成本偏差率（Cost Variance Rate）** — 实际成本与标准成本的差异比例，正值为超支，负值为节约
- **毛利率（Gross Margin）** — (销售收入-销售成本)/销售收入×100%
- **成本降低率（Cost Reduction Rate）** — 本期单位成本较上期的降低幅度
- **库存周转率（Inventory Turnover）** — 销售成本/平均库存，衡量原材料和成品库存管理效率

## 图表详细定义

### 1. 成本构成（饼图）

- **图表类型**: donut（环形饼图）
- **数据格式**:
```python
{
    "data": [
        {"name": "直接材料", "value": 58, "color": "#3498db"},
        {"name": "直接人工", "value": 18, "color": "#e67e22"},
        {"name": "制造费用", "value": 15, "color": "#9b59b6"},
        {"name": "能源动力", "value": 6, "color": "#f39c12"},
        {"name": "质量损失", "value": 3, "color": "#e74c3c"}
    ],
    "centerText": "单位成本: ¥285"
}
```
- **简洁解读**: 展示单位成本的构成比例，识别主要成本来源，为降本方向提供依据
- **洞察规则**: 材料成本占比>65%→降本重点在采购和工艺优化；人工成本占比>25%→需关注自动化改造；制造费用占比异常高→需检查分摊方法

---

### 2. 成本趋势（组合图）

- **图表类型**: combo（簇状柱形图+折线图）
- **数据格式**:
```python
{
    "xAxis": ["1月", "2月", "3月", "4月", "5月", "6月", "7月", "8月", "9月", "10月", "11月", "12月"],
    "bars": [
        {"name": "直接材料(元/件)", "data": [165, 162, 160, 158, 155, 152, 150, 148, 145, 142, 140, 138]},
        {"name": "直接人工(元/件)", "data": [52, 51, 50, 49, 48, 47, 46, 45, 44, 43, 42, 41]},
        {"name": "制造费用(元/件)", "data": [43, 42, 41, 40, 39, 38, 37, 36, 35, 34, 33, 32]}
    ],
    "lines": [
        {"name": "单位总成本(元/件)", "data": [260, 255, 251, 247, 242, 237, 233, 229, 224, 219, 215, 211]},
        {"name": "目标成本(元/件)", "data": [240, 238, 235, 233, 230, 228, 225, 223, 220, 218, 215, 212]}
    ]
}
```
- **简洁解读**: 追踪各项成本要素的变化趋势，识别成本波动原因，评估降本成效
- **洞察规则**: 成本持续下降且接近目标→降本措施有效；某月成本突然反弹→需追溯原材料涨价或效率下降；成本长期高于目标→需重新评估标准成本

---

### 3. BOM成本瀑布图

- **图表类型**: waterfall（瀑布图）
- **数据格式**:
```python
{
    "categories": ["原材料A", "原材料B", "原材料C", "外购件", "辅料", "直接人工", "制造费用", "能源", "质量损失", "总成本"],
    "values": [85, 62, 35, 28, 15, 52, 43, 17, 8, 345],
    "cumulative": [85, 147, 182, 210, 225, 277, 320, 337, 345, 345],
    "total": 345
}
```
- **简洁解读**: 从原材料到最终产品的成本层层叠加，清晰展示BOM各层级成本贡献
- **洞察规则**: 某原材料成本占比>25%→重点谈判该物料价格；外购件成本异常高→评估自制可行性；质量损失成本>5%→需质量改进

---

### 4. 部门成本对比（条形图）

- **图表类型**: horizontal_bar（水平条形图）
- **数据格式**:
```python
{
    "categories": ["机加工车间", "焊接车间", "涂装车间", "组装车间", "质检中心", "仓储物流", "设备维护"],
    "series": [
        {"name": "实际成本(万元)", "data": [285, 220, 195, 168, 85, 72, 58]},
        {"name": "预算成本(万元)", "data": [270, 210, 180, 160, 80, 70, 55]},
        {"name": "成本偏差(%)", "data": [5.6, 4.8, 8.3, 5.0, 6.3, 2.9, 5.5]}
    ]
}
```
- **简洁解读**: 对比各部门实际成本与预算，识别超支部门，为成本控制提供依据
- **洞察规则**: 偏差率>10%→需专项审计；某部门连续超支→需重新评估预算合理性；偏差率<0(节约)→需分析是否影响质量

---

### 5. 成本偏差率（仪表盘组）

- **图表类型**: gauge（仪表盘，多图并列）
- **数据格式**:
```python
{
    "gauges": [
        {"name": "直接材料偏差", "value": 3.2, "min": -10, "max": 10, "unit": "%", "thresholds": {"green": -5, "yellow": 0, "red": 5}},
        {"name": "直接人工偏差", "value": -2.5, "min": -10, "max": 10, "unit": "%", "thresholds": {"green": -5, "yellow": 0, "red": 5}},
        {"name": "制造费用偏差", "value": 1.8, "min": -10, "max": 10, "unit": "%", "thresholds": {"green": -5, "yellow": 0, "red": 5}},
        {"name": "能源成本偏差", "value": 6.5, "min": -10, "max": 10, "unit": "%", "thresholds": {"green": -5, "yellow": 0, "red": 5}},
        {"name": "总成本偏差", "value": 2.1, "min": -10, "max": 10, "unit": "%", "thresholds": {"green": -5, "yellow": 0, "red": 5}}
    ]
}
```
- **简洁解读**: 直观展示各项成本与标准成本的偏差，绿色为节约，黄色为接近标准，红色为超支
- **洞察规则**: 偏差率>5%(红色)→需立即分析原因并制定纠偏措施；偏差率<-5%(绿色)→需确认是否影响产品质量

---

### 6. 成本明细表（表格）

- **图表类型**: table（数据表格）
- **数据格式**:
```python
{
    "columns": ["产品型号", "产量(件)", "直接材料", "直接人工", "制造费用", "单位成本", "标准成本", "偏差率", "毛利率"],
    "rows": [
        ["A-100", 5000, "¥165", "¥52", "¥43", "¥260", "¥255", "+2.0%", "28%"],
        ["A-200", 4200, "¥142", "¥48", "¥38", "¥228", "¥225", "+1.3%", "32%"],
        ["B-100", 6800, "¥128", "¥42", "¥35", "¥205", "¥210", "-2.4%", "35%"],
        ["B-200", 3500, "¥185", "¥58", "¥48", "¥291", "¥285", "+2.1%", "25%"],
        ["C-100", 2800, "¥95", "¥35", "¥28", "¥158", "¥160", "-1.3%", "42%"]

```
- **简洁解读**: 详细记录各产品型号的成本构成和偏差情况，支持产品盈利能力分析
- **洞察规则**: 某产品毛利率<20%→需评估是否继续生产或提价；偏差率>5%→需追溯该产品的成本异常

## 深度洞察

1. **成本结构优化机会**：通过成本构成饼图发现材料成本占比高达58%，可重点推进VAVE（价值分析与价值工程）和供应商谈判，目标材料成本占比降低至55%。

2. **成本偏差根因分析**：通过成本偏差率仪表盘发现能源成本偏差达6.5%，结合能耗趋势图定位某高能耗设备异常运行，可制定针对性节能措施。

3. **产品盈利能力分层**：通过成本明细表识别C-100产品毛利率高达42%，可加大该产品产能和市场推广；B-200毛利率仅25%且成本偏差高，需评估是否优化或淘汰。

4. **BOM成本优化路径**：通过BOM瀑布图识别原材料A成本占比最高（85元/件），可推进该物料的国产化替代或批量采购谈判。

5. **部门成本责任追溯**：通过部门成本对比发现涂装车间成本偏差最高（8.3%），可深入分析涂装工艺参数、涂料利用率、返工率等因素。

---

## 运营建议

1. **P0 - 建立成本日监控机制**：每日通报前日各产品单位成本和偏差率，对偏差率>5%的产品当日完成根因分析。优先级来源：成本偏差率仪表盘红色预警即触发。

2. **P0 - 推进VAVE降本项目**：针对材料成本占比高的产品，成立VAVE专项小组，目标年度材料成本降低8-10%，每季度输出VAVE成果报告。

3. **P1 - 优化标准成本体系**：对连续3个月偏差率>3%的成本项目，重新核定标准成本，确保标准成本的合理性和可达成性。

4. **P1 - 加强供应商成本管理**：针对BOM瀑布图中成本占比高的原材料，开展供应商成本结构分析，推进年度降价谈判和备选供应商开发。

5. **P2 - 推进精益生产降本**：通过精益生产工具（5S、TPM、SMED）降低制造费用和人工成本，目标制造费用占比降低2个百分点。

6. **P2 - 建立产品盈利分析模型**：定期输出产品盈利能力报告，对毛利率<20%的产品制定提价或成本优化方案，对高毛利产品加大产能配置。

7. **P3 - 推进成本数字化管理**：建立成本数据采集自动化系统，实现成本数据实时采集、自动计算、实时预警，缩短成本分析周期。
