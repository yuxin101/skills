# O02 用户增长报告模板

适用于：用户增长分析、运营效果评估。

## 模板结构

```
┌─────────────────────────────────────────────────────┐
│  BANNER: 用户增长报告 + 周期                        │
├─────────────────────────────────────────────────────┤
│  KPI：新增用户 | 活跃用户 | 留存率 | 增长率         │
├─────────────────────────────────────────────────────┤
│  用户增长趋势图                                     │
├────────────────────────────┬────────────────────────┤
│  新增来源分析              │  活跃度分布            │
├────────────────────────────┼────────────────────────┤
│  留存曲线                  │  用户画像              │
└────────────────────────────┴────────────────────────┘
```

## 核心指标

- DAU/MAU、新增用户、活跃率
- 次日留存、7日留存、30日留存
- 各渠道新增占比
- 用户画像分布
