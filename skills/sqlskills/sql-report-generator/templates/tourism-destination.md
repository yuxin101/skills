# 旅游目的地运营看板

适用于：景区运营管理、文旅局大数据监控、旅行社目的地分析
对标：飞猪旅游数据平台、文旅部大数据平台、马蜂窝数据中心

## 模板布局

```
┌─────────────────────────────────────────────────────────────┐
│  BANNER: 旅游目的地运营看板 + 日期选择 + 景区筛选            │
├─────────────────────────────────────────────────────────────┤
│  KPI: 日游客量 | 累计游客量 | 旅游收入 | 人均消费 | 满意度   │
├─────────────────────────────────────────────────────────────┤
│  游客量趋势（面积图）        │  来源地TOP10（条形图）        │
├────────────────────────────────────┼────────────────────────┤
│  消费结构（饼图）              │  热门景区排行（表格）       │
├────────────────────────────────────┼────────────────────────┤
│  时段分布热力图                │  天气vs客流（散点图）      │
├────────────────────────────────────┴────────────────────────┤
│  深度洞察 + 运营建议                                      │
└─────────────────────────────────────────────────────────────┘
```

## 核心指标

- **日游客量** — 当日入园/到访游客总数 — 目标：同比 +10%
- **累计游客量** — 年度/月度累计 — 目标：完成全年预算 80%
- **旅游收入** — 门票+餐饮+住宿+购物综合收入 — 目标：人均消费 > 500 元
- **人均消费** — 每位游客平均消费金额 — 目标：环比增长 5%
- **游客满意度** — NPS 或评分（1-5分） — 目标：> 4.5 分
- **平均停留时长** — 游客在目的地平均停留天数 — 目标：> 2.5 天
- **回头客比例** — 二次到访游客占比 — 目标：> 30%

## 图表详细定义

### 1. 游客量趋势图
- **图表类型**: area（面积图）
- **数据格式**:
```python
{"categories": ["周一","周二","周三","周四","周五","周六","周日"],
 "series": [{"name": "本周游客量", "data": [8500, 9200, 9800, 10200, 15000, 28000, 25000]},
             {"name": "上周游客量", "data": [8000, 8800, 9500, 10000, 14500, 27000, 24000]}]}
```
- **简洁解读**: 展示游客量的周内波动规律和周末高峰效应，对比同期识别增长趋势
- **洞察规则**: 周末游客量为工作日 2-3 倍属正常；连续下降 >15% 需排查原因

### 2. 来源地 TOP10
- **图表类型**: clustered_bar（簇状条形图）
- **数据格式**:
```python
{"categories": ["本省","广东","浙江","江苏","北京","上海","山东","四川","湖北","福建"],
 "series": [{"name": "游客量(万)", "data": [120, 45, 38, 32, 28, 25, 22, 18, 15, 12]}]}
```
- **简洁解读**: 识别主要客源地，指导精准营销投放和交通接驳优化
- **洞察规则**: TOP3 来源地占比 >50% 说明客源集中度高，需拓展新市场

### 3. 消费结构
- **图表类型**: donut（环形图）
- **数据格式**:
```python
{"labels": ["门票","餐饮","住宿","购物","交通","娱乐","其他"],
 "values": [25, 20, 22, 15, 8, 6, 4]}
```
- **简洁解读**: 揭示游客消费构成，识别高价值消费品类
- **洞察规则**: 门票占比 >30% 说明二次消费不足；购物+娱乐占比 >25% 为健康结构

### 4. 热门景区排行
- **图表类型**: table（表格）
- **数据格式**:
```python
{"columns": ["景区名称","日游客量","收入(万)","满意度","同比增长"],
 "rows": [["AAAA景区A", 15000, 750, 4.6, "+15%"],
          ["AAAA景区B", 12000, 600, 4.5, "+12%"],
          ["AAA景区C", 8000, 320, 4.3, "+8%"]]}
```
- **简洁解读**: 综合对比各景区运营表现，识别明星景区和待提升景区
- **洞察规则**: 满意度 <4.0 或同比下降 >10% 的景区需重点关注

### 5. 时段分布热力图
- **图表类型**: heatmap（热力图）
- **数据格式**:
```python
{"x_labels": ["周一","周二","周三","周四","周五","周六","周日"],
 "y_labels": ["8-10时","10-12时","12-14时","14-16时","16-18时","18-20时"],
 "z_values": [[500,550,600,620,900,2500,2200],
               [2000,2200,2400,2500,3500,5000,4500],
               [1500,1600,1700,1750,2800,4200,3800],
               [1800,1900,2000,2100,3200,4800,4400],
               [1200,1300,1400,1450,2200,3500,3000],
               [300,350,380,400,800,1500,1200]]}
```
- **简洁解读**: 精确掌握客流高峰时段，优化工作人员排班和设施调度
- **洞察规则**: 10-16时为全天高峰，占日客流 60%+；如出现异常需排查活动或天气

### 6. 天气与客流关系
- **图表类型**: scatter（散点图）
- **数据格式**:
```python
{"x": [15, 20, 25, 30, 35, 22, 18, 28, 12, 8],
 "y": [12000, 15000, 16000, 18000, 14000, 15500, 13000, 17000, 9000, 6000],
 "labels": ["晴天","晴天","晴天","晴天","高温","多云","阴天","晴天","小雨","大雨"]}
```
- **简洁解读**: 分析天气对客流的影响程度，为天气预警调度提供数据支撑
- **洞察规则**: 高温(>35°C)和大雨天气客流下降 >30%，应启动室内备选方案

## 深度洞察

1. **周末效应显著**：周末游客量是工作日的 2.5-3 倍，但人均消费仅为工作日的 85%，说明周末以一日游为主
2. **客源集中度偏高**：本省游客占 40%，TOP3 来源地占 65%，远途客源拓展空间大
3. **二次消费不足**：门票收入占 25%，购物+娱乐仅占 21%，低于行业标准（30%+），需丰富消费场景
4. **天气敏感度高**：雨天客流下降 50%+，但室内景区影响较小，说明缺乏全天候产品

## 运营建议

1. 丰富周末过夜产品（如夜游、演出），提升周末人均消费 — 高优先级
2. 针对广东、浙江 TOP2 省外客源地投放精准广告 — 高优先级
3. 增加室内体验项目，降低天气对客流的影响 — 中优先级
4. 推出景区联票+餐饮住宿套餐，提升二次消费占比 — 中优先级
5. 10-12时高峰增设临时检票通道，缩短排队时间 — 高优先级
6. 建立天气预警机制，雨天推送室内备选路线给游客 — 低优先级
