# B08 趋势分析报告模板

适用于：时间序列深度分析。

## 模板结构

```
┌─────────────────────────────────────────────────────┐
│  BANNER: 趋势分析报告 + 周期                        │
├─────────────────────────────────────────────────────┤
│  KPI：均值 | 峰值 | 谷值 | 增长率                   │
├─────────────────────────────────────────────────────┤
│  长期趋势图（含趋势线）                             │
├────────────────────────────┬────────────────────────┤
│  季节性分析                │  周期性分析            │
├────────────────────────────┼────────────────────────┤
│  异常点检测                │  预测                  │
└────────────────────────────┴────────────────────────┘
```

## 分析维度

- 长期趋势
- 季节性波动
- 周期性规律
- 异常值检测
