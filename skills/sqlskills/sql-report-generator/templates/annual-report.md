# B04 年报模板

适用于：年度经营总结、年度汇报。

## 模板结构

```
┌─────────────────────────────────────────────────────┐
│  BANNER: 年度经营报告 + 年份                         │
├─────────────────────────────────────────────────────┤
│  年度核心指标卡片（6格）                             │
├─────────────────────────────────────────────────────┤
│  年度趋势图（12个月）                               │
├─────────────────────────────────────────────────────┤
│  季度对比 | 核心业务构成                            │
├─────────────────────────────────────────────────────┤
│  年度亮点 & 挑战                                    │
├─────────────────────────────────────────────────────┤
│  明年规划 & 目标                                    │
└─────────────────────────────────────────────────────┘
```

## 核心指标

- 年度总收入、同比增长率
- 年度净利润、净利率
- 年度用户增长、活跃用户数
- 市场份额变化

## 洞察生成

```
全年实现收入 {total_revenue:,.0f}，同比增长 {yoy:.1f}%。
Q4 表现最佳，贡献全年 {q4_pct:.1f}%。
年度目标达成率 {achievement:.1f}%。
```
