# O05 渠道分析报告模板

适用于：渠道效果对比、渠道优化。

## 模板结构

```
┌─────────────────────────────────────────────────────┐
│  BANNER: 渠道分析报告 + 周期                        │
├─────────────────────────────────────────────────────┤
│  KPI：总流量 | 转化率 | CPA | ROI                   │
├─────────────────────────────────────────────────────┤
│  渠道效果对比雷达图                                 │
├────────────────────────────┬────────────────────────┤
│  渠道转化漏斗对比          │  渠道趋势              │
├────────────────────────────┼────────────────────────┤
│  渠道成本收益              │  渠道明细              │
└────────────────────────────┴────────────────────────┘
```

## 渠道评估维度

- 流量规模、转化效率
- 获客成本、用户质量
- ROI、LTV
