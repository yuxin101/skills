# PR02 功能使用报告模板

适用于：功能渗透分析、功能优化。

## 模板结构

```
┌─────────────────────────────────────────────────────┐
│  BANNER: 功能使用报告 + 周期                        │
├─────────────────────────────────────────────────────┤
│  KPI：功能渗透率 | 使用频次 | 留存率 | 满意度       │
├─────────────────────────────────────────────────────┤
│  功能使用排行                                       │
├────────────────────────────┬────────────────────────┤
│  功能渗透漏斗              │  功能留存              │
├────────────────────────────┼────────────────────────┤
│  用户路径                  │  优化建议              │
└────────────────────────────┴────────────────────────┘
```

## 核心指标

- 功能渗透率
- 功能使用频次
- 功能留存率
- 功能满意度
