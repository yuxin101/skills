# P03 项目复盘报告模板

适用于：项目总结、经验沉淀。

## 模板结构

```
┌─────────────────────────────────────────────────────┐
│  BANNER: 项目复盘 + 项目名称                        │
├─────────────────────────────────────────────────────┤
│  项目概述：目标 | 结果 | 差异                       │
├─────────────────────────────────────────────────────┤
│  做得好的（Continue）                               │
├─────────────────────────────────────────────────────┤
│  需要改进的（Stop）                                 │
├─────────────────────────────────────────────────────┤
│  开始做的（Start）                                  │
├─────────────────────────────────────────────────────┤
│  行动计划                                           │
└─────────────────────────────────────────────────────┘
```

## 复盘框架

- 目标回顾
- 结果评估
- 原因分析
- 经验总结
- 行动计划
