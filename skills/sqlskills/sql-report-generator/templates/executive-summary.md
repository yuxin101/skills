# B05 高管摘要模板

适用于：向高层汇报、一页纸总结。

## 模板结构

```
┌─────────────────────────────────────────────────────┐
│  标题：{业务}摘要 {周期}                            │
├─────────────────────────────────────────────────────┤
│  核心指标（4个关键数字）                            │
│  ┌──────────┐ ┌──────────┐ ┌──────────┐ ┌──────────┐│
│  │ 大数字1  │ │ 大数字2  │ │ 大数字3  │ │ 大数字4  ││
│  └──────────┘ └──────────┘ └──────────┘ └──────────┘│
├─────────────────────────────────────────────────────┤
│  📊 关键发现（3-5条要点）                           │
│  ✅ 要点1                                           │
│  ✅ 要点2                                           │
│  ⚠️ 风险提示                                        │
├─────────────────────────────────────────────────────┤
│  📋 下步行动（3条）                                 │
└─────────────────────────────────────────────────────┘
```

## 设计原则

- 一页纸，不超过 300 字
- 重点数字加粗放大
- 结论先行，数据支撑
- 有明确的行动建议
