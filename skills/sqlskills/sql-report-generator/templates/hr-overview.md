# H01 人力概览报告模板

适用于：人力资源月度/季度汇报。

## 模板结构

```
┌─────────────────────────────────────────────────────┐
│  BANNER: 人力资源概览 + 周期                        │
├─────────────────────────────────────────────────────┤
│  KPI：总人数 | 新增 | 离职率 | 人力成本             │
├─────────────────────────────────────────────────────┤
│  人员结构图（部门/职级/年龄）                       │
├────────────────────────────┬────────────────────────┤
│  入离职趋势                │  部门人数分布          │
├────────────────────────────┼────────────────────────┤
│  人力成本分析              │  人员明细              │
└────────────────────────────┴────────────────────────┘
```

## 核心指标

- 总人数、新增人数、离职人数
- 离职率、入职率
- 人力成本、人均成本
- 部门分布、职级分布
