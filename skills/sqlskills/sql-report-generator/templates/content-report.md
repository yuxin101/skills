# M02 内容报告模板

适用于：内容运营分析。

## 模板结构

```
┌─────────────────────────────────────────────────────┐
│  BANNER: 内容报告 + 周期                            │
├─────────────────────────────────────────────────────┤
│  KPI：发布量 | 阅读量 | 互动量 | 转化量             │
├─────────────────────────────────────────────────────┤
│  内容效果趋势                                       │
├────────────────────────────┬────────────────────────┤
│  内容类型分布              │  Top 内容              │
├────────────────────────────┼────────────────────────┤
│  内容漏斗                  │  优化建议              │
└────────────────────────────┴────────────────────────┘
```

## 核心指标

- 发布量、阅读量、互动量
- 内容类型效果对比
- 高效内容特征分析
