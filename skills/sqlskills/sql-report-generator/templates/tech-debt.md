# T03 技术债务报告模板

适用于：技术债务管理。

## 模板结构

```
┌─────────────────────────────────────────────────────┐
│  BANNER: 技术债务报告 + 周期                        │
├─────────────────────────────────────────────────────┤
│  KPI：债务总量 | 新增债务 | 清理债务 | 债务率       │
├─────────────────────────────────────────────────────┤
│  债务分布图（按类型/模块）                          │
├────────────────────────────┬────────────────────────┤
│  债务优先级                │  清理进度              │
├────────────────────────────┼────────────────────────┤
│  高优债务                  │  清理计划              │
└────────────────────────────┴────────────────────────┘
```

## 核心指标

- 债务总量、债务类型分布
- 新增债务、清理债务
- 债务优先级排序
- 债务清理进度
