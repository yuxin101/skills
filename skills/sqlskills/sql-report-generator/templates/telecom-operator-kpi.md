## 核心指标

| 指标名称 | 英文缩写 | 定义 | 单位 | 目标参考 |
|---------|---------|------|------|---------|
| 总营收 | Revenue | 当月累计营业收入 | 亿元 | 同比≥5% |
| 月均用户收入 | ARPU | 月均每用户收入 | 元 | ≥50元 |
| 户均流量 | DOU | 月均每用户上网流量 | GB | ≥15GB |
| 用户离网率 | Churn Rate | 当月主动离网用户/期初用户 | % | ≤1.5% |
| 5G渗透率 | 5G Penetration | 5G套餐用户/总移动用户 | % | ≥60% |
| 用户净增量 | Net Adds | 新增用户-离网用户 | 万户 | ≥0 |
| 宽带渗透率 | Broadband Penetration | 宽带用户/家庭总数 | % | ≥80% |
| 政企收入占比 | B2B Revenue Ratio | 政企收入/总营收 | % | ≥30% |

## 图表详细定义

### 图表1：营收结构趋势图
- **标题**：月度营收结构趋势（近12个月）
- **图表类型**：堆叠面积图
- **数据格式**：
  ```json
  {
    "x_axis": ["2025-04", "2025-05", "...", "2026-03"],
    "series": [
      {"name": "语音及短信", "values": [12.3, 12.1, "..."]},
      {"name": "移动数据", "values": [45.2, 46.8, "..."]},
      {"name": "政企专线", "values": [18.5, 19.2, "..."]},
      {"name": "增值服务", "values": [8.7, 9.1, "..."]}
    ]
  }
  ```
- **解读**：展示营收各板块贡献度变化，识别增长引擎切换信号。
- **洞察规则**：若移动数据收入连续2个月增速低于3%，标记"流量经营承压"预警；若政企收入占比连续3个月上升，标记"结构优化良好"。

### 图表2：用户净增趋势图
- **标题**：月度用户净增走势（新增 vs 离网）
- **图表类型**：折线+柱状组合图
- **数据格式**：
  ```json
  {
    "x_axis": ["2025-04", "2025-05", "...", "2026-03"],
    "bar_series": {"name": "新增用户", "values": [85, 92, "..."]},
    "line_series": {"name": "离网用户", "values": [45, 52, "..."]}
  }
  ```
- **解读**：柱状为新增用户量，折线为离网用户量，两线间距即净增量。
- **洞察规则**：当离网用户量环比增长超10%时触发"离网预警"；当净增连续3个月为负时触发"用户池萎缩"红色预警。

### 图表3：ARPU分层对比图
- **标题**：各用户分层ARPU对比（本期 vs 去年同期）
- **图表类型**：分组柱状图
- **数据格式**：
  ```json
  {
    "categories": ["低端用户(<30元)", "中端用户(30-80元)", "中高端(80-150元)", "高端用户(>150元)", "政企用户"],
    "series": [
      {"name": "本期ARPU", "values": [18.5, 52.3, 108.7, 235.4, 320.1]},
      {"name": "去年同期", "values": [19.2, 49.8, 105.3, 228.6, 305.8]}
    ]
  }
  ```
- **解读**：观察各用户分层收入贡献变化，识别ARPU下降的来源分层。
- **洞察规则**：若低端用户占比扩大且中端用户占比缩小，标记"用户价值下移"预警；高端用户ARPU下降超5%触发"高价值用户流失风险"。

### 图表4：离网率漏斗分析图
- **标题**：用户离网原因分布漏斗
- **图表类型**：漏斗图
- **数据格式**：
  ```json
  {
    "stages": [
      {"name": "携号转网", "value": 3200, "percent": 28},
      {"name": "合约到期不续", "value": 2800, "percent": 24},
      {"name": "竞品低价抢夺", "value": 2200, "percent": 19},
      {"name": "服务质量投诉", "value": 1500, "percent": 13},
      {"name": "自然消号/其他", "value": 1750, "percent": 16}
    ]
  }
  ```
- **解读**：展示离网用户原因构成，定位主要流失驱动因素。
- **洞察规则**：携号转网占比超25%触发"竞品威胁"预警；服务质量投诉导致的离网连续2月上升触发"服务改进"预警。

### 图表5：5G渗透率区域热力图
- **标题**：各区域5G渗透率达成情况
- **图表类型**：热力地图/色块矩阵
- **数据格式**：
  ```json
  {
    "regions": ["华东", "华南", "华北", "华中", "西南", "西北", "东北"],
    "metrics": {
      "5G渗透率": [72.3, 68.5, 61.2, 55.8, 48.3, 35.6, 42.1],
      "5G DOU": [22.5, 21.8, 19.3, 17.6, 15.2, 12.8, 14.1]
    }
  }
  ```
- **解读**：展示区域间5G发展差异，指导网络建设和营销资源投放。
- **洞察规则**：渗透率低于全国均值10pp以上的区域标记"落后区域"；DOU与渗透率不匹配（高渗透低DOU）标记"流量激发不足"。

## 深度洞察

1. **收入结构转型信号**：当移动数据收入增速持续放缓（连续3个月<3%）而政企收入占比稳步提升时，说明收入结构正在从"流量驱动"向"价值驱动"转型，需关注增量市场的开拓节奏是否匹配存量市场的饱和趋势。

2. **用户价值分层预警**：若低端用户占比连续扩大且中高端用户ARPU持续下滑，可能反映价格战导致的用户价值下移。需区分是"自然升级受阻"还是"高价值用户主动降档"，前者需优化升级路径，后者需加固权益体系。

3. **5G投入产出拐点**：当5G渗透率超过60%且5G用户DOU达到4G用户的2倍以上时，标志着5G投资进入收获期。此阶段应重点监控5G资本开支占收比，确保在渗透率80%前实现投入产出拐点。

4. **携号转网竞争态势**：携号转网净流出连续2个月为负时，表明在区域市场竞争中处于劣势。需结合竞品套餐政策变动、网络质量测评排名等维度交叉分析，判断流失是短期波动还是结构性问题。

## 运营建议

| 优先级 | 建议内容 | 预期效果 |
|-------|---------|---------|
| P0 | 立即启动高价值用户挽留计划，对ARPU>150元且合约即将到期的用户推送专属续约权益包，包含流量翻倍+家庭宽带免费升级 | 降低高价值用户离网率0.3-0.5pp |
| P1 | 针对携号转网净流出最大的3个区域，部署"网络质量承诺+价格保护"双保险方案，并在营业厅设置竞品对比物料 | 转网净流出减少40%以上 |
| P1 | 优化5G套餐升级路径，对DOU>10GB的4G用户推送定向升档优惠，降低升档门槛（如首月免费体验） | 月均5G新增用户提升15-20% |
| P2 | 建立离网预测模型，提前30天识别离网风险用户（行为特征：流量骤降+投诉记录+合约临期），自动触发挽留工单 | 预防性挽留成功率提升至35% |
| P2 | 政企收入拓展：梳理区域内中小企业数字化转型需求，打包云+网+安全组合方案，设定季度政企客户净增目标 | 政企收入占比季度提升2pp |
