# M01 营销活动报告模板

适用于：营销活动效果评估。

## 模板结构

```
┌─────────────────────────────────────────────────────┐
│  BANNER: 营销活动报告 + 活动名称                    │
├─────────────────────────────────────────────────────┤
│  KPI：曝光 | 点击 | 转化 | ROI                      │
├─────────────────────────────────────────────────────┤
│  活动效果漏斗                                       │
├────────────────────────────┬────────────────────────┤
│  渠道效果对比              │  时间趋势              │
├────────────────────────────┼────────────────────────┤
│  成本收益分析              │  经验总结              │
└────────────────────────────┴────────────────────────┘
```

## 核心指标

- 曝光量、点击量、转化量
- CTR、CVR、CPC、CPA
- ROI、ROAS
- 渠道效果对比
