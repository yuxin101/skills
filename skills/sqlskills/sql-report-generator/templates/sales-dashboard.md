# O01 销售看板模板

适用于：销售业绩追踪、销售团队管理。

## 模板结构

```
┌─────────────────────────────────────────────────────┐
│  BANNER: 销售业绩看板 + 周期                        │
├─────────────────────────────────────────────────────┤
│  KPI：销售额 | 订单量 | 客单价 | 目标达成率         │
├─────────────────────────────────────────────────────┤
│  趋势图（含目标线）                                 │
├────────────────────────────┬────────────────────────┤
│  Top 销售员/产品排名      │  区域分布              │
├────────────────────────────┼────────────────────────┤
│  品类占比                  │  增长/下滑排名         │
└────────────────────────────┴────────────────────────┘
```

## 核心指标

- 总销售额、订单量、客单价
- 目标达成率、环比增长率
- Top 销售员、Top 产品
- 区域分布、品类构成
