# F04 财务比率分析模板

适用于：财务健康度分析。

## 模板结构

```
┌─────────────────────────────────────────────────────┐
│  BANNER: 财务比率分析 + 周期                        │
├─────────────────────────────────────────────────────┤
│  KPI：ROE | ROA | 毛利率 | 净利率                   │
├─────────────────────────────────────────────────────┤
│  杜邦分析图                                         │
├────────────────────────────┬────────────────────────┤
│  盈利能力                  │  偿债能力              │
├────────────────────────────┼────────────────────────┤
│  运营能力                  │  成长能力              │
└────────────────────────────┴────────────────────────┘
```

## 核心指标

- 盈利能力：毛利率、净利率、ROE、ROA
- 偿债能力：流动比率、速动比率、资产负债率
- 运营能力：周转率
- 成长能力：收入增长率、利润增长率
