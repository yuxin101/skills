# B07 对比报告模板

适用于：多维度横向对比分析。

## 模板结构

```
┌─────────────────────────────────────────────────────┐
│  BANNER: 对比报告 + 对比维度                        │
├─────────────────────────────────────────────────────┤
│  对比概览                                           │
├─────────────────────────────────────────────────────┤
│  对比图表（分组条形图/雷达图）                      │
├────────────────────────────┬────────────────────────┤
│  差异分析                  │  排名变化              │
├────────────────────────────┼────────────────────────┤
│  差异原因                  │  改进建议              │
└────────────────────────────┴────────────────────────┘
```

## 对比维度

- 时间对比：同比、环比
- 维度对比：渠道、区域、品类
- 目标对比：实际 vs 目标
