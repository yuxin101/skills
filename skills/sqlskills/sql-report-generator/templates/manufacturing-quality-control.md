## 核心指标

- **不良率（Defect Rate）** — 不良品数量占检验总数量的百分比，核心质量指标
- **返工率（Rework Rate）** — 需返工处理的产品占比，衡量工艺稳定性
- **报废率（Scrap Rate）** — 无法修复直接报废的产品占比，衡量材料浪费
- **CPK值（过程能力指数）** — 衡量工序能力是否满足规格要求，CPK≥1.33为合格
- **一次检验合格率（First Pass Yield）** — 首次检验即合格的产品比例，反映过程稳定性
- **质量成本（Cost of Quality）** — 预防成本+鉴定成本+内部损失+外部损失的总和
- **客户投诉率（Customer Complaint Rate）** — 客户投诉不良品数量占发货总数的比例
- **检验覆盖率（Inspection Coverage）** — 实际检验数量占应检验数量的比例

## 图表详细定义

### 1. 不良率趋势（折线图）

- **图表类型**: line（折线图）
- **数据格式**:
```python
{
    "xAxis": ["1月", "2月", "3月", "4月", "5月", "6月", "7月", "8月", "9月", "10月", "11月", "12月"],
    "series": [
        {"name": "不良率(%)", "data": [2.8, 2.5, 2.3, 2.1, 1.9, 1.8, 1.7, 1.6, 1.5, 1.4, 1.3, 1.2]},
        {"name": "目标值(%)", "data": [2.0, 2.0, 2.0, 1.8, 1.8, 1.8, 1.5, 1.5, 1.5, 1.2, 1.2, 1.2]},
        {"name": "行业标杆(%)", "data": [1.5, 1.5, 1.4, 1.4, 1.3, 1.3, 1.2, 1.2, 1.1, 1.1, 1.0, 1.0]}
    ]
}
```
- **简洁解读**: 追踪不良率变化趋势，对比目标值和行业标杆，评估质量改进成效
- **洞察规则**: 不良率持续下降且低于目标→质量改进有效；不良率反弹或长期横盘→需专项攻关；与行业标杆差距>50%→有较大提升空间

---

### 2. 缺陷帕累托图（柱形图+折线）

- **图表类型**: combo（柱形图+折线图组合）
- **数据格式**:
```python
{
    "categories": ["尺寸超差", "表面划伤", "外观瑕疵", "功能不良", "装配错误", "包装破损", "标识错误", "其他"],
    "bars": [
        {"name": "缺陷数量(件)", "data": [1850, 920, 680, 520, 380, 220, 150, 280]}
    ],
    "lines": [
        {"name": "累计占比(%)", "data": [40.2, 60.2, 75.0, 86.3, 94.6, 99.4, 99.9, 100.0]}
    ]
}
```
- **简洁解读**: 依据帕累托法则，识别造成80%缺陷的关键少数因素，聚焦改进重点
- **洞察规则**: 前3项缺陷累计占比>70%→集中攻克这3项即可显著改善质量；某项缺陷占比突然上升→需追溯工序变更

### 3. 工序质量热力图

- **图表类型**: heatmap（热力图）
- **数据格式**:
```python
{
    "xAxis": ["周一", "周二", "周三", "周四", "周五", "周六", "周日"],
    "yAxis": ["焊接工序", "组装工序", "涂装工序", "检测工序", "包装工序", "入库检验"],
    "data": [
        [1.2, 1.5, 1.8, 2.1, 1.9, 1.6, 1.3],
        [0.8, 0.9, 1.1, 1.3, 1.2, 1.0, 0.9],
        [2.5, 2.8, 3.2, 3.8, 3.5, 3.0, 2.6],
        [0.5, 0.6, 0.7, 0.9, 0.8, 0.7, 0.6],
        [1.0, 1.2, 1.4, 1.6, 1.5, 1.3, 1.1],
        [0.3, 0.4, 0.5, 0.6, 0.5, 0.4, 0.3]
    ],
    "colorScale": "Red-Yellow-Green"
}
```
- **简洁解读**: 以不良率着色，红色区域为质量风险点，一目了然定位问题工序和时段
- **洞察规则**: 某工序整周红色→工序本身存在系统性问题；周四整体不良率高→可能与产能冲刺或人员疲劳相关

### 4. CPK仪表盘组

- **图表类型**: gauge（仪表盘，多图并列）
- **数据格式**:
```python
{
    "gauges": [
        {"name": "焊接工序CPK", "value": 1.45, "min": 0, "max": 2.5, "thresholds": {"green": 1.33, "yellow": 1.0, "red": 0}},
        {"name": "组装工序CPK", "value": 1.28, "min": 0, "max": 2.5, "thresholds": {"green": 1.33, "yellow": 1.0, "red": 0}},
        {"name": "涂装工序CPK", "value": 0.95, "min": 0, "max": 2.5, "thresholds": {"green": 1.33, "yellow": 1.0, "red": 0}},
        {"name": "检测工序CPK", "value": 1.68, "min": 0, "max": 2.5, "thresholds": {"green": 1.33, "yellow": 1.0, "red": 0}},
        {"name": "包装工序CPK", "value": 1.52, "min": 0, "max": 2.5, "thresholds": {"green": 1.33, "yellow": 1.0, "red": 0}}
    ]
}
```
- **简洁解读**: CPK≥1.33为绿色合格，1.0-1.33为黄色预警，<1.0为红色不合格，快速识别工序能力短板
- **洞察规则**: CPK<1.0→工序能力严重不足需停线整改；CPK在1.0-1.33之间→需持续监控和优化；CPK>2.0→可考虑放宽检验频次

## 深度洞察

1. **质量波动根因定位**：通过工序质量热力图发现涂装工序不良率显著高于其他工序，结合帕累托图的表面划伤占比高，可定位涂装工艺存在系统性问题。

2. **过程能力预警机制**：当CPK值连续3天低于1.0时自动触发预警，联动设备维护、原材料检验、工艺参数检查，避免批量不良品产生。

3. **缺陷成本分析**：结合帕累托图缺陷分布和各类缺陷的返工/报废成本，计算质量损失成本占比，识别"高成本缺陷"优先解决。

4. **质量改进闭环验证**：针对某一缺陷类型实施改进措施后，追踪该缺陷在帕累托图中的占比变化和不良率趋势，验证改进效果。

5. **供应商质量关联**：将不良品追溯到原材料批次，分析不同供应商的材料对不良率的贡献度，支撑供应商质量评价。

---

## 运营建议

1. **P0 - 建立不良品快速追溯机制**：对不良率超过目标值150%的批次，24小时内完成根因分析报告，追溯至原材料批次、设备参数、操作人员。优先级来源：不良率趋势图突破目标线即触发。

2. **P0 - 强化CPK监控预警**：对CPK值处于黄色预警区(1.0-1.33)的工序，每日输出过程能力报告，制定工艺参数优化方案，目标CPK提升至1.33以上。

3. **P1 - 实施帕累托改进法**：对累计占比达80%的前3项缺陷成立专项改进小组，采用5Why分析法和鱼骨图定位根因，每两周输出改进进展。

4. **P1 - 优化检验抽样方案**：针对CPK>2.0的高稳定工序，评估降低检验频次或放宽抽样比例的可行性，释放检验资源聚焦高风险工序。

5. **P2 - 建立质量成本核算体系**：将预防成本、鉴定成本、内部损失、外部损失分项统计，输出质量成本占销售额比例，目标控制在2%以内。

6. **P2 - 推进质量看板数字化**：将线下纸质检验记录全面数字化，实现检验数据实时采集、自动计算不良率、实时推送预警，缩短问题发现到响应的周期。

7. **P3 - 开展质量意识培训**：针对不良率较高的时段(如周四、周五)，加强班前质量提醒，开展质量案例分析培训，提升全员质量意识。
