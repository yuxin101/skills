# F02 资产负债表模板

适用于：资产负债结构分析。

## 模板结构

```
┌─────────────────────────────────────────────────────┐
│  BANNER: 资产负债表 + 时点                          │
├─────────────────────────────────────────────────────┤
│  KPI：总资产 | 总负债 | 净资产 | 资产负债率         │
├─────────────────────────────────────────────────────┤
│  资产结构图                                         │
├────────────────────────────┬────────────────────────┤
│  负债结构                  │  权益结构              │
├────────────────────────────┼────────────────────────┤
│  偿债能力指标              │  明细表格              │
└────────────────────────────┴────────────────────────┘
```

## 核心指标

- 流动比率、速动比率
- 资产负债率、权益乘数
