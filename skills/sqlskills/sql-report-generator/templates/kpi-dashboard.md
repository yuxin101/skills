# B06 KPI 看板模板

适用于：关键指标追踪、目标管理。

## 模板结构

```
┌─────────────────────────────────────────────────────┐
│  BANNER: KPI 看板 + 周期                            │
├─────────────────────────────────────────────────────┤
│  KPI 卡片矩阵（2x4 或 4x2）                         │
│  每个卡片：指标名 | 当前值 | 目标 | 达成率 | 趋势   │
├─────────────────────────────────────────────────────┤
│  KPI 趋势对比图（多系列折线）                       │
├────────────────────────────┬────────────────────────┤
│  达成率排名                │  预警指标              │
├────────────────────────────┴────────────────────────┤
│  KPI 分析 & 改进建议                                │
└─────────────────────────────────────────────────────┘
```

## 状态标识

- 🟢 达成：≥100%
- 🟡 接近：80%-99%
- 🔮 风险：60%-79%
- 🔴 滞后：<60%
