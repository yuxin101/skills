# 新能源汽车销售&交付看板

适用于：新能源车企销售运营、交付管理、区域销售分析
对标：比亚迪数据大屏、蔚来 NIO App 数据中心、小鹏数据中台

## 模板布局

```
┌─────────────────────────────────────────────────────────────┐
│  BANNER: 新能源汽车销售&交付看板 + 月份选择 + 车型筛选        │
├─────────────────────────────────────────────────────────────┤
│  KPI: 月交付量 | 累计交付 | 订单量 | 等车周期 | 市占率       │
├─────────────────────────────────────────────────────────────┤
│  月度交付趋势（柱形图）        │  品牌份额对比（饼图）        │
├────────────────────────────────────┼────────────────────────┤
│  车型销量排行（条形图）        │  区域交付分布（柱形图）      │
├────────────────────────────────────┼────────────────────────┤
│  订单转化漏斗（漏斗图）        │  竞品对比矩阵（表格）       │
├────────────────────────────────────┴────────────────────────┤
│  深度洞察 + 运营建议                                      │
└─────────────────────────────────────────────────────────────┘
```

## 核心指标

- **月交付量** — 当月实际交付车辆数 — 目标：环比 +5%
- **累计交付量** — 年度累计交付 — 目标：达成年度目标 85%+
- **新增订单量** — 当月新增未交付订单 — 目标：环比 +8%
- **等车周期** — 从下订到交付平均天数 — 目标：<30 天
- **市占率** — 本品牌在新能源市场的份额 — 目标：环比 +0.3%
- **交付满意度** — 交付环节客户评分 — 目标：>4.5 分
- **退订率** — 订单退订比例 — 目标：<5%

## 图表详细定义

### 1. 月度交付趋势
- **图表类型**: clustered_column（簇状柱形图）
- **数据格式**:
```python
{"categories": ["1月","2月","3月","4月","5月","6月"],
 "series": [{"name": "交付量", "data": [15000, 12000, 18000, 20000, 22000, 25000]},
             {"name": "目标量", "data": [16000, 14000, 18000, 19000, 21000, 24000]}]}
```
- **简洁解读**: 对比实际交付与目标，识别交付缺口和季节性规律
- **洞察规则**: 连续 2 个月未达目标需预警；春节月（1-2月）交付量下降属正常

### 2. 品牌市场份额
- **图表类型**: donut（环形图）
- **数据格式**:
```python
{"labels": ["比亚迪","特斯拉","广汽埃安","蔚来","理想","小鹏","零跑","其他"],
 "values": [35, 15, 10, 8, 7, 5, 4, 16]}
```
- **简洁解读**: 直观展示新能源市场格局和本品牌竞争位置
- **洞察规则**: 市占率下降 >1% 需关注竞品动态；CR3 >50% 说明市场集中度高

### 3. 车型销量排行
- **图表类型**: clustered_bar（簇状条形图）
- **数据格式**:
```python
{"categories": ["车型A","车型B","车型C","车型D","车型E"],
 "series": [{"name": "交付量", "data": [8000, 6500, 5000, 3500, 2000]},
             {"name": "订单量", "data": [9000, 7500, 5500, 4000, 2500]}]}
```
- **简洁解读**: 识别明星车型和滞销车型，指导产能分配
- **洞察规则**: 订单-交付比 >1.3 说明产能不足；<0.8 说明库存积压

### 4. 区域交付分布
- **图表类型**: clustered_column（簇状柱形图）
- **数据格式**:
```python
{"categories": ["华东","华南","华北","西南","西北","东北","华中"],
 "series": [{"name": "交付量", "data": [8500, 5200, 4800, 2500, 1200, 800, 2000]}]}
```
- **简洁解读**: 展示全国交付分布，识别增长机会区域
- **洞察规则**: 西北/东北占比较低但增速最快，是拓展重点

### 5. 订单转化漏斗
- **图表类型**: funnel（漏斗图）
- **数据格式**:
```python
{"stages": ["线索留资", "试驾预约", "试驾完成", "下订", "锁单", "交付"],
 "values": [100000, 35000, 25000, 15000, 13000, 12000]}
```
- **简洁解读**: 追踪从线索到交付的全链路转化，定位流失最大环节
- **洞察规则**: 试驾预约→完成转化率 <70% 需优化试驾体验；下订→锁单 <85% 需优化交付沟通

### 6. 竞品对比矩阵
- **图表类型**: table（表格）
- **数据格式**:
```python
{"columns": ["品牌","月交付","同比","均价(万)","等车周期","续航(km)","评分"],
 "rows": [["本品牌", 25000, "+35%", "18.5", "28天", 550, 4.6],
          ["竞品A", 30000, "+25%", "22.0", "35天", 600, 4.5],
          ["竞品B", 18000, "+40%", "15.8", "21天", 450, 4.4]]}
```
- **简洁解读**: 多维度对比竞品，识别差异化优势和劣势
- **洞察规则**: 等车周期 >35天或价格高于竞品 20%+ 需调整策略

## 深度洞察

1. **交付量稳步增长**：连续 6 个月环比增长，Q2 目标达成率 95%，增长势头良好
2. **等车周期缩短**：从 35 天缩短至 28 天，产能提升效果显著
3. **车型A供不应求**：订单-交付比 1.13，需加快产能爬坡
4. **区域发展不均衡**：华东占 34%，西北仅占 5%，市场渗透空间大
5. **竞品A价格压制**：均价低于本品牌 16%，需评估是否调整定价或增加配置

## 运营建议

1. 加快车型A产能爬坡，目标将等车周期降至 25 天 — 高优先级
2. 加大西北/东北门店和充电网络建设 — 中优先级
3. 优化试驾流程，提升试驾→下订转化率至 70% — 高优先级
4. 针对竞品A推出限时金融方案或配置升级 — 中优先级
5. 建立订单交付预警机制，等车超 45 天主动关怀客户 — 高优先级
6. 在华东等成熟市场推出 OTA 升级等增值服务，提升用户粘性 — 低优先级
