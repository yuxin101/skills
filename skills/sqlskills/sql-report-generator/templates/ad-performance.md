# M03 广告投放报告模板

适用于：广告投放效果分析。

## 模板结构

```
┌─────────────────────────────────────────────────────┐
│  BANNER: 广告投放报告 + 周期                        │
├─────────────────────────────────────────────────────┤
│  KPI：消耗 | 展现 | 点击 | 转化 | ROI               │
├─────────────────────────────────────────────────────┤
│  投放趋势图                                         │
├────────────────────────────┬────────────────────────┤
│  创意效果对比              │  定向分析              │
├────────────────────────────┼────────────────────────┤
│  成本趋势                  │  优化建议              │
└────────────────────────────┴────────────────────────┘
```

## 核心指标

- 消耗金额、展现量、点击量
- CPC、CTR、CVR
- CPA、ROI
