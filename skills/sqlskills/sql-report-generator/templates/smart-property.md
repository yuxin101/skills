# 智慧物业&园区运营看板

适用于：物业管理公司运营管理、产业园区运营、楼宇资产监控
对标：万科万物云、龙湖智慧服务、碧桂园服务数字平台

## 模板布局

```
┌─────────────────────────────────────────────────────────────┐
│  BANNER: 智慧物业运营看板 + 楼宇/园区筛选 + 日期            │
├─────────────────────────────────────────────────────────────┤
│  KPI: 入住率 | 收缴率 | 工单量 | 完成率 | 故障数 | 能耗   │
├─────────────────────────────────────────────────────────────┤
│  入住率趋势（仪表盘+折线）      │  工单类型分布（饼图）    │
├────────────────────────────────────┼────────────────────────┤
│  能耗趋势（组合图）              │  区域/楼宇对比（条形图） │
├────────────────────────────────────┼────────────────────────┤
│  设备状态监控（表格）            │  费用收缴漏斗（漏斗图）  │
├────────────────────────────────────┴────────────────────────┤
│  深度洞察 + 运营建议                                      │
└─────────────────────────────────────────────────────────────┘
```

## 核心指标

- **入住率** — 已租面积/可租面积 — 目标：> 85%
- **物业费收缴率** — 实收/应收物业费 — 目标：> 95%
- **工单量** — 报事报修工单总数 — 基准：每月 <500 单/万平米
- **工单完成率** — 按时完成工单占比 — 目标：> 90%
- **设备故障数** — 电梯/消防/水电等故障 — 目标：< 5 次/月
- **能耗** — 单位面积综合能耗 — 目标：同比 -5%
- **客户满意度** — 物业服务评分 — 目标：> 4.2 分

## 图表详细定义

### 1. 入住率趋势
- **图表类型**: gauge（仪表盘）+ line（折线图）
- **数据格式**:
```python
gauge_data = {"title": "当前入住率", "value": 88, "min": 0, "max": 100,
               "ranges": [{"from": 0, "to": 70, "color": "#FF4444"},
                          {"from": 70, "to": 85, "color": "#FFB300"},
                          {"from": 85, "to": 100, "color": "#00C853"}]}
line_data = {"categories": ["1月","2月","3月","4月","5月","6月"],
              "series": [{"name": "入住率%", "data": [82, 83, 85, 86, 87, 88]}]}
```
- **简洁解读**: 实时显示入住率并追踪月度变化趋势
- **洞察规则**: 入住率 <80% 需加大招商；持续下滑 >3 个月预警租户流失

### 2. 工单类型分布
- **图表类型**: donut（环形图）
- **数据格式**:
```python
{"labels": ["电梯故障","水电气维修","环境保洁","绿化养护","停车管理","安全投诉","其他"],
 "values": [25, 22, 18, 12, 10, 8, 5]}
```
- **简洁解读**: 识别高频工单类型，优先配置对应维修资源
- **洞察规则**: 电梯故障 >30% 说明设备老化严重；环境投诉 >15% 需加强保洁频次

### 3. 能耗趋势
- **图表类型**: combo（组合图：柱形=总能耗，折线=单位面积能耗）
- **数据格式**:
```python
{"categories": ["1月","2月","3月","4月","5月","6月"],
 "columns": {"name": "总能耗(kWh)", "data": [45000, 42000, 38000, 35000, 32000, 30000]},
 "lines":   {"name": "单位kWh/m2", "data": [8.5, 8.0, 7.2, 6.5, 5.8, 5.2]}}
```
- **简洁解读**: 监控能耗变化，夏季制冷高峰期预警
- **洞察规则**: 单位面积能耗 >10 kWh/m2 说明节能设备需升级

### 4. 区域/楼宇对比
- **图表类型**: clustered_bar（簇状条形图）
- **数据格式**:
```python
{"categories": ["楼宇A","楼宇B","楼宇C","楼宇D","园区E"],
 "series": [{"name": "收缴率%", "data": [96, 92, 88, 85, 98]},
             {"name": "满意率%", "data": [4.5, 4.2, 3.9, 3.8, 4.6]}]}
```
- **简洁解读**: 多维度对比各楼宇/园区运营表现
- **洞察规则**: 收缴率 <90% 需专项催缴；满意率 <4.0 的项目需驻场改善

### 5. 设备状态监控
- **图表类型**: table（表格）
- **数据格式**:
```python
{"columns": ["设备名称","类型","位置","状态","上次保养","下次保养","故障次数"],
 "rows": [["1号电梯", "电梯", "A栋", "正常", "3月5日", "6月5日", 1],
          ["消防泵", "消防", "B1层", "正常", "2月20日", "5月20日", 0],
          ["空调主机1", "暖通", "屋顶", "告警", "1月10日", "4月10日", 3],
          ["监控系统", "弱电", "中控室", "正常", "3月1日", "6月1日", 0]]}
```
- **简洁解读**: 全面掌握设备健康状态，预防重大故障
- **洞察规则**: 故障次数 >5/月需大修或更换；保养超期设备高亮预警

### 6. 费用收缴漏斗
- **图表类型**: funnel（漏斗图）
- **数据格式**:
```python
{"stages": ["应收金额", "已出账", "已缴费", "逾期未缴", "催缴成功"],
 "values": [5000000, 4800000, 4400000, 400000, 280000]}
```
- **简洁解读**: 从应收→催缴追踪费用回收全链路
- **洞察规则**: 逾期未缴 >10% 需启动专项催缴；催缴成功率 >60% 说明催缴有效

## 深度洞察

1. **入住率持续攀升**：从 82% 增至 88%，招商去化效果显著，但楼宇D入住率仅 75% 拉低均值
2. **设备老化风险**：空调主机1三个月故障 3 次，需纳入大修计划
3. **能耗逐年下降**：得益于 LED 照明和智能温控改造，单位面积能耗同比 -12%
4. **园区E运营最优**：收缴率 98% 且满意率 4.6，其经验可复制到其他项目

## 运营建议

1. 针对楼宇D制定专项招商去化方案，目标 Q3 入住率提升至 85% — 高优先级
2. 将园区E的运营标准（收缴+满意率）制定为全公司标杆并推广 — 高优先级
3. 加快空调主机1大修审批，避免夏季高峰故障 — 高优先级
4. 建立设备预防性保养机制，将故障率控制在 3 次/月以内 — 中优先级
5. 推出物业费预缴优惠（年缴享折扣），提升提前缴费比例 — 中优先级
6. 部署能耗实时监控，夏季峰值预警自动推送 — 低优先级
