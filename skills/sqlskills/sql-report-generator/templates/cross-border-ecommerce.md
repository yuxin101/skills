# 跨境电商运营看板

适用于：跨境电商运营、多平台销售管理、跨境供应链监控
对标：亚马逊卖家中心、SHEIN 供应商数据平台、阿里国际站数据管家

## 模板布局

```
┌─────────────────────────────────────────────────────────────┐
│  BANNER: 跨境电商运营看板 + 币种切换 + 平台筛选               │
├─────────────────────────────────────────────────────────────┤
│  KPI: GMV(USD) | 订单量 | 客单价 | 汇率损益 | 物流时效 | 退货率 │
├─────────────────────────────────────────────────────────────┤
│  GMV趋势（折线图）             │  平台GMV分布（饼图）        │
├────────────────────────────────────┼────────────────────────┤
│  国家/地区GMV（条形图）        │  物流时效仪表盘             │
├────────────────────────────────────┼────────────────────────┤
│  汇率影响趋势（组合图）        │  退货原因分析（饼图）       │
├────────────────────────────────────┴────────────────────────┤
│  深度洞察 + 运营建议                                      │
└─────────────────────────────────────────────────────────────┘
```

## 核心指标

- **GMV（USD）** — 跨境成交总额（美元） — 目标：环比 +10%
- **订单量** — 总订单数 — 目标：环比 +8%
- **客单价（USD）** — GMV/订单量 — 目标：>$30
- **汇率损益** — 汇率波动导致的收入/成本变化 — 目标：损失 <$2%
- **平均物流时效** — 从发货到签收天数 — 目标：<15 天
- **退货率** — 退货订单/总订单 — 目标：<8%

## 图表详细定义

### 1. GMV 趋势
- **图表类型**: line（折线图）
- **数据格式**:
```python
{"categories": ["1月","2月","3月","4月","5月","6月"],
 "series": [{"name": "GMV(万USD)", "data": [50, 45, 58, 65, 72, 80]},
             {"name": "目标(万USD)", "data": [55, 50, 60, 65, 70, 75]}]}
```
- **简洁解读**: 追踪 GMV 增长趋势，对比目标完成进度
- **洞察规则**: 连续 2 个月未达目标需排查供应链或营销问题

### 2. 平台 GMV 分布
- **图表类型**: donut（环形图）
- **数据格式**:
```python
{"labels": ["Amazon US","Amazon EU","Shopee","Lazada","独立站","其他"],
 "values": [40, 20, 15, 10, 10, 5]}
```
- **简洁解读**: 分析多平台 GMV 结构，避免过度依赖单一平台
- **洞察规则**: 单一平台 >60% 存在政策风险；独立站占比 >15% 为健康结构

### 3. 国家/地区 GMV
- **图表类型**: clustered_bar（簇状条形图）
- **数据格式**:
```python
{"categories": ["美国","英国","德国","日本","东南亚","巴西","其他"],
 "series": [{"name": "GMV(万USD)", "data": [35, 12, 10, 8, 15, 5, 5]}]}
```
- **简洁解读**: 识别主力市场和增长市场，指导市场拓展
- **洞察规则**: 美国占比 >35% 属正常但需分散；新兴市场增速 >20% 可加大投入

### 4. 物流时效
- **图表类型**: gauge（仪表盘）
- **数据格式**:
```python
{"value": 12, "min": 0, "max": 30, "thresholds": [
    {"value": 10, "label": "优秀", "color": "#4CAF50"},
    {"value": 15, "label": "达标", "color": "#FFC107"},
    {"value": 30, "label": "预警", "color": "#F44336"}
]}
```
- **简洁解读**: 实时监控物流时效，超时自动预警
- **洞察规则**: 平均时效 >15 天需排查物流商；单一国家 >25 天需切换物流方案

### 5. 汇率影响趋势
- **图表类型**: combo（组合图）
- **数据格式**:
```python
{"categories": ["1月","2月","3月","4月","5月","6月"],
 "columns": {"name": "USD/CNY", "data": [7.10, 7.08, 7.15, 7.20, 7.18, 7.22]},
 "lines":   {"name": "汇率损益(万)", "data": [-2, -1, 1, 3, 2, 4]}}
```
- **简洁解读**: 监控汇率变动对利润的影响，指导定价和套期策略
- **洞察规则**: 汇率单月波动 >2% 需考虑锁汇或调价

### 6. 退货原因
- **图表类型**: pie（饼图）
- **数据格式**:
```python
{"labels": ["尺寸不符","质量问题","物流损坏","色差","不喜欢","其他"],
 "values": [30, 25, 15, 15, 10, 5]}
```
- **简洁解读**: 识别主要退货原因，指导产品改进和描述优化
- **洞察规则**: 尺寸+色差占比 >40% 说明商品详情页描述需改进

## 深度洞察

1. **美国市场增长放缓**：GMV 环比仅 +3%，低于整体 +10%，需加大美国营销
2. **汇率正向贡献**：USD/CNY 升至 7.22，累计正向损益 +4 万，美元定价产品受益
3. **物流时效优秀**：平均 12 天，但巴西线 >25 天拖累整体，需切换巴西物流商
4. **退货率偏高**：8.5% 超目标 8%，主因尺寸不符（30%），需优化尺码表

## 运营建议

1. 优化美国站产品详情页和广告投放，目标 GMV 环比 +10% — 高优先级
2. 切换巴西线物流商，将时效从 25 天降至 18 天 — 高优先级
3. 重新制作尺码指南（含对比图+视频），降低尺码退货率 — 高优先级
4. 增加独立站占比至 15%，降低单一平台依赖 — 中优先级
5. 对高客单价商品启用锁汇策略，降低汇率波动风险 — 中优先级
6. 在东南亚市场推出本地化选品，利用增速红利 — 低优先级
7. 建立物流时效实时监控告警机制 — 低优先级
