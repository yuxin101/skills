# T01 系统健康报告模板

适用于：系统运维汇报、SLA评估。

## 模板结构

```
┌─────────────────────────────────────────────────────┐
│  BANNER: 系统健康报告 + 周期                        │
├─────────────────────────────────────────────────────┤
│  KPI：可用性 | 平均响应时间 | 错误率 | 故障次数     │
├─────────────────────────────────────────────────────┤
│  可用性趋势图                                       │
├────────────────────────────┬────────────────────────┤
│  性能指标                  │  错误分布              │
├────────────────────────────┼────────────────────────┤
│  资源使用                  │  改进计划              │
└────────────────────────────┴────────────────────────┘
```

## 核心指标

- 可用性（SLA）
- 平均响应时间、P99响应时间
- 错误率、错误类型分布
- CPU、内存、磁盘使用率
