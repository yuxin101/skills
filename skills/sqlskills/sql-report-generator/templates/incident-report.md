# T02 故障报告模板

适用于：故障复盘、事故分析。

## 模板结构

```
┌─────────────────────────────────────────────────────┐
│  BANNER: 故障报告 + 故障ID                          │
├─────────────────────────────────────────────────────┤
│  故障概述：时间 | 影响 | 根因                       │
├─────────────────────────────────────────────────────┤
│  时间线                                             │
├─────────────────────────────────────────────────────┤
│  影响范围                                           │
├────────────────────────────┬────────────────────────┤
│  根因分析                  │  改进措施              │
├────────────────────────────┼────────────────────────┤
│  经验教训                  │  行动计划              │
└────────────────────────────┴────────────────────────┘
```

## 故障等级

- P0：核心业务中断
- P1：核心功能受影响
- P2：非核心功能受影响
- P3：轻微问题
