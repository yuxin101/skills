# C02 工单分析报告模板

适用于：工单数据深度分析。

## 模板结构

```
┌─────────────────────────────────────────────────────┐
│  BANNER: 工单分析报告 + 周期                        │
├─────────────────────────────────────────────────────┤
│  KPI：工单总量 | 平均处理时长 | 重开工单率 | 满意度 │
├─────────────────────────────────────────────────────┤
│  工单趋势图                                         │
├────────────────────────────┬────────────────────────┤
│  问题类型分布              │  处理时效分布          │
├────────────────────────────┼────────────────────────┤
│  高频问题                  │  改进建议              │
└────────────────────────────┴────────────────────────┘
```

## 核心指标

- 工单量趋势
- 问题类型占比
- 处理时效分布
- 重开工单分析
