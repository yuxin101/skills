## 核心指标

- **种植面积（Planting Area）** — 农场实际种植的作物面积，单位为亩或公顷
- **预估产量（Estimated Yield）** — 基于当前生长状况预测的作物产量，单位为吨或公斤
- **病虫害预警数（Pest Warning Count）** — 系统监测到的病虫害风险预警数量
- **土壤健康度（Soil Health Index）** — 综合土壤pH值、有机质、养分含量等指标的评分
- **气象适宜度（Weather Suitability）** — 当前气象条件对作物生长的适宜程度评分
- **农事完成率（Farming Task Completion）** — 已完成的农事任务占总计划任务的比例
- **灌溉用水量（Irrigation Water）** — 单位面积或单位产量的灌溉用水量
- **肥料利用率（Fertilizer Utilization）** — 作物吸收的肥料养分占施肥总量的比例

## 图表详细定义

### 1. 农事进度（甘特图/表格）

- **图表类型**: table（甘特图样式表格）
- **数据格式**:
```python
{
    "columns": ["农事任务", "计划开始", "计划结束", "实际开始", "实际结束", "完成进度", "负责人", "状态"],
    "rows": [
        ["春耕整地", "2024-03-01", "2024-03-15", "2024-03-02", "2024-03-14", "100%", "张农", "已完成"],
        ["播种施肥", "2024-03-16", "2024-03-25", "2024-03-17", "2024-03-24", "100%", "李农", "已完成"],
        ["苗期管理", "2024-04-01", "2024-04-30", "2024-04-01", "进行中", "65%", "王农", "进行中"],
        ["病虫害防治", "2024-05-01", "2024-05-15", "-", "-", "0%", "赵农", "未开始"],
        ["灌溉管理", "2024-05-10", "2024-06-30", "-", "-", "0%", "钱农", "未开始"],
        ["收获作业", "2024-09-15", "2024-10-15", "-", "-", "0%", "孙农", "未开始"]
    ]
}
```
- **简洁解读**: 展示农事任务的计划与实际进度对比，确保农事活动按时完成
- **洞察规则**: 任务进度<计划进度→需加快作业或调整计划；任务延期>7天→需评估对产量的影响

---

### 2. 产量趋势（柱形图）

- **图表类型**: clustered_column（簇状柱形图）
- **数据格式**:
```python
{
    "xAxis": ["2020年", "2021年", "2022年", "2023年", "2024年(预估)"],
    "series": [
        {"name": "小麦(吨)", "data": [1250, 1320, 1280, 1380, 1450]},
        {"name": "玉米(吨)", "data": [1680, 1750, 1720, 1850, 1920]},
        {"name": "大豆(吨)", "data": [420, 450, 435, 480, 510]}
    ]
}
```
- **简洁解读**: 追踪历年作物产量变化，评估种植技术和管理措施的成效
- **洞察规则**: 产量持续增长→种植管理措施有效；某年产量大幅下降→需追溯气象灾害或病虫害影响；预估产量低于目标→需调整管理措施

---

### 3. 病虫害预警热力图

- **图表类型**: heatmap（热力图）
- **数据格式**:
```python
{
    "xAxis": ["A地块", "B地块", "C地块", "D地块", "E地块", "F地块", "G地块", "H地块"],
    "yAxis": ["蚜虫", "红蜘蛛", "白粉病", "锈病", "稻瘟病", "纹枯病", "草地贪夜蛾"],
    "data": [
        [2, 3, 1, 4, 2, 1, 3, 2],
        [1, 2, 3, 2, 1, 2, 1, 1],
        [3, 4, 2, 3, 2, 3, 4, 2],
        [2, 1, 1, 2, 3, 1, 2, 1],
        [1, 1, 2, 1, 1, 2, 1, 1],
        [2, 2, 3, 2, 2, 3, 2, 2],
        [4, 3, 2, 3, 4, 2, 3, 3]
    ],
    "colorScale": "Green-Yellow-Red",
    "legend": {"1": "低风险", "2": "中低风险", "3": "中高风险", "4": "高风险"}
}
```
- **简洁解读**: 以颜色深浅表示各地块不同病虫害的风险等级，红色区域需重点关注和防治
- **洞察规则**: 某地块多病虫害高风险→需立即启动综合防治；某病虫害多地块高风险→需全农场统一防治

### 4. 气象数据（组合图）

- **图表类型**: combo（簇状柱形图+折线图）
- **数据格式**:
```python
{
    "xAxis": ["3月", "4月", "5月", "6月", "7月", "8月", "9月", "10月"],
    "bars": [
        {"name": "降水量(mm)", "data": [35, 55, 85, 120, 180, 160, 90, 45]},
        {"name": "蒸发量(mm)", "data": [45, 75, 110, 140, 165, 150, 95, 50]}
    ],
    "lines": [
        {"name": "平均气温(°C)", "data": [12, 18, 24, 28, 32, 31, 26, 18]},
        {"name": "作物适宜温度(°C)", "data": [15, 20, 25, 28, 30, 30, 25, 18]}
    ]
}
```
- **简洁解读**: 追踪关键气象要素的变化，评估气象条件对作物生长的影响
- **洞察规则**: 降水量<蒸发量持续>14天→需启动灌溉；气温偏离适宜温度>5°C持续>3天→需评估对生长的影响

### 5. 土壤数据（仪表盘组）

- **图表类型**: gauge（仪表盘，多图并列）
- **数据格式**:
```python

        {"name": "土壤湿度", "value": 68, "min": 0, "max": 100, "unit": "%", "thresholds": {"green": 60, "yellow": 40, "red": 20}},

        {"name": "磷含量", "value": 28, "min": 0, "max": 50, "unit": "mg/kg", "thresholds": {"green": 20, "yellow": 10, "red": 0}}

```
- **简洁解读**: 实时监测土壤关键指标，确保土壤条件满足作物生长需求

## 深度洞察

1. **作物生长异常预警**：通过农事进度表发现苗期管理进度仅65%且落后于计划，结合气象数据发现4月降水偏少，可定位干旱影响作物生长，需启动灌溉。

2. **病虫害风险预测**：通过病虫害预警热力图发现A地块草地贪夜蛾风险等级为4级（高风险），结合历史数据该虫害在该地块往年同期高发，可提前部署防治措施。

3. **土壤改良决策支持**：通过土壤数据仪表盘发现B地块pH值为5.2（偏酸），结合该地块小麦生长状况为"一般"，可制定土壤改良方案。

4. **气象灾害应对**：通过气象数据组合图发现7月降水量达180mm且气温高达32°C，高温高湿易引发病害，需加强田间巡查和病害预防。

5. **产量预测模型优化**：结合历年产量趋势、当前土壤数据、气象数据和农事进度，建立产量预测模型，动态调整预估产量。

## 运营建议

1. **P0 - 建立病虫害预警响应机制**：对病虫害预警热力图中风险等级≥3级的地块，48小时内完成现场勘查并启动防治措施。优先级来源：病虫害预警热力图红色区域即触发。

2. **P0 - 优化灌溉决策**：结合土壤湿度仪表盘和气象数据，建立智能灌溉模型，土壤湿度<40%且未来3天无有效降水时自动触发灌溉建议。

3. **P1 - 推进土壤改良计划**：针对土壤pH值<5.5或有机质<2%的地块，制定土壤改良方案，目标年度内土壤健康度提升至良好水平。

4. **P1 - 完善农事任务管理**：对农事进度表中延期>7天的任务，自动升级预警并调整后续农事计划，确保关键农事节点按时完成。

5. **P2 - 建立气象灾害预警**：当气象数据显示极端天气（高温、干旱、暴雨）即将发生时，提前3天推送预警并建议应对措施。

6. **P2 - 推进精准施肥**：基于土壤养分数据和作物需肥规律，建立变量施肥模型，目标肥料利用率提升15%，降低施肥成本。

7. **P3 - 推进农业数字化**：部署更多土壤传感器、气象站和无人机巡检，实现农场数据全面采集和智能分析，支撑精准农业决策。
