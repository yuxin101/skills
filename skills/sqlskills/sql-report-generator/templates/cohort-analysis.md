# O04 留存分析报告模板

适用于：用户留存深度分析。

## 模板结构

```
┌─────────────────────────────────────────────────────┐
│  BANNER: 用户留存分析 + 周期                        │
├─────────────────────────────────────────────────────┤
│  KPI：次日留存 | 7日留存 | 30日留存 | 平均留存      │
├─────────────────────────────────────────────────────┤
│  Cohort 热力图（全宽）                              │
├────────────────────────────┬────────────────────────┤
│  留存曲线对比              │  分渠道留存            │
├────────────────────────────┴────────────────────────┤
│  留存洞察 & 改进建议                                │
└─────────────────────────────────────────────────────┘
```

## Cohort 热力图

- X轴：留存周期（日/周）
- Y轴：注册周/月
- 颜色深浅：留存率高低
