# H02 招聘报告模板

适用于：招聘进度汇报、招聘效果分析。

## 模板结构

```
┌─────────────────────────────────────────────────────┐
│  BANNER: 招聘报告 + 周期                            │
├─────────────────────────────────────────────────────┤
│  KPI：HC | 已入职 | 招聘中 | 到岗率                 │
├─────────────────────────────────────────────────────┤
│  招聘漏斗图                                         │
├────────────────────────────┬────────────────────────┤
│  部门招聘进度              │  渠道效果              │
├────────────────────────────┼────────────────────────┤
│  招聘瓶颈分析              │  招聘明细              │
└────────────────────────────┴────────────────────────┘
```

## 招聘漏斗

简历 → 面试 → Offer → 入职
