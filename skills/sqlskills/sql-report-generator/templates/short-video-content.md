## 核心指标

| 指标名称 | 英文缩写 | 定义 | 单位 | 目标参考 |
|---------|---------|------|------|---------|
| 作品总播放量 | Views | 本周期所有作品播放次数之和 | 万次 | 周环比≥10% |
| 平均完播率 | Completion Rate | 完整播放次数/总播放次数 | % | ≥25% |
| 粉丝净增量 | Net Followers | 新增粉丝-取关粉丝 | 人 | 周均≥500 |
| 互动率 | Engagement Rate | (点赞+评论+收藏+转发)/播放量 | % | ≥5% |
| 5秒完播率 | 5s Retention | 前5秒留存用户/曝光用户 | % | ≥50% |
| 粉丝活跃率 | Fan Activity | 周活跃粉丝/总粉丝 | % | ≥15% |
| 商单完成率 | Campaign Fulfillment | 商单ROI达成合同要求比例 | % | ≥85% |
| 千次播放收益 | CPM/EPM | 广告收入/播放量×1000 | 元 | ≥2元 |

## 图表详细定义

### 图表1：播放量趋势与爆款标注
- **标题**：近30天日均播放量趋势（含爆款标注）
- **图表类型**：折线+散点组合图
- **数据格式**：
  ```json
  {
    "x_axis": ["03-01", "03-02", "03-03", "...", "03-30"],
    "daily_views": [12.5, 15.8, 8.2, 22.3, "...", 35.6],
    "viral_marks": [
      {"date": "03-12", "views": 185.2, "title": "这5件事让我月入过万", "boost_factor": 5.2},
      {"date": "03-25", "views": 92.8, "title": "真心话大冒险挑战", "boost_factor": 2.8}
    ],
    "trend_line": {"slope": 0.8, "direction": "up"}
  }
  ```
- **解读**：折线为日常播放量，散点标注爆款事件，虚线为趋势拟合线。
- **洞察规则**：播放量周均值连续3周环比下降>15%触发"内容力衰减"预警；爆款boost_factor>3倍时说明算法推荐有显著加成，应分析该视频的结构要素。

### 图表2：粉丝画像变化趋势
- **标题**：粉丝年龄×性别分布变化（本期 vs 初期）
- **图表类型**：堆叠面积图
- **数据格式**：
  ```json
  {
    "age_groups": ["18-24", "25-34", "35-44", "45+"],
    "initial_dist": [35, 40, 18, 7],
    "current_dist": [32, 42, 19, 7],
    "growth_by_group": [+1200, +2800, +850, +150]
  }
  ```
- **解读**：观察粉丝结构是否朝预期方向演进，内容风格与目标人群是否匹配。
- **洞察规则**：25-34岁主力人群占比下降>5pp触发"受众老化"预警；18-24岁人群增长但互动率低说明"低质增粉"。

### 图表3：互动热词词云
- **标题**：评论区高频词词云
- **图表类型**：词云图
- **数据格式**：

- **解读**：正向情感词为主说明内容口碑好；"求教程/多少钱"说明有商业化潜力；"恰饭"高频出现说明商业化内容引起粉丝反感。
- **洞察规则**：负面情感词占比>20%触发"内容口碑预警"；"恰饭"词频周环比上升>50%说明商业化内容频率过高。

### 图表4：商业化内容效果对比
- **标题**：自然内容 vs 商业内容关键指标对比
- **图表类型**：分组柱状图
- **数据格式**：
  ```json
  {
    "metrics": ["完播率", "互动率", "涨粉量", "分享率"],
    "organic": [32.5, 6.8, 850, 2.3],
    "commercial": [18.2, 3.5, 320, 0.8]
  }
  ```
- **解读**：量化"恰饭伤粉"程度，为商业合作频率和形式提供数据支撑。
- **洞察规则**：商业内容完播率<自然内容50%时说明植入方式生硬，需优化软植入脚本；商业内容互动率<3%时说明内容与受众需求脱节。

### 图表5：内容标签效果雷达图
- **标题**：不同内容标签表现雷达
- **图表类型**：雷达图
- **数据格式**：
  ```json
  {
    "labels": ["播放量", "完播率", "互动率", "涨粉效率", "评论深度", "转发率"],
    "tags": {
      "#职场干货": [85, 90, 88, 92, 75, 70],
  ```

- **洞察规则**：各标签在"播放量"维度得分高但在"涨粉效率"维度得分低，说明内容引流能力有余但粉丝转化能力不足。

## 深度洞察

1. **完播率的黄金分割线**：5秒完播率决定了算法是否继续推流，而整体完播率决定了内容能否进入更大流量池。两者逻辑不同：5秒靠"钩子"（悬念/冲突/利益承诺），整体完播率靠"节奏控制"（信息密度+情绪起伏）。高5秒完播但低整体完播说明"开头惊艳但内容注水"。

2. **涨粉效率 vs 爆款思维**：一场爆款带来的粉丝往往质量参差（蹭热点的路人粉），而稳定内容带来的粉丝留存率和活跃率更高。评估时应区分"爆款涨粉"和"日常涨粉"，后者才是账号健康度的真实指标。

3. **商业化内容的"舒适区悖论"**：广告主倾向于投粉丝量高的大号，但大号为了维护粉丝体验又必须克制商业化频率。解决路径是"内容即广告，广告即内容"——让商业内容本身成为爆款概率最高的选题类型，而非妥协于品牌方的话术要求。

4. **评论区是一座金矿**：评论区能提供的内容洞察远超互动率本身：用户提问揭示内容盲区，用户吐槽暴露内容槽点，用户二创标志内容破圈潜力。定期梳理评论区是最低成本的用户研究方法。

## 运营建议

| 优先级 | 建议内容 | 预期效果 |
|-------|---------|---------|
| P0 | 针对5秒完播率<50%的视频，建立"3秒钩子"素材库，要求每个视频前3秒必须包含：冲突画面/悬念提问/利益承诺之一 | 5秒完播率提升至55%+ |
| P0 | 对评论区高频"求教程/多少钱"类评论，优先制作配套教程类内容，满足粉丝需求的同时自然植入商业信息，降低"恰饭"感知 | 商业内容互动率提升50% |
| P1 | 建立内容标签效果矩阵，每周评估各标签的"流量-转化"综合得分，动态调整内容配比，将资源集中于高涨粉效率标签 | 粉丝周均增长提升30% |
| P1 | 设计"粉丝专属内容"机制（粉丝勋章才能看的番外/粉丝群优先看），提升粉丝活跃率和归属感，目标将粉丝活跃率从15%提升至25% | 粉丝月留存率提升10pp |
| P2 | 针对"商业内容完播率<自然内容50%"的问题，推行"甲方预审脚本+创作者保底"合作模式，给创作者留出20%的创意自由度，避免硬植入 | 商业内容完播率差距缩小至30%以内 |
