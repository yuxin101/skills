# 基金行业 - 投资组合监控看板模板

适用于：基金公司投资组合监控、基金经理业绩跟踪、风险控制管理
对标：蚂蚁财富、天天基金、蛋卷基金投资组合分析

## 模板布局

```
┌─────────────────────────────────────────────────────────────────┐
│  BANNER: 投资组合监控中心 + 组合选择器 + 时间周期              │
├─────────────────────────────────────────────────────────────────┤
│  KPI 卡片行：总资产净值 | 日收益率 | 夏普比率 | 最大回撤       │
├────────────────────────────┬────────────────────────────────────┤
│  资产净值趋势(面积图)      │  行业配置分布(饼图)              │
├────────────────────────────┼────────────────────────────────────┤
│  收益率分布散点图          │  风险指标详情(表格)              │
├────────────────────────────┼────────────────────────────────────┤
│  持仓集中度(柱形图)        │  业绩归因分析(瀑布图)            │
├────────────────────────────┴────────────────────────────────────┤
│  深度洞察 + 运营建议                                           │
└─────────────────────────────────────────────────────────────────┘
```

## 核心指标

- **总资产净值** — 组合总资产减去负债后的净值 — 目标值：稳定增长，年化≥8%
- **日收益率** — 当日净值增长率 — 目标值：超越基准指数
- **夏普比率** — 风险调整后收益，每承担1单位风险获得的超额收益 — 目标值：≥1.5
- **最大回撤** — 历史峰值到谷值的最大跌幅 — 目标值：≤15%
- **持仓集中度** — 前10大重仓股占比 — 目标值：40-60%
- **行业配置偏离度** — 相对基准的行业配置差异 — 目标值：±10%以内
- **换手率** — 一定时期内组合持仓调整频率 — 目标值：根据策略，主动型200-400%
- **Beta值** — 组合相对市场的敏感度 — 目标值：0.8-1.2

## 图表详细定义

### 1. 资产净值趋势
- **图表类型**: area
- **数据格式**:
```python
{
    "dates": ["2024-01-01", "2024-01-02", "2024-01-03", "...", "2024-03-31"],
    "nav": [1.0000, 1.0052, 1.0123, "...", 1.0892],  # 单位净值
    "total_value": [100000000, 100520000, 101230000, "...", 108920000],  # 总资产
    "benchmark_nav": [1.0000, 1.0031, 1.0088, "...", 1.0621]  # 基准净值
}
```
- **简洁解读**: 展示组合净值和总资产的历史走势，直观呈现收益情况
- **洞察规则**:
  - 净值曲线跑赢基准曲线为优秀
  - 连续3天下跌超3%需关注
  - 净值创历史新高应标记

### 2. 行业配置分布
- **图表类型**: donut
- **数据格式**:
```python
{
    "sectors": ["科技", "医药", "消费", "金融", "制造业", "能源", "其他"],
    "portfolio_weight": [28, 18, 15, 12, 10, 8, 9],  # 组合占比%
    "benchmark_weight": [25, 15, 18, 15, 12, 7, 8],  # 基准占比%
    "deviation": [3, 3, -3, -3, -2, 1, 1]  # 偏离度
}
```
- **简洁解读**: 对比组合与基准的行业配置，识别超配/低配领域
- **洞察规则**:
  - 单行业偏离>10%需说明理由
  - 超配行业持续跑输需调整
  - 低配行业持续跑赢需反思

### 3. 收益率分布散点图
- **图表类型**: scatter
- **数据格式**:
```python
{
    "positions": [
        {"name": "贵州茅台", "return": 18.5, "risk": 22.3, "weight": 8.2, "sector": "消费"},
        {"name": "宁德时代", "return": 25.3, "risk": 35.1, "weight": 6.8, "sector": "科技"},
        {"name": "招商银行", "return": 12.1, "risk": 18.5, "weight": 5.5, "sector": "金融"},
        {"name": "药明康德", "return": -8.2, "risk": 28.7, "weight": 4.2, "sector": "医药"}
    ]
}
```
- **简洁解读**: 每个持仓的收益-风险分布，气泡大小代表持仓权重
- **洞察规则**:
  - 第一象限(高收益低风险)为优质持仓
  - 第四象限(低收益高风险)需减仓
  - 权重过大且表现差的持仓需调整

### 4. 风险指标详情
- **图表类型**: table
- **数据格式**:
```python
{
    "columns": ["风险指标", "当前值", "目标值", "行业平均", "状态"],
    "data": [
        ["夏普比率", 1.85, "≥1.5", 1.42, "优秀"],
        ["最大回撤", "12.3%", "≤15%", "14.8%", "良好"],
        ["波动率", "18.5%", "≤20%", "19.2%", "良好"],
        ["Beta值", 1.05, "0.8-1.2", 1.00, "正常"],
        ["VaR(95%)", "-2.1%", "≤3%", "-2.5%", "优秀"],
        ["信息比率", 0.72, "≥0.5", 0.58, "良好"]
    ]
}
```
- **简洁解读**: 多维度风险指标监控，与目标和行业均值对比
- **洞察规则**:
  - 任一指标超目标值需风控报告
  - 连续3月夏普比率<1需策略检讨
  - 最大回撤接近目标值需降仓

### 5. 持仓集中度
- **图表类型**: clustered_column
- **数据格式**:
```python
{
    "concentration_levels": ["Top1", "Top3", "Top5", "Top10", "Top20"],
    "current_portfolio": [8.2, 21.5, 35.8, 52.3, 78.6],  # 当前占比%
    "peer_average": [7.5, 20.2, 33.5, 48.7, 75.2],  # 同业平均
    "benchmark": [6.8, 18.5, 30.2, 45.1, 72.3]  # 基准
}
```
- **简洁解读**: 展示组合持仓集中度，与同业和基准对比
- **洞察规则**:
  - Top10集中度>70%为过度集中
  - 显著高于同业需说明策略
  - 快速上升需警惕风险积聚

### 6. 业绩归因分析
- **图表类型**: waterfall
- **数据格式**:
```python
{
    "factors": ["基准收益", "行业配置", "个股选择", "交互效应", "交易成本", "总超额收益"],
    "contribution": [6.21, 0.85, 1.52, -0.28, -0.12, 1.97],  # 贡献度%
    "cumulative": [6.21, 7.06, 8.58, 8.30, 8.18, 8.18]  # 累计收益%
}
```
- **简洁解读**: 分解超额收益来源，评估投资决策有效性
- **洞察规则**:
  - 个股选择贡献>行业配置为优选型
  - 交互效应为负需检讨组合构建
  - 交易成本占比过高需优化执行

## 深度洞察

1. **风格漂移预警**: 当行业配置偏离度持续扩大且超额收益为负时，说明策略执行偏离原定风格，需及时纠偏或调整策略说明
2. **风险收益错配**: 部分持仓风险暴露过大但贡献有限，如第四象限持仓累计权重>10%，建议逐步减仓优化组合效率
3. **集中度风险积聚**: Top10持仓集中度快速上升且相关性增强时，组合抗风险能力下降，需警惕市场风格切换带来的冲击
4. **业绩持续性分析**: 过去3个月超额收益主要来自行业配置，个股选择贡献下降，需加强个股研究能力
5. **流动性风险提示**: 部分中小市值持仓换手率低、成交量小，极端情况下可能面临流动性折价，建议提前评估调整

## 运营建议

1. **动态仓位管理** — 优先级：高
   根据市场波动率和最大回撤阈值，建立自动化的仓位调整机制，回撤接近目标值时逐步降仓

2. **持仓结构优化** — 优先级：高
   清理第四象限(低收益高风险)持仓，将资金配置至第一象限或估值合理的优质标的，提升组合效率

3. **行业偏离度控制** — 优先级：中
   对偏离基准>10%的行业配置，要求投资经理提供详细说明，并设置止损线

4. **交易成本优化** — 优先级：中
   采用算法交易降低冲击成本，减少不必要的频繁交易，将年化交易成本控制在50bp以内

5. **风险预算分配** — 优先级：中
   根据夏普比率和信息比率，合理分配各投资经理的风险预算，建立风险调整后的绩效考核体系

6. **流动性管理** — 优先级：低
   对持仓股票的日均成交量进行监控，限制流动性差的个股仓位，确保极端情况下可顺利减仓

7. **归因分析常态化** — 优先级：低
   每月进行详细的业绩归因分析，识别投资决策的优劣势，持续改进投资框架和决策流程
