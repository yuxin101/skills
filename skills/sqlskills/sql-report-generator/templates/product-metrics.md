# PR01 产品指标报告模板

适用于：产品运营分析、产品健康度监控。

## 模板结构

```
┌─────────────────────────────────────────────────────┐
│  BANNER: 产品指标报告 + 周期                        │
├─────────────────────────────────────────────────────┤
│  KPI：DAU | MAU | 使用时长 | 留存率                 │
├─────────────────────────────────────────────────────┤
│  核心指标趋势                                       │
├────────────────────────────┬────────────────────────┤
│  功能使用分布              │  用户路径              │
├────────────────────────────┼────────────────────────┤
│  漏斗转化                  │  优化建议              │
└────────────────────────────┴────────────────────────┘
```

## 核心指标

- DAU/MAU、使用时长、启动次数
- 功能渗透率、功能留存
- 核心路径转化率
