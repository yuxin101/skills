# 银行行业 - 网点运营看板模板

适用于：银行网点日常运营监控、柜员效能管理、客户服务优化
对标：招行网点运营大屏、建行智慧网点系统、工行网点效能分析平台

## 模板布局

```
┌─────────────────────────────────────────────────────────────────┐
│  BANNER: 网点运营监控中心 + 日期选择器 + 网点筛选器            │
├─────────────────────────────────────────────────────────────────┤
│  KPI 卡片行：存款余额 | 贷款余额 | 日均交易量 | 客户等候时长   │
├────────────────────────────┬────────────────────────────────────┤
│  网点交易量排行榜(条形图)  │  存贷款趋势对比(组合图)          │
├────────────────────────────┼────────────────────────────────────┤
│  客户等候时间分布(热力图)  │  柜员效能排名(表格)              │
├────────────────────────────┼────────────────────────────────────┤
│  业务类型占比(饼图)        │  客户满意度趋势(折线图)          │
├────────────────────────────┴────────────────────────────────────┤
│  深度洞察 + 运营建议                                           │
└─────────────────────────────────────────────────────────────────┘
```

## 核心指标

- **存款余额** — 网点存款总额，反映吸储能力 — 目标值：环比增长≥3%
- **贷款余额** — 网点贷款总额，反映放贷规模 — 目标值：环比增长≥2%
- **日均交易量** — 每日平均业务办理笔数，反映业务繁忙度 — 目标值：≥200笔/日
- **客户平均等候时长** — 从取号到开始办理的平均等待时间 — 目标值：≤15分钟
- **柜员效能** — 单个柜员日均办理业务量 — 目标值：≥40笔/日
- **客户满意度** — 服务评价满意率 — 目标值：≥95%
- **业务差错率** — 业务办理差错占比 — 目标值：≤0.1%
- **智能设备使用率** — 自助设备业务办理占比 — 目标值：≥60%

## 图表详细定义

### 1. 网点交易量排行榜
- **图表类型**: horizontal_bar
- **数据格式**: 
```python
{
    "branches": ["城东支行", "市中心支行", "高新区支行", "开发区支行", "西区支行"],
    "transaction_count": [1850, 1720, 1680, 1520, 1380],
    "target": [1500, 1500, 1500, 1500, 1500]
}
```
- **简洁解读**: 展示各网点交易量排名，快速识别高低产网点
- **洞察规则**: 
  - 超目标20%以上为优秀
  - 低于目标15%为需关注
  - 连续3天排名末位需预警

### 2. 存贷款趋势对比
- **图表类型**: combo (柱形+折线)
- **数据格式**:
```python
{
    "dates": ["2024-01", "2024-02", "2024-03", "2024-04", "2024-05", "2024-06"],
    "deposit_balance": [12500, 12800, 13100, 12950, 13400, 13800],  # 亿元
    "loan_balance": [9800, 10100, 10350, 10500, 10800, 11100],
    "deposit_growth_rate": [2.4, 2.3, 1.2, 3.5, 3.0],  # 环比%
    "loan_growth_rate": [3.1, 2.5, 1.4, 2.9, 2.8]
}
```
- **简洁解读**: 对比存贷款余额及增长趋势，评估资金规模变化
- **洞察规则**:
  - 存贷比在70-85%为健康区间
  - 存款增速持续高于贷款增速需警惕资金闲置
  - 单月增速超5%需分析大额交易

## 深度洞察

1. **高峰拥堵预警**: 当检测到某时段等候时长持续>20分钟，系统自动推荐增开窗口或引导至自助设备，可减少客户等待时间30-40%
2. **柜员效能差异**: 效能排名末位柜员的办理时长比首位慢50%以上，建议实施"结对帮带"培训计划，预计3个月内可提升整体效能15%
3. **业务分流机会**: 存取款等简单业务占比超40%，通过智能引导可将其分流至自助设备，释放柜面资源服务高价值业务
4. **存贷款结构优化**: 存贷比持续低于75%说明资金运用不充分，建议加强贷款营销或向上级行申请资金调配
5. **周末服务缺失**: 周五等待时长显著高于其他工作日，建议评估周末营业或延长工作日营业时间的可行性

## 运营建议

1. **实施弹性排班机制** — 优先级：高
   根据历史客流数据，在高峰时段(周五上午、月初月末)增开窗口，低峰时段减少窗口，平衡柜员工作负荷

2. **推广智能分流系统** — 优先级：高
   在取号机增加业务类型识别，自动引导简单业务客户至自助设备，预计可减少柜面压力25%

3. **建立柜员效能激励机制** — 优先级：中
   对综合评分≥90的柜员给予绩效奖励，对评分<80的柜员安排专项培训，形成良性竞争氛围

4. **优化网点空间布局** — 优先级：中
   增设等候区座椅、饮水机、充电设施，提供免费WiFi和金融知识宣传屏，提升等待体验

5. **加强高价值客户服务** — 优先级：中
   为VIP客户设置专属窗口或预约服务，减少等待时间，提升客户粘性和综合贡献度

6. **开展业务技能竞赛** — 优先级：低
   每季度组织业务技能比武，表彰业务办理快、差错率低、满意度高的柜员，树立标杆

7. **完善客户反馈机制** — 优先级：低
   在业务办理后推送满意度评价短信，对差评客户24小时内回访，将问题消灭在萌芽状态
