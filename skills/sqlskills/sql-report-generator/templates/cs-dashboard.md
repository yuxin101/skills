# C01 客服看板模板

适用于：客服团队管理、服务质量监控。

## 模板结构

```
┌─────────────────────────────────────────────────────┐
│  BANNER: 客服看板 + 周期                            │
├─────────────────────────────────────────────────────┤
│  KPI：工单量 | 响应时间 | 解决率 | 满意度           │
├─────────────────────────────────────────────────────┤
│  工单趋势图                                         │
├────────────────────────────┬────────────────────────┤
│  问题类型分布              │  处理时效              │
├────────────────────────────┼────────────────────────┤
│  客服人员绩效              │  改进建议              │
└────────────────────────────┴────────────────────────┘
```

## 核心指标

- 工单量、响应时间、解决时间
- 首次解决率、满意度评分
- SLA 达成率
