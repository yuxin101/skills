# 电商行业 - 用户画像&分群看板模板

适用于：电商用户精细化运营、精准营销、用户生命周期管理
对标：阿里用户画像平台、京东用户数据中心、字节巨量引擎

## 模板布局

```
┌─────────────────────────────────────────────────────────────────┐
│  BANNER: 用户画像与分群中心 + 时间范围 + 用户群体筛选          │
├─────────────────────────────────────────────────────────────────┤
│  KPI 卡片行：活跃用户数 | 新增用户 | ARPU值 | 用户LTV         │
├────────────────────────────┬────────────────────────────────────┤
│  用户RFM分群(散点图)       │  用户画像雷达图                    │
├────────────────────────────┼────────────────────────────────────┤
│  用户生命周期分布(漏斗图)  │  渠道来源分析(饼图)                │
├────────────────────────────┼────────────────────────────────────┤
│  用户行为路径(桑基图)      │  分群价值贡献(表格)                │
├────────────────────────────┴────────────────────────────────────┤
│  深度洞察 + 运营建议                                           │
└─────────────────────────────────────────────────────────────────┘
```

## 核心指标

- **活跃用户数(DAU/MAU)** — 日/月活跃用户数量 — 目标值：环比增长≥5%
- **新增用户数** — 新注册用户数量 — 目标值：根据获客目标设定
- **ARPU值** — 人均收入贡献(Average Revenue Per User) — 目标值：持续提升
- **用户LTV** — 用户生命周期总价值(Lifetime Value) — 目标值：>获客成本3倍
- **复购率** — 重复购买用户占比 — 目标值：≥35%
- **用户留存率** — 次日/7日/30日留存率 — 目标值：次日≥40%,7日≥20%,30日≥10%
- **用户分群占比** — 各用户分群(高价值/普通/沉睡等)占比 — 目标值：高价值用户占比提升
- **用户健康度评分** — 综合评估用户活跃度和价值的指标 — 目标值：平均分≥70

## 图表详细定义

### 1. 用户RFM分群
- **图表类型**: scatter (3D或气泡图)
- **数据格式**:
```python
{
    "segments": [
        {"segment": "重要价值客户", "r_score": 5, "f_score": 5, "m_score": 5, "user_count": 8500, "color": "#FF6B6B"},
        {"segment": "重要发展客户", "r_score": 4, "f_score": 3, "m_score": 4, "user_count": 12000, "color": "#4ECDC4"},
        {"segment": "重要保持客户", "r_score": 3, "f_score": 4, "m_score": 4, "user_count": 9800, "color": "#45B7D1"},
        {"segment": "一般价值客户", "r_score": 3, "f_score": 3, "m_score": 3, "user_count": 15000, "color": "#96CEB4"},
        {"segment": "低价值客户", "r_score": 2, "f_score": 2, "m_score": 2, "user_count": 28000, "color": "#FFEAA7"},
        {"segment": "流失预警客户", "r_score": 1, "f_score": 2, "m_score": 2, "user_count": 6500, "color": "#DDA0DD"}
    ]
}
```
- **简洁解读**: 基于最近购买(R)、购买频率(F)、购买金额(M)三维分群
- **洞察规则**:
  - 重要价值客户占比<5%需提升
  - 流失预警客户>15%需激活
  - 各分群比例失衡需调整运营策略

### 2. 用户画像雷达图
- **图表类型**: radar
- **数据格式**:
```python

    "dimensions": ["购买力", "活跃度", "忠诚度", "互动性", "传播力", "品质偏好"],
    "high_value_users": [85, 92, 88, 75, 68, 82],
    "normal_users": [55, 65, 58, 52, 45, 60],
    "potential_users": [48, 72, 42, 68, 55, 50]
```

- **简洁解读**: 多维度对比不同用户群体特征,识别差异化需求
- **洞察规则**:
  - 高价值用户特征偏离需分析
  - 潜力用户短板需针对性提升
  - 各群体差异大需分群运营

### 3. 用户生命周期分布
- **图表类型**: funnel
- **数据格式**:
```python

    "stages": ["新增用户", "活跃用户", "首购用户", "复购用户", "忠诚用户", "核心用户"],
    "user_count": [50000, 35000, 22000, 12000, 6500, 2800],
    "conversion_rate": [100, 70, 44, 24, 13, 5.6],
    "stage_conversion": [100, 70, 62.9, 54.5, 54.2, 43.1]  # 阶段转化率

```
- **简洁解读**: 用户从新增到核心的转化漏斗,识别流失环节
- **洞察规则**:
  - 活跃→首购转化<60%需优化引导
  - 复购用户占比<25%需提升
  - 核心用户流失需紧急挽留

- **数据格式**:

## 深度洞察

1. **帕累托法则验证**: 高价值用户占比仅5.6%,却贡献了42%的GMV,说明用户价值高度集中,需重点维护VIP用户群体
2. **生命周期瓶颈**: 首购到复购的转化率仅54.5%,是最大流失环节,需加强首购后的营销触达和会员权益引导
3. **渠道效能差异**: 老用户推荐获客成本仅12元,远低于付费广告的85元,需强化推荐激励计划(如优惠券、积分)
4. **加购未购流失**: 加购到下单转化率仅62.5%,约3800个订单流失,通过购物车召回短信可挽回20-30%
5. **潜力用户特征**: 潜力用户活跃度高(72分)但购买力低(48分),说明有需求但受价格制约,适合推送性价比商品

## 运营建议

1. **建立分层运营体系** — 优先级：高
   针对不同用户分群制定差异化运营策略,高价值用户享受专属客服和优先发货,潜力用户推送优惠券激活

2. **强化会员权益体系** — 优先级：高
   设计多级会员体系(普通/VIP/SVIP),提供差异化权益(折扣、包邮、专属活动),提升复购率和忠诚度

3. **优化首购转化路径** — 优先级：高
   新用户首次访问提供新人礼包(优惠券+限时折扣),引导完成首购,首购用户赠送二次消费券提升复购

4. **建立流失预警机制** — 优先级：中
   基于用户行为数据(登录频次、购买间隔等)建立流失预测模型,对高风险用户主动触达挽回

5. **实施精准营销触达** — 优先级：中
   根据用户画像(购买偏好、价格敏感度、活跃时段)进行个性化推荐和精准推送,提升转化率

6. **激励用户推荐传播** — 优先级：中
   建立推荐奖励机制,邀请新用户注册双方得优惠券,降低获客成本同时提升用户粘性

7. **持续优化用户画像** — 优先级：低
   定期(月度)更新用户分群和画像,追踪用户生命周期变化,动态调整运营策略
