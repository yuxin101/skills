# P01 项目状态报告模板

适用于：项目进度汇报、项目管理。

## 模板结构

```
┌─────────────────────────────────────────────────────┐
│  BANNER: 项目状态报告 + 项目名称                    │
├─────────────────────────────────────────────────────┤
│  项目概况：进度 | 状态 | 风险                       │
├─────────────────────────────────────────────────────┤
│  里程碑进度图                                       │
├────────────────────────────┬────────────────────────┤
│  任务完成情况              │  风险问题              │
├────────────────────────────┼────────────────────────┤
│  资源使用                  │  下周计划              │
└────────────────────────────┴────────────────────────┘
```

## 项目状态

- 🟢 正常：按计划进行
- 🟡 预警：有风险需关注
- 🔴 延期：已延期需处理
