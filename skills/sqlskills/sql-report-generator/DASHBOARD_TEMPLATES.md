# 生产级行业标准看板模板库

> 对标：帆软 FineBI 216+ 模板 · 观远数据 · 网易有数 · 阿里 DataV · 华为云 DLV · 京东 EasyBI

## 📊 项目概述

本库包含 **19 大行业、40+ 个标准看板模板**，每个看板均对标头部大厂生产场景，包含：
- ✅ 完整的图表组合（使用已实现的 60 种图表）
- ✅ **每图简洁解读**（description 字段，一句话说明图表价值）
- ✅ **深度洞察**（insights 字段，多维度数据分析）
- ✅ **运营建议**（recommendations 字段，5-7 条可执行建议）
- ✅ 工具提示（Tooltip）交互支持

## 🏢 行业分类（19大行业）

# 🔧 交互功能：工具提示（Tooltip）

## 什么是工具提示？

**标准官方名称**
- 中文官方：工具提示
- 英文官方：Tooltip
- 业务/开发口语：悬停提示、鼠标悬浮提示、数据点详情弹窗

## 功能说明

它到底是什么？
就是你**鼠标悬停（Hover）**在图表的某个数据点、柱形、折线点、扇形、气泡上时，自动弹出的小浮层，展示：
- **维度名称**（如：2025-10、华东区、商品 A）
- **指标数值**（如：销售额 120 万）
- **同比 / 环比 / 占比**等附加信息

**完全属于 Power BI 原生自带交互功能**，不需要写代码，开箱即用。

## 生产看板里会看到的 3 种形态

### 1️⃣ 默认工具提示（自动生成）
系统根据你拖入的字段自动生成，最简单最常用。

**示例**：
```
商品A
销售额: ¥1,200,000
同比: +15%
```

### 2️⃣ 自定义工具提示（大厂标配）
可以自己做一个小报表页作为 tooltip，悬浮时显示：
- 多个图表
- 多个指标
- 表格明细
- 条件格式颜色

**生产环境几乎 100% 用这种！**

### 3️⃣ 钻取 / 突出显示 / 筛选（配套交互）
虽然不是 tooltip，但经常一起出现：
- **突出显示**：点一个柱子，其他图表变暗只高亮相关
- **钻取（Drillthrough）**：右键 → 钻取到详情页
- **筛选**：点击数据点，整个页面联动筛选

## 生产场景标准描述

> 图表支持 **鼠标悬停工具提示（Tooltip）** 交互，光标定位至数据点时自动弹出维度、指标及明细信息，支持自定义多指标、多图表 tooltip 展示，提升数据探查效率。

## 工具提示配置

### 默认配置
```python
{
    "enabled": True,
    "show_dimensions": True,
    "show_metrics": True,
    "show_comparison": True,
    "show_percentage": True,
    "show_delay": 200,  # 毫秒
    "hide_delay": 500,  # 毫秒
    "follow_mouse": True,
    "enable_highlight": True,
    "enable_filter": True
}
```

### 自定义配置（大厂生产环境）
```python
{
    "enabled": True,
    "tooltip_type": "custom",
    "title": "销售数据详情",
    "metrics": [
        {"field": "sales", "name": "销售额", "format": "currency"},
        {"field": "yoy", "name": "同比", "format": "percent"},
        {"field": "mom", "name": "环比", "format": "percent"},
        {"field": "rank", "name": "排名", "format": "number"}
    ],
    "dimensions": ["月份", "地区", "产品"],
    "table_data": {
        "columns": ["商品", "销售额"],
        "rows": [["商品A", "¥100万"], ["商品B", "¥80万"]]
    }
}
```

## 文件位置

- **工具提示模块**: `scripts/dashboard_tooltips.py` (17KB)
- **配置模板**: 内置 8 种图表的工具提示配置

## 快速使用

```python
from dashboard_tooltips import (
    TooltipGenerator,
    create_default_tooltip_config,
    create_custom_tooltip_config

# 使用默认配置

        {"field": "yoy", "name": "同比", "format": "percent", "precision": 1},
        {"field": "mom", "name": "环比", "format": "percent", "precision": 1}
    ],
    dimensions=["月份", "地区"]
)

# 生成 HTML
html = generator.generate_tooltip_html(data)
```
