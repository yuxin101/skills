# 图表规划指南

根据报告类型和数据特征，确定最佳图表组合。

---

## 报告类型 vs 图表组合

### 业务周报/月报（默认模板）
```
┌─────────────────────────────────────────┐
│  KPI 卡片行（4 个核心指标）               │
├─────────────────────────────────────────┤
│  趋势图（全宽）- 时间序列 + 洞察         │
├────────────────────┬────────────────────┤
│  分类对比图        │  维度拆解图         │
│  （横向条形图）    │  （堆叠条形图）      │
├────────────────────┼────────────────────┤
│  漏斗/流程图       │  明细表格            │
│  （转化分析）      │  （带条件格式）      │
└────────────────────┴────────────────────┘
```

### 转化漏斗分析报告
```
┌─────────────────────────────────────────┐
│  KPI 卡片行（转化率、流失率、平均时长）   │
├─────────────────────────────────────────┤
│  主漏斗图（全宽）- 各阶段转化率           │
├────────────────────┬────────────────────┤
│  分渠道漏斗对比    │  流失原因分析        │
│  （分组漏斗）      │  （横向条形图）      │
├────────────────────┴────────────────────┤
│  优化建议（洞察文字）                     │
└─────────────────────────────────────────┘
```

### 销售业绩看板
```
┌─────────────────────────────────────────┐
│  KPI 卡片行（销售额、订单量、客单价、目标达成率）│
├─────────────────────────────────────────┤
│  趋势图 + 目标线（全宽）                  │
├────────────────────┬────────────────────┤
│  Top 销售员/产品   │  区域分布            │
│  （横向条形图）    │  （地图/热力图）      │
├────────────────────┼────────────────────┤
│  品类占比          │  增长排名            │
│  （饼图/环形图）   │  （双向条形图）       │
└────────────────────┴────────────────────┘
```

## 图表选型决策矩阵

| 数据维度 | 1 个指标 | 2-3 个指标 | 4+ 个指标 |
|---------|---------|-----------|----------|
| 时间序列 | 折线图 | 多系列折线 | 面积堆叠 |
| 分类对比 | 横向条形图 | 分组条形图 | 热力图 |
| 占比构成 | 饼图（<5 类） | 堆叠条形图 | 矩形树图 |
| 相关性 | 散点图 | 气泡图 | 相关矩阵热力图 |
| 分布 | 直方图 | 箱线图 | 小提琴图 |
| 流程 | 漏斗图 | 分组漏斗 | 桑基图 |

## 报告布局原则

### 视觉层级
1. **顶部**：最重要的 KPI（用户第一眼看到）
2. **中上**：趋势图（回答"整体情况如何"）
3. **中部**：对比图（回答"哪里好/哪里差"）
4. **下部**：明细表格（回答"具体数据是什么"）
5. **底部**：洞察建议（回答"接下来做什么"）

### 信息密度
- **高管版**：3-4 个 KPI + 1 张趋势图 + 3 条洞察
- **运营版**：完整布局 + 明细表格
- **客户版**：精选 KPI + 核心结论 + 品牌元素

### 颜色使用
- 同类数据用同色
- 高亮重点数据用对比色
- 正向指标用绿色，负向用红色（或按品牌色）
- 最多使用 6 种颜色（色盲友好）

## 常见场景速查

| 业务问题 | 推荐图表组合 |
|---------|-------------|
| 销售趋势如何？ | 折线图 + 环比条形图 |
| 哪个品类表现最好？ | 横向条形图（Top 排名） |
| 渠道贡献占比？ | 堆叠条形图 + 饼图 |
| 转化率怎么样？ | 漏斗图 + 分渠道漏斗 |
| 用户留存情况？ | Cohort 热力图 |
| 各区域对比？ | 地图 + 横向条形图 |
| 目标达成情况？ | 折线图 + 目标线 + 进度条 |
