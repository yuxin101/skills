## 模板选择决策

```
用户需求是什么？
│
├── 定期汇报？
│   ├── 每周 → weekly-report
│   ├── 每月 → monthly-report
│   ├── 每季 → quarterly-report
│   └── 每年 → annual-report
│
├── 财务数据？
│   ├── 利润分析 → income-statement
│   ├── 资产负债 → balance-sheet
│   ├── 现金流 → cash-flow
│   └── 预算对比 → budget-analysis
│
├── 运营数据？
│   ├── 销售相关 → sales-dashboard
│   ├── 用户增长 → user-growth
│   ├── 转化漏斗 → conversion-funnel
│   └── 留存分析 → cohort-analysis
│
├── 给老板看？
│   └── 一页纸总结 → executive-summary
│
├── 人力相关？
│   ├── 人员概览 → hr-overview
│   ├── 招聘进度 → recruitment-report
│   └── 绩效考核 → performance-review
│
└── 其他专项？
    ├── 项目 → project-status / sprint-report
    ├── 营销 → campaign-report / ad-performance
    ├── 客服 → cs-dashboard
    ├── 产品 → product-metrics
    └── 技术 → system-health / incident-report
```
