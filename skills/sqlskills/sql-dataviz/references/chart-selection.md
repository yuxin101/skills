# 图表选型决策指南

## 核心问题：你想表达什么？

拿到数据后，先问自己这个问题，再选图表。

---

## 决策树

```
数据是什么关系？
│
├── 随时间变化？
│   ├── 连续时间序列 → 折线图
│   ├── 离散时间点比较 → 分组条形图
│   └── 累计/占比随时间变化 → 堆叠面积图
│
├── 分类之间比较？
│   ├── 类别 ≤ 10 → 纵向条形图
│   ├── 类别 > 10 或名称较长 → 横向条形图
│   ├── 多组对比 → 分组条形图
│   └── 排名展示 → 横向条形图（降序）
│
├── 整体中各部分占比？
│   ├── 类别 ≤ 5 → 堆叠条形图（推荐）或饼图
│   ├── 类别 > 5 → 矩形树图（Treemap）
│   └── 占比随时间变化 → 堆叠面积图（百分比）
│
├── 数值分布？
│   ├── 单变量分布 → 直方图
│   ├── 多组分布对比 → 箱线图 / 小提琴图
│   └── 密度估计 → KDE 图
│
├── 两变量关系？
│   ├── 相关性探索 → 散点图
│   ├── 带趋势线 → 回归散点图
│   └── 多变量相关矩阵 → 热力图
│
├── 地理分布？
│   └── 地图（Choropleth / 气泡地图）
│
├── 流向/转化？
│   ├── 用户转化漏斗 → 漏斗图
│   └── 流量流向 → 桑基图（Sankey）
│
└── 单一关键指标？
    └── 大数字 KPI 卡片（不需要图表）
```

## 常见场景速查

| 业务场景 | 推荐图表 | 示例 |
|---------|---------|------|
| 日/周/月销售趋势 | 折线图 | 近 30 天 GMV |
| 各品类销售额对比 | 横向条形图 | Top 10 品类 |
| 用户地区分布 | 横向条形图 / 地图 | 各省用户数 |
| 订单金额分布 | 直方图 | 0-100/100-500/500+ |
| 各渠道转化率对比 | 分组条形图 | PC vs 移动端 |
| 用户留存率 | 热力图（Cohort） | 按注册周分组 |
| 指标相关性 | 热力图 | 价格/销量/评分 |
| 漏斗转化 | 漏斗图 | 曝光→点击→购买 |
| 月度环比增长 | 折线图 + 柱状图组合 | 销售额 + 增长率 |
| 市场份额 | 堆叠条形图 | 各品牌占比 |

## 反模式（常见错误）

### ❌ 饼图滥用
```
错误：用饼图展示 10 个品类的销售占比
原因：人眼无法准确比较扇形大小
正确：横向条形图，按值排序
```

### ❌ 双 Y 轴滥用
```
错误：左轴销售额（万元），右轴增长率（%），两条线看起来相关
原因：双 Y 轴可以通过调整比例制造任意"相关性"
正确：分两张图，或用散点图展示真实相关性
```

### ❌ 截断 Y 轴
```
错误：条形图 Y 轴从 95 开始，让微小差异看起来很大
原因：视觉误导
正确：Y 轴从 0 开始；如果确实需要放大差异，用折线图并明确标注
```

### ❌ 3D 图表
```
错误：3D 饼图、3D 条形图
原因：透视变形导致读值不准
正确：永远用 2D
```

### ❌ 彩虹配色
```
错误：每个柱子用不同颜色，没有语义
原因：颜色应该传达信息，不是装饰
正确：同类数据用同色；用颜色高亮重点数据
```
