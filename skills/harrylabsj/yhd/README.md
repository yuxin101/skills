# YHD Skill v2.0

1号店购物助手 Skill，支持浏览器自动化完成搜索、每日秒杀、生鲜选购、加购物车等操作，支付环节留给用户控制。

## 功能特性

### v2.0 - 新增浏览器自动化

- 🔍 **商品搜索**：关键词搜索，支持筛选
- ⚡ **每日秒杀**：浏览每日 rotating flash sales
- 🥬 **生鲜选购**：检查产地直发、溯源码、新鲜度
- 💰 **价格对比**：普通价 vs 1号店会员价
- 🛒 **购物车操作**：登录态下添加商品
- 🎫 **优惠券**：自动检查免邮券、满减
- 🚚 **配送时段**：查看2小时配送窗口
- 📋 **订单预览**：生成完整订单信息（不含支付）

### v1.0 - 决策支持

- ✅ 每日秒杀时机策略
- ✅ 生鲜选购指南
- ✅ 1号店会员权益
- ✅ 价格对比建议
- ✅ 智能购物车技巧

## 安全边界

| 操作 | Agent | 用户 |
|------|-------|------|
| 搜索/浏览 | ✅ | - |
| 查看秒杀 | ✅ | - |
| 加购/用券 | ✅ | - |
| **支付/提交订单** | ❌ | ✅ |

## 工作流程

```
发现 → 对比 → 选择 → 加购 → 选配送 slot → 预览 → [用户支付]
```

```bash

```

```
用户：帮我买1号店的牛奶

```
