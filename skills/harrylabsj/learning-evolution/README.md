# Learning Evolution

Track, analyze, and evolve learning patterns from skill usage and user interactions.

## Quick Start

```bash
# Analyze usage patterns
./scripts/analyze-usage.sh --skill my-skill --period 30d

# Track effectiveness
./scripts/track-effectiveness.sh --skill my-skill

# Suggest evolutions
./scripts/suggest-evolutions.sh --skill my-skill
```

## Features

- **Usage Analysis**: Track how skills are used
- **Effectiveness Metrics**: Measure success rates
- **Pattern Recognition**: Identify usage patterns
- **Evolution Suggestions**: Recommend improvements
- **Version Comparison**: Compare skill versions

## Learning Dimensions

| Dimension | Description |
|-----------|-------------|
| usage | Frequency and patterns |
| effectiveness | Success and error rates |
| satisfaction | User feedback |
| adaptation | Evolution over time |

## Evolution Types

- **Incremental**: Small improvements
- **Breakthrough**: Major additions
- **Pivot**: Direction changes
- **Sunset**: Phase out
