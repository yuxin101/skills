# 抖音价格监控助手使用文档

## 抖音价格监控助手
专为抖音电商卖家打造的实时价格监控工具。

## 一、核心功能
- 自动解析抖音商品链接
- 实时价格降价提醒
- 企业微信告警推送
- 价格趋势追踪
- 多任务管理

## 二、版本说明
### 1. 免费基础版
- 最多支持 5 个监控任务
- 每月 10 次告警提醒
- 基础价格追踪功能

### 2. 专业付费版
- 最多支持 100 个监控任务
- 无限次告警提醒
- 完整功能开放
- 优先技术支持

### 3. 企业付费版
- 无限制监控任务
- 多店铺管理
- 定制化支持
- 私有化部署

## 三、安装方法
运行以下命令一键安装：

npx clawhub@latest install price-watcher-dy-mycar

## 四、使用方法
- 监控商品：监控抖音商品，价格低于 180 元时提醒
- 查看任务：显示我所有的监控任务
- 生成报告：生成商品价格分析报告

## 六、配置指南
### 配置步骤
1. 打开你的智能体技能配置页面
2. 找到 SKILL_CONFIG 配置项
3. 粘贴下方 JSON 内容，替换成你自己的信息即可：

{
  "douyin_cookie": "你的抖音网页Cookie（选填）",
  "wechat_webhook": "你的企业微信Webhook地址（选填）",
  "subscription_level": "basic"
}
