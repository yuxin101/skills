# V7 多周期短线策略

## 策略概述

基于**多周期分析**的短线交易策略，结合趋势策略和震荡策略，自适应市场状态。

## 核心逻辑

### 多周期框架
- **日线**: 定方向 (200MA + ADX)
- **4小时**: 定区间 (ADX判断趋势强度)
- **1小时**: 主信号 (策略入场)
- **15分钟**: 优化入场 (可选)

### 策略选择

| 市场状态 | 策略 | 说明 |
|---------|------|------|
| 趋势市 (ADX>25) | TrendStrategy | Supertrend + ADX过滤 |
| 震荡市 (ADX≤25) | OscillatorStrategy | 布林带 + RSI |

## 策略参数

### 趋势策略 (TrendStrategy)
- Supertrend周期: 20
- Supertrend乘数: 3.5
- ADX阈值: 20

### 震荡策略 (OscillatorStrategy)
- 布林带周期: 20
- 布林带标准差: 2.0
- RSI周期: 14
- 超卖线: 35
- 超买线: 65

### 风控参数
- 单笔风险: 1.5%
- 杠杆倍数: 4x
- 最大日亏损: 5%
- 最大日交易次数: 8次
- 止损ATR倍数: 1.5x
- 移动止损ATR倍数: 2.0x
