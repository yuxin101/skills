# Source Policy

## Priority Order
1. Government / official / filings
2. Mainstream media
3. Fact-check organizations
4. Social platforms (X, Reddit)

## Social Platform Use
- X/Reddit are supporting signals, not primary truth sources.
- Social-only evidence cannot produce a “true” verdict.

## Unreliable Source Handling
- If only unreliable sources are present, default to false/uncertain.

## Required Final Output Behavior
- Never show internal score to user.
- Provide integrated neutral narrative.
- Include limitation disclaimer.
